## Instrucciones de instalación  
Para usar este ejemplo, siga estos pasos:  
1. Descargar e instalar Aspose.Words para .NET  
   * Visite el[Página de descarga de Aspose.Words para .NET](https://releases.aspose.com/words/net/)y descargar la última versión.  
   * Extraer el paquete descargado y hacer referencia al`Aspose.Words.dll`archivo en su proyecto.  

2. Instalar a través de NuGet  
   * Abra su proyecto en Visual Studio.  
   * Vaya a Herramientas Administrador de paquetes de NuGet Administrar paquetes de NuGet para la solución.  
   * Buscar por:`Aspose.Words`en el Administrador de paquetes de NuGet e instalar el paquete.
