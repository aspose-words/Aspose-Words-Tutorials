## Instrucciones de instalación  

1. Descargar Aspose.Words para .NET  
   Obtenga la biblioteca desde la [Página de descarga de Aspose.Words](https://releases.aspose.com/words/net/).  

2. Instalar a través del Administrador de paquetes NuGet  
   - Abra su proyecto de Visual Studio.  
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.  
   - Buscar `Aspose.Words` en la pestaña Explorar.  
   - Seleccione el paquete y haga clic en Instalar.  

3. Instalar a través de la consola del administrador de paquetes NuGet  
   Ejecute el siguiente comando en la consola del administrador de paquetes:  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Agregar licencia  
   Para una funcionalidad completa, solicite una licencia. Puede obtener una licencia temporal en [Página de licencia temporal de Aspose](https://purchase.aspose.com/temporary-license/).
