# Insertar un gráfico de columnas simple en Word usando Aspose.Words

## Instrucciones de instalación  
Para usar este ejemplo, asegúrese de tener instalado Aspose.Words for .NET. Siga estos pasos:  

1. Descargar Aspose.Words para .NET:  
   Visita[Descargar Aspose.Words gratis para .NET](https://releases.aspose.com/words/net/)y descargar la última versión.  

2. Instalar la biblioteca:  
   * Agregue el paquete Aspose.Words NuGet a su proyecto. Ejecute el siguiente comando en la consola de NuGet Package Manager:  
     ```sh
     Install-Package Aspose.Words
     ```  
   * Alternativamente, añadir el`.dll`archivos a las referencias de su proyecto manualmente.  

3. Obtener una licencia:  
   * Obtener una licencia de[Aspose Compra](https://purchase.aspose.com/buy)o solicite una prueba gratuita de[Aspose Prueba gratuita](https://releases.aspose.com/).  

4. Aplicar la licencia:  
   Agregue el siguiente código a su proyecto para activar la licencia:  
   ```csharp
   License license = new License();
   license.SetLicense("path-to-your-license-file.lic");
   ```
