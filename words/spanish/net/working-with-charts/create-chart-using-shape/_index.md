## Instrucciones de instalación  

1. Descargar Aspose.Words para .NET:  
   Obtenga la última versión del sitio oficial [Página de descarga de Aspose.Words para .NET](https://releases.aspose.com/words/net/).  

2. Instalar mediante NuGet:  
   Agregue la biblioteca a su proyecto mediante el Administrador de paquetes NuGet. Ejecute el siguiente comando en la consola del Administrador de paquetes:  
   ```bash
   Install-Package Aspose.Words
   ```
