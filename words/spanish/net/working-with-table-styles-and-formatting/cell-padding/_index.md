## Instrucciones de instalación  
Para ejecutar el ejemplo y usar Aspose.Words para .NET, siga estos pasos:  

1. Descargar Aspose.Words para .NET  
   Consigue la última versión de Aspose.Words desde el[Aspose lanza página](https://releases.aspose.com/words/net/).  

2. Instalar a través de NuGet Package Manager  
   Abra su proyecto .NET en Visual Studio y ejecute el siguiente comando en NuGet Package Manager Console:  
   ```bash
   Install-Package Aspose.Words
   ```  

3. Configurar una versión temporal o con licencia  
   * Obtener un[Licencia Temporal Libre](https://purchase.aspose.com/temporary-license/)o comprar una licencia completa de la[Página de compra de Aspose](https://purchase.aspose.com/buy).  
   * Aplica la licencia en tu código:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
