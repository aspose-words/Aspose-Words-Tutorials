## Instrucciones de instalación  
Para ejecutar el ejemplo y usar Aspose.Words para .NET, siga estos pasos:  

1. Descarga Aspose.Words para .NET  
   Obtenga la última versión de Aspose.Words de la[Aspose libera la página](https://releases.aspose.com/words/net/)¿ Qué ?  

2. Instalar a través del administrador de paquetes NuGet  
   Abra su proyecto .NET en Visual Studio y ejecuta el siguiente comando en la consola de gestión de paquetes NuGet:  
   ```bash
   Install-Package Aspose.Words
   ```  

3. Establezca una versión temporal o con licencia  
   * Obtener una[licencia temporal gratuita](https://purchase.aspose.com/temporary-license/)o comprar una licencia completa de la [Aspire a comprar página](https://purchase.aspose.com/buy)¿ Qué ?  
   * Aplica la licencia en tu código:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
