## Instrucciones de instalación
1. Instalar Aspose.Words para .NET: Utilice NuGet Package Manager para instalar Aspose.Words. Puede ejecutar el siguiente comando en la consola de gestión de paquetes:
   ```
   Install-Package Aspose.Words
   ```

2. Create a New Project: Open Visual Studio and create a new C# project (Console Application or any type you prefer).

3. Añadir Referencia: asegúrese de que la biblioteca Aspose.Words sea referenciada en su proyecto.
