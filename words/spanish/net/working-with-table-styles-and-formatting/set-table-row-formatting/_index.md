## Instrucciones de instalación
1. Instalar Aspose.Words para .NET: Use el Gestor de paquetes NuGet para instalar Aspose.Words. Puede ejecutar el siguiente comando en la consola del Gestor de paquetes:
   ```
   Install-Package Aspose.Words
   ```

2. Crear un nuevo proyecto: abra Visual Studio y cree un nuevo proyecto C# (aplicación de consola o cualquier tipo que prefiera).

3. Agregar referencia: asegúrese de que la biblioteca Aspose.Words esté referenciada en su proyecto.

4. Solicitar licencia (opcional):  
   Para utilizar la versión completa, [solicitar una licencia](https://purchase.aspose.com/temporary-license/) o utilizar un [prueba gratuita](https://releases.aspose.com/words/net/).
