# Crear un documento de Word con una tabla usando Aspose.Words
  
Este ejemplo muestra cómo crear un documento de Word con una tabla usando la biblioteca Aspose.Words para .NET. El código inicializa un nuevo documento, construye una tabla con dos filas y aplica diferentes estilos y orientaciones a las celdas. Aprenderá a manipular el contenido y la alineación de las celdas, y a crear un documento estructurado mediante programación.

## Instrucciones de instalación:  
1. Instalar Aspose.Words para .NET:
   - Uso del administrador de paquetes NuGet:
     - Abra su proyecto en Visual Studio.
     - Haga clic con el botón derecho en el proyecto en el Explorador de soluciones y seleccione "Administrar paquetes NuGet".
     - Busque "Aspose.Words" y haga clic en "Instalar".

   - O utilice la consola del administrador de paquetes:
     ```
     Install-Package Aspose.Words
     ```
