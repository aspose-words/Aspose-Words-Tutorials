# Cree un documento de Word de varias páginas con Aspose.Words

Este ejemplo muestra cómo crear un documento de Word con Aspose.Words para .NET, lo que permite agregar contenido en varias páginas. Muestra cómo inicializar un nuevo documento, usar DocumentBuilder para insertar texto y saltos de página, y finalmente guardar el documento como archivo .docx. Ideal para desarrolladores que buscan automatizar la generación de documentos en sus aplicaciones .NET.

## Instrucciones de instalación
1. Instalar Aspose.Words para .NET: Use el Gestor de paquetes NuGet para instalar Aspose.Words. Ejecute el siguiente comando en la consola del Gestor de paquetes:
```
Install-Package Aspose.Words
```
