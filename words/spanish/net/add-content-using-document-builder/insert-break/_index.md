# Crear un documento de Word de varias páginas con Aspose.Words

Este ejemplo muestra cómo crear un documento de Word usando Aspose.Words para .NET, lo que le permite agregar contenido en varias páginas. Muestra cómo inicializar un nuevo documento, usar DocumentBuilder para insertar texto y saltos de página, y finalmente guardar el documento como un archivo .docx. Perfecto para desarrolladores que buscan automatizar la generación de documentos en sus aplicaciones .NET.

## Instrucciones de instalación
1. Instale Aspose.Words para .NET: Use NuGet Package Manager para instalar Aspose.Words. Ejecute el siguiente comando en la Consola del Administrador de paquetes:
```
Install-Package Aspose.Words
```
