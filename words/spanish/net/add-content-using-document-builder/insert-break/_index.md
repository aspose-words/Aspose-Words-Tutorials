# Crear un documento de palabra de varias páginas con Aspose.Words

Este ejemplo muestra cómo crear un documento de Word utilizando Aspose.Words para .NET, lo que le permite agregar contenido en varias páginas. Muestra cómo iniciar un nuevo documento, usar el DocumentBuilder para insertar texto y breaks de página y finalmente guardar el documento como un archivo .docx. Perfecto para los desarrolladores que buscan automatizar la generación de documentos en sus aplicaciones .NET.

## Instrucciones de instalación
1. Instalar Aspose.Words para .NET: Utilice NuGet Package Manager para instalar Aspose.Words. ejecuta el siguiente comando en la consola de gestión de paquetes:
```
Install-Package Aspose.Words
```
