## Instrucciones de instalación
1. Instalar Aspose.Words: Utilice el Administrador de paquetes de NuGet en Visual Studio.
   * Abra su proyecto en Visual Studio.
   * Haga clic con el botón derecho en su proyecto en el Explorador de soluciones.
   * Seleccione Administrar paquetes de NuGet.
   * Buscar por:`Aspose.Words`e instalarlo.

2. Agregar Uso de directiva: En el archivo de código, incluya la siguiente directiva para usar clases Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
