## Instrucciones de instalación
1. Instalar Aspose.Words: utilizar el Administrador de paquetes NuGet en Visual Studio.
   - Abra su proyecto en Visual Studio.
   - Haga clic derecho en su proyecto en el Explorador de soluciones.
   - Seleccione Administrar paquetes NuGet.
   - Buscar `Aspose.Words` e instalarlo.

2. Agregar directiva Using: En su archivo de código, incluya la siguiente directiva para usar las clases Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
