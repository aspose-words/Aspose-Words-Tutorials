## Instrucciones de instalación
1. Instalar Aspose.Words: Utilice el administrador de paquetes NuGet en Visual Studio.
   * Abre tu proyecto en Visual Studio.
   * Haga clic derecho en su proyecto en el Explorador de Soluciones.
   * Seleccione Gestionar los paquetes NuGet.
   * Busca por`"Aspose. Palabras"`y instalarlo.

2. Añadir la Directiva de Uso: En su archivo de código, incluya la siguiente directiva para usar las clases Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
