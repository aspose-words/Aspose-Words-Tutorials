# Crea un documento de Word con tabla de contenido
Este ejemplo muestra cómo usar Aspose.Words para .NET para crear un documento Word que incluye una Tabla de Contenidos interactiva (TOC) generada utilizando la clase DocumentBuilder. El código ilustra cómo insertar encabezados de varios niveles, formatarlos y guardar el documento resultante como un archivo .docx. Siga los pasos a continuación para una implementación fácil.

## Instrucciones de instalación
1. Prerequisitos: asegúrese de que tiene .NET instalado en su máquina.
2. Instalar Aspose.Words: 
   * Utilice la consola de gestión de paquetes NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * O, busque Aspose.Words en la interfaz gráfica de NuGet Package Manager.
