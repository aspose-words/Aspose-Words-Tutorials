# Crear un documento de Word con Tabla de contenido
En este ejemplo se muestra cómo utilizar Aspose.Words for .NET para crear un documento de Word que incluye una tabla de contenido interactiva (TOC) generada con la clase DocumentBuilder. El código ilustra cómo insertar encabezados de varios niveles, formatearlos y guardar el documento resultante como un archivo .docx. Siga los pasos a continuación para una implementación fácil.

## Instrucciones de instalación
1. Requisitos previos: Asegúrese de tener .NET instalado en su máquina.
2. Instalar Aspose.Words: 
   * Utilice la consola de administración de paquetes de NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * O bien, busque "Aspose.Words" en la GUI de NuGet Package Manager.
