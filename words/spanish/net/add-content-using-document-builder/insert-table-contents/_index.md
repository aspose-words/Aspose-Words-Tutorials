# Crear un documento de Word con tabla de contenido
Este ejemplo demuestra cómo usar Aspose.Words para .NET para crear un documento de Word que incluye una tabla de contenido (TOC) interactiva generada con la clase DocumentBuilder. El código ilustra cómo insertar encabezados de varios niveles, darles formato y guardar el documento resultante como un archivo .docx. Siga los pasos a continuación para una implementación sencilla.

## Instrucciones de instalación
1. Requisitos previos: asegúrese de tener .NET instalado en su máquina.
2. Instalar Aspose.Words: 
   - Utilice la consola del administrador de paquetes NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - O busque "Aspose.Words" en la GUI del Administrador de paquetes NuGet.
