# Crear un documento de Word con imagen flotante

Este ejemplo muestra cómo crear un documento de Word con Aspose.Words para .NET e insertar una imagen con un diseño flotante. El código proporcionado inicializa un documento y un DocumentBuilder, inserta una imagen con las dimensiones y la posición especificadas y, finalmente, guarda el documento. Esta guía es ideal para desarrolladores que buscan mejorar sus capacidades de generación de documentos con Aspose.Words.

## Instrucciones de instalación
1. Instalar Aspose.Words: Agregue el paquete NuGet Aspose.Words a su proyecto .NET. Puede hacerlo ejecutando el siguiente comando en la consola del Administrador de paquetes:
   ```
   Install-Package Aspose.Words
   ```

2. Agregar espacios de nombres obligatorios: asegúrese de que su archivo de código incluya los siguientes espacios de nombres:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
