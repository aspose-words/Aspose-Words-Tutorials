# Crear un documento de Word con una imagen flotante

Este ejemplo muestra cómo crear un documento de Word usando Aspose.Words para .NET e insertar una imagen con un diseño flotante. El código proporcionado inicializa un documento y DocumentBuilder, inserta una imagen con dimensiones y posicionamiento especificados y finalmente guarda el documento. Esta guía es ideal para desarrolladores que buscan mejorar sus capacidades de generación de documentos usando Aspose.Words.

## Instrucciones de instalación
1. Instale Aspose.Words: Agregue el paquete Aspose.Words NuGet a su proyecto .NET. Puede hacerlo ejecutando el siguiente comando en la Consola del administrador de paquetes:
   ```
   Install-Package Aspose.Words
   ```

2. Asegúrese de que su archivo de código incluya los siguientes espacios de nombres:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
