# Crear un documento de Word con imagen flotante

Este ejemplo demuestra cómo crear un documento de Word usando Aspose.Words para .NET e insertar una imagen con un diseño flotante. El código proporcionado inicializa un Document y DocumentBuilder, inserta una imagen con dimensiones y posicionamiento especificados y finalmente guarda el documento. Esta guía es ideal para los desarrolladores que buscan mejorar sus capacidades de generación de documentos utilizando Aspose.Words.

## Instrucciones de instalación
1. Instalar Aspose.Words: Agregue el paquete NuGet Aspose.Words a su proyecto .NET. Puede hacer esto ejecutando el siguiente comando en la consola de gestión de paquetes:
   ```
   Install-Package Aspose.Words
   ```

2. Añadir espacios de nombres requeridos: asegúrese de que su archivo de código incluya los siguientes espacios de nombres:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
