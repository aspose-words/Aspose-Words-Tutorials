Este ejemplo muestra cómo insertar una imagen en línea en un documento de Word usando la biblioteca Aspose.Words para .NET.`DocumentBuilder`En esta guía se proporciona una implementación sencilla para ayudarle a comenzar con el manejo de imágenes en documentos de Word.

## Instrucciones de instalación
1. Instalar Aspose.Words para .NET:
   * Utilice el Administrador de paquetes de NuGet en Visual Studio.
   * Ejecute el siguiente comando en la Consola del administrador de paquetes:
     ```
     Install-Package Aspose.Words
     ```

2. Configura tu proyecto:
   * Cree un nuevo proyecto .NET o abra uno existente.
   * Agregue las directivas necesarias usando en la parte superior de su archivo de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
