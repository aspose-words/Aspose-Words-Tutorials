# Insertar imagen en línea en un documento de Word usando Aspose.Words
Este ejemplo demuestra cómo insertar una imagen en línea en un documento de Word usando la biblioteca Aspose.Words para .NET. Al utilizar `DocumentBuilder` Con la clase [Aquí falta información], puede posicionar e insertar imágenes fácilmente en la posición actual del cursor en su documento. Esta guía proporciona una implementación sencilla para ayudarle a comenzar a gestionar imágenes en documentos de Word.

## Instrucciones de instalación
1. Instalar Aspose.Words para .NET:
   - Utilice el Administrador de paquetes NuGet en Visual Studio.
   - Ejecute el siguiente comando en la consola del administrador de paquetes:
     ```
     Install-Package Aspose.Words
     ```

2. Configura tu proyecto:
   - Cree un nuevo proyecto .NET o abra uno existente.
   - Agregue las directivas de uso necesarias en la parte superior de su archivo de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
