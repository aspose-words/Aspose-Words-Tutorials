Este ejemplo muestra cómo insertar una imagen en línea en un documento de Word utilizando la biblioteca Aspose.Words para .NET.`DocumentBuilder`En este manual se ofrece una implementación sencilla para ayudarle a comenzar con el manejo de imágenes en documentos de Word.

## Instrucciones de instalación
1. Instalar Aspose.Words para .NET:
   * Utilice el administrador de paquetes NuGet en Visual Studio.
   * Ejecute el siguiente comando en la consola de gestión de paquetes:
     ```
     Install-Package Aspose.Words
     ```

2. Establezca su proyecto:
   * Crear un nuevo proyecto .NET o abrir uno existente.
   * Añadir las directivas necesarias en la parte superior de su archivo de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
