# Crear listas de varios niveles en Word con Aspose.Words
Este ejemplo muestra cómo crear una lista multinivel en un documento de Word con Aspose.Words for .NET. Inicializa un nuevo documento y una instancia de DocumentBuilder, aplica un formato de numeración, crea elementos principales y secundarios y guarda el documento como un archivo .docx. Siga la guía paso a paso para formatear eficientemente el documento con listas numeradas.

## Instrucciones de instalación
Para usar Aspose.Words para .NET, siga estos pasos:
1. Descargar la biblioteca: Visitar la[Página de descarga de Aspose.Words para .NET](https://www.aspose.com/downloads/words/net).
2. Instalar a través de NuGet:
   * Abra su proyecto de Visual Studio.
   * Vaya a Herramientas Administrador de paquetes de NuGet Administrar paquetes de NuGet para la solución.
   * Buscar por:`Aspose.Words`y haga clic en Instalar.
3. Consulte la biblioteca en su código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Aplicar Licencia (Opcional):  
   Para utilizar la versión completa,[aplicar una licencia](https://purchase.aspose.com/temporary-license/)o usar un[prueba gratuita](https://releases.aspose.com/words/net/).
