# Crear formato de fuente en Word usando Aspose.Words

Esta guía proporciona un ejemplo simple de crear y formato de un documento de Word utilizando Aspose.Words para .NET. Aprenderá a personalizar las propiedades de fuente como tamaño, estilo y color, y luego escribir texto en el documento. Las instrucciones paso a paso hacen que sea fácil para los principiantes y desarrolladores experimentados incorporar Aspose.Words en sus aplicaciones.
