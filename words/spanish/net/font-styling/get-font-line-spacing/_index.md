# Obtener el interlineado de fuentes en un documento de Word
Aspose.Words para .NET simplifica la creación y manipulación de documentos de Word. Este ejemplo muestra cómo inicializar un documento de Word, configurar una fuente personalizada y recuperar el interlineado de una fuente específica. Tanto si automatiza la generación de documentos como si analiza estilos de texto, esta guía proporciona instrucciones claras y fragmentos de código prácticos para lograr sus objetivos de forma eficiente. Aspose.Words para .NET es una potente biblioteca para desarrolladores que trabajan con documentos de Word en C# o VB.NET.

## Instrucciones de instalación  
Para utilizar este ejemplo, siga estos pasos:  
1. Descargue e instale Aspose.Words para .NET  
   - Visita el [Página de descarga de Aspose.Words para .NET](https://releases.aspose.com/words/net/) y descargue la última versión.  
   - Extraiga el paquete descargado y haga referencia a `Aspose.Words.dll` archivo en su proyecto.  

2. Instalar mediante NuGet  
   - Abra su proyecto en Visual Studio.  
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.  
   - Buscar `Aspose.Words` en el Administrador de paquetes NuGet e instale el paquete.  

3. Solicitar una licencia (opcional pero recomendado)  
   - Obtener una licencia de la [Página de compra de Aspose](https://purchase.aspose.com/buy).  
   - Utilice un [licencia temporal](https://purchase.aspose.com/temporary-license/) para evaluación si es necesario.  
   - Agregue el siguiente código al comienzo de su aplicación para aplicar la licencia:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
