# Insertar una imagen en el encabezado de documento de Word

Aspose.Words for .NET simplifica el procesamiento de documentos de Word al permitir a los desarrolladores automatizar tareas como insertar imágenes en encabezados. Este tutorial muestra cómo usar las clases Document y DocumentBuilder para agregar una imagen con alineación, dimensiones y opciones de envoltura precisas al encabezado principal de un documento de Word. El proceso es sencillo, lo que lo hace ideal para los desarrolladores que buscan mejorar las plantillas de documentos dinámicamente.

## Instrucciones de instalación  

Para comenzar con Aspose.Words para .NET, siga estos sencillos pasos:  

1. Descargar Aspose.Words  
   * [Descargar la última versión](https://releases.aspose.com/words/net/)de Aspose.Words para .NET.  

2. Instalar Aspose.Words vía NuGet  
   * Abra su proyecto en Visual Studio.  
   * Vaya al menú Herramientas y seleccione Administrador de paquetes NuGet Administrar paquetes NuGet para la solución.  
   * Buscar por:`Aspose.Words`en el gestor de paquetes NuGet e instálelo.  

   O bien, utilice la Consola del administrador de paquetes:  
   ```shell
   Install-Package Aspose.Words
   ```  

3. Aplicar la licencia  
   * Obtener un[Licencia de prueba gratuita](https://releases.aspose.com/)o[Comprar una licencia](https://purchase.aspose.com/buy).  
   * Utilice el siguiente código para aplicar la licencia en su proyecto:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
