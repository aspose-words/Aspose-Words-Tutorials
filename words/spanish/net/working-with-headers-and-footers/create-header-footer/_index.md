# Crear un documento de Word con encabezado y pie de página usando Aspose.Words

Este ejemplo muestra cómo usar Aspose.Words for .NET para crear un documento de Word con un encabezado y un pie de página principales. La guía lo guía a través de la inicialización de un nuevo documento, pasando a las secciones de encabezado y pie de página, agregando texto personalizado y guardando el documento en formato DOCX. Con su poderosa clase DocumentBuilder, Aspose.Words hace que la manipulación de documentos de Word sea fluida y eficiente.

## Instrucciones de instalación  
1. Descargar Aspose.Words para .NET: Obtener la última versión de la[Página de lanzamientos de Aspose](https://releases.aspose.com/words/net/).  
2. Instalar Aspose.Words:  
   * Utilice el Administrador de paquetes de NuGet:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   * O haga referencia manual a la DLL del paquete descargado.  
3. Aplicar Licencia (Opcional): Obtener un[Licencia temporal](https://purchase.aspose.com/temporary-license/)o una licencia completa de[Aspose Compra](https://purchase.aspose.com/buy)y aplícalo en tu proyecto:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
