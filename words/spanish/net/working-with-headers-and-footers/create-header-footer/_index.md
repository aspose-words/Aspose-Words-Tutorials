# Crea un documento de Word con encabezado y pie de página usando Aspose.Words

Este ejemplo demuestra cómo usar Aspose.Words para .NET para crear un documento Word con un encabezado y pie de página primarios. La guía lo guía a través de la inicialización de un nuevo documento, pasando a las secciones de encabezado y pie de página, añadiendo texto personalizado y guardando el documento en formato DOCX. Con su potente clase DocumentBuilder, Aspose.Words hace que la manipulación de documentos Word sea fluida y eficiente. Ya sea que esté agregando elementos de marca o números de página, este ejemplo proporciona la base para personalizar documentos Word de manera programática.
