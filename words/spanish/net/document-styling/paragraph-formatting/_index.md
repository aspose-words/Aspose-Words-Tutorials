# Formatear párrafos con Aspose.Words
Esta guía ofrece un ejemplo de cómo dar formato a párrafos en un documento de Word con Aspose.Words para .NET. Aprenderá a configurar diversas opciones de formato de párrafo, como la alineación, la sangría y el espacio entre párrafos. El ejemplo ilustra el uso de las clases Document y DocumentBuilder, lo que permite crear documentos bien estructurados mediante programación.

## Instrucciones de instalación
1. Descargue Aspose.Words para .NET: Visite el sitio [Sitio web de Aspose](https://www.aspose.com/downloads/words/net) para descargar la última versión de Aspose.Words para .NET.

2. Agregar referencia: en su proyecto .NET, haga clic con el botón derecho en el proyecto en el Explorador de soluciones, seleccione Agregar > Referencia y busque para agregar el archivo DLL Aspose.Words.

3. Uso del paquete NuGet (opcional): como alternativa, puede instalar Aspose.Words a través del Administrador de paquetes NuGet con el siguiente comando:
   ```
   Install-Package Aspose.Words
   ```

4. Importar espacio de nombres: agregue la siguiente declaración using a su archivo de código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

5. Solicitar licencia (opcional):  
   Para utilizar la versión completa, [solicitar una licencia](https://purchase.aspose.com/temporary-license/) o utilizar un [prueba gratuita](https://releases.aspose.com/words/net/).
