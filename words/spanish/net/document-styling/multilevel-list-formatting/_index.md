# Crea listas de varios niveles en Word con Aspose.Words
Este ejemplo muestra cómo crear una lista de varios niveles en un documento de Word utilizando Aspose.Words para .NET. Siga la guía paso a paso para formatar eficientemente su documento con listas numeradas.

## Instrucciones de instalación
Para usar Aspose.Words para .NET, siga estos pasos:
1. Descarga la biblioteca: Visita el [Aspose.Words para la página de descarga de .NET](https://www.aspose.com/downloads/words/net)¿ Qué ?
2. Instalar a través de NuGet:
   * Abre tu proyecto de Visual Studio.
   * Navegar a Herramientas > Gerente de paquetes NuGet > Gestionar paquetes NuGet para Solución.
   * Busca por`"Aspose. Palabras"`y haga clic en Instalar.
3. Referencia la biblioteca en su código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
