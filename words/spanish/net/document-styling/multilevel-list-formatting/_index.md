## Instrucciones de instalación
Para utilizar Aspose.Words para .NET, siga estos pasos:
1. Descargar la biblioteca: Visita la [Página de descarga de Aspose.Words para .NET](https://www.aspose.com/downloads/words/net).
2. Instalar mediante NuGet:
   - Abra su proyecto de Visual Studio.
   - Vaya a Herramientas > Administrador de paquetes NuGet > Administrar paquetes NuGet para la solución.
   - Buscar `Aspose.Words` y haga clic en Instalar.
3. Haga referencia a la biblioteca en su código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Solicitar licencia (opcional):  
   Para utilizar la versión completa, [solicitar una licencia](https://purchase.aspose.com/temporary-license/) o utilizar un [prueba gratuita](https://releases.aspose.com/words/net/).
