# Crear un documento de Word con encabezado y pie de página usando Aspose.Words

Este ejemplo demuestra cómo usar Aspose.Words para .NET para crear un documento de Word con encabezado y pie de página principales. La guía le guía paso a paso por la inicialización de un nuevo documento, el acceso a las secciones de encabezado y pie de página, la adición de texto personalizado y el guardado del documento en formato DOCX. Gracias a su potente clase DocumentBuilder, Aspose.Words facilita la manipulación de documentos de Word. Tanto si añade elementos de marca como números de página, este ejemplo proporciona las bases para personalizar documentos de Word mediante programación.

## Instrucciones de instalación  
1. Descargue Aspose.Words para .NET: Obtenga la última versión desde [Página de lanzamiento de Aspose](https://releases.aspose.com/words/net/).  
2. Instalar Aspose.Words:  
   - Utilice el Administrador de paquetes NuGet:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   - O haga referencia manualmente a la DLL desde el paquete descargado.  
3. Solicitar Licencia (Opcional): Obtener una [licencia temporal](https://purchase.aspose.com/temporary-license/) o una licencia completa de [Compra de Aspose](https://purchase.aspose.com/buy) y aplicarlo en tu proyecto:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
