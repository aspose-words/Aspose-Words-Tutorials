# Insertar formas en documentos de Word usando Aspose.Words
Esta guía muestra cómo usar Aspose.Words para .NET para insertar y manipular formas, como cuadros de texto, en documentos de Word. Aprenderá cómo colocar formas absoluta o relativamente, establecer ángulos de rotación y guardar el documento en formato DOCX de transición ISO 29500:2008 utilizando`OoxmlSaveOptions`Este ejemplo simplifica el trabajo con formas en documentos de Word, lo que permite a los desarrolladores crear contenido visualmente atractivo con facilidad.

## Instrucciones de instalación  
1. Descargar Aspose.Words para .NET:  
   Obtener la última versión de Aspose.Words for .NET desde el[página de descarga](https://releases.aspose.com/words/net/).  

2. Instalar a través de NuGet:  
   Abra su proyecto en Visual Studio e instale la biblioteca con NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Referencia del espacio de nombres:  
   En su código, incluya el espacio de nombres para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Activar la licencia (opcional):  
   Si tiene una licencia válida, aplíquela en su proyecto para evitar limitaciones. Puede[Comprar una licencia](https://purchase.aspose.com/buy)o conseguir un[Licencia temporal](https://purchase.aspose.com/temporary- license/).
