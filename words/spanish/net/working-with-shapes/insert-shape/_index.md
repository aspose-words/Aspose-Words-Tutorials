# Insertar formas en documentos de palabra usando Aspose.Words
Esta guía muestra cómo usar Aspose.Words para .NET para insertar y manipular formas, como cuadros de texto, en documentos de Word. Aprenderá cómo posicionar formas absolutamente o relativamente, establecer ángulos de rotación y guardar el documento en el formato ISO 29500:2008 DOCX de transición utilizando `OoxmlSaveOptions`Este ejemplo simplifica el trabajo con formas en documentos de Word, permitiendo a los desarrolladores crear contenido visualmente atractivo con facilidad.

## Instrucciones de instalación  
1. Descarga Aspose.Words para .NET:  
   Obtenga la última versión de Aspose.Words para .NET desde el [página de descarga](https://releases.aspose.com/words/net/)¿ Qué ?  

2. Instalar a través de NuGet:  
   Abra su proyecto en Visual Studio e instale la biblioteca usando NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Añadir referencia del espacio de nombres:  
   En su código, incluya el espacio de nombres para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
