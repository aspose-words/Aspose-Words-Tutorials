# Insertar formas en documentos de Word con Aspose.Words
Esta guía muestra cómo usar Aspose.Words para .NET para insertar y manipular formas, como cuadros de texto, en documentos de Word. Aprenderá a posicionar formas de forma absoluta o relativa, establecer ángulos de rotación y guardar el documento en formato DOCX transicional ISO 29500:2008. `OoxmlSaveOptions`Este ejemplo simplifica el trabajo con formas en documentos de Word, lo que permite a los desarrolladores crear contenido visualmente atractivo con facilidad.

## Instrucciones de instalación  
1. Descargar Aspose.Words para .NET:  
   Obtenga la última versión de Aspose.Words para .NET desde [página de descarga](https://releases.aspose.com/words/net/).  

2. Instalar mediante NuGet:  
   Abra su proyecto en Visual Studio e instale la biblioteca usando el Administrador de paquetes NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Agregar referencia de espacio de nombres:  
   En su código, incluya el espacio de nombres para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Activar la licencia (opcional):  
   Si tiene una licencia válida, aplíquela en su proyecto para evitar limitaciones. Puede [comprar una licencia](https://purchase.aspose.com/buy) o conseguir uno [licencia temporal](https://purchase.aspose.com/temporary-license/).
