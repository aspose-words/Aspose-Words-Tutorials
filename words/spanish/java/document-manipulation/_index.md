## Respuestas rápidas
- **¿Qué biblioteca le permite crear documentos Word en Java?** Aspose.Words for Java.
- **¿Puedo convertir DOCX a PDF con la misma API?** Sí – una línea de código maneja la conversión.
- **¿Se admite la extracción de texto plano de un archivo Word?** Absolutamente, usando el método Document.getText().

## Preguntas frecuentes

**Q: ¿Cómo crear programáticamente un documento Word en Java?**  
A: Use las clases `Document` y `DocumentBuilder` de Aspose.Words for Java. Esto le permite añadir párrafos, tablas, imágenes y más sin Microsoft Office.

**Q: ¿Puedo convertir un archivo DOCX a PDF en una sola línea de código?**  
A: Sí. Cargue el DOCX con `new Document("input.docx")` y llame a `save("output.pdf", SaveFormat.PDF)`.

**Q: ¿Cuál es la mejor manera de extraer texto plano de un archivo Word?**  
A: Llame a `document.getText()` o recorra los nodos `Paragraph` para obtener el contenido de texto sin formato.

**Q: ¿Cómo puedo generar automáticamente una tabla de contenido?**  
A: Inserte un campo TOC usando `DocumentBuilder.insertTableOfContents()` y luego actualice los campos con `document.updateFields()`.

**Q: ¿Es posible combinar varios documentos Word en un solo archivo?**  
A: Absolutamente. Use `targetDocument.appendDocument(sourceDocument, ImportFormatMode.KEEP_SOURCE_FORMATTING)` para cada documento que desee combinar.
