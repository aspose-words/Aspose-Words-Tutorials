## Qué es **cómo añadir marca de agua**?
Añadir una marca de agua significa superponer texto o una imagen sobre cada página de un documento. Se utiliza comúnmente para “Confidencial”, “Borrador” o fines de marca. Con Aspose.Words for Java, puedes insertar marcas de agua programáticamente durante el proceso de **convert docx to pdf** o cualquier otra operación de exportación.

## ¿Por qué usar Aspose.Words for Java para añadir marcas de agua?
- **Full‑control API** – Insertar texto, imágenes o formas como marcas de agua sin abrir el archivo en Office.  
- **Cross‑format support** – El mismo código funciona para DOCX, PDF, HTML y más.  
- **Performance** – Conversión y marcación de agua a alta velocidad incluso en archivos grandes.  
- **Customization** – Ajustar opacidad, rotación, color y posición al instante.

## Preguntas Frecuentes

**Q: ¿Cómo puedo añadir una marca de agua de texto a un PDF generado a partir de un archivo DOCX?**  
A: Usa `DocumentBuilder.insertWatermarkText("Your Text")` antes de llamar a `document.save("output.pdf")`. Esto funciona durante el paso **convert docx to pdf**.

**Q: ¿Es posible añadir una marca de agua solo a ciertas páginas?**  
A: Sí. Inserta la marca de agua dentro de un bucle que verifica las propiedades `PageSetup` de las páginas que deseas dirigir.

**Q: ¿Puedo combinar la generación de códigos de barras con la marcación de agua?**  
A: Absolutamente. Genera la imagen del código de barras y luego insértala como una marca de agua de imagen usando `DocumentBuilder.insertWatermarkImage(barcodeImage)`.

**Q: ¿Qué formatos admiten marcas de agua?**  
A: Todos los formatos de exportación compatibles con Aspose.Words (PDF, DOCX, HTML, ODT, etc.) conservan las marcas de agua.

**Q: ¿Necesito una licencia separada para funciones de códigos de barras o gráficos?**  
A: No. Todas las funciones están incluidas en la licencia estándar de Aspose.Words for Java.
