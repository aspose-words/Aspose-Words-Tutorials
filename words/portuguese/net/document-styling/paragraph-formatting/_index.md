# Formatar parágrafos usando Aspose.Words
Este guia fornece um exemplo de como formatar parágrafos em um documento do Word usando o Aspose.Words para .NET. Você aprenderá a definir diversas opções de formatação de parágrafo, como alinhamento, recuo e espaçamento entre parágrafos. O exemplo ilustra o uso das classes Document e DocumentBuilder, permitindo a criação de documentos bem estruturados programaticamente.

## Instruções de instalação
1. Baixe Aspose.Words para .NET: Visite o [Site Aspose](https://www.aspose.com/downloads/words/net) para baixar a versão mais recente do Aspose.Words para .NET.

2. Adicionar referência: no seu projeto .NET, clique com o botão direito do mouse no projeto no Solution Explorer, selecione Adicionar > Referência e navegue para adicionar o arquivo DLL Aspose.Words.

3. Usando o pacote NuGet (opcional): Como alternativa, você pode instalar o Aspose.Words por meio do Gerenciador de pacotes NuGet com o seguinte comando:
   ```
   Install-Package Aspose.Words
   ```

4. Importar namespace: adicione a seguinte instrução using ao seu arquivo de código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

5. Aplicar licença (opcional):  
   Para usar a versão completa, [aplicar uma licença](https://purchase.aspose.com/temporary-license/) ou usar um [teste gratuito](https://releases.aspose.com/words/net/).
