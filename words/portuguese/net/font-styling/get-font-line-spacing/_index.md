# Obtenha espaçamento de linha de fonte em documento do Word
O Aspose.Words para .NET simplifica a criação e a manipulação de documentos do Word. Este exemplo demonstra como inicializar um documento do Word, definir uma fonte personalizada e recuperar o espaçamento entre linhas de uma fonte específica. Seja para automatizar a geração de documentos ou analisar estilos de texto, este guia fornece instruções claras e trechos de código práticos para atingir seus objetivos com eficiência. O Aspose.Words para .NET é uma biblioteca poderosa para desenvolvedores que trabalham com documentos do Word em C# ou VB.NET.

## Instruções de instalação  
Para usar este exemplo, siga estas etapas:  
1. Baixe e instale o Aspose.Words para .NET  
   - Visite o [Página de download do Aspose.Words para .NET](https://releases.aspose.com/words/net/) e baixe a versão mais recente.  
   - Extraia o pacote baixado e faça referência ao `Aspose.Words.dll` arquivo em seu projeto.  

2. Instalar via NuGet  
   - Abra seu projeto no Visual Studio.  
   - Acesse Ferramentas > Gerenciador de Pacotes NuGet > Gerenciar Pacotes NuGet para Solução.  
   - Procurar `Aspose.Words` no Gerenciador de Pacotes NuGet e instale o pacote.  

3. Aplicar uma licença (opcional, mas recomendado)  
   - Obtenha uma licença do [Página de compra do Aspose](https://purchase.aspose.com/buy).  
   - Use um [licença temporária](https://purchase.aspose.com/temporary-license/) para avaliação, se necessário.  
   - Adicione o seguinte código no início do seu aplicativo para aplicar a licença:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
