# Insira formas em documentos de palavras usando Aspose.Words
Este guia demonstra como usar Aspose.Words para .NET para inserir e manipular formas, como caixas de texto, em documentos do Word. Você aprenderá como posicionar formas absolutamente ou relativamente, definir ângulos de rotação e salvar o documento no formato ISO 29500:2008 DOCX de transição usando `OoxmlSaveOptions`Este exemplo simplifica o trabalho com formas em documentos do Word, permitindo aos desenvolvedores criar conteúdo visualmente atraente com facilidade.

## Instruções de instalação  
1. Descarregue Aspose.Words para .NET:  
   Obtenha a versão mais recente de Aspose.Words para .NET do [página de download](https://releases.aspose.com/words/net/)\- Não .  

2. Instalação através do NuGet:  
   Abra o seu projeto no Visual Studio e instale a biblioteca usando o NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Adicionar Referência de Espaço de Nome:  
   No seu código, inclua o espaço de nomes para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
