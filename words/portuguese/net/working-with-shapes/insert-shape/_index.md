# Inserir formas em documentos do Word usando Aspose.Words
Este guia demonstra como usar o Aspose.Words for .NET para inserir e manipular formas, como caixas de texto, em documentos do Word. Você aprenderá como posicionar formas absoluta ou relativamente, definir ângulos de rotação e salvar o documento no formato DOCX de transição ISO 29500:2008 usando`OoxmlSaveOptions`. Este exemplo simplifica o trabalho com formas em documentos do Word, permitindo que os desenvolvedores criem conteúdo visualmente atraente com facilidade.

## Instruções de instalação  
1. Baixar Aspose.Words for .NET:  
   Obtenha a versão mais recente do Aspose.Words for .NET a partir do[página de download](https://releases.aspose.com/words/net/).  

2. Instalar via NuGet:  
   Abra seu projeto no Visual Studio e instale a biblioteca usando o NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Adicionar Namespace Referência:  
   No seu código, inclua o namespace para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Ativar a Licença (Opcional):  
   Se você tiver uma licença válida, aplique-a em seu projeto para evitar limitações. Você pode[comprar uma licença](https://purchase.aspose.com/buy)ou obter a[Licença temporária](https://purchase.aspose.com/temporary-license/).
