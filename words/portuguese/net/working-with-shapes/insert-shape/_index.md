# Inserir formas em documentos do Word usando Aspose.Words
Este guia demonstra como usar o Aspose.Words para .NET para inserir e manipular formas, como caixas de texto, em documentos do Word. Você aprenderá a posicionar formas de forma absoluta ou relativa, definir ângulos de rotação e salvar o documento no formato ISO 29500:2008 Transitional DOCX usando `OoxmlSaveOptions`. Este exemplo simplifica o trabalho com formas em documentos do Word, permitindo que os desenvolvedores criem conteúdo visualmente atraente com facilidade.

## Instruções de instalação  
1. Baixe Aspose.Words para .NET:  
   Obtenha a versão mais recente do Aspose.Words para .NET em [página de download](https://releases.aspose.com/words/net/).  

2. Instalar via NuGet:  
   Abra seu projeto no Visual Studio e instale a biblioteca usando o Gerenciador de Pacotes NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Adicionar referência de namespace:  
   No seu código, inclua o namespace para Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Ativar a licença (opcional):  
   Se você possui uma licença válida, aplique-a em seu projeto para evitar limitações. Você pode [comprar uma licença](https://purchase.aspose.com/buy) ou pegue um [licença temporária](https://purchase.aspose.com/temporary-license/).
