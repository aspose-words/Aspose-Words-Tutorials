# Crie um documento do Word com cabeçalho e rodapé usando Aspose.Words

Este exemplo demonstra como usar o Aspose.Words para .NET para criar um documento do Word com cabeçalho e rodapé primários. O guia explica como inicializar um novo documento, navegar pelas seções de cabeçalho e rodapé, adicionar texto personalizado e salvar o documento no formato DOCX. Com sua poderosa classe DocumentBuilder, o Aspose.Words torna a manipulação de documentos do Word simples e eficiente. Seja adicionando elementos de identidade visual ou numeração de páginas, este exemplo fornece a base para personalizar documentos do Word programaticamente.

## Instruções de instalação  
1. Baixe Aspose.Words para .NET: Obtenha a versão mais recente do [Página de lançamentos do Aspose](https://releases.aspose.com/words/net/).  
2. Instalar Aspose.Words:  
   - Use o Gerenciador de Pacotes NuGet:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   - Ou faça referência manual à DLL do pacote baixado.  
3. Aplicar licença (opcional): obter uma [licença temporária](https://purchase.aspose.com/temporary-license/) ou uma licença completa de [Aspose Compra](https://purchase.aspose.com/buy) e aplique-o em seu projeto:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
