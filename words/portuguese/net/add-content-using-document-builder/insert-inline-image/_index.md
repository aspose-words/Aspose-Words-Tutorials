# Inserir imagem embutida em documento do Word usando Aspose.Words
Este exemplo demonstra como inserir uma imagem embutida em um documento do Word usando a biblioteca Aspose.Words para .NET. Ao utilizar a `DocumentBuilder` Com a classe, você pode posicionar e inserir imagens facilmente no local atual do cursor no seu documento. Este guia fornece uma implementação simples para ajudar você a começar a lidar com imagens em documentos do Word.

## Instruções de instalação
1. Instalar Aspose.Words para .NET:
   - Use o Gerenciador de Pacotes NuGet no Visual Studio.
   - Execute o seguinte comando no Console do Gerenciador de Pacotes:
     ```
     Install-Package Aspose.Words
     ```

2. Configure seu projeto:
   - Crie um novo projeto .NET ou abra um existente.
   - Adicione as diretivas using necessárias no início do seu arquivo de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
