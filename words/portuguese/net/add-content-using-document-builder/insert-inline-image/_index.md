# Insira imagem de linha em documento de Word usando Aspose.Words
Este exemplo demonstra como inserir uma imagem de linha em um documento do Word usando a biblioteca Aspose.Words para .NET.`DocumentBuilder`classe, você pode facilmente posicionar e inserir imagens na localização do cursor atual em seu documento. Este guia fornece uma implementação simples para ajudá-lo a começar com o tratamento de imagens em documentos do Word.

## Instruções de instalação
1. Instale Aspose.Words para .NET:
   * Use o Gerente de Pacotes NuGet no Visual Studio.
   * Execute o seguinte comando no Console do Gestor de Pacotes:
     ```
     Install-Package Aspose.Words
     ```

2. Configure o seu projeto:
   * Crie um novo projeto .NET ou abra um existente.
   * Adicione as instruções necessárias no topo do seu ficheiro de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
