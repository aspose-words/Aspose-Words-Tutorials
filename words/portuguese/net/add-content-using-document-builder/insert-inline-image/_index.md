# Inserir imagem em linha no documento do Word usando Aspose.Words
Este exemplo demonstra como inserir uma imagem em linha em um documento do Word usando a biblioteca Aspose.Words para .NET. Ao utilizar o`Document Builder`classe, você pode facilmente posicionar e inserir imagens no local do cursor atual em seu documento. Este guia fornece uma implementação direta para ajudá-lo a começar com o manuseio de imagens em documentos do Word.

## Instruções de instalação
1. Instale Aspose.Words para .NET:
   * Use o NuGet Package Manager no Visual Studio.
   * Execute o seguinte comando no Console do Gerenciador de Pacotes:
     ```
     Install-Package Aspose.Words
     ```

2. Configure seu projeto:
   * Crie um novo projeto .NET ou abra um já existente.
   * Adicione as diretivas necessárias na parte superior do seu arquivo de código:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
