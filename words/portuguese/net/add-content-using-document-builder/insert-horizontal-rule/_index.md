Este exemplo demonstra o uso do Aspose.Words for .NET para criar um documento do Word e adicionar uma regra horizontal usando o DocumentBuilder. Siga as etapas abaixo para aprender como formatar seus documentos de forma eficaz.

## Instruções de instalação
1. Instale Aspose.Words: Use o Gerenciador de Pacotes NuGet no Visual Studio.
   * Abra seu projeto no Visual Studio.
   * Clique com o botão direito do mouse em seu projeto no Solution Explorer.
   * Selecione Gerenciar pacotes NuGet.
   * Pesquisar por`Aspose.Words`e instalá-lo.

2. Adicionar Usando Diretiva: Em seu arquivo de código, inclua a seguinte diretiva para usar as classes Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
