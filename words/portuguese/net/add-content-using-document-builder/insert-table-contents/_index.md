# Crie um documento do Word com índice
Este exemplo demonstra como usar o Aspose.Words para .NET para criar um documento do Word que inclui um Sumário (TOC) interativo gerado usando a classe DocumentBuilder. O código ilustra como inserir títulos de vários níveis, formatá-los e salvar o documento resultante como um arquivo .docx. Siga os passos abaixo para uma implementação fácil.

## Instruções de instalação
1. Pré-requisitos: certifique-se de ter o .NET instalado na sua máquina.
2. Instalar Aspose.Words: 
   - Use o console do gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - Ou procure por "Aspose.Words" na GUI do Gerenciador de Pacotes NuGet.
