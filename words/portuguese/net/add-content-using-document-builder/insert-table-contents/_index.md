# Criar um documento de palavra com tabela de conteúdo
Este exemplo demonstra como usar Aspose.Words para .NET para criar um documento Word que inclui uma Tabela de Conteúdos interativa (TOC) gerada usando a classe DocumentBuilder. O código ilustra como inserir cabeçalhos de vários níveis, formatá-los e salvar o documento resultante como um arquivo .docx. Siga os passos abaixo para uma implementação fácil.

## Instruções de instalação
1. Preerequisitos: Certifique-se de ter o .NET instalado na sua máquina.
2. Instalar Aspose.Words: 
   * Use a Consola de Gestor de Pacotes NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Ou, procure por "Aspose.Words" na GUI do Gerente de Pacotes NuGet.
