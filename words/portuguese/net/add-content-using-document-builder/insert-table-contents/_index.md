# Criar um Documento de Word com ndice
Este exemplo demonstra como usar o Aspose.Words for .NET para criar um documento do Word que inclui uma tabela interativa de conteúdo (TOC) gerada usando a classe DocumentBuilder. O código ilustra como inserir cabeçalhos de vários níveis, formatá-los e salvar o documento resultante como um arquivo .docx. Siga as etapas abaixo para uma implementação fácil.

## Instruções de instalação
1. Pré-requisitos: Certifique-se de ter o .NET instalado em sua máquina.
2. Instale Aspose.Words: 
   * Usar o console do gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Ou, procure por "Aspose.Words" na GUI do NuGet Package Manager.
