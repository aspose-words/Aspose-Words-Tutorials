# Crie um documento de palavra com tabela usando Aspose.Words
  
Este exemplo demonstra como criar um documento Word com uma tabela usando a biblioteca Aspose.Words para .NET. O código inicializa um novo documento, constrói uma tabela com duas linhas e formata as células com vários estilos e orientações. Você aprenderá a manipular o conteúdo da célula, alinhamento e construir um documento estruturado programaticamente.

## Instruções de instalação:  
1. Instale Aspose.Words para .NET:
   * Usando o Gerente de Pacotes NuGet:
     * Abre o teu projeto no Visual Studio.
     * Clique com o botão direito no projeto no Solution Explorer e selecione "Gerenciar Pacotes NuGet".
     * Pesquise em Aspose.Words e clique em "Install".

   * Ou, use o Console do Gestor de Pacotes:
     ```
     Install-Package Aspose.Words
     ```
