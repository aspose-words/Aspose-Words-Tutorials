# Crie um documento do Word com tabela usando Aspose.Words
  
Este exemplo demonstra como criar um documento do Word com uma tabela usando a biblioteca Aspose.Words para .NET. O código inicializa um novo documento, constrói uma tabela com duas linhas e formata as células com vários estilos e orientações. Você aprenderá a manipular o conteúdo e o alinhamento das células e a criar um documento estruturado programaticamente.

## Instruções de instalação:  
1. Instalar Aspose.Words para .NET:
   - Usando o Gerenciador de Pacotes NuGet:
     - Abra seu projeto no Visual Studio.
     - Clique com o botão direito do mouse no projeto no Solution Explorer e selecione "Gerenciar pacotes NuGet".
     - Procure por "Aspose.Words" e clique em "Instalar".

   - Ou use o Console do Gerenciador de Pacotes:
     ```
     Install-Package Aspose.Words
     ```
