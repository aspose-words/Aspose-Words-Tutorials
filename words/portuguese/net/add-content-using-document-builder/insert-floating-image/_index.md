# Crie documento de texto com imagem flutuante

Este exemplo demonstra como criar um documento Word usando Aspose.Words para .NET e inserir uma imagem com um layout flutuante. O código fornecido inicializa um Document and DocumentBuilder, inserir uma imagem com dimensões e posicionamento especificados e, finalmente, salva o documento. Este guia é ideal para desenvolvedores que desejam melhorar suas capacidades de geração de documentos usando Aspose.Words.

## Instruções de instalação
1. Instale Aspose.Words: Adicione o pacote NuGet Aspose.Words ao seu projeto .NET. Você pode fazer isso executando o seguinte comando no Console do Gerente de Pacotes:
   ```
   Install-Package Aspose.Words
   ```

2. Adicionar espaços de nomes necessários: Certifique-se de que o seu arquivo de código inclui os seguintes espaços de nomes:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
