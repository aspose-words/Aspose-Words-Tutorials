# Criar documento do Word com imagem flutuante

Este exemplo demonstra como criar um documento do Word usando o Aspose.Words para .NET e inserir uma imagem com um layout flutuante. O código fornecido inicializa um Document e um DocumentBuilder, insere uma imagem com dimensões e posicionamento especificados e, por fim, salva o documento. Este guia é ideal para desenvolvedores que buscam aprimorar seus recursos de geração de documentos usando o Aspose.Words.

## Instruções de instalação
1. Instalar o Aspose.Words: Adicione o pacote NuGet Aspose.Words ao seu projeto .NET. Você pode fazer isso executando o seguinte comando no Console do Gerenciador de Pacotes:
   ```
   Install-Package Aspose.Words
   ```

2. Adicionar namespaces necessários: certifique-se de que seu arquivo de código inclua os seguintes namespaces:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
