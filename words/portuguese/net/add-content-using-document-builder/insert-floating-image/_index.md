# Criar documento Word com imagem flutuante

Este exemplo demonstra como criar um documento do Word usando Aspose.Words for .NET e inserir uma imagem com um layout flutuante. O código fornecido inicializa um Documento e DocumentBuilder, insere uma imagem com dimensões e posicionamento especificados e, finalmente, salva o documento. Este guia é ideal para desenvolvedores que desejam aprimorar suas capacidades de geração de documentos usando Aspose.Words.

## Instruções de instalação
1. Instale Aspose.Words: Adicione o pacote Aspose.Words NuGet ao seu projeto .NET. Você pode fazer isso executando o seguinte comando no Console do Gerenciador de Pacotes:
   ```
   Install-Package Aspose.Words
   ```

2. Adicione espaços de nomes obrigatórios: Certifique-se de que seu arquivo de código inclua os seguintes espaços de nomes:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
