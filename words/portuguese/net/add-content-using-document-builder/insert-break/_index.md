# Crie um documento de palavra de várias páginas com Aspose.Words

Este exemplo demonstra como criar um documento Word usando Aspose.Words para .NET, permitindo adicionar conteúdo em várias páginas. Ele mostra como inicializar um novo documento, usar o DocumentBuilder para inserir texto e pausas de página e, finalmente, salvar o documento como um arquivo .docx. Perfeito para desenvolvedores que desejam automatizar a geração de documentos em seus aplicativos .NET.

## Instruções de instalação
1. Instale Aspose.Words para .NET: Use o NuGet Package Manager para instalar Aspose.Words. Execute o seguinte comando no Console do Gerente de Pacotes:
```
Install-Package Aspose.Words
```
