# Crie um documento Word de várias páginas com Aspose.Words

Este exemplo demonstra como criar um documento do Word usando o Aspose.Words para .NET, permitindo adicionar conteúdo em várias páginas. Ele mostra como inicializar um novo documento, usar o DocumentBuilder para inserir texto e quebras de página e, por fim, salvar o documento como um arquivo .docx. Perfeito para desenvolvedores que buscam automatizar a geração de documentos em seus aplicativos .NET.

## Instruções de instalação
1. Instalar o Aspose.Words para .NET: Use o Gerenciador de Pacotes NuGet para instalar o Aspose.Words. Execute o seguinte comando no Console do Gerenciador de Pacotes:
```
Install-Package Aspose.Words
```
