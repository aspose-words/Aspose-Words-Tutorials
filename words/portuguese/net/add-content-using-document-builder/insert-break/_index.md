# Crie um documento do Word de várias páginas com Aspose.Words

Este exemplo demonstra como criar um documento do Word usando Aspose.Words for .NET, permitindo que você adicione conteúdo em várias páginas. Ele mostra como inicializar um novo documento, use o DocumentBuilder para inserir quebras de texto e página e, finalmente, salve o documento como um arquivo .docx. Perfeito para desenvolvedores que desejam automatizar a geração de documentos em seus aplicativos .NET.

## Instruções de instalação
1. Instale Aspose.Words for .NET: Use o Gerenciador de Pacotes NuGet para instalar o Aspose.Words. Execute o seguinte comando no Console do Gerenciador de Pacotes:
```
Install-Package Aspose.Words
```
