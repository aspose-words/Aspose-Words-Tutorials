# Formatar parágrafos usando Aspose.Words
Este guia fornece um exemplo de como formatar parágrafos em um documento do Word usando Aspose.Words para .NET. Você aprenderá como definir várias opções de formatamento de parágrafos, como alinhamento, indentamento e espaço após parágrafos. O exemplo ilustra o uso das classes Document e DocumentBuilder, permitindo a criação de documentos bem estruturados programaticamente.

## Instruções de instalação
1. Descarregue Aspose.Words para .NET: Visite o [Aspose site](https://www.aspose.com/downloads/words/net)Para baixar a versão mais recente de Aspose.Words para .NET.

2. Adicionar Referência: No seu projeto .NET, clique com o botão direito no projeto no Solution Explorer, selecione Adicionar > Referência e navegue para adicionar o arquivo DLL Aspose.Words.

3. Usando o pacote NuGet (opcional): Alternativamente, você pode instalar Aspose.Words através do Gerente de Pacote NuGet com o seguinte comando:
   ```
   Install-Package Aspose.Words
   ```

4. Importar espaço de nomes: Adicione o seguinte usando instrução ao seu arquivo de código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
