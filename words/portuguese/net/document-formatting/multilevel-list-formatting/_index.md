# Criar listas de vários níveis em palavra com Aspose.Words
Este exemplo demonstra como criar uma lista multinível em um documento do Word usando Aspose.Words for .NET. Ele inicializa um novo documento e uma instância do DocumentBuilder, aplica um formato de numeração, cria itens principais e sub-itens e salva o documento como um arquivo .docx. Siga o guia passo-a-passo para formatar eficientemente seu documento com listas numeradas.

## Instruções de instalação
Para usar o Aspose.Words for .NET, siga estes passos:
1. Baixe a biblioteca: Visite o[Página de download do Aspose.Words for .NET](https://www.aspose.com/downloads/words/net).
2. Instalar via NuGet:
   * Abra seu projeto do Visual Studio.
   * Navegue até Ferramentas > NuGet Package Manager > Gerenciar pacotes NuGet para solução.
   * Pesquisar por`Aspose.Words`e clique em Instalar.
3. Consulte a biblioteca em seu código:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Aplicar Licença (Opcional):  
   Para usar a versão completa,[aplicar uma licença](https://purchase.aspose.com/temporary-license/)ou utilizar a[Teste grátis](https://releases.aspose.com/words/net/).
