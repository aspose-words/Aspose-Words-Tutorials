# Inserir campo de formulário de caixa de combinação em documento do Word

O Aspose.Words para .NET facilita a criação e a manipulação programática de documentos do Word. Neste exemplo, você aprenderá a inserir um campo de formulário de caixa de combinação em um documento do Word. A caixa de combinação inclui itens predefinidos como "Um", "Dois" e "Três", com uma seleção padrão. Este tutorial passo a passo demonstra o uso de `DocumentBuilder` para criar documentos dinâmicos e interativos do Word sem esforço. Perfeito para desenvolvedores que buscam aprimorar seus aplicativos .NET com formulários personalizáveis do Word.

## Instruções de instalação  

Para usar o exemplo, você precisa do Aspose.Words para .NET instalado em seu ambiente de desenvolvimento. Siga estes passos:  

1. Baixe Aspose.Words para .NET  
   Obtenha a versão mais recente do [Downloads do Aspose](https://releases.aspose.com/words/net/) página.  

2. Instalar via NuGet  
   Abra seu projeto no Visual Studio e instale a biblioteca usando o Gerenciador de Pacotes NuGet:  
   - No Console do Gerenciador de Pacotes, digite:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

3. Aplicar licença (opcional)  
   Para funcionalidade completa, aplique uma licença:  
   - Compre uma licença da [Aspose Comprar](https://purchase.aspose.com/buy) página ou solicitar uma [Licença Temporária](https://purchase.aspose.com/temporary-license/).  
   - Carregue a licença no seu código:  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("path-to-license-file.lic");  
     ```
