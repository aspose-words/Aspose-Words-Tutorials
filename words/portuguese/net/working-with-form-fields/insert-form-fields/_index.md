# Insira um campo de formulário combo em documento de texto

Aspose.Words para .NET facilita a criação e manipulação programática de documentos do Word. Neste exemplo, você aprenderá como inserir um campo de formulário combo em um documento do Word. A caixa combo inclui itens predefinidos como "Um," "Duas," e "Tres," com um conjunto de seleção padrão. Este tutorial passo a passo demonstra o uso de `DocumentBuilder`Perfeito para desenvolvedores que procuram melhorar suas aplicações .NET com formulários Word personalizáveis.

## Instruções de instalação  

Para usar o exemplo, você precisa de Aspose.Words para .NET instalado no seu ambiente de desenvolvimento. Siga estes passos:  

1. Descarregue Aspose.Words para .NET  
   Obtenha a versão mais recente do [Aspone Downloads](https://releases.aspose.com/words/net/)página.  

2. Instalação através de NuGet  
   Abra o seu projeto no Visual Studio e instale a biblioteca usando o NuGet Package Manager:  
   * No Console do Gestor de Pacotes, digite:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

3. Licença de inscrição (opcional)  
   Para a funcionalidade completa, aplique uma licença:  
   * Comprar uma licença do [Pretenda comprar](https://purchase.aspose.com/buy)página ou solicitar um [Licença temporária](https://purchase.aspose.com/temporary-license/)\- Não .  
   * Carregue a licença no seu código:  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("path-to-license-file.lic");  
     ```
