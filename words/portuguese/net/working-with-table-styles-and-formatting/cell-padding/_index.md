## Instruções de instalação  
Para executar o exemplo e usar o Aspose.Words para .NET, siga estes passos:  

1. Baixar Aspose.Words for .NET  
   Obtenha a versão mais recente do Aspose.Words a partir do[Página de Lançamentos da Asposição](https://releases.aspose.com/words/net/).  

2. Instalar via NuGet Package Manager  
   Abra o seu projeto .NET no Visual Studio e execute o seguinte comando no Console do NuGet Package Manager:  
   ```bash
   Install-Package Aspose.Words
   ```  

3. Configurar uma versão temporária ou licenciada  
   * obter a[Licença temporária livre](https://purchase.aspose.com/temporary-license/)ou comprar uma licença completa do[Aspose Buy Page](https://purchase.aspose.com/buy).  
   * Aplique a licença em seu código:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
