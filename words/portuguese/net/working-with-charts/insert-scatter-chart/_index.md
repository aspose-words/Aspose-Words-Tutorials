# Criar gráfico de dispersão de palavras usando Aspose.Words

Aspose.Words for .NET simplifica a criação e manipulação de documentos do Word de forma programática. Este exemplo demonstra como inserir um gráfico de dispersão em um documento do Word. Ao inicializar um documento, usando o`Document Builder`para criação de conteúdo e inserção de dados de gráficos de forma programática, você pode adicionar gráficos de dispersão de nível profissional aos seus documentos de forma eficiente. Com sua API intuitiva, o Aspose.Words simplifica a automação de documentos para desenvolvedores .NET, economizando tempo e esforço.

## Instruções de instalação  
Para executar este exemplo, siga estes passos:  

1. Baixar e instalar Aspose.Words for .NET  
   * Baixe a biblioteca do site oficial da Aspose:[Aspose.Words for .NET Downloads](https://releases.aspose.com/words/net/).  
   * Instalar via NuGet Package Manager:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

2. Configure seu ambiente de desenvolvimento  
   * Abra o seu .NET IDE (por exemplo, Visual Studio).  
   * Crie um novo aplicativo de console ou integre-se a um projeto existente.  
   * Adicione uma referência à biblioteca Aspose.Words for .NET.  

3. Aplicar Licença  
   * Se você tiver uma licença, aplique-a em seu projeto usando:  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("Your_License_File.lic");  
     ```  
   * Em alternativa, obter a[Licença temporária livre](https://purchase.aspose.com/temporary-license/)para testar a biblioteca.
