## Instruções de instalação  
Para executar este exemplo, siga estes passos:  

1. Descarregar e instalar Aspose.Words para .NET  
   * Faça o download da biblioteca do site oficial da Aspose: [Aspose.Words para Downloads .NET](https://releases.aspose.com/words/net/)\- Não .  
   * Instalar através do Gerente de Pacotes NuGet:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

2. Configure seu ambiente de desenvolvimento  
   * Abra o seu IDE .NET (por exemplo, Visual Studio).  
   * Crie uma nova aplicação console ou integra-a num projeto existente.  
   * Adicione uma referência à Aspose.Words para a biblioteca .NET.  

3. Aplicar por licença  
   * Se tiver uma licença, aplique-a no seu projecto utilizando:  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("Your_License_File.lic");  
     ```  
   * Alternativamente, obtenha um[licença temporária gratuita](https://purchase.aspose.com/temporary-license/)Para testar a biblioteca.
