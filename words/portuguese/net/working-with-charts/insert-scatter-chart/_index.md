# Crie um gráfico de dispersão de palavras usando Aspose.Words

O Aspose.Words para .NET simplifica a criação e a manipulação programática de documentos do Word. Este exemplo demonstra como inserir um gráfico de dispersão em um documento do Word. Ao inicializar um documento, usando o `DocumentBuilder` Para criação de conteúdo e inserção programática de dados de gráficos, você pode adicionar gráficos de dispersão de nível profissional aos seus documentos com eficiência. Com sua API intuitiva, o Aspose.Words simplifica a automação de documentos para desenvolvedores .NET, economizando tempo e esforço.

## Instruções de instalação  
Para executar este exemplo, siga estas etapas:  

1. Baixe e instale o Aspose.Words para .NET  
   - Baixe a biblioteca do site oficial da Aspose: [Downloads do Aspose.Words para .NET](https://releases.aspose.com/words/net/).  
   - Instalar via Gerenciador de Pacotes NuGet:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

2. Configure seu ambiente de desenvolvimento  
   - Abra seu IDE .NET (por exemplo, Visual Studio).  
   - Crie um novo aplicativo de console ou integre-o a um projeto existente.  
   - Adicione uma referência à biblioteca Aspose.Words para .NET.  

3. Aplicar licença  
   - Se você tiver uma licença, aplique-a em seu projeto usando:  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("Your_License_File.lic");  
     ```  
   - Alternativamente, obtenha um [licença temporária gratuita](https://purchase.aspose.com/temporary-license/) para testar a biblioteca.
