# Crie um documento com gráficos usando Aspose.Words

Este tutorial demonstra como criar um documento do Word com um gráfico usando o Aspose.Words para .NET. Você aprenderá a inserir um gráfico de colunas, limpar séries existentes, adicionar novos dados e personalizar os limites do eixo Y. Com instruções passo a passo, incluindo inicialização, manipulação de dados e salvamento do documento como um arquivo DOCX, este guia simplifica o trabalho com gráficos no Aspose.Words para .NET. Seja para desenvolver relatórios profissionais ou visualizações dinâmicas de dados, este exemplo ajuda você a começar sem esforço.

## Instruções de instalação  

1. Baixe Aspose.Words para .NET  
   Obtenha a versão mais recente do Aspose.Words para .NET em [página oficial de download](https://releases.aspose.com/words/net/).  

2. Instalar via Gerenciador de Pacotes NuGet  
   - Abra seu projeto no Visual Studio.  
   - Navegue até Ferramentas > Gerenciador de Pacotes NuGet > Gerenciar Pacotes NuGet para Solução.  
   - Procurar `Aspose.Words` na aba Navegar.  
   - Selecione o pacote e clique em Instalar.  

3. Adicionar licença *(Opcional)*  
   - Se você possui uma licença, aplique-a em seu projeto. Para mais detalhes, visite [como solicitar uma licença](https://purchase.aspose.com/temporary-license/).  

4. Incluir namespaces  
   Certifique-se de adicionar os namespaces necessários em seu `.cs` arquivo:  
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   using Aspose.Words.Drawing.Charts;
   ```
