## Instruções de instalação  

1. Baixar Aspose.Words for .NET:  
   Obtenha a versão mais recente do oficial[Página de download do Aspose.Words for .NET](https://releases.aspose.com/words/net/).  

2. Instalar via NuGet:  
   Adicione a biblioteca ao seu projeto usando o NuGet Package Manager. Execute o seguinte comando no Console do Package Manager:  
   ```bash
   Install-Package Aspose.Words
   ```  
