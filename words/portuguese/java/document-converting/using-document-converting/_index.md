## Como Converter docx para pdf java usando Aspose.Words for Java

Vamos percorrer um guia passo a passo sobre como utilizar o recurso de conversão de documentos do Aspose.Words for Java. Neste exemplo, converteremos um documento Word para PDF.

### Etapa 1: Configurando o Ambiente

Antes de começar, certifique‑se de que você baixou e instalou o Aspose.Words for Java em seu projeto. Obtenha uma licença válida no site da Aspose para evitar quaisquer limitações de avaliação.

### Etapa 2: Carregando o Documento

Para começar, carregue o documento Word que você deseja converter:

```java
// Load the source Word document
Document doc = new Document("input.docx");
```

### Etapa 3: Convertendo para PDF

Em seguida, execute a conversão **docx to pdf java** salvando o documento como um arquivo PDF:

```java
// Save the document as PDF

```
