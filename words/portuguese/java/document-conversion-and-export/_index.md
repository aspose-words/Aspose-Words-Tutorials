## Perguntas Frequentes

**Q: Posso converter DOCX para PDF em um servidor Linux?**  
A: Sim. Aspose.Words for Java é independente de plataforma e funciona em qualquer SO com um JRE suportado.

**Q: Preciso ter o Microsoft Office instalado no servidor?**  
A: Não. A conversão é realizada inteiramente pela biblioteca Aspose.Words.

**Q: Como posso preservar fontes personalizadas durante a conversão?**  
A: Use `PdfSaveOptions.setEmbedFullFonts(true)` para incorporar todas as fontes usadas no PDF resultante.

**Q: É possível converter arquivos DOCX protegidos por senha?**  
A: Absolutamente. Carregue o documento com `LoadOptions.setPassword("yourPassword")` antes de salvá‑lo como PDF.

**Q: Qual é a maneira recomendada de converter em lote muitos documentos?**  
A: Crie um método reutilizável que carregue, converta e descarte cada documento, e execute‑o em um pool de threads para processamento paralelo.
