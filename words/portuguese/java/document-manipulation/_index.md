## Respostas Rápidas
- **Qual biblioteca permite criar documentos Word em Java?** Aspose.Words for Java.  
- **Posso converter DOCX para PDF com a mesma API?** Sim – uma única linha de código realiza a conversão.

## Perguntas Frequentes

**Q: Como criar programaticamente um documento Word em Java?**  
A: Use as classes `Document` e `DocumentBuilder` do Aspose.Words for Java. Isso permite adicionar parágrafos, tabelas, imagens e muito mais sem o Microsoft Office.

**Q: Posso converter um arquivo DOCX para PDF em uma única linha de código?**  
A: Sim. Carregue o DOCX com `new Document("input.docx")` e chame `save("output.pdf", SaveFormat.PDF)`.

**Q: Qual a melhor forma de extrair texto puro de um arquivo Word?**  
A: Chame `document.getText()` ou itere pelos nós `Paragraph` para obter o conteúdo de texto bruto.

**Q: Como gerar um índice automaticamente?**  
A: Insira um campo TOC usando `DocumentBuilder.insertTableOfContents()` e depois atualize os campos com `document.updateFields()`.

**Q: É possível mesclar vários documentos Word em um único arquivo?**  
A: Absolutamente. Use `targetDocument.appendDocument(sourceDocument, ImportFormatMode.KEEP_SOURCE_FORMATTING)` para cada documento que desejar combinar.
