## O que é Control Page Numbering Java?
Control page numbering Java refere‑se ao conjunto de chamadas de API que permitem definir onde os números de página aparecem, qual formato eles usam e como são reiniciados entre seções. Essa capacidade é essencial para criar documentos com aparência profissional, como contratos, manuais e relatórios de múltiplos capítulos.

## Perguntas frequentes

**Q: Posso mudar o formato do número de página (ex.: romano para arábico) programaticamente?**  
A: Sim, você pode definir a propriedade `PageNumberStyle` no campo `PageNumber` para qualquer formato suportado.

**Q: Como faço para pular números de página em páginas específicas, como páginas de título?**  
A: Insira um campo `PageNumber` com o interruptor `\* MERGEFORMAT` e defina o atributo `Suppress` no cabeçalho/rodapé daquela seção.

**Q: É possível reiniciar a numeração de páginas para cada capítulo automaticamente?**  
A: Absolutamente. Use a classe `Section` para criar uma nova seção para cada capítulo e habilite `PageSetup.RestartPageNumbering`.

**Q: A API lida eficientemente com documentos grandes?**  
A: Aspose.Words processa documentos de forma streaming, minimizando o uso de memória mesmo para arquivos com milhares de páginas.

**Q: Quais opções de licenciamento estão disponíveis para uso em produção?**
