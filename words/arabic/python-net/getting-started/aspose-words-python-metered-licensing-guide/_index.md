## دليل التنفيذ

### تفعيل التراخيص المقاسة

#### ملخص

تتيح لك هذه الميزة مراقبة كيفية استخدام تطبيقك لـ Aspose.Words، مما يوفر رؤى حول الاستهلاك والاعتمادات.

#### التنفيذ خطوة بخطوة

**1. تهيئة الترخيص المقنن**

ابدأ بإنشاء `Metered` المثال وضبط مفاتيحك:

```python
import aspose.words as aw

metered = aw.Metered()
metered.set_metered_key('YourPublicKey', 'YourPrivateKey')
```

**2. تتبع الاستخدام قبل التشغيل**

اطبع بيانات الائتمان والاستهلاك الأولية لفهم خط الأساس:

```python
print('Credit before operation:', metered.get_consumption_credit())
print('Consumption quantity before operation:', metered.get_consumption_quantity())
```

**3. إجراء عمليات المستندات**

استخدم Aspose.Words لمعالجة المستندات، مثل تحويل مستند Word إلى PDF:

```python
doc = aw.Document('path_to_your_document.docx')
doc.save('output_path.pdf')
```

**4. مراقبة الاستخدام بعد التشغيل**

بعد العملية، تحقق من مقدار التغيير الذي حدث في الائتمان والاستهلاك:

```python
import time

# انتظر للتأكد من إرسال البيانات إلى الخادم
time.sleep(10)  

print('Credit after operation:', metered.get_consumption_credit())
print('Consumption quantity after operation:', metered.get_consumption_quantity())
```
