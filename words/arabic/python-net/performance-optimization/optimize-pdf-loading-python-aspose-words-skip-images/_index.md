### تخطي صور PDF أثناء التحميل
يمكن أن يكون تخطي الصور أمرًا بالغ الأهمية للتطبيقات التي تتطلب فقط محتوى نصيًا من ملف PDF، مما يؤدي إلى تحسين أوقات التحميل وتقليل استخدام الذاكرة.

#### الخطوة 1: تحديد مسارات المستندات الخاصة بك
أولاً، حدد المسارات لمستندات الإدخال والإخراج:
```python
YOUR_DOCUMENT_DIRECTORY = 'path/to/your/documents/'
YOUR_OUTPUT_DIRECTORY = 'path/to/output/directory/'

def skip_pdf_images_demo():
    file_name = YOUR_DOCUMENT_DIRECTORY + 'Images.pdf'
```
#### الخطوة 2: تكوين PdfLoadOptions
إنشاء `PdfLoadOptions` قم بتكوين المثيل وتكوينه لتخطي الصور أو تضمينها:
```python
for is_skip_pdf_images in [True, False]:
    options = aw.loading.PdfLoadOptions()
    options.skip_pdf_images = is_skip_pdf_images
    options.page_index = 0
    options.page_count = 1
```
- **حدود**:
  - `skip_pdf_images`:قيمة منطقية لتحديد ما إذا كان ينبغي تخطي الصور أم لا.
  - `page_index` و `page_count`:حدد صفحات PDF التي سيتم تحميلها.

#### الخطوة 3: تحميل المستند
قم بتحميل المستند بالخيارات المحددة:
```python
doc = aw.Document(file_name=file_name, load_options=options)
```

#### الخطوة 4: التحقق من تحميل الصورة
التحقق من وجود الصور بناءً على التكوين:
```python
shape_collection = doc.get_child_nodes(aw.NodeType.SHAPE, True)

if is_skip_pdf_images:
    assert shape_collection.count == 0, 'Expected no images when skipping PDF images'
else:
    assert shape_collection.count != 0, 'Expected some images when not skipping PDF images'
# تنفيذ العرض التوضيحي
skip_pdf_images_demo()
```
