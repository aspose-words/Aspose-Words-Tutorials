## دليل التنفيذ

سيرشدك هذا القسم خلال عملية تحسين الإشارات المرجعية في PDF باستخدام Aspose.Words.

### إنشاء وإدارة الإشارات المرجعية

#### ملخص
تتيح الإشارات المرجعية في ملفات PDF للمستخدمين التنقل بسرعة بين أقسامها. بإدارتها بفعالية، تُحسّن تجربة المستخدم بشكل ملحوظ.

#### التنفيذ خطوة بخطوة

##### إضافة إشارات مرجعية باستخدام مستويات المخطط التفصيلي

يمكنك إضافة إشارات مرجعية وتعيين مستويات المخطط التفصيلي لإنشاء هيكل هرمي:

```python
builder = aw.DocumentBuilder(doc)
# ابدأ إشارة مرجعية باسم "الإشارة المرجعية 1"
builder.start_bookmark('Bookmark 1')
builder.writeln('Text inside Bookmark 1.')
builder.end_bookmark('Bookmark 1')

# إضافة إشارات مرجعية متداخلة
builder.start_bookmark('Bookmark 2')
builder.writeln('Text inside Nested Bookmark.')
builder.end_bookmark('Bookmark 2')
```

##### تكوين مستويات المخطط التفصيلي لتصدير PDF

تحدد مستويات المخطط التفصيلي كيفية عرض الإشارات المرجعية في القائمة المنسدلة:

```python
pdf_save_options = aw.saving.PdfSaveOptions()
outline_levels = pdf_save_options.outline_options.bookmarks_outline_levels
outline_levels.add('Bookmark 1', 1)
outline_levels.add('Bookmark 2', 2)

# حفظ المستند مع الإشارات المرجعية المحددة
doc.save('output.pdf', save_options=pdf_save_options)
```

##### إزالة ومسح الإشارات المرجعية

لتعديل بنية الإشارة المرجعية:

```python
# إزالة إشارة مرجعية محددة حسب الاسم
outline_levels.remove('Bookmark 2')

# مسح جميع مستويات المخطط التفصيلي، وتعيين الإشارات المرجعية إلى الوضع الافتراضي
outline_levels.clear()
```

### نصائح استكشاف الأخطاء وإصلاحها
- **مشكلة شائعة**:إذا لم تظهر الإشارات المرجعية بالشكل المتوقع في ملفات PDF، فتأكد من حفظ المستند باستخدام `PdfSaveOptions`.
- **تصحيح الأخطاء**:استخدم عبارات الطباعة أو التسجيل للتحقق من أسماء الإشارات المرجعية ومستويات المخطط التفصيلي.
