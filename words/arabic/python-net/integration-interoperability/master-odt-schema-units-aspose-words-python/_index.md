## دليل التنفيذ

### متوافق مع إصدارات مخطط ODT

#### ملخص

لضمان التوافق مع إصدارات محددة من مواصفات OpenDocument (مخطط ODT)، يسمح لك Aspose.Words بتحديد ما إذا كان يجب على مستندك الالتزام بشكل صارم بمواصفات الإصدار 1.1.

**خطوة بخطوة:**

##### الخطوة 1: إعداد خيارات الحفظ
```python
import aspose.words as aw

doc = aw.Document('path/to/your/input.docx')
save_options = aw.saving.OdtSaveOptions()
```

##### الخطوة 2: تكوين إصدار مخطط ODT
```python
# تم ضبطه على True للامتثال الصارم لإصدار ODT 1.1
save_options.is_strict_schema11 = True
```

##### الخطوة 3: حفظ المستند
```python
doc.save('path/to/your/output.odt', save_options)
```

### تكوين وحدات القياس

#### ملخص

يتيح لك Aspose.Words الاختيار بين الوحدات المترية (السنتيمتر) والإمبراطورية (البوصة) عند حفظ المستندات بتنسيق ODT. تضمن هذه المرونة توافق معلمات أسلوبك مع المعايير المطلوبة.

**خطوة بخطوة:**

##### الخطوة 1: اختيار وحدة القياس
```python
save_options = aw.saving.OdtSaveOptions()
# اختر بين السنتيمترات أو البوصات بناءً على احتياجاتك
save_options.measure_unit = aw.saving.OdtSaveMeasureUnit.CENTIMETERS
```

##### الخطوة 2: حفظ المستند بالوحدات
```python
doc.save('path/to/your/output.odt', save_options)
```

### تشفير مستندات ODT/OTT

#### ملخص

يتيح لك Aspose.Words تأمين مستنداتك بتشفيرها. يتناول هذا القسم كيفية استخدام كلمة مرور عند حفظ ملف ODT أو OTT.

**خطوة بخطوة:**

##### الخطوة 1: تهيئة خيارات المستند والحفظ
```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Hello world!")
save_options = aw.saving.OdtSaveOptions(aw.SaveFormat.ODT)
```

##### الخطوة 2: تعيين حماية كلمة المرور
```python
# تعيين كلمة مرور للتشفير
save_options.password = 'your_password_here'
doc.save('path/to/encrypted_output.odt', save_options)
```
