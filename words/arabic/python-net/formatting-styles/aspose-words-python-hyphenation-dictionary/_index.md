## دليل التنفيذ

### تسجيل قاموس الوصلات

#### ملخص
يتيح تسجيل القاموس لـ Aspose.Words تطبيق قواعد الوصل الخاصة بالموقع المحلي، والحفاظ على تدفق النص في الإعدادات متعددة اللغات.

#### عملية خطوة بخطوة

**1. تحديد الدلائل**

قم بتحديد المسارات لمستند الإدخال ودليل الإخراج:

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY'
arartifacts_directory = 'YOUR_OUTPUT_DIRECTORY'
```

**2. تسجيل القاموس**

استخدم Aspose.Words لتسجيل قاموس الوصلات لموقع "de-CH".

```python
aw.Hyphenation.register_dictionary('de-CH', document_directory + 'hyph_de_CH.dic')
```
*حدود:*
- `'de-CH'`:معرف الموقع.
- `document_directory + 'hyph_de_CH.dic'`:المسار إلى ملف قاموس الوصل.

**3. التحقق من التسجيل**

تأكد من تسجيل القاموس بشكل صحيح:

```python
assert aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be registered"
```

### تطبيق الوصلة

افتح مستندًا واحفظه مع تطبيق الواصلة عليه باستخدام القاموس المسجل حديثًا:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.registered.pdf')
```

### إلغاء تسجيل قاموس الوصلات

#### ملخص
يؤدي إلغاء التسجيل إلى إزالة القواعد الخاصة بالموقع المحلي، والعودة إلى سلوك الوصل الافتراضي.

**1. إلغاء تسجيل القاموس**

```python
aw.Hyphenation.unregister_dictionary('de-CH')
```
*غاية:* يقوم بإزالة تسجيل القاموس "de-CH" لمنع استخدامه في معالجة المستندات في المستقبل.

**2. التحقق من إلغاء التسجيل**

تأكد من أن القاموس لم يعد نشطًا:

```python
assert not aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be unregistered"
```

### الحفظ بدون استخدام الواصلة

أعد فتح مستندك وحفظه، هذه المرة دون تطبيق قواعد الوصل المسجلة مسبقًا:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.unregistered.pdf')
```
