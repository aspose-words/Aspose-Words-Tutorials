## دليل التنفيذ

### الميزة 1: استخدام التوقيع الرقمي

#### ملخص

توضح هذه الميزة كيفية إنشاء حامل شهادة رقمية واستخدامه لتوقيع المستندات. تتضمن تهيئة الشهادة، وتحميل المستند، وتطبيق توقيع رقمي باستخدام Aspose.Words.

#### التنفيذ خطوة بخطوة

**1. تهيئة حامل الشهادة**

إنشاء مثيل لـ `CertificateHolderExample` مع مسار الشهادة الرقمية وكلمة المرور الخاصة بك:

```python
certificate_holder = CertificateHolderExample("path/to/certificate.pfx", "your_password")
```

**2. التوقيع على الوثيقة**

استخدم `sign_document` طريقة تطبيق التوقيع:

```python
signature_image_data = open("path/to/signature.png", "rb").read()
certificate_holder.sign_document(
    "source.docx",
    "signed_output.docx",
    signer_id="SignatureLineID",
    image_data=signature_image_data
)
```

**توضيح:**
- `src_document_path`:المسار إلى المستند الذي تريد التوقيع عليه.
- `dst_document_path`:حيث سيتم حفظ المستند الموقع.
- `signer_id`:معرف لسطر التوقيع ضمن مستندك.
- `image_data`:مجموعة بايتات من صورة التوقيع.

#### خيارات تكوين المفاتيح

تأكد من أن شهادتك الرقمية صالحة وسهلة الوصول. تعامل مع الاستثناءات المتعلقة بمسارات الملفات أو كلمات المرور غير الصحيحة بسلاسة.

### الميزة 2: إدراج سطر التوقيع وتكوينه

#### ملخص

تتيح لك هذه الميزة إدراج سطر توقيع في مستند Word، والذي يمكن ملؤه لاحقًا بتوقيع رقمي فعلي.

#### التنفيذ خطوة بخطوة

**1. تهيئة SignatureLineExample**

قم بإعداد خيارات سطر التوقيع باستخدام معلومات التوقيع الخاصة بك:

```python
signature_line_example = SignatureLineExample("John Doe", "Manager", "SignatureLineID")
```

**2. أدخل سطر التوقيع**

يستخدم `insert_signature_line` لإضافة سطر توقيع إلى مستندك:

```python
document_path = "your_document.docx"
signature_line_object = signature_line_example.insert_signature_line(document_path)
```

**توضيح:**
- `document_path`:المسار إلى مستند Word الذي تريد إدراج سطر التوقيع فيه.
- يعيد `SignatureLine` كائن لمزيد من التلاعب إذا لزم الأمر.

#### خيارات تكوين المفاتيح

خصّص سطر التوقيع بخصائص إضافية، مثل التاريخ وسبب التوقيع. تأكد من `person_id` يتوافق مع نظام التتبع الداخلي الخاص بك.
