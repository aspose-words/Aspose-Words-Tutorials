# كيفية إصلاح الروابط المعطلة في ملفات CHM باستخدام Aspose.Words للغة Python

## مقدمة

هل تواجه مشاكل في الروابط المعطلة في ملفات .chm؟ قد تُسبب هذه المشكلة الشائعة الإحباط وتؤثر على سهولة استخدام مستندات المساعدة. في هذا البرنامج التعليمي، سنستكشف كيفية التعامل بكفاءة مع عناوين URL في ملف .chm التي تشير إلى موارد خارجية باستخدام مكتبة Aspose.Words للغة بايثون.

باتباع هذا الدليل، سوف تتعلم كيفية حل مشكلات الارتباط عن طريق تحديد اسم الملف الأصلي بـ `ChmLoadOptions`تُعد هذه العملية مثالية إذا كنت ترغب في تحسين موثوقية ملفات CHM وإمكانية الوصول إليها. 

**ما سوف تتعلمه:**
- تأثير الروابط المكسورة على قابلية استخدام ملف .chm
- إعداد Aspose.Words لـ Python للتعامل مع ملفات CHM
- استخدام `ChmLoadOptions` لإصلاح مشاكل الرابط
- التطبيقات العملية لهذه الميزة
- نصائح حول تحسين الأداء وإدارة الموارد

لنبدأ بإعداد المتطلبات الأساسية.

## دليل التنفيذ

دعنا نقسم التنفيذ إلى خطوات رئيسية لتحسين روابط CHM باستخدام واجهة برمجة تطبيقات Aspose.Words.

### تحديد اسم الملف الأصلي باستخدام ChmLoadOptions

**ملخص:**
تتيح لك هذه الميزة تحديد اسم الملف الأصلي لملف .chm، مما يضمن حل جميع الروابط الداخلية بشكل صحيح.

#### الخطوة 1: استيراد الوحدات الضرورية
ابدأ بالاستيراد `aspose.words` و `io`:

```python
import aspose.words as aw
import io
```

#### الخطوة 2: تكوين خيارات التحميل
إنشاء مثيل لـ `ChmLoadOptions` وضبط اسم الملف الأصلي:

```python
load_options = aw.loading.ChmLoadOptions()
load_options.original_file_name = 'amhelp.chm'
```
**توضيح:**
ضبط `original_file_name` يساعد Aspose.Words على حل الروابط داخل ملف CHM الخاص بك بدقة، مما يمنع عناوين URL التالفة.

#### الخطوة 3: تحميل المستند وحفظه
استخدم هذه الخيارات لتحميل مستند .chm:

```python
doc = aw.Document(
    stream=io.BytesIO(system_helper.io.File.read_all_bytes(YOUR_DOCUMENT_DIRECTORY + 'Document with ms-its links.chm')),
    load_options=load_options
)
```
احفظه كملف HTML، مع الحفاظ على الروابط المصححة:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'ExChmLoadOptions.OriginalFileName.html')
```
**نصيحة لاستكشاف الأخطاء وإصلاحها:**
تأكد من صحة مسار ملف .chm وإمكانية الوصول إليه. إذا كانت المسارات غير صحيحة، فعدّلها وفقًا لذلك في الكود.
