## دليل التنفيذ

### حماية كلمة المرور وحفظ إيصال التوجيه (H2)

#### ملخص

تتيح لك هذه الميزة تعيين كلمات مرور لمستندات مايكروسوفت وورد القديمة، مما يضمن أمان مستنداتك. كما أنها تحفظ معلومات إيصال التوجيه أثناء عملية الحفظ.

##### إعداد DocSaveOptions مع حماية كلمة المرور (H3)

أولاً، قم بإنشاء مستند جديد وتكوينه `DocSaveOptions`:

```python
import aspose.words as aw

def save_with_password_and_routing_slip():
    # إنشاء مستند جديد
    doc = aw.Document()
    builder = aw.DocumentBuilder(doc=doc)
    builder.write('Hello world!')

    # تكوين DocSaveOptions لحماية كلمة المرور
    options = aw.saving.DocSaveOptions(aw.SaveFormat.DOC)
    options.password = 'MyPassword'

    # الحفاظ على معلومات إيصال التوجيه
    options.save_routing_slip = True

    # حفظ المستند
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithPasswordAndRoutingSlip.doc"
    doc.save(file_name=output_path, save_options=options)

    # التحقق عن طريق التحميل باستخدام كلمة المرور
    load_options = aw.loading.LoadOptions(password='MyPassword')
    loaded_doc = aw.Document(file_name=output_path, load_options=load_options)
    assert 'Hello world!' == loaded_doc.get_text().strip()
```

**المعلمات موضحة:**
- `options.password`:تعيين كلمة المرور لحماية المستند.
- `options.save_routing_slip`:يحافظ على معلومات بطاقة التوجيه.

- استخدم كلمة مرور فريدة وقوية لتعزيز الأمان.

### استخدام المجلد المؤقت (H2)

#### ملخص

عند التعامل مع مستندات كبيرة، قد يؤدي استخدام مجلد مؤقت على القرص إلى تحسين الأداء من خلال تقليل استخدام الذاكرة.

##### تكوين DocSaveOptions للمجلدات المؤقتة (H3)

إليك كيفية إعداد مجلد مؤقت:

```python
import os
import aspose.words as aw

def save_using_temp_folder():
    # تحميل مستند موجود
    input_path = "YOUR_DOCUMENT_DIRECTORY/Rendering.docx"
    doc = aw.Document(file_name=input_path)

    # تكوين DocSaveOptions لاستخدام مجلد مؤقت
    options = aw.saving.DocSaveOptions()
    temp_folder = "YOUR_OUTPUT_DIRECTORY/TempFiles"

    # تأكد من وجود المجلد المؤقت
    os.makedirs(temp_folder, exist_ok=True)
    options.temp_folder = temp_folder

    # الحفظ باستخدام المجلد المؤقت
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithTempFolder.doc"
    doc.save(file_name=output_path, save_options=options)
```

**خيارات تكوين المفاتيح:**
- `options.temp_folder`:يحدد المسار الذي سيتم استخدامه لتخزين الملفات الوسيطة.

#### نصائح استكشاف الأخطاء وإصلاحها

- التحقق من أذونات الكتابة للمجلد المؤقت الخاص بك.
- تأكد من وجود مساحة كافية على القرص في الدليل المحدد.
