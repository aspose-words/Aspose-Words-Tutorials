#### الخطوة 1: إنشاء حقل مخصص
إنشاء حقول معلومات المستخدم المخصصة:


**المعلمات موضحة:**
- `DocumentBuilder`:يسهل إضافة المحتوى والتنسيق.
- `"INFO"`: يشير إلى نوع المعلومات.

#### الخطوة 2: تعديل الحقول الموجودة
تحديث أو إدارة الحقول الموجودة:

```python

```
