# البحث عن نص بسيط واستبداله في Word

4. محرر النصوص: Visual Studio أو أي IDE آخر متوافق مع C#.

## استيراد مساحات الأسماء

قبل الخوض في التفاصيل، عليك استيراد مساحات الأسماء اللازمة إلى مشروعك. إليك كيفية القيام بذلك:

```csharp
using Aspose.Words;
using Aspose.Words.Replacing;
```

الآن، لنشرح عملية البحث عن نص واستبداله في مستند وورد خطوة بخطوة. ستكون كل خطوة بسيطة وسهلة التنفيذ.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، لنُنشئ مسار مجلد المستندات. هذا هو المكان الذي سيتم فيه حفظ مستند Word بعد استبدال النص.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: إنشاء مستند جديد

بعد ذلك، ستنشئ مستند Word جديدًا باستخدام Aspose.Words. سيتم تعديل هذا المستند لعرض وظيفة البحث والاستبدال.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

هنا، نقوم بإنشاء `Document` كائن و `DocumentBuilder` الكائن. `DocumentBuilder` يساعدنا على كتابة النص في مستندنا.

## الخطوة 3: كتابة النص الأولي

الآن، لنكتب نصًا في المستند الذي سنستبدله لاحقًا. نستخدم `DocumentBuilder` لإدراج النص "مرحبا" _اسم العميل_"."

```csharp
builder.Writeln("Hello _CustomerName_,");
```

للتأكد من أن كل شيء يعمل بشكل صحيح حتى الآن، قمنا بطباعة نص المستند الأصلي على وحدة التحكم.

```csharp
Console.WriteLine("Original document text: " + doc.Range.Text);
```

## الخطوة 4: استبدال النص

هنا يحدث السحر! سنستبدل "_اسم العميل_"مع استخدام "جيمس بوند" `Replace` طريقة. 

```csharp
doc.Range.Replace("_CustomerName_", "James Bond", new FindReplaceOptions(FindReplaceDirection.Forward));
```

هنا، `FindReplaceOptions` يسمح لنا بتحديد اتجاه عملية البحث والاستبدال. نحن نستخدم `FindReplaceDirection.Forward` لاستبدال النص من بداية المستند إلى نهايته.

## الخطوة 5: التحقق من الاستبدال

للتأكد من نجاح الاستبدال، قم بطباعة نص المستند المعدل على وحدة التحكم.

```csharp
Console.WriteLine("Document text after replace: " + doc.Range.Text);
```

يجب أن ترى ذلك "_اسم العميل_"تم استبداله بـ "جيمس بوند".

## الخطوة 6: حفظ المستند

وأخيرًا، احفظ المستند المعدّل في الدليل المحدد.

```csharp
doc.Save(dataDir + "FindAndReplace.SimpleFindReplace.docx");
```
