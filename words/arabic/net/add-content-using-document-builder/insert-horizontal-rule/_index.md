## تعليمات التثبيت
1. تثبيت Aspose.Words: استخدم NuGet Package Manager في Visual Studio.
   - افتح مشروعك في Visual Studio.
   - انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
   - حدد إدارة حزم NuGet.
   - بحث عن `Aspose.Words` وتثبيته.

2. إضافة توجيه الاستخدام: في ملف التعليمات البرمجية الخاص بك، قم بتضمين التوجيه التالي لاستخدام فئات Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
