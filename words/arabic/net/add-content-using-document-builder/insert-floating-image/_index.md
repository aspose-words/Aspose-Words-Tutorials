# إنشاء مستند Word مع صورة عائمة

يوضح هذا المثال كيفية إنشاء مستند Word باستخدام Apose.Words for .NET وإدراج صورة مع تخطيط عائم. يقوم الرمز المقدم بتهيئة مستند و DocumentBuilder ، وإدراج صورة ذات أبعاد ومواقع محددة ، وأخيراً حفظ المستند. هذا الدليل مثالي للمطورين الذين يتطلعون إلى تعزيز قدرات إنشاء المستندات الخاصة بهم باستخدام Apose.Words.

## تعليمات التثبيت
1. تثبيت Apose.Words: إضافة حزمة Apose.Words NuGet إلى مشروع .NET الخاص بك. يمكنك القيام بذلك عن طريق تشغيل الأمر التالي في وحدة تحكم إدارة الحزمة:
   ```
   Install-Package Aspose.Words
   ```

2. إضافة مساحات الأسماء المطلوبة: تأكد من أن ملف الكود الخاص بك يتضمن مساحات الأسماء التالية:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
