## تعليمات التثبيت:  
1. تثبيت Apose.Words ل.NET:
   * باستخدام مدير حزمة NuGet:
     * افتح مشروعك في Visual Studio.
     * انقر بزر الماوس الأيمن فوق المشروع في مستكشف الحلول وحدد "إدارة حزم NuGet".
     * ابحث عن "Apose.Words" وانقر على "Install".

   * أو استخدم وحدة التحكم في إدارة الحزمة:
     ```
     Install-Package Aspose.Words
     ```
