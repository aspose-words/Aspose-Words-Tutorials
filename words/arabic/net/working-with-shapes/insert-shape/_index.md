# إدراج الأشكال في مستندات Word باستخدام Aspose.Words
يوضح هذا الدليل كيفية استخدام Aspose.Words لـ .NET لإدراج الأشكال، مثل مربعات النص، ومعالجتها في مستندات Word. ستتعلم كيفية تحديد موضع الأشكال بشكل مطلق أو نسبي، وضبط زوايا الدوران، وحفظ المستند بتنسيق DOCX الانتقالي ISO 29500:2008 باستخدام `OoxmlSaveOptions`يُبسط هذا المثال العمل مع الأشكال في مستندات Word، مما يُمكّن المطورين من إنشاء محتوى جذاب بصريًا بسهولة.

## تعليمات التثبيت  
1. تنزيل Aspose.Words لـ .NET:  
   احصل على أحدث إصدار من Aspose.Words لـ .NET من [صفحة التحميل](https://releases.aspose.com/words/net/).  

2. التثبيت عبر NuGet:  
   افتح مشروعك في Visual Studio وقم بتثبيت المكتبة باستخدام NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. إضافة مرجع مساحة الاسم:  
   في الكود الخاص بك، قم بتضمين مساحة الاسم لـ Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. تفعيل الترخيص (اختياري):  
   إذا كان لديك ترخيص صالح، فقم بتطبيقه في مشروعك لتجنب القيود. يمكنك [شراء ترخيص](https://purchase.aspose.com/buy) أو احصل على [رخصة مؤقتة](https://purchase.aspose.com/temporary-license/).
