# تعيين الوضع الأفقي أو الرأسي النسبي

## مقدمة

هل واجهتَ يومًا صعوبةً في تحديد مواقع الجداول بالشكل الذي تريده في مستندات Word؟ حسنًا، لستَ وحدك. سواءً كنتَ تُنشئ تقريرًا احترافيًا أو كتيبًا أنيقًا، يُمكن لمحاذاة الجداول أن تُحدث فرقًا كبيرًا. وهنا يأتي دور Aspose.Words for .NET. سيُرشدك هذا البرنامج التعليمي خطوةً بخطوة حول كيفية تحديد المواقع الأفقية أو الرأسية النسبية للجداول في مستندات Word. هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

1. Aspose.Words for .NET: إذا لم تقم بذلك بالفعل، يمكنك تنزيله [هنا](https://releases.aspose.com/words/net/).
2. بيئة التطوير: Visual Studio أو أي بيئة تطوير متكاملة أخرى متوافقة مع .NET.
3. المعرفة الأساسية بلغة C#: يفترض هذا البرنامج التعليمي أنك على دراية بأساسيات برمجة C#.

## استيراد مساحات الأسماء

أولاً، عليك استيراد مساحات الأسماء اللازمة. هذا ضروري للوصول إلى وظائف Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: تحميل المستند الخاص بك

للبدء، ستحتاج إلى تحميل مستند Word إلى البرنامج. إليك كيفية القيام بذلك:

```csharp
// المسار إلى دليل المستندات الخاص بك 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table wrapped by text.docx");
```

يُحدد هذا المقطع البرمجي مسار مجلد المستندات ويُحمّل المستند المُحدد الذي ترغب بالعمل عليه. تأكد من صحة مسار المستند لتجنب أي مشاكل في التحميل.

## الخطوة 2: الوصول إلى الجدول

بعد ذلك، علينا الوصول إلى الجدول داخل المستند. عادةً، يُفضّل العمل مع الجدول الأول في قسم النص.

```csharp
Table table = doc.FirstSection.Body.Tables[0];
```

هذا السطر من التعليمات البرمجية يسترجع الجدول الأول من نص المستند. إذا كان مستندك يحتوي على عدة جداول، يمكنك تعديل الفهرس وفقًا لذلك.

## الخطوة 3: ضبط الوضع الأفقي

الآن، لنُحدد الموضع الأفقي للجدول بالنسبة لعنصر مُحدد. في هذا المثال، سنحدد موضعه بالنسبة للعمود.

```csharp
table.HorizontalAnchor = RelativeHorizontalPosition.Column;
```

من خلال ضبط `HorizontalAnchor` ل `RelativeHorizontalPosition.Column`، فأنت تطلب من الجدول أن يمحاذاة نفسه أفقيًا فيما يتعلق بالعمود الذي يوجد فيه.

## الخطوة 4: ضبط الوضع الرأسي

كما هو الحال مع الوضع الأفقي، يمكنك أيضًا ضبط الوضع الرأسي. هنا، نضعه بالنسبة للصفحة.

```csharp
table.VerticalAnchor = RelativeVerticalPosition.Page;
```

ضبط `VerticalAnchor` ل `RelativeVerticalPosition.Page` يتأكد من محاذاة الجدول عموديًا وفقًا للصفحة.

## الخطوة 5: احفظ مستندك

أخيرًا، احفظ تغييراتك في مستند جديد. هذه خطوة أساسية لضمان حفظها.

```csharp
doc.Save(dataDir + "WorkingWithTables.SetFloatingTablePosition.docx");
```

يحفظ هذا الأمر المستند المعدل باسم جديد، مما يضمن عدم الكتابة فوق الملف الأصلي.
