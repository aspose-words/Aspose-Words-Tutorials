# طاولة منقسمة

## مقدمة

هل سبق لك أن وجدت نفسك تعمل على جدول كبير في مستند وورد وتمنيت لو كان بإمكانك تقسيمه إلى جدولين أصغر وأسهل إدارة؟ حسنًا، سنتناول اليوم بالتفصيل كيفية تحقيق ذلك باستخدام Aspose.Words لـ .NET. سواء كنت تتعامل مع جداول بيانات ضخمة أو هياكل مستندات معقدة، فإن تقسيم الجداول يُحسّن سهولة القراءة والتنظيم. لنستكشف عملية تقسيم جدول باستخدام Aspose.Words لـ .NET خطوة بخطوة.

## المتطلبات الأساسية

قبل أن ننتقل إلى البرنامج التعليمي، تأكد من أن لديك ما يلي:

1. مكتبة Aspose.Words لـ .NET: تأكد من تنزيل مكتبة Aspose.Words لـ .NET وتثبيتها. يمكنك الحصول عليها من [صفحة إصدارات Aspose](https://releases.aspose.com/words/net/).
2. بيئة التطوير: قم بإعداد بيئة تطوير تدعم إطار عمل .NET، مثل Visual Studio.
3. نموذج مستند: إعداد مستند Word (`Tables.docx`) مع جدول واحد على الأقل لتطبيق عملية التقسيم.

## استيراد مساحات الأسماء

أولاً، استورد مساحات الأسماء اللازمة لمشروعك. هذا يسمح لك بالوصول إلى الفئات والأساليب التي يوفرها Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## الخطوة 1: تحميل المستند

لنبدأ بتحميل المستند الذي يحتوي على الجدول الذي تريد تقسيمه. تأكد من تحديد المسار الصحيح للمستند.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Tables.docx");
```

## الخطوة 2: تحديد الجدول المراد تقسيمه

بعد ذلك، حدد الجدول الذي ترغب في تقسيمه واسترجعه. في هذا المثال، سنستهدف الجدول الأول في المستند.

```csharp
Table firstTable = (Table)doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: حدد الصف الذي تريد تقسيمه

حدد الصف الذي تريد تقسيم الجدول إليه. هنا، نقسم الجدول عند الصف الثالث (شاملًا).

```csharp
Row row = firstTable.Rows[2];
```

## الخطوة 4: إنشاء حاوية جدول جديدة

قم بإنشاء حاوية جدول جديدة لحمل الصفوف التي سيتم نقلها من الجدول الأصلي.

```csharp
Table table = (Table)firstTable.Clone(false);
```

## الخطوة 5: إدراج حاوية الجدول الجديدة

قم بإدراج حاوية الجدول الجديدة مباشرة بعد الجدول الأصلي في المستند.

```csharp
firstTable.ParentNode.InsertAfter(table, firstTable);
```

## الخطوة 6: إضافة فقرة عازلة

أضف فقرة عازلة بين الجدولين للتأكد من بقائهم منفصلين.

```csharp
firstTable.ParentNode.InsertAfter(new Paragraph(doc), firstTable);
```

## الخطوة 7: نقل الصفوف إلى الجدول الجديد

انقل الصفوف من الجدول الأصلي إلى حاوية الجدول الجديد. تستمر هذه الحلقة حتى يتم نقل الصف المحدد (بما في ذلك).

```csharp
Row currentRow;
do
{
    currentRow = firstTable.LastRow;
    table.PrependChild(currentRow);
} while (currentRow != row);
```

## الخطوة 8: حفظ المستند

وأخيرًا، احفظ المستند المعدّل مع تقسيم الجداول.

```csharp
doc.Save(dataDir + "WorkingWithTables.SplitTable.docx");
```
