## استيراد مساحات الأسماء

أولاً، ستحتاج إلى استيراد مساحات الأسماء اللازمة. سيُمكّنك هذا من الوصول إلى فئات وطرق Aspose.Words. إليك كيفية القيام بذلك:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Tables;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا نقسم العملية إلى خطوات سهلة المتابعة.

## الخطوة 1: تحميل المستند الخاص بك

الخطوة الأولى هي تحميل مستند Word. يجب أن يحتوي هذا المستند على الجداول التي تريد دمجها. إليك الكود لتحميل المستند:

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Tables.docx");
```

في هذا المثال، استبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار إلى مستندك.

## الخطوة 2: تحديد الجداول

بعد ذلك، عليك تحديد الجداول التي تريد دمجها. يتيح لك Aspose.Words الحصول على جداول من مستند باستخدام `GetChild` الطريقة. إليك الطريقة:

```csharp
Table firstTable = (Table) doc.GetChild(NodeType.Table, 0, true);
Table secondTable = (Table) doc.GetChild(NodeType.Table, 1, true);
```

في هذا الكود، نقوم بجلب الجدولين الأول والثاني من المستند.

## الخطوة 3: إضافة الصفوف من الجدول الثاني إلى الجدول الأول

الآن، حان وقت دمج الصفوف. سنضيف جميع الصفوف من الجدول الثاني إلى الجدول الأول. يتم ذلك باستخدام حلقة while بسيطة:

```csharp
// إضافة جميع الصفوف من الجدول الثاني إلى الجدول الأول
while (secondTable.HasChildNodes)
    firstTable.Rows.Add(secondTable.FirstRow);
```

تستمر هذه الحلقة حتى تتم إضافة كافة الصفوف من الجدول الثاني إلى الجدول الأول.

## الخطوة 4: إزالة الجدول الثاني

بعد إضافة الصفوف، لن تكون هناك حاجة للجدول الثاني. يمكنك إزالته باستخدام `Remove` طريقة:

```csharp
secondTable.Remove();
```

## الخطوة 5: حفظ المستند

أخيرًا، احفظ المستند المُعدَّل. تضمن هذه الخطوة كتابة تغييراتك في الملف.

```csharp
doc.Save(dataDir + "WorkingWithTables.CombineRows.docx");
```

وهذا كل شيء! لقد نجحت في دمج صفوف جدولين في جدول واحد باستخدام Aspose.Words لـ .NET.
