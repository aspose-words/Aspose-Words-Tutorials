# إبقاء الطاولة معًا

## مقدمة

هل شعرتَ يومًا بالإحباط عندما ينقسم جدول في مستند Word الخاص بك إلى صفحتين؟ كأن معلوماتك المُرتّبة بعناية قد قررت فجأةً التوقف في منتصفها! يُعدّ الحفاظ على الجداول معًا في صفحة واحدة أمرًا بالغ الأهمية لسهولة القراءة والعرض. سواءً كان ذلك لتقرير أو مقترح مشروع أو حتى مستندًا شخصيًا، فقد يكون تقسيم الجداول أمرًا مزعجًا للغاية. لحسن الحظ، يُقدّم Aspose.Words for .NET طريقةً فعّالة لحل هذه المشكلة. في هذا البرنامج التعليمي، سنشرح الخطوات اللازمة للحفاظ على جداولك سليمة ومُتناسقة. هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

1. Aspose.Words لـ .NET - إذا لم تقم بتثبيته بعد، فيمكنك تنزيله من [هنا](https://releases.aspose.com/words/net/).
2. مستند Word يحتوي على جدول - سنعمل مع مستند نموذجي يحتوي على جدول يمتد على عدة صفحات.
3. المعرفة الأساسية بلغة C# - يفترض هذا البرنامج التعليمي أن لديك فهمًا أساسيًا لبرمجة C#.

## استيراد مساحات الأسماء

أولاً، لنستورد مساحات الأسماء اللازمة. سيُتيح لنا هذا الوصول إلى الفئات والأساليب التي نحتاجها من Aspose.Words لـ .NET.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

لنُقسّم العملية إلى خطوات سهلة ومفهومة. سنبدأ بتحميل مستندنا وننتهي بحفظ المستند المُحدّث مع الحفاظ على بقاء الجدولين معًا.

## الخطوة 1: تحميل المستند

للعمل مع مستند Word، نحتاج أولاً إلى تحميله. سنستخدم `Document` الصف لهذا.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table spanning two pages.docx");
```

## الخطوة 2: الوصول إلى الجدول

بعد ذلك، نحتاج إلى تحديد الجدول الذي نريد الاحتفاظ به معًا. سنفترض أنه أول جدول في المستند.

```csharp
Table table = (Table) doc.GetChild(NodeType.Table, 0, true);
```

## الخطوة 3: تعيين KeepWithNext للفقرات

لمنع الجدول من الانهيار عبر الصفحات، نحتاج إلى ضبط `KeepWithNext` الخاصية لكل فقرة في الجدول، باستثناء الفقرات الأخيرة في الصف الأخير.

```csharp
foreach (Cell cell in table.GetChildNodes(NodeType.Cell, true))
{
    cell.EnsureMinimum();
    foreach (Paragraph para in cell.Paragraphs)
    {
        if (!(cell.ParentRow.IsLastRow && para.IsEndOfCell))
            para.ParagraphFormat.KeepWithNext = true;
    }
}
```

## الخطوة 4: حفظ المستند

أخيرًا، نحفظ المستند المُحدّث. سيؤدي هذا إلى تطبيق تغييراتنا وضمان بقاء الجدولين في صفحة واحدة.

```csharp
doc.Save(dataDir + "WorkingWithTables.KeepTableTogether.docx");
```
