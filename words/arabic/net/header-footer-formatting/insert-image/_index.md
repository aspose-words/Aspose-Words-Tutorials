# إدراج صورة في رأس مستند Word

## تعليمات التثبيت  

للبدء في استخدام Aspose.Words لـ .NET، اتبع الخطوات البسيطة التالية:  

1. تنزيل Aspose.Words  
   - [تنزيل أحدث إصدار](https://releases.aspose.com/words/net/) من Aspose.Words لـ .NET.  

2. تثبيت Aspose.Words عبر NuGet  
   - افتح مشروعك في Visual Studio.  
   - انتقل إلى قائمة الأدوات وحدد NuGet Package Manager > إدارة حزم NuGet للحل.  
   - بحث عن `Aspose.Words` في مدير حزمة NuGet وتثبيته.  

   أو استخدم وحدة التحكم في إدارة الحزم:  
   ```shell
   Install-Package Aspose.Words
   ```  

3. تطبيق الترخيص  
   - احصل على [رخصة تجريبية مجانية](https://releases.aspose.com/) أو [شراء ترخيص](https://purchase.aspose.com/buy).  
   - استخدم الكود التالي لتطبيق الترخيص في مشروعك:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
