# تحديد الخط الافتراضي عند العرض

## مقدمة

قد يكون ضمان عرض مستندات Word بشكل صحيح على مختلف المنصات أمرًا صعبًا، خاصةً عند التعامل مع توافق الخطوط. إحدى طرق الحفاظ على تناسق المظهر هي تحديد خط افتراضي عند عرض مستنداتك بتنسيق PDF أو تنسيقات أخرى. في هذا البرنامج التعليمي، سنستكشف كيفية تعيين خط افتراضي باستخدام Aspose.Words لـ .NET، لتبدو مستنداتك رائعة أينما عُرضت.

## المتطلبات الأساسية

قبل الغوص في الكود، دعنا نغطي ما ستحتاج إلى متابعته مع هذا البرنامج التعليمي:

- Aspose.Words لـ .NET: تأكد من تثبيت أحدث إصدار. يمكنك تنزيله. [هنا](https://releases.aspose.com/words/net/).
- بيئة التطوير: Visual Studio أو أي بيئة تطوير .NET أخرى.
- المعرفة الأساسية بلغة C#: يفترض هذا البرنامج التعليمي أنك مرتاح في برمجة C#.

## استيراد مساحات الأسماء

للبدء، عليك استيراد مساحات الأسماء اللازمة. سيسمح لك هذا بالوصول إلى الفئات والأساليب اللازمة للعمل مع Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

الآن، دعنا نقوم بتقسيم عملية تحديد الخط الافتراضي إلى خطوات سهلة المتابعة.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، حدد مسار مجلد المستندات. هذا هو المكان الذي ستُخزَّن فيه ملفات الإدخال والإخراج.

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: تحميل المستند الخاص بك

بعد ذلك، حمّل المستند الذي تريد عرضه. في هذا المثال، سنستخدم ملفًا باسم "Rendering.docx".

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## الخطوة 3: تكوين إعدادات الخط

إنشاء مثيل لـ `FontSettings` وحدد الخط الافتراضي. إذا لم يتم العثور على الخط المحدد أثناء العرض، فسيستخدم Aspose.Words أقرب خط متوفر على الجهاز.

```csharp
FontSettings fontSettings = new FontSettings();
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial Unicode MS";
```

## الخطوة 4: تطبيق إعدادات الخط على المستند

قم بتعيين إعدادات الخط المحددة للمستند الخاص بك.

```csharp
doc.FontSettings = fontSettings;
```

## الخطوة 5: حفظ المستند

أخيرًا، احفظ المستند بالصيغة المطلوبة. في هذه الحالة، سنحفظه بصيغة PDF.

```csharp
doc.Save(dataDir + "WorkingWithFonts.SpecifyDefaultFontWhenRendering.pdf");
```

## خاتمة

باتباع هذه الخطوات، يمكنك ضمان عرض مستندات Word بخط افتراضي محدد، مع الحفاظ على التناسق بين مختلف المنصات. يُعد هذا مفيدًا بشكل خاص للمستندات المشتركة على نطاق واسع أو المعروضة على أنظمة ذات خطوط متنوعة.


## الأسئلة الشائعة

### لماذا تحديد الخط الافتراضي في Aspose.Words؟
يؤدي تحديد الخط الافتراضي إلى ضمان ظهور مستندك بشكل متسق عبر منصات مختلفة، حتى إذا كانت الخطوط الأصلية غير متوفرة.

### ماذا يحدث إذا لم يتم العثور على الخط الافتراضي أثناء العرض؟
سيستخدم Aspose.Words أقرب خط متوفر على الجهاز للحفاظ على مظهر المستند قدر الإمكان.

### هل يمكنني تحديد خطوط افتراضية متعددة؟
لا، يمكنك تحديد خط افتراضي واحد فقط. مع ذلك، يمكنك استبدال الخط في حالات محددة باستخدام `FontSettings` فصل.

### هل Aspose.Words for .NET متوافق مع كافة إصدارات مستندات Word؟
نعم، يدعم Aspose.Words for .NET مجموعة واسعة من تنسيقات مستندات Word، بما في ذلك DOC، وDOCX، وRTF، والمزيد.

### أين يمكنني الحصول على الدعم إذا واجهت مشاكل؟
يمكنك الحصول على الدعم من مجتمع Aspose والمطورين على [منتدى دعم Aspose.Words](https://forum.aspose.com/c/words/8).
