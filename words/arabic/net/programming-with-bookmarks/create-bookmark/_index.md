# إنشاء إشارة مرجعية في مستند Word

## مقدمة

إنشاء الإشارات المرجعية في مستند Word يُحدث نقلة نوعية، خاصةً عند الرغبة في التنقل بين المستندات الكبيرة بسهولة. سنشرح اليوم عملية إنشاء الإشارات المرجعية باستخدام Aspose.Words لـ .NET. سيشرح هذا البرنامج التعليمي خطوة بخطوة، ويضمن فهمك لكل جزء من العملية. هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، يجب أن يكون لديك ما يلي:

1. Aspose.Words لمكتبة .NET: التنزيل والتثبيت من [هنا](https://releases.aspose.com/words/net/).
2. بيئة التطوير: Visual Studio أو أي بيئة تطوير .NET أخرى.
3. المعرفة الأساسية بلغة C#: فهم مفاهيم برمجة C# الأساسية.

## استيراد مساحات الأسماء

للعمل مع Aspose.Words لـ .NET، تحتاج إلى استيراد المساحات الأساسية الضرورية:

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## الخطوة 1: إعداد المستند ومنشئ المستندات

تهيئة المستند

أولاً، نحتاج إلى إنشاء مستند جديد وتهيئة `DocumentBuilder`. هذه هي نقطة البداية لإضافة المحتوى والإشارات المرجعية إلى مستندك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

الشرح: `Document` الهدف هو لوحتك. `DocumentBuilder` هو بمثابة قلمك، الذي يسمح لك بكتابة المحتوى وإنشاء إشارات مرجعية في المستند.

## الخطوة 2: إنشاء الإشارة المرجعية الرئيسية

بدء وإنهاء الإشارة المرجعية الرئيسية

لإنشاء إشارة مرجعية، عليك تحديد نقطتي البداية والنهاية. هنا، سننشئ إشارة مرجعية باسم "إشارتي المرجعية".

```csharp
builder.StartBookmark("My Bookmark");
builder.Writeln("Text inside a bookmark.");
```

الشرح: `StartBookmark` تشير الطريقة إلى بداية الإشارة المرجعية، و `Writeln` يضيف نصًا داخل الإشارة المرجعية.

## الخطوة 3: إنشاء إشارة مرجعية متداخلة

إضافة إشارة مرجعية متداخلة داخل الإشارة المرجعية الرئيسية

يمكنك دمج الإشارات المرجعية داخل إشارات مرجعية أخرى. هنا، نضيف "الإشارة المرجعية المتداخلة" ضمن "إشارتي المرجعية".

```csharp
builder.StartBookmark("Nested Bookmark");
builder.Writeln("Text inside a NestedBookmark.");
builder.EndBookmark("Nested Bookmark");
```

توضيح: يتيح دمج الإشارات المرجعية تنظيمًا أكثر هيكلية وتسلسلًا هرميًا للمحتوى. `EndBookmark` تغلق الطريقة الإشارة المرجعية الحالية.

## الخطوة 4: إضافة نص خارج الإشارة المرجعية المتداخلة

متابعة إضافة المحتوى

بعد الإشارة المرجعية المتداخلة، يمكننا الاستمرار في إضافة المزيد من المحتوى داخل الإشارة المرجعية الرئيسية.

```csharp
builder.Writeln("Text after Nested Bookmark.");
builder.EndBookmark("My Bookmark");
```

التوضيح: هذا يضمن أن الإشارة المرجعية الرئيسية تشمل كل من الإشارة المرجعية المتداخلة والنص الإضافي.

## الخطوة 5: تكوين خيارات حفظ PDF

إعداد خيارات حفظ PDF للإشارات المرجعية

عند حفظ المستند بتنسيق PDF، يمكننا تكوين خيارات لتضمين الإشارات المرجعية.

```csharp
PdfSaveOptions options = new PdfSaveOptions();
options.OutlineOptions.BookmarksOutlineLevels.Add("My Bookmark", 1);
options.OutlineOptions.BookmarksOutlineLevels.Add("Nested Bookmark", 2);
```

الشرح: `PdfSaveOptions` تسمح لك الفئة بتحديد كيفية حفظ المستند بتنسيق PDF. `BookmarksOutlineLevels` تعرف الخاصية على التسلسل الهرمي للإشارات المرجعية في ملف PDF.

## الخطوة 6: حفظ المستند

حفظ المستند بصيغة PDF

وأخيرًا، احفظ المستند بالخيارات المحددة.

```csharp
doc.Save(dataDir + "WorkingWithBookmarks.CreateBookmark.pdf", options);
```

الشرح: `Save` تحفظ هذه الطريقة المستند بالتنسيق والمكان المحددين. سيتضمن ملف PDF الآن الإشارات المرجعية التي أنشأناها.

## خاتمة

إنشاء إشارات مرجعية في مستند Word باستخدام Aspose.Words for .NET سهلٌ ومفيدٌ للغاية لتصفح المستندات وتنظيمها. سواءً كنت تُنشئ تقارير، أو كتبًا إلكترونية، أو تُدير مستندات كبيرة، فإن الإشارات المرجعية تُسهّل عليك الأمور. اتبع الخطوات الموضحة في هذا البرنامج التعليمي، وستحصل على ملف PDF مُضاف إليه إشارات مرجعية جاهز في وقت قصير.

## الأسئلة الشائعة

### هل يمكنني إنشاء إشارات مرجعية متعددة على مستويات مختلفة؟

بالتأكيد! يمكنك إنشاء أي عدد من الإشارات المرجعية وتحديد مستوياتها الهرمية عند حفظ المستند بصيغة PDF.

### كيف أقوم بتحديث نص الإشارة المرجعية؟

يمكنك الانتقال إلى الإشارة المرجعية باستخدام `DocumentBuilder.MoveToBookmark` ثم قم بتحديث النص.

### هل من الممكن حذف الإشارة المرجعية؟

نعم، يمكنك حذف الإشارة المرجعية باستخدام `Bookmarks.Remove` الطريقة عن طريق تحديد اسم الإشارة المرجعية.

### هل يمكنني إنشاء إشارات مرجعية بتنسيقات أخرى غير PDF؟

نعم، يدعم Aspose.Words الإشارات المرجعية بتنسيقات مختلفة، بما في ذلك DOCX وHTML وEPUB.

### كيف يمكنني التأكد من ظهور العلامات المرجعية بشكل صحيح في ملف PDF؟

تأكد من تحديد `BookmarksOutlineLevels` بشكل صحيح في `PdfSaveOptions`. وهذا يضمن تضمين الإشارات المرجعية في مخطط PDF.
