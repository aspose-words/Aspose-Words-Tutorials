# إضافة قيم التاريخ والوقت إلى محور الرسم البياني

## مقدمة

إنشاء المخططات البيانية في المستندات يُمثل وسيلة فعّالة لعرض البيانات. عند التعامل مع بيانات السلاسل الزمنية، تُعدّ إضافة قيم التاريخ والوقت إلى محور المخطط أمرًا بالغ الأهمية للوضوح. في هذا البرنامج التعليمي، سنشرح لك عملية إضافة قيم التاريخ والوقت إلى محور المخطط باستخدام Aspose.Words لـ .NET. سيساعدك هذا الدليل التفصيلي على إعداد بيئتك، وكتابة الشيفرة البرمجية، وفهم كل جزء من العملية. هيا بنا!

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من توفر المتطلبات الأساسية التالية:

1. Visual Studio أو أي .NET IDE: تحتاج إلى بيئة تطوير لكتابة وتشغيل كود .NET الخاص بك.
2. Aspose.Words for .NET: يجب أن تكون مكتبة Aspose.Words for .NET مثبتة لديك. يمكنك تنزيلها من [هنا](https://releases.aspose.com/words/net/).
3. المعرفة الأساسية بلغة C#: يفترض هذا البرنامج التعليمي أن لديك فهمًا أساسيًا لبرمجة C#.
4. ترخيص Aspose صالح: يمكنك الحصول على ترخيص مؤقت من [هنا](https://purchase.aspose.com/temporary-license/).

## استيراد مساحات الأسماء

للبدء، تأكد من استيراد مساحات الأسماء اللازمة في مشروعك. هذه الخطوة أساسية للوصول إلى فئات وطرق Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Drawing.Charts;
```

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، عليك تحديد المجلد الذي ستحفظ فيه مستندك. هذا مهم لتنظيم ملفاتك وضمان تشغيل الكود بشكل صحيح.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: إنشاء مستند جديد وDocumentBuilder

بعد ذلك، قم بإنشاء مثيل جديد لـ `Document` الصف و `DocumentBuilder` الكائنات. ستساعدك هذه الكائنات في إنشاء مستندك ومعالجته.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## الخطوة 3: إدراج مخطط في المستند

الآن، قم بإدراج مخطط في مستندك باستخدام `DocumentBuilder` في هذا المثال، نستخدم مخططًا عموديًا، ولكن يمكنك اختيار أنواع أخرى أيضًا.

```csharp
Shape shape = builder.InsertChart(ChartType.Column, 432, 252);
Chart chart = shape.Chart;
```

## الخطوة 4: مسح السلسلة الحالية

امسح أي سلسلة موجودة في المخطط للتأكد من أنك تبدأ من الصفر. هذه الخطوة ضرورية للبيانات المخصصة.

```csharp
chart.Series.Clear();
```

## الخطوة 5: إضافة قيم التاريخ والوقت إلى السلسلة

أضف قيم التاريخ والوقت إلى سلسلة المخططات. تتضمن هذه الخطوة إنشاء مصفوفات للتواريخ والقيم المقابلة لها.

```csharp
chart.Series.Add("Aspose Series 1",
    new[]
    {
        new DateTime(2017, 11, 06), new DateTime(2017, 11, 09), new DateTime(2017, 11, 15),
        new DateTime(2017, 11, 21), new DateTime(2017, 11, 25), new DateTime(2017, 11, 29)
    },
    new double[] { 1.2, 0.3, 2.1, 2.9, 4.2, 5.3 });
```

## الخطوة 6: تكوين المحور X

اضبط مقياس وعلامات التحديد للمحور X. هذا يضمن عرض التواريخ بشكل صحيح وعلى فترات زمنية مناسبة.

```csharp
ChartAxis xAxis = chart.AxisX;
xAxis.Scaling.Minimum = new AxisBound(new DateTime(2017, 11, 05).ToOADate());
xAxis.Scaling.Maximum = new AxisBound(new DateTime(2017, 12, 03).ToOADate());
xAxis.MajorUnit = 7;
xAxis.MinorUnit = 1;
xAxis.MajorTickMark = AxisTickMark.Cross;
xAxis.MinorTickMark = AxisTickMark.Outside;
```

## الخطوة 7: حفظ المستند

أخيرًا، احفظ مستندك في المجلد المحدد. بهذه الخطوة، تنتهي العملية، ويجب أن يحتوي مستندك الآن على مخطط بقيم التاريخ والوقت على المحور السيني.

```csharp
doc.Save(dataDir + "WorkingWithCharts.DateTimeValuesToAxis.docx");
```

## خاتمة

إضافة قيم التاريخ والوقت إلى محور الرسم البياني في مستند عملية سهلة باستخدام Aspose.Words لـ .NET. باتباع الخطوات الموضحة في هذا البرنامج التعليمي، يمكنك إنشاء مخططات بيانية واضحة وغنية بالمعلومات تُصوّر بيانات السلاسل الزمنية بفعالية. سواء كنت تُعدّ تقارير أو عروضًا تقديمية أو أي مستند يتطلب تمثيلًا تفصيليًا للبيانات، يوفر Aspose.Words الأدوات اللازمة للنجاح.

## الأسئلة الشائعة

### هل يمكنني استخدام أنواع أخرى من المخططات مع Aspose.Words لـ .NET؟

نعم، يدعم Aspose.Words أنواعًا مختلفة من المخططات، بما في ذلك المخططات الخطية والشريطية والدائرية والمزيد.

### كيف يمكنني تخصيص مظهر الرسم البياني الخاص بي؟

يمكنك تخصيص المظهر من خلال الوصول إلى خصائص الرسم البياني وتعيين الأنماط والألوان والمزيد.

### هل من الممكن إضافة سلاسل متعددة إلى الرسم البياني؟

بالتأكيد! يمكنك إضافة سلاسل متعددة إلى مخططك عن طريق استدعاء `Series.Add` الطريقة عدة مرات مع بيانات مختلفة.

### ماذا لو كنت بحاجة إلى تحديث بيانات الرسم البياني بشكل ديناميكي؟

بإمكانك تحديث بيانات الرسم البياني بشكل ديناميكي عن طريق معالجة خصائص السلسلة والمحور برمجيًا استنادًا إلى متطلباتك.

### أين يمكنني العثور على المزيد من الوثائق التفصيلية لـ Aspose.Words لـ .NET؟

يمكنك العثور على وثائق أكثر تفصيلا [هنا](https://reference.aspose.com/words/net/).
