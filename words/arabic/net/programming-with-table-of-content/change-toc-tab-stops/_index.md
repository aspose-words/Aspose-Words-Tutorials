# تغيير علامات التبويب في مستند Word

## مقدمة

هل تساءلت يومًا عن كيفية تحسين جدول المحتويات (TOC) في مستندات Word؟ ربما ترغب في محاذاة علامات التبويب بشكل مثالي لإضفاء لمسة احترافية. أنت في المكان المناسب! اليوم، سنتعمق في كيفية تغيير علامات تبويب جدول المحتويات باستخدام Aspose.Words لـ .NET. تابعونا، وأعدكم بأنكم ستغادرون ومعكم كل ما يلزم لجعل جدول المحتويات الخاص بكم يبدو أنيقًا ومرتبًا.

## المتطلبات الأساسية

قبل أن نبدأ، دعونا نتأكد من أن لديك كل ما تحتاجه:

1. Aspose.Words لـ .NET: يمكنك [قم بتحميله هنا](https://releases.aspose.com/words/net/).
2. بيئة التطوير: Visual Studio أو أي بيئة تطوير متكاملة متوافقة مع C#.
3. مستند Word: على وجه التحديد، المستند الذي يحتوي على جدول المحتويات.

هل فهمت كل ذلك؟ رائع! هيا بنا.

## استيراد مساحات الأسماء

أولاً، ستحتاج إلى استيراد مساحات الأسماء اللازمة. هذا أشبه بتجهيز أدواتك قبل بدء أي مشروع.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

لنُقسّم هذه العملية إلى خطوات بسيطة وسهلة الفهم. سنبدأ بتحميل المستند، وتعديل علامات تبويب جدول المحتويات، ثم حفظ المستند المُحدّث.

## الخطوة 1: تحميل المستند

لماذا؟ نحتاج إلى الوصول إلى مستند Word الذي يحتوي على جدول المحتويات الذي نريد تعديله.

كيف؟ إليك مقتطف برمجي بسيط لمساعدتك على البدء:

```csharp
// المسار إلى دليل المستندات الخاص بك
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// قم بتحميل المستند الذي يحتوي على جدول المحتويات
Document doc = new Document(dataDir + "Table of contents.docx");
```

تخيّل أن مستندك أشبه بكعكة، ونحن على وشك إضافة بعض الزينة. الخطوة الأولى هي إخراج الكعكة من العلبة.

## الخطوة 2: تحديد فقرات جدول المحتويات

لماذا؟ علينا تحديد الفقرات التي يتألف منها جدول المحتويات. 

كيف؟ كرر الفقرات وتحقق من أسلوبها:

```csharp
foreach(Paragraph para in doc.GetChildNodes(NodeType.Paragraph, true))
{
    if (para.ParagraphFormat.Style.StyleIdentifier >= StyleIdentifier.Toc1 &&
        para.ParagraphFormat.Style.StyleIdentifier <= StyleIdentifier.Toc9)
    {
        // تم العثور على فقرة جدول المحتويات
    }
}
```

تخيل الأمر كأنك تبحث في حشد من الناس عن أصدقائك. هنا، نبحث عن فقرات مُصممة على شكل جداول محتويات.

## الخطوة 3: تعديل علامات التبويب

لماذا؟ هنا يكمن السر. تغيير علامات التبويب يُضفي على جدول المحتويات مظهرًا أكثر أناقة.

كيف؟ أزل علامة التبويب الحالية وأضف علامة تبويب جديدة في موضع مُعدّل:

```csharp
foreach(Paragraph para in doc.GetChildNodes(NodeType.Paragraph, true))
{
    if (para.ParagraphFormat.Style.StyleIdentifier >= StyleIdentifier.Toc1 &&
        para.ParagraphFormat.Style.StyleIdentifier <= StyleIdentifier.Toc9)
    {
        TabStop tab = para.ParagraphFormat.TabStops[0];
        para.ParagraphFormat.TabStops.RemoveByPosition(tab.Position);
        para.ParagraphFormat.TabStops.Add(tab.Position - 50, tab.Alignment, tab.Leader);
    }
}
```

الأمر أشبه بتعديل أثاث غرفة معيشتك حتى يصبح مثاليًا. نقوم بتعديل علامات التبويب هذه لتحقيق الكمال.

## الخطوة 4: حفظ المستند المعدّل

لماذا؟ لضمان حفظ جميع أعمالك الشاقة وإمكانية عرضها أو مشاركتها.

كيف؟ احفظ المستند باسم جديد للحفاظ على الأصل سليمًا:

```csharp
// حفظ المستند المعدل
doc.Save(dataDir + "WorkingWithTableOfContent.ChangeTocTabStops.docx");
```

وفويلا! الآن أصبح جدول المحتويات الخاص بك يحتوي على علامات التبويب في المكان الذي تريده بالضبط.
