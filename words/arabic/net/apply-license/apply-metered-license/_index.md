# تطبيق رخصة القياس

2. مفاتيح ترخيص مُقاسة صالحة: تحتاجها لتفعيل الترخيص المُقاس. يمكنك الحصول عليها من [صفحة شراء Aspose](https://purchase.aspose.com/buy).

## استيراد مساحات الأسماء

قبل التعمق في الكود، علينا استيراد مساحات الأسماء اللازمة. هذا أمر بالغ الأهمية لأنه يسمح لنا بالوصول إلى الفئات والأساليب التي يوفرها Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Metered;
```

حسنًا، لنبدأ بشرحها. سنشرح العملية خطوة بخطوة، حتى لا يفوتك أي شيء.

## الخطوة 1: تهيئة الفئة المقاسة

أولاً وقبل كل شيء، نحتاج إلى إنشاء مثيل لـ `Metered` هذه الفئة مسؤولة عن إعداد الترخيص المقنن.

```csharp
Metered metered = new Metered();
```

## الخطوة 2: ضبط المفاتيح المقاسة

الآن بعد أن أصبح لدينا `Metered` على سبيل المثال، نحتاج إلى ضبط المفاتيح المُقاسة. هذه المفاتيح مُقدمة من Aspose وهي خاصة باشتراكك.

```csharp
metered.SetMeteredKey("your_public_key", "your_private_key");
```

يستبدل `"your_public_key"` و `"your_private_key"` بالمفاتيح الفعلية التي استلمتها من Aspose. هذه الخطوة تُعلم Aspose برغبتك في استخدام ترخيص مُقاس.

## الخطوة 3: تحميل المستند الخاص بك

بعد ذلك، لنحمّل مستند وورد باستخدام Aspose.Words. في هذا المثال، سنستخدم مستندًا باسم `Document.docx`تأكد من وجود هذا المستند في دليل المشروع الخاص بك.

```csharp
Document doc = new Document("Document.docx");
```

## الخطوة 4: التحقق من طلب الترخيص

للتأكد من صحة تطبيق الترخيص، لنُجرِ عمليةً على المستند. سنطبع ببساطة عدد الصفحات في وحدة التحكم.

```csharp
Console.WriteLine(doc.PageCount);
```

تضمن هذه الخطوة أن مستندك تم تحميله ومعالجته باستخدام الترخيص المقياس.

## الخطوة 5: التعامل مع الاستثناءات

من الجيد دائمًا التعامل مع أي استثناءات محتملة. لنُضِف كتلة try-catch إلى شيفرتنا لإدارة الأخطاء بسلاسة.

```csharp
try
{
    Metered metered = new Metered();
    metered.SetMeteredKey("your_public_key", "your_private_key");

    Document doc = new Document("Document.docx");

    Console.WriteLine(doc.PageCount);
}
catch (Exception e)
{
    Console.WriteLine("There was an error setting the license: " + e.Message);
}
```

يضمن هذا أنه في حالة حدوث أي خطأ، فسوف تحصل على رسالة خطأ ذات معنى بدلاً من تعطل تطبيقك.
