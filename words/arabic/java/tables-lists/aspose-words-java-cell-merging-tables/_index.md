# إتقان دمج الخلايا الرأسية والأفقية في الجداول باستخدام Aspose.Words Java

## مقدمة
يُعدّ التعامل مع تنسيقات خلايا الجدول أمرًا أساسيًا في أتمتة المستندات لتحسين عرض البيانات. سواءً عند إنشاء فواتير أو تقارير، يُحسّن دمج الخلايا سهولة القراءة والجمال. قد يكون التحكم في عمليات الدمج الرأسي والأفقي أمرًا صعبًا.

يُبسّط Aspose.Words لجافا هذه المهام باستخدام واجهة برمجة تطبيقات قوية، مما يُتيح إنشاء مستندات احترافية بكل سهولة. سيرشدك هذا البرنامج التعليمي إلى إتقان دمج الخلايا باستخدام Aspose.Words في جافا.

### ما سوف تتعلمه:
- دمج الخلايا عموديًا وأفقيًا باستخدام Aspose.Words Java
- إعداد بيئتك باستخدام تبعيات Maven أو Gradle

لنبدأ بالتأكد من أن لديك كل ما تحتاجه للمتابعة.

## دليل التنفيذ
دعنا نقسم عملية دمج خلايا الجدول إلى خطوات يمكن إدارتها، مع التركيز على الدمج الرأسي والأفقي.

### دمج الخلايا العمودية

#### ملخص:
دمج الخلايا الرأسية يجمع صفوفًا متعددة داخل عمود واحد، وهو مثالي لإنشاء رؤوس أو تجميع المعلومات ذات الصلة.

#### التنفيذ خطوة بخطوة:
**1. إنشاء مستند ومنشئ:**
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

**2. إدراج الخلايا باستخدام الدمج الرأسي:**

- **الخلية الأولى (بدء الدمج):** تم تعيينه كبداية لدمج عمودي.
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.FIRST); // قم بتحديد هذه الخلية كنقطة بداية للدمج.
  builder.write("Text in merged cells.");
  ```

- **الخلية الثانية (غير قابلة للدمج):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.NONE); // لم يتم تطبيق الدمج هنا.
  builder.write("Text in unmerged cell.");
  builder.endRow(); // إنهاء الصف الحالي.
  ```

- **الخلية الثالثة (متابعة الدمج):** يتم دمجها مع الخلية الأولى عموديا.
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.PREVIOUS); // يستمر الدمج الرأسي من الخلية السابقة.
  builder.endRow(); // أكمل الصف الثاني.
  ```

**3. حفظ المستند:**
```java
doc.save("VerticalMergeOutput.docx");
```

### دمج الخلايا الأفقية

#### ملخص:
يجمع الدمج الأفقي الخلايا عبر صف واحد، وهو مثالي لإنشاء رؤوس شاملة أو معلومات ممتدة.

#### التنفيذ خطوة بخطوة:
**1. إنشاء مستند ومنشئ:**
أعد استخدام نفس رمز التهيئة كما في السابق.

**2. إدراج الخلايا باستخدام الدمج الأفقي:**

- **الخلية الأولى (بدء الدمج):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.FIRST); // يبدأ الدمج الأفقي.
  builder.write("Text in merged cells.");
  ```

- **الخلية الثانية (متابعة الدمج):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.PREVIOUS); // يستمر من الخلية الأولى أفقيًا.
  builder.endRow(); // إنهاء الصف الحالي، واستكمال الدمج الأفقي.
  ```

**3. حفظ المستند:**
```java
doc.save("HorizontalMergeOutput.docx");
```

### حشو الخلية

#### ملخص:
تؤدي إضافة الحشو إلى الخلايا إلى تحسين إمكانية القراءة من خلال إنشاء مسافة بيضاء بين النص والحدود.

#### التنفيذ خطوة بخطوة:
**1. تعيين الحشوات على الخلايا:**
```java
builder.getCellFormat().setPaddings(5.0, 10.0, 40.0, 50.0); // الحشوات العلوية، اليمنى، السفلية، اليسرى بالنقاط.
```

**2. إدراج خلية مع الحشو:**
```java
builder.startTable();
builder.insertCell();
builder.write("Lorem ipsum dolor sit amet...");
builder.endRow();
builder.endTable();
doc.save("PaddingOutput.docx");
```

## التطبيقات العملية
إن فهم كيفية دمج الخلايا وإضافة الحشو يمكن أن يعزز المستندات بطرق مختلفة:
1. **إنشاء الفاتورة**:استخدم الدمج الرأسي لأوصاف العناصر التي تمتد عبر عدة صفوف، مما يؤدي إلى تحسين الوضوح.
2. **إنشاء التقارير**:تعتبر عمليات الدمج الأفقية مثالية لتكوين عناوين أقسام موحدة عبر الجداول.
3. **نماذج السيرة الذاتية**:أضف حشوًا لضمان أن يكون النص الموجود داخل أقسام السيرة الذاتية مريحًا للعين.
