### تحميل مستند RTF مع التعرف على UTF-8

**ملخص:**
تتيح لك هذه الميزة فتح وقراءة مستندات RTF التي تحتوي على نص مشفر بتنسيق UTF-8، مما يضمن عرض جميع الأحرف بشكل صحيح.

#### الخطوة 1: استيراد الفئات الضرورية
ابدأ باستيراد الفئات المطلوبة من مكتبة Aspose.Words:
```java
import com.aspose.words.Document;
import com.aspose.words.RtfLoadOptions;
```
تتيح لك هذه الواردات العمل مع المستندات وتحديد خيارات التحميل لملفات RTF.

#### الخطوة 2: تكوين خيارات التحميل
إنشاء مثيل لـ `RtfLoadOptions` وتكوينه للتعرف على نص UTF-8:
```java
// إنشاء RtfLoadOptions لتحديد تكوينات التحميل
RtfLoadOptions loadOptions = new RtfLoadOptions();
loadOptions.setRecognizeUtf8Text(true);
```
جلسة `RecognizeUtf8Text` يضمن تعيين "true" أن المحلل يتعرف على الأحرف المشفرة بتنسيق UTF-8 ويفسرها بشكل صحيح داخل مستند RTF الخاص بك.

#### الخطوة 3: تحميل المستند
استخدم الخيارات المكوّنة لتحميل ملف RTF:
```java
// قم بتحميل مستند RTF باستخدام خيارات التحميل المحددة
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/UTF-8_characters.rtf", loadOptions);
```
ال `Document` يأخذ المنشئ مسار الملف والمسار المحدد مسبقًا `loadOptions`. استبدل "YOUR_DOCUMENT_DIRECTORY/UTF-8_characters.rtf" بمسار الملف الفعلي لديك.

#### الخطوة 4: استخراج النص
وأخيرًا، استخرج النص من المستند وقم بطباعته:
```java
// احصل على النص من القسم الأول من المستند وقم بطباعته
String text = doc.getFirstSection().getBody().getText().trim();
System.out.println(text);
```
يسترجع هذا الكود النص من نص القسم الأول في ملف RTF الخاص بك ويقص أي مسافة بادئة أو لاحقة.
