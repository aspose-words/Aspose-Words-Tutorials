# إتقان خيارات تحميل Markdown في Aspose.Words لـ Java

تعرّف على كيفية تسخير قوة Aspose.Words لجافا لإدارة مستندات Markdown بفعالية. سيوضح لك هذا الدليل كيفية الحفاظ على الأسطر الفارغة والتعرف على تنسيق التسطير في مشاريعك.

## مقدمة

هل تواجه صعوبة في تحويل ملفات Markdown مع الحفاظ على بنيتها الأصلية؟ لست وحدك. Markdown تنسيق متعدد الاستخدامات، لكن بساطته قد تُسبب تحديات عند استيرادها إلى تنسيقات مستندات أكثر ثراءً مثل Word. مع Aspose.Words لجافا، تتحول هذه العقبات إلى فرص. سيرشدك هذا البرنامج التعليمي خلال استخدام `MarkdownLoadOptions` فئة للاحتفاظ بالأسطر الفارغة وتفسير تنسيق التسطير في ملفات Markdown.

**ما سوف تتعلمه:**
- كيفية الحفاظ على الأسطر الفارغة أثناء استيراد العلامات.
- كيفية التعرف على تنسيق التسطير وتطبيقه من خلال بناء الجملة Markdown.
- عملية الإعداد لـ Aspose.Words باستخدام Maven أو Gradle.

بفضل هذه المهارات، يمكنك ضمان استيراد مستنداتك المُعدّلة إلى Word بدقة كما هو مُراد. لنبدأ بشرح المتطلبات الأساسية!

## دليل التنفيذ

### الحفاظ على الأسطر الفارغة في Markdown

يعد الحفاظ على الأسطر الفارغة أمرًا بالغ الأهمية عندما تريد الحفاظ على التخطيط المرئي للمستند.

#### ملخص
تضمن هذه الميزة الاحتفاظ بجميع فواصل الأسطر المتعمدة في علامات التمييز الخاصة بك أثناء التحويل، مما يحافظ على مستند Word الخاص بك متوافقًا مع تنسيقه الأصلي.

#### التنفيذ خطوة بخطوة:

##### تكوين خيارات التحميل

```java
// إنشاء مثيل MarkdownLoadOptions
MarkdownLoadOptions loadOptions = new MarkdownLoadOptions();
loadOptions.setPreserveEmptyLines(true);
```

##### تحميل المستند مع الخيارات

```java
String mdText = "Line1\n\nLine2\n";
Document doc = new Document(new ByteArrayInputStream(mdText.getBytes()), loadOptions);

// التحقق من الحفاظ على المحتوى
Assert.assertEquals("\rLine1\r\rLine2\r\f", doc.getText());
```

**توضيح**: 
- `setPreserveEmptyLines(true)`:يضمن الحفاظ على الأسطر الفارغة في Markdown.
- يؤكد التأكيد أن التحويل يحتفظ بفواصل الأسطر بشكل صحيح.

### التعرف على تنسيق التسطير

يؤدي تحويل صيغة التسطير إلى تنسيق فعلي إلى تحسين عرض المستند.

#### ملخص
التعرف على قواعد التسطير في Markdown وتطبيقها (`++text++`) حيث يضمن تنسيق الخط السفلي في Word أن يكون نمط مستندك متسقًا مع مصدر العلامات الخاص به.

#### التنفيذ خطوة بخطوة:

##### تكوين خيارات التحميل للخطوط السفلية

```java
// تمكين استيراد تنسيق التسطير
loadOptions.setImportUnderlineFormatting(true);
```

##### تحميل المستند والتحقق من التنسيق

```java
try (ByteArrayInputStream stream = new ByteArrayInputStream("++12 and B++".getBytes(StandardCharsets.US_ASCII))) {
    Document doc = new Document(stream, loadOptions);

    Paragraph para = (Paragraph)doc.getChild(NodeType.PARAGRAPH, 0, true);
    Assert.assertEquals(Underline.SINGLE, para.getRuns().get(0).getFont().getUnderline());
}
```

**توضيح**: 
- `setImportUnderlineFormatting(true)`:المتحولون `++text++` لتسطير النص.
- يتحقق التأكيد من أن التشغيل الأول في الفقرة يحتوي على تنسيق خط سفلي واحد.
