# Aspose.Words के साथ Word में बहुस्तरीय सूचियाँ बनाएँ
यह उदाहरण Aspose.Words for .NET का उपयोग करके Word दस्तावेज़ में एक बहुस्तरीय सूची बनाने का तरीका दर्शाता है। यह एक नया दस्तावेज़ और एक DocumentBuilder इंस्टेंस आरंभ करता है, एक नंबरिंग प्रारूप लागू करता है, मुख्य और उप-आइटम बनाता है, और दस्तावेज़ को .docx फ़ाइल के रूप में सहेजता है। क्रमांकित सूचियों के साथ अपने दस्तावेज़ को कुशलतापूर्वक प्रारूपित करने के लिए चरण-दर-चरण मार्गदर्शिका का पालन करें।

## स्थापना निर्देश
.NET के लिए Aspose.Words का उपयोग करने के लिए, इन चरणों का पालन करें:
1. लाइब्रेरी डाउनलोड करें: यहां जाएं [.NET के लिए Aspose.Words डाउनलोड पृष्ठ](https://www.aspose.com/downloads/words/net).
2. NuGet के माध्यम से स्थापित करें:
   - अपना विज़ुअल स्टूडियो प्रोजेक्ट खोलें.
   - टूल्स > NuGet पैकेज मैनेजर > समाधान के लिए NuGet पैकेज प्रबंधित करें पर जाएँ।
   - निम्न को खोजें `Aspose.Words` और इंस्टॉल पर क्लिक करें.
3. अपने कोड में लाइब्रेरी का संदर्भ दें:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. लाइसेंस लागू करें (वैकल्पिक):  
   पूर्ण संस्करण का उपयोग करने के लिए, [लाइसेंस लागू करें](https://purchase.aspose.com/temporary-license/) या का उपयोग करें [मुफ्त परीक्षण](https://releases.aspose.com/words/net/).
