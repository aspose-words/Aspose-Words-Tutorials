# विषय-सूची के साथ एक वर्ड दस्तावेज़ बनाएँ
यह उदाहरण दर्शाता है कि .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ कैसे बनाया जाता है जिसमें DocumentBuilder वर्ग का उपयोग करके जेनरेट की गई इंटरैक्टिव तालिका (TOC) शामिल होती है। कोड बताता है कि विभिन्न स्तरों के शीर्षक कैसे डालें, उन्हें फ़ॉर्मेट करें और परिणामी दस्तावेज़ को .docx फ़ाइल के रूप में कैसे सेव करें। आसान कार्यान्वयन के लिए नीचे दिए गए चरणों का पालन करें।

## स्थापना निर्देश
1. पूर्वापेक्षाएँ: सुनिश्चित करें कि आपकी मशीन पर .NET स्थापित है।
2. Aspose.Words स्थापित करें: 
   - NuGet पैकेज प्रबंधक कंसोल का उपयोग करें:
     ```
     Install-Package Aspose.Words
     ```
   - या, NuGet पैकेज मैनेजर GUI में "Aspose.Words" खोजें।
