# फ्लोटिंग इमेज के साथ वर्ड दस्तावेज़ बनाएं

यह उदाहरण दिखाता है कि .NET के लिए Aspose.Words का उपयोग करके वर्ड दस्तावेज़ कैसे बनाया जाए और एक फ्लोटिंग लेआउट वाली छवि कैसे डाली जाए। प्रदान किया गया कोड एक दस्तावेज़ और दस्तावेज़निर्माता को शुरू करता है, निर्दिष्ट आयामों और स्थिति के साथ एक छवि डालता है, और अंततः दस्तावेज़ को सहेजता है। यह गाइड डेवलपर्स के लिए आदर्श है जो Aspose.Words का उपयोग करके अपनी दस्तावेज़ निर्माण क्षमताओं को बढ़ाने की तलाश में हैं।

## स्थापना निर्देश
1. Aspose.Words स्थापित करेंः अपने .NET परियोजना में Aspose.Words NuGet पैकेज जोड़ें. आप पैकेज प्रबंधक कंसोल में निम्नलिखित कमांड चलाकर ऐसा कर सकते हैंः
   ```
   Install-Package Aspose.Words
   ```

2. आवश्यक नामस्थान जोड़ेंः सुनिश्चित करें कि आपकी कोड फ़ाइल में निम्नलिखित नामस्थान शामिल हैंः
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
