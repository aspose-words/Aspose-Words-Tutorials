# फ्लोटिंग इमेज के साथ वर्ड डॉक्यूमेंट बनाएं

यह उदाहरण दर्शाता है कि .NET के लिए Aspose.Words का उपयोग करके Word दस्तावेज़ कैसे बनाया जाता है और फ़्लोटिंग लेआउट के साथ एक छवि कैसे सम्मिलित की जाती है। प्रदान किया गया कोड एक दस्तावेज़ और DocumentBuilder को आरंभीकृत करता है, निर्दिष्ट आयामों और स्थिति के साथ एक छवि सम्मिलित करता है, और अंत में दस्तावेज़ को सहेजता है। यह मार्गदर्शिका उन डेवलपर्स के लिए आदर्श है जो Aspose.Words का उपयोग करके अपनी दस्तावेज़ निर्माण क्षमताओं को बढ़ाना चाहते हैं।

## स्थापना निर्देश
1. Aspose.Words इंस्टॉल करें: अपने .NET प्रोजेक्ट में Aspose.Words NuGet पैकेज जोड़ें। आप पैकेज मैनेजर कंसोल में निम्न कमांड चलाकर ऐसा कर सकते हैं:
   ```
   Install-Package Aspose.Words
   ```

2. आवश्यक नामस्थान जोड़ें: सुनिश्चित करें कि आपकी कोड फ़ाइल में निम्नलिखित नामस्थान शामिल हैं:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
