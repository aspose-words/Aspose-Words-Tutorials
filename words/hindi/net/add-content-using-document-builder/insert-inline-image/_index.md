# Word दस्तावेज़ में Aspose.Words का उपयोग करके इनलाइन छवि डालें
यह उदाहरण दिखाता है कि .NET के लिए Aspose.Words पुस्तकालय का उपयोग करके एक इनलाइन छवि को वर्ड दस्तावेज़ में कैसे डाला जाए।`दस्तावेज़निर्माता`वर्ग, आप आसानी से अपनी दस्तावेज़ में वर्तमान कर्सर स्थान पर चित्रों की स्थिति और सम्मिलित कर सकते हैं. यह गाइड आपको वर्ड दस्तावेज़ों में छवि प्रबंधन के साथ शुरू करने में मदद करने के लिए एक सीधा कार्यान्वयन प्रदान करता है।

## स्थापना निर्देश
1. .NET के लिए Aspose.Words स्थापित करेंः
   * विज़ुअल स्टूडियो में NuGet पैकेज प्रबंधक का उपयोग करें.
   * पैकेज प्रबंधक कंसोल में निम्न कमांड चलाएँः
     ```
     Install-Package Aspose.Words
     ```

2. अपनी परियोजना स्थापित करें
   * एक नई .NET परियोजना बनाएं या एक मौजूदा परियोजना खोलें।
   * अपनी कोड फ़ाइल के शीर्ष पर आवश्यक निर्देशों का उपयोग करके जोड़ेंः
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
