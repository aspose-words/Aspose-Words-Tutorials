# Aspose.Words का उपयोग करके Word दस्तावेज़ में इनलाइन छवि डालें
यह उदाहरण दर्शाता है कि .NET के लिए Aspose.Words लाइब्रेरी का उपयोग करके Word दस्तावेज़ में इनलाइन छवि कैसे डालें। `DocumentBuilder` क्लास का उपयोग करके, आप अपने दस्तावेज़ में वर्तमान कर्सर स्थान पर आसानी से छवियाँ रख सकते हैं और सम्मिलित कर सकते हैं। यह मार्गदर्शिका आपको Word दस्तावेज़ों में छवि प्रबंधन के साथ आरंभ करने में सहायता करने के लिए एक सरल कार्यान्वयन प्रदान करती है।

## स्थापना निर्देश
1. .NET के लिए Aspose.Words स्थापित करें:
   - विज़ुअल स्टूडियो में NuGet पैकेज मैनेजर का उपयोग करें।
   - पैकेज प्रबंधक कंसोल में निम्नलिखित कमांड चलाएँ:
     ```
     Install-Package Aspose.Words
     ```

2. अपना प्रोजेक्ट सेट करें:
   - एक नया .NET प्रोजेक्ट बनाएं या मौजूदा प्रोजेक्ट खोलें.
   - अपनी कोड फ़ाइल के शीर्ष पर आवश्यक using निर्देश जोड़ें:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
