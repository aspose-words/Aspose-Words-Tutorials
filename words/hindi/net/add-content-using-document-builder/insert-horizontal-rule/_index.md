# क्षैतिज नियम के साथ वर्ड दस्तावेज़ बनाएँ
यह उदाहरण Word दस्तावेज़ बनाने और DocumentBuilder का उपयोग करके क्षैतिज नियम जोड़ने के लिए .NET के लिए Aspose.Words के उपयोग को दर्शाता है। अपने दस्तावेज़ों को प्रभावी ढंग से फ़ॉर्मेट करने का तरीका जानने के लिए नीचे दिए गए चरणों का पालन करें।

## स्थापना निर्देश
1. Aspose.Words स्थापित करें: Visual Studio में NuGet पैकेज मैनेजर का उपयोग करें।
   - अपना प्रोजेक्ट Visual Studio में खोलें.
   - समाधान एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करें।
   - NuGet पैकेज प्रबंधित करें चुनें.
   - निम्न को खोजें `Aspose.Words` और इसे स्थापित करें.

2. उपयोग निर्देश जोड़ें: अपनी कोड फ़ाइल में, Aspose.Words क्लासेस का उपयोग करने के लिए निम्नलिखित निर्देश शामिल करें:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
