# क्षैतिज नियम के साथ वर्ड दस्तावेज़ बनाएं
यह उदाहरण एक वर्ड दस्तावेज़ बनाने के लिए .NET के लिए Aspose.Words का उपयोग करता है और DocumentBuilder का उपयोग करके एक क्षैतिज नियम जोड़ता है। अपने दस्तावेजों को प्रभावी ढंग से प्रारूपित करने के लिए नीचे दिए गए चरणों का पालन करें।

## स्थापना निर्देश
1. Aspose.Words स्थापित करेंः विज़ुअल स्टूडियो में NuGet पैकेज प्रबंधक का उपयोग करें.
   * विज़ुअल स्टूडियो में अपनी परियोजना खोलें।
   * समाधान एक्सप्लोरर में अपनी परियोजना पर राइट क्लिक करें।
   * NuGet पैकेज प्रबंधित करें चुनें.
   * खोजें`"अस्पोज. शब्द"`और इसे स्थापित करें.

2. उपयोग निर्देश जोड़ेंः अपनी कोड फ़ाइल में Aspose.Words वर्गों का उपयोग करने के लिए निम्नलिखित निर्देश शामिल करेंः
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
