### UTF-8 मान्यता के साथ RTF दस्तावेज़ लोड करें

**अवलोकन:**
यह सुविधा आपको UTF-8 एनकोडेड पाठ वाले RTF दस्तावेज़ों को खोलने और पढ़ने की अनुमति देती है, जिससे यह सुनिश्चित होता है कि सभी वर्ण सही ढंग से प्रदर्शित हों।

#### चरण 1: आवश्यक कक्षाएं आयात करें
Aspose.Words लाइब्रेरी से आवश्यक क्लासेस आयात करके प्रारंभ करें:
```java
import com.aspose.words.Document;
import com.aspose.words.RtfLoadOptions;
```
ये आयात आपको दस्तावेजों के साथ काम करने और RTF फ़ाइलों के लिए लोड विकल्प निर्दिष्ट करने की अनुमति देते हैं।

#### चरण 2: लोड विकल्प कॉन्फ़िगर करें
इसका एक उदाहरण बनाएं `RtfLoadOptions` और इसे UTF-8 पाठ पहचानने के लिए कॉन्फ़िगर करें:
```java
// लोडिंग कॉन्फ़िगरेशन निर्दिष्ट करने के लिए RtfLoadOptions बनाएँ
RtfLoadOptions loadOptions = new RtfLoadOptions();
loadOptions.setRecognizeUtf8Text(true);
```
सेटिंग `RecognizeUtf8Text` सत्य पर सेट करने से यह सुनिश्चित होता है कि पार्सर आपके RTF दस्तावेज़ में UTF-8 एनकोडेड वर्णों को पहचानता है और सही ढंग से व्याख्या करता है।

#### चरण 3: दस्तावेज़ लोड करें
RTF फ़ाइल लोड करने के लिए कॉन्फ़िगर किए गए विकल्पों का उपयोग करें:
```java
// निर्दिष्ट लोड विकल्पों के साथ RTF दस्तावेज़ लोड करें
Document doc = new Document("YOUR_DOCUMENT_DIRECTORY/UTF-8_characters.rtf", loadOptions);
```
The `Document` कन्स्ट्रक्टर एक फ़ाइल पथ और पहले से सेट किया गया पथ लेता है `loadOptions`"YOUR_DOCUMENT_DIRECTORY/UTF-8_characters.rtf" को अपने वास्तविक फ़ाइल पथ से बदलें।

#### चरण 4: पाठ निकालें
अंत में, दस्तावेज़ से पाठ निकालें और प्रिंट करें:
```java
// दस्तावेज़ के पहले खंड से पाठ प्राप्त करें और प्रिंट करें
String text = doc.getFirstSection().getBody().getText().trim();
System.out.println(text);
```
यह कोड आपकी RTF फ़ाइल के प्रथम खंड के मुख्य भाग से पाठ प्राप्त करता है तथा किसी भी प्रारंभिक या अंतिम रिक्त स्थान को काट देता है।
