## कार्यान्वयन मार्गदर्शिका

### एआई सारांशीकरण

बड़े दस्तावेज़ों को जल्दी से समझने के लिए टेक्स्ट को सारांशित करना ज़रूरी है। यहाँ बताया गया है कि आप Aspose.Words और OpenAI के साथ ऐसा कैसे कर सकते हैं:

#### एकल दस्तावेज़ सारांशीकरण
**अवलोकन:** यह सुविधा आपको एकल दस्तावेज़ को प्रभावी ढंग से सारांशित करने की अनुमति देती है।

- **दस्तावेज़ लोड करें:**
  ```python
  first_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **AI मॉडल कॉन्फ़िगर करें:**
  - सारांशीकरण के लिए OpenAI के GPT मॉडल का उपयोग करें।
  ```python
  api_key = 'YOUR_API_KEY'  
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model()
           .with_organization('Organization')
           .with_project('Project'))
  ```

- **संक्षेपण विकल्प सेट करें:**
  ```python
  options = aw.ai.SummarizeOptions()
  options.summary_length = aw.ai.SummaryLength.SHORT
  ```

- **सारांशीकरण करें:**
  ```python
  one_document_summary = model.summarize(source_document=first_doc, options=options)
  one_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.One.docx')
  ```

#### बहु-दस्तावेज़ सारांशीकरण

एक साथ कई दस्तावेज़ों का सारांश तैयार करने के लिए:

- **अतिरिक्त दस्तावेज़ लोड करें:**
  ```python
  second_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **सारांश की लंबाई समायोजित करें:**
  ```python
  options.summary_length = aw.ai.SummaryLength.LONG
  ```

- **एकाधिक दस्तावेज़ों का सारांश बनाएँ:**
  ```python
  multi_document_summary = model.summarize(source_documents=[first_doc, second_doc], options=options)
  multi_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.Multi.docx')
  ```

### एआई अनुवाद

दस्तावेजों का विभिन्न भाषाओं में अनुवाद करने से नए बाजार और दर्शक खुल सकते हैं।

#### अवलोकन:
यह सुविधा गूगल मॉडल का उपयोग करके पाठ का अनुवाद करती है।

- **दस्तावेज़ लोड करें:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **अनुवाद मॉडल कॉन्फ़िगर करें:**
  - अनुवाद के लिए Google AI का उपयोग करें.
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GEMINI_15_FLASH)
           .with_api_key(api_key)
           .as_google_ai_model())
  ```

- **दस्तावेज़ का अनुवाद करें:**
  ```python
  translated_doc = model.translate(doc, aw.ai.Language.ARABIC)
  translated_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiTranslate.docx')
  ```

### एआई व्याकरण जाँच

व्याकरण की जाँच करके दस्तावेज़ की गुणवत्ता में सुधार करना।

#### अवलोकन:
यह सुविधा आपके दस्तावेज़ों में व्याकरण संबंधी त्रुटियों की जांच करती है और उन्हें सुधारती है।

- **दस्तावेज़ लोड करें:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **व्याकरण मॉडल कॉन्फ़िगर करें:**
  - व्याकरण जाँच के लिए OpenAI के GPT मॉडल का उपयोग करें।
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model())
  ```

- **व्याकरण विकल्प सेट करें:**
  ```python
  grammar_options = aw.ai.CheckGrammarOptions()
  grammar_options.improve_stylistics = True
  ```

- **दस्तावेज़ जांचें और सहेजें:**
  ```python
  proofed_doc = model.check_grammar(doc, grammar_options)
  proofed_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiGrammar.docx')
  ```
