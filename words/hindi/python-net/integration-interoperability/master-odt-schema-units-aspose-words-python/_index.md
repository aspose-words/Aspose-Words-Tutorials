# पायथन में Aspose.Words के साथ ODT स्कीमा और इकाइयों में महारत हासिल करना

## परिचय

क्या आप यह सुनिश्चित करने के लिए संघर्ष कर रहे हैं कि आपके दस्तावेज़ विशिष्ट ओपन डॉक्यूमेंट फ़ॉर्मेट (ODF) मानकों का पालन करते हैं या फ़ाइलों को परिवर्तित करते समय माप इकाइयों पर सटीक नियंत्रण की आवश्यकता है? "Aspose.Words Python" लाइब्रेरी के साथ, आप इन चुनौतियों का आसानी से सामना कर सकते हैं। यह गाइड ODT स्कीमा सेटिंग और इकाई रूपांतरणों में महारत हासिल करने के लिए Python के लिए Aspose.Words का लाभ उठाने के बारे में है।

**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

### ODT स्कीमा संस्करणों के अनुरूप

#### अवलोकन

OpenDocument विनिर्देश (ODT स्कीमा) के विशिष्ट संस्करणों के साथ संगतता सुनिश्चित करने के लिए, Aspose.Words आपको यह परिभाषित करने की अनुमति देता है कि आपके दस्तावेज़ को संस्करण 1.1 विनिर्देशों का सख्ती से पालन करना चाहिए या नहीं।

**क्रमशः:**

##### चरण 1: सेव विकल्प सेट करना
```python
import aspose.words as aw

doc = aw.Document('path/to/your/input.docx')
save_options = aw.saving.OdtSaveOptions()
```

##### चरण 2: ODT स्कीमा संस्करण कॉन्फ़िगर करें
```python
# ODT संस्करण 1.1 के साथ सख्त अनुपालन के लिए इसे True पर सेट करें
save_options.is_strict_schema11 = True
```

##### चरण 3: दस्तावेज़ सहेजें
```python
doc.save('path/to/your/output.odt', save_options)
```

### मापन इकाइयाँ कॉन्फ़िगर करना

#### अवलोकन

ODT प्रारूप में दस्तावेज़ सहेजते समय Aspose.Words आपको मीट्रिक (सेंटीमीटर) और इंपीरियल (इंच) इकाइयों के बीच चयन करने देता है। यह लचीलापन सुनिश्चित करता है कि आपके स्टाइल पैरामीटर आवश्यक मानकों से मेल खाते हैं।

**क्रमशः:**

##### चरण 1: मापन इकाई का चयन
```python
save_options = aw.saving.OdtSaveOptions()
# अपनी आवश्यकताओं के आधार पर सेंटीमीटर या इंच में से चुनें
save_options.measure_unit = aw.saving.OdtSaveMeasureUnit.CENTIMETERS
```

##### चरण 2: दस्तावेज़ को इकाइयों के साथ सहेजें
```python
doc.save('path/to/your/output.odt', save_options)
```

### ODT/OTT दस्तावेज़ों को एन्क्रिप्ट करना

#### अवलोकन

Aspose.Words आपको अपने दस्तावेज़ों को एन्क्रिप्ट करके सुरक्षित करने की अनुमति देता है। यह अनुभाग बताता है कि ODT या OTT फ़ाइल को सहेजते समय पासवर्ड सुरक्षा कैसे लागू करें।

**क्रमशः:**

##### चरण 1: दस्तावेज़ आरंभ करें और विकल्प सहेजें
```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Hello world!")
save_options = aw.saving.OdtSaveOptions(aw.SaveFormat.ODT)
```

##### चरण 2: पासवर्ड सुरक्षा सेट करें
```python
# एन्क्रिप्शन के लिए पासवर्ड सेट करें
save_options.password = 'your_password_here'
doc.save('path/to/encrypted_output.odt', save_options)
```

## व्यावहारिक अनुप्रयोगों

यहां कुछ वास्तविक परिदृश्य दिए गए हैं जहां इन सुविधाओं को लागू किया जा सकता है:

1. **दस्तावेज़ अनुपालन**यह सुनिश्चित करना कि कानूनी दस्तावेज़ संगठनात्मक या नियामक मानकों के अनुरूप हों।
2. **क्रॉस-प्लेटफ़ॉर्म संगतता**: उन प्रणालियों में उपयोग के लिए दस्तावेजों को अनुकूलित करना जो ODT स्कीमा संस्करणों का सख्ती से पालन करते हैं।
3. **सुरक्षित दस्तावेज़ साझाकरण**ईमेल या क्लाउड सेवाओं के माध्यम से साझा करने से पहले संवेदनशील जानकारी को एन्क्रिप्ट करना।
