## Frequently Asked Questions

**Q: Can I change the page‑number format (e.g., Roman to Arabic) programmatically?**  
A: Yes, you can set the `PageNumberStyle` property on the `PageNumber` field to any supported format.

**Q: How do I skip page numbers on specific pages, such as title pages?**  
A: Insert a `PageNumber` field with the `\* MERGEFORMAT` switch and set the `Suppress` attribute on the header/footer for that section.

**Q: Is it possible to reset page numbers for each chapter automatically?**  
A: Absolutely. Use the `Section` class to create a new section for each chapter and enable `PageSetup.RestartPageNumbering`.

**Q: Does the API handle large documents efficiently?**  
A: Aspose.Words processes documents in a streaming fashion, minimizing memory usage even for files with thousands of pages.

**Q: What licensing options are available for production use?**
