## Frequently Asked Questions

**Q: Can I use this approach with on‑premise AI models?**  
A: Yes – Aspose.Words works with any HTTP‑accessible AI endpoint, including self‑hosted models.

**Q: How do I keep the original document formatting after translation?**  
A: Retrieve the translated text, then replace the original runs while preserving the existing style definitions.

**Q: Is there a limit to the size of documents I can process?**  
A: Aspose.Words can handle large files, but memory consumption grows with document complexity; consider streaming large PDFs.

**Q: Do I need to train my own models for summarization?**  
A: Not necessarily – pre‑trained models like GPT‑4 or Gemini provide high‑quality summaries out‑of‑the‑box.

**Q: How can I monitor AI usage costs?**  
A: Log each request’s token count and associate it with a billing tag; many AI providers offer usage dashboards.
