## Installation Instructions  
To use this example, ensure you have Aspose.Words for .NET installed. Follow these steps:  

1. Download Aspose.Words for .NET:  
   Visit [Aspose.Words for .NET Download](https://releases.aspose.com/words/net/) and download the latest version.  

2. Install the Library:  
   - Add the Aspose.Words NuGet package to your project. Run the following command in the NuGet Package Manager Console:  
     ```sh
     Install-Package Aspose.Words
     ```  
   - Alternatively, add the `.dll` files to your project references manually.  

3. Get a License:  
   - Obtain a license from [Aspose Purchase](https://purchase.aspose.com/buy) or request a free trial from [Aspose Free Trial](https://releases.aspose.com/).  

4. Apply the License:  
   Add the following code to your project to activate the license:  
   ```csharp
   License license = new License();
   license.SetLicense("path-to-your-license-file.lic");
   ```  

5. Run the Example:  
   Copy the provided code, customize it if necessary, and execute it in your development environment. Make sure to include references to `Aspose.Words`.
