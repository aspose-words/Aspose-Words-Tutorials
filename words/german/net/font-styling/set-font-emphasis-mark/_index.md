# Setzen Sie mit Aspose.Words ein Hervorhebungszeichen im Word-Dokument

Das Setzen eines Hervorhebungszeichens in Word-Dokumenten ist eine hervorragende Möglichkeit, bestimmten Text hervorzuheben und ihn so in Ihren Inhalten hervorzuheben. Mit Aspose.Words für .NET können Sie mit wenigen Codezeilen mühelos Hervorhebungszeichen, wie z. B. einen ausgefüllten Kreis, einfügen. Dieses Beispiel zeigt die Verwendung von `DocumentBuilder` Mit dieser Klasse können Sie Word-Dokumente bearbeiten, Text mit Hervorhebungen formatieren und das Ergebnis im DOCX-Format speichern. Folgen Sie dieser Anleitung, um Ihre Dokumentformatierungsaufgaben mit professioneller Präzision zu optimieren.

## Installationsanweisungen  
Um das bereitgestellte Codebeispiel auszuführen und Aspose.Words für .NET zu verwenden, führen Sie die folgenden Schritte aus:  

1. Laden Sie Aspose.Words für .NET herunter:  
   - Holen Sie sich die Bibliothek von der [Aspose-Veröffentlichungen](https://releases.aspose.com/words/net/) Seite.  

2. Installieren Sie die Bibliothek:  
   - Installieren Sie es über den NuGet-Paket-Manager in Visual Studio:  
     - Öffnen Sie Visual Studio.  
     - Navigieren Sie zu Tools > NuGet-Paket-Manager > NuGet-Pakete für Lösung verwalten.  
     - Suchen nach `Aspose.Words` und klicken Sie auf Installieren.  

   - Alternativ können Sie die NuGet-Paket-Manager-Konsole verwenden:  
     ```shell
     Install-Package Aspose.Words
     ```  

3. Einrichten einer temporären Lizenz (optional):  
   - Für die uneingeschränkte Nutzung erhalten Sie eine [kostenlose temporäre Lizenz](https://purchase.aspose.com/temporary-license/).  
   - Wenden Sie die Lizenz in Ihrem Projekt an:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
