# Erstellen Sie mehrere Listen in Word mit Aspose.Words
Dieses Beispiel zeigt, wie man eine mehrstufige Liste in einem Word-Dokument mit Aspose.Words für .NET erstellt. Es initialisiert ein neues Dokument und eine Dokumentbuilder-Instanz, wendet ein Nummerierungsformat an, erstellt Haupt- und Unterartikel und speichert das Dokument als .docx-Datei. Folgen Sie der Schritt-für-Schritt-Anleitung, um Ihr Dokument effizient mit nummerierten Listen zu formatieren.

## Installationsanweisungen
Um Aspose.Words für .NET zu verwenden, folgen Sie diesen Schritten:
1. Download der Bibliothek: Besuchen Sie [Aspose.Words für die .NET- Downloadseite](https://www.aspose.com/downloads/words/net)\- Ich weiß .
2. Installieren Sie über NuGet:
   * Öffnen Sie Ihr Visual Studio Projekt.
   * Navigieren Sie zu Tools > NuGet Package Manager > Manage NuGet Packages for Solution.
   * Suchen Sie nach `"Aspose.Wörter"`und klicken Sie auf Installieren.
3. Verweisen Sie die Bibliothek in Ihrem Code:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
