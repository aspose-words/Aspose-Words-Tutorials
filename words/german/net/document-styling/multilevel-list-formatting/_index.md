# Erstellen Sie mehrstufige Listen in Word mit Aspose.Words
Dieses Beispiel zeigt, wie Sie mit Aspose.Words für .NET eine mehrstufige Liste in einem Word-Dokument erstellen. Es initialisiert ein neues Dokument und eine DocumentBuilder-Instanz, wendet ein Nummerierungsformat an, erstellt Haupt- und Unterelemente und speichert das Dokument als DOCX-Datei. Folgen Sie der Schritt-für-Schritt-Anleitung, um Ihr Dokument effizient mit nummerierten Listen zu formatieren.

## Installationsanweisungen
Um Aspose.Words für .NET zu verwenden, führen Sie die folgenden Schritte aus:
1. Laden Sie die Bibliothek herunter: Besuchen Sie die [Aspose.Words für .NET-Downloadseite](https://www.aspose.com/downloads/words/net).
2. Über NuGet installieren:
   - Öffnen Sie Ihr Visual Studio-Projekt.
   - Navigieren Sie zu Tools > NuGet-Paket-Manager > NuGet-Pakete für Lösung verwalten.
   - Suchen nach `Aspose.Words` und klicken Sie auf Installieren.
3. Verweisen Sie in Ihrem Code auf die Bibliothek:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Lizenz anwenden (optional):  
   Um die Vollversion zu nutzen, [eine Lizenz beantragen](https://purchase.aspose.com/temporary-license/) oder verwenden Sie eine [kostenlose Testversion](https://releases.aspose.com/words/net/).
