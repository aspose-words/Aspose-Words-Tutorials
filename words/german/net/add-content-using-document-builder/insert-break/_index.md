# Erstellen Sie ein Multi-Page Word-Dokument mit Aspose.Words

Dieses Beispiel zeigt, wie man ein Word-Dokument mit Aspose.Words für .NET erstellt, so dass Sie Inhalte auf mehreren Seiten hinzufügen können. Es zeigt, wie man ein neues Dokument initialisiert, den DocumentBuilder verwendet, um Text- und Seitenumbrüche einzufügen und schließlich das Dokument als .docx- Datei zu speichern. Perfekt für Entwickler, die die Dokumentgenerierung in ihren .NET-Anwendungen automatisieren möchten.

## Installationshinweise
1. Installieren Sie Aspose.Words für .NET: Verwenden Sie NuGet Package Manager, um Aspose.Words zu installieren. Führen Sie den folgenden Befehl in der Package Manager Console aus:
```
Install-Package Aspose.Words
```
