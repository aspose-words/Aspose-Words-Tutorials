# Erstellen Sie ein Word-Dokument mit Inhaltsverzeichnis
Dieses Beispiel zeigt, wie man Aspose.Words für .NET verwendet, um ein Word- Dokument zu erstellen, das ein interaktives Table of Contents (TOC) enthält, das mit der DocumentBuilder-Klasse generiert wird. Der Code zeigt, wie Sie Überschriften verschiedener Ebenen einfügen, formatieren und das resultierende Dokument als .docx-Datei speichern können.

## Installationshinweise
1. Voraussetzungen: Stellen Sie sicher, dass .NET auf Ihrem Rechner installiert ist.
2. Installieren Sie Aspose.Words: 
   * Verwenden Sie NuGet Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
   * Oder suchen Sie nach "Aspose.Words" in der NuGet Package Manager GUI.
