## Installationsanweisungen
1. Voraussetzungen: Stellen Sie sicher, dass .NET auf Ihrem Computer installiert ist.
2. Installieren Sie Aspose.Words: 
   - Verwenden Sie die NuGet-Paket-Manager-Konsole:
     ```
     Install-Package Aspose.Words
     ```
   - Oder suchen Sie in der GUI des NuGet-Paket-Managers nach „Aspose.Words“.
