# Erstellen Sie ein Word-Dokument mit schwebendem Bild

Dieses Beispiel zeigt, wie Sie mit Aspose.Words für .NET ein Word-Dokument erstellen und ein Bild mit schwebendem Layout einfügen. Der bereitgestellte Code initialisiert ein Dokument und einen DocumentBuilder, fügt ein Bild mit angegebenen Abmessungen und Positionierung ein und speichert das Dokument anschließend. Diese Anleitung ist ideal für Entwickler, die ihre Dokumenterstellungsfunktionen mit Aspose.Words erweitern möchten.

## Installationsanweisungen
1. Installieren Sie Aspose.Words: Fügen Sie das Aspose.Words NuGet-Paket zu Ihrem .NET-Projekt hinzu. Führen Sie dazu den folgenden Befehl in der Paket-Manager-Konsole aus:
   ```
   Install-Package Aspose.Words
   ```

2. Fügen Sie die erforderlichen Namespaces hinzu: Stellen Sie sicher, dass Ihre Codedatei die folgenden Namespaces enthält:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
