## Häufig gestellte Fragen

**Q: Kann ich das Seitenzahl‑Format (z. B. römisch zu arabisch) programmatisch ändern?**  
A: Ja, Sie können die Eigenschaft `PageNumberStyle` im `PageNumber`‑Feld auf jedes unterstützte Format setzen.

**Q: Wie kann ich Seitenzahlen auf bestimmten Seiten, wie Titelseiten, überspringen?**  
A: Fügen Sie ein `PageNumber`‑Feld mit dem Schalter `\* MERGEFORMAT` ein und setzen Sie das Attribut `Suppress` in der Kopf‑ bzw. Fußzeile für diesen Abschnitt.

**Q: Ist es möglich, die Seitenzahlen für jedes Kapitel automatisch zurückzusetzen?**  
A: Absolut. Verwenden Sie die Klasse `Section`, um für jedes Kapitel einen neuen Abschnitt zu erstellen, und aktivieren Sie `PageSetup.RestartPageNumbering`.

**Q: Verarbeitet die API große Dokumente effizient?**  
A: Aspose.Words verarbeitet Dokumente in einem Streaming‑Modus, wodurch der Speicherverbrauch selbst bei Dateien mit tausenden Seiten minimiert wird.

**Q: Welche Lizenzoptionen stehen für den Produktionseinsatz zur Verfügung?**  
A: Sie können mit einer temporären Lizenz zur Evaluierung beginnen und auf eine unbefristete oder Abonnement‑Lizenz für den kommerziellen Einsatz upgraden.
