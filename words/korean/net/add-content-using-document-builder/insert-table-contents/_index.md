## 설치 지침
1. 전제 조건: 컴퓨터에 .NET를 설치한 것을 확인하세요.
2. Aspose.Words를 설치합니다. 
   * NuGet 패키지 관리자 콘솔을 사용하세요:
     ```
     Install-Package Aspose.Words
     ```
   * 또는 NuGet 패키지 관리자 GUI에서 "Aspose.Words"를 검색하십시오.
