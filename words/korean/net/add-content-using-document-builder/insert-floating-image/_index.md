# 부동의 이미지를 사용하여 워드 문서를 생성

이 예는 .NET에 대한 Aspose.Words를 사용하여 워드 문서를 생성하고 부동의 레이아웃을 가진 이미지를 삽입하는 방법을 보여줍니다. 제공된 코드는 문서 및 문서 빌더를 초기화하고, 지정된 차원 및 위치와 함께 이미지를 삽입하고, 마지막으로 문서를 저장합니다. 이 가이드는 Aspose.Words를 사용하여 문서 생성 기능을 향상시키고자 하는 개발자들에게 이상적입니다.

## 설치 지침
1. Aspose.Words을 설치합니다. Aspose.Words NuGet 패키지를 .NET 프로젝트에 추가합니다. 패키지 관리자 콘솔에서 다음 명령을 실행하면 할 수 있습니다.
   ```
   Install-Package Aspose.Words
   ```

2. 필요한 이름 공간을 추가하십시오. 코드 파일에는 다음과 같은 이름 공간을 포함하도록 하십시오.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
