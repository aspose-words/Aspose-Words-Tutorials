# 떠다니는 이미지로 Word 문서 만들기

이 예제에서는 Aspose.Words for .NET을 사용하여 Word 문서를 만들고 떠다니는 레이아웃의 이미지를 삽입하는 방법을 보여줍니다. 제공된 코드는 Document와 DocumentBuilder를 초기화하고, 지정된 크기와 위치로 이미지를 삽입한 후, 마지막으로 문서를 저장합니다. 이 가이드는 Aspose.Words를 사용하여 문서 생성 기능을 향상시키고자 하는 개발자에게 이상적입니다.

## 설치 지침
1. Aspose.Words 설치: Aspose.Words NuGet 패키지를 .NET 프로젝트에 추가하세요. 패키지 관리자 콘솔에서 다음 명령을 실행하면 됩니다.
   ```
   Install-Package Aspose.Words
   ```

2. 필수 네임스페이스 추가: 코드 파일에 다음 네임스페이스가 포함되어 있는지 확인하세요.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
