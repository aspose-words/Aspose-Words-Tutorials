# Aspose.Words를 사용하여 여러 페이지로 구성된 Word 문서 만들기

이 예제는 Aspose.Words for .NET을 사용하여 Word 문서를 만드는 방법을 보여줍니다. 여러 페이지에 걸쳐 콘텐츠를 추가할 수 있습니다. 새 문서를 초기화하고, DocumentBuilder를 사용하여 텍스트와 페이지 나누기를 삽입하고, 마지막으로 문서를 .docx 파일로 저장하는 방법을 보여줍니다. .NET 애플리케이션에서 문서 생성을 자동화하려는 개발자에게 적합합니다.

## 설치 지침
1. Aspose.Words for .NET 설치: NuGet 패키지 관리자를 사용하여 Aspose.Words를 설치하세요. 패키지 관리자 콘솔에서 다음 명령을 실행하세요.
```
Install-Package Aspose.Words
```
