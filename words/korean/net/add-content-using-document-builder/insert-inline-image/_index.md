# Aspose.Words를 사용하여 워드 문서에 인라인 이미지를 삽입하십시오
이 예제에서는 .NET의 Aspose.Words 라이브러리를 사용하여 Word 문서에 인라인 이미지를 삽입하는 방법을 보여줍니다.`문서 제작자`이 가이드에서는 워드 문서에서 이미지 처리 작업을 시작하는데 도움이 되는 간단한 구현을 제공합니다.

## 설치 지침
1. .NET에 Aspose.Words를 설치합니다.
   * Visual Studio에서 NuGet 패키지 관리자를 사용하십시오.
   * 패키지 관리자 콘솔에서 다음 명령을 실행하세요:
     ```
     Install-Package Aspose.Words
     ```

2. 사업 을 설정 하십시오:
   * 새로운 .NET 프로젝트를 만들거나 기존 프로젝트를 열 수 있습니다.
   * 코드 파일 상단에 필요한 지침을 사용하여 추가합니다.
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
