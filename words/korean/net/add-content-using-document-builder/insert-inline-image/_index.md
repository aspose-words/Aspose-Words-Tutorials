# Aspose.Words를 사용하여 Word 문서에 인라인 이미지 삽입
이 예제에서는 .NET용 Aspose.Words 라이브러리를 사용하여 Word 문서에 인라인 이미지를 삽입하는 방법을 보여줍니다. `DocumentBuilder` 클래스를 사용하면 문서의 현재 커서 위치에 이미지를 쉽게 배치하고 삽입할 수 있습니다. 이 가이드는 Word 문서에서 이미지 처리를 시작하는 데 도움이 되는 간단한 구현 방법을 제공합니다.

## 설치 지침
1. Aspose.Words for .NET을 설치하세요:
   - Visual Studio에서 NuGet 패키지 관리자를 사용합니다.
   - 패키지 관리자 콘솔에서 다음 명령을 실행합니다.
     ```
     Install-Package Aspose.Words
     ```

2. 프로젝트 설정:
   - 새로운 .NET 프로젝트를 만들거나 기존 프로젝트를 엽니다.
   - 코드 파일의 맨 위에 필요한 using 지시문을 추가합니다.
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
