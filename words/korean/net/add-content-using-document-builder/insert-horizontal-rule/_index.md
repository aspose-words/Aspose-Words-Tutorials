## 설치 지침
1. Aspose.Words 설치: Visual Studio에서 NuGet 패키지 관리자를 사용합니다.
   - Visual Studio에서 프로젝트를 엽니다.
   - 솔루션 탐색기에서 프로젝트를 마우스 오른쪽 버튼으로 클릭합니다.
   - NuGet 패키지 관리를 선택합니다.
   - 검색 `Aspose.Words` 그리고 설치하세요.

2. Using 지시문 추가: Aspose.Words 클래스를 사용하려면 코드 파일에 다음 지시문을 포함하세요.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
