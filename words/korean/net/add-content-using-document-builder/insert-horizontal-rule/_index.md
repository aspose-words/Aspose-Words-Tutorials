## 설치 지침
1. Aspose.Words을 설치합니다. Visual Studio에서 NuGet 패키지 관리자를 사용하세요.
   * 비주얼 스튜디오에서 프로젝트를 열세요.
   * 솔루션 익스플로러에서 프로젝트의 오른쪽 클릭을 해보세요.
   * NuGet 패키지를 관리하는 것을 선택하십시오.
   * 검색`"말하라. 말하라".`그리고 설치해

2. 사용 지침을 추가하십시오. 코드 파일에 Aspose.Words 클래스를 사용하기 위해 다음 지침을 포함하십시오.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
