# Aspose.Words를 사용하여 Word 문서에 도형 삽입
이 가이드에서는 Aspose.Words for .NET을 사용하여 Word 문서에 텍스트 상자와 같은 도형을 삽입하고 조작하는 방법을 보여줍니다. 도형을 절대 또는 상대적으로 배치하고, 회전 각도를 설정하고, ISO 29500:2008 Transitional DOCX 형식으로 문서를 저장하는 방법을 알아봅니다. `OoxmlSaveOptions`이 예제는 Word 문서에서 도형 작업을 간소화하여 개발자가 시각적으로 매력적인 콘텐츠를 쉽게 만들 수 있도록 해줍니다.

## 설치 지침  
1. Aspose.Words for .NET을 다운로드하세요:  
   .NET용 Aspose.Words의 최신 버전을 여기에서 받으세요. [다운로드 페이지](https://releases.aspose.com/words/net/).  

2. NuGet을 통해 설치:  
   Visual Studio에서 프로젝트를 열고 NuGet 패키지 관리자를 사용하여 라이브러리를 설치합니다.  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. 네임스페이스 참조 추가:  
   코드에 Aspose.Words에 대한 네임스페이스를 포함하세요.  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. 라이선스 활성화(선택 사항):
