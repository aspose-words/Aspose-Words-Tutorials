# Aspose.Words을 사용하여 단어 분포 차트를 만들

.NET에 대한 Aspose.Words는 워드 문서를 프로그래밍적으로 생성하고 조작하는 것을 단순화합니다. 이 예제에서는 워드 문서에 스캐더 차트를 삽입하는 방법을 보여줍니다.`문서 제작자`콘텐츠 제작 및 프로그램적으로 차트 데이터를 삽입하기 위해, 당신은 효율적으로 전문적인 수준의 스캐더 차트를 문서에 추가 할 수 있습니다. 직관적인 API를 통해, Aspose.Words는 .NET 개발자를 위한 문서 자동화를 효율화, 시간과 노력을 절약합니다.

## 설치 지침  
이 예제를 실행하려면 다음 단계를 수행하십시오.  

1. .NET에 Aspose.Words를 다운로드하고 설치합니다  
   * 라이브러리를 공식 Aspose 웹사이트에서 다운로드 합니다.[.NET 다운로드에 대한 Aspose.Words](https://releases.aspose.com/words/net/)\- 그래  
   * NuGet 패키지 관리자 를 통해 설치:  
     ```bash  
     Install-Package Aspose.Words  
     ```  

2. 발전 환경 을 설정 하십시오  
   * .NET IDE를 열 (예를 들어, 비주얼 스튜디오)  
   * 새로운 콘솔 애플리케이션을 만들거나 기존 프로젝트에 통합하십시오.  
   * .NET 라이브러리에서 Aspose.Words에 참조를 추가합니다.  

3. 면허 신청  
   * 만약 당신이 라이선스를 가지고 있다면, 다음을 사용하여 프로젝트를 적용하십시오.  
     ```csharp  
     Aspose.Words.License license = new Aspose.Words.License();  
     license.SetLicense("Your_License_File.lic");  
     ```  
   * 대체로,[무료 임시 면허](https://purchase.aspose.com/temporary-license/)도서관을 시험해 볼 수 있습니다.
