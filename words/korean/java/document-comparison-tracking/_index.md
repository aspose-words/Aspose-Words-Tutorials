## 자주 묻는 질문

**Q: aspose words java를 웹 애플리케이션에서 사용할 수 있나요?**  
A: 예. 이 라이브러리는 서블릿 컨테이너, Spring Boot 및 모든 Java 기반 웹 프레임워크에서 작동합니다.

**Q: 비교 시 이미지가 어떻게 처리되나요?**  
A: 이미지는 바이너리 콘텐츠를 기준으로 비교됩니다. 차이가 있으면 이미지 변경을 나타내는 수정이 추가됩니다.

**Q: 비교 중 특정 요소(예: 머리글)를 무시할 수 있나요?**  
A: 물론입니다. `ComparisonOptions`를 구성하여 머리글, 바닥글 또는 특정 스타일을 무시하도록 설정할 수 있습니다.

**Q: 어떤 출력 형식이 추적된 변경을 유지합니까?**  
A: DOCX, DOC, RTF는 수정 마크업을 유지합니다. PDF로 내보낼 때는 변경을 시각적으로 렌더링하도록 선택할 수 있습니다.

**Q: aspose words java가 다수의 문서를 한 번에 비교하는 것을 지원하나요?**  
A: 예. 문서 쌍을 순회하면서 단일 `Comparison` 인스턴스를 재사용하면 대량 배치를 효율적으로 처리할 수 있습니다.
