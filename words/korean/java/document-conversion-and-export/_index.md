## Frequently Asked Questions

**Q: Can I convert DOCX to PDF on a Linux server?**  
A: Yes. Aspose.Words for Java is platform‑independent and works on any OS with a supported JRE.  
**Q: Linux 서버에서 DOCX를 PDF로 변환할 수 있나요?**  
A: 예. Aspose.Words for Java는 플랫폼에 독립적이며 지원되는 JRE가 설치된 모든 OS에서 동작합니다.

**Q: Do I need Microsoft Office installed on the server?**  
A: No. The conversion is performed entirely by the Aspose.Words library.  
**Q: 서버에 Microsoft Office를 설치해야 하나요?**  
A: 아니요. 변환은 전적으로 Aspose.Words 라이브러리에 의해 수행됩니다.

**Q: How can I preserve custom fonts during conversion?**  
A: Use `PdfSaveOptions.setEmbedFullFonts(true)` to embed all used fonts into the resulting PDF.  
**Q: 변환 중에 사용자 정의 폰트를 보존하려면 어떻게 해야 하나요?**  
A: `PdfSaveOptions.setEmbedFullFonts(true)`를 사용하여 사용된 모든 폰트를 결과 PDF에 임베드하십시오.

**Q: Is it possible to convert password‑protected DOCX files?**  
A: Absolutely. Load the document with `LoadOptions.setPassword("yourPassword")` before saving as PDF.  
**Q: 비밀번호로 보호된 DOCX 파일을 변환할 수 있나요?**  
A: 물론입니다. PDF로 저장하기 전에 `LoadOptions.setPassword("yourPassword")`를 사용해 문서를 로드하십시오.

**Q: What is the recommended way to batch‑convert many documents?**  
A: Create a reusable method that loads, converts, and disposes of each document, and run it in a thread‑pool for parallel processing.  
**Q: 많은 문서를 배치 변환하는 권장 방법은 무엇인가요?**  
A: 각 문서를 로드·변환·해제하는 재사용 가능한 메서드를 만든 뒤, 스레드 풀에서 병렬 처리하도록 실행하십시오.
