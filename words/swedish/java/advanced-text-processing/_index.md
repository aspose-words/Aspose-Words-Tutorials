## Vanliga frågor

**Q: Kan jag ändra sidnummerformatet (t.ex. romerskt till arabiskt) programatiskt?**  
A: Ja, du kan sätta `PageNumberStyle`‑egenskapen på `PageNumber`‑fältet till vilket stödformat som helst.

**Q: Hur hoppar jag över sidnummer på specifika sidor, såsom titelsidor?**  
A: Infoga ett `PageNumber`‑fält med `\* MERGEFORMAT`‑växeln och sätt `Suppress`‑attributet på sidhuvudet/sidfoten för den sektionen.

**Q: Är det möjligt att återställa sidnummer för varje kapitel automatiskt?**  
A: Absolut. Använd `Section`‑klassen för att skapa en ny sektion för varje kapitel och aktivera `PageSetup.RestartPageNumbering`.

**Q: Hantera API:t stora dokument effektivt?**  
A: Aspose.Words bearbetar dokument i ett strömningsläge, vilket minimerar minnesanvändning även för filer med tusentals sidor.

**Q: Vilka licensalternativ finns tillgängliga för produktionsbruk?**  
A: Du kan börja med en tillfällig licens för utvärdering och uppgradera till en evig eller prenumerationslicens för kommersiell distribution.
