# Skapa listor på flera nivåer i Word med Aspose.Words
Det här exemplet visar hur man skapar en flernivålista i ett Word-dokument med hjälp av Aspose.Words för .NET. Programmet initierar ett nytt dokument och en DocumentBuilder-instans, tillämpar ett numreringsformat, skapar huvud- och underobjekt och sparar dokumentet som en .docx-fil. Följ steg-för-steg-guiden för att effektivt formatera ditt dokument med numrerade listor.

## Installationsanvisningar
För att använda Aspose.Words för .NET, följ dessa steg:
1. Ladda ner biblioteket: Besök [Aspose.Words för .NET nedladdningssida](https://www.aspose.com/downloads/words/net).
2. Installera via NuGet:
   - Öppna ditt Visual Studio-projekt.
   - Navigera till Verktyg > NuGet-pakethanteraren > Hantera NuGet-paket för lösningen.
   - Leta efter `Aspose.Words` och klicka på Installera.
3. Referera till biblioteket i din kod:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Använd licens (valfritt):  
   För att använda den fullständiga versionen, [ansök om en licens](https://purchase.aspose.com/temporary-license/) eller använd en [gratis provperiod](https://releases.aspose.com/words/net/).
