# Skapa Word-dokument med huvud och sidfot med hjälp av aspose.Words

Det här exemplet visar hur man använder Aspose.Words för .NET för att skapa ett Word-dokument med ett primärt huvud och sidfot. Guiden går igenom att initiera ett nytt dokument, flytta till sidhuvudet och sidfoten sektioner, lägga till anpassad text, och spara dokumentet i DOCX- format. Med sin kraftfulla DocumentBuilder klass, Aspose.Words gör Word dokument manipulation sömlös och effektiv. Oavsett om du lägger till branding element eller sidnummer, detta exempel ger grunden för att anpassa Word-dokument programmatiskt.
