# Infoga ett bubbeldiagram i Word med hjälp av Aspose.Words

Lägg enkelt till ett bubbeldiagram i ditt Word-dokument med Aspose.Words för .NET! Den här guiden guidar dig genom hur du skapar ett nytt dokument, infogar ett bubbeldiagram, lägger till dataserier med X-, Y- och storleksvärden och sparar dokumentet. Med Aspose.Words har det aldrig varit enklare att hantera Word-dokument programmatiskt.

## Installationsanvisningar  

1. Ladda ner Aspose.Words för .NET  
   Hämta biblioteket från [Aspose.Words nedladdningssida](https://releases.aspose.com/words/net/).  

2. Installera via NuGet-pakethanteraren  
   - Öppna ditt Visual Studio-projekt.  
   - Navigera till Verktyg > NuGet-pakethanteraren > Hantera NuGet-paket för lösningen.  
   - Leta efter `Aspose.Words` på fliken Bläddra.  
   - Välj paketet och klicka på Installera.  

3. Installera via NuGet Package Manager-konsolen  
   Kör följande kommando i pakethanterarkonsolen:  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Lägg till licens  
   För full funktionalitet, ansök om en licens. Du kan få en tillfällig licens från [Aspose tillfällig licens sida](https://purchase.aspose.com/temporary-license/).
