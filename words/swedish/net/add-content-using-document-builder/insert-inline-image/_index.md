I detta exempel visas hur man lägger in en inline bild i ett Word-dokument med hjälp av biblioteket Aspose.Words för .NET.`Dokumentbyggare`I denna guide finns en enkel implementering som hjälper dig att komma igång med bildhantering i Word-dokument.

## Installationsinstruktioner
1. Installa Aspose.Words för .NET:
   * Använd NuGet Package Manager i Visual Studio.
   * Rör följande kommandon i paketetjänstkonsolen:
     ```
     Install-Package Aspose.Words
     ```

2. Ställ in ditt projekt:
   * Skapa ett nytt .NET-projekt eller öppna ett befintligt.
   * Lägg till de nödvändiga instruktionerna längst upp i kodfilen:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
