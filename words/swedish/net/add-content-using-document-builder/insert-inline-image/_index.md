# Infoga inline- bild i Word- dokument med hjälp av Aspose.Words
Detta exempel visar hur man infogar en inline-avbildning i ett Word-dokument med hjälp av Aspose.Words-biblioteket för .NET. Genom att använda`Dokumentuppbyggnad`klass, kan du enkelt placera och infoga bilder på den aktuella markören plats i dokumentet. Denna guide ger en enkel implementation för att hjälpa dig att komma igång med bildhantering i Word- dokument.

## Monteringsanvisningar
1. Installera Aspose.Words för .NET:
   * Använd NuGet Package Manager i Visual Studio.
   * Kör följande kommando i pakethanterarens konsol:
     ```
     Install-Package Aspose.Words
     ```

2. Sätt upp ditt projekt:
   * Skapa ett nytt .NET-projekt eller öppna ett befintligt.
   * Lägg till de nödvändiga med hjälp av direktiv högst upp i din kodfil:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
