# Infoga inbäddad bild i Word-dokument med Aspose.Words
Det här exemplet visar hur man infogar en inbäddad bild i ett Word-dokument med hjälp av Aspose.Words-biblioteket för .NET. Genom att använda `DocumentBuilder` I klassen kan du enkelt placera och infoga bilder vid markörens aktuella position i dokumentet. Den här guiden ger en enkel implementering som hjälper dig att komma igång med bildhantering i Word-dokument.

## Installationsanvisningar
1. Installera Aspose.Words för .NET:
   - Använd NuGet-pakethanteraren i Visual Studio.
   - Kör följande kommando i pakethanterarkonsolen:
     ```
     Install-Package Aspose.Words
     ```

2. Konfigurera ditt projekt:
   - Skapa ett nytt .NET-projekt eller öppna ett befintligt.
   - Lägg till de nödvändiga using-direktiven högst upp i din kodfil:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
