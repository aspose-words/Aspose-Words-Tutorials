# Skapa Word-dokument med flytande bild

Detta exempel visar hur man skapar ett Word-dokument med hjälp av Aspose.Words för .NET och lägger in en bild med en flytande layout. Den tillhandahållna koden initialiserar ett Dokument och DocumentBuilder, lägger in en bild med angivna dimensioner och placering och slutligen sparar dokumentet. Denna guide är idealisk för utvecklare som vill förbättra sina dokumentgenereringskapaciteter med hjälp av Aspose.Words.

## Installationsinstruktioner
1. Installa Aspose.Words: Lägg till NuGet-paketet Aspose.Words till ditt .NET- projekt.
   ```
   Install-Package Aspose.Words
   ```

2. Lägga till nödvändiga namneutrymmen: Se till att kodfilen innehåller följande namneutrymmen:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
