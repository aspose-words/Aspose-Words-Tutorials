# Skapa Word-dokument med flytande bild

Det här exemplet visar hur man skapar ett Word-dokument med Aspose.Words för .NET och infogar en bild med en flytande layout. Den medföljande koden initierar ett dokument och en DocumentBuilder, infogar en bild med angivna dimensioner och positionering och sparar slutligen dokumentet. Den här guiden är idealisk för utvecklare som vill förbättra sina dokumentgenereringsmöjligheter med Aspose.Words.

## Installationsanvisningar
1. Installera Aspose.Words: Lägg till Aspose.Words NuGet-paketet i ditt .NET-projekt. Du kan göra detta genom att köra följande kommando i pakethanterarkonsolen:
   ```
   Install-Package Aspose.Words
   ```

2. Lägg till obligatoriska namnrymder: Se till att din kodfil innehåller följande namnrymder:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
