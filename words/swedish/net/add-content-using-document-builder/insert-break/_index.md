# Skapa ett flersidigt Word-dokument med Aspose.Words

Det här exemplet visar hur man skapar ett Word-dokument med Aspose.Words för .NET, vilket gör att du kan lägga till innehåll över flera sidor. Det visar hur man initierar ett nytt dokument, använder DocumentBuilder för att infoga text och sidbrytningar och slutligen sparar dokumentet som en .docx-fil. Perfekt för utvecklare som vill automatisera dokumentgenerering i sina .NET-applikationer.

## Installationsanvisningar
1. Installera Aspose.Words för .NET: Använd NuGet Package Manager för att installera Aspose.Words. Kör följande kommando i Package Manager-konsolen:
```
Install-Package Aspose.Words
```
