# Skapa ett flersidigt Word-dokument med Aspose.Words

Det här exemplet visar hur man skapar ett Word-dokument med hjälp av Aspose.Words för .NET, så att du kan lägga till innehåll över flera sidor. Det visar hur man initierar ett nytt dokument, använder DocumentBuilder för att infoga text och sidbrytningar, och slutligen spara dokumentet som en .docx- fil. Perfekt för utvecklare som vill automatisera dokumentgenerering i sina .NET-program.
