# Skapa ett flersidigt Word-dokument med Aspose.Words

Detta exempel visar hur man skapar ett Word-dokument med hjälp av Aspose.Words för .NET, vilket gör att du kan lägga till innehåll över flera sidor. Det visar hur man initialiserar ett nytt dokument, använder DocumentBuilder för att införa text och sidbrist och slutligen sparar dokumentet som en .docx-fil. Perfekt för utvecklare som vill automatisera dokumentgenerering i sina .NET- applikationer.

1. Installa Aspose.Words för .NET: Använd NuGet Package Manager för att installera Aspose.Words. Kör följande kommandon i Package Manager Console:
```
Install-Package Aspose.Words
```
