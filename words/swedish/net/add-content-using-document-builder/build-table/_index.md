# Skapa ett Word-dokument med tabell med hjälp av Aspose.Words
  
Detta exempel visar hur man skapar ett Word-dokument med en tabell med hjälp av biblioteket Aspose.Words för .NET. Koden initialiserar ett nytt dokument, bygger en tabell med två rader och formaterar cellerna med olika stilar och orienteringar. Du lär dig att manipulera cellinnehåll, justering och bygga ett strukturerat dokument programmatiskt.

## Installationsinstruktioner:  
1. Installa Aspose.Words för .NET:
   * Använd NuGet Package Manager:
     * Öppna ditt projekt i Visual Studio.
     * Rättklicka på projektet i Solution Explorer och välj "Manage NuGet Packages".
     * Sök efter Aspose.Words och klicka på "Installa".

   * Antingen använder du Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
