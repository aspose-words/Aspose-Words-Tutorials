# Skapa ett orddokument med tabell med hjälp av aspose.Words
  
Det här exemplet visar hur man skapar ett Word-dokument med en tabell med hjälp av Aspose.Words-biblioteket för .NET. Koden initierar ett nytt dokument, konstruerar ett bord med två rader och formaterar cellerna med olika stilar och orienteringar. Du lär dig att manipulera cellinnehåll, anpassning och bygga ett strukturerat dokument programmatiskt.

## Monteringsanvisningar:  
1. Installera Aspose.Words för .NET:
   * Använda NuGet pakethanterare:
     * Öppna ditt projekt i Visual Studio.
     * Högerklicka på projektet i Solution Explorer och välj "Hantera NuGet-paket".
     * Sök efter "Apose.Words" och klicka på "Install".

   * Eller använd pakethanterarens konsol:
     ```
     Install-Package Aspose.Words
     ```
