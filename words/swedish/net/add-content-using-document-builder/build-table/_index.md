# Skapa ett Word-dokument med tabell med hjälp av Aspose.Words
  
Det här exemplet visar hur man skapar ett Word-dokument med en tabell med hjälp av Aspose.Words-biblioteket för .NET. Koden initierar ett nytt dokument, konstruerar en tabell med två rader och formaterar cellerna med olika stilar och orienteringar. Du lär dig att manipulera cellinnehåll, justering och bygga ett strukturerat dokument programmatiskt.

## Installationsanvisningar:  
1. Installera Aspose.Words för .NET:
   - Använda NuGet-pakethanteraren:
     - Öppna ditt projekt i Visual Studio.
     - Högerklicka på projektet i Solution Explorer och välj "Hantera NuGet-paket".
     - Sök efter "Aspose.Words" och klicka på "Installera".

   - Eller använd pakethanterarkonsolen:
     ```
     Install-Package Aspose.Words
     ```
