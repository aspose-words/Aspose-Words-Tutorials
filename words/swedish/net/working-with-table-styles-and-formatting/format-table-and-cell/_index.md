# Skapa Word-dokument med anpassade bord och skugga

Detta exempel visar hur man använder Aspose.Words för .NET för att skapa ett Word-dokument som innehåller en tabell med anpassade gränsstilar och skuggningstilar för dess celler. Du lär dig att initialisera ett dokument, ställa in tabellgränser, tillämpa bakgrundsfärger på enskilda celler och spara dokumentet i DOCX-format.

## Installationsinstruktioner
1. Ladda ner Aspose.Words för .NET: Besök [Att skapa webbplats](https://www.aspose.com)och ladda ner den senaste versionen av Aspose.Words för .NET.
2. Lägg till referens: Lägg till en referens till DLL-filen Aspose.Words i ditt .NET-projekt.
3. Installa via NuGet: Alternativt kan du installera paketet med hjälp av NuGet Package Manager med följande kommando:
   ```
   Install-Package Aspose.Words
   ```
4. Ange Namespace: Lägg till följande med hjälp av direktiv i koden:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```

4. Ansökningstillstånd (Förval):  
   För att använda den fullständiga versionen, [ansöka om licens](https://purchase.aspose.com/temporary-license/)eller använda en [Gratis prövning](https://releases.aspose.com/words/net/)\- Jag är inte rädd.
