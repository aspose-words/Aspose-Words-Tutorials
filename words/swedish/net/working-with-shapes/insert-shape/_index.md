# Lägg in former i Word-dokument med hjälp av Aspose.Words
I denna guide visas hur man använder Aspose.Words för .NET för att införa och manipulera former, till exempel textfält, i Word-dokument. Du lär dig hur man positionerar former absolut eller relativt, anger rotationsvinklar och sparar dokumentet i ISO 29500:2008 Transitional DOCX-format med hjälp av `OoxmlSaveOptions`. Detta exempel förenklar arbetet med former i Word- dokument, vilket gör det möjligt för utvecklare att skapa visuellt tilltalande innehåll med lätthet.

## Installationsinstruktioner  
1. Ladda ner Aspose.Words för .NET:  
   Få den senaste versionen av Aspose.Words för .NET från [nedladdningssida](https://releases.aspose.com/words/net/)\- Jag är inte rädd.  

2. Installa via NuGet:  
   Öppna ditt projekt i Visual Studio och installera biblioteket med NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Lägg till Namespace Referens:  
   I koden ska du ange namnrummet för Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Aktivera licensen (Förval):
