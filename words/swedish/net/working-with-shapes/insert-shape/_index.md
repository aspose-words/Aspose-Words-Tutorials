# Infoga former i Word-dokument med hjälp av Aspose.Words
Den här guiden visar hur man använder Aspose.Words för .NET för att infoga och manipulera former, till exempel textrutor, i Word-dokument. Du lär dig hur du placerar former absolut eller relativt, ställer in rotationsvinklar och sparar dokumentet i ISO 29500:2008 Transitional DOCX-format med hjälp av... `OoxmlSaveOptions`Det här exemplet förenklar arbetet med former i Word-dokument, vilket gör det möjligt för utvecklare att enkelt skapa visuellt tilltalande innehåll.

## Installationsanvisningar  
1. Ladda ner Aspose.Words för .NET:  
   Hämta den senaste versionen av Aspose.Words för .NET från [nedladdningssida](https://releases.aspose.com/words/net/).  

2. Installera via NuGet:  
   Öppna ditt projekt i Visual Studio och installera biblioteket med hjälp av NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Lägg till namnrymdsreferens:  
   I din kod, inkludera namnrymden för Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Aktivera licensen (valfritt):  
   Om du har en giltig licens, använd den i ditt projekt för att undvika begränsningar. Du kan [köpa en licens](https://purchase.aspose.com/buy) eller få en [tillfällig licens](https://purchase.aspose.com/temporary-license/).
