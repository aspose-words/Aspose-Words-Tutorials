# Skapa Word-dokument med sidhuvud och sidfot med Aspose.Words

Det här exemplet visar hur man använder Aspose.Words för .NET för att skapa ett Word-dokument med ett primärt sidhuvud och en sidfot. Guiden vägleder dig genom att initiera ett nytt dokument, gå vidare till sidhuvud- och sidfotssektionerna, lägga till anpassad text och spara dokumentet i DOCX-format. Med sin kraftfulla DocumentBuilder-klass gör Aspose.Words hantering av Word-dokument sömlös och effektiv. Oavsett om du lägger till varumärkeselement eller sidnummer ger det här exemplet grunden för att anpassa Word-dokument programmatiskt.

## Installationsanvisningar  
1. Ladda ner Aspose.Words för .NET: Hämta den senaste versionen från [Aspose-utgåvorsida](https://releases.aspose.com/words/net/).  
2. Installera Aspose.Words:  
   - Använd NuGet-pakethanteraren:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   - Eller referera manuellt till DLL-filen från det nedladdade paketet.  
3. Ansök om licens (valfritt): Erhåll en [tillfällig licens](https://purchase.aspose.com/temporary-license/) eller en fullständig licens från [Aspose-köp](https://purchase.aspose.com/buy) och tillämpa det i ditt projekt:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
