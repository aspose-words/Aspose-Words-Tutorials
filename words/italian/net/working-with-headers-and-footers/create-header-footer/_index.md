# Crea documento Word con intestazione e piè di pagina utilizzando Aspese.Words

Questo esempio dimostra come usare Aspose.Words for .NET per creare un documento Word con un'intestazione primaria e un piè di pagina. La guida ti guida attraverso l'inizializzazione di un nuovo documento, passando alle sezioni intestazione e piè di pagina, aggiungendo testo personalizzato, e salvando il documento in formato DOCX. Con la sua potente classe DocumentBuilder, Aspose.Words rende la manipolazione di documenti Word senza problemi ed efficiente.

## Istruzioni per l'installazione  
1. Scarica Aspese.Parole per .NET: Ottieni l'ultima versione dalla[Apponga la pagina dei rilasci](https://releases.aspose.com/words/net/).  
2. Installa Aspose.Parole:  
   * Utilizzare NuGet Package Manager:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   * O riferimento manuale alla DLL dal pacchetto scaricato.  
3. Applica licenza (opzionale): Ottieni un[licenza temporanea](https://purchase.aspose.com/temporary-license/)o una licenza completa da[Acquisto aspettato](https://purchase.aspose.com/buy)e applicarlo nel tuo progetto:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
