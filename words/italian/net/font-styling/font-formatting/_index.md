## Istruzioni per l'installazione 
1. Prerequisiti: assicurati di avere .NET installato sul tuo computer.
2. Installa Aspose.Words: 
   - Utilizzare NuGet Package Manager in Visual Studio:
     ```
     Install-Package Aspose.Words
     ```
   - Oppure scarica la DLL direttamente da [Sito web di Aspose](https://www.aspose.com).
3. Applica licenza (facoltativo):
