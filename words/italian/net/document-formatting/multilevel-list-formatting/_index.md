Questo esempio dimostra come creare una lista multilivello in un documento Word usando Aspose.Words for .NET. ## Istruzioni per l'installazione
Per utilizzare Aspose.Words for .NET, seguire questi passaggi:
1. Scarica la libreria: Visita la[Parole per la pagina di download .NET](https://www.aspose.com/downloads/words/net).
2. Installare via NuGet:
   * Apri il tuo progetto Visual Studio.
   * Vai agli strumenti > NuGet Package Manager > Gestisci i pacchetti NuGet per la soluzione.
   * Cerca`Aspese.Parole`e fare clic su Installa.
3. Riferimento alla libreria nel tuo codice:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Applica licenza (opzionale):  
   Per utilizzare la versione completa,[applicare una licenza](https://purchase.aspose.com/temporary-license/)o usare una[prova gratuita](https://releases.aspose.com/words/net/).
