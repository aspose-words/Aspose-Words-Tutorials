# Inserire forme nei documenti Word utilizzando Aspose.Words
Questa guida illustra come utilizzare Aspose.Words per .NET per inserire e manipolare forme, come caselle di testo, nei documenti Word. Imparerai come posizionare le forme in modo assoluto o relativo, impostare gli angoli di rotazione e salvare il documento nel formato DOCX di transizione ISO 29500:2008 utilizzando `OoxmlSaveOptions`Questo esempio semplifica l'utilizzo delle forme nei documenti Word, consentendo agli sviluppatori di creare facilmente contenuti visivamente accattivanti.

## Istruzioni per l'installazione  
1. Scarica Aspose.Words per .NET:  
   Ottieni l'ultima versione di Aspose.Words per .NET da [pagina di download](https://releases.aspose.com/words/net/).  

2. Installa tramite NuGet:  
   Apri il tuo progetto in Visual Studio e installa la libreria utilizzando NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Aggiungi riferimento allo spazio dei nomi:  
   Nel codice, includi lo spazio dei nomi per Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
