## Istruzioni per l'installazione  

1. Scarica Aspese.Parole per .NET:  
   Ricevi l'ultima versione dal ufficiale[Parole per la pagina di download .NET](https://releases.aspose.com/words/net/).  

2. Installare via NuGet:  
   Aggiungi la libreria al tuo progetto usando NuGet Package Manager. Esegui il seguente comando nella Console Package Manager:  
   ```bash
   Install-Package Aspose.Words
   ```
