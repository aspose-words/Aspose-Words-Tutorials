# Crea un documento di Word con Indice
Questo esempio dimostra come usare Aspose.Words for .NET per creare un documento Word che include una Tabella dei Contenuti interattiva (TOC) generata utilizzando la classe DocumentBuilder. Il codice illustra come inserire le voci di vari livelli, formattarle e salvare il documento risultante come file .docx. Seguire i passaggi qui sotto per una facile implementazione.

## Istruzioni per l'installazione
1. Prerequisiti: Assicurarsi di avere .NET installato sulla vostra macchina.
2. Installa Aspose.Parole: 
   * Usa la console di gestione dei pacchetti NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Oppure, cerca "Aspose.Words" nella GUI di NuGet Package Manager.
