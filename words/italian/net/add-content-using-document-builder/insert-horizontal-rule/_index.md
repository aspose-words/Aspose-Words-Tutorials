Questo esempio dimostra l'uso di Aspose.Words for .NET per creare un documento Word e aggiungere una regola orizzontale utilizzando DocumentBuilder. Seguire i passaggi qui sotto per imparare a formattare i documenti in modo efficace.

## Istruzioni per l'installazione
1. Installa Aspose.Parole: Usa NuGet Package Manager in Visual Studio.
   * Apri il tuo progetto in Visual Studio.
   * Clicca con il tasto destro sul tuo progetto nell'esploratore di soluzioni.
   * Selezionare Gestisci pacchetti NuGet.
   * Cerca`Aspese.Parole`e installarlo.

2. Aggiungere la direttiva di utilizzo: Nel file di codice, includere la seguente direttiva per utilizzare le classi Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
