## Istruzioni per l'installazione
1. Installa Aspose.Words: usa NuGet Package Manager in Visual Studio.
   - Apri il progetto in Visual Studio.
   - Fare clic con il pulsante destro del mouse sul progetto in Esplora soluzioni.
   - Selezionare Gestisci pacchetti NuGet.
   - Cercare `Aspose.Words` e installarlo.

2. Aggiungi direttiva Using: nel tuo file di codice, includi la seguente direttiva per utilizzare le classi Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
