# Crea un documento Word con un'immagine mobile

Questo esempio illustra come creare un documento Word utilizzando Aspose.Words per .NET e inserire un'immagine con un layout flottante. Il codice fornito inizializza un Document e un DocumentBuilder, inserisce un'immagine con dimensioni e posizionamento specificati e infine salva il documento. Questa guida è ideale per gli sviluppatori che desiderano migliorare le proprie capacità di generazione di documenti utilizzando Aspose.Words.

## Istruzioni per l'installazione
1. Installa Aspose.Words: aggiungi il pacchetto NuGet Aspose.Words al tuo progetto .NET. Puoi farlo eseguendo il seguente comando nella console di Gestione Pacchetti:
   ```
   Install-Package Aspose.Words
   ```

2. Aggiungi spazi dei nomi obbligatori: assicurati che il tuo file di codice includa i seguenti spazi dei nomi:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
