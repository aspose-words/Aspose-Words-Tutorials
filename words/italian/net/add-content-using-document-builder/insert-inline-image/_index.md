# Inserisci un'immagine in linea nel documento Word utilizzando Aspose.Words
Questo esempio illustra come inserire un'immagine in linea in un documento Word utilizzando la libreria Aspose.Words per .NET. Utilizzando `DocumentBuilder` classe, puoi posizionare e inserire facilmente le immagini nella posizione corrente del cursore nel documento. Questa guida fornisce un'implementazione semplice per aiutarti a iniziare a gestire le immagini nei documenti Word.

## Istruzioni per l'installazione
1. Installa Aspose.Words per .NET:
   - Utilizzare NuGet Package Manager in Visual Studio.
   - Eseguire il seguente comando nella console di Package Manager:
     ```
     Install-Package Aspose.Words
     ```

2. Imposta il tuo progetto:
   - Crea un nuovo progetto .NET o aprine uno esistente.
   - Aggiungi le direttive using necessarie all'inizio del tuo file di codice:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
