# Crea un documento di Word con la tabella usando Aspose.Words
  
Questo esempio dimostra come creare un documento Word con una tabella usando la libreria Aspese.Words per .NET. Il codice inizializza un nuovo documento, costruisce una tabella con due righe e formatta le celle con vari stili e orientamenti. Imparerai a manipolare il contenuto delle celle, l'allineamento e a costruire un documento strutturato programmaticamente.

## Istruzioni per l'installazione:  
1. Installa Aspose.Parole per .NET:
   * Utilizzando NuGet Package Manager:
     * Apri il tuo progetto in Visual Studio.
     * Fare clic con il tasto destro sul progetto in Solution Explorer e selezionare "Gestisci pacchetti NuGet."
     * Cercare "Aspose.Words" e fare clic su "Installa."

   * Oppure, utilizzare la console gestore pacchetti:
     ```
     Install-Package Aspose.Words
     ```
