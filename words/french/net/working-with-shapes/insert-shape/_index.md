# Insérer des formes dans les documents Word en utilisant Aspose.Words
Ce guide montre comment utiliser Aspose.Words pour .NET pour insérer et manipuler des formes, telles que des boîtes de texte, dans des documents Word. Vous apprendrez à positionner des formes absolument ou relativement, à définir des angles de rotation et à enregistrer le document dans le format ISO 29500:2008 Transitional DOCX en utilisant `OoxmlSaveOptions`Cet exemple simplifie le travail avec les formes dans les documents Word, permettant aux développeurs de créer facilement un contenu visuellement attrayant.

## Instructions d'installation  
1. Téléchargez Aspose.Words pour .NET:  
   Obtenez la dernière version de Aspose.Words pour .NET depuis le [page de téléchargement](https://releases.aspose.com/words/net/)\- Je suis désolé .  

2. Installer par NuGet:  
   Ouvrez votre projet dans Visual Studio et installez la bibliothèque à l'aide du gestionnaire de paquets NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Ajouter une référence à l' espace de noms:  
   Dans votre code, inclure l'espace de noms pour Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
