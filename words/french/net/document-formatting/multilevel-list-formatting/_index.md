# Créer des listes multiniveaux en Word avec Aspose.Words
Cet exemple montre comment créer une liste multiniveaux dans un document Word en utilisant Aspose.Words pour .NET. Il initialise un nouveau document et une instance DocumentBuilder, applique un format de numérotation, crée des éléments principaux et des sous-éléments, et enregistre le document sous forme de fichier .docx. Suivez le guide étape par étape pour formater efficacement votre document avec des listes numérotées.

## Instructions d'installation
Pour utiliser Pose.Words pour .NET, suivez les étapes suivantes :
1. Télécharger la bibliothèque: Visitez la[Pose.Words pour .NET page de téléchargement](https://www.aspose.com/downloads/words/net)C'est vrai.
2. Installer via NuGet :
   * Ouvrez votre projet Visual Studio.
   * Naviguez vers les outils > NuGet Package Manager > Gérez les paquets NuGet pour la solution.
   * Rechercher`Pose.Words`et cliquez sur Installer.
3. Référencez la bibliothèque dans votre code :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
