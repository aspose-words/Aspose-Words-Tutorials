# Format Paragraphes Utilisation d'Aspos.Words
Ce guide fournit un exemple de la façon de formater les paragraphes dans un document Word en utilisant Aspose.Words pour .NET. Vous apprendrez à définir diverses options de formatage des paragraphes comme l'alignement, l'indentation et l'espace après les paragraphes. L'exemple illustre l'utilisation des classes Document et DocumentBuilder, permettant la création de documents bien structurés programmatiquement.

## Instructions d'installation
1. Télécharger Aspose.Words pour .NET: Visitez le[Poser un site Web](https://www.aspose.com/downloads/words/net)pour télécharger la dernière version d'Aspose.Words pour .NET.

2. Ajouter une référence: Dans votre projet .NET, faites un clic droit sur le projet dans Solution Explorer, sélectionnez Ajouter > Référence, et naviguez pour ajouter le fichier DLL Aspose.Words.

3. Utilisation de NuGet Package (optionnel) : Vous pouvez également installer Aspose.Words via NuGet Package Manager avec la commande suivante :
   ```
   Install-Package Aspose.Words
   ```

4. Importer l'espace de noms : Ajouter la déclaration suivante à votre fichier de codes :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

5. Appliquer une licence (facultative) :
