# Créer un document Word avec Aspose.Words
Cet exemple montre comment utiliser la bibliothèque Aspose.Words for .NET pour créer un document Word programmatiquement. En utilisant la classe DocumentBuilder, vous pouvez insérer des paragraphes style avec des paramètres de police personnalisés dans votre document. Ce guide vous guidera dans chaque étape, vous assurant de générer efficacement des documents Word pour vos applications.

## Instructions d'installation
1. Installez la bibliothèque Aspose.Words :
   * Vous pouvez installer Aspose.Words pour .NET via NuGet Package Manager dans Visual Studio:
     ```
     Install-Package Aspose.Words
     ```

2. Configurez votre projet :
   * Créez un nouveau projet .NET dans Visual Studio.
   * Ajoutez une référence à la bibliothèque Aspose.Words.

3. Ajouter le code de l'exemple :
   * Copiez l'exemple de code fourni dans votre projet.
   * S'assurer d'inclure les éléments nécessaires`utilisant`les directives en haut de la page:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     using Aspose.Words.Fields;
     using System.Drawing;
     ```
