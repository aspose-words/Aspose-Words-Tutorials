# Créer un document Word avec Aspose.Words
Cet exemple montre comment utiliser la bibliothèque Aspose.Words pour .NET pour créer un document Word par programmation. Grâce à la classe DocumentBuilder, vous pouvez insérer des paragraphes stylisés avec des polices personnalisées dans votre document. Ce guide vous guidera pas à pas pour vous permettre de générer efficacement des documents Word pour vos applications.

## Instructions d'installation
1. Installez la bibliothèque Aspose.Words :
   - Vous pouvez installer Aspose.Words pour .NET via le gestionnaire de packages NuGet dans Visual Studio :
     ```
     Install-Package Aspose.Words
     ```

2. Configurez votre projet :
   - Créez un nouveau projet .NET dans Visual Studio.
   - Ajoutez une référence à la bibliothèque Aspose.Words.

3. Ajoutez l'exemple de code :
   - Copiez l’exemple de code fourni dans votre projet.
   - Assurez-vous d'inclure les informations nécessaires `using` directives en haut :
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     using Aspose.Words.Fields;
     using System.Drawing;
     ```
