# Créer un document Word multipage avec Aspose.Words

Cet exemple montre comment créer un document Word avec Aspose.Words pour .NET, ce qui permet d'ajouter du contenu sur plusieurs pages. Il montre comment initialiser un nouveau document, utiliser DocumentBuilder pour insérer du texte et des sauts de page, et enfin enregistrer le document au format .docx. Idéal pour les développeurs souhaitant automatiser la génération de documents dans leurs applications .NET.

## Instructions d'installation
1. Installer Aspose.Words pour .NET : utilisez le gestionnaire de packages NuGet pour installer Aspose.Words. Exécutez la commande suivante dans la console du gestionnaire de packages :
```
Install-Package Aspose.Words
```
