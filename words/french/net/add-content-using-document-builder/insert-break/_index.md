# Créer un document Word multi-pages avec Aspose.Words

Cet exemple montre comment créer un document Word en utilisant Aspose.Words pour .NET, vous permettant d'ajouter du contenu sur plusieurs pages. Il montre comment initialiser un nouveau document, utiliser le DocumentBuilder pour insérer le texte et les pauses de page, et enfin enregistrer le document comme un fichier .docx. Parfait pour les développeurs qui cherchent à automatiser la génération de documents dans leurs applications .NET.

1. Installer Aspose.Words pour .NET : Utilisez NuGet Package Manager pour installer Aspose.Words. Exécutez la commande suivante dans la console de gestion de paquets :
```
Install-Package Aspose.Words
```
