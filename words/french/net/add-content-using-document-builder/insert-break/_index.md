# Créer un document Word à plusieurs pages avec Aspose.Words

Cet exemple montre comment créer un document Word à l'aide d' Aspose.Words pour .NET, vous permettant d'ajouter du contenu sur plusieurs pages. Il montre comment initialement un nouveau document, utiliser le DocumentBuilder pour insérer du texte et des pauses de page, et enfin enregistrer le document comme un fichier .docx. Parfait pour les développeurs qui souhaitent automatiser la génération de documents dans leurs applications .NET.

## Instructions d'installation
1. Installez Aspose.Words pour .NET: Utilisez NuGet Package Manager pour installer Aspose.Words. Exécutez la commande suivante dans la console de gestion de paquets:
```
Install-Package Aspose.Words
```
