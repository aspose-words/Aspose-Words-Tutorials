## Instructions d'installation
1. Installez Pose.Words: Utilisez NuGet Package Manager dans Visual Studio.
   * Ouvrez votre projet dans Visual Studio.
   * Faites un clic droit sur votre projet dans l'Explorateur de solutions.
   * Sélectionnez Gérer les paquets NuGet.
   * Rechercher`Pose.Words`et l'installer.

2. Ajouter la directive d'utilisation: Dans votre fichier de code, inclure la directive suivante pour utiliser Aspose.Words classes:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
