## Instructions d'installation
1. Installez Aspose.Words: Utilisez le gestionnaire de paquets NuGet dans Visual Studio.
   * Ouvrez votre projet dans Visual Studio.
   * Cliquez à droite sur votre projet dans l'explorateur de solutions.
   * Sélectionnez Gérer les paquets NuGet.
   * Recherche de `"Aspose. Les mots"`et l'installer.

2. Ajouter une directive d'utilisation: Dans votre fichier de code, inclure la directive suivante pour utiliser les classes Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
