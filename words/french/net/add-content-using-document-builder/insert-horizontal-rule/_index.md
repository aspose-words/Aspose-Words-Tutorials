## Instructions d'installation
1. Installez Aspose.Words : utilisez le gestionnaire de packages NuGet dans Visual Studio.
   - Ouvrez votre projet dans Visual Studio.
   - Cliquez avec le bouton droit sur votre projet dans l’Explorateur de solutions.
   - Sélectionnez Gérer les packages NuGet.
   - Rechercher `Aspose.Words` et installez-le.

2. Ajouter la directive Using : Dans votre fichier de code, incluez la directive suivante pour utiliser les classes Aspose.Words :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
