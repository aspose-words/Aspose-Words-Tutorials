# Créer un document Word avec un tableau à l'aide d'Aspose.Words
  
Cet exemple montre comment créer un document Word avec un tableau à l'aide de la bibliothèque Aspose.Words pour .NET. Le code initialise un nouveau document, construit un tableau de deux lignes et formate les cellules avec différents styles et orientations. Vous apprendrez à manipuler le contenu des cellules, l'alignement et à créer un document structuré par programmation.

## Instructions d'installation :  
1. Installer Aspose.Words pour .NET :
   - Utilisation du gestionnaire de packages NuGet :
     - Ouvrez votre projet dans Visual Studio.
     - Cliquez avec le bouton droit sur le projet dans l'Explorateur de solutions et sélectionnez « Gérer les packages NuGet ».
     - Recherchez « Aspose.Words » et cliquez sur « Installer ».

   - Ou utilisez la console du gestionnaire de packages :
     ```
     Install-Package Aspose.Words
     ```
