# Créer un document Word avec une table des matières
Cet exemple montre comment utiliser Aspose.Words for .NET pour créer un document Word qui comprend une table des matières interactive (TOC) générée à l'aide de la classe DocumentBuilder. Le code illustre comment insérer des en- têtes de différents niveaux, les formater et enregistrer le document résultant comme un fichier .docx. Suivez les étapes ci-dessous pour une mise en œuvre facile.

## Instructions d'installation
1. Prérequis : Assurez-vous que le .NET est installé sur votre machine.
2. Installez Pose.Words: 
   * Utilisez NuGet Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
   * Ou, recherchez "Aspose.Words" dans l'interface du gestionnaire de paquets NuGet.
