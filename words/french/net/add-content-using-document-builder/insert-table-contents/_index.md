# Créer un document Word avec un tableau de contenu
Cet exemple montre comment utiliser Aspose.Words pour .NET pour créer un document Word qui comprend une Table de contenu interactive (TOC) générée à l'aide de la classe DocumentBuilder. Le code illustre comment insérer des titres de différents niveaux, les formater et enregistrer le document résultant en tant que fichier .docx. Suivez les étapes ci-dessous pour une mise en œuvre facile.

## Instructions d'installation
1. Pré-requis: Assurez-vous d'avoir .NET installé sur votre machine.
2. Installez Aspose.Words: 
   * Utilisez la console de gestion de paquets NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Ou, recherchez Aspose.Words dans l'interface graphique du gestionnaire de paquets NuGet.
