# Créer un document Word avec table des matières
Cet exemple montre comment utiliser Aspose.Words pour .NET pour créer un document Word incluant une table des matières interactive (TOC) générée par la classe DocumentBuilder. Le code illustre comment insérer des titres de différents niveaux, les mettre en forme et enregistrer le document obtenu au format .docx. Suivez les étapes ci-dessous pour une implémentation facile.

## Instructions d'installation
1. Prérequis : assurez-vous que .NET est installé sur votre machine.
2. Installer Aspose.Mots : 
   - Utiliser la console du gestionnaire de packages NuGet :
     ```
     Install-Package Aspose.Words
     ```
   - Ou recherchez « Aspose.Words » dans l’interface graphique du gestionnaire de packages NuGet.
