# Créer un document Word avec une image flottante

Cet exemple montre comment créer un document Word en utilisant Aspose.Words pour .NET et insérer une image avec une mise en page flottante. Le code fourni initialise un Document et DocumentBuilder, insère une image avec des dimensions et un positionnement spécifiés, et finalement enregistre le document. Ce guide est idéal pour les développeurs cherchant à améliorer leurs capacités de génération de documents en utilisant Aspose.Words.

## Instructions d'installation
1. Installez Aspose.Words: Ajoutez le paquet Aspose.Words NuGet à votre projet .NET. Vous pouvez le faire en exécutant la commande suivante dans la console de gestion de paquets:
   ```
   Install-Package Aspose.Words
   ```

2. Ajouter les espaces de noms obligatoires : Assurez-vous que votre fichier de codes inclut les espaces de noms suivants :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
