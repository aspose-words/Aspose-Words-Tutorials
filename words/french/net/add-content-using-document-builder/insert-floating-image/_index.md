# Créer un document Word avec une image flottante

Cet exemple montre comment créer un document Word avec Aspose.Words pour .NET et insérer une image avec une mise en page flottante. Le code fourni initialise un Document et un DocumentBuilder, insère une image aux dimensions et au positionnement spécifiés, puis enregistre le document. Ce guide est idéal pour les développeurs souhaitant améliorer leurs capacités de génération de documents avec Aspose.Words.

## Instructions d'installation
1. Installer Aspose.Words : ajoutez le package NuGet Aspose.Words à votre projet .NET. Pour ce faire, exécutez la commande suivante dans la console du gestionnaire de packages :
   ```
   Install-Package Aspose.Words
   ```

2. Ajouter les espaces de noms requis : assurez-vous que votre fichier de code inclut les espaces de noms suivants :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
