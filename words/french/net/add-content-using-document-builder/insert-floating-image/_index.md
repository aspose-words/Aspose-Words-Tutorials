# Créer un document Word avec une image flottante

Cet exemple montre comment créer un document Word à l'aide d'Aspose.Words pour .NET et insérer une image avec une mise en page flottante. Le code fourni initie un Document et DocumentBuilder, insère une image avec des dimensions et un positionnement spécifiés, et enfin stocke le document. Ce guide est idéal pour les développeurs qui cherchent à améliorer leurs capacités de génération de documents à l'aide d'Aspose.Words.

## Instructions d'installation
1. Installez Aspose.Words: Ajoutez le paquet NuGet Aspose.Words à votre projet .NET. Vous pouvez le faire en exécutant la commande suivante dans la console de gestion de paquets:
   ```
   Install-Package Aspose.Words
   ```

2. Ajouter les espaces de nom requis: Assurez-vous que votre fichier de code comprend les espaces de nom suivants:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
