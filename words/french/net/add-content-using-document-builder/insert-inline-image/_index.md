# Insérer l'image en ligne dans le document Word en utilisant Aspose.Words
Cet exemple montre comment insérer une image en ligne dans un document Word à l'aide de la bibliothèque Aspose.Words pour .NET.`DocumentBuilder`classe, vous pouvez facilement positionner et insérer des images à l'emplacement actuel du curseur dans votre document. Ce guide fournit une implémentation simple pour vous aider à commencer à gérer l'image dans les documents Word.

## Instructions d'installation
1. Installez Pose.Words pour .NET:
   * Utilisez NuGet Package Manager dans Visual Studio.
   * Exécutez la commande suivante dans la console de gestion de paquets :
     ```
     Install-Package Aspose.Words
     ```

2. Configurez votre projet :
   * Créez un nouveau projet .NET ou ouvrez un projet existant.
   * Ajoutez les directives nécessaires en utilisant le haut de votre fichier de code:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
