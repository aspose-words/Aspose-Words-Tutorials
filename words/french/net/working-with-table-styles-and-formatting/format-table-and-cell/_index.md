# Créer un document Word avec des limites de table personnalisées et des ombres

Cet exemple montre comment utiliser Aspose.Words pour .NET pour créer un document Word qui comprend une table avec des styles de bord et d'ombrage personnalisés pour ses cellules. Vous apprendrez à initialiser un document, définir des limites de table, appliquer des couleurs d'arrière-plan aux cellules individuelles et enregistrer le document au format DOCX.

## Instructions d'installation
1. Téléchargez Aspose.Words pour .NET: Visitez le [Site web à viser](https://www.aspose.com)et télécharger la dernière version d'Aspose.Words pour .NET.
2. Ajouter une référence: Dans votre projet .NET, ajoutez une référence au fichier DLL Aspose.Words.
3. Installez via NuGet: Alternativement, vous pouvez installer le package à l'aide du gestionnaire de paquets NuGet avec la commande suivante:
   ```
   Install-Package Aspose.Words
   ```
4. Inclure le nom de domaine: Ajoutez le texte suivant à l'aide de la directive dans votre code:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```
