# Créer une table stylisée dans un document Word à l'aide de Aspose.Words

Dans ce tutoriel, nous vous guiderons dans le processus de création d'un document Word avec une table personnalisée à l'aide de Aspose.Words pour .NET. Vous apprendrez à initialiser un nouveau document, à insérer des lignes et des colonnes de table, et à appliquer un style de table personnalisé avec des paramètres de bord spécifiques et un rembourrage.
