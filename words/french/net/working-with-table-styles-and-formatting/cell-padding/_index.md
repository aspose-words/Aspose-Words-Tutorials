## Instructions d'installation  
Pour exécuter l'exemple et utiliser Aspose.Words pour .NET, suivez ces étapes:  

1. Télécharger Aspose.Words pour .NET  
   Obtenez la dernière version de Aspose.Words de la page [Page de sortie de l'aspose](https://releases.aspose.com/words/net/)\- Je suis désolé .  

2. Installez par le gestionnaire de paquets NuGet  
   Ouvrez votre projet .NET dans Visual Studio et exécutez la commande suivante dans la console NuGet Package Manager:  
   ```bash
   Install-Package Aspose.Words
   ```
