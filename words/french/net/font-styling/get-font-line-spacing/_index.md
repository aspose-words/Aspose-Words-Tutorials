# Obtenir l'espacement des lignes de police dans un document Word
Aspose.Words pour .NET simplifie la création et la manipulation de documents Word. Cet exemple montre comment initialiser un document Word, définir une police personnalisée et récupérer l'interligne d'une police spécifique. Que vous automatisiez la génération de documents ou analysiez des styles de texte, ce guide fournit des instructions claires et des extraits de code pratiques pour atteindre vos objectifs efficacement. Aspose.Words pour .NET est une bibliothèque puissante pour les développeurs travaillant avec des documents Word en C# ou VB.NET.

## Instructions d'installation  
Pour utiliser cet exemple, suivez ces étapes :  
1. Téléchargez et installez Aspose.Words pour .NET  
   - Visitez le [Page de téléchargement d'Aspose.Words pour .NET](https://releases.aspose.com/words/net/) et téléchargez la dernière version.  
   - Extrayez le package téléchargé et référencez le `Aspose.Words.dll` fichier dans votre projet.  

2. Installer via NuGet  
   - Ouvrez votre projet dans Visual Studio.  
   - Accédez à Outils > Gestionnaire de packages NuGet > Gérer les packages NuGet pour la solution.  
   - Rechercher `Aspose.Words` dans le gestionnaire de packages NuGet et installez le package.  

3. Appliquer une licence (facultatif mais recommandé)  
   - Obtenir une licence auprès du [Page d'achat d'Aspose](https://purchase.aspose.com/buy).  
   - Utiliser un [permis temporaire](https://purchase.aspose.com/temporary-license/) pour évaluation si nécessaire.  
   - Ajoutez le code suivant au début de votre application pour appliquer la licence :  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
