## Instructions d'installation  
Pour utiliser cet exemple, assurez-vous d'avoir Aspose.Words pour .NET installé. Suivez ces étapes :  

1. Télécharger Aspose.Words pour .NET:  
   Visite[Pose.Words pour .NET Télécharger](https://releases.aspose.com/words/net/)et télécharger la dernière version.  

2. Installer la bibliothèque :  
   * Ajoutez le paquet Aspose.Words NuGet à votre projet. Exécutez la commande suivante dans la console NuGet Package Manager :  
     ```sh
     Install-Package Aspose.Words
     ```  
