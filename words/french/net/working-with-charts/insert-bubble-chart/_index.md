## Instructions d'installation  

1. Télécharger Aspose.Words pour .NET  
   Obtenez la bibliothèque à partir du [Page de téléchargement d'Aspose.Words](https://releases.aspose.com/words/net/).  

2. Installer via le gestionnaire de packages NuGet  
   - Ouvrez votre projet Visual Studio.  
   - Accédez à Outils > Gestionnaire de packages NuGet > Gérer les packages NuGet pour la solution.  
   - Rechercher `Aspose.Words` dans l'onglet Parcourir.  
   - Sélectionnez le package et cliquez sur Installer.  

3. Installation via la console du gestionnaire de packages NuGet  
   Exécutez la commande suivante dans la console du gestionnaire de packages :  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Ajouter une licence  
   Pour bénéficier de toutes les fonctionnalités, appliquez une licence. Vous pouvez obtenir une licence temporaire auprès de [Page de licence temporaire Aspose](https://purchase.aspose.com/temporary-license/).
