Ajoutez facilement un graphique Bubble à votre document Word en utilisant Aspose.Words for .NET! Ce guide vous permet de créer un nouveau document, d'insérer un graphique Bubble, d'ajouter des séries de données avec des valeurs de X, Y et de taille, et de sauvegarder le document.

## Instructions d'installation  

1. Télécharger Aspose.Words pour .NET  
   Obtenez la bibliothèque de la[Pose.Words page de téléchargement](https://releases.aspose.com/words/net/)C'est vrai.  

2. Installer via NuGet Package Manager  
   * Ouvrez votre projet Visual Studio.  
   * Naviguez vers les outils > NuGet Package Manager > Gérez les paquets NuGet pour la solution.  
   * Rechercher`Pose.Words`dans l'onglet Parcourir.  
   * Sélectionnez le paquet et cliquez sur Installer.  

3. Installer via NuGet Package Manager Console  
   Exécutez la commande suivante dans la console de gestion de paquets :  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Ajouter une licence  
   Pour toutes les fonctionnalités, appliquez une licence. Vous pouvez obtenir une licence temporaire de la[Poser une page de licence temporaire](https://purchase.aspose.com/temporary-license/)C'est vrai.
