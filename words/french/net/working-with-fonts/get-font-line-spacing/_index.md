## Instructions d'installation  
Pour utiliser cet exemple, suivez les étapes suivantes :  
1. Télécharger et installer Aspose.Words pour .NET  
   * Visitez le[Pose.Words pour .NET page de téléchargement](https://releases.aspose.com/words/net/)et télécharger la dernière version.  
   * Extraire le paquet téléchargé et référencer le`Pose.Words.dll`fichier dans votre projet.  

2. Installer via NuGet  
   * Ouvrez votre projet dans Visual Studio.  
   * Allez dans Outils > Gestionnaire de paquets NuGet > Gérer les paquets NuGet pour la solution.  
   * Rechercher`Pose.Words`dans le gestionnaire de paquets NuGet et installer le paquet.  

3. Appliquer une licence (facultative mais recommandée)

