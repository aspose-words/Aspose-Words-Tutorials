# Créer des listes multiniveaux dans Word avec Aspose.Words
Cet exemple montre comment créer une liste multiniveau dans un document Word avec Aspose.Words pour .NET. Il initialise un nouveau document et une instance de DocumentBuilder, applique un format de numérotation, crée des éléments principaux et secondaires, et enregistre le document au format .docx. Suivez le guide étape par étape pour formater efficacement votre document avec des listes numérotées.

## Instructions d'installation
Pour utiliser Aspose.Words pour .NET, suivez ces étapes :
1. Téléchargez la bibliothèque : Visitez le [Page de téléchargement d'Aspose.Words pour .NET](https://www.aspose.com/downloads/words/net).
2. Installer via NuGet :
   - Ouvrez votre projet Visual Studio.
   - Accédez à Outils > Gestionnaire de packages NuGet > Gérer les packages NuGet pour la solution.
   - Rechercher `Aspose.Words` et cliquez sur Installer.
3. Faites référence à la bibliothèque dans votre code :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Appliquer la licence (facultatif) :  
   Pour utiliser la version complète, [appliquer une licence](https://purchase.aspose.com/temporary-license/) ou utiliser un [essai gratuit](https://releases.aspose.com/words/net/).
