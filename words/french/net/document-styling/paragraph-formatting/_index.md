# Formats des paragraphes en utilisant Aspose.Words
Ce guide fournit un exemple de la façon de formater des paragraphes dans un document Word en utilisant Aspose.Words pour .NET. Vous apprendrez à définir diverses options de formatage de paragraphes telles que l'alignement, l'indentation et l'espace après les paragraphes. L'exemple illustre l'utilisation des classes Document et DocumentBuilder, permettant la création de documents bien structurés de manière programmatique.

## Instructions d'installation
1. Téléchargez Aspose.Words pour .NET: Visitez le [Site web à viser](https://www.aspose.com/downloads/words/net)Pour télécharger la dernière version d'Aspose.Words pour .NET.

2. Ajouter une référence: Dans votre projet .NET, cliquez à droite sur le projet dans Solution Explorer, sélectionnez Ajouter > Références, et parcourez pour ajouter le fichier DLL Aspose.Words.

3. Utilisation du paquet NuGet (optionnel): Alternativement, vous pouvez installer Aspose.Words via le gestionnaire de paquet NuGet avec la commande suivante:
   ```
   Install-Package Aspose.Words
   ```

4. Importation de l' espace de noms: Ajoutez le texte suivant à votre fichier de code en utilisant l' énoncé:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
