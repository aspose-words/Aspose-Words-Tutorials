# Formater des paragraphes avec Aspose.Words
Ce guide présente un exemple de mise en forme de paragraphes dans un document Word avec Aspose.Words pour .NET. Vous apprendrez à définir différentes options de mise en forme, telles que l'alignement, le retrait et l'espacement après les paragraphes. Cet exemple illustre l'utilisation des classes Document et DocumentBuilder, permettant la création de documents bien structurés par programmation.

## Instructions d'installation
1. Téléchargez Aspose.Words pour .NET : Visitez le [Site Web d'Aspose](https://www.aspose.com/downloads/words/net) pour télécharger la dernière version d'Aspose.Words pour .NET.

2. Ajouter une référence : dans votre projet .NET, cliquez avec le bouton droit sur le projet dans l’Explorateur de solutions, sélectionnez Ajouter > Référence et naviguez pour ajouter le fichier DLL Aspose.Words.

3. Utilisation du package NuGet (facultatif) : Vous pouvez également installer Aspose.Words via le gestionnaire de packages NuGet avec la commande suivante :
   ```
   Install-Package Aspose.Words
   ```

4. Importer l'espace de noms : ajoutez l'instruction using suivante à votre fichier de code :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

5. Appliquer la licence (facultatif) :  
   Pour utiliser la version complète, [appliquer une licence](https://purchase.aspose.com/temporary-license/) ou utiliser un [essai gratuit](https://releases.aspose.com/words/net/).
