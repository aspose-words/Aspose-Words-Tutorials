## Réponses rapides
- **Quelle bibliothèque vous permet de créer des documents Word en Java ?** Aspose.Words for Java.

## Questions fréquentes

**Q : Comment créer programmétiquement un document Word en Java ?**  
A: Utilisez les classes `Document` et `DocumentBuilder` d’Aspose.Words for Java. Cela vous permet d’ajouter des paragraphes, des tableaux, des images, et plus encore sans Microsoft Office.

**Q : Puis‑je convertir un fichier DOCX en PDF en une seule ligne de code ?**  
A: Oui. Chargez le DOCX avec `new Document("input.docx")` et appelez `save("output.pdf", SaveFormat.PDF)`.

**Q : Quelle est la meilleure façon d’extraire le texte brut d’un fichier Word ?**  
A: Appelez `document.getText()` ou parcourez les nœuds `Paragraph` pour récupérer le texte brut.

**Q : Comment générer automatiquement une table des matières ?**  
A: Insérez un champ TOC avec `DocumentBuilder.insertTableOfContents()` puis mettez à jour les champs avec `document.updateFields()`.

**Q : Est‑il possible de fusionner plusieurs documents Word en un seul fichier ?**  
A: Absolument. Utilisez `targetDocument.appendDocument(sourceDocument, ImportFormatMode.KEEP_SOURCE_FORMATTING)` pour chaque document que vous souhaitez combiner.
