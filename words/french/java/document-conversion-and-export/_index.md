## Réponses rapides
- **Quelle est la façon la plus simple de convertir un document Word en PDF en Java ?**  
  Utilisez `Document.save("output.pdf", SaveFormat.PDF)` d'Aspose.Words.
- **Ai‑je besoin d’une licence pour une utilisation en production ?**  
  Oui – une licence valide d'Aspose.Words for Java supprime les filigranes d'évaluation.
- **Puis‑je convertir directement des fichiers DOCX en PDF ?**  
  Absolument, le même appel d'API fonctionne pour `.docx`, `.doc`, `.rtf` et d'autres formats Word.
- **Quelle version de Java est requise ?**  
  Java 8+ est entièrement pris en charge ; les versions plus récentes fonctionnent également.
- **La conversion par lots est‑elle possible ?**  
  Oui – parcourez les fichiers et appelez `save` pour chaque document.

## Questions fréquentes

**Q : Puis‑je convertir des fichiers Word protégés par mot de passe en PDF ?**  
R : Oui. Chargez le document avec le mot de passe approprié en utilisant `Document.load(path, loadOptions)` puis enregistrez‑le au format PDF.

**Q : Comment contrôler la qualité de sortie du PDF ?**  
R : Utilisez `PdfSaveOptions` pour définir la compression des images, l’incorporation des polices et le niveau de conformité avant d’appeler `save`.

**Q : Est‑il possible de convertir un lot de fichiers Word en une seule fois ?**  
R : Absolument. Parcourez votre liste de fichiers, chargez chaque document et appelez `save` avec le format PDF dans la boucle.

**Q : Quelles options de licence sont disponibles pour Aspose.Words for Java ?**  
R : Il existe des licences d’évaluation temporaires, des licences développeur et des licences entreprise. Choisissez celle qui correspond à la taille de votre projet.

**Q : La conversion prend‑elle en charge les langues de droite à gauche comme l’arabe ou l’hébreu ?**  
R : Oui, Aspose.Words prend pleinement en charge les langues RTL et préserve la mise en page lors de la conversion.
