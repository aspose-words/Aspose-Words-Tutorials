## Telepítési utasítások
1. Az Aspose.Words telepítése: Használja a NuGet csomagkezelőt a Visual Studio-ban.
   - Nyisd meg a projektedet a Visual Studioban.
   - Kattintson jobb gombbal a projektjére a Megoldáskezelőben.
   - Válassza a NuGet-csomagok kezelése lehetőséget.
   - Keresés `Aspose.Words` és telepítse.

2. Using Directive hozzáadása: A kódfájlodban add meg a következő direktívát az Aspose.Words osztályok használatához:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
