# Töltsön egy Word dokumentumot vízszintes szabálylal
Ez a példa bemutatja a .NET-hez használt Aspose.Words használatát a Word dokumentum létrehozásához és egy vízszintes szabály hozzáadásához a DocumentBuilder segítségével. Kövesse az alábbi lépéseket, hogy megtanulja, hogyan formátálhatja dokumentumait hatékonyan.

## A telepítési utasítások
1. telepítse az Aspose.Words: Használja a NuGet csomagkezelőt a Visual Studio- ban.
   * Nyissa ki a projektét a Visual Studio-ban.
   * Jobb kattintás a megoldás-kutató projektje.
   * Válassza Kihasználó csomagok kezelését.
   * Keresd a`Aspose.Words`és telepítem.

2. Adjon hozzá a Használat-irányelv: A kódfájljaba adja be a következő irányelvet az Aspose.Words osztályok használatára:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
