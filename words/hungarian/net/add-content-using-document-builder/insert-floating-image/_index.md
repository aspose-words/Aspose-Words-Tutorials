# Word dokumentum létrehozása lebegő képpel

Ez a példa bemutatja, hogyan hozhat létre Word-dokumentumot az Aspose.Words for .NET használatával, és hogyan szúrhat be egy képet lebegő elrendezéssel. A megadott kód inicializálja a Document és a DocumentBuilder elemeket, beszúr egy képet a megadott méretekkel és pozícióval, majd végül menti a dokumentumot. Ez az útmutató ideális azoknak a fejlesztőknek, akik az Aspose.Words segítségével szeretnék bővíteni dokumentumgenerálási képességeiket.

## Telepítési utasítások
1. Aspose.Words telepítése: Adja hozzá az Aspose.Words NuGet csomagot a .NET projekthez. Ezt a következő parancs futtatásával teheti meg a Package Manager Console-ban:
   ```
   Install-Package Aspose.Words
   ```

2. Kötelező névterek hozzáadása: Győződjön meg arról, hogy a kódfájl tartalmazza a következő névtereket:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
