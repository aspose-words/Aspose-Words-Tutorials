# Stel een nadruk op een document met het woord Aspose.Words

Het plaatsen van een nadruk in Word-documenten is een uitstekende manier om specifieke tekst te benadrukken, zodat deze in uw inhoud opvalt. Met Aspose.Words voor .NET kunt u gemakkelijk nadrukken, zoals een onder-solide cirkel, toepassen met behulp van een paar regels code.`DocumentBuilder`Word- documenten kunnen worden gemanipuleerd, tekst met nadrukken worden geformateerd en het resultaat in DOCX-formaat worden opgeslagen.

## Installatie-instructies  
Om het gegeven code voorbeeld uit te voeren en Aspose.Words te gebruiken voor .NET, volg deze stappen:  

1. Download Aspose.Words voor .NET:  
   * Haal de bibliotheek van de [Verwijderingen van de stof](https://releases.aspose.com/words/net/)\- Een blad.  

2. Installeer de bibliotheek:  
   * Installeer het via NuGet Package Manager in Visual Studio:  
     * Open Visual Studio.  
     * Ga naar Tools > NuGet Pakket Manager > NuGet Pakketten beheren voor Solution.  
     * Zoek naar `Aspose.Words`Klik op Installeer.  

   * Als alternatief gebruik je de NuGet Package Manager Console:  
     ```shell
     Install-Package Aspose.Words
     ```  

3. Stel een tijdelijke licentie op (optioneel):  
   * Voor onbeperkt gebruik, ontvang een [gratis tijdelijke vergunning](https://purchase.aspose.com/temporary-license/)\- Ik heb het niet gedaan .  
   * De licentie in uw project toepassen:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
