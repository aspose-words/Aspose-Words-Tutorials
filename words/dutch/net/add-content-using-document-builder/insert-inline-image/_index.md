# Inline-afbeelding invoegen in Word-document met Aspose.Words
Dit voorbeeld laat zien hoe u een inline-afbeelding in een Word-document kunt invoegen met behulp van de Aspose.Words-bibliotheek voor .NET. Door gebruik te maken van de `DocumentBuilder` Met de klasse kunt u eenvoudig afbeeldingen positioneren en invoegen op de huidige cursorpositie in uw document. Deze handleiding biedt een eenvoudige implementatie om u te helpen aan de slag te gaan met het verwerken van afbeeldingen in Word-documenten.

## Installatie-instructies
1. Installeer Aspose.Words voor .NET:
   - Gebruik NuGet Package Manager in Visual Studio.
   - Voer de volgende opdracht uit in de Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```

2. Stel uw project in:
   - Maak een nieuw .NET-project of open een bestaand project.
   - Voeg de benodigde using-richtlijnen bovenaan uw codebestand toe:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
