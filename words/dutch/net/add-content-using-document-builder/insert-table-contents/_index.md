# Een Word-document aanmaken met inhoudstabel
Dit voorbeeld laat zien hoe je Aspose.Words voor .NET gebruikt om een Word- document aan te maken dat een interactieve Inhoudsopgave (TOC) bevat die is gegenereerd met behulp van de DocumentBuilder-klasse. De code illustreert hoe je rubrieken van verschillende niveaus invoegt, ze formatteert en het resulterende document opslaat als een .docx-bestand.
