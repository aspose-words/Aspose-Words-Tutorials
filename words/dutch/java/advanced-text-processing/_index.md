## Veelgestelde vragen

**Q: Kan ik het paginanummerformaat (bijv. Romeins naar Arabisch) programmatisch wijzigen?**  
A: Ja, je kunt de `PageNumberStyle`‑eigenschap op het `PageNumber`‑veld instellen op elk ondersteund formaat.

**Q: Hoe sla ik paginanummers over op specifieke pagina's, zoals titelpagina's?**  
A: Voeg een `PageNumber`‑veld toe met de `\* MERGEFORMAT`‑switch en stel het `Suppress`‑attribuut in op de header/footer voor die sectie.

**Q: Is het mogelijk om paginanummers automatisch voor elk hoofdstuk te resetten?**  
A: Absoluut. Gebruik de `Section`‑klasse om een nieuwe sectie voor elk hoofdstuk te maken en schakel `PageSetup.RestartPageNumbering` in.

**Q: Handelt de API grote documenten efficiënt af?**  
A: Aspose.Words verwerkt documenten in een streaming‑modus, waardoor het geheugenverbruik wordt geminimaliseerd, zelfs voor bestanden met duizenden pagina's.

**Q: Welke licentieopties zijn beschikbaar voor productiegebruik?**
