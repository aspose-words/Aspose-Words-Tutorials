## Instrukcje instalacji  

Aby użyć przykładu, musisz zainstalować Aspose.Words dla .NET w środowisku rozwoju.  

1. Pobierz Aspose.Words na .NET  
   / Zbierz najnowszą wersję z [Wystarczy pobrać](https://releases.aspose.com/words/net/)Strona.  

2. Instalacja za pośrednictwem NuGet  
   Otwórz projekt w Visual Studio i zainstaluj bibliotekę za pomocą Menedżera pakietów NuGet:  
   * W konsoli zarządzającej pakietami wpisz:  
     ```bash  
     Install-Package Aspose.Words  
     ```
