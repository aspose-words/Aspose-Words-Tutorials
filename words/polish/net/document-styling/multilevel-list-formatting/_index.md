# Tworzenie list wielopoziomowych w programie Word za pomocą Aspose.Words
Ten przykład pokazuje, jak utworzyć listę wielopoziomową w dokumencie Word przy użyciu Aspose.Words dla .NET. Inicjuje nowy dokument i wystąpienie DocumentBuilder, stosuje format numeracji, tworzy elementy główne i podrzędne oraz zapisuje dokument jako plik .docx. Postępuj zgodnie z przewodnikiem krok po kroku, aby wydajnie sformatować dokument za pomocą list numerowanych.

## Instrukcje instalacji
Aby użyć Aspose.Words dla .NET, wykonaj następujące kroki:
1. Pobierz bibliotekę: Odwiedź [Strona pobierania Aspose.Words dla .NET](https://www.aspose.com/downloads/words/net).
2. Instalacja za pomocą NuGet:
   - Otwórz projekt programu Visual Studio.
   - Przejdź do Narzędzia > Menedżer pakietów NuGet > Zarządzaj pakietami NuGet dla rozwiązania.
   - Szukaj `Aspose.Words` i kliknij Zainstaluj.
3. Odwołaj się do biblioteki w swoim kodzie:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Zastosuj licencję (opcjonalnie):  
   Aby korzystać z pełnej wersji, [zastosować licencję](https://purchase.aspose.com/temporary-license/) lub użyj [bezpłatny okres próbny](https://releases.aspose.com/words/net/).
