## Instrukcje instalacji  
1. Pobierz Aspose.Words na .NET:  
   Zdobądź najnowszą wersję Aspose.Words dla .NET z [strona pobrania](https://releases.aspose.com/words/net/)\- Nie .  

2. Instalacja za pośrednictwem NuGet:  
   Otwórz projekt w Visual Studio i zainstaluj bibliotekę za pomocą Menedżera pakietów NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Dodaj odniesienie do przestrzeni nazw:  
   W swoim kodzie wpisz przestrzenie nazwisk Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
