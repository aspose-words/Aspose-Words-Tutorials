# Wstawianie kształtów do dokumentów Word za pomocą Aspose.Words
W tym przewodniku pokazano, jak używać Aspose.Words dla .NET do wstawiania i manipulowania kształtami, takimi jak pola tekstowe, w dokumentach Word. Dowiesz się, jak pozycjonować kształty bezwzględnie lub względnie, ustawiać kąty obrotu i zapisywać dokument w formacie ISO 29500:2008 Transitional DOCX przy użyciu `OoxmlSaveOptions`. Ten przykład upraszcza pracę z kształtami w dokumentach Word, umożliwiając deweloperom łatwe tworzenie wizualnie atrakcyjnej zawartości.

## Instrukcje instalacji  
1. Pobierz Aspose.Words dla .NET:  
   Pobierz najnowszą wersję Aspose.Words dla .NET ze strony [strona do pobrania](https://releases.aspose.com/words/net/).  

2. Instalacja za pomocą NuGet:  
   Otwórz projekt w programie Visual Studio i zainstaluj bibliotekę za pomocą Menedżera pakietów NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Dodaj odniesienie do przestrzeni nazw:  
   W swoim kodzie uwzględnij przestrzeń nazw dla Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
