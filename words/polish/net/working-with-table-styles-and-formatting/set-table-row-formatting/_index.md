## Instrukcje instalacji
1. Zainstaluj Aspose.Words dla .NET: użyj pakietu NuGet Manager do zainstalowania pakietu Aspose.Words. Możesz uruchomić następującą komendę w konsoli pakietu Manager:
   ```
   Install-Package Aspose.Words
   ```

2. Create a New Project: Open Visual Studio and create a new C# project (Console Application or any type you prefer).

3. Dodaj odniesienie: Upewnij się, że biblioteka Aspose.Words jest odniesieniem do projektu.
