## Instrukcje instalacji  
W projekcie należy wykonać następujące kroki:  

1. Pobierz Aspose.Words:  
   Odwiedź [Aspose.Words dla strony pobierania .NET](https://releases.aspose.com/words/net/)i pobierz najnowszą wersję biblioteki.  

2. Instalacja za pośrednictwem NuGet:  
   Otwórz projekt .NET w Visual Studio, przejdź do Menedżera pakietu NuGet (Narzędzia > Menedżer pakietu NuGet > Zarządzaj pakietami NuGet dla rozwiązania), wyszukaj Aspose.Words i instaluj pakiet.  

   W konsole do zarządzania pakietami wyślij następującą instrukcję:  
   ```shell
   Install-Package Aspose.Words
   ```
