## Instrukcje instalacji  

### Warunki wstępne  
1. Upewnij się, że masz .NET Framework 4.7.2 lub nowsza zainstalowana na systemie.  
2. Załatw i zainstaluj Visual Studio (zalecane jest 2019 lub późniejsze).  

### Kroki instalacji  
1. Pobierz Aspose.Words dla biblioteki .NET z oficjalnej strony internetowej: [Aspose.Words dla .NET Download](https://releases.aspose.com/words/net/)\- Nie .  
2. Instaluj bibliotekę za pośrednictwem Menedżera pakietów NuGet:  
   * Otwórz swój projekt w Visual Studio.  
   * Przejdź do Narzędzi > Menedżer pakietu NuGet > Zarządzaj pakietami NuGet dla rozwiązania.  
   * Szukaj `"Aspose.Word"`i kliknij zainstalowanie.  
3. Dodaj w swoim projekcie odniesienie do biblioteki:  
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```
