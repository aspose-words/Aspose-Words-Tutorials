## Instrukcje instalacji  
Aby użyć tego przykładu, upewnij się, że masz zainstalowany Aspose.Words dla .NET. Wykonaj następujące kroki:  

1. Pobierz Aspose.Words dla .NET:  
   Odwiedzać [Aspose.Words dla .NET Pobierz](https://releases.aspose.com/words/net/) i pobierz najnowszą wersję.  

2. Zainstaluj bibliotekę:  
   - Dodaj pakiet Aspose.Words NuGet do swojego projektu. Uruchom następujące polecenie w konsoli NuGet Package Manager:  
     ```sh
     Install-Package Aspose.Words
     ```  
   - Alternatywnie, dodaj `.dll` pliki do odniesień do projektu ręcznie.  

3. Uzyskaj licencję:  
   - Uzyskaj licencję od [Zakup Aspose](https://purchase.aspose.com/buy) lub poproś o bezpłatną wersję próbną [Bezpłatna wersja próbna Aspose](https://releases.aspose.com/).  

4. Zastosuj licencję:  
   Dodaj następujący kod do swojego projektu, aby aktywować licencję:  
   ```csharp
   License license = new License();
   license.SetLicense("path-to-your-license-file.lic");
   ```
