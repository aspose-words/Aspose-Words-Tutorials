## Instrukcje instalacji  
Aby użyć tego przykładu, upewnij się, że masz zainstalowany Aspose.Words dla .NET.  

1. Pobierz Aspose.Words na .NET:  
   Odwiedź[Aspose.Words dla .NET Download](https://releases.aspose.com/words/net/)I pobierz najnowszą wersję.  

2. Zainstalowanie biblioteki:  
   * Dodaj pakiet NuGet Aspose.Words do projektu. Wydaj następującą instrukcję w konsoli NuGet Package Manager:  
     ```sh
     Install-Package Aspose.Words
     ```  
   * Alternatywnie dodać `.dll`w ręcznie wpisane do referencji projektu.
