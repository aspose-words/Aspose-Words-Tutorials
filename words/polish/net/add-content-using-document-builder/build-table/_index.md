# Utwórz dokument Word z tabelą za pomocą Aspose.Words
  
Ten przykład pokazuje, jak utworzyć dokument Word z tabelą przy użyciu biblioteki Aspose.Words dla .NET. Kod inicjuje nowy dokument, konstruuje tabelę z dwoma wierszami i formatuje komórki za pomocą różnych stylów i orientacji. Nauczysz się manipulować zawartością komórek, wyrównaniem i programowo budować ustrukturyzowany dokument.

## Instrukcje instalacji:  
1. Zainstaluj Aspose.Words dla .NET:
   - Korzystanie z Menedżera pakietów NuGet:
     - Otwórz projekt w programie Visual Studio.
     - Kliknij prawym przyciskiem myszy projekt w Eksploratorze rozwiązań i wybierz opcję „Zarządzaj pakietami NuGet”.
     - Wyszukaj „Aspose.Words” i kliknij „Zainstaluj”.

   - Można też skorzystać z Konsoli Menedżera Pakietów:
     ```
     Install-Package Aspose.Words
     ```
