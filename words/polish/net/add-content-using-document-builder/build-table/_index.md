## Instrukcje instalacji:  
1. Zainstalowanie Aspose.Words dla .NET:
   * Korzystanie z Menedżera pakietu NuGet:
     * Otwórz swój projekt w Visual Studio.
     * Kliknij prawym kliknięciem na projekt w programie Solution Explorer i wybierz "Zarządzaj pakietami NuGet".
     * Poszukaj Aspose.Words i kliknij "Instaluj".

   * Albo użyj konsoli do zarządzania pakietami:
     ```
     Install-Package Aspose.Words
     ```
