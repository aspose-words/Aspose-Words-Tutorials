## Instrukcje instalacji
1. Zainstaluj Aspose.Words: Użyj Menedżera pakietów NuGet w programie Visual Studio.
   - Otwórz projekt w programie Visual Studio.
   - Kliknij prawym przyciskiem myszy swój projekt w Eksploratorze rozwiązań.
   - Wybierz opcję Zarządzaj pakietami NuGet.
   - Szukaj `Aspose.Words` i zainstaluj.

2. Dodaj dyrektywę Using: W pliku kodu dodaj następującą dyrektywę, aby użyć klas Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
