## Instrukcje instalacji
1. Wymagania wstępne: Upewnij się, że na Twoim komputerze jest zainstalowany .NET.
2. Zainstaluj Aspose.Words: 
   - Użyj konsoli Menedżera pakietów NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - Można też wyszukać „Aspose.Words” w interfejsie graficznym Menedżera pakietów NuGet.
