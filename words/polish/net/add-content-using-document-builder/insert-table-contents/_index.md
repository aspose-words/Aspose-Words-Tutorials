# Utwórz dokument Word z tabelą treści
Ten przykład pokazuje, jak użyć Aspose.Words dla .NET do tworzenia dokumentu Word zawierającego interaktywną tablicę treści (TOC) generowaną za pomocą klasy DocumentBuilder. Kod ilustruje, jak wstawić nagłówki różnych poziomów, formatować je i zapisać otrzymany dokument jako plik .docx.

## Instrukcje instalacji
1. Warunki wstępne: Upewnij się, że masz zainstalowaną .NET na komputerze.
2. Zainstalować Aspose.Words: 
   * Użyj konsoli do zarządzania pakietami NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Albo wyszukaj Aspose.Words w interfejsie interfejsu NuGet Package Manager.
