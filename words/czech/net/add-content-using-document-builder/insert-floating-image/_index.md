# Vytvořte dokument Word s plovoucím obrázkem

Tento příklad ukazuje, jak vytvořit dokument Word pomocí Aspose.Words pro .NET a vložit obrázek s plovoucím rozvržením. Poskytnutý kód inicializuje Document a DocumentBuilder, vloží obrázek se zadanými rozměry a umístěním a nakonec uloží dokument. Tato příručka je ideální pro vývojáře, kteří chtějí vylepšit své možnosti generování dokumentů pomocí Aspose.Words.

## Pokyny k instalaci
1. Instalace Aspose.Words: Přidejte balíček NuGet Aspose.Words do svého projektu .NET. To provedete spuštěním následujícího příkazu v konzoli Správce balíčků:
   ```
   Install-Package Aspose.Words
   ```

2. Přidání požadovaných jmenných prostorů: Ujistěte se, že váš soubor s kódem obsahuje následující jmenné prostory:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
