## Návod k instalaci
1. Nainstalujte Aspose.Words: Použijte NuGet Package Manager ve Visual Studio.
   * Otevřít projekt ve Visual Studio.
   * Klikněte pravým tlačítkem myši na Váš projekt v Solution Explorer.
   * Vyberte Spravovat NuGet balíčky.
   * Hledat`Aspose. Words`a nainstalovat ho.

2. Přidat pomocí směrnice: Ve vašem kódovém souboru uveďte následující směrnici pro použití tříd Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
