# Vytvořte vícestránkový dokument Wordu pomocí Aspose.Words

Tento příklad ukazuje, jak vytvořit dokument Word pomocí Aspose.Words pro .NET, což vám umožní přidávat obsah na více stránek. Ukazuje, jak inicializovat nový dokument, použít DocumentBuilder k vložení textu a zalomení stránek a nakonec uložit dokument jako soubor .docx. Ideální pro vývojáře, kteří chtějí automatizovat generování dokumentů ve svých .NET aplikacích.

## Pokyny k instalaci
1. Instalace Aspose.Words pro .NET: K instalaci Aspose.Words použijte Správce balíčků NuGet. Spusťte následující příkaz v konzoli Správce balíčků:
```
Install-Package Aspose.Words
```
