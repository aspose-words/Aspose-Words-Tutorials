# Vytvořit multi-Page Word dokument s Aspose.Words

Tento příklad ukazuje, jak vytvořit dokument Word pomocí Aspose.Words pro .NET, což vám umožní přidat obsah přes více stránek. Ukazuje, jak inicializovat nový dokument, pomocí DocumentBuilder vložit text a stránky přestávky, a nakonec uložit dokument jako .docx soubor. Ideální pro vývojáře, kteří hledají automatizovat tvorbu dokumentů ve svých .NET aplikacích.

## Návod k instalaci
1. Nainstalujte Aspose.Words pro .NET: Pomocí NuGet Package Manager nainstalujte Aspose.Words. Spusťte následující příkaz v konzoli Správce balíčků:
```
Install-Package Aspose.Words
```
