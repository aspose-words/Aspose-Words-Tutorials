# Vložení obrázku do dokumentu Word pomocí Aspose.Words
Tento příklad ukazuje, jak vložit vložený obrázek do dokumentu Word pomocí knihovny Aspose.Words pro .NET. Využitím `DocumentBuilder` třídy můžete snadno umisťovat a vkládat obrázky na aktuální pozici kurzoru v dokumentu. Tato příručka nabízí přímočarou implementaci, která vám pomůže začít se zpracováním obrázků v dokumentech Wordu.

## Pokyny k instalaci
1. Instalace Aspose.Words pro .NET:
   - Použijte Správce balíčků NuGet ve Visual Studiu.
   - Spusťte v konzoli Správce balíčků následující příkaz:
     ```
     Install-Package Aspose.Words
     ```

2. Nastavení projektu:
   - Vytvořte nový .NET projekt nebo otevřete existující.
   - Přidejte potřebné direktivy using na začátek souboru s kódem:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
