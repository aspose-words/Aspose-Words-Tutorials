# Vytvořte dokument Word s obsahem
Tento příklad ukazuje, jak pomocí Aspose.Words pro .NET vytvořit dokument aplikace Word, který obsahuje interaktivní obsah (TOC) vygenerovaný pomocí třídy DocumentBuilder. Kód ilustruje, jak vkládat nadpisy různých úrovní, formátovat je a ukládat výsledný dokument jako soubor .docx. Pro snadnou implementaci postupujte podle níže uvedených kroků.

## Pokyny k instalaci
1. Předpoklady: Ujistěte se, že máte na svém počítači nainstalováno rozhraní .NET.
2. Nainstalujte Aspose.Slova: 
   - Použití konzole Správce balíčků NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - Nebo vyhledejte „Aspose.Words“ v grafickém rozhraní Správce balíčků NuGet.
