# Vytvořte víceúrovňové seznamy ve Wordu pomocí Aspose.Words
Tento příklad ukazuje, jak vytvořit víceúrovňový seznam v dokumentu Word pomocí Aspose.Words pro .NET. Inicializuje nový dokument a instanci DocumentBuilder, použije formát číslování, vytvoří hlavní a podpoložky a uloží dokument jako soubor .docx. Postupujte podle podrobného návodu, abyste efektivně naformátovali dokument s číslovanými seznamy.

## Pokyny k instalaci
Chcete-li použít Aspose.Words pro .NET, postupujte takto:
1. Stáhněte si knihovnu: Navštivte [Stránka pro stažení Aspose.Words pro .NET](https://www.aspose.com/downloads/words/net).
2. Instalace přes NuGet:
   - Otevřete svůj projekt ve Visual Studiu.
   - Přejděte do nabídky Nástroje > Správce balíčků NuGet > Spravovat balíčky NuGet pro řešení.
   - Hledat `Aspose.Words` a klikněte na tlačítko Instalovat.
3. Odkaz na knihovnu ve vašem kódu:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Použít licenci (volitelné):  
   Chcete-li používat plnou verzi, [požádat o licenci](https://purchase.aspose.com/temporary-license/) nebo použijte [bezplatná zkušební verze](https://releases.aspose.com/words/net/).
