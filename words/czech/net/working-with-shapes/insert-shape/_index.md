# Vkládání tvarů do dokumentů Wordu pomocí Aspose.Words
Tato příručka ukazuje, jak používat Aspose.Words pro .NET k vkládání a manipulaci s tvary, jako jsou textová pole, v dokumentech Word. Naučíte se, jak umístit tvary absolutně nebo relativně, nastavit úhly natočení a uložit dokument ve formátu ISO 29500:2008 Transitional DOCX pomocí... `OoxmlSaveOptions`Tento příklad zjednodušuje práci s tvary v dokumentech aplikace Word a umožňuje vývojářům snadno vytvářet vizuálně atraktivní obsah.

## Pokyny k instalaci  
1. Stáhněte si Aspose.Words pro .NET:  
   Získejte nejnovější verzi Aspose.Words pro .NET z [stránka ke stažení](https://releases.aspose.com/words/net/).  

2. Instalace přes NuGet:  
   Otevřete projekt ve Visual Studiu a nainstalujte knihovnu pomocí Správce balíčků NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Přidat odkaz na jmenný prostor:  
   Do kódu zahrňte jmenný prostor pro Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
