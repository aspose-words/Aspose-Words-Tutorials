# Vložit tvary do textu dokumenty pomocí Aspose.Words
Tato příručka ukazuje, jak používat Aspose.Slova pro .NET vložit a manipulovat tvary, jako jsou textové krabice, v dokumentech Word. Vy se naučíte, jak umístit tvary absolutně nebo relativně, nastavit rotační úhly, a uložit dokument ve formátu ISO 29500:2008 Přechodný DOCX pomocí`OoxmlSaveOptions`. Tento příklad zjednodušuje práci s tvary v dokumentech Word, což umožňuje vývojářům vytvářet vizuálně atraktivní obsah s lehkostí.

## Návod k instalaci  
1. Stáhnout Aspose.Words pro .NET:  
   Získejte nejnovější verzi Aspose.Words pro .NET z[stáhnout stránku@ info: whatsthis](https://releases.aspose.com/words/net/).  

2. Nainstalovat prostřednictvím NuGet:  
   Otevřít svůj projekt ve Visual Studio a nainstalovat knihovnu pomocí NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Přidat odkaz na jmenný prostor:  
   Do svého kódu, včetně jmenného prostoru pro Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Aktivovat licenci (volitelné):
