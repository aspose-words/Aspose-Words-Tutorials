# Vytvořit víceúrovňové seznamy ve slově s Aspose.Words
Tento příklad ukazuje, jak vytvořit víceúrovňový seznam v dokumentu Word pomocí aplikace Aspose.Words for .NET. Inicializuje nový dokument a instance DocumentBuilder, použije formát číslování, vytvoří hlavní a podpoložky a uloží dokument jako soubor .docx. Postupujte podle krok za krokem průvodce efektivně formátovat svůj dokument s očíslovanými seznamy.

## Návod k instalaci
Chcete-li použít Aspose.Words pro .NET, postupujte podle těchto kroků:
1. Stáhnout knihovnu: Navštivte[Aspose.Words pro .NET stáhnout stránku](https://www.aspose.com/downloads/words/net).
2. Nainstalovat prostřednictvím NuGet:
   * Otevřete si projekt Visual Studio.
   * Navigovat k nástrojům > NuGet Package Manager > Spravovat NuGet balíčky pro řešení.
   * Hledat`Aspose. Words`a klikněte na Install.
3. Reference knihovny ve vašem kódu:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. Použít licenci (volitelné):  
   Chcete-li použít plnou verzi,[požádat o licenci](https://purchase.aspose.com/temporary-license/)nebo[zkušební verze zdarma](https://releases.aspose.com/words/net/).
