# Vytvořit slovní dokument s hlavičkou a zápatí pomocí Aspose.Words

Tento příklad ukazuje, jak používat Aspose.Slova pro .NET vytvořit dokument Word s primární hlavičkou a zápatí. Průvodce vás provede inicializací nového dokumentu, pohybující se do hlavičky a zápatí sekce, přidání vlastního textu, a uložení dokumentu ve formátu DOCX. Se svým výkonným DocumentBuilder třídy, Aspose.Slova dělá Word manipulaci s dokumenty bezproblémové a efektivní. Ať už budete přidávat branding prvky nebo čísla stránek, tento příklad poskytuje základ pro přizpůsobení Word dokumentů programově.

## Návod k instalaci  
1. Stáhnout Aspose.Words pro .NET: Získejte nejnovější verzi z[Aspose releases page](https://releases.aspose.com/words/net/).  
2. Nainstalujte Aspose.Words:  
   * Použít NuGet správce balíčků:  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   * Nebo ručně označte DLL ze staženého balíku.  
3. Použít licenci (volitelné): Získat[dočasná licence](https://purchase.aspose.com/temporary-license/)nebo plná licence od[Navrhnout nákup](https://purchase.aspose.com/buy)a aplikovat jej ve vašem projektu:  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
