# Použití formátování řádků v dokumentech Word pomocí Aspose.Words

## Pokyny k instalaci  
Chcete-li ve svém projektu použít Aspose.Words pro .NET, postupujte takto:  

1. Stáhněte si Aspose.Words pro .NET  
   - Navštivte [Stránka pro stažení Aspose.Words pro .NET](https://releases.aspose.com/words/net/) a stáhněte si nejnovější verzi.

2. Instalace přes NuGet  
   - Otevřete svůj projekt ve Visual Studiu.  
   - Přejděte do nabídky Nástroje > Správce balíčků NuGet > Spravovat balíčky NuGet pro řešení.  
   - Hledat `Aspose.Words` a klikněte na tlačítko Instalovat.

3. Zahrnout jmenný prostor  
   - Přidejte do kódu následující řádek:  
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Tables;
     ```

4. Aktivace licence (volitelné)  
   - Abyste se vyhnuli omezením v hodnocení, požádejte o [dočasná licence](https://purchase.aspose.com/temporary-license/) nebo si zakoupit [plná licence](https://purchase.aspose.com/buy).  
   - Načtěte licenci do kódu:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
