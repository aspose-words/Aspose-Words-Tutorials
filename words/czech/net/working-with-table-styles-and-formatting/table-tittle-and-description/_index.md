## Návod k instalaci:

Chcete-li začít s Aspose.Words pro .NET a spustit tento příklad, postupujte podle těchto jednoduchých kroků:

1. Nainstalujte Aspose.Words pro .NET:
   Můžete snadno nainstalovat Aspose.Words pro .NET přes NuGet. Spusťte následující příkaz ve své konzoli Správce balíčků:
   ```
   Install-Package Aspose.Words
   ```

2. Použít licenci (volitelné):
