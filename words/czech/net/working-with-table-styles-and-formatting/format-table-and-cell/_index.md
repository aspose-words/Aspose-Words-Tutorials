# Vytvořit slovní dokument s vlastními hranicemi tabulky a stínění

Tento příklad ukazuje, jak využít Aspose.Slova pro .NET vytvořit dokument Word, který obsahuje tabulku s vlastní hranice a stínovací styly pro své buňky. Naučíte se inicializovat dokument, nastavit hranice tabulky, aplikovat barvy pozadí na jednotlivé buňky, a uložit dokument ve formátu DOCX.

## Návod k instalaci
1. Stáhnout Aspose.Words pro .NET: Navštivte[Aspose website](https://www.aspose.com)a stáhnout nejnovější verzi Aspose.Words pro .NET.
2. Přidat odkaz: Ve vašem projektu .NET přidejte odkaz na soubor Aspose.Words DLL.
3. Nainstalujte přes NuGet: Alternativně můžete balíček nainstalovat pomocí NuGet Package Manager s následujícím příkazem:
   ```
   Install-Package Aspose.Words
   ```
4. Zahrnout Namespace: Přidat následující pomocí směrnice do svého kódu:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```

4. Použít licenci (volitelné):  
   Chcete-li použít plnou verzi,[požádat o licenci](https://purchase.aspose.com/temporary-license/)nebo[zkušební verze zdarma](https://releases.aspose.com/words/net/).
