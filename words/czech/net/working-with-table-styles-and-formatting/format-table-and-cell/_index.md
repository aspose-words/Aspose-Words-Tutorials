# Vytvořte dokument Word s vlastními hranicemi tabulky a stínem

Tento příklad ukazuje, jak použít Aspose.Words pro .NET k vytvoření dokumentu Word, který obsahuje tabulku s přizpůsobenými styly hraní a stíhání pro své buňky. Naučíte se iniciovat dokument, nastavit hranice tabulky, aplikovat barvy pozadí na jednotlivé buňky a uložit dokument ve formátu DOCX.

## Příkazy na instalaci
1. Stáhněte si Aspose.Words pro .NET: Navštivte [Webové stránky](https://www.aspose.com)a stáhnout nejnovější verzi Aspose.Words pro .NET.
2. Přidejte odkaz: Ve svém projektu .NET přidejte odkaz na soubor DLL Aspose.Words.
3. Instalujte prostřednictvím programu NuGet: Alternativně můžete instalovat balíček pomocí programu NuGet Package Manager s následujícím příkazem:
   ```
   Install-Package Aspose.Words
   ```
4. Vkládejte v kód název: Přidejte do kódu následující směrnici:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```
