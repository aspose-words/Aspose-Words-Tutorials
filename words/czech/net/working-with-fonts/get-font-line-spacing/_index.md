# Získejte Font Line Spaceing v dokumentu Word
Aspose.Words for .NET simplifies Word document creation and manipulation. This example demonstrates how to initialize a Word document, set a custom font, and retrieve the line spacing of a specific font. Whether you're automating document generation or analyzing text styles, this guide provides clear instructions and practical code snippets to achieve your goals efficiently. Aspose.Words for .NET is a powerful library for developers working with Word documents in C# or VB.NET.

## Návod k instalaci  
Pro použití tohoto příkladu, postupujte podle těchto kroků:  
1. Stáhnout a nainstalovat Aspose.Words pro .NET  
   * Navštivte[Aspose.Words pro .NET stáhnout stránku](https://releases.aspose.com/words/net/)a stáhnout nejnovější verzi.  
   * Extrahovat stažený balíček a odkaz na`Aspose. Words.dll`Složka ve vašem projektu.  

2. Nainstalovat přes NuGet  
   * Otevřít projekt ve Visual Studio.  
   * Přejděte na nástroje > NuGet Package Manager > Spravovat NuGet balíčky pro řešení.  
   * Hledat`Aspose. Words`do NuGet Package Manager a nainstalovat balíček.  

3. Použít licenci (volitelné, ale doporučené)  
   * Získání licence od[Upravit nákupní stránku](https://purchase.aspose.com/buy).  
   * Použít[dočasná licence](https://purchase.aspose.com/temporary-license/)pro hodnocení v případě potřeby.  
   * Přidejte následující kód na začátku vaší aplikace k použití licence:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
