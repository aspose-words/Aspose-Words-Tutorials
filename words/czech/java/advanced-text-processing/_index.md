## Často kladené otázky

**Q: Mohu programově změnit formát číslování stránek (např. římské na arabské)?**  
A: Ano, můžete nastavit vlastnost `PageNumberStyle` na poli `PageNumber` na libovolný podporovaný formát.

**Q: Jak mohu přeskočit číslování stránek na konkrétních stránkách, například na titulní stránce?**  
A: Vložte pole `PageNumber` s přepínačem `\* MERGEFORMAT` a nastavte atribut `Suppress` v záhlaví/zápatí pro tuto sekci.

**Q: Je možné automaticky resetovat číslování stránek pro každou kapitolu?**  
A: Rozhodně. Použijte třídu `Section` k vytvoření nové sekce pro každou kapitolu a povolte `PageSetup.RestartPageNumbering`.

**Q: Zvládá API efektivně velké dokumenty?**  
A: Aspose.Words zpracovává dokumenty ve streamovacím režimu, minimalizuje využití paměti i u souborů s tisíci stránkami.
