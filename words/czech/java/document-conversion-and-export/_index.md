## Rychlé odpovědi
- **Jaký je nejjednodušší způsob, jak převést Word dokument do PDF v Javě?**  
  Použijte `Document.save("output.pdf", SaveFormat.PDF)` z Aspose.Words.
- **Potřebuji licenci pro produkční použití?**  
  Ano – platná licence Aspose.Words for Java odstraňuje vodotisky z hodnocení.
- **Mohu přímo převést soubory DOCX do PDF?**  
  Rozhodně, stejná API metoda funguje pro `.docx`, `.doc`, `.rtf` a další formáty Wordu.
- **Jaká verze Javy je vyžadována?**  
  Java 8+ je plně podporována; novější verze také fungují.
- **Je možná hromadná konverze?**  
  Ano – projděte soubory ve smyčce a pro každý dokument zavolejte `save`.

## Často kladené otázky

**Q: Mohu převést Word soubory chráněné heslem do PDF?**  
A: Ano. Načtěte dokument s příslušným heslem pomocí `Document.load(path, loadOptions)` a poté jej uložte jako PDF.

**Q: Jak mohu řídit kvalitu výstupu PDF?**  
A: Použijte `PdfSaveOptions` k nastavení komprese obrázků, vložení fontů a úrovně souladu před voláním `save`.

**Q: Je možné převést hromadu Word souborů najednou?**  
A: Rozhodně. Procházejte seznam souborů, načtěte každý dokument a v cyklu zavolejte `save` s formátem PDF.

**Q: Jaké licenční možnosti jsou k dispozici pro Aspose.Words for Java?**  
A: Existují dočasné evaluační licence, vývojářské licence a podnikové licence. Vyberte tu, která odpovídá velikosti vašeho projektu.

**Q: Podporuje konverze jazyky psané zprava doleva, jako arabštinu nebo hebrejštinu?**  
A: Ano, Aspose.Words plně podporuje RTL jazyky a během konverze zachovává rozvržení.
