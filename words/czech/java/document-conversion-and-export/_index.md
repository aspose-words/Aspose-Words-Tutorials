## Často kladené otázky

**Q: Mohu převést DOCX na PDF na Linux serveru?**  
A: Ano. Aspose.Words for Java je platformně nezávislý a funguje na jakémkoli OS s podporovaným JRE.

**Q: Potřebuji mít na serveru nainstalovaný Microsoft Office?**  
A: Ne. Konverze je prováděna výhradně knihovnou Aspose.Words.

**Q: Jak mohu během konverze zachovat vlastní písma?**  
A: Použijte `PdfSaveOptions.setEmbedFullFonts(true)`, aby se všechna použita písma vložila do výsledného PDF.

**Q: Je možné převést soubory DOCX chráněné heslem?**  
A: Rozhodně. Načtěte dokument pomocí `LoadOptions.setPassword("yourPassword")` před uložením jako PDF.

**Q: Jaký je doporučený způsob hromadného převodu mnoha dokumentů?**  
A: Vytvořte znovupoužitelnou metodu, která načte, převede a uvolní každý dokument, a spusťte ji v thread‑poolu pro paralelní zpracování.
