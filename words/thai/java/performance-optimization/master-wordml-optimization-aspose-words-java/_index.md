### คู่มือการใช้งาน

#### คุณสมบัติรูปแบบสวย
**ภาพรวม:**
คุณลักษณะ 'PrettyFormat' จะสร้าง WordML ด้วยโครงสร้าง XML ที่มีการเยื้องย่อหน้าอย่างสวยงามและอ่านง่าย ช่วยให้ดีบักและทำความเข้าใจได้ง่ายยิ่งขึ้น

##### ขั้นตอนที่ 1: สร้างเอกสาร
เริ่มต้นด้วยการสร้างใหม่ `Document` วัตถุและการใช้ `DocumentBuilder` เพื่อเพิ่มเนื้อหา:

```java
import com.aspose.words.Document;
import com.aspose.words.DocumentBuilder;

// เริ่มต้นเอกสาร
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.writeln("Hello world!");
```

##### ขั้นตอนที่ 2: กำหนดค่า WordML2003SaveOptions
ตั้งค่า `WordML2003SaveOptions` เพื่อเปิดใช้งานการจัดรูปแบบสวยงาม:

```java
import com.aspose.words.WordML2003SaveOptions;

// เริ่มต้นตัวเลือกการบันทึก
WordML2003SaveOptions options = new WordML2003SaveOptions();
options.setPrettyFormat(true); // เปิดใช้งานรูปแบบสวยงามสำหรับเอาท์พุต XML

doc.save("YOUR_DOCUMENT_DIRECTORY/WordML2003SaveOptions.PrettyFormat.xml", options);
```

**คำอธิบาย-**
- **`setPrettyFormat(true)`:** กำหนดค่าเอกสารที่จะบันทึกโดยมีการจัดรูปแบบที่อ่านได้ รวมถึงการเยื้องและแบ่งบรรทัด

#### คุณสมบัติการเพิ่มประสิทธิภาพหน่วยความจำ
**ภาพรวม:**
การจัดการหน่วยความจำอย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญเมื่อต้องจัดการกับเอกสารขนาดใหญ่ คุณลักษณะ 'MemoryOptimization' ช่วยลดการใช้หน่วยความจำระหว่างการบันทึกข้อมูล

##### ขั้นตอนที่ 1: เริ่มต้นเอกสาร
สร้างใหม่ `Document` วัตถุ:

```java
import com.aspose.words.Document;
import com.aspose.words.DocumentBuilder;

// สร้างเอกสารใหม่
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.writeln("Hello world!");
```

##### ขั้นตอนที่ 2: ตั้งค่าการเพิ่มประสิทธิภาพหน่วยความจำ
กำหนดค่าตัวเลือกการบันทึกของคุณเพื่อเพิ่มประสิทธิภาพการใช้หน่วยความจำ:

```java
import com.aspose.words.WordML2003SaveOptions;

// เริ่มต้น WordML2003SaveOptions
WordML2003SaveOptions options = new WordML2003SaveOptions();
options.setMemoryOptimization(true); // เปิดใช้งานการเพิ่มประสิทธิภาพหน่วยความจำ

doc.save("YOUR_DOCUMENT_DIRECTORY/WordML2003SaveOptions.MemoryOptimization.xml", options);
```

**คำอธิบาย-**
- **`setMemoryOptimization(true)`:** ลดการใช้หน่วยความจำในระหว่างการบันทึกเอกสาร ซึ่งเป็นสิ่งสำคัญสำหรับการจัดการไฟล์ขนาดใหญ่อย่างมีประสิทธิภาพ
