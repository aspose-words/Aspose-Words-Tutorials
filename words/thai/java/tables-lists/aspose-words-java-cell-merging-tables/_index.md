## คู่มือการใช้งาน
มาแบ่งกระบวนการในการผสานเซลล์ตารางออกเป็นขั้นตอนที่สามารถจัดการได้ โดยเน้นที่การผสานทั้งในแนวตั้งและแนวนอน

### การรวมเซลล์แนวตั้ง

#### ภาพรวม:
การรวมเซลล์แนวตั้งจะรวมแถวหลายแถวไว้ด้วยกันในคอลัมน์เดียว เหมาะสำหรับการสร้างส่วนหัวหรือการจัดกลุ่มข้อมูลที่เกี่ยวข้อง

#### การดำเนินการทีละขั้นตอน:
**1. สร้างเอกสารและตัวสร้าง:**
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

**2. แทรกเซลล์ด้วยการผสานแนวตั้ง:**

- **เซลล์แรก (เริ่มการผสาน):** ตั้งเป็นจุดเริ่มต้นของการผสานแนวตั้ง
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.FIRST); // ทำเครื่องหมายเซลล์นี้ให้เป็นจุดเริ่มต้นในการผสาน
  builder.write("Text in merged cells.");
  ```

- **เซลล์ที่สอง (ไม่รวม):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.NONE); // ไม่มีการใช้การผสานที่นี่
  builder.write("Text in unmerged cell.");
  builder.endRow(); // สิ้นสุดแถวปัจจุบัน
  ```

- **เซลล์ที่สาม (ดำเนินการผสาน):** ผสานกับเซลล์แรกในแนวตั้ง
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.PREVIOUS); // ดำเนินการผสานแนวตั้งต่อจากเซลล์ก่อนหน้า
  builder.endRow(); // เสร็จสิ้นแถวที่ 2
  ```

**3. บันทึกเอกสาร:**
```java
doc.save("VerticalMergeOutput.docx");
```

### การรวมเซลล์แนวนอน

#### ภาพรวม:
การผสานแนวนอนจะรวมเซลล์ต่างๆ ในแถวเดียว ซึ่งเหมาะอย่างยิ่งสำหรับการสร้างส่วนหัวที่ครอบคลุมหรือข้อมูลที่ครอบคลุม

#### การดำเนินการทีละขั้นตอน:
**1. สร้างเอกสารและตัวสร้าง:**
ใช้โค้ดเริ่มต้นเดิมซ้ำเหมือนครั้งก่อน

**2. แทรกเซลล์ด้วยการผสานแนวนอน:**

- **เซลล์แรก (เริ่มการผสาน):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.FIRST); // เริ่มต้นการรวมแนวนอน
  builder.write("Text in merged cells.");
  ```

- **เซลล์ที่สอง (ดำเนินการผสาน):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.PREVIOUS); // ต่อจากเซลล์แรกในแนวนอน
  builder.endRow(); // สิ้นสุดแถวปัจจุบัน และทำการผสานแนวนอนให้เสร็จสมบูรณ์
  ```

**3. บันทึกเอกสาร:**
```java
doc.save("HorizontalMergeOutput.docx");
```

### การเติมเซลล์

#### ภาพรวม:
การเพิ่มช่องว่างลงในเซลล์จะช่วยเพิ่มความสามารถในการอ่านได้โดยการสร้างช่องว่างระหว่างข้อความและเส้นขอบ

#### การดำเนินการทีละขั้นตอน:
**1. ตั้งค่าการเติมช่องว่างบนเซลล์:**
```java
builder.getCellFormat().setPaddings(5.0, 10.0, 40.0, 50.0); // เติมจุดบน ขวา ล่าง ซ้าย
```

**2. แทรกเซลล์ด้วยการเติม:**
```java
builder.startTable();
builder.insertCell();
builder.write("Lorem ipsum dolor sit amet...");
builder.endRow();
builder.endTable();
doc.save("PaddingOutput.docx");
```

## การประยุกต์ใช้งานจริง
การทำความเข้าใจเกี่ยวกับวิธีการรวมเซลล์และเพิ่มช่องว่างสามารถปรับปรุงเอกสารได้หลายวิธี:
1. **การสร้างใบแจ้งหนี้**:ใช้การผสานแนวตั้งสำหรับคำอธิบายรายการที่ขยายหลายแถว ซึ่งจะช่วยปรับปรุงความชัดเจน
2. **การสร้างรายงาน**การผสานแนวนอนเหมาะอย่างยิ่งสำหรับการรวมส่วนหัวส่วนต่างๆ ทั่วทั้งตาราง
3. **เทมเพลตประวัติย่อ**:เพิ่มช่องว่างเพื่อให้แน่ใจว่าข้อความในส่วนต่าง ๆ ของประวัติย่อจะสบายตา

เมื่อทำงานกับเอกสารขนาดใหญ่หรือการจัดการตารางจำนวนมาก:
- **เพิ่มประสิทธิภาพการโหลดเอกสาร:** ใช้ `Document` ผู้สร้างอย่างมีประสิทธิภาพโดยโหลดเฉพาะส่วนที่จำเป็นของเอกสารหากเป็นไปได้
