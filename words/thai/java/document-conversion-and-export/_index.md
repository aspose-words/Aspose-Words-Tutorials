## คำถามที่พบบ่อย

**Q: ฉันสามารถแปลง DOCX เป็น PDF บนเซิร์ฟเวอร์ Linux ได้หรือไม่?**  
A: ได้ Aspose.Words for Java ไม่ขึ้นกับแพลตฟอร์มและทำงานได้บนระบบปฏิบัติการใดก็ได้ที่มี JRE ที่รองรับ

**Q: จำเป็นต้องติดตั้ง Microsoft Office บนเซิร์ฟเวอร์หรือไม่?**  
A: ไม่ จำเป็นต้องใช้ไลบรารี Aspose.Words ทำการแปลงทั้งหมด

**Q: ฉันจะรักษาฟอนต์ที่กำหนดเองระหว่างการแปลงได้อย่างไร?**  
A: ใช้ `PdfSaveOptions.setEmbedFullFonts(true)` เพื่อฝังฟอนต์ทั้งหมดที่ใช้ลงใน PDF ที่ได้

**Q: สามารถแปลงไฟล์ DOCX ที่มีการป้องกันด้วยรหัสผ่านได้หรือไม่?**  
A: แน่นอน โหลดเอกสารด้วย `LoadOptions.setPassword("yourPassword")` ก่อนบันทึกเป็น PDF

**Q: วิธีที่แนะนำสำหรับการแปลงเอกสารจำนวนมากเป็นชุดคืออะไร?**  
A: สร้างเมธอดที่ใช้ซ้ำได้สำหรับโหลด, แปลง, และทำลายแต่ละเอกสาร แล้วรันใน thread‑pool เพื่อประมวลผลแบบขนาน
