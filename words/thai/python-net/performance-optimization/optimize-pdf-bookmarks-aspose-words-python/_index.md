## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับกระบวนการเพิ่มประสิทธิภาพบุ๊กมาร์ก PDF โดยใช้ Aspose.Words

### การสร้างและการจัดการบุ๊กมาร์ก

#### ภาพรวม
การสร้างบุ๊กมาร์กใน PDF ช่วยให้ผู้ใช้สามารถนำทางไปยังส่วนต่างๆ ได้อย่างรวดเร็ว การจัดการบุ๊กมาร์กอย่างมีประสิทธิภาพจะช่วยเพิ่มประสบการณ์ของผู้ใช้ได้อย่างมาก

#### การดำเนินการแบบทีละขั้นตอน

##### การเพิ่มบุ๊กมาร์กด้วยระดับโครงร่าง

คุณสามารถเพิ่มบุ๊กมาร์กและกำหนดระดับโครงร่างเพื่อสร้างโครงสร้างลำดับชั้นได้:

```python
builder = aw.DocumentBuilder(doc)
# เริ่มบุ๊กมาร์กที่ชื่อ 'บุ๊กมาร์ก 1'
builder.start_bookmark('Bookmark 1')
builder.writeln('Text inside Bookmark 1.')
builder.end_bookmark('Bookmark 1')

# การเพิ่มบุ๊กมาร์กแบบซ้อนกัน
builder.start_bookmark('Bookmark 2')
builder.writeln('Text inside Nested Bookmark.')
builder.end_bookmark('Bookmark 2')
```

##### การกำหนดค่าระดับเค้าร่างสำหรับการส่งออก PDF

ระดับโครงร่างจะกำหนดว่าจะแสดงบุ๊กมาร์กอย่างไรในเมนูแบบดรอปดาวน์:

```python
pdf_save_options = aw.saving.PdfSaveOptions()
outline_levels = pdf_save_options.outline_options.bookmarks_outline_levels
outline_levels.add('Bookmark 1', 1)
outline_levels.add('Bookmark 2', 2)

# บันทึกเอกสารด้วยบุ๊กมาร์กแบบมีโครงร่าง
doc.save('output.pdf', save_options=pdf_save_options)
```

##### การลบและการล้างบุ๊กมาร์ก

ในการปรับเปลี่ยนโครงสร้างบุ๊กมาร์ก:

```python
# ลบบุ๊กมาร์กเฉพาะตามชื่อ
outline_levels.remove('Bookmark 2')

# ล้างระดับเค้าร่างทั้งหมด ตั้งค่าบุ๊กมาร์กเป็นค่าเริ่มต้น
outline_levels.clear()
```

### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาทั่วไป**:หากบุ๊กมาร์กไม่ปรากฏตามที่คาดหวังใน PDF ให้แน่ใจว่าคุณได้บันทึกเอกสารด้วย `PdfSaveOptions`-
- **การแก้จุดบกพร่อง**:ใช้คำสั่งพิมพ์หรือการบันทึกข้อมูลเพื่อตรวจสอบชื่อบุ๊กมาร์กและระดับโครงร่าง
