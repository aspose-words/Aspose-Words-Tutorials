# เพิ่มประสิทธิภาพไฟล์ Excel ด้วย Aspose.Words สำหรับ Python: เทคนิคการบีบอัดและการปรับแต่ง

ค้นพบเทคนิคอันทรงพลังในการบีบอัด จัดระเบียบ และปรับปรุงประสิทธิภาพของเอกสาร Excel ของคุณอย่างมีประสิทธิภาพโดยใช้ Aspose.Words สำหรับ Python บทช่วยสอนนี้จะแนะนำคุณตลอดกระบวนการเพิ่มประสิทธิภาพไฟล์ XLSX โดยการลดขนาดไฟล์ บันทึกหลายส่วนเป็นเวิร์กชีตแยกต่างหาก และเปิดใช้งานการตรวจจับรูปแบบวันที่และเวลาโดยอัตโนมัติ

## การแนะนำ

การจัดการข้อมูลเอกสารขนาดใหญ่ มักจะส่งผลให้ไฟล์ XLSX มีขนาดใหญ่ขึ้น ซึ่งยุ่งยากในการจัดการและแบ่งปัน ไม่ว่าจะต้องจัดการกับแผนภูมิ ตาราง หรือรายงานจำนวนมาก การจัดเก็บและจัดระเบียบอย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญ Aspose.Words for Python นำเสนอโซลูชันที่แข็งแกร่งโดยให้ตัวเลือกการบีบอัดขั้นสูงและการตั้งค่าการบันทึกแบบกำหนดเอง

ในบทช่วยสอนนี้ คุณจะได้เรียนรู้วิธีการ:
- บีบอัดเอกสาร XLSX เพื่อลดขนาดไฟล์ให้เหมาะสมที่สุด
- บันทึกส่วนเอกสารแต่ละส่วนเป็นเวิร์กชีตแยกกัน
- เปิดใช้งานการตรวจจับอัตโนมัติของรูปแบบวันที่และเวลาในไฟล์ของคุณ

เมื่ออ่านคู่มือนี้จบ คุณจะมีความรู้เชิงปฏิบัติเกี่ยวกับการเพิ่มประสิทธิภาพและการเข้าถึงไฟล์ Excel ของคุณ

## คู่มือการใช้งาน
เราจะแบ่งการใช้งานออกเป็นคุณลักษณะที่แตกต่างกัน โดยอธิบายแต่ละขั้นตอนด้วยตัวอย่างโค้ดและการกำหนดค่า

### คุณสมบัติ 1: บีบอัดเอกสาร XLSX
**ภาพรวม**:คุณลักษณะนี้ช่วยลดขนาดไฟล์เอกสาร Excel ของคุณด้วยการใช้การบีบอัดสูงสุดเมื่อบันทึกเป็นไฟล์ XLSX

#### การดำเนินการทีละขั้นตอน:
##### โหลดเอกสารของคุณ
เริ่มต้นด้วยการโหลดเอกสารที่คุณต้องการบีบอัด:

```python
import aspose.words as aw

YOUR_DOCUMENT_DIRECTORY = 'path/to/your/document/directory'
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Shape with linked chart.docx')
```

##### กำหนดค่าการตั้งค่าการบีบอัด
สร้างอินสแตนซ์ของ `XlsxSaveOptions` และตั้งค่าระดับการบีบอัดให้สูงสุด:

```python
xlsx_save_options = aw.saving.XlsxSaveOptions()
xlsx_save_options.compression_level = aw.saving.CompressionLevel.MAXIMUM
xlsx_save_options.save_format = aw.SaveFormat.XLSX
```

##### บันทึกด้วยการบีบอัด
สุดท้าย ให้บันทึกเอกสารของคุณโดยใช้ตัวเลือกเหล่านี้เพื่อให้ได้ไฟล์ XLSX ที่ถูกบีบอัด:

```python
YOUR_OUTPUT_DIRECTORY = 'path/to/your/output/directory'
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'CompressedOutput.xlsx', save_options=xlsx_save_options)
```

### คุณสมบัติ 2: บันทึกเอกสารเป็นเวิร์กชีตแยกต่างหาก
**ภาพรวม**:คุณลักษณะนี้ช่วยให้สามารถบันทึกแต่ละส่วนของเอกสารลงในเวิร์กชีตของตัวเองได้ ช่วยให้จัดระเบียบข้อมูลได้ดีขึ้น

#### การดำเนินการทีละขั้นตอน:
##### โหลดเอกสารขนาดใหญ่ของคุณ

```python
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Big document.docx')
```

##### ตั้งค่าโหมดส่วน
กำหนดค่า `XlsxSaveOptions` เพื่อบันทึกแต่ละส่วนเป็นเวิร์กชีตแยกกัน:

```python
xlsx_save_options = aw.saving.XlsxSaveOptions()
xlsx_save_options.section_mode = aw.saving.XlsxSectionMode.MULTIPLE_WORKSHEETS
```

##### บันทึกด้วยแผ่นงานหลายแผ่น
ดำเนินการบันทึกฟังก์ชั่น:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'MultipleWorksheetsOutput.xlsx', save_options=xlsx_save_options)
```

### คุณสมบัติที่ 3: ระบุโหมดการแยกวิเคราะห์ DateTime
**ภาพรวม**: เปิดใช้งานการตรวจจับรูปแบบวันที่และเวลาอัตโนมัติเพื่อรับรองความถูกต้องและความสอดคล้องในเอกสารของคุณ

#### การดำเนินการทีละขั้นตอน:
##### โหลดเอกสารด้วยข้อมูลวันที่และเวลา

```python
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Xlsx DateTime.docx')
```

##### กำหนดค่าการแยกวิเคราะห์ DateTime
ตั้งค่าการตรวจจับอัตโนมัติสำหรับรูปแบบวันที่และเวลาโดยใช้ `XlsxSaveOptions`-

```python
save_options = aw.saving.XlsxSaveOptions()
save_options.date_time_parsing_mode = aw.saving.XlsxDateTimeParsingMode.AUTO
```

##### บันทึกด้วยรูปแบบวันที่-เวลาที่ตรวจพบอัตโนมัติ
บันทึกเอกสารเพื่อใช้การตั้งค่าเหล่านี้:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'DateTimeParsingModeOutput.xlsx', save_options=save_options)
```
