## คู่มือการใช้งาน
เราจะสำรวจคุณลักษณะแต่ละอย่างทีละอย่างเพื่อสาธิตการใช้งาน

### แรสเตอร์องค์ประกอบที่ซับซ้อนใน PCL
การแรสเตอร์องค์ประกอบที่ซับซ้อนช่วยให้มั่นใจได้ว่าการแปลง เช่น การหมุนหรือการปรับขนาดจะได้รับการรักษาไว้อย่างแม่นยำเมื่อทำการพิมพ์ คุณสามารถทำได้ดังนี้:

#### ภาพรวม
การเปิดใช้งานการแรสเตอร์ไรเซชันขององค์ประกอบที่แปลงแล้วนั้นถือเป็นสิ่งจำเป็นสำหรับการรักษาความเที่ยงตรงของภาพระหว่างงานพิมพ์ โดยเฉพาะอย่างยิ่งกับการออกแบบที่ซับซ้อน

```python
import aspose.words as aw
# โหลดเอกสาร
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Rendering.docx')
save_options = aw.saving.PclSaveOptions()
save_options.save_format = aw.SaveFormat.PCL
save_options.rasterize_transformed_elements = True  # เปิดใช้งานการแรสเตอร์ขององค์ประกอบที่ถูกแปลงแล้ว
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.RasterizeElements.pcl', save_options=save_options)
```

**คำอธิบายพารามิเตอร์:**
- `rasterize_transformed_elements`:รับประกันว่าการเปลี่ยนแปลงใดๆ ที่ใช้กับองค์ประกอบจะถูกยังคงอยู่ในผลลัพธ์ที่พิมพ์ออกมา

### ประกาศฟอนต์ Fallback สำหรับ PCL
หากไม่สามารถใช้แบบอักษรที่ระบุได้ การมีแบบอักษรสำรองจะช่วยให้เอกสารของคุณพิมพ์ออกมาได้โดยไม่มีองค์ประกอบที่ขาดหายไป นี่คือวิธีตั้งค่า:

#### ภาพรวม
ระบุแบบอักษรทดแทนที่จะใช้หากไม่พบแบบอักษรต้นฉบับระหว่างการพิมพ์

```python
import aspose.words as aw
doc = aw.Document()
builder = aw.DocumentBuilder(doc=doc)
builder.font.name = 'Non-existent font'  # ใช้ชื่อแบบอักษรที่ไม่พร้อมใช้งานโดยตั้งใจ
derived_text = builder.write('Hello world!')

save_options = aw.saving.PclSaveOptions()
save_options.fallback_font_name = 'Times New Roman'  # ตั้งค่าแบบอักษรสำรอง
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.SetPrinterFont.pcl', save_options=save_options)
```

**คำอธิบายพารามิเตอร์:**
- `fallback_font_name`:ชื่อของแบบอักษรที่จะใช้หากแบบอักษรเดิมไม่มี

### เพิ่มการแทนที่ฟอนต์เครื่องพิมพ์ใน PCL
แทนที่แบบอักษรเฉพาะสำหรับเอกสารระหว่างการพิมพ์เพื่อความเข้ากันได้ดีขึ้น:

#### ภาพรวม
เปลี่ยนแบบอักษรที่ระบุด้วยแบบอักษรอื่นเมื่อทำการพิมพ์ ช่วยให้มั่นใจว่าข้อความจะปรากฏสม่ำเสมอกันบนอุปกรณ์ต่างๆ

```python
import aspose.words as aw
doc = aw.Document()
builder = aw.DocumentBuilder(doc=doc)
builder.font.name = 'Courier'
builder.write('Hello world!')

save_options = aw.saving.PclSaveOptions()
save_options.add_printer_font('Courier New', 'Courier')  # แทนที่ 'Courier' ด้วย 'Courier New'
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.AddPrinterFont.pcl', save_options=save_options)
```

**คำอธิบายพารามิเตอร์:**
- `add_printer_font`:ปรับเปลี่ยนแบบอักษรเดิมเพื่อใช้แทนการพิมพ์

### เก็บรักษาข้อมูลถาดกระดาษใน PCL
การรักษาการตั้งค่าถาดกระดาษเป็นสิ่งสำคัญเมื่อต้องจัดการกับเครื่องพิมพ์หลายถาด:

#### ภาพรวม
รักษาการตั้งค่าถาดเฉพาะสำหรับส่วนต่างๆ ของเอกสารของคุณ เพื่อให้แน่ใจว่าใช้กระดาษอย่างถูกต้องระหว่างงานพิมพ์

```python
import aspose.words as aw
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Rendering.docx')

for section in doc.sections:
    section.page_setup.first_page_tray = 15  # ตั้งถาดหน้าแรกเป็น 15
    section.page_setup.other_pages_tray = 12  # ตั้งค่าถาดหน้าอื่นเป็น 12

doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.GetPreservedPaperTrayInformation.pcl')
```

**คำอธิบายพารามิเตอร์:**
- `first_page_tray` และ `other_pages_tray`: กำหนดถาดกระดาษสำหรับหน้าแรกและหน้าถัดไป
