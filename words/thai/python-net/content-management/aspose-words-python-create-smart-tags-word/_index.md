## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นคุณสมบัติต่างๆ ของสมาร์ทแท็กกัน

### สร้างสมาร์ทแท็ก (H2)

#### ภาพรวม
การสร้างสมาร์ทแท็กเกี่ยวข้องกับการเพิ่มองค์ประกอบข้อความที่จดจำได้ลงในเอกสารของคุณและเชื่อมโยงกับคุณสมบัติ XML ที่กำหนดเอง หัวข้อนี้จะแนะนำคุณเกี่ยวกับการสร้างสมาร์ทแท็กประเภทวันที่และประเภทหุ้น

#### การดำเนินการแบบทีละขั้นตอน

##### 1. ตั้งค่าเอกสารของคุณ
เริ่มต้นด้วยการนำเข้า Aspose.Words และเริ่มต้นเอกสาร Word ใหม่:
```python
import aspose.words as aw

def create_smart_tags():
    doc = aw.Document()
```

##### 2. สร้างสมาร์ทแท็กประเภทวันที่
เพิ่มข้อความที่รู้จักเป็นวันที่ และกำหนดค่าคุณสมบัติ XML แบบกำหนดเอง
```python
# เพิ่มสมาร์ทแท็กประเภทวันที่พร้อมคุณสมบัติ XML แบบกำหนดเอง
smart_tag_date = aw.markup.SmartTag(doc)
smart_tag_date.append_child(aw.Run(doc, 'May 29, 2019'))
smart_tag_date.element = 'date'
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Day', '', '29'))
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Month', '', '5'))
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Year', '', '2019'))
smart_tag_date.uri = 'urn:schemas-microsoft-com:office:smarttags'
doc.first_section.body.first_paragraph.append_child(smart_tag_date)
```

##### 3. สร้างสมาร์ทแท็กประเภทสัญลักษณ์หุ้น
กำหนดค่าสมาร์ทแท็กอื่นสำหรับสัญลักษณ์หุ้น
```python
# เพิ่มสมาร์ทแท็กประเภทเครื่องหมายหุ้น
smart_tag_stock = aw.markup.SmartTag(doc)
smart_tag_stock.element = 'stockticker'
smart_tag_stock.uri = 'urn:schemas-microsoft-com:office:smarttags'
smart_tag_stock.append_child(aw.Run(doc, 'MSFT'))
doc.first_section.body.first_paragraph.append_child(smart_tag_stock)
```

##### 4. บันทึกเอกสารของคุณ
สุดท้ายให้บันทึกเอกสารด้วยสมาร์ทแท็กที่กำหนดค่าไว้ทั้งหมด
```python
# บันทึกเอกสารไปยังเส้นทางที่ระบุ
output_path = YOUR_OUTPUT_DIRECTORY + 'SmartTag.create.doc'
doc.save(output_path)
print(f'Document saved to {output_path}')
```

### การลบสมาร์ทแท็ก (H2)

#### ภาพรวม
บางครั้งคุณจำเป็นต้องทำความสะอาดเอกสารของคุณโดยการลบสมาร์ทแท็กที่มีอยู่ ในส่วนนี้จะแสดงวิธีการดำเนินการดังกล่าว

#### การนำไปปฏิบัติ

##### 1. โหลดเอกสาร
เริ่มต้นด้วยการโหลดเอกสาร Word ที่มีสมาร์ทแท็ก
```python
def remove_smart_tags():
    input_path = YOUR_DOCUMENT_DIRECTORY + 'SmartTag.create.doc'
    doc = aw.Document(input_path)
```

##### 2. ถอดสมาร์ทแท็กทั้งหมดออก
ดำเนินการวิธีการลบสมาร์ทแท็กทั้งหมดออกจากเอกสารของคุณ
```python
# ลบสมาร์ทแท็กทั้งหมดและตรวจสอบจำนวนก่อนและหลังการลบ
initial_count = doc.get_child_nodes(aw.NodeType.SMART_TAG, True).count
doc.remove_smart_tags()
final_count = doc.get_child_nodes(aw.NodeType.SMART_TAG, True).count
print(f'Initial number of smart tags: {initial_count}')
print(f'Final number of smart tags: {final_count}')
```

### เข้าถึงคุณสมบัติสมาร์ทแท็ก (H2)

#### ภาพรวม
การทำความเข้าใจและการจัดการคุณสมบัติของสมาร์ทแท็กสามารถปรับปรุงวิธีการประมวลผลข้อมูลได้ หัวข้อนี้จะกล่าวถึงการเข้าถึงคุณสมบัติเหล่านี้

#### การนำไปปฏิบัติ

##### 1. โหลดเอกสารด้วยสมาร์ทแท็ก
โหลดเอกสารและดึงสมาร์ทแท็กทั้งหมด
```python
def access_smart_tag_properties():
    input_path = YOUR_DOCUMENT_DIRECTORY + 'SmartTag.create.doc'
    doc = aw.Document(input_path)
```

##### 2. ดึงข้อมูลและเข้าถึงคุณสมบัติ
เข้าถึงคุณสมบัติของสมาร์ทแท็กเฉพาะโดยแสดงการโต้ตอบต่างๆ
```python
# แยกสมาร์ทแท็กออกจากเอกสาร
smart_tags = [node.as_smart_tag() for node in doc.get_child_nodes(aw.NodeType.SMART_TAG, True)]
print(f'Total number of smart tags: {len(smart_tags)}')

# เข้าถึงคุณสมบัติและสาธิตตัวเลือกการจัดการ
properties = smart_tags[-1].properties
for prop in properties:
    print(f'Property name: {prop.name}, value: {prop.value}')

if properties.contains('Day'):
    day_value = properties.get_by_name('Day').value
    print(f'Day property value: {day_value}')

month_index = properties.index_of_key('Month')
print(f'Month index in properties: {month_index}')
```

##### 3. ปรับเปลี่ยนคุณสมบัติ
ลบหรือล้างคุณสมบัติเฉพาะตามความจำเป็น
```python
# ลบคุณสมบัติที่เฉพาะเจาะจงและล้างคุณสมบัติทั้งหมด
if 'Year' in [prop.name for prop in properties]:
    properties.remove('Year')
    print('Year property removed.')

properties.clear()
print(f'Properties count after clearing: {properties.count}')
```

สมาร์ทแท็กสามารถใช้งานได้ในสถานการณ์จริงต่างๆ เช่น:

2. **การดึงข้อมูล**:ดึงประเภทข้อมูลเฉพาะอย่างมีประสิทธิภาพสำหรับการวิเคราะห์จากเอกสารขนาดใหญ่
3. **การทำงานร่วมกันที่ได้รับการปรับปรุง**:ลดความยุ่งยากในการแชร์เอกสารโดยการจดจำและจัดรูปแบบข้อมูลที่สำคัญโดยอัตโนมัติ
