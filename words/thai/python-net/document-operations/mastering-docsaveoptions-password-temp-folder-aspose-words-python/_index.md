## การแนะนำ

คุณกำลังมองหาวิธีเพิ่มความปลอดภัยให้กับเอกสาร Microsoft Word ของคุณพร้อมทั้งเพิ่มประสิทธิภาพการประมวลผลไฟล์หรือไม่ ไม่ว่าจะเป็นการปกป้องข้อมูลที่ละเอียดอ่อนด้วยรหัสผ่านหรือการจัดการไฟล์ขนาดใหญ่โดยใช้โฟลเดอร์ชั่วคราว Aspose.Words สำหรับ Python มอบเครื่องมืออันทรงพลังเพื่อตอบสนองความต้องการเหล่านี้ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการป้องกันด้วยรหัสผ่านและการใช้โฟลเดอร์ชั่วคราวในกระบวนการบันทึกเอกสาร

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน

### การป้องกันด้วยรหัสผ่านและการเก็บรักษาใบนำส่ง (H2)

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณตั้งรหัสผ่านสำหรับรูปแบบเอกสาร Microsoft Word เวอร์ชันเก่าได้ เพื่อให้แน่ใจว่าเอกสารของคุณปลอดภัย นอกจากนี้ ยังรักษาข้อมูลใบกำกับเส้นทางระหว่างขั้นตอนการบันทึกอีกด้วย

##### ตั้งค่า DocSaveOptions พร้อมการป้องกันด้วยรหัสผ่าน (H3)

ขั้นแรกให้สร้างเอกสารใหม่และกำหนดค่า `DocSaveOptions`-

```python
import aspose.words as aw

def save_with_password_and_routing_slip():
    # สร้างเอกสารใหม่
    doc = aw.Document()
    builder = aw.DocumentBuilder(doc=doc)
    builder.write('Hello world!')

    # กำหนดค่า DocSaveOptions สำหรับการป้องกันด้วยรหัสผ่าน
    options = aw.saving.DocSaveOptions(aw.SaveFormat.DOC)
    options.password = 'MyPassword'

    # เก็บรักษาข้อมูลใบแจ้งเส้นทาง
    options.save_routing_slip = True

    # บันทึกเอกสาร
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithPasswordAndRoutingSlip.doc"
    doc.save(file_name=output_path, save_options=options)

    # ตรวจสอบโดยโหลดด้วยรหัสผ่าน
    load_options = aw.loading.LoadOptions(password='MyPassword')
    loaded_doc = aw.Document(file_name=output_path, load_options=load_options)
    assert 'Hello world!' == loaded_doc.get_text().strip()
```

**คำอธิบายพารามิเตอร์:**
- `options.password`: กำหนดรหัสผ่านสำหรับการป้องกันเอกสาร
- `options.save_routing_slip`: เก็บรักษาข้อมูลใบแจ้งเส้นทาง

#### เคล็ดลับการแก้ไขปัญหา

- ให้แน่ใจว่าเส้นทางไดเร็กทอรีเอาต์พุตมีอยู่ก่อนการบันทึก
- ใช้รหัสผ่านที่ไม่ซ้ำใครและแข็งแกร่งเพื่อเพิ่มความปลอดภัย

### การใช้งานโฟลเดอร์ชั่วคราว (H2)

#### ภาพรวม

เมื่อต้องจัดการกับเอกสารขนาดใหญ่ การใช้โฟลเดอร์ชั่วคราวบนดิสก์สามารถปรับปรุงประสิทธิภาพการทำงานได้โดยลดการใช้หน่วยความจำ

##### กำหนดค่า DocSaveOptions สำหรับโฟลเดอร์ชั่วคราว (H3)

วิธีตั้งค่าโฟลเดอร์ชั่วคราวมีดังนี้:

```python
import os
import aspose.words as aw

def save_using_temp_folder():
    # โหลดเอกสารที่มีอยู่
    input_path = "YOUR_DOCUMENT_DIRECTORY/Rendering.docx"
    doc = aw.Document(file_name=input_path)

    # กำหนดค่า DocSaveOptions ให้ใช้โฟลเดอร์ชั่วคราว
    options = aw.saving.DocSaveOptions()
    temp_folder = "YOUR_OUTPUT_DIRECTORY/TempFiles"

    # ให้แน่ใจว่าโฟลเดอร์ชั่วคราวมีอยู่
    os.makedirs(temp_folder, exist_ok=True)
    options.temp_folder = temp_folder

    # บันทึกโดยใช้โฟลเดอร์ชั่วคราว
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithTempFolder.doc"
    doc.save(file_name=output_path, save_options=options)
```

**ตัวเลือกการกำหนดค่าคีย์:**
- `options.temp_folder`: ระบุเส้นทางที่จะใช้สำหรับจัดเก็บไฟล์กลาง

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบสิทธิ์การเขียนสำหรับโฟลเดอร์ชั่วคราวของคุณ
- ตรวจสอบให้แน่ใจว่ามีพื้นที่ดิสก์เพียงพอในไดเร็กทอรีที่ระบุ

## การประยุกต์ใช้งานจริง

ต่อไปนี้เป็นการประยุกต์ใช้งานจริงของฟีเจอร์เหล่านี้:

1. **การแบ่งปันเอกสารที่ปลอดภัย**:ใช้การป้องกันด้วยรหัสผ่านเมื่อแชร์เอกสารสำคัญกับพันธมิตรภายนอก
2. **การประมวลผลไฟล์ขนาดใหญ่**:เพิ่มประสิทธิภาพการใช้หน่วยความจำโดยใช้ประโยชน์จากโฟลเดอร์ชั่วคราวในระหว่างการประมวลผลแบบแบตช์หรือการย้ายข้อมูล
3. **การควบคุมเวอร์ชันเอกสาร**:รักษาใบส่งของเพื่อรักษาประวัติเอกสารและเวิร์กโฟลว์การอนุมัติ
