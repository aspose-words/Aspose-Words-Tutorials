## คู่มือการใช้งาน
มาใช้งานฟีเจอร์ที่จะแสดงข้อมูลเวอร์ชัน Aspose.Words กันเถอะ

### ภาพรวมคุณสมบัติ:
หัวข้อนี้สาธิตวิธีการแยกและพิมพ์ชื่อผลิตภัณฑ์และเวอร์ชันของ Aspose.Words สำหรับ Python ผ่านทาง .NET โดยใช้คลาสในตัว

#### ขั้นตอนที่ 1: นำเข้าไลบรารี
เริ่มต้นด้วยการนำเข้า `aspose.words` โมดูลที่ให้คุณเข้าถึงคุณสมบัติทั้งหมดได้

```python
import aspose.words as aw
```

#### ขั้นตอนที่ 2: ดึงข้อมูลเวอร์ชัน
ใช้ `BuildVersionInfo` คลาสเพื่อรับชื่อผลิตภัณฑ์และหมายเลขเวอร์ชัน คลาสนี้ให้ข้อมูลโดยละเอียดเกี่ยวกับไลบรารี Aspose.Words ที่ติดตั้ง

```python
product_name = aw.BuildVersionInfo.product
version_number = aw.BuildVersionInfo.version
```

#### ขั้นตอนที่ 3: แสดงข้อมูล
พิมพ์ข้อมูลที่เรียกค้นได้โดยใช้สตริงตัวอักษรที่จัดรูปแบบของ Python เพื่อความชัดเจนและอ่านง่าย

```python
print(f'I am currently using {product_name}, version number {version_number}!')
```

### พารามิเตอร์และค่าส่งคืน:
- `BuildVersionInfo.product`: ส่งคืนสตริงที่แสดงชื่อผลิตภัณฑ์
- `BuildVersionInfo.version`: ให้สตริงที่มีหมายเลขเวอร์ชัน

2. **การแก้จุดบกพร่อง**ตรวจสอบอย่างรวดเร็วว่าการอัปเดตหรือดาวน์เกรดอาจแก้ไขปัญหาได้หรือไม่โดยการตรวจสอบเวอร์ชันปัจจุบัน
