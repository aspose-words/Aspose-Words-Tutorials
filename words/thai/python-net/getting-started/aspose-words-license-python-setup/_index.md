## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การตั้งค่าใบอนุญาตจากไฟล์และจากสตรีม

### การตั้งค่าใบอนุญาตจากไฟล์

ฟีเจอร์นี้ช่วยให้คุณสามารถใช้สิทธิ์การใช้งาน Aspose.Words โดยใช้เส้นทางไฟล์ที่ระบุ

#### ภาพรวม
ด้วยการใช้ใบอนุญาตจากไฟล์ แอปพลิเคชันของคุณจะสามารถพิสูจน์ตัวตนด้วย Aspose.Words ได้ ทำให้สามารถปลดล็อกคุณสมบัติพรีเมียมทั้งหมดได้

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: นำเข้าโมดูลที่จำเป็น**

```python
import aspose.words as aw
```

**ขั้นตอนที่ 2: กำหนดฟังก์ชันการสมัครใบอนุญาต**

```python
def apply_license_from_file(license_path):
    """
    Apply a license for Aspose.Words using the specified file path.
    
    Parameters:
    - license_path (str): The local file system path to the valid license file.
    """
    # สร้างอินสแตนซ์ของใบอนุญาต
    license = aw.License()
    # ตั้งค่าใบอนุญาตโดยส่งเส้นทางไฟล์
    license.set_license(license_path)
```

- **พารามิเตอร์**- `license_path` ควรเป็นสตริงที่แสดงเส้นทางแบบเต็มไปยังไฟล์ใบอนุญาตของคุณ
- **ค่าส่งคืน**: ฟังก์ชั่นนี้จะไม่ส่งคืนสิ่งใดเลย แต่จะตั้งค่าใบอนุญาตภายใน

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ที่ระบุถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าไฟล์ใบอนุญาตถูกต้องและไม่เสียหาย

### การตั้งค่าใบอนุญาตจากสตรีม

คุณลักษณะนี้ช่วยให้มีสภาพแวดล้อมแบบไดนามิกมากขึ้นโดยสามารถโหลดไฟล์ลงในหน่วยความจำได้แทนที่จะเข้าถึงโดยตรงบนดิสก์

#### ภาพรวม
การใช้สตรีมสามารถเพิ่มประสิทธิภาพได้ โดยเฉพาะเมื่อต้องจัดการกับไฟล์ขนาดใหญ่หรือแอปพลิเคชันที่ใช้เครือข่าย

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: นำเข้าโมดูลที่จำเป็น**

```python
import aspose.words as aw
from io import BytesIO
```

**ขั้นตอนที่ 2: กำหนดฟังก์ชันเพื่อใช้ใบอนุญาตโดยใช้สตรีม**

```python
def apply_license_from_stream(stream):
    """
    Apply a license for Aspose.Words by passing a file stream.
    
    Parameters:
    - stream (BytesIO): A stream containing the valid license file content.
    """
    # สร้างอินสแตนซ์ของใบอนุญาต
    license = aw.License()
    # ตั้งค่าใบอนุญาตโดยใช้สตรีมที่ให้มา
    with stream as my_stream:
        license.set_license(my_stream)
```

- **พารามิเตอร์**- `stream` ควรเป็นวัตถุ BytesIO ที่ประกอบด้วยข้อมูลใบอนุญาตของคุณ
- **ค่าส่งคืน**:คล้ายกับวิธีการไฟล์ ฟังก์ชันนี้จะตั้งค่าใบอนุญาตภายใน

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าสตรีมได้รับการเริ่มต้นอย่างถูกต้องด้วยเนื้อหาใบอนุญาตที่ถูกต้อง
- จัดการข้อยกเว้นสำหรับการดำเนินการ I/O อย่างเหมาะสมเพื่อหลีกเลี่ยงข้อผิดพลาดขณะรันไทม์
