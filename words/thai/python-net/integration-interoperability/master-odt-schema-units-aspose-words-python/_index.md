# เรียนรู้ ODT Schema และ Units ด้วย Aspose.Words ใน Python

## การแนะนำ

คุณกำลังดิ้นรนเพื่อให้แน่ใจว่าเอกสารของคุณเป็นไปตามมาตรฐาน Open Document Format (ODF) เฉพาะหรือไม่ หรือต้องการควบคุมหน่วยการวัดที่แม่นยำเมื่อแปลงไฟล์หรือไม่ ด้วยไลบรารี "Aspose.Words Python" คุณสามารถรับมือกับความท้าทายเหล่านี้ได้อย่างง่ายดาย คู่มือนี้เกี่ยวกับการใช้ประโยชน์จาก Aspose.Words สำหรับ Python เพื่อเชี่ยวชาญการตั้งค่าโครงร่าง ODT และการแปลงหน่วย

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการปรับเอกสารให้สอดคล้องกับรูปแบบ ODT ที่แตกต่างกัน
- การตั้งค่าหน่วยการวัดในไฟล์ ODT ด้วยความแม่นยำ
- การเข้ารหัสเอกสาร ODT/OTT โดยใช้รหัสผ่าน

มาเจาะลึกข้อกำหนดเบื้องต้นที่คุณต้องมีก่อนที่เราจะเริ่มสำรวจฟีเจอร์เหล่านี้

## คู่มือการใช้งาน

### สอดคล้องกับเวอร์ชันของ Schema ODT

#### ภาพรวม

เพื่อให้แน่ใจว่าเอกสารจะเข้ากันได้กับเวอร์ชันเฉพาะต่างๆ ของข้อกำหนด OpenDocument (รูปแบบ ODT) Aspose.Words ช่วยให้คุณกำหนดได้ว่าเอกสารของคุณควรปฏิบัติตามข้อกำหนดของเวอร์ชัน 1.1 อย่างเคร่งครัดหรือไม่

**ทีละขั้นตอน:**

##### ขั้นตอนที่ 1: การตั้งค่าตัวเลือกการบันทึก
```python
import aspose.words as aw

doc = aw.Document('path/to/your/input.docx')
save_options = aw.saving.OdtSaveOptions()
```

##### ขั้นตอนที่ 2: กำหนดค่าเวอร์ชัน Schema ของ ODT
```python
# ตั้งค่าเป็น True เพื่อให้เป็นไปตาม ODT เวอร์ชัน 1.1 อย่างเคร่งครัด
save_options.is_strict_schema11 = True
```

##### ขั้นตอนที่ 3: บันทึกเอกสาร
```python
doc.save('path/to/your/output.odt', save_options)
```

### การกำหนดค่าหน่วยการวัด

#### ภาพรวม

Aspose.Words ช่วยให้คุณเลือกหน่วยเมตริก (เซนติเมตร) หรืออิมพีเรียล (นิ้ว) เมื่อบันทึกเอกสารในรูปแบบ ODT ความยืดหยุ่นนี้ช่วยให้มั่นใจได้ว่าพารามิเตอร์รูปแบบของคุณตรงตามมาตรฐานที่ต้องการ

**ทีละขั้นตอน:**

##### ขั้นตอนที่ 1: เลือกหน่วยการวัด
```python
save_options = aw.saving.OdtSaveOptions()
# เลือกระหว่างเซนติเมตรหรือนิ้วตามความต้องการของคุณ
save_options.measure_unit = aw.saving.OdtSaveMeasureUnit.CENTIMETERS
```

##### ขั้นตอนที่ 2: บันทึกเอกสารด้วยหน่วย
```python
doc.save('path/to/your/output.odt', save_options)
```

### การเข้ารหัสเอกสาร ODT/OTT

#### ภาพรวม

Aspose.Words ช่วยให้คุณรักษาความปลอดภัยเอกสารของคุณโดยการเข้ารหัส หัวข้อนี้จะกล่าวถึงวิธีการใช้การป้องกันด้วยรหัสผ่านเมื่อบันทึกไฟล์ ODT หรือ OTT

**ทีละขั้นตอน:**

##### ขั้นตอนที่ 1: เริ่มต้นเอกสารและบันทึกตัวเลือก
```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Hello world!")
save_options = aw.saving.OdtSaveOptions(aw.SaveFormat.ODT)
```

##### ขั้นตอนที่ 2: ตั้งค่าการป้องกันด้วยรหัสผ่าน
```python
# ตั้งรหัสผ่านสำหรับการเข้ารหัส
save_options.password = 'your_password_here'
doc.save('path/to/encrypted_output.odt', save_options)
```
