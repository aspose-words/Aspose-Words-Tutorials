## คู่มือการใช้งาน

### การลงทะเบียนพจนานุกรมการแบ่งคำ

#### ภาพรวม
การลงทะเบียนพจนานุกรมช่วยให้ Aspose.Words สามารถใช้กฎการแบ่งคำเฉพาะตำแหน่งได้ ทำให้รักษาการไหลของข้อความในการตั้งค่าหลายภาษา

#### กระบวนการทีละขั้นตอน

**1. ระบุไดเรกทอรี**

กำหนดเส้นทางสำหรับเอกสารอินพุตและไดเร็กทอรีเอาต์พุตของคุณ:

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY'
arartifacts_directory = 'YOUR_OUTPUT_DIRECTORY'
```

**2. ลงทะเบียนพจนานุกรม**

ใช้ Aspose.Words เพื่อลงทะเบียนพจนานุกรมการแบ่งคำสำหรับตำแหน่ง "de-CH"

```python
aw.Hyphenation.register_dictionary('de-CH', document_directory + 'hyph_de_CH.dic')
```
*พารามิเตอร์:*
- `'de-CH'`: ตัวระบุตำแหน่ง
- `document_directory + 'hyph_de_CH.dic'`: เส้นทางไปยังไฟล์พจนานุกรมการแบ่งคำ

**3. ตรวจสอบการลงทะเบียน**

ตรวจสอบให้แน่ใจว่าพจนานุกรมได้รับการลงทะเบียนอย่างถูกต้อง:

```python
assert aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be registered"
```

### การใช้การแบ่งคำ

เปิดเอกสารและบันทึกโดยใช้การแบ่งคำโดยใช้พจนานุกรมที่ลงทะเบียนใหม่:

### การยกเลิกการลงทะเบียนพจนานุกรมการแบ่งคำ

#### ภาพรวม
การยกเลิกการลงทะเบียนจะลบกฎเฉพาะตำแหน่ง และกลับสู่พฤติกรรมการแบ่งคำแบบเริ่มต้น

**1. ยกเลิกการลงทะเบียนพจนานุกรม**

```python
aw.Hyphenation.unregister_dictionary('de-CH')
```
*วัตถุประสงค์:* ลบการลงทะเบียนพจนานุกรม "de-CH" เพื่อป้องกันการใช้งานในการประมวลผลเอกสารในอนาคต

**2. ตรวจสอบการยกเลิกการลงทะเบียน**

ยืนยันว่าพจนานุกรมไม่ได้ใช้งานอีกต่อไป:

```python
assert not aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be unregistered"
```

### การบันทึกโดยไม่ต้องใช้เครื่องหมายขีดกลาง

เปิดและบันทึกเอกสารของคุณอีกครั้ง ครั้งนี้โดยไม่ต้องใช้กฎการแบ่งคำที่ลงทะเบียนไว้ก่อนหน้านี้:

```python

```
