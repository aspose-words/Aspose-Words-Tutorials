# สร้างเอกสาร Word ด้วยภาพลอย

ตัวอย่างนี้สาธิตวิธีการสร้างเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET และแทรกภาพด้วยเค้าโครงลอยตัว โค้ดที่ให้มาจะเริ่มต้น Document และ DocumentBuilder แทรกภาพด้วยขนาดและตำแหน่งที่ระบุ และบันทึกเอกสารในที่สุด คู่มือนี้เหมาะสำหรับนักพัฒนาที่ต้องการปรับปรุงความสามารถในการสร้างเอกสารโดยใช้ Aspose.Words

## คำแนะนำในการติดตั้ง
1. ติดตั้ง Aspose.Words: เพิ่มแพ็กเกจ Aspose.Words NuGet ลงในโปรเจ็กต์ .NET ของคุณ คุณสามารถทำได้โดยเรียกใช้คำสั่งต่อไปนี้ในคอนโซลตัวจัดการแพ็กเกจ:
   ```
   Install-Package Aspose.Words
   ```

2. เพิ่มเนมสเปซที่จำเป็น: ตรวจสอบให้แน่ใจว่าไฟล์โค้ดของคุณมีเนมสเปซต่อไปนี้:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
