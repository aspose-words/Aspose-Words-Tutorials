# สร้างเอกสาร Word ที่มีตารางโดยใช้ Aspose.Words
  
ตัวอย่างนี้สาธิตวิธีการสร้างเอกสาร Word ที่มีตารางโดยใช้ไลบรารี Aspose.Words สำหรับ .NET โค้ดจะเริ่มต้นเอกสารใหม่ สร้างตารางที่มี 2 แถว และจัดรูปแบบเซลล์ด้วยรูปแบบและการวางแนวต่างๆ คุณจะได้เรียนรู้การจัดการเนื้อหาเซลล์ การจัดตำแหน่ง และสร้างเอกสารที่มีโครงสร้างด้วยโปรแกรม

## คำแนะนำในการติดตั้ง:  
1. ติดตั้ง Aspose.Words สำหรับ .NET:
   - การใช้ตัวจัดการแพ็คเกจ NuGet:
     - เปิดโปรเจ็กต์ของคุณใน Visual Studio
     - คลิกขวาที่โครงการใน Solution Explorer และเลือก "จัดการแพ็คเกจ NuGet"
     - ค้นหา "Aspose.Words" และคลิก "ติดตั้ง"

   - หรือใช้คอนโซลตัวจัดการแพ็คเกจ:
     ```
     Install-Package Aspose.Words
     ```
