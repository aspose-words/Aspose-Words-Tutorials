# สร้างเอกสาร Word กับตาราง โดยใช้ Aspose.Words
  
ตัวอย่างนี้แสดงให้เห็นวิธีการสร้างเอกสาร Word กับตารางโดยใช้ห้องสมุด Aspose.Words สําหรับ .NET. คódเริ่มต้นเอกสารใหม่ สร้างตารางที่มีสองแถว และฟอร์มเมทเซลล์ด้วยสไตล์และทิศทางต่าง ๆ คุณจะได้เรียนรู้การ thao tácเนื้อหาเซลล์, การจัดสรร และสร้างเอกสารโครงสร้างแบบโปรแกรม

## คําแนะนําการติดตั้ง:  
1. ติดตั้ง Aspose.Words สําหรับ .NET:
   * การใช้งาน NuGet Package Manager:
     * เปิดโครงการของคุณใน Visual Studio
     * คลิกขวาบนโครงการใน Solution Explorer และเลือก "จัดการแพ็คเกจ NuGet"
     * ค้นหา "Aspose.Words" แล้วคลิก "ติดตั้ง".

   * หรือใช้คอนโซลผู้จัดการแพ็คเกจ
     ```
     Install-Package Aspose.Words
     ```
