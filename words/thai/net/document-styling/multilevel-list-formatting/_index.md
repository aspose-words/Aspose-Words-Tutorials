# สร้างรายการระดับหลายระดับใน Word ด้วย Aspose.Words
ตัวอย่างนี้แสดงให้เห็นวิธีการสร้างรายการหลายระดับในเอกสาร Word โดยใช้ Aspose.Words สําหรับ .NET. มันเริ่มต้นเอกสารใหม่และตัวอย่าง DocumentBuilder, ใช้รูปแบบหมายเลข, สร้างหลักและปัจจัยล่าง, และบันทึกเอกสารเป็นไฟล์ .docx. ติดตามคําแนะนําขั้นตอนเพื่อกรีดเอกสารของคุณได้อย่างมีประสิทธิภาพด้วยรายการที่มีหมายเลข.

## คําแนะนําการติดตั้ง
เพื่อใช้ Aspose.Words สําหรับ .NET ติดตามขั้นตอนนี้
1. ดาวน์โหลดห้องสมุด: ไปดู [Aspose.Words สําหรับหน้าดาวน์โหลด .NET](https://www.aspose.com/downloads/words/net). .
2. โครงการผ่าน NuGet:
   * เปิดโครงการ Visual Studio
   * ไปยัง Tools > NuGet Package Manager > การจัดการ NuGet Packages for Solution
   * ค้นหา`"พูดคําพูด"`แล้วคลิกติดตั้ง
3. หมายเลขห้องสมุดในรหัสของคุณ:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. การสมัครใบอนุญาต (ทางเลือก):  
   เพื่อใช้ฉบับเต็ม[ใช้ใบอนุญาต](https://purchase.aspose.com/temporary- license/)หรือใช้ [ทดลองใช้ฟรี](https://releases.aspose.com/words/net/). .
