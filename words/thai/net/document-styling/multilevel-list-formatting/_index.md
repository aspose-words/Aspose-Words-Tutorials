# สร้างรายการหลายระดับใน Word ด้วย Aspose.Words
ตัวอย่างนี้สาธิตวิธีการสร้างรายการหลายระดับในเอกสาร Word โดยใช้ Aspose.Words สำหรับ .NET โดยจะเริ่มต้นเอกสารใหม่และอินสแตนซ์ DocumentBuilder ใช้รูปแบบการนับเลข สร้างรายการหลักและรายการย่อย และบันทึกเอกสารเป็นไฟล์ .docx ปฏิบัติตามคำแนะนำทีละขั้นตอนเพื่อจัดรูปแบบเอกสารของคุณอย่างมีประสิทธิภาพด้วยรายการที่มีหมายเลข

## คำแนะนำในการติดตั้ง
ในการใช้ Aspose.Words สำหรับ .NET ให้ทำตามขั้นตอนเหล่านี้:
1. ดาวน์โหลดห้องสมุด : เข้าไปที่ [หน้าดาวน์โหลด Aspose.Words สำหรับ .NET](https://www-aspose.com/downloads/words/net).
2. ติดตั้งผ่าน NuGet:
   - เปิดโครงการ Visual Studio ของคุณ
   - ไปที่เครื่องมือ > ตัวจัดการแพ็กเกจ NuGet > จัดการแพ็กเกจ NuGet สำหรับโซลูชัน
   - ค้นหา `Aspose.Words` และคลิกติดตั้ง
3. อ้างอิงไลบรารีในโค้ดของคุณ:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
4. สมัครใบอนุญาต (ทางเลือก):  
   หากต้องการใช้เวอร์ชันเต็ม [ยื่นขอใบอนุญาต](https://purchase.aspose.com/temporary-license/) หรือใช้ [ทดลองใช้งานฟรี](https://releases-aspose.com/words/net/).
