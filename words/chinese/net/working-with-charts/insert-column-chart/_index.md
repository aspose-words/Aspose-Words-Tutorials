## 安装说明  

1. 下载 Aspose.Words for .NET  
   - 访问 [Aspose.Words for .NET下载页面](https://releases.aspose.com/words/net/) 并下载最新版本。  

2. 安装库  
   - 使用 Visual Studio 中的 NuGet 包管理器安装 Aspose.Words for .NET：  
     ```
     Install-Package Aspose.Words
     ```

3. 包含命名空间  
   - 将必要的命名空间添加到您的项目：  
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```

4. 设置免费试用或许可证  
   - 从 [免费试用](https://releases.aspose.com/) 或申请 [临时执照](https://purchase.aspose.com/temporary-license/) 以释放图书馆的全部功能。
