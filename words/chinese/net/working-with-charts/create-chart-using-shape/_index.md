## 安装说明  

1. 下载 Aspose.Words for .NET：  
   从官方获取最新版本 [Aspose.Words for .NET下载页面](https://releases。aspose.com/words/net/).  

2. 通过 NuGet 安装：  
   使用 NuGet 包管理器将库添加到你的项目中。在包管理器控制台中运行以下命令：  
   ```bash
   Install-Package Aspose.Words
   ```  
