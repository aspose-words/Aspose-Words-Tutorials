   访问[对于 .NET 下载页面的Aspose.Words](https://releases.aspose.com/words/net/)您可以下载最新版本的图书馆.  

2. 通过 NuGet 安装:  
   在视觉工作室中打开您的 .NET 项目,进入 NuGet 包管理器 (工具 > NuGet 包管理器 > 管理 NuGet 包解决方案),搜索Aspose.Words",并安装包.  

   另行,在包管理器控制台中运行下列命令:
