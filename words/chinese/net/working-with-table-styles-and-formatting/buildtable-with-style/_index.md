## 安装说明
1. 下载 Aspose.Words：访问 [Aspose.Words for .NET](https://www.nuget.org/packages/Aspose.Words/) NuGet 页面并下载最新版本。

2. 创建 .NET 项目：打开 Visual Studio 并创建一个新的控制台或类库项目。

3. 添加 NuGet 包：
   - 在解决方案资源管理器中右键单击您的项目。
   - 选择“管理 NuGet 包”。
   - 搜索“Aspose.Words”并单击“安装”。

4. 添加所需的使用语句：在代码文件中包括必要的使用指令：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
