## 安装说明
1. 安装 Aspose.Words for .NET：使用 NuGet 包管理器安装 Aspose.Words。您可以在包管理器控制台中运行以下命令：
   ```
   Install-Package Aspose.Words
   ```

2. 创建新项目：打开 Visual Studio 并创建一个新的 C# 项目（控制台应用程序或任何您喜欢的类型）。

3. 添加引用：确保您的项目中引用了 Aspose.Words 库。

4. 申请许可证（可选）：  
   要使用完整版本， [申请许可证](https://purchase.aspose.com/temporary-license/) 或者使用 [免费试用](https://releases。aspose.com/words/net/).
