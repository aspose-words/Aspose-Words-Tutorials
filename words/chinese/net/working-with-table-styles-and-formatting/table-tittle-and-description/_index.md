## 安装指示:

使用 Aspostion.Words for.NET 并运行此示例, 遵循这些简单步骤 :

1. 安裝套件。 net: 的 words:
   您可以很容易地通过 NuGet 为 .NET 安装 Apose.Words 。 在您的软件包管理器控制台运行以下命令 :
   ```
   Install-Package Aspose.Words
   ```
