## 装备说明:

为了开始使用 .NET 的Aspose.Words并运行这个例子,

1. 安装 .NET 的Aspose.Words:
   通过 NuGet 轻松安装 .NET 的 Aspose.Words. 在您的包管理器控制台中运行下列命令:
   ```
   Install-Package Aspose.Words
   ```
