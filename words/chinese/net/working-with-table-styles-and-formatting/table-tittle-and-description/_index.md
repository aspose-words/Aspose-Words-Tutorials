## 安装说明：

要开始使用 Aspose.Words for .NET 并运行此示例，请按照以下简单步骤操作：

1. 安装 Aspose.Words for .NET：
   您可以通过 NuGet 轻松安装 Aspose.Words for .NET。在程序包管理器控制台中运行以下命令：
   ```
   Install-Package Aspose.Words
   ```

2. 申请许可证（可选）：  
   要使用完整版本， [申请许可证](https://purchase.aspose.com/temporary-license/) 或者使用 [免费试用](https://releases。aspose.com/words/net/).
