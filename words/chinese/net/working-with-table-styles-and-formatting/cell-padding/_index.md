# 在 Aspose.Words 中创建带填充的 Word 表格单元格

## 安装说明  
要运行示例并使用 Aspose.Words for .NET，请按照以下步骤操作：  

1. 下载 Aspose.Words for .NET  
   从 [Aspose 发布页面](https://releases。aspose.com/words/net/).  

2. 通过 NuGet 包管理器安装  
   在 Visual Studio 中打开您的 .NET 项目并在 NuGet 包管理器控制台中运行以下命令：  
   ```bash
   Install-Package Aspose.Words
   ```  

3. 设置临时或许可版本  
   - 获得 [免费临时驾照](https://purchase.aspose.com/temporary-license/) 或从购买完整许可证 [Aspose购买页面](https://purchase。aspose.com/buy).  
   - 在您的代码中应用许可证：  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
