# 创建带有浮动图像的 Word 文档

本示例演示如何使用 Aspose.Words for .NET 创建 Word 文档并插入带有浮动布局的图像。提供的代码将初始化 Document 和 DocumentBuilder，插入具有指定尺寸和位置的图像，并最终保存文档。本指南非常适合希望使用 Aspose.Words 增强文档生成功能的开发人员。

## 安装说明
1. 安装 Aspose.Words：将 Aspose.Words NuGet 包添加到您的 .NET 项目中。您可以在包管理器控制台中运行以下命令来执行此操作：
   ```
   Install-Package Aspose.Words
   ```

2. 添加所需的命名空间：确保您的代码文件包含以下命名空间：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
