## 安装指示
1. 安装pose. words: 将 Apose. Words Nuget 软件包添加到您的. NET 工程中。 您可以在软件包管理器控制台运行以下命令来做到这一点 :
   ```
   Install-Package Aspose.Words
   ```

2. 添加需要的命名空间 : 确保您的代码文件包含以下命名空间 :
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
