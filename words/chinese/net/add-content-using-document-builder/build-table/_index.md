此示例演示如何使用 .NET 的 Aspose.Words 库创建包含表格的 Word 文档。代码会初始化一个新文档，构建一个包含两行的表格，并使用各种样式和方向设置单元格的格式。您将学习如何以编程方式操作单元格内容、对齐方式以及构建结构化文档。

## 安装说明：  
1. 安装 Aspose.Words for .NET：
   - 使用 NuGet 包管理器：
     - 在 Visual Studio 中打开您的项目。
     - 在解决方案资源管理器中右键单击项目并选择“管理 NuGet 包”。
     - 搜索“Aspose.Words”并单击“安装”。

   - 或者，使用包管理器控制台：
     ```
     Install-Package Aspose.Words
     ```
