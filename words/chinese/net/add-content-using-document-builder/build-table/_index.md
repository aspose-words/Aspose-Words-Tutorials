# 使用Aspose.Words创建一个 Word 文档
  
这个例子展示了如何使用 .NET 的Aspose.Words库创建一个 Word 文档.该代码初始化了一个新文档,构建一个两个行的表格,并将各种风格和方向的细胞格式化.您将学习如何操作细胞内容,对齐和编程结构文档.

## 装备说明:  
1. 安装 .NET 的Aspose.Words:
   * 使用 NuGet 包管理器:
     * 在视觉工作室打开你的项目.
     * 在解决方案探险器中右键点击项目,然后选择"管理NuGet包".
     * 搜索Aspose.Words并点击"安装".

   * 或使用包管理器控制台:
     ```
     Install-Package Aspose.Words
     ```
