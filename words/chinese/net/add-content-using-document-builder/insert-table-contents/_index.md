## 装备说明
1. 要求:确保您的机器上安装了 .NET.
2. 安装Aspose.Words: 
   * 使用 NuGet 包管理器控制台:
     ```
     Install-Package Aspose.Words
     ```
   * 或在 NuGet 包管理器 GUI 中搜索 "Aspose.Words"
