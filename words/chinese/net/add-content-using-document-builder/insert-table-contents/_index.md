## 安装说明
1. 先决条件：确保您的机器上安装了 .NET。
2. 安装 Aspose.Words： 
   - 使用 NuGet 包管理器控制台：
     ```
     Install-Package Aspose.Words
     ```
   - 或者，在 NuGet 包管理器 GUI 中搜索“Aspose.Words”。
