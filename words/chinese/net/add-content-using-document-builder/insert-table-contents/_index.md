## 安装指示
1. 先决条件:确保您在机器上安装了.NET。
2. 安裝套件。 文字 : 
   * 使用 NuGet 软件包管理器控制台:
     ```
     Install-Package Aspose.Words
     ```
   * 或者,在 NuGet 软件包管理器 GUI 中搜索“ 对象. words ” 。
