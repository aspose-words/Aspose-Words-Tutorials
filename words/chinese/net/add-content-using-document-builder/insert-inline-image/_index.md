## 安装指示
1. 安裝套件。 net: 的 words:
   * 在视觉工作室使用 NuGet 软件包管理器 。
   * 在软件包管理器控制台运行以下命令 :
     ```
     Install-Package Aspose.Words
     ```

   * 创建新的.NET 工程或打开已有的工程 。
   * 在代码文件顶部添加使用指令的必要命令 :
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
