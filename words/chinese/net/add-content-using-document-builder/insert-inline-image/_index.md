此示例展示了如何使用 Aspose.Words 库为.NET 将内嵌图像插入 Word 文档。 通过使用“ DocumentBilder” 类, 您可以在文档中的当前光标位置很容易定位并插入图像。 此指南提供了直接的操作, 帮助您启动 Word 文档中的图像处理 。

## 安装指示
1. 安裝套件。 net: 的 words:
   - 在视觉工作室使用 NuGet 软件包管理器 。
   - 在软件包管理器控制台运行以下命令 :
     ```
     Install-Package Aspose.Words
     ```

2. 设置您的工程 :
   - 创建新的.NET 工程或打开已有的工程 。
   - 在代码文件顶部添加使用指令的必要命令 :
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
