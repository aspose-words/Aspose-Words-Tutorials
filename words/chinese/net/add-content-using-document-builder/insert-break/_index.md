# 使用 Aspose.Words 创建多页 Word 文档

本示例演示如何使用 Aspose.Words for .NET 创建 Word 文档，并允许您在多个页面上添加内容。它展示了如何初始化新文档、使用 DocumentBuilder 插入文本和分页符，以及最终将文档保存为 .docx 文件。非常适合希望在 .NET 应用程序中自动生成文档的开发人员。

## 安装说明
1. 安装 Aspose.Words for .NET：使用 NuGet 包管理器安装 Aspose.Words。在包管理器控制台中运行以下命令：
```
Install-Package Aspose.Words
```
