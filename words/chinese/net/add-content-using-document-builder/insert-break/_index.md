## 安装指示
1. 安裝套件. NET 的 words: 使用 NuGet 软件包管理器安装 Apose. words。 在软件包管理器控制台运行以下命令 :
   ```
   Install-Package Aspose.Words
   ```

2. 创建一个新工程: 打开视觉工作室, 创建一个新的 .NET 工程, 或者打开一个已有的工程 。

3. 添加引用: 确保您的工程引用 Aspose.Words DLL 安装后引用 。

4. 写入代码 : 使用所提供的代码示例创建多页 Word 文档 。

5. 运行工程: 执行生成 Word 文档的工程。 请检查保存的. docx 文件的输出目录 。 

6. 审查输出: 打开生成的文档, 以核实文本是否正确放置在多个页面上 。
