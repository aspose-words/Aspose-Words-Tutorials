## 安装说明
1. 安装 Aspose.Words：在 Visual Studio 中使用 NuGet 包管理器。
   - 在 Visual Studio 中打开您的项目。
   - 在解决方案资源管理器中右键单击您的项目。
   - 选择管理 NuGet 包。
   - 搜索 `Aspose.Words` 并安装它。

2. 添加使用指令：在您的代码文件中，包含以下指令以使用 Aspose.Words 类：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
