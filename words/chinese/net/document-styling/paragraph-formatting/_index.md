## 安装说明
1. 下载 Aspose.Words for .NET：访问 [Aspose 网站](https://www.aspose.com/downloads/words/net) 下载最新版本的 Aspose.Words for .NET。

2. 添加引用：在您的 .NET 项目中，右键单击解决方案资源管理器中的项目，选择添加 > 引用，然后浏览以添加 Aspose.Words DLL 文件。

3. 使用 NuGet 包（可选）：或者，您可以使用以下命令通过 NuGet 包管理器安装 Aspose.Words：
   ```
   Install-Package Aspose.Words
   ```

4. 导入命名空间：将以下 using 语句添加到代码文件中：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

5. 申请许可证（可选）：  
   要使用完整版本， [申请许可证](https://purchase.aspose.com/temporary-license/) 或者使用 [免费试用](https://releases。aspose.com/words/net/).
