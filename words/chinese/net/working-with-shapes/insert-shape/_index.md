# 使用 Aspose.Words 在 Word 文档中插入形状
本指南演示如何使用 Aspose.Words for .NET 在 Word 文档中插入和操作形状（例如文本框）。您将学习如何绝对或相对定位形状、设置旋转角度，以及如何使用以下工具将文档保存为 ISO 29500:2008 过渡 DOCX 格式： `OoxmlSaveOptions`。此示例简化了 Word 文档中形状的处理，使开发人员能够轻松创建具有视觉吸引力的内容。

## 安装说明  
1. 下载 Aspose.Words for .NET：  
   从获取最新版本的 Aspose.Words for .NET [下载页面](https://releases。aspose.com/words/net/).  

2. 通过 NuGet 安装：  
   在 Visual Studio 中打开您的项目并使用 NuGet 包管理器安装库：  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. 添加命名空间引用：  
   在您的代码中，包含 Aspose.Words 的命名空间：  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. 激活许可证（可选）：  
   如果您拥有有效的许可证，请将其应用于您的项目以避免限制。您可以 [购买许可证](https://purchase.aspose.com/buy) 或者得到 [临时执照](https://purchase。aspose.com/temporary-license/).
