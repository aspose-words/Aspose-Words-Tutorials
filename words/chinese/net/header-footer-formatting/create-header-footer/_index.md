# 使用 Aspose.Words 创建带有页眉和页脚的 Word 文档

此示例演示如何使用 Aspose.Words for .NET 创建带有主页眉和页脚的 Word 文档。本指南将引导您完成初始化新文档、移动到页眉和页脚部分、添加自定义文本以及将文档保存为 DOCX 格式的步骤。凭借其强大的 DocumentBuilder 类，Aspose.Words 使 Word 文档操作变得无缝且高效。无论您是添加品牌元素还是页码，此示例都为您以编程方式自定义 Word 文档奠定了基础。

## 安装说明  
1. 下载 Aspose.Words for .NET：从 [Aspose 发布页面](https://releases。aspose.com/words/net/).  
2. 安装 Aspose.Words：  
   - 使用 NuGet 包管理器：  
     ```bash  
     Install-Package Aspose.Words  
     ```  
   - 或者从下载的包中手动引用 DLL。  
3. 申请许可证（可选）：获取 [临时执照](https://purchase.aspose.com/temporary-license/) 或完整许可证 [Aspose 购买](https://purchase.aspose.com/buy) 并将其应用到您的项目中：  
   ```csharp  
   License license = new License();  
   license.SetLicense("Aspose.Words.lic");  
   ```
