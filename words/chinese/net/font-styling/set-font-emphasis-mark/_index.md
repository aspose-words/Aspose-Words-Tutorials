# 使用 Aspose.Words 在 Word 文档中设置强调标记

在 Word 文档中设置强调符号是突出显示特定文本的绝佳方法，可确保其在内容中脱颖而出。使用 Aspose.Words for .NET，您只需几行代码即可轻松添加强调符号，例如实心圆。此示例演示了如何使用 `DocumentBuilder` 类用于操作 Word 文档、使用强调符号格式化文本，并将结果保存为 DOCX 格式。遵循本指南，以专业的精度简化您的文档格式化任务。

## 安装说明  
要运行提供的代码示例并使用 Aspose.Words for .NET，请按照以下步骤操作：  

1. 下载 Aspose.Words for .NET：  
   - 从 [Aspose 版本](https://releases.aspose.com/words/net/) 页。  

2. 安装库：  
   - 通过 Visual Studio 中的 NuGet 包管理器安装它：  
     - 打开 Visual Studio。  
     - 导航到工具>NuGet 包管理器>管理解决方案的 NuGet 包。  
     - 搜索 `Aspose.Words` 并点击“安装”。  

   - 或者，使用 NuGet 包管理器控制台：  
     ```shell
     Install-Package Aspose.Words
     ```  

3. 设置临时许可证（可选）：  
   - 如需不受限制地使用，请获取 [免费临时驾照](https://purchase。aspose.com/temporary-license/).  
   - 在您的项目中应用许可证：  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
