# 使用 Aspose.Words 创建 Word 文档（Java）

## 快速答案
- **Java Word 处理的主要库是什么？** Aspose.Words for Java。  
- **可以将 Word 转换为 ODT 吗？** 可以 – 使用 `Save` 方法并指定 `SaveFormat.ODT`。  
- **支持 SVG 导出吗？** 完全支持，使用 `SaveFormat.SVG` 将任意文档页导出为 SVG。

## 常见问题

**Q: 如何在 Java 中从头创建一个新的 Word 文档？**  
A: 实例化 `Document` 类，添加 `Section`，然后使用 `Paragraph` 和 `Run` 对象填充内容。这是以编程方式 **create word document java** 的核心方法。

**Q: Aspose.Words 能直接将 Word 文件转换为 ODT 吗？**  
A: 能。加载源文档后调用 `save("output.odt", SaveFormat.ODT)`。

**Q: 是否支持将文档页导出为 SVG？**  
A: 完全支持。保存时使用 `SaveFormat.SVG`；每一页都会生成单独的 SVG 文件。

**Q: rtf 到 docx 转换的最简方法是什么？**  
A: 使用 `new Document("input.rtf")` 加载 RTF 文件，然后通过 `SaveFormat.DOCX` 保存为 DOCX。

**Q: 如何在转换 CHM 为 HTML 时保留内部链接？**  
A: 使用 `Document` 加载 CHM 文件并保存为 HTML；Aspose.Words 会自动重写链接。
