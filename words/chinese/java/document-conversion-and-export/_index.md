## 常见问题

**Q: 可以在 Linux 服务器上将 DOCX 转换为 PDF 吗？**  
A: 可以。Aspose.Words for Java 与平台无关，适用于任何安装了受支持 JRE 的操作系统。

**Q: 服务器上需要安装 Microsoft Office 吗？**  
A: 不需要。转换完全由 Aspose.Words 库完成。

**Q: 如何在转换过程中保留自定义字体？**  
A: 使用 `PdfSaveOptions.setEmbedFullFonts(true)` 将所有使用的字体嵌入生成的 PDF。

**Q: 能否转换受密码保护的 DOCX 文件？**  
A: 完全可以。在保存为 PDF 之前，使用 `LoadOptions.setPassword("yourPassword")` 加载文档。

**Q: 批量转换大量文档的推荐做法是什么？**  
A: 创建一个可复用的方法来加载、转换并释放每个文档，并在线程池中并行执行以提升处理速度。
