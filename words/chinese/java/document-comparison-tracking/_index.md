## 常见问题

**Q: 我可以在 Web 应用程序中使用 aspose words java 吗？**  
A: 可以。该库可在 servlet 容器、Spring Boot 以及任何基于 Java 的 Web 框架中使用。

**Q: 比较时如何处理图像？**  
A: 图像通过其二进制内容进行比较；如果不同，会添加一个修订，指示图像更改。

**Q: 在比较时是否可以忽略特定元素（例如页眉）？**  
A: 当然。您可以配置 `ComparisonOptions` 来忽略页眉、页脚，甚至特定样式。

**Q: 哪些输出格式会保留跟踪的更改？**  
A: DOCX、DOC 和 RTF 会保留修订标记。导出为 PDF 时，您可以选择以可视方式呈现更改。

**Q: aspose words java 是否支持批量比较大量文档？**  
A: 支持。通过遍历文档对并复用单个 `Comparison` 实例，您可以高效地处理大批量文档。
