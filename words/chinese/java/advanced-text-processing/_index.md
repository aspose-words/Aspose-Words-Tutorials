## 常见问题

**Q: 我可以以编程方式更改页码格式（例如，从罗马数字改为阿拉伯数字）吗？**  
A: 是的，您可以在 `PageNumber` 字段上设置 `PageNumberStyle` 属性为任何受支持的格式。

**Q: 我如何在特定页面（例如标题页）跳过页码？**  
A: 插入带有 `\* MERGEFORMAT` 开关的 `PageNumber` 字段，并在该节的页眉/页脚上设置 `Suppress` 属性。

**Q: 能否自动为每章重置页码？**  
A: 完全可以。使用 `Section` 类为每章创建新节，并启用 `PageSetup.RestartPageNumbering`。

**Q: API 能高效处理大文档吗？**  
A: Aspose.Words 以流式方式处理文档，即使是包含数千页的文件也能最大限度地降低内存使用。

**Q: 生产使用有哪些授权选项？**  
A: 您可以先使用临时许可证进行评估，然后升级为永久许可证或订阅许可证以进行商业部署。
