# Δημιουργήστε τραπέζι με στυλ

## Εισαγωγή

Η δημιουργία κομψών, επαγγελματικών εγγράφων συχνά απαιτεί κάτι περισσότερο από απλό κείμενο. Οι πίνακες είναι ένας φανταστικός τρόπος οργάνωσης δεδομένων, αλλά η ελκυστική τους εμφάνιση είναι μια εντελώς διαφορετική πρόκληση. Μπείτε στο Aspose.Words για .NET! Σε αυτό το σεμινάριο, θα εμβαθύνουμε στο πώς να δημιουργήσετε έναν πίνακα με στυλ, κάνοντας τα έγγραφά σας στο Word να φαίνονται κομψά και επαγγελματικά.

## Προαπαιτούμενα

Πριν προχωρήσουμε στον αναλυτικό οδηγό, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

1. Aspose.Words για .NET: Αν δεν το έχετε κάνει ήδη, κατεβάστε και εγκαταστήστε [Aspose.Words για .NET](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Θα πρέπει να έχετε ρυθμίσει ένα περιβάλλον ανάπτυξης. Το Visual Studio είναι μια εξαιρετική επιλογή για αυτό το σεμινάριο.
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να παρακολουθείτε πιο εύκολα.

## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Αυτό θα σας δώσει πρόσβαση στις κλάσεις και τις μεθόδους που απαιτούνται για τον χειρισμό εγγράφων του Word.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## Βήμα 1: Δημιουργία νέου εγγράφου και DocumentBuilder

Πρώτα απ 'όλα, πρέπει να δημιουργήσετε ένα νέο έγγραφο και ένα `DocumentBuilder` αντικείμενο. Αυτό `DocumentBuilder` θα σας βοηθήσει να κατασκευάσετε τον πίνακα στο έγγραφό σας.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 2: Ξεκινήστε την κατασκευή του πίνακα

Τώρα που έχουμε έτοιμα το έγγραφό μας και το εργαλείο δημιουργίας, ας ξεκινήσουμε τη δημιουργία του πίνακα.

```csharp
Table table = builder.StartTable();
```

## Βήμα 3: Εισαγωγή της πρώτης γραμμής

Ένας πίνακας χωρίς γραμμές είναι απλώς μια κενή δομή. Πρέπει να εισαγάγουμε τουλάχιστον μία γραμμή πριν μπορέσουμε να ορίσουμε οποιαδήποτε μορφοποίηση πίνακα.

```csharp
builder.InsertCell();
```

## Βήμα 4: Ορισμός του στυλ πίνακα

Με την εισαγωγή του πρώτου κελιού, ήρθε η ώρα να προσθέσουμε κάποιο στυλ στον πίνακά μας. Θα χρησιμοποιήσουμε το `StyleIdentifier` για να εφαρμόσετε ένα προκαθορισμένο στυλ.

```csharp
// Ορίστε το στυλ πίνακα που χρησιμοποιείται με βάση το μοναδικό αναγνωριστικό στυλ
table.StyleIdentifier = StyleIdentifier.MediumShading1Accent1;
```

## Βήμα 5: Ορισμός επιλογών στυλ

Οι επιλογές στυλ πίνακα καθορίζουν ποια μέρη του πίνακα θα μορφοποιηθούν. Για παράδειγμα, μπορούμε να επιλέξουμε να μορφοποιήσουμε την πρώτη στήλη, τις ζώνες γραμμών και την πρώτη γραμμή.

```csharp
// Εφαρμογή των χαρακτηριστικών που θα πρέπει να μορφοποιηθούν από το στυλ
table.StyleOptions = TableStyleOptions.FirstColumn | TableStyleOptions.RowBands | TableStyleOptions.FirstRow;
```

## Βήμα 6: Προσαρμογή πίνακα ώστε να ταιριάζει με τα περιεχόμενα

Για να διασφαλίσουμε ότι το τραπέζι μας φαίνεται καθαρό και τακτοποιημένο, μπορούμε να χρησιμοποιήσουμε το `AutoFit` μέθοδος για την προσαρμογή του πίνακα ώστε να ταιριάζει με το περιεχόμενό του.

```csharp
table.AutoFit(AutoFitBehavior.AutoFitToContents);
```

## Βήμα 7: Εισαγωγή δεδομένων στον πίνακα

Τώρα είναι η ώρα να γεμίσουμε τον πίνακά μας με κάποια δεδομένα. Θα ξεκινήσουμε με τη γραμμή κεφαλίδας και στη συνέχεια θα προσθέσουμε κάποια δείγματα δεδομένων.

### Εισαγωγή γραμμής κεφαλίδας

```csharp
builder.Writeln("Item");
builder.CellFormat.RightPadding = 40;
builder.InsertCell();
builder.Writeln("Quantity (kg)");
builder.EndRow();
```

#### Εισαγωγή γραμμών δεδομένων

```csharp
builder.InsertCell();
builder.Writeln("Apples");
builder.InsertCell();
builder.Writeln("20");
builder.EndRow();

builder.InsertCell();
builder.Writeln("Bananas");
builder.InsertCell();
builder.Writeln("40");
builder.EndRow();

builder.InsertCell();
builder.Writeln("Carrots");
builder.InsertCell();
builder.Writeln("50");
builder.EndRow();
```

## Βήμα 8: Αποθήκευση του εγγράφου

Αφού εισαγάγετε όλα τα δεδομένα, το τελευταίο βήμα είναι η αποθήκευση του εγγράφου.

```csharp
doc.Save(dataDir + "WorkingWithTableStylesAndFormatting.BuildTableWithStyle.docx");
```
