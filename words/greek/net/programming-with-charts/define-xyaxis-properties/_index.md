# Ορισμός ιδιοτήτων άξονα XY σε ένα διάγραμμα

## Εισαγωγή

Τα γραφήματα είναι ένα ισχυρό εργαλείο για την οπτικοποίηση δεδομένων. Όταν χρειάζεται να δημιουργήσετε επαγγελματικά έγγραφα με δυναμικά γραφήματα, το Aspose.Words για .NET είναι μια ανεκτίμητη βιβλιοθήκη. Αυτό το άρθρο θα σας καθοδηγήσει στη διαδικασία ορισμού των ιδιοτήτων του άξονα XY σε ένα γράφημα χρησιμοποιώντας το Aspose.Words για .NET, αναλύοντας κάθε βήμα για να διασφαλιστεί η σαφήνεια και η ευκολία κατανόησης.

## Προαπαιτούμενα

Πριν ξεκινήσετε την κωδικοποίηση, υπάρχουν μερικές προϋποθέσεις που πρέπει να έχετε στη διάθεσή σας:

1. Aspose.Words για .NET: Βεβαιωθείτε ότι έχετε τη βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να [κατεβάστε το εδώ](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Χρειάζεστε ένα ολοκληρωμένο περιβάλλον ανάπτυξης (IDE) όπως το Visual Studio.
3. .NET Framework: Βεβαιωθείτε ότι το περιβάλλον ανάπτυξής σας έχει ρυθμιστεί για ανάπτυξη .NET.
4. Βασικές γνώσεις C#: Αυτός ο οδηγός προϋποθέτει ότι έχετε βασική κατανόηση του προγραμματισμού C#.

## Εισαγωγή χώρων ονομάτων

Αρχικά, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας. Αυτό διασφαλίζει ότι έχετε πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που απαιτούνται για τη δημιουργία και τον χειρισμό εγγράφων και γραφημάτων.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Drawing.Charts;
```

Θα αναλύσουμε τη διαδικασία σε απλά βήματα, καθένα από τα οποία εστιάζει σε ένα συγκεκριμένο μέρος του ορισμού των ιδιοτήτων του άξονα XY σε ένα γράφημα.

## Βήμα 1: Αρχικοποίηση του Εγγράφου και του DocumentBuilder

Αρχικά, πρέπει να αρχικοποιήσετε ένα νέο έγγραφο και ένα `DocumentBuilder` αντικείμενο. Το `DocumentBuilder` βοηθά στην εισαγωγή περιεχομένου στο έγγραφο.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## Βήμα 2: Εισαγωγή γραφήματος

Στη συνέχεια, θα εισαγάγετε ένα γράφημα στο έγγραφο. Σε αυτό το παράδειγμα, θα χρησιμοποιήσουμε ένα γράφημα περιοχής. Μπορείτε να προσαρμόσετε τις διαστάσεις του γραφήματος όπως απαιτείται.

```csharp
// Εισαγωγή γραφήματος
Shape shape = builder.InsertChart(ChartType.Area, 432, 252);
Chart chart = shape.Chart;
```

## Βήμα 3: Διαγραφή προεπιλεγμένης σειράς και προσθήκη προσαρμοσμένων δεδομένων

Από προεπιλογή, το γράφημα θα έχει ορισμένες προκαθορισμένες σειρές. Θα τις διαγράψουμε και θα προσθέσουμε τις προσαρμοσμένες σειρές δεδομένων μας.

```csharp
chart.Series.Clear();
chart.Series.Add("Aspose Series 1",
	new DateTime[]
	{
		new DateTime(2002, 01, 01), new DateTime(2002, 06, 01), new DateTime(2002, 07, 01),
		new DateTime(2002, 08, 01), new DateTime(2002, 09, 01)
	},
	new double[] { 640, 320, 280, 120, 150 });
```

## Βήμα 4: Ορίστε τις ιδιότητες του άξονα X

Τώρα, ήρθε η ώρα να ορίσετε τις ιδιότητες για τον άξονα X. Αυτό περιλαμβάνει τον ορισμό του τύπου κατηγορίας, την προσαρμογή της διασταύρωσης του άξονα και την προσαρμογή των σημαδιών υποδιαίρεσης και των ετικετών.

```csharp
ChartAxis xAxis = chart.AxisX;
xAxis.CategoryType = AxisCategoryType.Category;
xAxis.Crosses = AxisCrosses.Custom;
xAxis.CrossesAt = 3; // Μετρημένο σε μονάδες απεικόνισης του άξονα Y (εκατοντάδες).
xAxis.ReverseOrder = true;
xAxis.MajorTickMark = AxisTickMark.Cross;
xAxis.MinorTickMark = AxisTickMark.Outside;
xAxis.TickLabelOffset = 200;
```

## Βήμα 5: Ορίστε τις ιδιότητες του άξονα Y

Ομοίως, θα ορίσετε τις ιδιότητες για τον άξονα Y. Αυτό περιλαμβάνει τον ορισμό της θέσης της ετικέτας υποδιαίρεσης, των κύριων και δευτερευόντων μονάδων, της μονάδας εμφάνισης και της κλιμάκωσης.

```csharp
ChartAxis yAxis = chart.AxisY;
yAxis.TickLabelPosition = AxisTickLabelPosition.High;
yAxis.MajorUnit = 100;
yAxis.MinorUnit = 50;
yAxis.DisplayUnit.Unit = AxisBuiltInUnit.Hundreds;
yAxis.Scaling.Minimum = new AxisBound(100);
yAxis.Scaling.Maximum = new AxisBound(700);
```

## Βήμα 6: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο. Αυτό θα δημιουργήσει το έγγραφο του Word με το προσαρμοσμένο γράφημα.

```csharp
doc.Save(dataDir + "WorkingWithCharts.DefineXYAxisProperties.docx");
```
