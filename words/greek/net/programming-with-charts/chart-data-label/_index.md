## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας σε C#. Αυτό το βήμα είναι κρίσιμο, καθώς διασφαλίζει ότι έχετε πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που παρέχονται από το Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Saving;
using Aspose.Words.Charts;
```

## Βήμα 1: Αρχικοποίηση του Εγγράφου και του DocumentBuilder

Για να δημιουργήσουμε και να χειριστούμε έγγραφα Word, πρέπει πρώτα να αρχικοποιήσουμε μια παρουσία του `Document` τάξη και ένα `DocumentBuilder` αντικείμενο.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

### Εξήγηση

- Έγγραφο εγγράφου: Δημιουργεί μια νέα παρουσία της κλάσης Document.
- Εργαλείο δημιουργίας εγγράφων: Το εργαλείο δημιουργίας εγγράφων βοηθά στην εισαγωγή περιεχομένου στο αντικείμενο εγγράφου.

## Βήμα 2: Εισαγωγή γραφήματος

Στη συνέχεια, θα εισαγάγουμε ένα ραβδόγραμμα στο έγγραφο χρησιμοποιώντας το `DocumentBuilder` αντικείμενο.

```csharp
Shape shape = builder.InsertChart(ChartType.Bar, 432, 252);
Chart chart = shape.Chart;
```

### Εξήγηση

- Σχήμα σχήματος: Αντιπροσωπεύει το γράφημα ως σχήμα στο έγγραφο.
- builder.InsertChart(ChartType.Bar, 432, 252): Εισάγει ένα γράφημα ράβδων με καθορισμένες διαστάσεις.

## Βήμα 3: Πρόσβαση στη σειρά γραφημάτων

Για να προσαρμόσουμε τις ετικέτες δεδομένων, πρέπει πρώτα να έχουμε πρόσβαση στις σειρές στο γράφημα.

```csharp
ChartSeries series0 = shape.Chart.Series[0];
```

### Εξήγηση

- ChartSeries series0: Ανακτά την πρώτη σειρά του γραφήματος, την οποία θα προσαρμόσουμε.

## Βήμα 4: Προσαρμογή ετικετών δεδομένων

Οι ετικέτες δεδομένων μπορούν να προσαρμοστούν ώστε να εμφανίζουν διάφορες πληροφορίες. Θα διαμορφώσουμε τις ετικέτες ώστε να εμφανίζουν το κλειδί υπομνήματος, το όνομα της σειράς και την τιμή, ενώ θα αποκρύπτουν το όνομα και το ποσοστό της κατηγορίας.

```csharp
ChartDataLabelCollection labels = series0.DataLabels;
labels.ShowLegendKey = true;
labels.ShowLeaderLines = true;
labels.ShowCategoryName = false;
labels.ShowPercentage = false;
labels.ShowSeriesName = true;
labels.ShowValue = true;
labels.Separator = "/";
```

### Εξήγηση

- Ετικέτες ChartDataLabelCollection: Πρόσβαση στις ετικέτες δεδομένων της σειράς.
- labels.ShowLegendKey: Εμφανίζει το κλειδί υπομνήματος.
- labels.ShowLeaderLines: Εμφανίζει γραμμές οδηγού για ετικέτες δεδομένων που βρίσκονται πολύ έξω από τα σημεία δεδομένων.
- labels.ShowCategoryName: Αποκρύπτει το όνομα της κατηγορίας.
- labels.ShowPercentage: Αποκρύπτει την τιμή ποσοστού.
- labels.ShowSeriesName: Εμφανίζει το όνομα της σειράς.
- labels.ShowValue: Εμφανίζει την τιμή των σημείων δεδομένων.
- labels.Separator: Ορίζει το διαχωριστικό για τις ετικέτες δεδομένων.

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο.

```csharp
doc.Save(dataDir + "WorkingWithCharts.ChartDataLabel.docx");
```

### Εξήγηση

- doc.Save: Αποθηκεύει το έγγραφο με το καθορισμένο όνομα στον παρεχόμενο κατάλογο.
