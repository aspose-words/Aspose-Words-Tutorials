## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, ας ρυθμίσουμε το έργο μας και ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτοί οι χώροι ονομάτων είναι κρίσιμοι για την πρόσβαση στη λειτουργικότητα του Aspose.Words.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.ReportingServices;
```

## Βήμα 1: Δημιουργία νέου εγγράφου


Η διαδρομή ξεκινά με τη δημιουργία ενός νέου εγγράφου και την αρχικοποίηση ενός `DocumentBuilder`Το `DocumentBuilder` Η κλάση παρέχει ένα σύνολο μεθόδων για τον εύκολο χειρισμό του περιεχομένου του εγγράφου.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Δημιουργήστε ένα νέο έγγραφο
Document doc = new Document();

// Αρχικοποίηση του DocumentBuilder
DocumentBuilder builder = new DocumentBuilder(doc);
```

### Εξήγηση

Σε αυτό το βήμα, έχουμε ρυθμίσει το έγγραφο και το εργαλείο δημιουργίας που θα χρησιμοποιήσουμε για την εισαγωγή και τη μορφοποίηση του περιεχομένου μας. `dataDir` Η μεταβλητή περιέχει τη διαδρομή όπου θα αποθηκεύσουμε το τελικό μας έγγραφο.

## Βήμα 2: Εισαγωγή γραφήματος

Στη συνέχεια, θα προσθέσουμε ένα γράφημα πίτας στο έγγραφό μας. `InsertChart` μέθοδος του `DocumentBuilder` Το μάθημα το κάνει αυτό εξαιρετικά εύκολο.

```csharp
// Εισαγωγή κυκλικού γραφήματος
Shape shape = builder.InsertChart(ChartType.Pie, 432, 252);

// Πρόσβαση στο αντικείμενο γραφήματος
Chart chart = shape.Chart;
```

### Εξήγηση

Εδώ, εισάγουμε ένα γράφημα πίτας στο έγγραφό μας. Το `InsertChart` Η μέθοδος απαιτεί τον τύπο, το πλάτος και το ύψος του γραφήματος ως παραμέτρους. Μετά την εισαγωγή του γραφήματος, έχουμε πρόσβαση στο αντικείμενο του γραφήματος για περαιτέρω χειρισμό του.

## Βήμα 3: Προσαρμόστε τη σειρά γραφημάτων

Τώρα, θα διαγράψουμε τυχόν υπάρχουσες σειρές στο γράφημα και θα προσθέσουμε τις προσαρμοσμένες σειρές μας. Αυτές οι σειρές θα αντιπροσωπεύουν τα σημεία δεδομένων μας.

```csharp
// Διαγραφή υπάρχουσας σειράς γραφημάτων
chart.Series.Clear();

// Προσθήκη νέας σειράς στο γράφημα
ChartSeries series = chart.Series.Add("Aspose Series 1",
    new string[] { "Category 1", "Category 2", "Category 3" },
    new double[] { 2.7, 3.2, 0.8 });
```

### Εξήγηση

Σε αυτό το βήμα, βεβαιωνόμαστε ότι το γράφημά μας είναι άδειο, διαγράφοντας τυχόν προϋπάρχουσες σειρές. Στη συνέχεια, προσθέτουμε μια νέα σειρά με προσαρμοσμένες κατηγορίες και τιμές, οι οποίες θα εμφανίζονται στο γράφημα πίτας.

## Βήμα 4: Ορισμός προεπιλεγμένων επιλογών για ετικέτες δεδομένων

Οι ετικέτες δεδομένων είναι ζωτικής σημασίας για να κάνετε το γράφημά σας κατατοπιστικό. Θα ορίσουμε επιλογές για να εμφανίσουμε το ποσοστό, την τιμή και θα προσαρμόσουμε το διαχωριστικό.

```csharp
// Πρόσβαση στη συλλογή ετικετών δεδομένων
ChartDataLabelCollection labels = series.DataLabels;

// Ορισμός επιλογών ετικέτας δεδομένων
labels.ShowPercentage = true;
labels.ShowValue = true;
labels.ShowLeaderLines = false;
labels.Separator = " - ";
```

### Εξήγηση

Εδώ, έχουμε πρόσβαση στο `DataLabels` ιδιότητα της σειράς μας για να προσαρμόσετε την εμφάνιση και τις πληροφορίες που εμφανίζονται σε κάθε ετικέτα δεδομένων. Έχουμε επιλέξει να εμφανίζουμε τόσο το ποσοστό όσο και την τιμή, να αποκρύπτουμε τις γραμμές οδηγού και να ορίζουμε ένα προσαρμοσμένο διαχωριστικό.

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, θα αποθηκεύσουμε το έγγραφό μας στον καθορισμένο κατάλογο. Αυτό το βήμα διασφαλίζει ότι όλες οι αλλαγές μας θα εγγραφούν σε ένα αρχείο.

```csharp
// Αποθήκευση του εγγράφου
doc.Save(dataDir + "WorkingWithCharts.DefaultOptionsForDataLabels.docx");
```

### Εξήγηση

Σε αυτό το τελευταίο βήμα, αποθηκεύουμε το έγγραφό μας χρησιμοποιώντας το `Save` μέθοδος. Το έγγραφο θα αποθηκευτεί στον κατάλογο που καθορίζεται από `dataDir`, με το όνομα "WorkingWithCharts.DefaultOptionsForDataLabels.docx".
