# Δημιουργία ενός εγγράφου λέξεων με τον πίνακα περιεχομένων
Αυτό το παράδειγμα δείχνει πώς να χρησιμοποιήσετε Aspose.Words για .NET για να δημιουργήσετε ένα έγγραφο Word που περιλαμβάνει έναν διαδραστικό πίνακα περιεχομένων (TOC) που παράγεται χρησιμοποιώντας την κατηγορία DocumentBuilder. Ο κώδικας δείχνει πώς να εισάγετε τίτλους διαφόρων επιπέδων, να τα διαμορφώσετε, και να αποθηκεύσετε το έγγραφο που προκύπτει ως .docx αρχείο. Ακολουθήστε τα παρακάτω βήματα για μια εύκολη εφαρμογή.

## Οδηγίες εγκατάστασης
1. Προαπαιτούμενα: Βεβαιωθείτε ότι έχετε .NET εγκατασταθεί στο μηχάνημα σας.
2. Εγκαταστήστε το Aspose.Words: 
   * Χρήση της κονσόλας NuGet Package Manager:
     ```
     Install-Package Aspose.Words
     ```
   * Ή, αναζήτηση για "Aspose.Words" στο NuGet Package Manager GUI.
