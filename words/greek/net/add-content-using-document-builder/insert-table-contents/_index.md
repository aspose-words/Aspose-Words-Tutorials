# Δημιουργήστε ένα έγγραφο Word με πίνακα περιεχομένων
Αυτό το παράδειγμα δείχνει πώς να χρησιμοποιήσετε το Aspose.Words για .NET για να δημιουργήσετε ένα έγγραφο του Word που περιλαμβάνει έναν διαδραστικό Πίνακα Περιεχομένων (TOC) που δημιουργείται χρησιμοποιώντας την κλάση DocumentBuilder. Ο κώδικας δείχνει πώς να εισαγάγετε επικεφαλίδες διαφόρων επιπέδων, να τις μορφοποιήσετε και να αποθηκεύσετε το έγγραφο που προκύπτει ως αρχείο .docx. Ακολουθήστε τα παρακάτω βήματα για μια εύκολη υλοποίηση.

## Οδηγίες εγκατάστασης
1. Προαπαιτούμενα: Βεβαιωθείτε ότι έχετε εγκατεστημένο το .NET στον υπολογιστή σας.
2. Εγκατάσταση Aspose.Words: 
   - Χρησιμοποιήστε την Κονσόλα Διαχείρισης Πακέτων NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - Ή, αναζητήστε "Aspose.Words" στο γραφικό περιβάλλον χρήστη του NuGet Package Manager.
