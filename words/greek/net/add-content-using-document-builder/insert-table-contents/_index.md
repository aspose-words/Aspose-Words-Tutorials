# Δημιουργήστε ένα έγγραφο Word με πίνακα περιεχομένου
Αυτό το παράδειγμα δείχνει πώς να χρησιμοποιήσετε Aspose.Words για το .NET για τη δημιουργία ενός εγγράφου Word που περιλαμβάνει ένα διαδραστικό Πίνακα Περιεχομένου (TOC) που δημιουργείται χρησιμοποιώντας την κατηγορία DocumentBuilder. Ο κώδικας απεικονίζει πώς να εισαγάγετε τίτλους διαφόρων επιπέδων, να τα μορφοποιείτε και να αποθηκεύσετε το εγγράφο που προκύπτει ως αρχείο .docx. Ακολουθήστε τα παρακάτω βήματα για μια εύκολη εφαρμογή.

## Διδασκαλίες εγκατάστασης
1. Προϋποθέσεις: Βεβαιωθείτε ότι έχετε εγκατεστημένο .NET στον υπολογιστή σας.
2. Εγκαταστήστε Aspose.Words: 
   * Χρησιμοποιήστε τη κονσόλα διαχειριστής πακέτων NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Ή, αναζητήστε το Aspose.Words στο GUI του NuGet Package Manager.
