# Δημιουργία εγγράφου Word με κινούμενη εικόνα

Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα έγγραφο Word χρησιμοποιώντας το Aspose.Words για .NET και να εισαγάγετε μια εικόνα με αιωρούμενη διάταξη. Ο παρεχόμενος κώδικας αρχικοποιεί ένα Έγγραφο και ένα DocumentBuilder, εισάγει μια εικόνα με καθορισμένες διαστάσεις και θέση και τέλος αποθηκεύει το έγγραφο. Αυτός ο οδηγός είναι ιδανικός για προγραμματιστές που θέλουν να βελτιώσουν τις δυνατότητες δημιουργίας εγγράφων τους χρησιμοποιώντας το Aspose.Words.

## Οδηγίες εγκατάστασης
1. Εγκατάσταση Aspose.Words: Προσθέστε το πακέτο Aspose.Words NuGet στο έργο .NET σας. Μπορείτε να το κάνετε αυτό εκτελώντας την ακόλουθη εντολή στην Κονσόλα Διαχείρισης Πακέτων:
   ```
   Install-Package Aspose.Words
   ```

2. Προσθήκη απαιτούμενων χώρων ονομάτων: Βεβαιωθείτε ότι το αρχείο κώδικά σας περιλαμβάνει τους ακόλουθους χώρους ονομάτων:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
