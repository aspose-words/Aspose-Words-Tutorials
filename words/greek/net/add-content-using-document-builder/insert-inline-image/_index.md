# Εισάγετε Inline εικόνα στο έγγραφο Word χρησιμοποιώντας Aspose.Words
Αυτό το παράδειγμα δείχνει πώς να εισάγετε μια εικόνα inline σε ένα έγγραφο Word χρησιμοποιώντας τη βιβλιοθήκη Aspose.Words για .NET.`Κατασκευαστής εγγράφωνName`τάξη, μπορείτε εύκολα να τοποθετήσετε και να εισάγετε εικόνες στην τρέχουσα θέση του δρομέα στο έγγραφό σας. Αυτός ο οδηγός παρέχει μια απλή εφαρμογή για να σας βοηθήσει να ξεκινήσετε με τον χειρισμό εικόνας στα έγγραφα Word.

## Οδηγίες εγκατάστασης
1. Εγκαταστήστε το Aspose.Words για .NET:
   * Χρησιμοποιήστε το NuGet Package Manager στο Visual Studio.
   * Εκτελέστε την ακόλουθη εντολή στην κονσόλα Package Manager:
     ```
     Install-Package Aspose.Words
     ```

2. Ρυθμίστε το πρόγραμμά σας:
   * Δημιουργήστε ένα νέο έργο .NET ή ανοίξτε ένα υπάρχον.
   * Προσθέστε τις απαραίτητες οδηγίες στην κορυφή του αρχείου κώδικα σας:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
