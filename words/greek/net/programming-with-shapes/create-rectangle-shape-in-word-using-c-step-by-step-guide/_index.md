# Δημιουργία σχήματος ορθογωνίου στο Word με C# – Πλήρες Tutorial

Κάποτε χρειάστηκε να **δημιουργήσετε σχήμα ορθογωνίου** σε ένα έγγραφο Word αλλά δεν ήξερατε από πού να ξεκινήσετε; Δεν είστε μόνοι—πολλοί προγραμματιστές αντιμετωπίζουν το ίδιο πρόβλημα όταν θέλουν να **προσθέσουν σκιά στο σχήμα** για πιο επαγγελματική εμφάνιση. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες για **εισαγωγή σχήματος στο Word**, εφαρμογή ήπιας σκιάς, και τέλος **c# generate word document** αρχεία που μπορείτε να διανείμετε στους χρήστες.

Θα καλύψουμε τα πάντα, από τη ρύθμιση του έργου μέχρι τη ρύθμιση των ιδιοτήτων της σκιάς, και θα κλείσουμε με ένα έτοιμο‑για‑εκτέλεση δείγμα κώδικα. Χωρίς περιττές πληροφορίες, μόνο τα πρακτικά βήματα που κάνουν τη δουλειά.

## Τι Θα Μάθετε

- Πώς να **δημιουργήσετε σχήμα ορθογωνίου** με Aspose.Words (ή Open XML) σε C#  
- Οι ακριβείς ιδιότητες που χρειάζεστε για **προσθήκη σκιάς στο σχήμα** ώστε να υπάρχει βάθος  
- Πού να τοποθετήσετε το σχήμα χρησιμοποιώντας `DocumentBuilder`  
- Πώς να αποθηκεύσετε το αρχείο ώστε να ανοίγει σωστά στο Microsoft Word  
- Συμβουλές, παγίδες και παραλλαγές για πραγματικές περιπτώσεις  

### Προαπαιτούμενα

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί σε .NET Core και .NET Framework)  
- Ένα πακέτο NuGet που μπορεί να επεξεργαστεί αρχεία Word – θα χρησιμοποιήσουμε **Aspose.Words for .NET** επειδή το API του είναι συνοπτικό. Αν προτιμάτε το Open XML SDK, οι έννοιες είναι ίδιες, μόνο οι κλάσεις διαφέρουν.  
- Visual Studio, VS Code ή οποιοδήποτε IDE για C# προτιμάτε  

> **Pro tip:** Αν έχετε περιορισμένο προϋπολογισμό, η Aspose προσφέρει δωρεάν δοκιμαστική έκδοση που είναι ιδανική για εκμάθηση. Απλώς αντικαταστήστε τη γραμμή άδειας με ένα σχόλιο όταν δοκιμάζετε.

## Βήμα 1: Εγκατάσταση της Βιβλιοθήκης Επεξεργασίας Word

Πρώτα, προσθέστε τη βιβλιοθήκη στο έργο σας. Ανοίξτε ένα τερματικό στο φάκελο της λύσης και τρέξτε:

```bash
dotnet add package Aspose.Words
```

Αν χρησιμοποιείτε το Open XML SDK, η εντολή θα ήταν `dotnet add package DocumentFormat.OpenXml`. Το υπόλοιπο του οδηγού υποθέτει Aspose.Words, αλλά η αντικατάσταση των κλήσεων API είναι απλή.

## Βήμα 2: Δημιουργία Νέου Κενό Έγγραφο

Τώρα που η βιβλιοθήκη είναι έτοιμη, μπορούμε να **δημιουργήσουμε σχήμα ορθογωνίου** ξεκινώντας με ένα καθαρό αντικείμενο `Document`. Σκεφτείτε το ως λευκό καμβά.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;

// Step 2: Initialize a blank Word document
Document document = new Document();
DocumentBuilder builder = new DocumentBuilder(document);
```

Το `DocumentBuilder` μας παρέχει έναν υψηλού επιπέδου τρόπο εισαγωγής περιεχομένου χωρίς να βυθιζόμαστε σε χαμηλού επιπέδου δέντρα κόμβων.

## Βήμα 3: Εισαγωγή του Σχήματος Ορθογωνίου

Με το builder στα χέρια, μπορούμε να **εισάγουμε σχήμα στο Word**. Η μέθοδος `InsertShape` δέχεται τον τύπο σχήματος και τις διαστάσεις του (πλάτος, ύψος) σε μονάδες point.

```csharp
// Step 3: Insert a rectangle shape – 150pt wide, 80pt high
Shape rectangle = builder.InsertShape(ShapeType.Rectangle, 150, 80);
```

Σε αυτό το σημείο το ορθογώνιο εμφανίζεται στο έγγραφο, αλλά φαίνεται λίγο επίπεδο. Εδώ μπαίνει το επόμενο βήμα.

## Βήμα 4: Προσθήκη Σκιάς στο Σχήμα

Οι σκιές δίνουν στο σχήμα αίσθηση βάθους. Το αντικείμενο `Shadow` μας επιτρέπει να ρυθμίσουμε ακριβεία το blur, την απόσταση, τη γωνία, το χρώμα και τη διαφάνεια. Παρακάτω υπάρχει μια πλήρης διαμόρφωση που λειτουργεί καλά για τις περισσότερες αναφορές.

```csharp
// Step 4: Configure a subtle shadow
rectangle.Shadow = new Shadow
{
    BlurRadius = 5.0,          // Soft edges
    Distance = 4.0,            // How far the shadow is offset
    Angle = 45,                // Direction in degrees (45° = down‑right)
    Color = Color.Black,       // Shadow color
    Transparency = 0.3         // 30 % transparent for a gentle look
};
```

**Γιατί αυτές οι τιμές;**  
- **BlurRadius** των `5.0` διατηρεί την άκρη ομαλή χωρίς να φαίνεται θολή.  
- **Distance** των `4.0` μετατοπίζει τη σκιά ακριβώς όσο χρειάζεται για να είναι ορατή.  
- **Angle** `45` μιμείται φυσικό φωτισμό από πάνω‑αριστερά, μια κοινή σύμβαση UI.  
- **Transparency** `0.3` αποτρέπει τη σκιά από το να κυριαρχεί πάνω στο γέμισμα του σχήματος.

Αν χρειάζεστε πιο δραματικό αποτέλεσμα, αυξήστε το `BlurRadius` και μειώστε τη `Transparency`. Για μια ήπια, σχεδόν αόρατη άνοδο, αντιστρέψτε αυτές τις τιμές.

## Βήμα 5: Αποθήκευση του Εγγράφου

Τέλος, γράψτε το αρχείο στο δίσκο. Η μέθοδος `Save` ανιχνεύει τη μορφή από την επέκταση του αρχείου, έτσι το `.docx` δίνει τη σύγχρονη μορφή Word.

```csharp
// Step 5: Persist the document
string outputPath = @"C:\Temp\ShadowRectangle.docx";
document.Save(outputPath);
```

Ανοίξτε το `ShadowRectangle.docx` στο Microsoft Word και θα δείτε ένα καθαρό ορθογώνιο με ήπια σκιά—ακριβώς αυτό που θέλατε όταν ρωτήσατε “**how to add shape**” με επαγγελματικό φινίρισμα.

## Πλήρες Παράδειγμα Λειτουργίας

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι το ολοκληρωμένο, έτοιμο‑για‑εκτέλεση πρόγραμμα. Αντιγράψτε‑και‑επικολλήστε σε μια εφαρμογή console και πατήστε **F5**.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;

namespace WordShapeDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1️⃣ Create a new blank document
            Document doc = new Document();
            DocumentBuilder builder = new DocumentBuilder(doc);

            // 2️⃣ Insert a rectangle shape (150pt × 80pt)
            Shape rect = builder.InsertShape(ShapeType.Rectangle, 150, 80);

            // 3️⃣ Add a subtle shadow
            rect.Shadow = new Shadow
            {
                BlurRadius = 5.0,
                Distance = 4.0,
                Angle = 45,
                Color = Color.Black,
                Transparency = 0.3
            };

            // 4️⃣ Save the file
            string filePath = @"C:\Temp\ShadowRectangle.docx";
            doc.Save(filePath);

            System.Console.WriteLine($"Document saved to {filePath}");
        }
    }
}
```

### Αναμενόμενο Αποτέλεσμα

- Το παραγόμενο `ShadowRectangle.docx` περιέχει **ένα σχήμα ορθογωνίου** κεντραρισμένο στη θέση του κέρσορα.  
- Το ορθογώνιο εμφανίζει **ήπια, 30 % διαφανή μαύρη σκιά** με μετατόπιση 45°.  
- Δεν προστίθεται άλλο περιεχόμενο, κρατώντας το αρχείο ελαφρύ και εύκολο ενσωμάτωσης σε μεγαλύτερες αναφορές.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις

### Τι αν χρειάζομαι διαφορετικό σχήμα;

Αντικαταστήστε το `ShapeType.Rectangle` με οποιαδήποτε άλλη τιμή του enum `ShapeType` (π.χ., `Ellipse`, `Triangle`). Το API σκιάς λειτουργεί με τον ίδιο τρόπο, οπότε μπορείτε να επαναχρησιμοποιήσετε τη διαμόρφωση.

### Πώς αλλάζω το χρώμα γεμίσματος;

```csharp
rect.FillColor = Color.LightBlue;   // or any System.Drawing.Color
```

### Μπορώ να προσθέσω το σχήμα σε συγκεκριμένη παράγραφο;

Ναι. Μετακινήστε το `DocumentBuilder` στην επιθυμητή παράγραφο με `builder.MoveToParagraph(index)` πριν καλέσετε το `InsertShape`. Αυτό εξασφαλίζει ότι το σχήμα θα εμφανιστεί ακριβώς εκεί που το χρειάζεστε.

### Τι γίνεται με παλαιότερες μορφές Word (.doc);

Απλώς αλλάξτε την επέκταση:

```csharp
doc.Save(@"C:\Temp\ShadowRectangle.doc", SaveFormat.Doc);
```

Η λειτουργία σκιάς υποστηρίζεται από το Word 2003 και μετά, οπότε θα δείτε το εφέ.

### Χρήση Open XML SDK αντί για Aspose;

Τα βήματα παραμένουν: δημιουργήστε ένα `WordprocessingDocument`, προσθέστε ένα στοιχείο `Drawing`, ορίστε τις ιδιότητες `<a:shadow>`. Το XML είναι πιο εκτενές, αλλά οι ίδιες έννοιες (μέγεθος, blur, απόσταση, γωνία) ισχύουν.

## Συμβουλές για Αποφυγή Παγίδων

- **Μην ξεχάσετε την άδεια** αν χρησιμοποιείτε εμπορική έκδοση της Aspose· διαφορετικά θα εμφανιστεί υδατογράφημα.  
- **Οι μονάδες είναι points**, όχι pixels. Ένα τυπικό pixel οθόνης ≈ 0.75 pt, οπότε προσαρμόστε τις διαστάσεις αναλόγως.  
- **Οι ιδιότητες σκιάς αγνοούνται** αν το `WrapType` του σχήματος είναι `Inline`. Χρησιμοποιήστε `WrapType = WrapType.Square` για αιωρούμενα σχήματα που σέβονται την απόδοση σκιάς.  
- **Η αποθήκευση σε δικτυακό φάκελο** μπορεί να απαιτεί κατάλληλα δικαιώματα· πάντα δοκιμάζετε τη διαδρομή πρώτα.
