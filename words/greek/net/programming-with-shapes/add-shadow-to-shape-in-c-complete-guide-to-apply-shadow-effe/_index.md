# Προσθήκη σκιάς σε σχήμα σε C# – Πλήρης Οδηγός

Έχετε αναρωτηθεί ποτέ πώς να **προσθέσετε σκιά σε σχήμα** σε ένα έγγραφο Word χρησιμοποιώντας C#; Δεν είστε οι μόνοι. Πολλοί προγραμματιστές συναντούν πρόβλημα όταν χρειάζονται αυτή τη διακριτική πτώση‑σκιά για να ξεχωρίσει ένα διάγραμμα, αλλά δεν μπορούν να βρουν ένα σύντομο, έτοιμο‑για‑εκτέλεση παράδειγμα.  

Καλή είδηση: αυτό το tutorial σας δίνει τον ακριβή κώδικα που χρειάζεστε για **προσθήκη σκιάς σε σχήμα**, εξηγεί γιατί κάθε γραμμή είναι σημαντική και σας δείχνει πώς να ρυθμίσετε το εφέ—είτε θέλετε μια αχνή γκρι ομίχλη είτε μια έντονη σκιά 45 °. Στη διαδικασία θα **εφαρμόσουμε εφέ σκιάς**, **αλλάξουμε το χρώμα της σκιάς**, και θα συζητήσουμε το κλασικό σενάριο **σκιάς 45 μοιρών**.

## Τι Θα Μάθετε

- Πώς να φορτώσετε ένα DOCX, να εντοπίσετε ένα σχήμα και να ενεργοποιήσετε τη σκιά του.
- Το νόημα πίσω από κάθε ιδιότητα σκιάς (ορατότητα, χρώμα, διαφάνεια, μέγεθος, απόσταση, γωνία).
- Τρόπους για **εφαρμογή εφέ σκιάς** δυναμικά, όπως βρόχο μέσω όλων των σχημάτων ή διαχείριση ομαδοποιημένων αντικειμένων.
- Συμβουλές για **αλλαγή χρώματος σκιάς** με ασφάλεια και αντιμετώπιση εγγράφων που δεν περιέχουν σχήματα.
- Πώς να πετύχετε μια ακριβή **σκιά 45 μοιρών** χωρίς εικασίες γωνιών.

Δεν απαιτείται εξωτερική τεκμηρίωση—απλώς αντιγράψτε, επικολλήστε και τρέξτε. Στο τέλος θα έχετε ένα λειτουργικό πρόγραμμα που προσθέτει μια επαγγελματική σκιά σε οποιοδήποτε σχήμα.

## Προαπαιτήσεις

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης σε .NET Framework 4.7+).
- Aspose.Words for .NET (δωρεάν δοκιμή ή αδειοδοτημένη έκδοση). Εγκατάσταση μέσω NuGet: `dotnet add package Aspose.Words`.
- Ένα βασικό αρχείο Word (`input.docx`) που περιέχει ήδη τουλάχιστον ένα σχήμα (π.χ. ένα ορθογώνιο ή εικόνα).

> **Pro tip:** Αν δεν έχετε σχήμα, εισάγετε ένα χειροκίνητα στο Word πρώτα· το tutorial υποθέτει ότι το πρώτο σχήμα είναι ο στόχος.

## Βήμα 1: Ρύθμιση του Έργου και Φόρτωση του Εγγράφου

Πρώτα, δημιουργήστε μια εφαρμογή console (ή οποιοδήποτε έργο C#) και προσθέστε την αναφορά Aspose.Words. Στη συνέχεια φορτώστε το DOCX που περιέχει το σχήμα που θέλετε να βελτιώσετε.

```csharp
using System;
using System.Drawing;               // For Color
using Aspose.Words;
using Aspose.Words.Drawing;          // For Shape and ShadowFormat

class Program
{
    static void Main()
    {
        // Load the Word document that contains the shape.
        Document doc = new Document("YOUR_DIRECTORY/input.docx");
```

**Γιατί είναι σημαντικό:** `Document` είναι το σημείο εισόδου για όλες τις εργασίες επεξεργασίας Word. Φορτώνοντας το αρχείο νωρίς, εξασφαλίζετε ότι κάθε επόμενη λειτουργία εκτελείται στην σωστή αναπαράσταση στη μνήμη.

## Βήμα 2: Ανάκτηση του Στόχου Σχήματος

Στη συνέχεια, εντοπίστε το σχήμα που προτίθεστε να τροποποιήσετε. Το παράδειγμα παίρνει το πρώτο σχήμα, αλλά μπορείτε να προσαρμόσετε το δείκτη ή να φιλτράρετε ανά τύπο σχήματος.

```csharp
        // Retrieve the first shape in the document (adjust the index if needed).
        Shape targetShape = (Shape)doc.GetChild(NodeType.Shape, 0, true);
        if (targetShape == null)
        {
            Console.WriteLine("No shape found. Add a shape to input.docx and try again.");
            return;
        }
```

**Εξήγηση:**  
- `GetChild(NodeType.Shape, 0, true)` διασχίζει το δέντρο του εγγράφου βάθος‑πρώτα και επιστρέφει το πρώτο σχήμα που συναντά.  
- Ο έλεγχος για `null` αποτρέπει ένα `NullReferenceException` όταν το έγγραφο δεν έχει σχήματα—μια κοινή περίπτωση που παγιδεύει τους αρχάριους.

## Βήμα 3: Ενεργοποίηση της Σκιάς

Η σκιά ενός σχήματος είναι απενεργοποιημένη από προεπιλογή. Η ενεργοποίησή της είναι τόσο απλή όσο η αλλαγή μιας Boolean σημαίας.

```csharp
        // Turn on the shadow effect for the shape.
        targetShape.ShadowFormat.Visible = true;
```

**Τι συμβαίνει:** Ορίζοντας το `Visible` σε `true` λέτε στο Word να αποδώσει μια σκιά. Χωρίς αυτή τη γραμμή, οποιεσδήποτε άλλες ρυθμίσεις σκιάς θα αγνοηθούν.

---

## Βήμα 4: Διαμόρφωση της Εμφάνισης της Σκιάς

Τώρα ορίζουμε την εμφάνιση της σκιάς. Ο παρακάτω κώδικας ταιριάζει με το τυπικό στυλ “μαύρο, 30 % διαφανές, 5 pt θολό, 3 pt μετατόπιση, γωνία 45°”.

```csharp
        // Configure the shadow's appearance.
        // • Black color
        // • 30 % transparent
        // • 5 pt blur radius (size)
        // • 3 pt offset distance
        // • 45° direction (angle)
        targetShape.ShadowFormat.Color = Color.Black;          // change shadow color
        targetShape.ShadowFormat.Transparency = 0.3;           // 30 % transparent
        targetShape.ShadowFormat.Size = 5;                     // blur radius
        targetShape.ShadowFormat.Distance = 3;                 // offset distance
        targetShape.ShadowFormat.Angle = 45;                   // 45 degree shadow
```

**Γιατί κάθε ιδιότητα είναι σημαντική:**

| Ιδιότητα | Επίδραση | Τυπική χρήση |
|----------|----------|--------------|
| `Visible` | Ενεργοποιεί/απενεργοποιεί τη σκιά | Κεντρική για **εφαρμογή εφέ σκιάς** |
| `Color` | Καθορίζει το χρώμα της σκιάς | Αλλάξτε σε γκρι για διακριτικότητα, κόκκινο για έμφαση |
| `Transparency` | 0 = αδιαφανές, 1 = πλήρως διαφανές | 0.3 δίνει μαλακό, ρεαλιστικό αποτέλεσμα |
| `Size` | Ελέγχει την ακτίνα θολώματος (σε σημεία) | Μεγαλύτερες τιμές δημιουργούν “πτερωτό” εφέ |
| `Distance` | Πόσο μακριά είναι η σκιά από το σχήμα | Μικρές αποστάσεις κρατούν το σχήμα σταθερό |
| `Angle` | Κατεύθυνση σε μοίρες (0 = δεξιά, 90 = πάνω) | 45 δίνει κλασική διαγώνια πτώση σκιάς |

Πειραματιστείτε ελεύθερα—π.χ., ορίστε `Color = Color.Gray` για **αλλαγή χρώματος σκιάς** σε πιο ανοιχτό τόνο, ή χρησιμοποιήστε `Angle = 135` για σκιά που πέφτει κάτω‑αριστερά.

## Bonus: Εφαρμογή Σκιάς σε Όλα τα Σχήματα του Εγγράφου

Αν χρειάζεται να **εφαρμόσετε εφέ σκιάς** σε κάθε σχήμα, κάντε βρόχο στη συλλογή αντί να στοχεύετε έναν μόνο κόμβο.

```csharp
        // Loop through every shape and add the same shadow.
        NodeCollection shapes = doc.GetChildNodes(NodeType.Shape, true);
        foreach (Shape shp in shapes)
        {
            shp.ShadowFormat.Visible = true;
            shp.ShadowFormat.Color = Color.Black;
            shp.ShadowFormat.Transparency = 0.3;
            shp.ShadowFormat.Size = 5;
            shp.ShadowFormat.Distance = 3;
            shp.ShadowFormat.Angle = 45;
        }
```

**Διαχείριση ειδικών περιπτώσεων:** Ορισμένα σχήματα (π.χ. WordArt) μπορεί να αγνοούν ορισμένες ιδιότητες. Δοκιμάστε πάντα σε αντιπροσωπευτικό δείγμα.

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω προσαρμοσμένο χρωματικό gradient για τη σκιά;**  
Α: Το Aspose.Words υποστηρίζει μόνο στερεά χρώματα για το `ShadowFormat.Color`. Για gradients, θα πρέπει να εξάγετε το σχήμα ως εικόνα και να εφαρμόσετε εφέ σε επίπεδο γραφικών.

**Ε: Τι γίνεται αν το έγγραφο περιέχει ομαδοποιημένα σχήματα;**  
Α: Κάθε μέλος μιας ομάδας είναι ξεχωριστός κόμβος `Shape`. Ο βρόχος που εμφανίζεται στην ενότητα “Bonus” θα τα διαχειριστεί αυτόματα.

**Ε: Λειτουργεί αυτό με αρχεία Word 2007‑2019;**  
Α: Ναι. Το Aspose.Words αφαιρεί την εξάρτηση από τη μορφή αρχείου, έτσι ο ίδιος κώδικας λειτουργεί για `.doc`, `.docx` και ακόμη `.rtf`.

**Ε: Πώς κάνω τη σκιά ξανά αόρατη;**  
Α: Ορίστε `targetShape.ShadowFormat.Visible = false;` και αποθηκεύστε ξανά το έγγραφο.
