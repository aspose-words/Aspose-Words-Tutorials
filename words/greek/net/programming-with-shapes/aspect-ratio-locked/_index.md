# Αναλογία διαστάσεων κλειδωμένη

## Εισαγωγή

Έχετε αναρωτηθεί ποτέ πώς να διατηρήσετε τις τέλειες αναλογίες εικόνων και σχημάτων στα έγγραφά σας στο Word; Μερικές φορές, πρέπει να βεβαιωθείτε ότι οι εικόνες και τα σχήματά σας δεν παραμορφώνονται κατά την αλλαγή μεγέθους. Εδώ είναι που το κλείδωμα της αναλογίας διαστάσεων είναι χρήσιμο. Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να ορίσετε την αναλογία διαστάσεων για σχήματα σε έγγραφα του Word χρησιμοποιώντας το Aspose.Words για .NET. Θα το αναλύσουμε σε εύκολα βήματα, διασφαλίζοντας ότι μπορείτε να εφαρμόσετε αυτές τις δεξιότητες στα έργα σας με σιγουριά.

## Προαπαιτούμενα

Πριν εμβαθύνουμε στον κώδικα, ας δούμε τι χρειάζεστε για να ξεκινήσετε:

- Aspose.Words για βιβλιοθήκη .NET: Πρέπει να έχετε εγκατεστημένο το Aspose.Words για .NET. Αν δεν το έχετε ήδη κάνει, μπορείτε να το κάνετε. [κατεβάστε το εδώ](https://releases.aspose.com/words/net/).
- Περιβάλλον Ανάπτυξης: Βεβαιωθείτε ότι έχετε ρυθμίσει ένα περιβάλλον ανάπτυξης .NET. Το Visual Studio είναι μια δημοφιλής επιλογή.
- Βασικές γνώσεις C#: Κάποια εξοικείωση με τον προγραμματισμό C# θα είναι χρήσιμη.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτοί οι χώροι ονομάτων θα μας δώσουν πρόσβαση στις κλάσεις και τις μεθόδους που χρειαζόμαστε για να εργαστούμε με έγγραφα και σχήματα του Word.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
```

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πριν ξεκινήσουμε να χειριζόμαστε σχήματα, πρέπει να δημιουργήσουμε έναν κατάλογο όπου θα αποθηκεύονται τα έγγραφά μας. Για λόγους απλότητας, θα χρησιμοποιήσουμε ένα σύμβολο κράτησης θέσης. `YOUR DOCUMENT DIRECTORY`Αντικαταστήστε αυτό με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Δημιουργία νέου εγγράφου

Στη συνέχεια, θα δημιουργήσουμε ένα νέο έγγραφο του Word χρησιμοποιώντας το Aspose.Words. Αυτό το έγγραφο θα χρησιμεύσει ως καμβάς μας για την προσθήκη σχημάτων και εικόνων.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

Εδώ, δημιουργούμε μια παρουσία του `Document` τάξη και χρησιμοποιήστε ένα `DocumentBuilder` για να μας βοηθήσουν να δημιουργήσουμε το περιεχόμενο του εγγράφου.

## Βήμα 3: Εισαγωγή εικόνας

Τώρα, ας εισαγάγουμε μια εικόνα στο έγγραφό μας. Θα χρησιμοποιήσουμε το `InsertImage` μέθοδος του `DocumentBuilder` κλάση. Βεβαιωθείτε ότι έχετε μια εικόνα στον καθορισμένο κατάλογο.

```csharp
Shape shape = builder.InsertImage(dataDir + "Transparent background logo.png");
```

Αντικαθιστώ `dataDir + "Transparent background logo.png"` με τη διαδρομή προς το αρχείο εικόνας σας.

## Βήμα 4: Κλείδωμα της αναλογίας διαστάσεων

Μόλις εισαχθεί η εικόνα, μπορούμε να κλειδώσουμε την αναλογία διαστάσεων. Το κλείδωμα της αναλογίας διαστάσεων διασφαλίζει ότι οι αναλογίες της εικόνας παραμένουν σταθερές κατά την αλλαγή μεγέθους.

```csharp
shape.AspectRatioLocked = true;
```

Σύνθεση `AspectRatioLocked` να `true` διασφαλίζει ότι η εικόνα διατηρεί την αρχική της αναλογία διαστάσεων.

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, θα αποθηκεύσουμε το έγγραφο στον καθορισμένο κατάλογο. Αυτό το βήμα καταγράφει όλες τις αλλαγές που έχουμε κάνει στο αρχείο του εγγράφου.

```csharp
doc.Save(dataDir + "WorkingWithShapes.AspectRatioLocked.docx");
```

## Σύναψη

Συγχαρητήρια! Μάθατε με επιτυχία πώς να ορίζετε την αναλογία διαστάσεων για σχήματα σε έγγραφα Word χρησιμοποιώντας το Aspose.Words για .NET. Ακολουθώντας αυτά τα βήματα, μπορείτε να διασφαλίσετε ότι οι εικόνες και τα σχήματά σας διατηρούν τις αναλογίες τους, κάνοντας τα έγγραφά σας να φαίνονται επαγγελματικά και κομψά. Μη διστάσετε να πειραματιστείτε με διαφορετικές εικόνες και σχήματα για να δείτε πώς λειτουργεί η λειτουργία κλειδώματος αναλογίας διαστάσεων σε διάφορα σενάρια.

Η εικόνα θα αλλάξει μέγεθος αναλογικά, διατηρώντας την αρχική της αναλογία πλάτους προς ύψος.

### Μπορώ να το εφαρμόσω αυτό και σε άλλα σχήματα εκτός από εικόνες;
Απολύτως! Η λειτουργία κλειδώματος αναλογίας διαστάσεων μπορεί να εφαρμοστεί σε οποιοδήποτε σχήμα, συμπεριλαμβανομένων ορθογωνίων, κύκλων και άλλων.

### Είναι το Aspose.Words για .NET συμβατό με το .NET Core;
Ναι, το Aspose.Words για .NET υποστηρίζει τόσο το .NET Framework όσο και το .NET Core.
