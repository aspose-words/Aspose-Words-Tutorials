# Δημιουργία Σχήματος Ορθογωνίου στο Word – Πλήρης Οδηγός Aspose.Words

Κάποτε χρειάστηκε να **δημιουργήσετε σχήμα ορθογωνίου** σε ένα έγγραφο Word αλλά δεν ήξερετε από πού να ξεκινήσετε; Δεν είστε μόνοι—πολλοί προγραμματιστές αντιμετωπίζουν αυτό το πρόβλημα όταν αυτοματοποιούν αναφορές ή τιμολόγια. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα που δείχνει πώς να **προσθέσετε σχήμα σε Word**, να εφαρμόσετε μια διακριτική σκιά και να ελέγξετε τη διαφάνεια του σχήματος, όλα με το Aspose.Words για .NET.

Στο τέλος του οδηγού θα έχετε ένα αρχείο `.docx` που περιέχει ένα καθαρό ορθογώνιο με μια επαγγελματική σκιά—ιδανικό για branding, call‑outs ή απλώς για να δώσετε στο έγγραφό σας μια πιο επαγγελματική εμφάνιση. Δεν απαιτούνται εξωτερικά εργαλεία, μόνο λίγες γραμμές C#.

## Τι Θα Χρειαστείτε

- **Aspose.Words for .NET** (η τελευταία έκδοση μέχρι αρχές 2026). Μπορείτε να το αποκτήσετε από το NuGet (`Install-Package Aspose.Words`).
- Ένα περιβάλλον ανάπτυξης .NET (Visual Studio, Rider ή VS Code με την επέκταση C#).
- Βασική εξοικείωση με τη σύνταξη C#—τίποτα περίπλοκο, μόνο οι συνηθισμένες δηλώσεις `using` και η δημιουργία αντικειμένων.

Αν έχετε ήδη όλα αυτά, υπέροχα—ας ξεκινήσουμε.

## Δημιουργία Σχήματος Ορθογωνίου – Βασικά Βήματα

Παρακάτω βρίσκεται ο πλήρης κώδικας. Αντιγράψτε‑και‑επικολλήστε το σε ένα νέο project κονσόλας, πατήστε **F5** και θα δείτε το `ShadowDemo.docx` να εμφανίζεται στο φάκελο που θα ορίσετε.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;   // Needed for Color

// Step 1: Create a new blank document.
Document document = new Document();

// Step 2: Insert a rectangle shape and define its size.
Shape rectangleShape = new Shape(document, ShapeType.Rectangle)
{
    Width  = 200,   // Width in points (≈2.78 inches)
    Height = 100    // Height in points (≈1.39 inches)
};

// Step 3: Apply a shadow with fine‑grained control over its appearance.
rectangleShape.Shadow = new Shadow
{
    BlurRadius   = 5.0,                     // Softness of the shadow edge
    Distance     = 4.0,                     // How far the shadow is offset
    Direction    = 45,                      // Angle of the offset (degrees)
    Color        = Color.Gray,              // Shadow colour
    Transparency = 0.2,                     // Opacity (0 = opaque, 1 = fully transparent)
    Spread       = 0.3                      // Size of the shadow spread
};

// Step 4: Add the shape to the first paragraph of the document.
document.FirstSection.Body.FirstParagraph.AppendChild(rectangleShape);

// Step 5: Save the document with the shadowed shape.
document.Save("ShadowDemo.docx");
```

### Γιατί Λειτουργεί Αυτό

- **`Document`** είναι το σημείο εισόδου· αντιπροσωπεύει ολόκληρο το αρχείο Word.
- **`Shape`** με `ShapeType.Rectangle` λέει στο Aspose ότι θέλουμε ένα ορθογώνιο αντικείμενο σχεδίασης.
- Ορίζοντας **`Width`** και **`Height`** δίνουμε στο σχήμα καθορισμένο μέγεθος· διαφορετικά θα είναι ένα μικροσκοπικό placeholder.
- Το αντικείμενο **`Shadow`** μας επιτρέπει να ρυθμίσουμε κάθε οπτικό στοιχείο: θολότητα, απόσταση, κατεύθυνση, χρώμα, διαφάνεια και spread. Αυτό είναι η καρδιά του *apply shadow to shape*.
- Τέλος, **`AppendChild`** ενσωματώνει το σχήμα στην πρώτη παράγραφο του εγγράφου, που είναι ο πιο απλός τρόπος για *add shape to Word* χωρίς να ασχοληθούμε με πίνακες ή κεφαλίδες.

Όταν ανοίξετε το `ShadowDemo.docx`, θα δείτε ένα γκρι ορθογώνιο να βρίσκεται άνετα στο έγγραφο, με τη σκιά του να κατεβαίνει προς τα δεξιά σε γωνία 45°. Η σκιά δεν είναι ένα συμπαγές μπλοκ· η ακτίνα θολώματος μαλακώνει τις άκρες και η διαφάνεια την κάνει να μοιάζει με φυσική σκιά αντί για σκληρό overlay.

## Προσθήκη Σχήματος σε Έγγραφο Word – Επιλογές Τοποθέτησης

Το παράδειγμα χρησιμοποιεί την **πρώτη παράγραφο** επειδή είναι ο γρηγορότερος τρόπος να δείτε κάτι στην οθόνη. Σε πραγματικές συνθήκες ίσως θέλετε να:

- Εισάγετε το σχήμα σε μια συγκεκριμένη **ενότητα** ή **κεφαλίδα/υποσέλιδο**.
- Τοποθετήσετε το μέσα σε **κελί πίνακα** για ευθυγράμμιση με δεδομένα πίνακα.
- Το τυλίξετε με επιλογές **αναδίπλωσης κειμένου** (π.χ., `WrapType.Square`) ώστε το κείμενο γύρω του να ρέει γύρω από το ορθογώνιο.

Ακολουθεί μια γρήγορη παραλλαγή που τοποθετεί το σχήμα σε μια νέα παράγραφο με προσαρμοσμένο στυλ:

```csharp
Paragraph para = new Paragraph(document);
para.ParagraphFormat.StyleIdentifier = StyleIdentifier.Heading2;
para.AppendChild(rectangleShape);
document.FirstSection.Body.AppendChild(para);
```

*Συμβουλή:* Πάντα προσθέτετε το σχήμα **μετά** τη ρύθμιση των ιδιοτήτων του· διαφορετικά ίσως χρειαστεί να καλέσετε `UpdateLayout` για να ανανεώσετε την οπτική εμφάνιση.

## Εφαρμογή Σκιάς σε Σχήμα – Λεπτομερής Ρύθμιση της Εμφάνισης

Οι σκιές μπορούν να αλλάξουν δραματικά την αισθητική ενός εγγράφου. Η κλάση `Shadow` εκθέτει πολλές ιδιότητες:

| Ιδιότητα      | Τι Ελέγχει                                   | Τυπικές Τιμές |
|---------------|----------------------------------------------|----------------|
| `BlurRadius`  | Μαλακότητα των άκρων της σκιάς               | 2.0 – 10.0      |
| `Distance`    | Απόσταση της σκιάς από το σχήμα              | 1.0 – 8.0       |
| `Direction`   | Γωνία σε μοίρες (0 = αριστερά, 90 = πάνω)    | 0 – 360         |
| `Color`       | Χρώμα σκιάς (οποιοδήποτε `System.Drawing.Color`) | Gray, Black, Custom |
| `Transparency`| Αδιαφάνεια (0 = πλήρως αδιαφανές, 1 = αόρατο) | 0.0 – 0.5       |
| `Spread`      | Επέκταση της σκιάς πριν την εφαρμογή θολώματος| 0.0 – 1.0       |

Αν θέλετε μια **διακριτική, επαγγελματική εμφάνιση**, κρατήστε το `BlurRadius` γύρω στα 4‑6 και το `Transparency` κοντά στο 0.2, όπως στον κώδικα παραπάνω. Για **δραματικό αποτέλεσμα**, αυξήστε το `Distance` στα 6, ορίστε το `Direction` στα 135° και μειώστε το `Transparency` στο 0.05.

## Ρύθμιση Διαφάνειας Σχήματος και Spread Σκιάς

Η διαφάνεια δεν αφορά μόνο τη σκιά· μπορείτε επίσης να κάνετε το ίδιο το ορθογώνιο μερικώς διαφανές:

```csharp
rectangleShape.FillColor = Color.LightBlue;
rectangleShape.Transparency = 0.3; // 30% transparent fill
```

Ο συνδυασμός μισής διαφάνειας γεμίσματος με μια μαλακή σκιά συχνά δίνει μια σύγχρονη αίσθηση UI—ιδανικό για dashboards ή mock‑ups σχεδίασης ενσωματωμένα σε αναφορές.

### Περιπτώσεις Ακρότητας που Πρέπει να Προσέξετε

1. **Παλαιότερες εκδόσεις του Word** (πριν το 2007) δεν υποστηρίζουν ορισμένες ιδιότητες σκιάς. Αν στοχεύετε σε αρχεία `.doc`, σκεφτείτε να απλοποιήσετε τη σκιά (π.χ., ορίστε `BlurRadius` στο 0).
2. **Οθόνες υψηλής ανάλυσης DPI** μπορεί να αποδώσουν τη σκιά ελαφρώς διαφορετικά. Δοκιμάστε στο περιβάλλον στόχο αν η οπτική πιστότητα είναι κρίσιμη.
3. **Αλληλοεπικάλυψη σχημάτων**—το Aspose αποδίδει τις σκιές με τη σειρά που προστίθενται. Εισάγετε τα σχήματα από το παρασκήνιο προς το προσκήνιο για να αποφύγετε ανεπιθύμητη κάλυψη.

## Αποθήκευση και Επαλήθευση του Αποτελέσματος

Η μέθοδος `Document.Save` ανιχνεύει αυτόματα τη μορφή εξόδου από την επέκταση του αρχείου. Για ένα αρχείο **`.docx`** λαμβάνετε τη μορφή Open XML, την οποία κατανοούν οι περισσότεροι σύγχρονοι επεξεργαστές Word. Αν χρειάζεστε μια **PDF** έκδοση με το ίδιο στυλ, απλώς αλλάξτε την επέκταση:

```csharp
document.Save("ShadowDemo.pdf");
```

Ανοίγοντας το παραγόμενο `ShadowDemo.docx` (ή `ShadowDemo.pdf`) θα πρέπει να δείτε ένα καθαρό **ορθογώνιο με σκιά**, επιβεβαιώνοντας ότι έχετε επιτυχώς *create rectangle shape* και *apply shadow to shape* χρησιμοποιώντας το Aspose.Words.

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω διαφορετικό σχήμα, όπως έλλειψη;**  
Α: Φυσικά. Αντικαταστήστε το `ShapeType.Rectangle` με `ShapeType.Ellipse` (ή οποιοδήποτε άλλο enum `ShapeType`). Οι ιδιότητες σκιάς παραμένουν ίδιες.

**Ε: Τι γίνεται αν θέλω το ορθογώνιο να είναι κλικαρίσιμο;**  
Α: Μπορείτε να αντιστοιχίσετε έναν υπερσύνδεσμο στο σχήμα:

```csharp
rectangleShape.Href = "https://example.com";
```

**Ε: Λειτουργεί αυτό σε .NET 6+;**  
Α: Ναι. Το Aspose.Words 23.11 και μεταγενέστερες εκδόσεις υποστηρίζουν πλήρως .NET 6, .NET 7 και .NET 8. Απλώς αναφέρετε το κατάλληλο πακέτο NuGet.

**Ε: Πώς αλλάζω το χρώμα της σκιάς ώστε να ταιριάζει με το brand μου;**  
Α: Χρησιμοποιήστε οποιοδήποτε `System.Drawing.Color` θέλετε:

```csharp
rectangleShape.Shadow.Color = Color.FromArgb(255, 30, 144, 255); // DodgerBlue
```
