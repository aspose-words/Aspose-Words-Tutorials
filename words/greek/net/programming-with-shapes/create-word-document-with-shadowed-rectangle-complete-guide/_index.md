# Δημιουργία Εγγράφου Word με Ορθογώνιο με Σκιά – Πλήρης Οδηγός

Κάποτε χρειάστηκε να **δημιουργήσετε έγγραφο word** που να φαίνεται λίγο πιο επαγγελματικό από μια απλή σελίδα κειμένου; Ίσως να φτιάχνετε ένα πρότυπο αναφοράς ή ένα φυλλάδιο και ένα απλό ορθογώνιο με ήπια σκιά να κάνει τη δουλειά. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από το πώς να εισάγετε ένα σχήμα ορθογωνίου, να ενεργοποιήσετε τη σκιά και να προσαρμόσετε το χρώμα, το θολό και τις μετατοπίσεις — όλα με C# και Aspose.Words.

Θα καλύψουμε επίσης **πώς να προσθέσετε σκιά** με τρόπο που λειτουργεί είτε στο Word 2016, 2019, είτε στην πιο πρόσφατη έκδοση του Office 365. Στο τέλος θα έχετε ένα έτοιμο προς αποθήκευση αρχείο *.docx* που δείχνει ένα ωραία σκιασμένο ορθογώνιο, και θα κατανοήσετε το «γιατί» πίσω από κάθε ιδιότητα που ορίζετε.

## Προαπαιτούμενα

- .NET 6 (ή οποιαδήποτε πρόσφατη έκδοση του .NET Framework)  
- Πακέτο NuGet Aspose.Words for .NET (`Install-Package Aspose.Words`)  
- Βασική εξοικείωση με τη σύνταξη C#  
- Ένα IDE όπως το Visual Studio (αλλά οποιοσδήποτε επεξεργαστής αρκεί)

Δεν απαιτούνται πρόσθετες βιβλιοθήκες· όλα τα υπόλοιπα περιλαμβάνονται στο Aspose.Words.

## Βήμα 1 – Αρχικοποίηση του Εγγράφου και του Builder (Create Word Document)

Για να **δημιουργήσετε έγγραφο word** προγραμματιστικά ξεκινάτε με την κλάση `Document`. Η `DocumentBuilder` είναι το πινέλο σας· σας επιτρέπει να προσθέτετε κείμενο, σχήματα και άλλα στοιχεία.

```csharp
using System;
using System.Drawing;
using Aspose.Words;
using Aspose.Words.Drawing;

class ShadowRectangleDemo
{
    static void Main()
    {
        // Step 1: Create a new blank document and a builder to edit it
        Document document = new Document();
        DocumentBuilder builder = new DocumentBuilder(document);
```

*Γιατί είναι σημαντικό:* Το αντικείμενο `Document` αντιπροσωπεύει ολόκληρο το αρχείο .docx. Χωρίς αυτό δεν υπάρχει που να συνδέσετε το ορθογώνιο ή τη σκιά του.

## Βήμα 2 – Εισαγωγή Σχήματος Ορθογωνίου (Insert Rectangle Shape)

Τώρα πραγματικά **εισάγουμε σχήμα ορθογωνίου**. Η μέθοδος `InsertShape` δέχεται ένα enum `ShapeType`, καθώς και το πλάτος και το ύψος σε points.

```csharp
        // Step 2: Insert a rectangle shape of the desired size (200x100 points)
        Shape rectangle = builder.InsertShape(ShapeType.Rectangle, 200, 100);
```

*Συμβουλή:* 1 point ≈ 1/72 inch, οπότε 200 pts είναι περίπου 2,78 ίντσες πλάτος. Ρυθμίστε αυτές τις τιμές ώστε να ταιριάζουν στο layout σας.

## Βήμα 3 – Ενεργοποίηση της Σκιάς (How to Add Shadow)

Οι σκιές είναι απενεργοποιημένες από προεπιλογή. Αλλάξτε τη σημαία `Visible` για να την ενεργοποιήσετε.

```csharp
        // Step 3: Turn on the shadow for the shape
        rectangle.ShadowFormat.Visible = true;
```

*Τι συμβαίνει;* Όταν το `Visible` είναι true, το Word θα αποδώσει μια σκιά βάσει των υπόλοιπων ιδιοτήτων που θα ορίσετε στη συνέχεια.

## Βήμα 4 – Προσαρμογή Εμφάνισης Σκιάς (Set Shadow Color, Blur, Offsets)

Εδώ **ορίζετε το χρώμα της σκιάς**, την ακτίνα θολώματος και τις μετατοπίσεις X/Y. Πειραματιστείτε—διαφορετικές τιμές δίνουν ένα απαλό φωτισμό, μια έντονη πτώση ή ακόμη και ένα «πλωτό» εφέ.

```csharp
        // Step 4: Define the shadow appearance – colour, blur radius and offsets
        rectangle.ShadowFormat.Color = Color.Gray;   // shadow colour
        rectangle.ShadowFormat.Blur = 5.0;           // blur radius (points)
        rectangle.ShadowFormat.OffsetX = 4.0;        // horizontal offset (points)
        rectangle.ShadowFormat.OffsetY = 4.0;        // vertical offset (points)
```

*Γιατί αυτές οι τιμές;* Ένα θόλωμα 5 pts δίνει μια ήπια, αχνή άκρη, ενώ μια μετατόπιση 4 pts μετακινεί τη σκιά κάτω‑δεξιά, προσομοιώνοντας πηγή φωτός από πάνω‑αριστερά. Αλλάξτε το `Color` σε `Color.Black` για μεγαλύτερη αντίθεση, ή χρησιμοποιήστε `Color.FromArgb(128, 0, 0, 0)` για ημιδιαφανή μαύρο.

### Περιπτώσεις Άκρων & Παραλλαγές

- **Χωρίς θόλωμα:** Ορίστε `Blur = 0` για καθαρή, σκληρή σκιά.  
- **Αρνητικές μετατοπίσεις:** Χρησιμοποιήστε `OffsetX = -4` για να μετακινήσετε τη σκιά αριστερά.  
- **Διάφορα σχήματα:** Οι ίδιες ιδιότητες σκιάς λειτουργούν για κύκλους, τρίγωνα ή ακόμη και ελεύθερα σχήματα—απλώς αλλάξτε το `ShapeType` στο Βήμα 2.  
- **Συμβατότητα:** Το Aspose.Words γράφει τα δεδομένα σκιάς σε μορφή Office Open XML, η οποία λειτουργεί σε Word 2010‑2021 και Office 365.

## Βήμα 5 – Αποθήκευση του Εγγράφου (Create Word Document)

Τέλος, αποθηκεύστε το αρχείο στο δίσκο. Μπορείτε να επιλέξετε οποιαδήποτε υποστηριζόμενη μορφή (`.docx`, `.pdf`, `.odt`, …) αλλά για αυτόν τον οδηγό θα μείνουμε στην κλασική μορφή Word.

```csharp
        // Step 5: Save the document with the shaped shadow
        document.Save("ShadowRectangle.docx");
    }
}
```

Όταν ανοίξετε το **ShadowRectangle.docx** στο Microsoft Word, θα δείτε ένα γκρι ορθογώνιο με μια ήπια, θολή σκιά μετατοπισμένη προς τα κάτω‑δεξιά—ακριβώς όπως το προγραμματίσαμε.

### Αναμενόμενο Αποτέλεσμα

- Ένα αρχείο *.docx* μιας σελίδας.  
- Ένα ορθογώνιο 200 pt × 100 pt κεντραρισμένο στη θέση του δρομέα όταν κλήθηκε το `InsertShape`.  
- Μια γκρι σκιά που εμφανίζεται 4 pts δεξιά και 4 pts κάτω, με θόλωμα 5 pt.

Αν το σχήμα φαίνεται εκτός κέντρου, μπορείτε να μετακινήσετε τον δρομέα με `builder.MoveTo` πριν την εισαγωγή, ή να προσαρμόσετε τις ιδιότητες `Left` και `Top` του σχήματος μετά την εισαγωγή.

## Συχνές Ερωτήσεις & Αντιμετώπιση Προβλημάτων

**Ε: Η σκιά δεν εμφανίζεται στο Word.**  
Α: Βεβαιωθείτε ότι το `ShadowFormat.Visible` είναι `true`. Επίσης, ελέγξτε ότι χρησιμοποιείτε πρόσφατη έκδοση του Aspose.Words (η δυνατότητα σκιάς προστέθηκε στην έκδοση 20.3).  

**Ε: Μπορώ να εφαρμόσω διαβάθμιση στο χρώμα της σκιάς;**  
Α: Όχι άμεσα μέσω `ShadowFormat`. Η UI του Word υποστηρίζει σκιά με διαβάθμιση, αλλά το σχήμα Open XML (που ακολουθεί το Aspose.Words) εκθέτει μόνο σκιές μονής απόχρωσης. Θα πρέπει να επεξεργαστείτε το υποκείμενο XML χειροκίνητα—σενάριο πιο προχωρημένο.

**Ε: Τι γίνεται αν χρειάζομαι διαφανές ορθογώνιο με μόνο σκιά;**  
Α: Ορίστε `rectangle.FillColor = Color.Transparent;` μετά την εισαγωγή. Η σκιά θα εξακολουθεί να αποδίδεται επειδή είναι ανεξάρτητη από το γέμισμα.

## Συμβουλές για Παραγωγικό Κώδικα

- **Επαναχρησιμοποίηση του builder:** Αν προσθέτετε πολλά σχήματα, κρατήστε το ίδιο αντικείμενο `DocumentBuilder`· η δημιουργία νέου για κάθε σχήμα προσθέτει περιττό φόρτο.  
- **Μαζικές αποθηκεύσεις:** Κάντε αποθήκευση μία φορά μετά από όλες τις τροποποιήσεις· η συχνή I/O επιβραδύνει τη δημιουργία μεγάλων εγγράφων.  
- **Διαχείριση σφαλμάτων:** Τυλίξτε όλο το τμήμα σε `try / catch` και καταγράψτε τις εξαιρέσεις `Aspose.Words`; συνήθως περιέχουν χρήσιμους αριθμούς γραμμής αν το πρότυπο εγγράφου είναι κατεστραμμένο.

## Επόμενα Βήματα (Σχετικά Θέματα)

- **Πώς να προσθέσετε σκιά** σε εικόνες ή πλαίσια κειμένου (παρόμοια χρήση `ShadowFormat`).  
- **Εισαγωγή ορθογωνίου σχήματος** μέσα σε κελί πίνακα για προσαρμοσμένο στυλ κελιού.  
- **Δημιουργία ορθογωνίου στο Word** χρησιμοποιώντας το εγγενές XML του Word (για όσους προτιμούν ακατέργαστο Open XML).  
- **Ορισμός χρώματος σκιάς** δυναμικά βάσει εισόδου χρήστη ή χρωμάτων θέματος.

Πειραματιστείτε με διαφορετικά χρώματα, ακτίνες θολώματος και μετατοπίσεις—ίσως ένα απαλό μπλε φως για εταιρική αναφορά, ή μια βαθιά μαύρη σκιά για δραματικό φυλλάδιο. Οι δυνατότητες είναι ατελείωτες, και οι αλλαγές στον κώδικα ελάχιστες.
