# Δημιουργία σχήματος ορθογωνίου και πώς να προσθέσετε σκιά σε C#

Κάποτε χρειάστηκε να **δημιουργήσετε σχήμα ορθογωνίου** σε ένα έγγραφο Word αλλά δεν ήξερες πώς να του δώσεις μια διακριτική σκιά‑πτώσης; Δεν είσαι μόνος/η. Σε πολλές περιπτώσεις αναφορών ή branding, ένα απλό ορθογώνιο με μια απαλή, ημιδιαφανή σκιά μπορεί να κάνει τη διάταξη πιο επαγγελματική χωρίς πολύ κόπο.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα **πώς να δημιουργήσετε ένα έγγραφο** χρησιμοποιώντας το Aspose.Words, μετά θα δείξουμε **πώς να προσθέσετε σκιά**, **πώς να εφαρμόσετε θόλωση στη σκιά**, και ακόμη **πώς να κάνετε τη σκιά διαφανή**. Στο τέλος θα έχετε ένα έτοιμο κομμάτι κώδικα C# που παράγει ένα αρχείο *.docx* με ένα ωραία σκιασμένο ορθογώνιο — όλα σε λίγα λεπτά.

## Τι θα χρειαστείτε

- .NET 6 ή νεότερο (το API λειτουργεί επίσης με .NET Framework 4.6+)
- Aspose.Words for .NET (η δωρεάν δοκιμή λειτουργεί για αυτό το παράδειγμα)
- Έναν επεξεργαστή κώδικα – Visual Studio, VS Code, Rider, ό,τι προτιμάτε
- Βασικές γνώσεις C# – τίποτα περίπλοκο, απλώς η δυνατότητα να τρέξετε μια εφαρμογή κονσόλας

Αν έχετε όλα αυτά, μπορούμε να περάσουμε κατευθείαν στη λύση.

## Βήμα 1 – Πώς να δημιουργήσετε έγγραφο και να αρχικοποιήσετε τον καμβά

Πρώτα απ’ όλα: χρειάζεστε ένα κενό αντικείμενο `Document`. Σκεφτείτε το ως ένα άδειο φύλλο χαρτί που το Aspose.Words θα μετατρέψει αργότερα σε αρχείο Word.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;   // For Color

// Create a new blank document
Document doc = new Document();
```

Γιατί δημιουργούμε ένα `Document` αντί να φορτώνουμε ένα πρότυπο; Ξεκινώντας από το μηδέν εξασφαλίζουμε ότι δεν υπάρχουν κρυφά στυλ ή ενότητες που να παρεμβαίνουν στο ορθογώνιο μας. Επίσης κρατά το μέγεθος του αρχείου μικρό — καλή πρακτική όταν παράγετε πολλά έγγραφα σε βρόχο.

## Βήμα 2 – Δημιουργία σχήματος ορθογωνίου (ο πυρήνας της κύριας λέξης‑κλειδί)

Τώρα δημιουργούμε **σχήμα ορθογωνίου**. Η κλάση `Shape` είναι ευέλικτη· της λέτε τον τύπο (Rectangle), το μέγεθος και πώς πρέπει να τυλίγεται με το κείμενο γύρω.

```csharp
// Define a rectangular shape
Shape rect = new Shape(doc, ShapeType.Rectangle)
{
    Width = 200,               // Width in points (≈2.8 inches)
    Height = 100,              // Height in points (≈1.4 inches)
    WrapType = WrapType.Inline // Makes the shape behave like a character
};
```

Παρατηρήστε τη χρήση της σύνταξης αρχικοποιητή αντικειμένου – είναι σύντομη και μειώνει την πιθανότητα να ξεχάσετε να ορίσετε κάποια ιδιότητα αργότερα. Το ορθογώνιο θα τοποθετηθεί μέσα στην πρώτη παράγραφο, την οποία θα προσθέσουμε στο επόμενο βήμα.

## Βήμα 3 – Πώς να προσθέσετε σκιά και να προσαρμόσετε την εμφάνισή της

Η προσθήκη σκιάς δεν είναι μόνο μία γραμμή κώδικα· έχετε πολλές ιδιότητες να ρυθμίσετε. Εδώ μπουν σε παιχνίδι οι δευτερεύουσες λέξεις‑κλειδιά **apply blur to shadow** και **make shadow transparent**.

```csharp
// Configure the shadow
rect.Shadow.Format.Color = Color.DarkGray;   // Shadow colour
rect.Shadow.Format.BlurRadius = 5.0;         // Apply blur to shadow (points)
rect.Shadow.Format.OffsetX = 3;              // Horizontal offset
rect.Shadow.Format.OffsetY = 3;              // Vertical offset
rect.Shadow.Format.Transparency = 0.3;       // 30 % transparent (make shadow transparent)
```

Μια γρήγορη σημείωση για τους αριθμούς: `BlurRadius` ίσο με 5 δίνει ήπια θόλωση· αυξήστε το σε 10 για πιο απαλό αποτέλεσμα, ή μειώστε το σε 2 για πιο καθαρή άκρη. Η τιμή `Transparency` κυμαίνεται από 0 (αδιαφανές) έως 1 (αόρατο). Ρυθμίστε την ανάλογα με τις απαιτήσεις αντίθεσης του brand σας.

### Pro tip

Αν χρειαστείτε σκιά χρώματος (π.χ. εταιρικό μπλε), απλώς αντικαταστήστε το `Color.DarkGray` με `Color.FromArgb(80, 0, 120, 215)`. Το πρώτο όρισμα είναι το κανάλι άλφα – κρατήστε το χαμηλό για διακριτικότητα.

## Βήμα 4 – Εισαγωγή του σχήματος στο έγγραφο

Με το ορθογώνιο και τη σκιά του έτοιμα, τοποθετούμε το σχήμα στην πρώτη παράγραφο του εγγράφου. Αυτό το βήμα εξασφαλίζει ότι το σχήμα εμφανίζεται στην κορυφή του αρχείου.

```csharp
// Append the shape to the first paragraph of the first section
doc.FirstSection.Body.FirstParagraph.AppendChild(rect);
```

Γιατί στην πρώτη παράγραφο; Είναι μια ασφαλής προεπιλογή που λειτουργεί ακόμη και όταν το έγγραφο είναι εντελώς κενό. Αν έχετε συγκεκριμένη θέση (π.χ. μετά από έναν τίτλο), θα εντοπίσετε εκείνο το κόμβο και θα εισάγετε το σχήμα εκεί.

## Βήμα 5 – Αποθήκευση του αρχείου και επαλήθευση του αποτελέσματος

Τέλος, αποθηκεύουμε το έγγραφο στο δίσκο. Μπορείτε να επιλέξετε οποιοδήποτε μονοπάτι θέλετε· απλώς βεβαιωθείτε ότι ο φάκελος υπάρχει.

```csharp
// Save the document
doc.Save(@"C:\Temp\ShadowRectangle.docx");
```

Όταν ανοίξετε το *ShadowRectangle.docx* στο Microsoft Word, θα δείτε ένα ορθογώνιο 200 × 100 σημείων με σκιά σκούρου γκρι, ελαφρώς θολή, 30 % διαφανή, μετατοπισμένη τρία σημεία προς τα δεξιά και κάτω. Το αποτέλεσμα είναι διακριτικό αλλά προσθέτει βάθος σε διαφορετικά επίπεδες διατάξεις.

## Συνηθισμένες παραλλαγές και ειδικές περιπτώσεις

### Αλλαγή του χρώματος της σκιάς δυναμικά

Αν η εφαρμογή σας υποστηρίζει θέματα, μπορείτε να διαβάσετε το χρώμα της σκιάς από ένα αρχείο ρυθμίσεων:

```csharp
Color themeShadow = ColorTranslator.FromHtml(ConfigurationManager.AppSettings["ShadowColor"]);
rect.Shadow.Format.Color = themeShadow;
```

### Καθιστώντας το σχήμα μη‑ενσωματωμένο (non‑inline)

Μερικές φορές θέλετε το ορθογώνιο να «πλέει» πάνω από το κείμενο. Αλλάξτε το `WrapType` σε `WrapType.Square` και ορίστε το `RelativeHorizontalPosition` σε `RelativeHorizontalPosition.Margin` για μεγαλύτερο έλεγχο.

```csharp
rect.WrapType = WrapType.Square;
rect.RelativeHorizontalPosition = RelativeHorizontalPosition.Margin;
rect.Left = 72; // 1 inch from the left margin
```

### Διαχείριση πολλαπλών σελίδων

Αν χρειάζεστε ορθογώνιο σε κάθε σελίδα, κάντε βρόχο στα `doc.Sections` και προσθέστε ένα κλωνοποιημένο σχήμα στην πρώτη παράγραφο κάθε ενότητας. Θυμηθείτε να καλέσετε `rect.Clone(true)` για να αντιγράψετε και τις ρυθμίσεις σκιάς.

- **Δημιουργήσαμε σχήμα ορθογωνίου** χρησιμοποιώντας το Aspose.Words
- **Προσθέσαμε σκιά** με χρώμα, μετατόπιση, θόλωση και διαφάνεια
- Επιδείξαμε **apply blur to shadow** και **make shadow transparent**
- Αποθηκεύσαμε ένα αρχείο Word που μπορείτε να ανοίξετε αμέσως

Όλα αυτά επιτεύχθηκαν με λίγες μόνο γραμμές κώδικα, αποδεικνύοντας ότι οι εξελιγμένες οπτικές προσαρμογές δεν απαιτούν πάντα βαριές βιβλιοθήκες γραφικών.

## Τι ακολουθεί;

- Πειραματιστείτε με άλλους τύπους `ShapeType` (Ellipse, Cloud, κ.λπ.) και δείτε πώς συμπεριφέρονται οι σκιές.
- Συνδυάστε το ορθογώνιο με πλαίσια κειμένου για να δημιουργήσετε ετικέτες‑call‑outs.
- Εξερευνήστε **πώς να δημιουργήσετε έγγραφα‑πρότυπα** που περιέχουν ήδη placeholders για σχήματα, και στη συνέχεια να τα γεμίζετε προγραμματιστικά.

Αλλάξτε ελεύθερα το radius της θόλωσης, το χρώμα ή τη διαφάνεια μέχρι η σκιά να φαίνεται ακριβώς όπως θέλετε για τη γλώσσα σχεδίασής σας. Το API είναι επιεικές, και οι αλλαγές είναι άμεσα ορατές όταν ξανατρέξετε την εφαρμογή κονσόλας.

Καλή προγραμματιστική δουλειά, και τα έγγραφά σας να έχουν πάντα αυτή τη μικρή δόση βάθους!
