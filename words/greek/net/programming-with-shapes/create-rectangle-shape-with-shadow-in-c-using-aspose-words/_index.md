# Δημιουργία σχήματος ορθογωνίου με σκιά σε C# χρησιμοποιώντας Aspose.Words

Έχετε χρειαστεί ποτέ να **δημιουργήσετε σχήμα ορθογωνίου** σε ένα έγγραφο Word αλλά δεν ήξερες πώς να του προσθέσεις μια διακριτική σκιά πτώσης; Δεν είστε μόνοι—πολλοί προγραμματιστές αντιμετωπίζουν αυτό το πρόβλημα όταν αρχίζουν να πειραματίζονται με την αυτοματοποίηση εγγράφων. Σε αυτόν τον οδηγό θα περάσουμε βήμα‑βήμα πώς να **προσθέσετε σκιά σε σχήμα** χρησιμοποιώντας Aspose.Words, και θα απαντήσουμε στις ερωτήσεις «**πώς να προσθέσετε σκιά**», «**πώς να δημιουργήσετε ορθογώνιο**» και «**πώς να ορίσετε σκιά**» καθ' όλη τη διάρκεια.

Θα ξεκινήσουμε με ένα καθαρό `Document`, θα σχεδιάσουμε ένα ορθογώνιο, θα ενεργοποιήσουμε τη σκιά του, θα ρυθμίσουμε το θόλωμα, την απόσταση, τη γωνία και το χρώμα, και τέλος θα αποθηκεύσουμε το αρχείο. Στο τέλος θα έχετε ένα έτοιμο `.docx` που δείχνει ένα γκριζαρισμένο ορθογώνιο που «πλέει» ακριβώς πάνω από τη σελίδα. Καμία μυστική, μόνο απλός κώδικας που μπορείτε να αντιγράψετε‑επικολλήσετε σε οποιοδήποτε έργο .NET.

## Προαπαιτούμενα

Πριν βουτήξουμε, βεβαιωθείτε ότι έχετε:

* **Aspose.Words for .NET** (η πιο πρόσφατη έκδοση μέχρι Μάρτιο 2026). Μπορείτε να το κατεβάσετε από το NuGet με `Install-Package Aspose.Words`.
* Ένα περιβάλλον ανάπτυξης .NET – Visual Studio, Rider ή ακόμη και VS Code με την επέκταση C# λειτουργούν άψογα.
* Βασικές γνώσεις C# – τίποτα περίπλοκο, απλώς η ικανότητα να δημιουργήσετε μια εφαρμογή console ή WinForms.

Αυτό είναι όλο. Καμία πρόσθετη βιβλιοθήκη, κανένα κρυφό βήμα. Έτοιμοι; Ας ξεκινήσουμε.

## Βήμα 1: Αρχικοποίηση ενός νέου κεντρικού εγγράφου

Για να **δημιουργήσετε σχήμα ορθογωνίου**, χρειάζεται πρώτα ένας κοντέινερ – ένα αντικείμενο `Document` – που αντιπροσωπεύει το αρχείο Word.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;

// Step 1: Create a new empty document
Document document = new Document();
```

Η κλάση `Document` είναι το σημείο εισόδου για όλα όσα κάνει το Aspose.Words. Σκεφτείτε το ως ένα κενό καμβά· χωρίς αυτό δεν μπορείτε να προσθέσετε σχήματα, πίνακες ή κείμενο.

## Βήμα 2: Δημιουργία του ορθογωνίου που θα κρατήσει τη σκιά

Τώρα θα **πώς να δημιουργήσετε ορθογώνιο** δημιουργώντας ένα `Shape` τύπου `Rectangle`. Ορίζουμε επίσης το μέγεθός του σε points (1 point ≈ 1/72 ίντσα).

```csharp
// Step 2: Create a rectangular shape that will hold the shadow
Shape rectangleShape = new Shape(document, ShapeType.Rectangle);
rectangleShape.Width  = 200; // width in points
rectangleShape.Height = 100; // height in points
```

Γιατί 200 × 100 points; Είναι ένα λογικό μέγεθος για επίδειξη – αρκετά μεγάλο ώστε η σκιά να φαίνεται καθαρά, αλλά όχι τόσο τεράστιο ώστε να κατακλύζει τη σελίδα. Αλλάξτε ελεύθερα αυτούς τους αριθμούς ώστε να ταιριάζουν με το layout σας.

## Βήμα 3: Ενεργοποίηση του εφέ σκιάς και ρύθμιση της εμφάνισής του

Εδώ βρίσκεται η καρδιά του οδηγού: **πώς να προσθέσετε σκιά** και **πώς να ορίσετε σκιά**. Το Aspose.Words εκθέτει ένα αντικείμενο `Shadow` σε κάθε σχήμα, επιτρέποντάς σας να ενεργοποιήσετε το εφέ και να ρυθμίσετε τις οπτικές παραμέτρους.

```csharp
// Step 3: Enable the shadow effect and configure its appearance
rectangleShape.Shadow.Enabled    = true;                     // turn the shadow on
rectangleShape.Shadow.BlurRadius = 5;                       // blur radius in pixels
rectangleShape.Shadow.Distance   = 8;                       // distance from the shape in pixels
rectangleShape.Shadow.Angle      = 45;                      // direction of the light source (degrees)
rectangleShape.Shadow.Color      = System.Drawing.Color.Gray; // shadow color
```

* **BlurRadius** μαλακώνει τις άκρες – μεγαλύτερη τιμή κάνει τη σκιά πιο διάχυτη.
* **Distance** απομακρύνει τη σκιά περισσότερο από το ορθογώνιο.
* **Angle** καθορίζει από πού φαίνεται να έρχεται το φως· 45° δίνει μια διαγώνια, φυσική εμφάνιση.
* **Color** σας επιτρέπει να επιλέξετε οποιοδήποτε `System.Drawing.Color`. Το γκρι είναι ασφαλές προεπιλεγμένο, αλλά μπορείτε να το κάνετε τολμηρό με `Color.Black` ή διακριτικό με `Color.LightGray`.

Συμβουλή: Αν ορίσετε `Enabled = false`, όλες οι άλλες ρυθμίσεις σκιάς αγνοούνται, οπότε ελέγχετε πάντα αυτή τη σημαία.

## Βήμα 4: Εισαγωγή του σχήματος στο σώμα του εγγράφου

Με το ορθογώνιο έτοιμο και τη σκιά του ρυθμισμένη, πρέπει να το τοποθετήσουμε στο έγγραφο. Ο πιο απλός τρόπος είναι να το προσαρτήσετε στην πρώτη παράγραφο της πρώτης ενότητας.

```csharp
// Step 4: Insert the shape into the first paragraph of the document body
document.FirstSection.Body.FirstParagraph.AppendChild(rectangleShape);
```

Αν το έγγραφό σας περιέχει ήδη κείμενο, μπορείτε να εντοπίσετε ένα συγκεκριμένο `Paragraph` ή ακόμη και ένα κελί `Table` και να εισάγετε το σχήμα εκεί. Η μέθοδος `AppendChild` είναι ευέλικτη – λειτουργεί με οποιοδήποτε τύπο `Node`.

## Βήμα 5: Αποθήκευση του εγγράφου και επαλήθευση του αποτελέσματος

Τέλος, γράφουμε το αρχείο στο δίσκο. Αλλάξτε τη διαδρομή σε όποιον φάκελο θέλετε· ο φάκελος πρέπει να υπάρχει, διαφορετικά θα πάρετε εξαίρεση.

```csharp
// Step 5: Save the document with the shadowed shape
document.Save(@"C:\Temp\ShadowedRectangle.docx");
```

Ανοίξτε το παραγόμενο `ShadowedRectangle.docx` στο Microsoft Word (ή LibreOffice) και θα πρέπει να δείτε ένα γκρι ορθογώνιο με καθαρή, διαγώνια σκιά που κατευθύνεται κάτω‑δεξιά. Αν η σκιά φαίνεται πολύ αχνή, αυξήστε το `BlurRadius` ή το `Distance` και ξανατρέξτε τον κώδικα – η πειραματική δοκιμή είναι μέρος της διασκέδασης.

### Αναμενόμενο αποτέλεσμα

* Ένα έγγραφο Word μιας σελίδας.
* Ένα γκρι ορθογώνιο 200 × 100 point τοποθετημένο στην πάνω‑αριστερή γωνία της σελίδας.
* Μια διακριτική γκρι σκιά μετατοπισμένη κατά 8 pixel με γωνία 45°, θολή κατά 5 pixel.

## Πώς να προσθέσετε σκιά σε σχήμα – πιο βαθιά ανάλυση

Μπορεί να αναρωτιέστε, *«Μπορώ να ανιματίσω τη σκιά ή να την αλλάξω ανάλογα με την είσοδο του χρήστη;»* Αν και το Aspose.Words δεν υποστηρίζει animation, μπορείτε προγραμματιστικά να προσαρμόσετε τις ιδιότητες σκιάς πριν την αποθήκευση, δημιουργώντας ουσιαστικά πολλές εκδόσεις του ίδιου εγγράφου με διαφορετικές εμφανίσεις. Για παράδειγμα, επανάληψη πάνω σε μια συλλογή χρωμάτων:

```csharp
Color[] shadowColors = { Color.Gray, Color.Black, Color.DarkSlateGray };
foreach (var col in shadowColors)
{
    rectangleShape.Shadow.Color = col;
    document.Save($@"C:\Temp\Shadow_{col.Name}.docx");
}
```

Αυτό το μικρό απόσπασμα δείχνει **πώς να ορίσετε σκιά** δυναμικά—ιδανικό για τη δημιουργία θεματικών αναφορών.

## Πώς να δημιουργήσετε ορθογώνιο – εναλλακτικά σχήματα

Αν χρειάζεστε στρογγυλεμένο ορθογώνιο, απλώς αλλάξτε το `ShapeType`:

```csharp
Shape rounded = new Shape(document, ShapeType.RoundRectangle);
rounded.Width  = 200;
rounded.Height = 100;
rounded.Shadow.Enabled = true; // shadow works the same way
```

Ή, για τέλειο τετράγωνο, ορίστε το `Width` ίσο με το `Height`. Οι ίδιες ιδιότητες σκιάς ισχύουν, οπότε καλύπτεστε ήδη για **πώς να προσθέσετε σκιά** σε οποιοδήποτε σχήμα επιλέξετε.

## Συνηθισμένα προβλήματα και αντιμετώπιση

| Συμπτωμα | Πιθανή αιτία | Διόρθωση |
|---------|--------------|----------|
| Η σκιά δεν εμφανίζεται | `Shadow.Enabled` παραμένει `false` | Ορίστε `rectangleShape.Shadow.Enabled = true;` |
| Η σκιά φαίνεται πολύ αιχμηρή | `BlurRadius` ορίστηκε σε 0 | Αυξήστε το `BlurRadius` τουλάχιστον σε 3 |
| Το έγγραφο πετάει `FileNotFoundException` κατά την αποθήκευση | Ο φάκελος προορισμού δεν υπάρχει | Δημιουργήστε πρώτα το φάκελο ή χρησιμοποιήστε έγκυρη διαδρομή |
| Το σχήμα είναι αόρατο | `Width`/`Height` ορίστηκαν σε 0 | Βεβαιωθείτε ότι και οι δύο διαστάσεις είναι > 0 |

Η προσοχή σε αυτά τα ζητήματα σας σώζει από το κλασικό «γιατί δεν εμφανίζεται το σχήμα μου;» στιγμιότυπο.

## Ανακεφαλαίωση – τι πετύχαμε

* **Δημιουργία σχήματος ορθογωνίου** σε νέο έγγραφο Word χρησιμοποιώντας Aspose.Words.  
* **Προσθήκη σκιάς σε σχήμα** με την ενεργοποίηση της σημαίας `Shadow.Enabled` και ρύθμιση θολώματος, απόστασης, γωνίας και χρώματος.  
* Επιδεικνύτηκε **πώς να προσθέσετε σκιά**, **πώς να δημιουργήσετε ορθογώνιο** και **πώς να ορίσετε σκιά** σε καθαρό, επαναχρησιμοποιήσιμο απόσπασμα κώδικα.  
* Παρέχεται ένα πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα που μπορείτε να επικολλήσετε σε οποιοδήποτε έργο C#.
