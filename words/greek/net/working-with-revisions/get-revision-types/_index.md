# Λήψη τύπων λέξεων αναθεώρησης

## Εισαγωγή

Έχετε βρεθεί ποτέ σε μια θάλασσα από αναθεωρήσεις εγγράφων, αναρωτώμενοι ποιος μετέφερε τι και πότε; Δεν είστε οι μόνοι. Η διαχείριση αναθεωρήσεων εγγράφων μπορεί να είναι μια κουραστική εργασία, ειδικά όταν πρόκειται για εκτεταμένα έγγραφα. Αλλά μην ανησυχείτε! Με το Aspose.Words για .NET, μπορείτε εύκολα να εντοπίσετε και να διαχειριστείτε αυτές τις αναθεωρήσεις. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε βήμα προς βήμα σε μια διαδικασία για το πώς να αποκτήσετε τύπους λέξεων αναθεώρησης σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET. Λοιπόν, πάρτε τις θέσεις σας και ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ασχοληθούμε με τον κώδικα, υπάρχουν μερικά πράγματα που θα χρειαστείτε:

1. Aspose.Words για τη βιβλιοθήκη .NET: Αν δεν το έχετε κάνει ήδη, κατεβάστε το από [εδώ](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Visual Studio ή οποιοδήποτε άλλο IDE συμβατό με .NET.
3. Βασικές γνώσεις C#: Η κατανόηση της γλώσσας προγραμματισμού C# θα είναι ωφέλιμη.
4. Ένα έγγραφο Word με αναθεωρήσεις: Βεβαιωθείτε ότι έχετε ένα `.docx` αρχείο με τις εντοπισμένες αλλαγές για να ελέγξετε τον κώδικα.

## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας σε C#. Αυτό θα σας επιτρέψει να αποκτήσετε πρόσβαση στις λειτουργίες που παρέχονται από το Aspose.Words για .NET.

```csharp
using Aspose.Words;
using Aspose.Words.Revision;
using System;
```

Ας χωρίσουμε το παράδειγμα σε πολλά βήματα για καλύτερη κατανόηση και εφαρμογή.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πρώτα απ 'όλα, πρέπει να ορίσετε τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ θα βρίσκεται το έγγραφο του Word με τις αναθεωρήσεις.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον φάκελο εγγράφων σας.

## Βήμα 2: Φόρτωση του εγγράφου του Word

Στη συνέχεια, πρέπει να φορτώσετε το έγγραφο του Word στο έργο σας. Αυτό το έγγραφο θα πρέπει να έχει τις αναθεωρήσεις που θέλετε να αναλύσετε.

```csharp
Document doc = new Document(dataDir + "Revisions.docx");
```

Βεβαιωθείτε ότι το αρχείο `Revisions.docx` υπάρχει στον καθορισμένο κατάλογο.

## Βήμα 3: Πρόσβαση στη Συλλογή Παραγράφων

Τώρα που το έγγραφό σας έχει φορτωθεί, πρέπει να έχετε πρόσβαση στις παραγράφους που βρίσκονται στην πρώτη ενότητα του κυρίως μέρους του εγγράφου. Αυτό θα σας βοηθήσει να επανεξετάσετε κάθε παράγραφο για να ελέγξετε για αναθεωρήσεις.

```csharp
ParagraphCollection paragraphs = doc.FirstSection.Body.Paragraphs;
```

## Βήμα 4: Επαναλάβετε τις παραγράφους και ελέγξτε για αναθεωρήσεις

Εδώ είναι που συμβαίνει η μαγεία. Θα επαναλάβετε κάθε παράγραφο και θα ελέγξετε αν έχει μετακινηθεί (διαγραφεί ή εισαχθεί).

```csharp
for (int i = 0; i < paragraphs.Count; i++)
{
    if (paragraphs[i].IsMoveFromRevision)
        Console.WriteLine("Paragraph {0} has been moved (deleted).", i);
    if (paragraphs[i].IsMoveToRevision)
        Console.WriteLine("Paragraph {0} has been moved (inserted).", i);
}
```

Αυτός ο βρόχος διατρέχει κάθε παράγραφο και χρησιμοποιεί το `IsMoveFromRevision` και `IsMoveToRevision` ιδιότητες για να προσδιορίσετε εάν η παράγραφος μετακινήθηκε (διαγράφηκε) ή μετακινήθηκε (εισήχθη).
