# Μορφή 1Bpp με ευρετήριο

## Εισαγωγή

Αναρωτηθήκατε ποτέ πώς να αποθηκεύσετε ένα έγγραφο του Word ως ασπρόμαυρη εικόνα με λίγες μόνο γραμμές κώδικα; Λοιπόν, είστε τυχεροί! Σήμερα, θα εξερευνήσουμε ένα ωραίο μικρό κόλπο χρησιμοποιώντας το Aspose.Words για .NET που σας επιτρέπει να μετατρέψετε τα έγγραφά σας σε εικόνες με ευρετήριο 1Bpp. Αυτή η μορφή είναι ιδανική για ορισμένους τύπους ψηφιακής αρχειοθέτησης, εκτύπωσης ή όταν χρειάζεται να εξοικονομήσετε χώρο. Θα αναλύσουμε κάθε βήμα για να το κάνουμε τόσο εύκολο. Είστε έτοιμοι να ξεκινήσετε; Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν λερώσουμε τα χέρια μας, υπάρχουν μερικά πράγματα που πρέπει να έχετε στη διάθεσή σας:

- Aspose.Words για .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη. Μπορείτε [κατεβάστε το εδώ](https://releases.aspose.com/words/net/).
- Περιβάλλον ανάπτυξης .NET: Το Visual Studio είναι μια καλή επιλογή, αλλά μπορείτε να χρησιμοποιήσετε οποιοδήποτε περιβάλλον σας βολεύει.
- Βασικές γνώσεις C#: Μην ανησυχείτε, θα το κρατήσουμε απλό, αλλά λίγη εξοικείωση με την C# θα βοηθήσει.
- Ένα έγγραφο του Word: Να έχετε ένα δείγμα εγγράφου του Word έτοιμο για μετατροπή.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, πρέπει να εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό είναι κρίσιμο καθώς μας επιτρέπει να έχουμε πρόσβαση στις κλάσεις και τις μεθόδους που χρειαζόμαστε από το Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Θα χρειαστεί να καθορίσετε τη διαδρομή προς τον κατάλογο του εγγράφου σας. Εδώ αποθηκεύεται το έγγραφο του Word και εκεί θα αποθηκευτεί η εικόνα που έχει μετατραπεί.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Φόρτωση του εγγράφου του Word

Τώρα, ας φορτώσουμε το έγγραφο του Word σε ένα Aspose.Words `Document` αντικείμενο. Αυτό το αντικείμενο αντιπροσωπεύει το αρχείο Word σας και σας επιτρέπει να το χειριστείτε.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης εικόνας

Στη συνέχεια, πρέπει να ρυθμίσουμε το `ImageSaveOptions`Εδώ συμβαίνει η μαγεία. Θα το ρυθμίσουμε ώστε να αποθηκεύει την εικόνα σε μορφή PNG με λειτουργία χρώματος με ευρετήριο 1Bpp.

```csharp
ImageSaveOptions saveOptions = new ImageSaveOptions(SaveFormat.Png)
{
    PageSet = new PageSet(1),
    ImageColorMode = ImageColorMode.BlackAndWhite,
    PixelFormat = ImagePixelFormat.Format1bppIndexed
};
```

- SaveFormat.Png: Αυτό καθορίζει ότι θέλουμε να αποθηκεύσουμε το έγγραφο ως εικόνα PNG.
- PageSet(1): Αυτό υποδεικνύει ότι μετατρέπουμε μόνο την πρώτη σελίδα.
- ImageColorMode.BlackAndWhite: Αυτό ορίζει την εικόνα σε ασπρόμαυρη.
- ImagePixelFormat.Format1bppIndexed: Αυτό ορίζει τη μορφή εικόνας σε 1Bpp με ευρετήριο.

## Βήμα 4: Αποθήκευση του εγγράφου ως εικόνα

Τέλος, αποθηκεύουμε το έγγραφο ως εικόνα χρησιμοποιώντας το `Save` μέθοδος του `Document` αντικείμενο.

```csharp
doc.Save(dataDir + "WorkingWithImageSaveOptions.Format1BppIndexed.Png", saveOptions);
```

Αυτή η μέθοδος είναι εξαιρετικά χρήσιμη για τη δημιουργία εικόνων υψηλής αντίθεσης και εξοικονόμησης χώρου από τα έγγραφά σας. Τώρα, μπορείτε εύκολα να την ενσωματώσετε στα έργα και τις ροές εργασίας σας. Καλή κωδικοποίηση!

## Συχνές ερωτήσεις

### Τι είναι μια εικόνα με ευρετήριο 1Bpp;
Μια εικόνα με ευρετήριο 1Bpp (1 Bit ανά εικονοστοιχείο) είναι μια ασπρόμαυρη μορφή εικόνας όπου κάθε εικονοστοιχείο αναπαρίσταται από ένα μόνο bit, είτε 0 είτε 1. Αυτή η μορφή είναι εξαιρετικά αποδοτική ως προς τον χώρο.

### Μπορώ να μετατρέψω πολλές σελίδες ενός εγγράφου Word ταυτόχρονα;
Ναι, μπορείτε. Τροποποιήστε το `PageSet` ιδιοκτησία στο `ImageSaveOptions` για να συμπεριλάβετε πολλές σελίδες ή ολόκληρο το έγγραφο.

### Χρειάζομαι άδεια χρήσης για να χρησιμοποιήσω το Aspose.Words για .NET;
Ναι, το Aspose.Words για .NET απαιτεί άδεια χρήσης για πλήρη λειτουργικότητα. Μπορείτε να αποκτήσετε μια [προσωρινή άδεια εδώ](https://purchase.aspose.com/temporary-license/).

### Σε ποιες άλλες μορφές εικόνας μπορώ να μετατρέψω το έγγραφο του Word μου;
Το Aspose.Words υποστηρίζει διάφορες μορφές εικόνας, όπως JPEG, BMP και TIFF. Απλώς αλλάξτε το `SaveFormat` στο `ImageSaveOptions`.

### Πού μπορώ να βρω περισσότερη τεκμηρίωση για το Aspose.Words για .NET;
Μπορείτε να βρείτε λεπτομερή τεκμηρίωση στο [Aspose.Words για τη σελίδα τεκμηρίωσης .NET](https://reference.aspose.com/words/net/).
