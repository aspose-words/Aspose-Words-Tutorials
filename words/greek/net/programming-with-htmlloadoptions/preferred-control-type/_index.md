# Προτιμώμενος τύπος ελέγχου σε έγγραφο του Word

## Εισαγωγή

Βουτάμε σε ένα συναρπαστικό σεμινάριο σχετικά με τον τρόπο εργασίας με επιλογές φόρτωσης HTML στο Aspose.Words για .NET, εστιάζοντας συγκεκριμένα στον ορισμό του προτιμώμενου τύπου ελέγχου κατά την εισαγωγή ενός πεδίου φόρμας σύνθετου πλαισίου σε ένα έγγραφο του Word. Αυτός ο οδηγός βήμα προς βήμα θα σας βοηθήσει να κατανοήσετε πώς να χειρίζεστε και να αποδίδετε αποτελεσματικά το περιεχόμενο HTML στα έγγραφά σας του Word χρησιμοποιώντας το Aspose.Words για .NET.

## Προαπαιτούμενα

Πριν προχωρήσουμε στον κώδικα, υπάρχουν μερικά πράγματα που πρέπει να έχετε στη διάθεσή σας:

1. Aspose.Words για .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να την κατεβάσετε από το [δικτυακός τόπος](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Θα πρέπει να έχετε ρυθμίσει ένα περιβάλλον ανάπτυξης, όπως το Visual Studio.
3. Βασικές γνώσεις C#: Απαιτείται βασική κατανόηση του προγραμματισμού C# για την παρακολούθηση του σεμιναρίου.
4. Περιεχόμενο HTML: Η βασική γνώση της HTML είναι χρήσιμη, καθώς σε αυτό το παράδειγμα θα εργαστούμε με περιεχόμενο HTML.

## Εισαγωγή χώρων ονομάτων

Αρχικά, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων για να ξεκινήσουμε:

```csharp
using System;
using System.IO;
using System.Text;
using Aspose.Words;
using Aspose.Words.Loading;
```

Τώρα, ας αναλύσουμε το παράδειγμα σε πολλά βήματα για να διασφαλίσουμε τη σαφήνεια και την κατανόηση.

## Βήμα 1: Ρύθμιση του περιεχομένου HTML σας

Αρχικά, πρέπει να ορίσουμε το περιεχόμενο HTML που θέλουμε να εισαγάγουμε στο έγγραφο του Word. Ακολουθεί το απόσπασμα HTML που θα χρησιμοποιήσουμε:

```csharp
const string html = @"
    <html>
        <select name='ComboBox' size='1'>
            <option value='val1'>item1</option>
            <option value='val2'></option>                        
        </select>
    </html>
";
```

Αυτή η HTML περιέχει ένα απλό σύνθετο πλαίσιο με δύο επιλογές. Θα φορτώσουμε αυτήν την HTML σε ένα έγγραφο του Word και θα καθορίσουμε τον τρόπο απόδοσής της.

## Βήμα 2: Ορισμός του καταλόγου εγγράφων

Στη συνέχεια, καθορίστε τον κατάλογο όπου θα αποθηκευτεί το έγγραφο του Word. Αυτό βοηθά στην οργάνωση των αρχείων σας και στη διατήρηση της καθαρής διαχείρισης διαδρομών.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το έγγραφο του Word.

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών φόρτωσης HTML

Εδώ, διαμορφώνουμε τις επιλογές φόρτωσης HTML, εστιάζοντας ιδιαίτερα στο `PreferredControlType` ιδιότητα. Αυτό καθορίζει τον τρόπο με τον οποίο θα πρέπει να αποδοθεί το σύνθετο πλαίσιο στο έγγραφο του Word.

```csharp
HtmlLoadOptions loadOptions = new HtmlLoadOptions { PreferredControlType = HtmlControlType.StructuredDocumentTag };
```

Ρυθμίζοντας `PreferredControlType` να `HtmlControlType.StructuredDocumentTag`, διασφαλίζουμε ότι το σύνθετο πλαίσιο αποδίδεται ως ετικέτα δομημένου εγγράφου (SDT) στο έγγραφο του Word.

## Βήμα 4: Φόρτωση περιεχομένου HTML στο έγγραφο

Χρησιμοποιώντας τις διαμορφωμένες επιλογές φόρτωσης, φορτώνουμε το περιεχόμενο HTML σε ένα νέο έγγραφο του Word.

```csharp
Document doc = new Document(new MemoryStream(Encoding.UTF8.GetBytes(html)), loadOptions);
```

Εδώ, μετατρέπουμε τη συμβολοσειρά HTML σε έναν πίνακα byte και τη φορτώνουμε στο έγγραφο χρησιμοποιώντας μια ροή μνήμης. Αυτό διασφαλίζει ότι το περιεχόμενο HTML ερμηνεύεται και αποδίδεται σωστά από το Aspose.Words.

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφο στον καθορισμένο κατάλογο σε μορφή DOCX.

```csharp
doc.Save(dataDir + "WorkingWithHtmlLoadOptions.PreferredControlType.docx", SaveFormat.Docx);
```

Αυτό αποθηκεύει το έγγραφο του Word με το αποδιδόμενο στοιχείο ελέγχου σύνθετου πλαισίου στην καθορισμένη θέση.
