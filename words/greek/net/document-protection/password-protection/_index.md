# Προστασία με κωδικό πρόσβασης σε έγγραφο του Word

## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας. Αυτό το βήμα διασφαλίζει ότι έχετε πρόσβαση σε όλες τις λειτουργίες που προσφέρει το Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
using System;
```

## Βήμα 1: Ρύθμιση του Έργου

Προτού προσθέσετε προστασία με κωδικό πρόσβασης στο έγγραφό σας, πρέπει να ρυθμίσετε το έργο σας. Ας ξεκινήσουμε.

### Δημιουργία νέου έργου

Ανοίξτε το Visual Studio και δημιουργήστε μια νέα εφαρμογή κονσόλας C#. Ονομάστε την με κάτι που μπορείτε να θυμάστε, όπως "WordDocumentProtection".

### Εγκατάσταση του Aspose.Words για .NET

Μπορείτε να εγκαταστήσετε το Aspose.Words για .NET μέσω του NuGet Package Manager. Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων, επιλέξτε "Διαχείριση πακέτων NuGet" και αναζητήστε "Aspose.Words". Εγκαταστήστε το πακέτο.

```shell
Install-Package Aspose.Words
```

## Βήμα 2: Φόρτωση ή δημιουργία εγγράφου Word

Τώρα που το έργο μας έχει ρυθμιστεί, ας δημιουργήσουμε ένα έγγραφο του Word που μπορούμε να προστατεύσουμε.

Στο δικό σου `Program.cs` αρχείο, αρχικοποιήστε μια νέα παρουσία του `Document` κλάση. Αυτή η κλάση αντιπροσωπεύει το έγγραφο του Word με το οποίο θα εργαστείτε.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

## Βήμα 3: Εφαρμογή προστασίας με κωδικό πρόσβασης

Εδώ συμβαίνει η μαγεία. Θα εφαρμόσουμε προστασία με κωδικό πρόσβασης στο έγγραφό μας για να αποτρέψουμε την μη εξουσιοδοτημένη πρόσβαση.

### Επιλέξτε Τύπο Προστασίας

Το Aspose.Words προσφέρει διαφορετικά είδη προστασίας, όπως `NoProtection`, `ReadOnly`, `AllowOnlyComments`, και `AllowOnlyFormFields`Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε `NoProtection` αλλά με κωδικό πρόσβασης, που ουσιαστικά σημαίνει ότι το έγγραφο είναι επεξεργάσιμο αλλά απαιτεί κωδικό πρόσβασης για την κατάργηση της προστασίας.

### Εφαρμογή προστασίας

Χρησιμοποιήστε το `Protect` μέθοδος του `Document` κλάση για την εφαρμογή προστασίας με κωδικό πρόσβασης. 

```csharp
// Εφαρμόστε προστασία εγγράφων.
doc.Protect(ProtectionType.NoProtection, "password");
```

## Βήμα 4: Αποθήκευση του προστατευμένου εγγράφου

Τέλος, ας αποθηκεύσουμε το προστατευμένο έγγραφό μας σε έναν καθορισμένο κατάλογο.


Χρησιμοποιήστε το `Save` μέθοδος για την αποθήκευση του εγγράφου σας. Δώστε τη διαδρομή όπου θέλετε να αποθηκεύσετε το έγγραφο μαζί με το όνομα αρχείου.

```csharp
doc.Save(dataDir + "DocumentProtection.PasswordProtection.docx");
```

Ναι, το Aspose.Words υποστηρίζει διάφορους τύπους προστασίας, συμπεριλαμβανομένων `ReadOnly`, `AllowOnlyComments`, και `AllowOnlyFormFields`.

Ναι, το Aspose.Words είναι συμβατό με .NET Core, .NET Framework και άλλες πλατφόρμες .NET.

Απολύτως! Μπορείτε να φορτώσετε ένα υπάρχον έγγραφο χρησιμοποιώντας το `Document` κλάση και στη συνέχεια εφαρμόστε προστασία.
