# Ορισμός συστήματος φακέλων γραμματοσειρών και προσαρμοσμένου φακέλου

## Εισαγωγή

Φανταστείτε ότι δημιουργείτε ένα έγγραφο με ένα μοναδικό στυλ γραμματοσειράς, μόνο και μόνο για να διαπιστώσετε ότι οι γραμματοσειρές δεν εμφανίζονται σωστά σε άλλο μηχάνημα. Απογοητευτικό, σωστά; Εδώ μπαίνει στο παιχνίδι η ρύθμιση των φακέλων γραμματοσειρών. Με το Aspose.Words για .NET, μπορείτε να ορίσετε φακέλους γραμματοσειρών συστήματος και προσαρμοσμένους φακέλους γραμματοσειρών για να διασφαλίσετε ότι τα έγγραφά σας θα φαίνονται πάντα όπως πρέπει. Ας δούμε πώς μπορείτε να το πετύχετε αυτό.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- Aspose.Words για τη βιβλιοθήκη .NET: Αν δεν το έχετε κάνει ήδη, κατεβάστε το [εδώ](https://releases.aspose.com/words/net/).
- Περιβάλλον Ανάπτυξης: Ένα IDE όπως το Visual Studio.
- Βασικές γνώσεις C#: Η εξοικείωση με την C# θα σας βοηθήσει να παρακολουθήσετε τα παραδείγματα κώδικα.

## Εισαγωγή χώρων ονομάτων

Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας:

```csharp
using System;
using System.Collections.Generic;
using Aspose.Words;
using Aspose.Words.Fonts;
```

Τώρα, ας αναλύσουμε τη διαδικασία σε απλά βήματα.

## Βήμα 1: Φόρτωση του εγγράφου

Για να ξεκινήσετε, φορτώστε το έγγραφο του Word σε ένα αρχείο Aspose.Words. `Document` αντικείμενο. Αυτό το έγγραφο θα είναι αυτό όπου θέλετε να ορίσετε τους φακέλους γραμματοσειρών.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
```

## Βήμα 2: Αρχικοποίηση ρυθμίσεων γραμματοσειράς

Δημιουργήστε μια νέα παρουσία του `FontSettings`Αυτό το αντικείμενο θα σας επιτρέψει να διαχειριστείτε τις πηγές γραμματοσειρών.

```csharp
FontSettings fontSettings = new FontSettings();
```

## Βήμα 3: Ανάκτηση πηγών γραμματοσειρών συστήματος

Ανακτήστε τις προεπιλεγμένες πηγές γραμματοσειρών συστήματος. Σε έναν υπολογιστή με Windows, αυτό συνήθως περιλαμβάνει τον κατάλογο "Windows\Fonts".

```csharp
List<FontSourceBase> fontSources = new List<FontSourceBase>(fontSettings.GetFontsSources());
```

## Βήμα 4: Προσθήκη φακέλου προσαρμοσμένων γραμματοσειρών

Προσθέστε έναν προσαρμοσμένο φάκελο που περιέχει τις επιπλέον γραμματοσειρές σας. Αυτό είναι χρήσιμο εάν έχετε συγκεκριμένες γραμματοσειρές που δεν είναι εγκατεστημένες στον κατάλογο γραμματοσειρών συστήματος.

```csharp
FolderFontSource folderFontSource = new FolderFontSource("C:\\MyFonts\\", true);
fontSources.Add(folderFontSource);
```

## Βήμα 5: Ενημέρωση πηγών γραμματοσειρών

Μετατρέψτε τη λίστα των πηγών γραμματοσειρών ξανά σε έναν πίνακα και ορίστε την σε `FontSettings` αντικείμενο.

```csharp
FontSourceBase[] updatedFontSources = fontSources.ToArray();
fontSettings.SetFontsSources(updatedFontSources);
```

## Βήμα 6: Εφαρμογή ρυθμίσεων γραμματοσειράς στο έγγραφο

Τέλος, εφαρμόστε το διαμορφωμένο `FontSettings` στο έγγραφό σας και αποθηκεύστε το στην επιθυμητή μορφή, όπως PDF.

```csharp
doc.FontSettings = fontSettings;
doc.Save(dataDir + "WorkingWithFonts.SetFontsFoldersSystemAndCustomFolder.pdf");
```

## Σύναψη

Και να το! Ακολουθώντας αυτά τα βήματα, μπορείτε να διασφαλίσετε ότι τα έγγραφά σας στο Word χρησιμοποιούν τις σωστές γραμματοσειρές, είτε πρόκειται για γραμματοσειρές συστήματος είτε για προσαρμοσμένες γραμματοσειρές που είναι αποθηκευμένες σε έναν συγκεκριμένο κατάλογο. Αυτή η ρύθμιση βοηθά στη διατήρηση της ακεραιότητας της εμφάνισης του εγγράφου σας σε διαφορετικά περιβάλλοντα.

## Συχνές ερωτήσεις

### Τι συμβαίνει εάν μια γραμματοσειρά λείπει τόσο από τον φάκελο συστήματος όσο και από τους προσαρμοσμένους φακέλους;

Το Aspose.Words θα χρησιμοποιήσει μια προεπιλεγμένη γραμματοσειρά για να αντικαταστήσει τη γραμματοσειρά που λείπει, διασφαλίζοντας ότι το έγγραφο παραμένει ευανάγνωστο.

### Μπορώ να προσθέσω πολλούς φακέλους προσαρμοσμένων γραμματοσειρών;

Ναι, μπορείτε να προσθέσετε πολλούς προσαρμοσμένους φακέλους γραμματοσειρών επαναλαμβάνοντας τη διαδικασία δημιουργίας `FolderFontSource` αντικείμενα και προσθήκη τους στη λίστα πηγών γραμματοσειρών.

### Είναι δυνατή η χρήση διαδρομών δικτύου για προσαρμοσμένους φακέλους γραμματοσειρών;

Ναι, μπορείτε να καθορίσετε μια διαδρομή δικτύου στο `FolderFontSource` κατασκευαστής.

### Ποιες μορφές αρχείων υποστηρίζει το Aspose.Words για την αποθήκευση εγγράφων;

Το Aspose.Words υποστηρίζει διάφορες μορφές, όπως DOCX, PDF, HTML και άλλες.

### Πώς μπορώ να χειριστώ τις ειδοποιήσεις αντικατάστασης γραμματοσειράς;

Μπορείτε να διαχειριστείτε τις ειδοποιήσεις αντικατάστασης γραμματοσειράς χρησιμοποιώντας το `FontSettings` τάξης `FontSubstitutionWarning` συμβάν.
