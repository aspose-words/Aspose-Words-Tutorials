# Ενεργοποίηση Απενεργοποίηση Υποκατάστασης Γραμματοσειράς

## Εισαγωγή

Έχετε βρεθεί ποτέ σε μια κατάσταση όπου οι σχολαστικά επιλεγμένες γραμματοσειρές σας σε ένα έγγραφο του Word αντικαθίστανται όταν προβάλλονται σε άλλον υπολογιστή; Ενοχλητικό, σωστά; Αυτό συμβαίνει λόγω της αντικατάστασης γραμματοσειρών, μιας διαδικασίας όπου το σύστημα αντικαθιστά μια γραμματοσειρά που λείπει με μια διαθέσιμη. Αλλά μην ανησυχείτε! Με το Aspose.Words για .NET, μπορείτε εύκολα να διαχειριστείτε και να ελέγξετε την αντικατάσταση γραμματοσειρών. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στα βήματα για να ενεργοποιήσετε ή να απενεργοποιήσετε την αντικατάσταση γραμματοσειρών στα έγγραφά σας του Word, διασφαλίζοντας ότι τα έγγραφά σας φαίνονται πάντα ακριβώς όπως τα θέλετε.

## Προαπαιτούμενα

Πριν προχωρήσουμε στα βήματα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

- Aspose.Words για .NET: Λήψη της τελευταίας έκδοσης [εδώ](https://releases.aspose.com/words/net/).
- Visual Studio: Οποιαδήποτε έκδοση υποστηρίζει .NET.
- Βασικές γνώσεις C#: Αυτό θα σας βοηθήσει να παρακολουθήσετε τα παραδείγματα κωδικοποίησης.

## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, βεβαιωθείτε ότι έχετε εισαγάγει τους απαραίτητους χώρους ονομάτων στο έργο σας. Προσθέστε τους στο επάνω μέρος του αρχείου C#:

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

Τώρα, ας αναλύσουμε τη διαδικασία σε απλά, διαχειρίσιμα βήματα.

## Βήμα 1: Ρύθμιση του έργου σας

Αρχικά, δημιουργήστε ένα νέο έργο στο Visual Studio και προσθέστε μια αναφορά στη βιβλιοθήκη Aspose.Words για .NET. Εάν δεν το έχετε κάνει ήδη, κατεβάστε το από το [Ιστότοπος Aspose](https://releases.aspose.com/words/net/).

## Βήμα 2: Φόρτωση του εγγράφου σας

Στη συνέχεια, φορτώστε το έγγραφο με το οποίο θέλετε να εργαστείτε. Δείτε πώς μπορείτε να το κάνετε:

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Rendering.docx");
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας. Αυτός ο κώδικας φορτώνει το έγγραφο στη μνήμη, ώστε να μπορείτε να το χειριστείτε.

## Βήμα 3: Ρύθμιση παραμέτρων γραμματοσειράς

Τώρα, ας δημιουργήσουμε ένα `FontSettings` αντικείμενο για τη διαχείριση των ρυθμίσεων αντικατάστασης γραμματοσειράς:

```csharp
FontSettings fontSettings = new FontSettings();
```

## Βήμα 4: Ορισμός προεπιλεγμένης αντικατάστασης γραμματοσειράς

Ορίστε την προεπιλεγμένη αντικατάσταση γραμματοσειράς σε μια γραμματοσειρά της επιλογής σας. Αυτή η γραμματοσειρά θα χρησιμοποιηθεί εάν η αρχική γραμματοσειρά δεν είναι διαθέσιμη:

```csharp
fontSettings.SubstitutionSettings.DefaultFontSubstitution.DefaultFontName = "Arial";
```

Σε αυτό το παράδειγμα, χρησιμοποιούμε την Arial ως προεπιλεγμένη γραμματοσειρά.

## Βήμα 5: Απενεργοποίηση αντικατάστασης πληροφοριών γραμματοσειράς

Για να απενεργοποιήσετε την αντικατάσταση πληροφοριών γραμματοσειράς, η οποία εμποδίζει το σύστημα να αντικαταστήσει τις γραμματοσειρές που λείπουν με τις διαθέσιμες, χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
fontSettings.SubstitutionSettings.FontInfoSubstitution.Enabled = false;
```

## Βήμα 6: Εφαρμογή ρυθμίσεων γραμματοσειράς στο έγγραφο

Τώρα, εφαρμόστε αυτές τις ρυθμίσεις στο έγγραφό σας:

```csharp
doc.FontSettings = fontSettings;
```

## Βήμα 7: Αποθήκευση του εγγράφου σας

Τέλος, αποθηκεύστε το τροποποιημένο έγγραφό σας. Μπορείτε να το αποθηκεύσετε σε οποιαδήποτε μορφή θέλετε. Για αυτό το σεμινάριο, θα το αποθηκεύσουμε ως PDF:

```csharp
doc.Save(dataDir + "WorkingWithFonts.EnableDisableFontSubstitution.pdf");
```

## Σύναψη

Και να το! Ακολουθώντας αυτά τα βήματα, μπορείτε εύκολα να ελέγξετε την αντικατάσταση γραμματοσειρών στα έγγραφά σας στο Word χρησιμοποιώντας το Aspose.Words για .NET. Αυτό διασφαλίζει ότι τα έγγραφά σας διατηρούν την προβλεπόμενη εμφάνιση και αίσθηση, ανεξάρτητα από το πού τα βλέπετε.

## Συχνές ερωτήσεις

### Μπορώ να χρησιμοποιήσω άλλες γραμματοσειρές εκτός από την Arial για αντικατάσταση;

Απολύτως! Μπορείτε να καθορίσετε οποιαδήποτε γραμματοσειρά είναι διαθέσιμη στο σύστημά σας αλλάζοντας το όνομα της γραμματοσειράς στο `DefaultFontName` ιδιοκτησία.

### Τι συμβαίνει εάν η καθορισμένη προεπιλεγμένη γραμματοσειρά δεν είναι διαθέσιμη;

Εάν η προεπιλεγμένη γραμματοσειρά δεν είναι διαθέσιμη, το Aspose.Words θα χρησιμοποιήσει έναν μηχανισμό εφεδρείας συστήματος για να βρει μια κατάλληλη αντικατάσταση.

### Μπορώ να ενεργοποιήσω ξανά την αντικατάσταση γραμματοσειρών αφού την απενεργοποιήσω;

Ναι, μπορείτε να ενεργοποιήσετε/απενεργοποιήσετε το `Enabled` ιδιοκτησία του `FontInfoSubstitution` πίσω στο `true` αν θέλετε να ενεργοποιήσετε ξανά την αντικατάσταση γραμματοσειρών.

### Υπάρχει τρόπος να ελέγξω ποιες γραμματοσειρές αντικαθίστανται;

Ναι, το Aspose.Words παρέχει μεθόδους για την καταγραφή και παρακολούθηση της αντικατάστασης γραμματοσειρών, επιτρέποντάς σας να δείτε ποιες γραμματοσειρές αντικαθίστανται.

### Μπορώ να χρησιμοποιήσω αυτήν τη μέθοδο για άλλες μορφές εγγράφων εκτός από το DOCX;

Σίγουρα! Το Aspose.Words υποστηρίζει διάφορες μορφές και μπορείτε να εφαρμόσετε αυτές τις ρυθμίσεις γραμματοσειράς σε οποιαδήποτε υποστηριζόμενη μορφή.
