# Λήψη λίστας με τις διαθέσιμες γραμματοσειρές

## Εισαγωγή

Δυσκολεύεστε ποτέ να διαχειριστείτε γραμματοσειρές στα έγγραφά σας στο Word; Αν είστε προγραμματιστής .NET, το Aspose.Words για .NET είναι εδώ για να σας σώσει! Αυτή η ισχυρή βιβλιοθήκη όχι μόνο σας βοηθά να δημιουργείτε και να χειρίζεστε έγγραφα Word μέσω προγραμματισμού, αλλά προσφέρει επίσης εκτεταμένες δυνατότητες διαχείρισης γραμματοσειρών. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε σε ένα βήμα προς βήμα σεμινάριο για το πώς να λάβετε μια λίστα με τις διαθέσιμες γραμματοσειρές χρησιμοποιώντας το Aspose.Words για .NET. Θα το αναλύσουμε σε εύπεπτα βήματα για να διασφαλίσουμε ότι μπορείτε να παρακολουθείτε με ευκολία. Ας εμβαθύνουμε λοιπόν και ας κάνουμε τη διαχείριση γραμματοσειρών παιχνιδάκι!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, υπάρχουν μερικά πράγματα που θα χρειαστείτε:

- Aspose.Words για .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να την κατεβάσετε από [εδώ](https://releases.aspose.com/words/net/).
- Visual Studio: Αυτό το παράδειγμα χρησιμοποιεί το Visual Studio ως περιβάλλον ανάπτυξης.
- .NET Framework: Βεβαιωθείτε ότι έχετε εγκαταστήσει το .NET Framework στον υπολογιστή σας.
- Κατάλογος εγγράφων: Μια διαδρομή καταλόγου όπου αποθηκεύονται τα έγγραφά σας.

## Εισαγωγή χώρων ονομάτων

Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας:

```csharp
using System;
using System.Collections.Generic;
using Aspose.Words;
using Aspose.Words.Fonts;
```

## Βήμα 1: Αρχικοποίηση ρυθμίσεων γραμματοσειράς

Το πρώτο βήμα είναι να αρχικοποιήσετε τις ρυθμίσεις γραμματοσειράς. Αυτό θα σας επιτρέψει να διαχειριστείτε τις πηγές γραμματοσειρών για τα έγγραφά σας.

```csharp
FontSettings fontSettings = new FontSettings();
List<FontSourceBase> fontSources = new List<FontSourceBase>(fontSettings.GetFontsSources());
```

- Ρυθμίσεις γραμματοσειράς: Αυτή η κλάση χρησιμοποιείται για να καθορίσει τις ρυθμίσεις για την αντικατάσταση γραμματοσειρών και τις πηγές γραμματοσειρών.
- fontSources: Δημιουργούμε μια λίστα με τις υπάρχουσες πηγές γραμματοσειρών από τις τρέχουσες ρυθμίσεις γραμματοσειράς.

## Βήμα 2: Ορισμός καταλόγου εγγράφων

Στη συνέχεια, καθορίστε τη διαδρομή προς τον κατάλογο του εγγράφου σας. Εδώ θα αναζητήσει το Aspose.Words γραμματοσειρές.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

- dataDir: Αυτή η μεταβλητή συμβολοσειράς περιέχει τη διαδρομή προς τον κατάλογο όπου βρίσκονται οι γραμματοσειρές σας. Αντικαταστήστε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή.

## Βήμα 3: Προσθήκη φακέλου προσαρμοσμένων γραμματοσειρών

Τώρα, προσθέστε έναν νέο φάκελο προέλευσης για να δώσετε εντολή στο Aspose.Words να αναζητήσει γραμματοσειρές σε αυτόν τον φάκελο.

```csharp
FolderFontSource folderFontSource = new FolderFontSource(dataDir, true);
```

- FolderFontSource: Αυτή η κλάση αντιπροσωπεύει μια πηγή γραμματοσειράς φακέλου. Η δεύτερη παράμετρος (`true`υποδεικνύει εάν θα γίνεται αναδρομική αναζήτηση γραμματοσειρών σε υποφακέλους.

## Βήμα 4: Ενημέρωση πηγών γραμματοσειρών

Προσθέστε τον φάκελο προσαρμοσμένων γραμματοσειρών στη λίστα των υπαρχουσών πηγών γραμματοσειρών και ενημερώστε τις ρυθμίσεις γραμματοσειράς.

```csharp
fontSources.Add(folderFontSource);
FontSourceBase[] updatedFontSources = fontSources.ToArray();
```

- fontSources.Add(folderFontSource): Προσθέτει τον προσαρμοσμένο φάκελο γραμματοσειρών στις υπάρχουσες πηγές γραμματοσειρών.
- updatedFontSources: Μετατρέπει τη λίστα των πηγών γραμματοσειρών σε έναν πίνακα.

## Βήμα 5: Ανάκτηση και εμφάνιση γραμματοσειρών

Τέλος, ανακτήστε τις διαθέσιμες γραμματοσειρές και εμφανίστε τις λεπτομέρειες τους.

```csharp
foreach (PhysicalFontInfo fontInfo in updatedFontSources[0].GetAvailableFonts())
{
    Console.WriteLine("FontFamilyName : " + fontInfo.FontFamilyName);
    Console.WriteLine("FullFontName  : " + fontInfo.FullFontName);
    Console.WriteLine("Version  : " + fontInfo.Version);
    Console.WriteLine("FilePath : " + fontInfo.FilePath);
}
```

- GetAvailableFonts(): Ανακτά τη λίστα με τις διαθέσιμες γραμματοσειρές από την πρώτη πηγή γραμματοσειρών στην ενημερωμένη λίστα.
- fontInfo: Μια παρουσία του `PhysicalFontInfo` που περιέχει λεπτομέρειες για κάθε γραμματοσειρά.

## Σύναψη

Συγχαρητήρια! Ανακτήσατε με επιτυχία μια λίστα με τις διαθέσιμες γραμματοσειρές χρησιμοποιώντας το Aspose.Words για .NET. Αυτό το σεμινάριο σας καθοδηγεί σε κάθε βήμα, από την αρχικοποίηση των ρυθμίσεων γραμματοσειράς έως την εμφάνιση λεπτομερειών γραμματοσειράς. Με αυτές τις γνώσεις, μπορείτε πλέον να διαχειρίζεστε εύκολα τις γραμματοσειρές στα έγγραφά σας στο Word. Να θυμάστε ότι το Aspose.Words για .NET είναι ένα ισχυρό εργαλείο που μπορεί να βελτιώσει σημαντικά τις δυνατότητες επεξεργασίας εγγράφων σας. Συνεχίστε λοιπόν και εξερευνήστε περισσότερες λειτουργίες για να κάνετε τη διαδικασία ανάπτυξης ακόμα πιο αποτελεσματική.

## Συχνές ερωτήσεις

### Μπορώ να χρησιμοποιήσω το Aspose.Words για .NET με άλλα .NET frameworks;
Ναι, το Aspose.Words για .NET είναι συμβατό με διάφορα .NET frameworks, συμπεριλαμβανομένων των .NET Core και .NET 5+.

### Πώς μπορώ να εγκαταστήσω το Aspose.Words για .NET;
Μπορείτε να το εγκαταστήσετε μέσω του NuGet Package Manager στο Visual Studio αναζητώντας "Aspose.Words".

### Είναι δυνατή η προσθήκη πολλαπλών φακέλων προσαρμοσμένων γραμματοσειρών;
Ναι, μπορείτε να προσθέσετε πολλούς προσαρμοσμένους φακέλους γραμματοσειρών δημιουργώντας πολλούς `FolderFontSource` παρουσίες και προσθήκη τους στη λίστα πηγών γραμματοσειρών.

### Μπορώ να ανακτήσω λεπτομέρειες γραμματοσειράς από μια συγκεκριμένη πηγή γραμματοσειράς;
Ναι, μπορείτε να ανακτήσετε λεπτομέρειες γραμματοσειράς από οποιαδήποτε πηγή γραμματοσειράς καθορίζοντας τον δείκτη της πηγής γραμματοσειράς στο `updatedFontSources` παράταξη.

### Υποστηρίζει το Aspose.Words για .NET την αντικατάσταση γραμματοσειρών;
Ναι, υποστηρίζει την αντικατάσταση γραμματοσειράς για να διασφαλιστεί ότι το κείμενο αποδίδεται σωστά, ακόμη και αν η αρχική γραμματοσειρά δεν είναι διαθέσιμη.
