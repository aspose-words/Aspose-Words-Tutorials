# Ορισμός φακέλων γραμματοσειρών

## Εισαγωγή

Γεια σας! Είστε έτοιμοι να βουτήξετε στον κόσμο των προσαρμοσμένων γραμματοσειρών στο Aspose.Words για .NET; Ας ξεκινήσουμε. Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία ρύθμισης προσαρμοσμένων φακέλων γραμματοσειρών, διασφαλίζοντας ότι τα έγγραφά σας θα φαίνονται ακριβώς όπως τα θέλετε. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτός ο οδηγός θα σας καθοδηγήσει σε κάθε βήμα. Ας κάνουμε, λοιπόν, αυτές τις γραμματοσειρές να φαίνονται υπέροχες!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

- Aspose.Words για .NET: Μπορείτε [λήψη](https://releases.aspose.com/words/net/) το αν δεν το έχετε κάνει ήδη.
- Visual Studio: Οποιαδήποτε έκδοση θα λειτουργήσει, αλλά η πιο πρόσφατη είναι πάντα η καλύτερη.
- Ένα έγγραφο: Θα χρησιμοποιήσουμε ένα έγγραφο του Word για αυτό το σεμινάριο. Μπορείτε να δημιουργήσετε το δικό σας ή να χρησιμοποιήσετε ένα υπάρχον.
- Προσαρμοσμένες γραμματοσειρές: Έχετε έτοιμες μερικές προσαρμοσμένες γραμματοσειρές. Θα τις χρησιμοποιήσουμε για να δείξουμε πώς να ορίσετε φακέλους γραμματοσειρών.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ' όλα, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό είναι απαραίτητο για την πρόσβαση στις κλάσεις και τις μεθόδους που χρειαζόμαστε από το Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Fonts;
```

Με την εισαγωγή αυτών των χώρων ονομάτων, είμαστε έτοιμοι να ξεκινήσουμε τη ρύθμιση των προσαρμοσμένων φακέλων γραμματοσειρών μας.

## Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας

Ας ξεκινήσουμε ορίζοντας τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ αποθηκεύεται το έγγραφο του Word σας. Θα χρησιμοποιήσουμε μια μεταβλητή που ονομάζεται `dataDir` για να αποθηκεύσετε αυτήν τη διαδρομή.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογό σας. Αυτό είναι κρίσιμο επειδή το Aspose.Words θα πρέπει να γνωρίζει πού να βρει το έγγραφό σας.

## Βήμα 2: Ορισμός πηγών γραμματοσειράς

Στη συνέχεια, πρέπει να ρυθμίσουμε τις πηγές γραμματοσειρών. Εδώ λέμε στο Aspose.Words πού να βρει τις προσαρμοσμένες γραμματοσειρές μας. Θα χρησιμοποιήσουμε το `FontSettings.DefaultInstance.SetFontsSources` μέθοδος για να επιτευχθεί αυτό.

```csharp
FontSettings.DefaultInstance.SetFontsSources(new FontSourceBase[]
{
	new SystemFontSource(), new FolderFontSource("C:\\MyFonts\\", true)
});
```

Να τι κάνουμε:

- SystemFontSource: Αυτό υποδεικνύει στο Aspose.Words να χρησιμοποιήσει τις προεπιλεγμένες γραμματοσειρές του συστήματος.
- FolderFontSource: Εδώ καθορίζουμε τον φάκελο που περιέχει τις προσαρμοσμένες γραμματοσειρές μας. Αντικατάσταση `"C:\\MyFonts\\"` με τη διαδρομή προς τον κατάλογο προσαρμοσμένων γραμματοσειρών σας. Το `true` Η παράμετρος υποδεικνύει ότι θα πρέπει επίσης να συμπεριληφθούν και οι υποκατάλογοι.

## Βήμα 3: Φόρτωση του εγγράφου σας

Τώρα που έχουμε ρυθμίσει τις πηγές γραμματοσειρών μας, ήρθε η ώρα να φορτώσουμε το έγγραφο με το οποίο θέλουμε να εργαστούμε. Θα χρησιμοποιήσουμε το `Document` τάξη από το Aspose. Λέξεις για αυτό.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

Βεβαιωθείτε ότι `"Rendering.docx"` είναι το όνομα του εγγράφου σας στο Word. Εάν το έγγραφό σας έχει διαφορετικό όνομα, φροντίστε να το ενημερώσετε ανάλογα.

## Βήμα 4: Αποθηκεύστε το έγγραφό σας ως PDF

Τέλος, ας αποθηκεύσουμε το έγγραφό μας ως PDF για να δούμε τις προσαρμοσμένες γραμματοσειρές σε δράση. Θα χρησιμοποιήσουμε το `Save` μέθοδος του `Document` τάξη.

```csharp
doc.Save(dataDir + "WorkingWithFonts.SetFontsFolders.pdf");
```

Αυτό θα αποθηκεύσει το έγγραφό σας ως PDF στον καθορισμένο κατάλογο, χρησιμοποιώντας τις προσαρμοσμένες γραμματοσειρές που ορίσαμε νωρίτερα.

## Σύναψη

Και να το! Έχετε ρυθμίσει με επιτυχία φακέλους προσαρμοσμένων γραμματοσειρών στο Aspose.Words για .NET και έχετε αποθηκεύσει το έγγραφό σας ως PDF με αυτές τις προσαρμοσμένες γραμματοσειρές. Πολύ ωραίο, σωστά; Η προσαρμογή των γραμματοσειρών μπορεί να κάνει τεράστια διαφορά στην εμφάνιση των εγγράφων σας και τώρα ξέρετε ακριβώς πώς να το κάνετε. Καλή κωδικοποίηση!
