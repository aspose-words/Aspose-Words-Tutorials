# Διακριτή διαγραφή

## Εισαγωγή

Καλώς ορίσατε σε αυτόν τον λεπτομερή οδηγό σχετικά με τον τρόπο εφαρμογής μορφοποίησης διακριτής διαγραφής σε κείμενο χρησιμοποιώντας το Aspose.Words για .NET. Αν θέλετε να βελτιώσετε τις δεξιότητές σας στην επεξεργασία εγγράφων και να προσθέσετε μια μοναδική πινελιά στο κείμενό σας, βρίσκεστε στο σωστό μέρος. Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- Aspose.Words για .NET: Κατεβάστε το [εδώ](https://releases.aspose.com/words/net/).
- .NET Framework: Βεβαιωθείτε ότι έχετε εγκαταστήσει το .NET Framework στο σύστημά σας.
- Περιβάλλον Ανάπτυξης: Ένα IDE όπως το Visual Studio.
- Βασικές γνώσεις C#: Απαραίτητη η εξοικείωση με τον προγραμματισμό C#.

## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Αυτοί είναι απαραίτητοι για την πρόσβαση στη βιβλιοθήκη Aspose.Words και στις δυνατότητές της.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Saving;
```

## Βήμα 1: Αρχικοποίηση του DocumentBuilder

Ο `DocumentBuilder` Η κλάση είναι ένα ισχυρό εργαλείο στο Aspose.Words που σας επιτρέπει να προσθέτετε περιεχόμενο στο έγγραφό σας με ευκολία.

```csharp
// Αρχικοποιήστε ένα DocumentBuilder.
DocumentBuilder builder = new DocumentBuilder();
```

## Βήμα 2: Ορισμός ιδιότητας διακριτής διαγραφής

Τώρα, ας εφαρμόσουμε την ιδιότητα διακριτής διαγραφής στο κείμενό μας. Αυτό περιλαμβάνει τον ορισμό του `StrikeThrough` ιδιοκτησία του `Font` αντιτίθεμαι `true`.

```csharp
// Κάντε το κείμενο διακριτό.
builder.Font.StrikeThrough = true;
```

## Βήμα 3: Γράψτε κείμενο με διακριτή διαγραφή

Με την ιδιότητα διακριτής διαγραφής ορισμένη, μπορούμε τώρα να προσθέσουμε το κείμενό μας. `Writeln` Η μέθοδος θα προσθέσει το κείμενο στο έγγραφο.

```csharp
// Γράψτε κείμενο με διακριτή διαγραφή.
builder.Writeln("This text will be StrikeThrough");
```
