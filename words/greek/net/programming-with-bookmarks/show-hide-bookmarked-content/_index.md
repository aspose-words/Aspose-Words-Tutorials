# Εμφάνιση απόκρυψης περιεχομένου σελιδοδεικτών σε έγγραφο του Word

## Εισαγωγή

Είστε έτοιμοι να βυθιστείτε στον κόσμο της διαχείρισης εγγράφων με το Aspose.Words για .NET; Είτε είστε προγραμματιστής που θέλει να αυτοματοποιήσει εργασίες εγγράφων είτε απλώς κάποιος που ενδιαφέρεται για τον προγραμματισμό αρχείων Word, βρίσκεστε στο σωστό μέρος. Σήμερα, θα εξερευνήσουμε πώς να εμφανίζουμε και να αποκρύπτουμε περιεχόμενο σελιδοδεικτών σε ένα έγγραφο Word χρησιμοποιώντας το Aspose.Words για .NET. Αυτός ο οδηγός βήμα προς βήμα θα σας κάνει επαγγελματία στον έλεγχο της ορατότητας περιεχομένου με βάση τους σελιδοδείκτες. Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν μπούμε στα πιο απλά, υπάρχουν μερικά πράγματα που θα χρειαστείτε:

1. Visual Studio: Οποιαδήποτε έκδοση συμβατή με το .NET.
2. Aspose.Words για .NET: Κατεβάστε το [εδώ](https://releases.aspose.com/words/net/).
3. Βασική Κατανόηση της C#: Αν μπορείτε να γράψετε ένα απλό πρόγραμμα τύπου "Hello World", είστε έτοιμοι.
4. Ένα έγγραφο του Word με σελιδοδείκτες: Θα χρησιμοποιήσουμε ένα δείγμα εγγράφου με σελιδοδείκτες για αυτό το σεμινάριο.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό διασφαλίζει ότι έχουμε όλα τα εργαλεία που χρειαζόμαστε για την εργασία μας.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Bookmark;
```

Με αυτούς τους χώρους ονομάτων στη θέση τους, είμαστε έτοιμοι να ξεκινήσουμε το ταξίδι μας.

## Βήμα 1: Ρύθμιση του έργου σας

Εντάξει, ας ξεκινήσουμε ρυθμίζοντας το έργο μας στο Visual Studio.

### Δημιουργία νέου έργου

Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο εφαρμογής κονσόλας (.NET Core). Ονομάστε το με κάτι πιασάρικο, όπως "BookmarkVisibilityManager".

### Προσθήκη Aspose.Words για .NET

Θα χρειαστεί να προσθέσετε το Aspose.Words για .NET στο έργο σας. Μπορείτε να το κάνετε αυτό μέσω του NuGet Package Manager.

1. Μεταβείτε στα Εργαλεία > Διαχειριστής πακέτων NuGet > Διαχείριση πακέτων NuGet για λύση.
2. Αναζήτηση για "Aspose.Words".
3. Εγκαταστήστε το πακέτο.

Τέλεια! Τώρα που το έργο μας έχει ρυθμιστεί, ας προχωρήσουμε στη φόρτωση του εγγράφου μας.

## Βήμα 2: Φόρτωση του εγγράφου

Πρέπει να φορτώσουμε το έγγραφο του Word που περιέχει τους σελιδοδείκτες. Για αυτό το σεμινάριο, θα χρησιμοποιήσουμε ένα δείγμα εγγράφου με το όνομα "Bookmarks.docx".

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Bookmarks.docx");
```

Αυτό το απόσπασμα κώδικα ορίζει τη διαδρομή προς τον κατάλογο εγγράφων σας και φορτώνει το έγγραφο στον `doc` αντικείμενο.

## Βήμα 3: Εμφάνιση/Απόκρυψη περιεχομένου σελιδοδεικτών

Τώρα έρχεται το διασκεδαστικό κομμάτι – η εμφάνιση ή η απόκρυψη του περιεχομένου με βάση τους σελιδοδείκτες. Θα δημιουργήσουμε μια μέθοδο που ονομάζεται `ShowHideBookmarkedContent` να το χειριστεί αυτό.

Ακολουθεί η μέθοδος που θα ενεργοποιήσει την ορατότητα του περιεχομένου που έχει προστεθεί στους σελιδοδείκτες:

```csharp
public void ShowHideBookmarkedContent(Document doc, string bookmarkName, bool isHidden)
{
    Bookmark bm = doc.Range.Bookmarks[bookmarkName];

    Node currentNode = bm.BookmarkStart;
    while (currentNode != null && currentNode.NodeType != NodeType.BookmarkEnd)
    {
        if (currentNode.NodeType == NodeType.Run)
        {
            Run run = currentNode as Run;
            run.Font.Hidden = isHidden;
        }
        currentNode = currentNode.NextSibling;
    }
}
```

### Ανάλυση της μεθόδου

- Ανάκτηση σελιδοδεικτών: `Bookmark bm = doc.Range.Bookmarks[bookmarkName];` ανακτά τον σελιδοδείκτη.
- Διέλευση Κόμβων: Διέλευση από τους κόμβους εντός του σελιδοδείκτη.
- Εναλλαγή ορατότητας: Εάν ο κόμβος είναι `Run` (μια συνεχής σειρά κειμένου), ορίζουμε το `Hidden` ιδιοκτησία.

## Βήμα 4: Εφαρμογή της μεθόδου

Έχοντας έτοιμη τη μέθοδό μας, ας την εφαρμόσουμε για να εμφανίσουμε ή να αποκρύψουμε περιεχόμενο με βάση έναν σελιδοδείκτη.

```csharp
ShowHideBookmarkedContent(doc, "MyBookmark1", true);
```

Αυτή η γραμμή κώδικα θα αποκρύψει το περιεχόμενο μέσα στον σελιδοδείκτη με το όνομα "MyBookmark1".

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, ας αποθηκεύσουμε το τροποποιημένο έγγραφό μας.

```csharp
doc.Save(dataDir + "WorkingWithBookmarks.ShowHideBookmarks.docx");
```

Αυτό αποθηκεύει το έγγραφο με τις αλλαγές που έχουμε κάνει.
