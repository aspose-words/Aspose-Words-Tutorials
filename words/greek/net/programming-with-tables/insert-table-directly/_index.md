# Εισαγωγή πίνακα απευθείας

## Εισαγωγή
Η δημιουργία πινάκων μέσω προγραμματισμού μπορεί να αποτελέσει μια αρκετά δύσκολη πρόκληση, ειδικά όταν πρόκειται για σύνθετες δομές εγγράφων. Αλλά μην ανησυχείτε, είμαστε εδώ για να σας την αναλύσουμε! Σε αυτόν τον οδηγό, θα σας δείξουμε τα βήματα εισαγωγής ενός πίνακα απευθείας σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτό το σεμινάριο θα σας βοηθήσει να κατακτήσετε τη διαδικασία με ευκολία.

## Προαπαιτούμενα

Πριν εμβαθύνουμε στον κώδικα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ξεκινήσετε. Ακολουθεί μια γρήγορη λίστα ελέγχου:

1. Aspose.Words για βιβλιοθήκη .NET: Βεβαιωθείτε ότι έχετε κατεβάσει και εγκαταστήσει τη βιβλιοθήκη Aspose.Words για .NET. Μπορείτε να την αποκτήσετε από το [σελίδα λήψης](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Ένα περιβάλλον ανάπτυξης όπως το Visual Studio.
3. Βασικές γνώσεις C#: Κατανόηση των βασικών στοιχείων του προγραμματισμού C#.
4. Κατάλογος εγγράφων: Μια διαδρομή καταλόγου όπου θα αποθηκεύσετε τα έγγραφά σας.

Με αυτές τις προϋποθέσεις, είστε έτοιμοι να ξεκινήσετε τον προγραμματισμό!

## Εισαγωγή χώρων ονομάτων

Αρχικά, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτοί οι χώροι ονομάτων θα μας παρέχουν τις κλάσεις και τις μεθόδους που απαιτούνται για την εργασία με έγγραφα του Word.

```csharp
using System;
using System.Drawing;
using Aspose.Words;
using Aspose.Words.Tables;
```

Τώρα που έχουμε ορίσει τους χώρους ονομάτων μας, ας προχωρήσουμε στο συναρπαστικό κομμάτι—τη δημιουργία και την εισαγωγή πινάκων απευθείας σε ένα έγγραφο του Word.

## Βήμα 1: Ρύθμιση του εγγράφου

Ας ξεκινήσουμε δημιουργώντας ένα νέο έγγραφο του Word. Εδώ θα εισαχθεί ο πίνακάς μας.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document();
```

Αυτός ο κώδικας αρχικοποιεί ένα νέο έγγραφο του Word. Θα χρειαστεί να αντικαταστήσετε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας.

## Βήμα 2: Δημιουργία του αντικειμένου πίνακα

Στη συνέχεια, δημιουργούμε το αντικείμενο πίνακα. Εδώ θα ορίσουμε τη δομή του πίνακά μας.

```csharp
// Ξεκινάμε δημιουργώντας το αντικείμενο πίνακα. Σημειώστε ότι πρέπει να περάσουμε το αντικείμενο εγγράφου.
// στον κατασκευαστή κάθε κόμβου. Αυτό συμβαίνει επειδή κάθε κόμβος που δημιουργούμε πρέπει να ανήκει
// σε κάποιο έγγραφο.
Table table = new Table(doc);
doc.FirstSection.Body.AppendChild(table);
```

Εδώ, δημιουργούμε έναν νέο πίνακα και τον προσθέτουμε στο σώμα της πρώτης ενότητας του εγγράφου μας.

## Βήμα 3: Προσθήκη γραμμών και κελιών

Ένας πίνακας αποτελείται από γραμμές και κελιά. Ας προσθέσουμε αυτά τα στοιχεία βήμα προς βήμα.

### Προσθήκη γραμμής

```csharp
// Εδώ θα μπορούσαμε να καλέσουμε το EnsureMinimum για να δημιουργήσει τις γραμμές και τα κελιά για εμάς. Αυτή η μέθοδος χρησιμοποιείται
// για να διασφαλιστεί ότι ο καθορισμένος κόμβος είναι έγκυρος. Σε αυτήν την περίπτωση, ένας έγκυρος πίνακας θα πρέπει να έχει τουλάχιστον μία γραμμή και ένα κελί.
// Αντίθετα, θα αναλάβουμε εμείς οι ίδιοι τη δημιουργία της γραμμής και του πίνακα.
// Αυτός θα ήταν ο καλύτερος τρόπος για να το κάνουμε αυτό αν δημιουργούσαμε έναν πίνακα μέσα σε έναν αλγόριθμο.
Row row = new Row(doc);
row.RowFormat.AllowBreakAcrossPages = true;
table.AppendChild(row);
```

Αυτός ο κώδικας δημιουργεί μια νέα γραμμή και την προσθέτει στον πίνακά μας.

### Προσθήκη κελιών στη γραμμή

Τώρα, ας προσθέσουμε μερικά κελιά στη σειρά μας. 

```csharp
Cell cell = new Cell(doc);
cell.CellFormat.Shading.BackgroundPatternColor = Color.LightBlue;
cell.CellFormat.Width = 80;
cell.AppendChild(new Paragraph(doc));
cell.FirstParagraph.AppendChild(new Run(doc, "Row 1, Cell 1 Text"));
row.AppendChild(cell);
```

Σε αυτό το απόσπασμα, δημιουργούμε ένα κελί, ορίζουμε το χρώμα φόντου του σε ανοιχτό μπλε και ορίζουμε το πλάτος του. Στη συνέχεια, προσθέτουμε μια παράγραφο και μια γραμμή στο κελί για να χωρέσει το κείμενό μας.

## Βήμα 4: Κλωνοποίηση κυττάρων

Για να επιταχύνουμε τη διαδικασία προσθήκης κελιών, μπορούμε να κλωνοποιήσουμε υπάρχοντα κελιά.

```csharp
// Στη συνέχεια, θα επαναλαμβάναμε τη διαδικασία για τα άλλα κελιά και γραμμές στον πίνακα.
// Μπορούμε επίσης να επιταχύνουμε τα πράγματα κλωνοποιώντας υπάρχοντα κελιά και γραμμές.
row.AppendChild(cell.Clone(false));
row.LastCell.AppendChild(new Paragraph(doc));
row.LastCell.FirstParagraph.AppendChild(new Run(doc, "Row 1, Cell 2 Text"));
```

Αυτός ο κώδικας κλωνοποιεί το υπάρχον κελί και το προσθέτει στη γραμμή. Στη συνέχεια, προσθέτουμε μια παράγραφο και μια εκτέλεση στο νέο κελί.

## Βήμα 5: Εφαρμογή ρυθμίσεων αυτόματης προσαρμογής

Τέλος, ας εφαρμόσουμε ρυθμίσεις αυτόματης προσαρμογής στον πίνακά μας για να διασφαλίσουμε ότι οι στήλες έχουν σταθερό πλάτος.

```csharp
// Τώρα μπορούμε να εφαρμόσουμε οποιεσδήποτε ρυθμίσεις αυτόματης προσαρμογής.
table.AutoFit(AutoFitBehavior.FixedColumnWidths);
```

## Βήμα 6: Αποθήκευση του εγγράφου

Αφού ο πίνακάς μας έχει ρυθμιστεί πλήρως, ήρθε η ώρα να αποθηκεύσουμε το έγγραφο.

```csharp
doc.Save(dataDir + "WorkingWithTables.InsertTableDirectly.docx");
```

Αυτός ο κώδικας αποθηκεύει το έγγραφο με τον πίνακα που έχει εισαχθεί.
