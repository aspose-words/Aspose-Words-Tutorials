# Ορισμός σχετικής οριζόντιας ή κάθετης θέσης

## Εισαγωγή

Νιώσατε ποτέ κολλημένοι με το πώς να τοποθετείτε πίνακες ακριβώς όπως θέλετε στα έγγραφά σας στο Word; Λοιπόν, δεν είστε οι μόνοι. Είτε δημιουργείτε μια επαγγελματική αναφορά είτε ένα κομψό φυλλάδιο, η ευθυγράμμιση πινάκων μπορεί να κάνει τη διαφορά. Εδώ είναι που το Aspose.Words για .NET είναι χρήσιμο. Αυτό το σεμινάριο θα σας καθοδηγήσει βήμα προς βήμα σχετικά με το πώς να ορίσετε σχετικές οριζόντιες ή κάθετες θέσεις για πίνακες στα έγγραφά σας στο Word. Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

1. Aspose.Words για .NET: Αν δεν το έχετε κάνει ήδη, μπορείτε να το κατεβάσετε [εδώ](https://releases.aspose.com/words/net/).
2. Περιβάλλον Ανάπτυξης: Visual Studio ή οποιοδήποτε άλλο IDE συμβατό με .NET.
3. Βασικές γνώσεις C#: Αυτό το σεμινάριο προϋποθέτει ότι είστε εξοικειωμένοι με τα βασικά του προγραμματισμού C#.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Αυτό είναι απαραίτητο για την πρόσβαση στις λειτουργίες του Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## Βήμα 1: Φόρτωση του εγγράφου σας

Για να ξεκινήσετε, θα χρειαστεί να φορτώσετε το έγγραφο του Word στο πρόγραμμα. Δείτε πώς μπορείτε να το κάνετε:

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table wrapped by text.docx");
```

Αυτό το απόσπασμα κώδικα ορίζει τη διαδρομή προς τον κατάλογο εγγράφων σας και φορτώνει το συγκεκριμένο έγγραφο στο οποίο θέλετε να εργαστείτε. Βεβαιωθείτε ότι η διαδρομή του εγγράφου σας είναι σωστή για να αποφύγετε τυχόν προβλήματα φόρτωσης.

## Βήμα 2: Πρόσβαση στον πίνακα

Στη συνέχεια, πρέπει να έχουμε πρόσβαση στον πίνακα μέσα στο έγγραφο. Συνήθως, θα θέλατε να εργαστείτε με τον πρώτο πίνακα στην ενότητα του κυρίως μέρους.

```csharp
Table table = doc.FirstSection.Body.Tables[0];
```

Αυτή η γραμμή κώδικα ανακτά τον πρώτο πίνακα από το σώμα του εγγράφου. Εάν το έγγραφό σας έχει πολλούς πίνακες, μπορείτε να προσαρμόσετε το ευρετήριο ανάλογα.

## Βήμα 3: Ορισμός οριζόντιας θέσης

Τώρα, ας ορίσουμε την οριζόντια θέση του πίνακα σε σχέση με ένα συγκεκριμένο στοιχείο. Σε αυτό το παράδειγμα, θα τον τοποθετήσουμε σε σχέση με τη στήλη.

```csharp
table.HorizontalAnchor = RelativeHorizontalPosition.Column;
```

Ρυθμίζοντας το `HorizontalAnchor` να `RelativeHorizontalPosition.Column`, λέτε στον πίνακα να ευθυγραμμιστεί οριζόντια σε σχέση με τη στήλη στην οποία βρίσκεται.

## Βήμα 4: Ορισμός κάθετης θέσης

Όπως και με την οριζόντια τοποθέτηση, μπορείτε επίσης να ορίσετε την κατακόρυφη θέση. Εδώ, την τοποθετούμε σε σχέση με τη σελίδα.

```csharp
table.VerticalAnchor = RelativeVerticalPosition.Page;
```

Ρύθμιση του `VerticalAnchor` να `RelativeVerticalPosition.Page` διασφαλίζει ότι ο πίνακας είναι κάθετα ευθυγραμμισμένος σύμφωνα με τη σελίδα.

## Βήμα 5: Αποθήκευση του εγγράφου σας

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο έγγραφο. Αυτό είναι ένα κρίσιμο βήμα για να βεβαιωθείτε ότι οι αλλαγές σας θα διατηρηθούν.

```csharp
doc.Save(dataDir + "WorkingWithTables.SetFloatingTablePosition.docx");
```

Αυτή η εντολή αποθηκεύει το τροποποιημένο έγγραφο με νέο όνομα, διασφαλίζοντας ότι δεν θα αντικαταστήσετε το αρχικό σας αρχείο.
