# Μορφή γραμμής Απενεργοποίηση διαχωρισμού σε σελίδες

## Εισαγωγή

Όταν εργάζεστε με πίνακες σε έγγραφα του Word, ίσως θελήσετε να βεβαιωθείτε ότι οι γραμμές δεν διακόπτονται μεταξύ των σελίδων, κάτι που μπορεί να είναι απαραίτητο για τη διατήρηση της αναγνωσιμότητας και της μορφοποίησης των εγγράφων σας. Το Aspose.Words για .NET παρέχει έναν εύκολο τρόπο απενεργοποίησης των διακοπτών γραμμών μεταξύ των σελίδων.

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία απενεργοποίησης των αλλαγών γραμμών σε σελίδες σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:
- Εγκατεστημένο το Aspose.Words για τη βιβλιοθήκη .NET.
- Ένα έγγραφο του Word με έναν πίνακα που εκτείνεται σε πολλές σελίδες.

## Εισαγωγή χώρων ονομάτων

Αρχικά, εισαγάγετε τους απαραίτητους χώρους ονομάτων στο έργο σας:

```csharp
using Aspose.Words;
using Aspose.Words.Tables;
```

## Βήμα 1: Φόρτωση του εγγράφου

Φορτώστε το έγγραφο που περιέχει τον πίνακα που εκτείνεται σε πολλές σελίδες.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας 
string dataDir = "YOUR DOCUMENT DIRECTORY";

Document doc = new Document(dataDir + "Table spanning two pages.docx");
```

## Βήμα 2: Πρόσβαση στον πίνακα

Αποκτήστε πρόσβαση στον πρώτο πίνακα στο έγγραφο. Αυτό προϋποθέτει ότι ο πίνακας που θέλετε να τροποποιήσετε είναι ο πρώτος πίνακας στο έγγραφο.

```csharp
Table table = (Table) doc.GetChild(NodeType.Table, 0, true);
```

## Βήμα 3: Απενεργοποίηση διαχωρισμού σελίδων για όλες τις γραμμές

Περιηγηθείτε σε κάθε γραμμή στον πίνακα και ορίστε το `AllowBreakAcrossPages` ιδιοκτησία σε `false`Αυτό διασφαλίζει ότι οι γραμμές δεν θα διασπώνται στις σελίδες.

```csharp
// Απενεργοποίηση διαχωρισμού σελίδων για όλες τις γραμμές στον πίνακα.
foreach (Row row in table.Rows)
    row.RowFormat.AllowBreakAcrossPages = false;
```

## Βήμα 4: Αποθήκευση του εγγράφου

Αποθηκεύστε το τροποποιημένο έγγραφο στον καθορισμένο κατάλογο.

```csharp
doc.Save(dataDir + "WorkingWithTables.RowFormatDisableBreakAcrossPages.docx");
```

## Σύναψη

Σε αυτό το σεμινάριο, δείξαμε πώς να απενεργοποιήσετε τις αλλαγές γραμμών σε σελίδες σε ένα έγγραφο του Word χρησιμοποιώντας το Aspose.Words για .NET. Ακολουθώντας τα βήματα που περιγράφονται παραπάνω, μπορείτε να διασφαλίσετε ότι οι γραμμές του πίνακά σας παραμένουν άθικτες και δεν διαιρούνται σε σελίδες, διατηρώντας την αναγνωσιμότητα και τη μορφοποίηση του εγγράφου.

## Συχνές ερωτήσεις

### Μπορώ να απενεργοποιήσω τις αλλαγές γραμμών σε όλες τις σελίδες για μια συγκεκριμένη γραμμή αντί για όλες τις γραμμές;  
Ναι, μπορείτε να απενεργοποιήσετε τις αλλαγές γραμμών για συγκεκριμένες γραμμές, αποκτώντας πρόσβαση στην επιθυμητή γραμμή και ορίζοντας την `AllowBreakAcrossPages` ιδιοκτησία σε `false`.

### Λειτουργεί αυτή η μέθοδος για πίνακες με συγχωνευμένα κελιά;  
Ναι, αυτή η μέθοδος λειτουργεί για πίνακες με συγχωνευμένα κελιά. Η ιδιότητα `AllowBreakAcrossPages` ισχύει για ολόκληρη τη γραμμή, ανεξάρτητα από τη συγχώνευση κελιών.

### Θα λειτουργήσει αυτή η μέθοδος αν ο πίνακας είναι ενσωματωμένος μέσα σε έναν άλλο πίνακα;  
Ναι, μπορείτε να αποκτήσετε πρόσβαση και να τροποποιήσετε τους ένθετους πίνακες με τον ίδιο τρόπο. Βεβαιωθείτε ότι αναφέρεστε σωστά στον ένθετο πίνακα μέσω του ευρετηρίου του ή άλλων ιδιοτήτων.

### Πώς μπορώ να ελέγξω αν μια γραμμή επιτρέπει τη διάσπαση σε σελίδες;  
Μπορείτε να ελέγξετε αν μια γραμμή επιτρέπει τη διάσπαση σελίδων αποκτώντας πρόσβαση στο `AllowBreakAcrossPages` ιδιοκτησία του `RowFormat` και ελέγχοντας την αξία του.

### Υπάρχει τρόπος να εφαρμοστεί αυτή η ρύθμιση σε όλους τους πίνακες ενός εγγράφου;  
Ναι, μπορείτε να κάνετε επανάληψη σε όλους τους πίνακες του εγγράφου και να εφαρμόσετε αυτήν τη ρύθμιση σε κάθε έναν από αυτούς.
