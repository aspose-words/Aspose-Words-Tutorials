# Αποθήκευση Εγγράφου ως PDF – Πλήρης Οδηγός Aspose.Words

Έχετε χρειαστεί ποτέ να **αποθηκεύσετε έγγραφο ως PDF** αλλά ανησυχείτε ότι το αποτέλεσμα μπορεί να φαίνεται διαφορετικό λόγω ελλιπών γραμματοσειρών; Δεν είστε μόνοι. Σε πολλές επιχειρηματικές εφαρμογές ένα αρχείο Word φτάνει στον διακομιστή, και η επόμενη γραμμή κώδικα πρέπει να παράγει ένα τέλειο PDF—ακόμη και όταν η αρχική γραμματοσειρά δεν είναι εγκατεστημένη.  

Σε αυτόν τον οδηγό θα σας δείξουμε ακριβώς πώς να **μετατρέψετε Word σε PDF**, να καταγράψετε προειδοποιήσεις **Aspose font substitution** και να **ανιχνεύσετε ελλιπείς γραμματοσειρές** ώστε να τις διορθώσετε πριν γίνουν εφιάλτης στην παραγωγή. Στο τέλος θα έχετε ένα έτοιμο‑για‑εκτέλεση απόσπασμα C# που κάνει όλα αυτά χωρίς κρυφή μαγεία.

> **Τι θα αποκτήσετε**  
> • Ένα πλήρες, εκτελέσιμο δείγμα κώδικα που φορτώνει ένα DOCX, καταχωρεί μια callback προειδοποίησης και αποθηκεύει ένα PDF.  
> • Μια εξήγηση του γιατί η callback προειδοποίησης είναι απαραίτητη για την ανίχνευση ελλιπών γραμματοσειρών.  
> • Πρακτικές συμβουλές για τη διαχείριση της αντικατάστασης γραμματοσειρών σε πραγματικές εφαρμογές.

## Προαπαιτήσεις

Πριν βουτήξουμε, βεβαιωθείτε ότι έχετε:

| Απαίτηση | Γιατί είναι σημαντικό |
|----------|------------------------|
| **Aspose.Words for .NET** (latest version) | Παρέχει την κλάση `Document` και την υποδομή προειδοποιήσεων. |
| **.NET 6+** (or .NET Framework 4.6+) | Εγγυάται συμβατότητα με την πιο πρόσφατη επιφάνεια API. |
| **A DOCX** that may reference fonts not installed on the server | Μας δίνει κάτι για να δοκιμάσουμε τη διαδρομή *ανίχνευσης ελλιπών γραμματοσειρών*. |
| **Visual Studio** (or any C# IDE) | Κάνει εύκολη την εκτέλεση και αποσφαλμάτωση του δείγματος. |

Δεν απαιτούνται πρόσθετα πακέτα NuGet πέρα από το `Aspose.Words`. Αν δεν το έχετε εγκαταστήσει ακόμη, εκτελέστε:

```bash
dotnet add package Aspose.Words
```

## Βήμα 1 – Φόρτωση του Πηγαίου Εγγράφου (Μετατροπή Word σε PDF)

Το πρώτο που κάνουμε είναι να ανοίξουμε το αρχείο Word. Το Aspose.Words διαβάζει ολόκληρη τη δομή του εγγράφου, συμπεριλαμβανομένων των αναφορών γραμματοσειρών, ώστε να γνωρίζει ακριβώς ποιες γραμματοσειρές χρειάζονται για τη μετατροπή σε PDF.

```csharp
using Aspose.Words;
using Aspose.Words.Warning;

// Replace with the actual path to your DOCX
string inputPath = @"C:\Docs\input.docx";

Document doc = new Document(inputPath);
```

> **Γιατί αυτό είναι σημαντικό:**  
> Η προημεροληπτική φόρτωση του εγγράφου επιτρέπει στο σύστημα προειδοποιήσεων να εξετάσει κάθε τμήμα κειμένου. Αν μια γραμματοσειρά δεν βρεθεί τοπικά, το Aspose θα εκδώσει μια προειδοποίηση `FontSubstitution` αργότερα—ιδανική για σενάρια **ανίχνευσης ελλιπών γραμματοσειρών**.

## Βήμα 2 – Καταχώρηση Callback Προειδοποίησης (Aspose Font Substitution)

Το Aspose.Words δεν ρίχνει εξαίρεση για ελλιπείς γραμματοσειρές· αντίθετα, εκδίδει προειδοποιήσεις. Ενσωματώνοντας ένα προσαρμοσμένο `IWarningCallback`, μπορούμε να συλλάβουμε αυτές τις προειδοποιήσεις και να αποφασίσουμε τι να κάνουμε—να τις καταγράψουμε, να αντικαταστήσουμε γραμματοσειρές ή ακόμη και να ακυρώσουμε τη μετατροπή.

```csharp
// Attach our custom callback before saving
doc.WarningCallback = new FontWarningHandler();
```

Η υλοποίηση του callback βρίσκεται μερικές γραμμές πιο κάτω, αλλά η ιδέα είναι απλή: ακούστε για `WarningType.FontSubstitution` και εκτυπώστε ένα φιλικό μήνυμα.

## Βήμα 3 – Αποθήκευση του Εγγράφου ως PDF

Τώρα τελικά **αποθηκεύουμε το έγγραφο ως PDF**. Αν πραγματοποιηθεί κάποια αντικατάσταση γραμματοσειράς, η callback θα έχει ήδη εκτυπώσει τις λεπτομέρειες στην κονσόλα.

```csharp
// Destination PDF path
string outputPath = @"C:\Docs\output.pdf";

// Perform the conversion
doc.Save(outputPath);
Console.WriteLine($"✅ PDF saved to {outputPath}");
```

Αυτό είναι—δύο γραμμές κώδικα μετατρέπουν ένα πιθανώς προβληματικό αρχείο Word σε καθαρό PDF ενώ σας ειδοποιούν για τυχόν ελλιπείς γραμματοσειρές.

## Βήμα 4 – Ο Διαχειριστής Προειδοποιήσεων Γραμματοσειρών (Ανίχνευση Ελλιπών Γραμματοσειρών)

Παρακάτω είναι η πλήρης υλοποίηση του διαχειριστή προειδοποιήσεων. Παρατηρήστε την προϋπόθεση `if (info.Type == WarningType.FontSubstitution)`—μας ενδιαφέρουν μόνο οι προειδοποιήσεις σχετικές με γραμματοσειρές, όχι άλλες όπως παρωχημένα χαρακτηριστικά.

```csharp
/// <summary>
/// Custom warning callback that logs font substitution warnings.
/// </summary>
class FontWarningHandler : IWarningCallback
{
    public void Warning(WarningInfo info)
    {
        // We’re only interested in font substitution warnings.
        if (info.Type == WarningType.FontSubstitution)
        {
            // The description already contains the missing font name.
            Console.WriteLine($"⚠️ Font substitution detected: {info.Description}");
        }
    }
}
```

**Αναμενόμενη έξοδος κονσόλας** όταν λείπει μια γραμματοσειρά:

```
⚠️ Font substitution detected: Font 'MySpecialFont' was not found. Substituted with 'Arial'.
✅ PDF saved to C:\Docs\output.pdf
```

Αν όλες οι γραμματοσειρές είναι παρούσες, θα δείτε μόνο τη γραμμή επιτυχίας.

## Βήμα 5 – Πλήρες, Έτοιμο‑για‑Εκτέλεση Παράδειγμα

Συνδυάζοντας όλα, εδώ είναι ένα μοναδικό αρχείο που μπορείτε να προσθέσετε σε ένα έργο κονσόλας και να το εκτελέσετε αμέσως.

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Warning;

namespace AsposePdfDemo
{
    class Program
    {
        static void Main()
        {
            // 1️⃣ Load the source DOCX (convert word to pdf later)
            string inputPath = @"C:\Docs\input.docx";
            Document doc = new Document(inputPath);

            // 2️⃣ Register the warning callback (detect missing fonts)
            doc.WarningCallback = new FontWarningHandler();

            // 3️⃣ Save as PDF (save document as pdf)
            string outputPath = @"C:\Docs\output.pdf";
            doc.Save(outputPath);

            Console.WriteLine($"✅ PDF saved to {outputPath}");
        }
    }

    /// <summary>
    /// Handles font substitution warnings emitted by Aspose.Words.
    /// </summary>
    class FontWarningHandler : IWarningCallback
    {
        public void Warning(WarningInfo info)
        {
            if (info.Type == WarningType.FontSubstitution)
            {
                Console.WriteLine($"⚠️ Font substitution detected: {info.Description}");
            }
        }
    }
}
```

**Εκτελέστε το**:

```bash
dotnet run
```

Θα πρέπει να δείτε είτε μόνο το μήνυμα επιτυχίας είτε μια προειδοποίηση ακολουθούμενη από επιτυχία, ανάλογα με τις γραμματοσειρές που είναι εγκατεστημένες στο σύστημά σας.

## Επαγγελματικές Συμβουλές & Συνηθισμένα Πιθανά Σφάλματα

| Κατάσταση | Τι να προσέξετε | Συνιστώμενη διόρθωση |
|-----------|-------------------|-----------------|
| **Απουσία προσαρμοσμένων αρχείων γραμματοσειρών** | Η προειδοποίηση θα αναφέρει το αρχικό όνομα της γραμματοσειράς. | Εγκαταστήστε τη γραμματοσειρά στον διακομιστή ή ενσωματώστε την στο DOCX (`File → Options → Save → Embed fonts`). |
| **Μεγάλα έγγραφα προκαλούν επιβράδυνση** | Κάθε αναζήτηση γραμματοσειράς προσθέτει επιπλέον φόρτο. | Προφορτώστε τις απαιτούμενες γραμματοσειρές σε μια προσαρμοσμένη συλλογή `FontSettings` και επαναχρησιμοποιήστε το ίδιο αντικείμενο `Document`. |
| **Εκτέλεση σε κοντέινερ χωρίς γραμματοσειρές** | Θα λάβετε πληθώρα προειδοποιήσεων αντικατάστασης. | Κρεμάστε τα απαιτούμενα αρχεία `.ttf`/`.otf` στο κοντέινερ και κατευθύνετε το Aspose σε αυτά μέσω `FontSettings`. |
| **Χρειάζεστε συγκεκριμένη εναλλακτική γραμματοσειρά** | Το Aspose χρησιμοποιεί προεπιλογή Arial. | Ορίστε `FontSettings.SubstitutionSettings.DefaultFontSubstitution` στη προτιμώμενη εναλλακτική σας. |
| **Οι χαρακτήρες Unicode εμφανίζονται ως κουτιά** | Απουσία γλυφών για τη στοχευόμενη γραμματοσειρά. | Ενσωματώστε μια γραμματοσειρά που καλύπτει Unicode όπως “Noto Sans” και ενεργοποιήστε την ενσωμάτωση γραμματοσειρών (`doc.FontInfos.FontEmbeddingMode = FontEmbeddingMode.Embedding`). |

## Πώς Αυτό Σας Βοηθά να Μετατρέψετε Word σε PDF Απρόσκοπτα

- **Αξιοπιστία** – Ακούγοντας τις προειδοποιήσεις γραμματοσειρών, δεν στέλνετε ποτέ ένα PDF που φαίνεται λανθασμένο επειδή ο διακομιστής δεν είχε τη γραμματοσειρά.
- **Διαφάνεια** – Η έξοδος της κονσόλας σας λέει ακριβώς ποιες γραμματοσειρές αντικαταστάθηκαν, κάνοντας τον εντοπισμό σφαλμάτων απλό.
- **Φορητότητα** – Ο ίδιος κώδικας λειτουργεί σε Windows, Linux και Docker containers, εφόσον παρέχετε τις απαιτούμενες γραμματοσειρές.

## Επόμενα Βήματα (Εξερευνήστε Περισσότερα)

Τώρα που έχετε κατακτήσει την **αποθήκευση εγγράφου ως PDF** και την **ανίχνευση ελλιπών γραμματοσειρών**, ίσως θέλετε να:

1. **Επεξεργασία κατά παρτίδες** ενός φακέλου αρχείων DOCX, καταγράφοντας όλα τα προβλήματα γραμματοσειρών σε αρχείο CSV.
2. **Αυτόματη ενσωμάτωση ελλιπών γραμματοσειρών** φορτώνοντάς τες στο `FontSettings` κατά το χρόνο εκτέλεσης.
3. **Προσαρμογή εξόδου PDF** – προσθήκη υδατογραφιών, ρύθμιση συμμόρφωσης PDF/A ή κρυπτογράφηση του αρχείου.
4. **Ενσωμάτωση με ASP.NET Core** – έκθεση ενός API endpoint που δέχεται ροή DOCX και επιστρέφει ροή PDF, ενώ εξακολουθεί να αναφέρει την αντικατάσταση γραμματοσειρών.

Κάθε ένα από αυτά τα θέματα βασίζεται άμεσα στις έννοιες που καλύφθηκαν εδώ, και το ίδιο πρότυπο `IWarningCallback` ισχύει.

## Συμπέρασμα

Διασχίσαμε μια πλήρη λύση που **αποθηκεύει έγγραφο ως PDF** χρησιμοποιώντας το Aspose.Words, ενώ ταυτόχρονα **ανιχνεύει ελλιπείς γραμματοσειρές** μέσω του ενσωματωμένου συστήματος προειδοποιήσεων. Ο κώδικας είναι σύντομος, αυτόνομος και έτοιμος για παραγωγή. Αντιμετωπίζοντας τις προειδοποιήσεις `FontSubstitution` αποκτάτε την εμπιστοσύνη ότι κάθε PDF που δημιουργείτε αντανακλά πιστά την αρχική διάταξη του Word—χωρίς απρόσμενες αντικαταστάσεις “Arial” στο τελικό αρχείο.

Δοκιμάστε το στα δικά σας έργα, προσαρμόστε το callback ώστε να καταγράφει σε αρχείο ή σύστημα παρακολούθησης, και σύντομα θα αναρωτιέστε πώς κάνατε ποτέ μετατροπή Word σε PDF χωρίς αυτό.

Καλό προγραμματισμό, και εύχομαι τα PDFs σας να φαίνονται πάντα ακριβώς όπως το θέλετε!
