## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό το βήμα είναι κρίσιμο, καθώς ρυθμίζει το περιβάλλον για τη χρήση των λειτουργιών του Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

Τώρα, ας αναλύσουμε τη διαδικασία σε εύκολα βήματα. Κάθε βήμα θα σας καθοδηγήσει σε ένα συγκεκριμένο στάδιο της ενσωμάτωσης γραμματοσειρών στο έγγραφο PDF σας χρησιμοποιώντας το Aspose.Words για .NET.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πριν εμβαθύνετε στον κώδικα, πρέπει να ρυθμίσετε τον κατάλογο εγγράφων σας. Εδώ βρίσκεται το δείγμα εγγράφου του Word (`Rendering.docx`) και το PDF εξόδου θα παραμείνει.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ θα συμβεί όλη η μαγεία!

## Βήμα 2: Φόρτωση του εγγράφου του Word

Στη συνέχεια, θα φορτώσετε το έγγραφο του Word στο Aspose.Words `Document` αντικείμενο. Αυτό είναι το έγγραφο με το οποίο θα εργαστείτε.

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

Σε αυτήν τη γραμμή, δημιουργούμε ένα νέο `Document` αντικείμενο και φορτώστε το `Rendering.docx` αρχείο από τον κατάλογο εγγράφων μας.

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF

Τώρα, ήρθε η ώρα να διαμορφώσουμε τις επιλογές αποθήκευσης PDF. Συγκεκριμένα, θα ορίσουμε το `EmbedFullFonts` ιδιοκτησία σε `true` για να διασφαλίσετε ότι όλες οι γραμματοσειρές που χρησιμοποιούνται στο έγγραφο είναι ενσωματωμένες στο PDF.

```csharp
PdfSaveOptions saveOptions = new PdfSaveOptions { EmbedFullFonts = true };
```

Αυτή η γραμμή δημιουργεί ένα νέο `PdfSaveOptions` αντικείμενο και ορίζει το `EmbedFullFonts` ιδιοκτησία σε `true`Αυτό διασφαλίζει ότι το PDF που δημιουργείται θα περιλαμβάνει όλες τις γραμματοσειρές που χρησιμοποιούνται στο έγγραφο.

## Βήμα 4: Αποθηκεύστε το έγγραφο ως PDF

Τέλος, θα αποθηκεύσετε το έγγραφο του Word ως PDF με τις καθορισμένες επιλογές αποθήκευσης. Αυτό το βήμα μετατρέπει το έγγραφο και ενσωματώνει τις γραμματοσειρές.

```csharp
doc.Save(dataDir + "WorkingWithPdfSaveOptions.EmbeddedFontsInPdf.pdf", saveOptions);
```

Σε αυτήν τη γραμμή, αποθηκεύουμε το έγγραφο ως PDF στον κατάλογο εγγράφων, ενσωματώνοντας όλες τις γραμματοσειρές που χρησιμοποιούνται στο έγγραφο του Word.
