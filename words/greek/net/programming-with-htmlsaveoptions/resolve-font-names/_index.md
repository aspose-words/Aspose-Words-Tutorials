## Εισαγωγή χώρων ονομάτων

Πριν ξεκινήσουμε τον προγραμματισμό, βεβαιωθείτε ότι έχετε εισαγάγει τους απαραίτητους χώρους ονομάτων στο έργο σας. Αυτό είναι κρίσιμο για την πρόσβαση στις λειτουργίες του Aspose.Words.

```csharp
using Aspose.Words;
using Aspose.Words.Saving;
```

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων

Πρώτα απ 'όλα, ας ορίσουμε τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ βρίσκεται το έγγραφο του Word και εκεί θα αποθηκεύσετε το αποτέλεσμα.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Εξήγηση:
Εδώ, `dataDir` διατηρεί τη διαδρομή προς τον κατάλογο εγγράφων σας. Αντικαταστήστε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στο σύστημά σας.

## Βήμα 2: Φόρτωση του εγγράφου του Word

Στη συνέχεια, πρέπει να φορτώσουμε το έγγραφο του Word που θέλουμε να επεξεργαστούμε. Αυτό το έγγραφο θα πρέπει να έχει τις γραμματοσειρές που θέλετε να επιλύσετε.

```csharp
Document doc = new Document(dataDir + "Missing font.docx");
```

Εξήγηση:
Δημιουργούμε ένα `Document` αντικείμενο και φορτώστε το έγγραφο Word με το όνομα "Missing font.docx" από το `dataDir`.

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

Τώρα, ας ρυθμίσουμε τις επιλογές για την αποθήκευση του εγγράφου ως HTML. Εδώ, θα διασφαλίσουμε ότι τα ονόματα των γραμματοσειρών έχουν επιλυθεί σωστά.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html)
{
    PrettyFormat = true,
    ResolveFontNames = true
};
```

Εξήγηση:
Δημιουργούμε μια παρουσία του `HtmlSaveOptions` με `SaveFormat.Html`Το `PrettyFormat` η επιλογή κάνει την έξοδο HTML πιο ευανάγνωστη και `ResolveFontNames` διασφαλίζει ότι τα ονόματα γραμματοσειρών επιλύονται.

## Βήμα 4: Αποθήκευση του εγγράφου ως HTML

Τέλος, αποθηκεύουμε το έγγραφο ως αρχείο HTML χρησιμοποιώντας τις διαμορφωμένες επιλογές αποθήκευσης.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.ResolveFontNames.html", saveOptions);
```

Εξήγηση:
Ονομάζουμε το `Save` μέθοδος στο `Document` αντικείμενο, καθορίζοντας τη διαδρομή εξόδου και τις επιλογές αποθήκευσης που ρυθμίσαμε. Αυτό θα δημιουργήσει ένα αρχείο HTML με τα ονόματα γραμματοσειρών που έχουν επιλυθεί.
