# Προσθήκη προθέματος ονόματος κλάσης Css

## Εισαγωγή

Καλώς ορίσατε! Αν βουτάτε στον κόσμο του Aspose.Words για .NET, σας περιμένει μια έκπληξη. Σήμερα, θα εξερευνήσουμε πώς να προσθέσετε ένα πρόθεμα ονόματος κλάσης CSS κατά την αποθήκευση ενός εγγράφου Word ως HTML χρησιμοποιώντας το Aspose.Words για .NET. Αυτή η λειτουργία είναι εξαιρετικά χρήσιμη όταν θέλετε να αποφύγετε διενέξεις ονομάτων κλάσεων στα αρχεία HTML σας.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- Aspose.Words για .NET: Αν δεν το έχετε εγκαταστήσει ακόμα, [κατεβάστε το εδώ](https://releases.aspose.com/words/net/).
- Περιβάλλον Ανάπτυξης: Visual Studio ή οποιοδήποτε άλλο C# IDE.
- Ένα έγγραφο του Word: Θα χρησιμοποιήσουμε ένα έγγραφο με όνομα `Rendering.docx`Τοποθετήστε το στον κατάλογο του έργου σας.

## Εισαγωγή χώρων ονομάτων

Αρχικά, βεβαιωθείτε ότι έχετε εισαγάγει τους απαραίτητους χώρους ονομάτων στο έργο C# σας. Προσθέστε τους στην αρχή του αρχείου κώδικά σας:

```csharp
using System;
using Aspose.Words;
using Aspose.Words.Saving;
```

Τώρα, ας δούμε τον οδηγό βήμα προς βήμα!

## Βήμα 1: Ρύθμιση του έργου σας

Πριν ξεκινήσουμε την προσθήκη ενός προθέματος ονόματος κλάσης CSS, ας ρυθμίσουμε το έργο μας.

### Βήμα 1.1: Δημιουργία νέου έργου

Ανοίξτε το Visual Studio σας και δημιουργήστε ένα νέο έργο εφαρμογής κονσόλας. Ονομάστε το κάτι πιασάρικο, όπως `AsposeCssPrefixExample`.

### Βήμα 1.2: Προσθήκη Aspose.Words για .NET

Αν δεν το έχετε κάνει ήδη, προσθέστε το Aspose.Words για .NET στο έργο σας μέσω του NuGet. Απλώς ανοίξτε την κονσόλα NuGet Package Manager και εκτελέστε:

```bash
Install-Package Aspose.Words
```

Τέλεια! Τώρα, είμαστε έτοιμοι να ξεκινήσουμε τον προγραμματισμό.

## Βήμα 2: Φόρτωση του εγγράφου σας

Το πρώτο πράγμα που πρέπει να κάνουμε είναι να φορτώσουμε το έγγραφο του Word που θέλουμε να μετατρέψουμε σε HTML.

### Βήμα 2.1: Ορισμός της διαδρομής εγγράφου

Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας. Για τους σκοπούς αυτού του σεμιναρίου, ας υποθέσουμε ότι το έγγραφό σας βρίσκεται σε έναν φάκελο με το όνομα `Documents` μέσα στον κατάλογο του έργου σας.

```csharp
string dataDir = @"C:\YourProject\Documents\";
```

### Βήμα 2.2: Φόρτωση του εγγράφου

Τώρα, ας φορτώσουμε το έγγραφο χρησιμοποιώντας το Aspose.Words:

```csharp
Document doc = new Document(dataDir + "Rendering.docx");
```

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης HTML

Στη συνέχεια, πρέπει να ρυθμίσουμε τις επιλογές αποθήκευσης HTML ώστε να περιλαμβάνουν ένα πρόθεμα ονόματος κλάσης CSS.

### Βήμα 3.1: Δημιουργία επιλογών αποθήκευσης HTML

Δημιουργήστε ένα στιγμιότυπο του `HtmlSaveOptions` αντικείμενο και ορίστε τον τύπο φύλλου στυλ CSS σε `External`.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions
{
    CssStyleSheetType = CssStyleSheetType.External
};
```

### Βήμα 3.2: Ορισμός προθέματος ονόματος κλάσης CSS

Τώρα, ας ορίσουμε το `CssClassNamePrefix` ιδιότητα στο επιθυμητό πρόθεμα. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε `"pfx_"`.

```csharp
saveOptions.CssClassNamePrefix = "pfx_";
```

## Βήμα 4: Αποθήκευση του εγγράφου ως HTML

Τέλος, ας αποθηκεύσουμε το έγγραφο ως αρχείο HTML με τις διαμορφωμένες επιλογές μας.


Καθορίστε τη διαδρομή του αρχείου HTML εξόδου και αποθηκεύστε το έγγραφο.

```csharp
doc.Save(dataDir + "WorkingWithHtmlSaveOptions.AddCssClassNamePrefix.html", saveOptions);
```

## Βήμα 5: Επαλήθευση της εξόδου

Αφού εκτελέσετε το έργο σας, μεταβείτε στο `Documents` φάκελο. Θα πρέπει να βρείτε ένα αρχείο HTML με το όνομα `WorkingWithHtmlSaveOptions.AddCssClassNamePrefix.html`Ανοίξτε αυτό το αρχείο σε ένα πρόγραμμα επεξεργασίας κειμένου ή σε ένα πρόγραμμα περιήγησης για να επαληθεύσετε ότι οι κλάσεις CSS έχουν το πρόθεμα `pfx_`.

## Σύναψη

Και να το! Ακολουθώντας αυτά τα βήματα, έχετε προσθέσει με επιτυχία ένα πρόθεμα ονόματος κλάσης CSS στην έξοδο HTML χρησιμοποιώντας το Aspose.Words για .NET. Αυτή η απλή αλλά ισχυρή λειτουργία μπορεί να σας βοηθήσει να διατηρήσετε καθαρά και χωρίς συγκρούσεις στυλ στα έγγραφα HTML σας.

## Συχνές ερωτήσεις

### Μπορώ να χρησιμοποιήσω διαφορετικό πρόθεμα για κάθε λειτουργία αποθήκευσης;
Ναι, μπορείτε να προσαρμόσετε το πρόθεμα κάθε φορά που αποθηκεύετε ένα έγγραφο αλλάζοντας το `CssClassNamePrefix` ιδιοκτησία.

### Υποστηρίζει αυτή η μέθοδος inline CSS;
Ο `CssClassNamePrefix` Η ιδιότητα λειτουργεί με εξωτερικό CSS. Για ενσωματωμένο CSS, θα χρειαστείτε μια διαφορετική προσέγγιση.

### Πώς μπορώ να συμπεριλάβω άλλες επιλογές αποθήκευσης HTML;
Μπορείτε να διαμορφώσετε διάφορες ιδιότητες του `HtmlSaveOptions` για να προσαρμόσετε την έξοδο HTML. Ελέγξτε το [απόδειξη με έγγραφα](https://reference.aspose.com/words/net/) για περισσότερες λεπτομέρειες.

### Είναι δυνατή η αποθήκευση του HTML σε μια ροή;
Απολύτως! Μπορείτε να αποθηκεύσετε το έγγραφο σε μια ροή μεταβιβάζοντας το αντικείμενο ροής στο `Save` μέθοδος.

### Πώς μπορώ να λάβω υποστήριξη σε περίπτωση που αντιμετωπίσω προβλήματα;
Μπορείτε να λάβετε υποστήριξη από το [Φόρουμ Aspose](https://forum.aspose.com/c/words/8).
