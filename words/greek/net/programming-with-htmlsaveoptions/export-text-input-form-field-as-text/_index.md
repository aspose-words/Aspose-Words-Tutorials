# Εξαγωγή πεδίου φόρμας εισαγωγής κειμένου ως κείμενο

## Εισαγωγή

Βουτάτε, λοιπόν, στον κόσμο του Aspose.Words για .NET; Φανταστική επιλογή! Αν θέλετε να μάθετε πώς να εξάγετε ένα πεδίο φόρμας εισαγωγής κειμένου ως κείμενο, βρίσκεστε στο σωστό μέρος. Είτε ξεκινάτε τώρα είτε βελτιώνετε τις δεξιότητές σας, αυτός ο οδηγός θα σας καθοδηγήσει σε όλα όσα πρέπει να γνωρίζετε. Ας ξεκινήσουμε, έτσι δεν είναι;

## Προαπαιτούμενα

Πριν εμβαθύνουμε στα πιο σημαντικά, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ακολουθήσετε ομαλά:

- Aspose.Words για .NET: Λήψη και εγκατάσταση της τελευταίας έκδοσης από [εδώ](https://releases.aspose.com/words/net/).
- IDE: Visual Studio ή οποιοδήποτε περιβάλλον ανάπτυξης C#.
- Βασικές γνώσεις C#: Κατανόηση βασικής σύνταξης C# και εννοιών αντικειμενοστρεφούς προγραμματισμού.
- Έγγραφο: Ένα δείγμα εγγράφου του Word (`Rendering.docx`) με πεδία φόρμας εισαγωγής κειμένου.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, πρέπει να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Αυτοί είναι σαν τα δομικά στοιχεία που κάνουν τα πάντα να λειτουργούν άψογα.

```csharp
using System;
using System.IO;
using Aspose.Words;
using Aspose.Words.Saving;
```

Εντάξει, τώρα που έχουμε έτοιμους τους χώρους ονομάτων μας, ας ξεκινήσουμε!

## Βήμα 1: Ρύθμιση του Έργου

Πριν μπούμε στον κώδικα, ας βεβαιωθούμε ότι το έργο μας έχει ρυθμιστεί σωστά.

## Δημιουργία του Έργου

1. Άνοιγμα Visual Studio: Ξεκινήστε ανοίγοντας το Visual Studio ή το περιβάλλον ανάπτυξης C# που προτιμάτε.
2. Δημιουργία νέου έργου: Μεταβείτε στο `File > New > Project`Επιλογή `Console App (.NET Core)` ή οποιοδήποτε άλλο σχετικό είδος έργου.
3. Ονομάστε το έργο σας: Δώστε στο έργο σας ένα ουσιαστικό όνομα, κάτι σαν `AsposeWordsExportExample`.

## Προσθήκη Aspose.Words

1. Διαχείριση πακέτων NuGet: Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων και επιλέξτε `Manage NuGet Packages`.
2. Αναζήτηση για Aspose.Words: Στον Διαχειριστή Πακέτων NuGet, αναζητήστε `Aspose.Words`.
3. Εγκατάσταση Aspose.Words: Κάντε κλικ στο `Install` για να προσθέσετε τη βιβλιοθήκη Aspose.Words στο έργο σας.

## Βήμα 2: Φόρτωση του εγγράφου του Word

Τώρα που το έργο μας έχει ρυθμιστεί, ας φορτώσουμε το έγγραφο του Word που περιέχει τα πεδία φόρμας εισαγωγής κειμένου.

1. Καθορίστε τον κατάλογο εγγράφων: Ορίστε τη διαδρομή προς τον κατάλογο όπου είναι αποθηκευμένο το έγγραφό σας.
2. Τοποθέτηση του εγγράφου: Χρησιμοποιήστε το `Document` κλάση για να φορτώσετε το έγγραφο του Word σας.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Rendering.docx");
```

## Βήμα 3: Προετοιμασία του καταλόγου εξαγωγής

Πριν από την εξαγωγή, ας βεβαιωθούμε ότι ο κατάλογος εξαγωγής μας είναι έτοιμος. Εδώ θα αποθηκευτεί το αρχείο HTML και οι εικόνες μας.

1. Ορίστε τον κατάλογο εξαγωγής: Καθορίστε τη διαδρομή όπου θα αποθηκευτούν τα εξαγόμενα αρχεία.
2. Έλεγχος και καθαρισμός του καταλόγου: Βεβαιωθείτε ότι ο κατάλογος υπάρχει και είναι κενός.

```csharp
string imagesDir = Path.Combine(dataDir, "Images");

if (Directory.Exists(imagesDir))
    Directory.Delete(imagesDir, true);

Directory.CreateDirectory(imagesDir);
```

## Βήμα 4: Ρύθμιση παραμέτρων επιλογών αποθήκευσης

Εδώ ακριβώς συμβαίνει η μαγεία. Πρέπει να ρυθμίσουμε τις επιλογές αποθήκευσης για να εξαγάγουμε το πεδίο φόρμας εισαγωγής κειμένου ως απλό κείμενο.

1. Δημιουργία επιλογών αποθήκευσης: Αρχικοποίηση νέου `HtmlSaveOptions` αντικείμενο.
2. Ορισμός επιλογής εξαγωγής κειμένου: Διαμορφώστε το `ExportTextInputFormFieldAsText` ιδιοκτησία σε `true`.
3. Ορισμός φακέλου εικόνων: Ορίστε τον φάκελο στον οποίο θα αποθηκευτούν οι εικόνες.

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html)
{
    ExportTextInputFormFieldAsText = true,
    ImagesFolder = imagesDir
};
```

## Βήμα 5: Αποθήκευση του εγγράφου ως HTML

Τέλος, ας αποθηκεύσουμε το έγγραφο του Word ως αρχείο HTML χρησιμοποιώντας τις διαμορφωμένες επιλογές αποθήκευσης.

1. Ορισμός της διαδρομής εξόδου: Καθορίστε τη διαδρομή όπου θα αποθηκευτεί το αρχείο HTML.
2. Αποθήκευση εγγράφου: Χρησιμοποιήστε το `Save` μέθοδος του `Document` κλάση για εξαγωγή του εγγράφου.

```csharp
doc.Save(dataDir + "ExportedDocument.html", saveOptions);
```
