# Επανάκληση με συλλαβισμό

Η παύλα, αν και είναι ζωτικής σημασίας για τη σωστή διάταξη κειμένου, μπορεί να είναι λίγο πονοκέφαλος. Αλλά μαντέψτε τι; Το Aspose.Words για .NET σας βοηθάει. Σας ενδιαφέρει; Ας εμβαθύνουμε στις λεπτομέρειες του πώς μπορείτε να εφαρμόσετε μια επανάκληση με παύλα χρησιμοποιώντας το Aspose.Words για .NET.

## Εισαγωγή χώρων ονομάτων

Πρώτα απ 'όλα, ας εισαγάγουμε τους απαραίτητους χώρους ονομάτων. Αυτό διασφαλίζει ότι ο κώδικάς μας έχει πρόσβαση σε όλες τις κλάσεις και τις μεθόδους που χρειαζόμαστε από το Aspose.Words.

```csharp
using Aspose.Words;
using System;
using System.IO;
```

## Βήμα 1: Καταχωρίστε την επανακλήση με συλλαβισμό

Για να ξεκινήσουμε, πρέπει να καταχωρήσουμε την επανακλήση με συλλαβισμό. Εδώ λέμε στο Aspose.Words να χρησιμοποιήσει την προσαρμοσμένη λογική συλλαβισμού μας.

```csharp
try
{
    // Εγγραφή επανακλήσης με παύλα.
    Hyphenation.Callback = new CustomHyphenationCallback();
}
catch (Exception e)
{
    Console.WriteLine($"Error registering hyphenation callback: {e.Message}");
}
```

Εδώ, δημιουργούμε μια παρουσία της προσαρμοσμένης ανάκλησης και την αντιστοιχίζουμε σε `Hyphenation.Callback`.

## Βήμα 2: Ορίστε τη διαδρομή εγγράφου

Στη συνέχεια, πρέπει να ορίσουμε τον κατάλογο όπου αποθηκεύονται τα έγγραφά μας. Αυτό είναι κρίσιμο καθώς θα φορτώνουμε και θα αποθηκεύουμε έγγραφα από αυτήν τη διαδρομή.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή προς τα έγγραφά σας.

## Βήμα 3: Φόρτωση του εγγράφου

Τώρα, ας φορτώσουμε το έγγραφο που απαιτεί παύλα.

```csharp
Document document = new Document(dataDir + "German text.docx");
```

Εδώ, φορτώνουμε ένα έγγραφο κειμένου στα Γερμανικά. Μπορείτε να το αντικαταστήσετε `"German text.docx"` με το όνομα αρχείου του εγγράφου σας.

## Βήμα 4: Αποθήκευση του εγγράφου

Αφού φορτώσουμε το έγγραφο, το αποθηκεύουμε σε ένα νέο αρχείο, εφαρμόζοντας παράλληλα την επανακλήση με παύλα.

```csharp
document.Save(dataDir + "TreatmentByCesureWithRecall.pdf");
```

Αυτή η γραμμή αποθηκεύει το έγγραφο ως PDF με εφαρμογή παύλας.

## Βήμα 5: Χειρισμός εξαίρεσης λεξικού που λείπει με συλλαβισμό

Μερικές φορές, μπορεί να αντιμετωπίσετε ένα πρόβλημα όπου λείπει το λεξικό παύλας. Ας το αντιμετωπίσουμε.

```csharp
catch (Exception e) when (e.Message.StartsWith("Missing hyphenation dictionary"))
{
    Console.WriteLine(e.Message);
}
finally
{
    Hyphenation.Callback = null;
}
```

Σε αυτό το μπλοκ, εντοπίζουμε την συγκεκριμένη εξαίρεση που σχετίζεται με τα λεξικά που λείπουν και εκτυπώνουμε το μήνυμα.

## Βήμα 6: Υλοποίηση της κλάσης επανακλήσης προσαρμοσμένου συλλαβισμού

Τώρα, ας εφαρμόσουμε το `CustomHyphenationCallback` κλάση που χειρίζεται το αίτημα για λεξικά συλλαβισμού.

```csharp
public class CustomHyphenationCallback : IHyphenationCallback
{
    public void RequestDictionary(string language)
    {
        string dictionaryFolder = MyDir;
        string dictionaryFullFileName;
        switch (language)
        {
            case "en-US":
                dictionaryFullFileName = Path.Combine(dictionaryFolder, "hyph_en_US.dic");
                break;
            case "de-CH":
                dictionaryFullFileName = Path.Combine(dictionaryFolder, "hyph_de_CH.dic");
                break;
            default:
                throw new Exception($"Missing hyphenation dictionary for {language}.");
        }
        // Εγγραφή λεξικού για την ζητούμενη γλώσσα.
        Hyphenation.RegisterDictionary(language, dictionaryFullFileName);
    }
}
```

Σε αυτή την τάξη, το `RequestDictionary` Η μέθοδος καλείται κάθε φορά που χρειάζεται ένα λεξικό παύλας. Ελέγχει τη γλώσσα και καταχωρεί το κατάλληλο λεξικό.
