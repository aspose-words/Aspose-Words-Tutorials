## Βήμα 1: Μετατροπή Docx σε Byte Array

Για να μετατρέψετε ένα αρχείο Docx σε έναν πίνακα byte, ακολουθήστε τα εξής βήματα:
```csharp
// Φόρτωση του αρχείου Docx από δίσκο ή ροή
Document doc = new Document("input.docx");

// Αποθήκευση του εγγράφου σε ένα MemoryStream
MemoryStream outStream = new MemoryStream();
doc.Save(outStream, SaveFormat.Docx);

// Μετατροπή MemoryStream σε πίνακα byte
byte[] docBytes = outStream.ToArray();
```

## Βήμα 2: Μετατροπή πίνακα byte πίσω σε έγγραφο

Για να μετατρέψετε έναν πίνακα byte ξανά σε αντικείμενο εγγράφου:
```csharp
// Μετατροπή πίνακα byte πίσω σε MemoryStream
MemoryStream inStream = new MemoryStream(docBytes);

// Φόρτωση του εγγράφου από το MemoryStream
Document docFromBytes = new Document(inStream);
```
