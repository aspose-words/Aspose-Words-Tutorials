# Αποθήκευση docx ως txt – Πλήρης Οδηγός για τη Μετατροπή Αρχείων Word σε Απλό Κείμενο

Έχετε χρειαστεί ποτέ να **αποθηκεύσετε docx ως txt** αλλά δεν ήσασταν σίγουροι ποια κλήση API κάνει πραγματικά τη δουλειά; Δεν είστε μόνοι· πολλοί προγραμματιστές κοιτάζουν ένα αρχείο Word και αναρωτιούνται πώς να εξάγουν το ακατέργαστο κείμενο—ιδιαίτερα όταν το έγγραφο περιέχει εξισώσεις.  

Σε αυτό το tutorial θα σας δείξουμε, βήμα προς βήμα, πώς να **μετατρέψετε Word σε txt**, να εξάγετε αυτά τα ενσωματωμένα αντικείμενα Office Math και να καταλήξετε με ένα καθαρό αρχείο απλού κειμένου. Στο τέλος θα μπορείτε να εκτελέσετε ένα μόνο πρόγραμμα C# που παίρνει οποιοδήποτε *.docx* και γράφει μια έκδοση *.txt* (ή ακόμη και MathML/LaTeX)—χωρίς καμία χειροκίνητη αντιγραφή‑επικόλληση.

## Τι Θα Μάθετε

- Πώς να **αποθηκεύσετε docx ως txt** χρησιμοποιώντας το Aspose.Words για .NET.  
- Η επιλογή `OfficeMathExportMode` που σας επιτρέπει να **εξάγετε εξισώσεις** ως MathML.  
- Παραλλαγές για εξαγωγή σε LaTeX ή μόνο απλό κείμενο.  
- Συνηθισμένα προβλήματα, όπως ελλιπείς γραμματοσειρές ή μη υποστηριζόμενα χαρακτηριστικά εξισώσεων.  
- Ένα πλήρες, έτοιμο‑για‑εκτέλεση δείγμα κώδικα που μπορείτε να ενσωματώσετε σε οποιοδήποτε έργο .NET.  

> **Συμβουλή επαγγελματία:** Αν χρειάζεστε μόνο το κειμενικό περιεχόμενο και δεν σας ενδιαφέρουν οι εξισώσεις, μπορείτε να παραλείψετε εντελώς τη γραμμή `OfficeMathExportMode`. Εξοικονομεί μερικά χιλιοστά του δευτερολέπτου.

## Βήμα 1 – Φόρτωση του Πηγαίου Εγγράφου

Το πρώτο που κάνετε είναι να πείτε στο Aspose.Words ποιο αρχείο Word θέλετε να μετατρέψετε. Σκεφτείτε το `Document` ως την πύλη σε όλα όσα περιέχονται στο *.docx*.

```csharp
using Aspose.Words;

// Step 1: Load the source document
Document doc = new Document("YOUR_DIRECTORY/input.docx");
```

> **Γιατί αυτό το βήμα είναι σημαντικό:** Η φόρτωση του αρχείου αναλύει το πακέτο OpenXML, δημιουργεί ένα μοντέλο αντικειμένων στη μνήμη και σας δίνει πρόσβαση σε κείμενο, παραγράφους, πίνακες και αντικείμενα Office Math. Αν η διαδρομή του αρχείου είναι λανθασμένη, θα λάβετε `FileNotFoundException`—για αυτό ελέγξτε ξανά τη θέση.

---

## Βήμα 2 – Διαμόρφωση Επιλογών Αποθήκευσης TXT (Εξαγωγή Εξισώσεων ως MathML)

Από προεπιλογή, η αποθήκευση ενός εγγράφου ως απλό κείμενο αφαιρεί όλα όσα δεν είναι απλό κείμενο. Αυτό περιλαμβάνει τις εξισώσεις, οι οποίες εξαφανίζονται σιωπηλά. Για να **εξάγετε εξισώσεις**, πρέπει να πούμε στο Aspose.Words πώς να διαχειριστεί τα αντικείμενα `OfficeMath`.

```csharp
// Step 2: Configure TXT save options to export Office Math as MathML
// You can also choose LaTeX or PlainText by changing the enum value
TxtSaveOptions txtSaveOptions = new TxtSaveOptions
{
    OfficeMathExportMode = OfficeMathExportMode.MathML
};
```

- **`OfficeMathExportMode.MathML`** – Εξάγει κάθε εξίσωση ως ένα τμήμα MathML ενσωματωμένο στο αρχείο κειμένου.  
- **`OfficeMathExportMode.LaTeX`** – Σας παρέχει σήμανση LaTeX αντί αυτού (χρήσιμο για επιστημονικές ροές εργασίας).  
- **`OfficeMathExportMode.Text`** – Αντικαθιστά τις εξισώσεις με έναν δείκτη όπως «[Equation]».  

> **Ακραία περίπτωση:** Ορισμένες παλαιότερες εξισώσεις Word (OMML) μπορεί να μην έχουν τέλεια αναπαράσταση MathML. Σε αυτές τις σπάνιες περιπτώσεις το Aspose.Words επιστρέφει μια περιγραφή κειμένου, την οποία μπορείτε να εντοπίσετε ελέγχοντας το `txtSaveOptions.OfficeMathExportMode`.

---

## Βήμα 3 – Αποθήκευση του Εγγράφου ως Αρχείο Απλού Κειμένου

Τώρα που έχουμε το αντικείμενο `Document` και τις ρυθμισμένες `TxtSaveOptions`, απλώς καλούμε το `Save`. Η μέθοδος γράφει ένα αρχείο `.txt` στο δίσκο, σεβόμενη τη λειτουργία εξαγωγής που επιλέξαμε.

```csharp
// Step 3: Save the document as a plain‑text file using the configured options
doc.Save("YOUR_DIRECTORY/Math.txt", txtSaveOptions);
```

Μετά την εκτέλεση αυτής της γραμμής, ανοίξτε το `Math.txt` και θα δείτε κανονικές παραγράφους ακολουθούμενες από μπλοκ MathML όπως:

```xml
<math xmlns="http://www.w3.org/1998/Math/MathML">
  <mi>x</mi><mo>=</mo><mfrac><mi>-b</mi><mi>2a</mi></mfrac>
</math>
```

Αν αλλάξατε σε `OfficeMathExportMode.Text`, θα δείτε αντί αυτού:

```
[Equation]
```

## Πλήρες Παράδειγμα Λειτουργίας

Παρακάτω υπάρχει μια αυτόνομη εφαρμογή κονσόλας που μπορείτε να αντιγράψετε‑επικολλήσετε σε ένα νέο έργο C#. Περιλαμβάνει όλες τις οδηγίες `using`, διαχείριση σφαλμάτων και έναν μικρό βοηθό που εκτυπώνει μια επιβεβαίωση στην κονσόλα.

```csharp
using System;
using Aspose.Words;

namespace DocxToTxtDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // Validate arguments
            if (args.Length < 2)
            {
                Console.WriteLine("Usage: DocxToTxtDemo <input.docx> <output.txt>");
                return;
            }

            string inputPath = args[0];
            string outputPath = args[1];

            try
            {
                // Load the .docx file
                Document doc = new Document(inputPath);

                // Configure save options – change MathML to LaTeX or Text if needed
                TxtSaveOptions options = new TxtSaveOptions
                {
                    OfficeMathExportMode = OfficeMathExportMode.MathML
                };

                // Save as .txt
                doc.Save(outputPath, options);

                Console.WriteLine($"✅ Successfully saved '{inputPath}' as '{outputPath}'.");
                Console.WriteLine("Open the file to see extracted equations in MathML format.");
            }
            catch (Exception ex)
            {
                Console.WriteLine($"❌ Error: {ex.Message}");
            }
        }
    }
}
```

**Πώς να εκτελέσετε:**  

```bash
dotnet run --project DocxToTxtDemo.csproj "sample.docx" "sample.txt"
```

Το πρόγραμμα εκτυπώνει ένα φιλικό μήνυμα επιτυχίας, ή ένα σφάλμα αν κάτι πάει στραβά (όπως ένα αρχείο που λείπει ή ανεπαρκή δικαιώματα).

## Συχνές Ερωτήσεις (FAQ)

### 1. Μπορώ να **μετατρέψω word σε txt** χωρίς να εγκαταστήσω το Aspose.Words;

Ναι, θα μπορούσατε να χρησιμοποιήσετε το Open XML SDK για να διαβάσετε παραγράφους, αλλά δεν θα διαχειριστεί τις εξισώσεις αυτόματα. Το Aspose.Words αφαιρεί αυτήν την πολυπλοκότητα, γι' αυτό είναι η προτεινόμενη προσέγγιση για μια αξιόπιστη λύση **πώς να εξάγετε εξισώσεις**.

### 2. Τι γίνεται αν το έγγραφό μου περιέχει εικόνες—θα εμφανιστούν στο txt;

Όχι. Τα αρχεία απλού κειμένου δεν αποθηκεύουν δυαδικά δεδομένα, έτσι οι εικόνες παραλείπονται εντελώς. Αν χρειάζεστε περιγραφή κειμένου των εικόνων, θα πρέπει να προσθέσετε alt‑text χειροκίνητα ή να χρησιμοποιήσετε OCR πριν τη μετατροπή.

### 3. Λειτουργεί αυτό σε macOS/Linux;

Απολύτως. Το Aspose.Words για .NET είναι δια‑πλατφόρμα όσο τρέχετε .NET 5+ ή .NET Core. Απλώς βεβαιωθείτε ότι οι διαδρομές αρχείων χρησιμοποιούν τους κατάλληλους διαχωριστές καταλόγου.

### 4. Πώς μπορώ να **αποθηκεύσω το έγγραφο ως txt** διατηρώντας τις αλλαγές γραμμής;

`TxtSaveOptions` σέβεται τη αρχική διάταξη παραγράφων, έτσι κάθε παράγραφος του Word γίνεται μια νέα γραμμή στην έξοδο. Αν χρειάζεστε προσαρμοσμένη διαχείριση αλλαγών γραμμής, ορίστε `options.AddBidiMarks = true` ή επεξεργαστείτε το παραγόμενο string μετά την αποθήκευση.

## Συμβουλές, Τεχνάσματα και Ακραίες Περιπτώσεις

- **Μεγάλα έγγραφα:** Όταν επεξεργάζεστε αρχεία > 100 MB, σκεφτείτε τη ροή εξόδου (`doc.Save(Stream, options)`) για να αποφύγετε υψηλή χρήση μνήμης.  
- **Μη υποστηριζόμενες εξισώσεις:** Αν μια εξίσωση περιέχει προσαρμοσμένα σύμβολα, το Aspose.Words μπορεί να επιστρέψει έναν κειμενικό δείκτη. Ελέγξτε την έξοδο και, αν χρειάζεται, επεξεργαστείτε με έναν επικυρωτή MathML.  
- **Μετατροπή κατά παρτίδες:** Τυλίξτε τον κώδικα σε βρόχο `foreach` που διατρέχει έναν φάκελο με αρχεία *.docx*. Θυμηθείτε να επαναχρησιμοποιήσετε ένα μόνο αντικείμενο `TxtSaveOptions` για βελτιωμένη απόδοση.  
- **Κωδικοποίηση:** Από προεπιλογή, το Aspose.Words γράφει σε UTF‑8. Αν χρειάζεστε διαφορετική κωδικοσελίδα (π.χ., Windows‑1252), ορίστε `options.Encoding = Encoding.GetEncoding(1252)`.
