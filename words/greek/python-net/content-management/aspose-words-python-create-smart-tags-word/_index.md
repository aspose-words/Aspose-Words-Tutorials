## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση των έξυπνων ετικετών σε διαφορετικά χαρακτηριστικά.

### Δημιουργία Έξυπνων Ετικετών (H2)

#### Επισκόπηση
Η δημιουργία έξυπνων ετικετών περιλαμβάνει την προσθήκη αναγνωρίσιμων στοιχείων κειμένου στο έγγραφό σας και τη συσχέτισή τους με προσαρμοσμένες ιδιότητες XML. Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας έξυπνης ετικέτας τύπου ημερομηνίας και τύπου μετοχικού τίτλου.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ρύθμιση του εγγράφου σας
Ξεκινήστε εισάγοντας το Aspose.Words και αρχικοποιώντας ένα νέο έγγραφο του Word:
```python
import aspose.words as aw

def create_smart_tags():
    doc = aw.Document()
```

##### 2. Δημιουργήστε μια Έξυπνη Ετικέτα Τύπου Ημερομηνίας
Προσθέστε κείμενο που αναγνωρίζεται ως ημερομηνία και διαμορφώστε τις προσαρμοσμένες ιδιότητες XML του.
```python
# Προσθέστε μια έξυπνη ετικέτα τύπου ημερομηνίας με προσαρμοσμένες ιδιότητες XML.
smart_tag_date = aw.markup.SmartTag(doc)
smart_tag_date.append_child(aw.Run(doc, 'May 29, 2019'))
smart_tag_date.element = 'date'
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Day', '', '29'))
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Month', '', '5'))
smart_tag_date.properties.add(aw.markup.CustomXmlProperty('Year', '', '2019'))
smart_tag_date.uri = 'urn:schemas-microsoft-com:office:smarttags'
doc.first_section.body.first_paragraph.append_child(smart_tag_date)
```

##### 3. Δημιουργήστε μια έξυπνη ετικέτα τύπου μετοχικού τίτλου
Ρυθμίστε μια άλλη έξυπνη ετικέτα για τα στοιχεία μετοχών.
```python
# Προσθέστε μια έξυπνη ετικέτα τύπου μετοχικού τίτλου.
smart_tag_stock = aw.markup.SmartTag(doc)
smart_tag_stock.element = 'stockticker'
smart_tag_stock.uri = 'urn:schemas-microsoft-com:office:smarttags'
smart_tag_stock.append_child(aw.Run(doc, 'MSFT'))
doc.first_section.body.first_paragraph.append_child(smart_tag_stock)
```

##### 4. Αποθηκεύστε το έγγραφό σας
Τέλος, αποθηκεύστε το έγγραφο με όλες τις διαμορφωμένες έξυπνες ετικέτες.
```python
# Αποθηκεύστε το έγγραφο σε μια καθορισμένη διαδρομή.
output_path = YOUR_OUTPUT_DIRECTORY + 'SmartTag.create.doc'
doc.save(output_path)
print(f'Document saved to {output_path}')
```

### Αφαίρεση Έξυπνων Ετικετών (H2)

#### Επισκόπηση
Μερικές φορές χρειάζεται να καθαρίσετε το έγγραφό σας αφαιρώντας υπάρχουσες έξυπνες ετικέτες. Αυτή η ενότητα δείχνει πώς να το πετύχετε αυτό.

#### Εκτέλεση

##### 1. Τοποθετήστε το έγγραφο
Ξεκινήστε φορτώνοντας το έγγραφο του Word που περιέχει έξυπνες ετικέτες.
```python
def remove_smart_tags():
    input_path = YOUR_DOCUMENT_DIRECTORY + 'SmartTag.create.doc'
    doc = aw.Document(input_path)
```

##### 2. Αφαίρεση όλων των Έξυπνων Ετικετών
Εκτελέστε μια μέθοδο για να καταργήσετε όλες τις έξυπνες ετικέτες από το έγγραφό σας.
```python
# Αφαιρέστε όλες τις έξυπνες ετικέτες και επαληθεύστε την καταμέτρηση πριν και μετά την αφαίρεση.
initial_count = doc.get_child_nodes(aw.NodeType.SMART_TAG, True).count
doc.remove_smart_tags()
final_count = doc.get_child_nodes(aw.NodeType.SMART_TAG, True).count
print(f'Initial number of smart tags: {initial_count}')
print(f'Final number of smart tags: {final_count}')
```

### Ιδιότητες έξυπνης ετικέτας πρόσβασης (H2)

#### Επισκόπηση
Η κατανόηση και ο χειρισμός των ιδιοτήτων μιας έξυπνης ετικέτας μπορεί να βελτιώσει τον τρόπο επεξεργασίας των δεδομένων. Αυτή η ενότητα καλύπτει την πρόσβαση σε αυτές τις ιδιότητες.

#### Εκτέλεση

##### 1. Φόρτωση του εγγράφου με έξυπνες ετικέτες
Φορτώστε το έγγραφο και ανακτήστε όλες τις έξυπνες ετικέτες.
```python
def access_smart_tag_properties():
    input_path = YOUR_DOCUMENT_DIRECTORY + 'SmartTag.create.doc'
    doc = aw.Document(input_path)
```

##### 2. Ανάκτηση και πρόσβαση σε ιδιότητες
Πρόσβαση σε ιδιότητες συγκεκριμένων έξυπνων ετικετών, επιδεικνύοντας διάφορες αλληλεπιδράσεις.
```python
# Εξαγωγή έξυπνων ετικετών από το έγγραφο.
smart_tags = [node.as_smart_tag() for node in doc.get_child_nodes(aw.NodeType.SMART_TAG, True)]
print(f'Total number of smart tags: {len(smart_tags)}')

# Αποκτήστε πρόσβαση σε ιδιότητες και δείξτε επιλογές χειρισμού.
properties = smart_tags[-1].properties
for prop in properties:
    print(f'Property name: {prop.name}, value: {prop.value}')

if properties.contains('Day'):
    day_value = properties.get_by_name('Day').value
    print(f'Day property value: {day_value}')

month_index = properties.index_of_key('Month')
print(f'Month index in properties: {month_index}')
```

##### 3. Τροποποίηση ιδιοτήτων
Αφαιρέστε ή διαγράψτε συγκεκριμένες ιδιότητες, όπως απαιτείται.
```python
# Καταργήστε μια συγκεκριμένη ιδιότητα και διαγράψτε όλες τις ιδιότητες.
if 'Year' in [prop.name for prop in properties]:
    properties.remove('Year')
    print('Year property removed.')

properties.clear()
print(f'Properties count after clearing: {properties.count}')
```
