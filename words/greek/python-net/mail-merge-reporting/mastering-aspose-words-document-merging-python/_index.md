## Οδηγός Εφαρμογής

Εξερευνήστε δύο βασικές λειτουργίες: "Διατήρηση αρίθμησης πηγής" και "Εισαγωγή στο σελιδοδείκτη". Κάθε λειτουργία αναλύεται σε βήματα υλοποίησης.

### Χαρακτηριστικό 1: Διατήρηση αρίθμησης πηγής

#### Επισκόπηση
Αυτή η λειτουργία επιλύει τις συγκρούσεις αρίθμησης λιστών κατά τη συγχώνευση εγγράφων, διατηρώντας συνεπείς ακολουθίες αρίθμησης για προσαρμοσμένες λίστες.

#### Βήματα Υλοποίησης
**Βήμα 1: Προετοιμάστε τα έγγραφά σας**
Φορτώστε το έγγραφο προέλευσης και δημιουργήστε ένα κλώνο του:

```python
src_doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Custom list numbering.docx')
dst_doc = src_doc.clone()
```

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών μορφής εισαγωγής**
Ρυθμίστε τις επιλογές μορφής εισαγωγής για να διατηρήσετε ή να τροποποιήσετε την αρίθμηση πηγής:

```python
import_format_options = aw.ImportFormatOptions()
import_format_options.keep_source_numbering = True  # Ορίστε σε False για την επανάριθμηση
```

**Βήμα 3: Εισαγωγή κόμβων**
Χρήση `NodeImporter` για να μεταφέρετε κόμβους από το έγγραφο προέλευσης, εφαρμόζοντας συγκεκριμένες επιλογές μορφοποίησης:

```python
importer = aw.NodeImporter(
    src_doc=src_doc,
    dst_doc=dst_doc,
    import_format_mode=aw.ImportFormatMode.KEEP_DIFFERENT_STYLES,
    import_format_options=import_format_options
)

for paragraph in src_doc.first_section.body.paragraphs:
    imported_node = importer.import_node(paragraph.as_paragraph(), True)
    dst_doc.first_section.body.append_child(imported_node)
```

**Βήμα 4: Ενημέρωση ετικετών λίστας**
Βεβαιωθείτε ότι η αρίθμηση της λίστας αντικατοπτρίζει το συγχωνευμένο περιεχόμενο:

```python
dst_doc.update_list_labels()
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι λίστες εγγράφων προέλευσης έχουν σωστή μορφοποίηση.
- Βεβαιωθείτε ότι η λειτουργία μορφής εισαγωγής ευθυγραμμίζεται με το επιθυμητό αποτέλεσμα.

### Λειτουργία 2: Εισαγωγή σε σελιδοδείκτη

#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την εισαγωγή του περιεχομένου ενός εγγράφου σε έναν συγκεκριμένο σελιδοδείκτη μέσα σε ένα άλλο έγγραφο, ιδανικό για δυναμική ενσωμάτωση περιεχομένου.

#### Βήματα Υλοποίησης
**Βήμα 1: Δημιουργία και προετοιμασία εγγράφων**
Αρχικοποιήστε το κύριο έγγραφό σας με έναν καθορισμένο σελιδοδείκτη:

```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.start_bookmark('InsertionPoint')
builder.write('We will insert a document here: ')
builder.end_bookmark('InsertionPoint')
```

**Βήμα 2: Δημιουργία εγγράφου περιεχομένου**
Αναπτύξτε το περιεχόμενο που θέλετε να εισαγάγετε και αποθηκεύστε το:

```python
doc_to_insert = aw.Document()
builder = aw.DocumentBuilder(doc_to_insert)
builder.write('Hello world!')
doc_to_insert.save('YOUR_OUTPUT_DIRECTORY/NodeImporter.insert_at_bookmark.docx')
```

**Βήμα 3: Εισαγωγή περιεχομένου**
Εντοπίστε τον σελιδοδείκτη και χρησιμοποιήστε τον `insert_document` για να τοποθετήσετε το περιεχόμενό σας:

```python
bookmark = doc.range.bookmarks.get_by_name('InsertionPoint')
insert_document(bookmark.bookmark_start.parent_node, doc_to_insert)
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι το όνομα του σελιδοδείκτη είναι σωστό.
- Επιβεβαιώστε ότι το περιεχόμενο του εισαγόμενου εγγράφου ανταποκρίνεται στις προσδοκίες.
