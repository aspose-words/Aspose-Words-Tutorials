## Οδηγός Εφαρμογής

### Σύνοψη Τεχνητής Νοημοσύνης

Η σύνοψη κειμένου είναι απαραίτητη για την γρήγορη κατανόηση μεγάλων εγγράφων. Δείτε πώς μπορείτε να το κάνετε αυτό με το Aspose.Words και το OpenAI:

#### Σύνοψη Ενιαίου Εγγράφου
**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να συνοψίσετε αποτελεσματικά ένα μεμονωμένο έγγραφο.

- **Φόρτωση του εγγράφου:**
  ```python
  first_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **Ρύθμιση παραμέτρων μοντέλου τεχνητής νοημοσύνης:**
  - Χρησιμοποιήστε το μοντέλο GPT του OpenAI για σύνοψη.
  ```python
  api_key = 'YOUR_API_KEY'  
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model()
           .with_organization('Organization')
           .with_project('Project'))
  ```

- **Ορισμός επιλογών σύνοψης:**
  ```python
  options = aw.ai.SummarizeOptions()
  options.summary_length = aw.ai.SummaryLength.SHORT
  ```

- **Εκτέλεση σύνοψης:**
  ```python
  one_document_summary = model.summarize(source_document=first_doc, options=options)
  one_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.One.docx')
  ```

#### Σύνοψη πολλαπλών εγγράφων

Για τη σύνοψη πολλών εγγράφων ταυτόχρονα:

- **Φόρτωση πρόσθετων εγγράφων:**
  ```python
  second_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **Προσαρμογή μήκους σύνοψης:**
  ```python
  options.summary_length = aw.ai.SummaryLength.LONG
  ```

- **Συνοψίστε Πολλαπλά Έγγραφα:**
  ```python
  multi_document_summary = model.summarize(source_documents=[first_doc, second_doc], options=options)
  multi_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.Multi.docx')
  ```

### Μετάφραση Τεχνητής Νοημοσύνης

Η μετάφραση εγγράφων σε διαφορετικές γλώσσες μπορεί να ανοίξει νέες αγορές και κοινό.

#### Επισκόπηση:
Αυτή η λειτουργία μεταφράζει κείμενο χρησιμοποιώντας μοντέλα Google.

- **Φόρτωση του εγγράφου:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **Ρύθμιση παραμέτρων μοντέλου μετάφρασης:**
  - Χρησιμοποιήστε την Τεχνητή Νοημοσύνη της Google για μεταφράσεις.
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GEMINI_15_FLASH)
           .with_api_key(api_key)
           .as_google_ai_model())
  ```

- **Μεταφράστε το έγγραφο:**
  ```python
  translated_doc = model.translate(doc, aw.ai.Language.ARABIC)
  translated_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiTranslate.docx')
  ```

### Έλεγχος γραμματικής με τεχνητή νοημοσύνη

Βελτίωση της ποιότητας των εγγράφων μέσω γραμματικού ελέγχου.

#### Επισκόπηση:
Αυτή η λειτουργία ελέγχει και διορθώνει γραμματικά λάθη στα έγγραφά σας.

- **Φόρτωση του εγγράφου:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **Ρύθμιση παραμέτρων γραμματικού μοντέλου:**
  - Χρησιμοποιήστε το μοντέλο GPT του OpenAI για έλεγχο γραμματικής.
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model())
  ```

- **Ορισμός επιλογών γραμματικής:**
  ```python
  grammar_options = aw.ai.CheckGrammarOptions()
  grammar_options.improve_stylistics = True
  ```

- **Έλεγχος και αποθήκευση εγγράφου:**
  ```python
  proofed_doc = model.check_grammar(doc, grammar_options)
  proofed_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiGrammar.docx')
  ```
