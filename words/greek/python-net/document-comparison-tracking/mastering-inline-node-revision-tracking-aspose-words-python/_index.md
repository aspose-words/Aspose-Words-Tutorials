## Οδηγός Εφαρμογής
Τώρα, ας δούμε τα βήματα για την εφαρμογή της παρακολούθησης αναθεωρήσεων ενσωματωμένων κόμβων.
### Χαρακτηριστικό: Ενσωματωμένη παρακολούθηση αναθεωρήσεων κόμβου
Αυτή η λειτουργία σάς επιτρέπει να εντοπίζετε και να διαχειρίζεστε διαφορετικούς τύπους αναθεωρήσεων σε ένα έγγραφο του Word. Ας το αναλύσουμε βήμα προς βήμα.
#### Βήμα 1: Φόρτωση του εγγράφου σας
Φορτώστε το έγγραφό σας χρησιμοποιώντας το Aspose.Words:
```python
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Revision_runs.docx')
```
Εδώ, `Document` είναι η κλάση που χρησιμοποιείται για την αναπαράσταση και τον χειρισμό εγγράφων Word στο Aspose.Words. Βεβαιωθείτε ότι η διαδρομή δείχνει σε ένα έγγραφο με εντοπισμένες αλλαγές.
#### Βήμα 2: Έλεγχος αριθμού αναθεωρήσεων
Πριν εμβαθύνουμε στις μεμονωμένες αναθεωρήσεις, ας ελέγξουμε πόσες αναθεωρήσεις υπάρχουν:
```python
assert len(doc.revisions) == 6  # Προσαρμόστε ανάλογα με τον πραγματικό αριθμό αναθεωρήσεων
```
Αυτή η διεκδίκηση ελέγχει τον αριθμό των αναθεωρήσεων. Εάν δεν ταιριάζει με τον πραγματικό αριθμό του εγγράφου σας, προσαρμόστε την ανάλογα.
#### Βήμα 3: Προσδιορισμός τύπων αναθεώρησης
Διαφορετικοί τύποι αναθεώρησης περιλαμβάνουν εισαγωγές, αλλαγές μορφοποίησης, μετακινήσεις και διαγραφές. Ας τα προσδιορίσουμε:
```python
# Λήψη του γονικού κόμβου της πρώτης αναθεώρησης ως αντικείμενο εκτέλεσης
run = doc.revisions[0].parent_node.as_run()
first_paragraph = run.parent_paragraph
runs = first_paragraph.runs

assert len(runs) == 6  # Βεβαιωθείτε ότι υπάρχουν έξι εκτελέσεις στην παράγραφο
```
Τώρα, ας προσδιορίσουμε συγκεκριμένους τύπους αναθεωρήσεων:
- **Εισαγωγή αναθεώρησης:**
```python
# Ελέγξτε εάν η τρίτη εκτέλεση είναι αναθεώρηση εισαγωγής
assert runs[2].is_insert_revision
```
- **Αναθεώρηση μορφής:**
```python
# Επαλήθευση αλλαγών μορφής εντός της ίδιας εκτέλεσης
assert runs[2].is_format_revision
```
- **Μετακίνηση αναθεωρήσεων:**
  - Από την Αναθεώρηση:
```python
assert runs[4].is_move_from_revision  # Αρχική θέση πριν από τη μετακίνηση
```
  - Προς Αναθεώρηση:
```python
assert runs[1].is_move_to_revision   # Νέα θέση μετά τη μετακόμιση
```
- **Διαγραφή αναθεώρησης:**
```python
# Επιβεβαίωση μιας αναθεώρησης διαγραφής στην τελευταία εκτέλεση
assert runs[5].is_delete_revision
```
