import aspose.words as aw

## Προσθήκη περιεχομένου στο έγγραφο

Η πραγματική δύναμη του Aspose.Words για Python έγκειται στην ικανότητά του να προσθέτει πλούσιο περιεχόμενο στο έγγραφο του Word. Μπορείτε να εισάγετε δυναμικά κείμενο, πίνακες, εικόνες και άλλα. Παρακάτω είναι ένα παράδειγμα προσθήκης περιεχομένου στο κενό έγγραφο που δημιουργήθηκε προηγουμένως:

import aspose.words as aw

## Ενσωμάτωση μορφοποίησης και στυλ

Για να δημιουργήσετε έγγραφα επαγγελματικής εμφάνισης, πιθανότατα θα θελήσετε να εφαρμόσετε μορφοποίηση και στυλ στο περιεχόμενο που προσθέτετε. Το Aspose.Words για Python προσφέρει ένα ευρύ φάσμα επιλογών μορφοποίησης, όπως στυλ γραμματοσειράς, χρώματα, στοίχιση, εσοχή και άλλα. Ας δούμε ένα παράδειγμα εφαρμογής μορφοποίησης σε μια παράγραφο:

```python
import aspose.words as aw

def format_paragraph():
    # Φόρτωση του εγγράφου
    doc = aw.Document("output.docx")

    # Πρόσβαση στην πρώτη παράγραφο του εγγράφου
    paragraph = doc.first_section.body.first_paragraph

    # Εφαρμογή μορφοποίησης στην παράγραφο
    paragraph.alignment = aw.ParagraphAlignment.CENTER

    # Αποθήκευση του ενημερωμένου εγγράφου
    doc.save("output.docx")
```

## Προσθήκη πινάκων στο έγγραφο

Οι πίνακες χρησιμοποιούνται συνήθως σε έγγραφα του Word για την οργάνωση δεδομένων. Με το Aspose.Words για Python, μπορείτε εύκολα να δημιουργήσετε πίνακες και να τους συμπληρώσετε με περιεχόμενο. Παρακάτω είναι ένα παράδειγμα προσθήκης ενός απλού πίνακα στο έγγραφο:

```python
import aspose.words as aw

def add_table_to_document():
    # Φόρτωση του εγγράφου
    doc = aw.Document()
	table = aw.tables.Table(doc)
	doc.first_section.body.append_child(table)
	# Οι πίνακες περιέχουν γραμμές, οι οποίες περιέχουν κελιά, τα οποία μπορεί να έχουν παραγράφους
	# με τυπικά στοιχεία όπως τρεξίματα, σχήματα, ακόμη και άλλα τραπέζια.
	# Η κλήση της μεθόδου "EnsureMinimum" σε έναν πίνακα θα διασφαλίσει ότι
	# Ο πίνακας έχει τουλάχιστον μία γραμμή, κελί και παράγραφο.
	first_row = aw.tables.Row(doc)
	table.append_child(first_row)
	first_cell = aw.tables.Cell(doc)
	first_row.append_child(first_cell)
	paragraph = aw.Paragraph(doc)
	first_cell.append_child(paragraph)
	# Προσθήκη κειμένου στο πρώτο κελί της πρώτης γραμμής του πίνακα.
	run = aw.Run(doc=doc, text='Hello world!')
	paragraph.append_child(run)
	# Αποθήκευση του ενημερωμένου εγγράφου
	doc.save(file_name=ARTIFACTS_DIR + 'Table.CreateTable.docx')
```
