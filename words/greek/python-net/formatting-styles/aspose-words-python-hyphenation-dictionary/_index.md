## Οδηγός Εφαρμογής

### Καταχώριση ενός Λεξικού Συλλαβισμού

#### Επισκόπηση
Η καταχώριση ενός λεξικού επιτρέπει στο Aspose.Words να εφαρμόζει κανόνες συλλαβισμού που αφορούν συγκεκριμένες τοπικές ρυθμίσεις, διατηρώντας τη ροή του κειμένου σε πολύγλωσσα περιβάλλοντα.

#### Βήμα προς βήμα διαδικασία

**1. Καθορίστε Καταλόγους**

Ορίστε διαδρομές για το έγγραφο εισόδου και τον κατάλογο εξόδου:

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY'
arartifacts_directory = 'YOUR_OUTPUT_DIRECTORY'
```

**2. Εγγραφή του Λεξικού**

Χρησιμοποιήστε το Aspose.Words για να καταχωρήσετε ένα λεξικό συλλαβισμού για την τοπική ρύθμιση "de-CH".

```python
aw.Hyphenation.register_dictionary('de-CH', document_directory + 'hyph_de_CH.dic')
```
*Παράμετροι:*
- `'de-CH'`: Αναγνωριστικό τοπικών ρυθμίσεων.
- `document_directory + 'hyph_de_CH.dic'`: Διαδρομή προς το αρχείο λεξικού συλλαβισμού.

**3. Επαλήθευση εγγραφής**

Βεβαιωθείτε ότι το λεξικό έχει καταχωρηθεί σωστά:

```python
assert aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be registered"
```

### Εφαρμογή συλλαβισμού

Ανοίξτε ένα έγγραφο και αποθηκεύστε το εφαρμόζοντας παύλα χρησιμοποιώντας το νεοεγγεγραμμένο λεξικό:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.registered.pdf')
```

### Κατάργηση καταχώρισης λεξικού συλλαβισμού

#### Επισκόπηση
Η κατάργηση της καταχώρησης καταργεί τους κανόνες που αφορούν συγκεκριμένες τοπικές ρυθμίσεις, επαναφέροντας την προεπιλεγμένη συμπεριφορά συλλαβισμού.

**1. Απεγγραφή του Λεξικού**

```python
aw.Hyphenation.unregister_dictionary('de-CH')
```
*Σκοπός:* Καταργεί την καταχώρηση στο λεξικό "de-CH" για να αποτρέψει τη χρήση της σε μελλοντική επεξεργασία εγγράφων.

**2. Επαλήθευση διαγραφής**

Επιβεβαιώστε ότι το λεξικό δεν είναι πλέον ενεργό:

```python
assert not aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be unregistered"
```

### Αποθήκευση χωρίς παύλα

Ανοίξτε ξανά και αποθηκεύστε το έγγραφό σας, αυτή τη φορά χωρίς να εφαρμόσετε προηγουμένως καταχωρημένους κανόνες παύλας:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.unregistered.pdf')
```
