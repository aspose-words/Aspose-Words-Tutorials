## Οδηγός Εφαρμογής
### Δημιουργία Προσαρμοσμένων Στυλ
#### Επισκόπηση
Τα προσαρμοσμένα στυλ διασφαλίζουν την ομοιογενή μορφοποίηση σε όλο το έγγραφό σας χωρίς κόπο. Αυτή η ενότητα σας καθοδηγεί στη δημιουργία ενός νέου στυλ από την αρχή.
#### Βήμα 1: Ορίστε το στυλ
Ξεκινήστε ορίζοντας τις ιδιότητες του προσαρμοσμένου στυλ σας, όπως το όνομα, τα χαρακτηριστικά γραμματοσειράς, την απόσταση μεταξύ παραγράφων, τα περιγράμματα κ.λπ.
```python
# Δημιουργήστε ένα νέο στυλ στη συλλογή στυλ του εγγράφου
doc.styles.add(aw.StyleType.PARAGRAPH, "SEOStyle")
# Ορισμός χαρακτηριστικών γραμματοσειράς
font = doc.styles["SEOStyle"].font
font.name = "Arial"
font.size = 14
font.bold = True
# Ρύθμιση παραμέτρων μορφοποίησης παραγράφου
paragraph_format = doc.styles["SEOStyle"].paragraph_format
paragraph_format.space_before = 10
paragraph_format.space_after = 10
```
#### Βήμα 2: Εφαρμογή του στυλ σε κείμενο
Εφαρμόστε το προσαρμοσμένο στυλ σας σε ένα συγκεκριμένο μέρος του εγγράφου.
```python
# Μετακινηθείτε στο τέλος του εγγράφου και προσθέστε κείμενο με το νέο στυλ
doc_builder = aw.DocumentBuilder(doc)
doc_builder.move_to_document_end()
doc_builder.write("This is a paragraph styled with SEOStyle.")
# Εφαρμογή του προσαρμοσμένου στυλ
doc_builder.current_paragraph.applied_style = doc.styles["SEOStyle"]
```
#### Βήμα 3: Αποθήκευση του εγγράφου σας
Αφού εφαρμόσετε τα στυλ, αποθηκεύστε το έγγραφό σας για να διατηρήσετε τις αλλαγές.
```python
# Αποθήκευση του εγγράφου
doc.save("StyledDocument.docx")
```
