## Οδηγός Εφαρμογής

### Ενεργοποίηση αδειοδότησης με μετρητή

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να παρακολουθείτε τον τρόπο με τον οποίο η εφαρμογή σας χρησιμοποιεί το Aspose.Words, παρέχοντας πληροφορίες σχετικά με την κατανάλωση και τις πιστώσεις.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση Άδειας Ογκοχρέωσης**

Ξεκινήστε δημιουργώντας ένα `Metered` παράδειγμα και ορισμός των κλειδιών σας:

```python
import aspose.words as aw

metered = aw.Metered()
metered.set_metered_key('YourPublicKey', 'YourPrivateKey')
```

**2. Παρακολούθηση χρήσης πριν από τη λειτουργία**

Εκτυπώστε τα αρχικά δεδομένα πίστωσης και κατανάλωσης για να κατανοήσετε την αρχική τιμή:

```python
print('Credit before operation:', metered.get_consumption_credit())
print('Consumption quantity before operation:', metered.get_consumption_quantity())
```

**3. Εκτέλεση λειτουργιών εγγράφων**

Χρησιμοποιήστε το Aspose.Words για την επεξεργασία εγγράφων, όπως η μετατροπή ενός εγγράφου Word σε PDF:

```python
doc = aw.Document('path_to_your_document.docx')
doc.save('output_path.pdf')
```

**4. Παρακολούθηση χρήσης μετά τη λειτουργία**

Μετά την επέμβαση, ελέγξτε πόσο έχουν αλλάξει η πίστωση και η κατανάλωση:

```python
import time

# Περιμένετε για να βεβαιωθείτε ότι τα δεδομένα αποστέλλονται στον διακομιστή
time.sleep(10)  

print('Credit after operation:', metered.get_consumption_credit())
print('Consumption quantity after operation:', metered.get_consumption_quantity())
```
