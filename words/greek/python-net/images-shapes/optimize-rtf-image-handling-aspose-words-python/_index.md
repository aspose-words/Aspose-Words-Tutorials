## Οδηγός Εφαρμογής

### Αποθήκευση εικόνων ως WMF σε RTF

Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύετε εικόνες σε μορφή μετα-αρχείου των Windows κατά την εξαγωγή εγγράφων σε μορφή RTF, κάτι που είναι επωφελές για λόγους συμβατότητας και απόδοσης.

#### Επισκόπηση

Η αποθήκευση εικόνων ως WMF βοηθά στη μείωση του μεγέθους του αρχείου και στη βελτίωση της απόδοσης σε διαφορετικές πλατφόρμες. Αυτή η μέθοδος είναι ιδιαίτερα χρήσιμη για σύνθετα διανυσματικά γραφικά.

#### Βήμα προς βήμα εφαρμογή

##### Βήμα 1: Δημιουργία εγγράφου και εισαγωγή εικόνων

Ξεκινήστε δημιουργώντας ένα νέο έγγραφο και εισάγοντας τις εικόνες σας:

```python
import aspose.words as aw

def save_images_as_wmf_example():
    for save_images_as_wmf in [False, True]:
        doc = aw.Document()
        builder = aw.DocumentBuilder(doc=doc)

        # Εισαγωγή εικόνας JPEG
        builder.writeln('Jpeg image:')
        jpeg_image_shape = builder.insert_image(file_name='YOUR_DOCUMENT_DIRECTORY/Logo.jpg')
        assert aw.drawing.ImageType.JPEG == jpeg_image_shape.image_data.image_type
        builder.insert_paragraph()

        # Εισαγωγή εικόνας PNG
        builder.writeln('Png image:')
        png_image_shape = builder.insert_image(file_name='YOUR_DOCUMENT_DIRECTORY/Transparent background logo.png')
        assert aw.drawing.ImageType.PNG == png_image_shape.image_data.image_type

        # Ρύθμιση παραμέτρων επιλογών αποθήκευσης RTF
        rtf_save_options = aw.saving.RtfSaveOptions()
        rtf_save_options.save_images_as_wmf = save_images_as_wmf

        # Αποθήκευση του εγγράφου ως RTF
        doc.save(file_name='YOUR_OUTPUT_DIRECTORY/RtfSaveOptions.SaveImagesAsWmf.rtf', save_options=rtf_save_options)

        # Επαλήθευση μορφών εικόνας στο αποθηκευμένο έγγραφο
        doc = aw.Document(file_name='YOUR_OUTPUT_DIRECTORY/RtfSaveOptions.SaveImagesAsWmf.rtf')
        shapes = doc.get_child_nodes(aw.NodeType.SHAPE, True)
        if save_images_as_wmf:
            assert aw.drawing.ImageType.WMF == shapes[0].as_shape().image_data.image_type
            assert aw.drawing.ImageType.WMF == shapes[1].as_shape().image_data.image_type
        else:
            assert aw.drawing.ImageType.JPEG == shapes[0].as_shape().image_data.image_type
            assert aw.drawing.ImageType.PNG == shapes[1].as_shape().image_data.image_type

save_images_as_wmf_example()
```

##### Επεξήγηση βασικών παραμέτρων:
- `save_images_as_wmf`: Μια λογική τιμή που καθορίζει εάν οι εικόνες πρέπει να αποθηκεύονται ως WMF.
- `RtfSaveOptions.save_images_as_wmf`Ρυθμίζει τις παραμέτρους της εξαγωγής RTF για τη μετατροπή εικόνων σε μορφή WMF.

#### Συμβουλές αντιμετώπισης προβλημάτων

Εάν αντιμετωπίσετε προβλήματα:
- Βεβαιωθείτε ότι οι διαδρομές των εικόνων σας είναι σωστές.

### Εξαγωγή εικόνων για παλιούς αναγνώστες σε RTF

Αυτή η λειτουργία εστιάζει στην εξαγωγή εικόνων με ρυθμίσεις που βελτιώνουν τη συμβατότητα με παλαιότερα προγράμματα ανάγνωσης RTF.

#### Επισκόπηση

Παλαιότερα προγράμματα ανάγνωσης RTF ενδέχεται να έχουν περιορισμούς στον χειρισμό ορισμένων μορφών εικόνας. Αυτή η λειτουργικότητα βοηθά να διασφαλιστεί ότι το έγγραφό σας είναι προσβάσιμο σε ένα ευρύ φάσμα λογισμικού, προσαρμόζοντας τις παραμέτρους εξαγωγής.

#### Βήμα προς βήμα εφαρμογή

##### Βήμα 1: Ρύθμιση επιλογών εγγράφου και εξαγωγής

Δείτε πώς μπορείτε να διαμορφώσετε το έγγραφό σας για βέλτιστη συμβατότητα:

```python
import aspose.words as aw

def export_images_example():
    for export_images_for_old_readers in (False, True):
        doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Rendering.docx')

        # Ρύθμιση παραμέτρων επιλογών αποθήκευσης RTF
        options = aw.saving.RtfSaveOptions()
        options.export_compact_size = True  # Μειώστε το μέγεθος του αρχείου με κάποιο κόστος συμβατότητας
        options.export_images_for_old_readers = export_images_for_old_readers

        # Αποθήκευση του εγγράφου με καθορισμένες επιλογές
        doc.save('YOUR_OUTPUT_DIRECTORY/RtfSaveOptions.export_images.rtf', options)

        # Επαληθεύστε ότι το αποθηκευμένο RTF περιέχει κατάλληλες λέξεις-κλειδιά
        with open('YOUR_OUTPUT_DIRECTORY/RtfSaveOptions.export_images.rtf', 'rb') as file:
            data = file.read().decode('utf-8')
            if export_images_for_old_readers:
                assert 'nonshppict' in data
                assert 'shprslt' in data
            else:
                assert 'nonshppict' not in data
                assert 'shprslt' not in data

export_images_example()
```

##### Βασικές επιλογές διαμόρφωσης:
- `export_compact_size`: Μειώνει το μέγεθος του αρχείου, αλλά ενδέχεται να επηρεάσει ορισμένες λειτουργίες της εικόνας.
- `export_images_for_old_readers`: Εξασφαλίζει ότι οι εικόνες είναι συμβατές με παλαιότερα προγράμματα ανάγνωσης RTF.

#### Συμβουλές αντιμετώπισης προβλημάτων

Εάν αντιμετωπίσετε προβλήματα:
- Επιβεβαιώστε ότι το έγγραφο εισαγωγής σας έχει σωστή μορφοποίηση και είναι προσβάσιμο.
- Βεβαιωθείτε ότι οι ρυθμίσεις συμβατότητας ευθυγραμμίζονται με την προβλεπόμενη περίπτωση χρήσης του εγγράφου σας.
