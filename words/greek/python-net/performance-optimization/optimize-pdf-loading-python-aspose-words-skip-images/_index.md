### Παράλειψη εικόνων PDF κατά τη φόρτωση
Η παράλειψη εικόνων μπορεί να είναι ζωτικής σημασίας για εφαρμογές όπου απαιτείται μόνο περιεχόμενο κειμένου από ένα PDF, βελτιώνοντας τους χρόνους φόρτωσης και μειώνοντας τη χρήση μνήμης.

#### Βήμα 1: Ορίστε τις διαδρομές των εγγράφων σας
Αρχικά, καθορίστε διαδρομές για έγγραφα εισόδου και εξόδου:
```python
YOUR_DOCUMENT_DIRECTORY = 'path/to/your/documents/'
YOUR_OUTPUT_DIRECTORY = 'path/to/output/directory/'

def skip_pdf_images_demo():
    file_name = YOUR_DOCUMENT_DIRECTORY + 'Images.pdf'
```
#### Βήμα 2: Ρύθμιση παραμέτρων PdfLoadOptions
Δημιουργήστε ένα `PdfLoadOptions` παράδειγμα και ρυθμίστε το ώστε να παραλείπει ή να περιλαμβάνει εικόνες:
```python
for is_skip_pdf_images in [True, False]:
    options = aw.loading.PdfLoadOptions()
    options.skip_pdf_images = is_skip_pdf_images
    options.page_index = 0
    options.page_count = 1
```
- **Παράμετροι**:
  - `skip_pdf_images`: Μια λογική τιμή για να αποφασιστεί εάν οι εικόνες θα πρέπει να παραλειφθούν.
  - `page_index` και `page_count`: Καθορίστε τις σελίδες PDF που θα φορτωθούν.

#### Βήμα 3: Φόρτωση του εγγράφου
Φορτώστε το έγγραφο με τις καθορισμένες επιλογές:
```python
doc = aw.Document(file_name=file_name, load_options=options)
```

#### Βήμα 4: Επαλήθευση φόρτωσης εικόνας
Ελέγξτε εάν υπάρχουν εικόνες με βάση τη διαμόρφωση:
```python
shape_collection = doc.get_child_nodes(aw.NodeType.SHAPE, True)

if is_skip_pdf_images:
    assert shape_collection.count == 0, 'Expected no images when skipping PDF images'
else:
    assert shape_collection.count != 0, 'Expected some images when not skipping PDF images'
# Εκτελέστε την επίδειξη
skip_pdf_images_demo()
```
