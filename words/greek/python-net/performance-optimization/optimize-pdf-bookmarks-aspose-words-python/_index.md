### Δημιουργία και Διαχείριση Σελιδοδεικτών

#### Επισκόπηση
Οι σελιδοδείκτες σε ένα PDF επιτρέπουν στους χρήστες να πλοηγούνται γρήγορα στις ενότητες. Διαχειριζόμενοι αυτούς τους σελιδοδείκτες αποτελεσματικά, βελτιώνετε σημαντικά την εμπειρία χρήστη.

#### Βήμα προς βήμα εφαρμογή

##### Προσθήκη σελιδοδεικτών με επίπεδα περιγράμματος

Μπορείτε να προσθέσετε σελιδοδείκτες και να αντιστοιχίσετε επίπεδα διάρθρωσης για να δημιουργήσετε μια ιεραρχική δομή:

```python
builder = aw.DocumentBuilder(doc)
# Έναρξη σελιδοδείκτη με το όνομα 'Σελιδοδείκτης 1'
builder.start_bookmark('Bookmark 1')
builder.writeln('Text inside Bookmark 1.')
builder.end_bookmark('Bookmark 1')

# Προσθήκη ενσωματωμένων σελιδοδεικτών
builder.start_bookmark('Bookmark 2')
builder.writeln('Text inside Nested Bookmark.')
builder.end_bookmark('Bookmark 2')
```

##### Ρύθμιση παραμέτρων επιπέδων περιγράμματος για εξαγωγή PDF

Τα επίπεδα περιγράμματος υπαγορεύουν τον τρόπο εμφάνισης των σελιδοδεικτών στο αναπτυσσόμενο μενού:

```python
pdf_save_options = aw.saving.PdfSaveOptions()
outline_levels = pdf_save_options.outline_options.bookmarks_outline_levels
outline_levels.add('Bookmark 1', 1)
outline_levels.add('Bookmark 2', 2)

# Αποθήκευση εγγράφου με σελιδοδείκτες με περίγραμμα
doc.save('output.pdf', save_options=pdf_save_options)
```

##### Αφαίρεση και διαγραφή σελιδοδεικτών

Για να τροποποιήσετε τη δομή των σελιδοδεικτών:

```python
# Αφαίρεση συγκεκριμένου σελιδοδείκτη με βάση το όνομα
outline_levels.remove('Bookmark 2')

# Διαγραφή όλων των επιπέδων περιγράμματος, ορίζοντας τους σελιδοδείκτες στις προεπιλογές
outline_levels.clear()
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**: Εάν οι σελιδοδείκτες δεν εμφανίζονται όπως αναμένεται στα PDF, βεβαιωθείτε ότι έχετε αποθηκεύσει το έγγραφο με `PdfSaveOptions`.
- **Αποσφαλμάτωση**Χρησιμοποιήστε εντολές εκτύπωσης ή καταγραφή για να επαληθεύσετε τα ονόματα σελιδοδεικτών και τα επίπεδα διάρθρωσης.
