### Καθορισμός αρχικού ονόματος αρχείου με ChmLoadOptions

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε το αρχικό όνομα αρχείου ενός αρχείου .chm, διασφαλίζοντας ότι όλοι οι εσωτερικοί σύνδεσμοι έχουν επιλυθεί σωστά.

#### Βήμα 1: Εισαγωγή απαραίτητων ενοτήτων
Ξεκινήστε με την εισαγωγή `aspose.words` και `io`:

```python
import aspose.words as aw
import io
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών φόρτωσης
Δημιουργήστε μια παρουσία του `ChmLoadOptions` και ορίστε το αρχικό όνομα αρχείου:

```python
load_options = aw.loading.ChmLoadOptions()
load_options.original_file_name = 'amhelp.chm'
```
**Εξήγηση:**
Ρύθμιση του `original_file_name` βοηθά το Aspose.Words να επιλύει με ακρίβεια συνδέσμους μέσα στο αρχείο CHM σας, αποτρέποντας κατεστραμμένες διευθύνσεις URL.

#### Βήμα 3: Φόρτωση και αποθήκευση του εγγράφου
Χρησιμοποιήστε αυτές τις επιλογές για να φορτώσετε ένα έγγραφο .chm:

```python
doc = aw.Document(
    stream=io.BytesIO(system_helper.io.File.read_all_bytes(YOUR_DOCUMENT_DIRECTORY + 'Document with ms-its links.chm')),
    load_options=load_options
)
```
Αποθηκεύστε το ως αρχείο HTML, διατηρώντας τους διορθωμένους συνδέσμους:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'ExChmLoadOptions.OriginalFileName.html')
```
**Συμβουλή αντιμετώπισης προβλημάτων:**
Βεβαιωθείτε ότι η διαδρομή προς το αρχείο .chm είναι σωστή και προσβάσιμη. Εάν οι διαδρομές είναι λανθασμένες, προσαρμόστε τες ανάλογα στον κώδικά σας.
