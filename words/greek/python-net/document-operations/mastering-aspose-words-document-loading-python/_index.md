#### Φόρτωση εγγράφου από αρχείο

**Επισκόπηση:**

Η φόρτωση ενός εγγράφου απευθείας από μια τοπική διαδρομή αρχείου συστήματος είναι απλή. Δείτε πώς μπορείτε να το κάνετε:

##### Βήμα 1: Αρχικοποίηση της κλάσης Loader

Δημιουργήστε μια παρουσία της προσαρμοσμένης κλάσης μας που έχει σχεδιαστεί για να χειρίζεται τη φόρτωση εγγράφων.

```python
class LoadDocumentsWithComHelper:
    def __init__(self):
        self.com_helper = aw.ComHelper()
```

##### Βήμα 2: Ορίστε τη μέθοδο φόρτωσης αρχείων

Υλοποιήστε μια μέθοδο που παίρνει μια διαδρομή αρχείου και χρησιμοποιεί `com_helper.open` για να φορτώσετε το έγγραφο.

```python
def open_document_from_file(self, file_path):
    """
    Opens a document using a local system filename.
    
    :param file_path: Path to the document file
    """
    doc = self.com_helper.open(file_name=file_path)
    return doc.get_text().strip()
```

**Εξήγηση:** Ο `open` Η μέθοδος διαβάζει το καθορισμένο αρχείο και επιστρέφει ένα `Document` αντικείμενο, από το οποίο μπορείτε να εξαγάγετε κείμενο ή άλλα δεδομένα.

#### Φόρτωση εγγράφου από μια ροή

**Επισκόπηση:**

Σε περιπτώσεις όπου τα έγγραφα δεν αποθηκεύονται τοπικά αλλά η πρόσβαση σε αυτά γίνεται μέσω ροών (π.χ., απαντήσεις δικτύου), η αποτελεσματική φόρτωσή τους είναι το κλειδί.

##### Βήμα 1: Ορίστε τη μέθοδο για τη φόρτωση ροής

Υλοποιήστε μια άλλη μέθοδο για τη διαχείριση της φόρτωσης εγγράφων από μια ροή εισόδου:

```python
from io import BytesIO

def open_document_from_stream(self, stream):
    """
    Opens a document using an input stream.
    
    :param stream: A BytesIO stream containing the document data
    """
    doc = self.com_helper.open(stream=stream)
    return doc.get_text().strip()
```

**Εξήγηση:** Αυτή η μέθοδος χρησιμοποιεί `BytesIO` για την προσομοίωση αντικειμένων που μοιάζουν με αρχεία από ροές byte, επιτρέποντας την απρόσκοπτη φόρτωση εγγράφων χωρίς να χρειάζεται φυσικό αρχείο.
