## Οδηγός Εφαρμογής

### Περιορισμός επιπέδου επικεφαλίδων στο XPS Outline (Λειτουργία 1)

#### Επισκόπηση

Αυτή η λειτουργία σάς βοηθά να ελέγχετε το βάθος των επικεφαλίδων που περιλαμβάνονται στο περίγραμμα ενός εγγράφου XPS, διασφαλίζοντας ότι επισημαίνονται μόνο οι σχετικές ενότητες για σκοπούς πλοήγησης.

#### Ρύθμιση και απόσπασμα κώδικα

```python
import aspose.words as aw

class LimitedHeadingsXps:
    def __init__(self):
        self.doc = aw.Document()
        self.builder = aw.DocumentBuilder(doc=self.doc)
        
    def setup_headings(self):
        # Εισαγωγή επικεφαλίδων που θα χρησιμεύσουν ως καταχωρίσεις Περιεχομένου (TOC) των επιπέδων 1, 2 και 3
        self.builder.paragraph_format.style_identifier = aw.StyleIdentifier.HEADING1
        self.builder.writeln('Heading 1')
        self.builder.paragraph_format.style_identifier = aw.StyleIdentifier.HEADING2
        self.builder.writeln('Heading 1.1')
        self.builder.writeln('Heading 1.2')
        self.builder.paragraph_format.style_identifier = aw.StyleIdentifier.HEADING3
        self.builder.writeln('Heading 1.2.1')
        self.builder.writeln('Heading 1.2.2')
    
    def save_with_limited_outline(self, output_path):
        # Δημιουργήστε το XpsSaveOptions για να τροποποιήσετε τη μετατροπή του εγγράφου σε .XPS
        save_options = aw.saving.XpsSaveOptions()
        save_options.outline_options.headings_outline_levels = 2  # Περιορισμός σε επικεφαλίδες επιπέδου 2
        self.doc.save(file_name=output_path + 'LimitedHeadingsOutline.xps', save_options=save_options)

# Παράδειγμα χρήσης:
xps_save = LimitedHeadingsXps()
xps_save.setup_headings()
xps_save.save_with_limited_outline('YOUR_DOCUMENT_DIRECTORY/')
```

#### Εξήγηση

- **`setup_headings()`**: Αυτή η μέθοδος χρησιμοποιεί το `DocumentBuilder` για να εισαγάγετε επικεφαλίδες διαφόρων επιπέδων στο έγγραφο.
- **`save_with_limited_outline(output_path)`**: Εδώ, διαμορφώνουμε `XpsSaveOptions` για να περιορίσετε τα επίπεδα διάρθρωσης σε 2. Αυτό διασφαλίζει ότι μόνο οι επικεφαλίδες έως το επίπεδο 2 περιλαμβάνονται στο παράθυρο πλοήγησης του εγγράφου XPS.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι το περιβάλλον Python σας έχει ρυθμιστεί σωστά με εγκατεστημένο το Aspose.Words.
- Ελέγξτε τις διαδρομές αρχείων και τα δικαιώματα καταλόγου εάν αντιμετωπίσετε σφάλματα αποθήκευσης.

### Υπογραφή εγγράφου XPS με ψηφιακή υπογραφή (Λειτουργία 2)

#### Επισκόπηση

Η ψηφιακή υπογραφή εγγράφων διασφαλίζει την αυθεντικότητά τους, παρέχοντας ένα επίπεδο ασφάλειας κρίσιμο για ευαίσθητες πληροφορίες. Αυτή η λειτουργία σάς επιτρέπει να εφαρμόζετε ψηφιακές υπογραφές κατά την αποθήκευση εγγράφων σε μορφή XPS.

#### Ρύθμιση και απόσπασμα κώδικα

```python
import aspose.words as aw
import datetime

class SignedXpsDocument:
    def __init__(self, input_path):
        self.doc = aw.Document(file_name=input_path)
        
    def sign_document(self, certificate_path, password, output_path):
        # Δημιουργήστε στοιχεία ψηφιακής υπογραφής
        certificate_holder = aw.digitalsignatures.CertificateHolder.create(
            file_name=certificate_path, password=password)
        options = aw.digitalsignatures.SignOptions()
        options.sign_time = datetime.datetime.now()
        options.comments = 'Some comments'
        
        digital_signature_details = aw.saving.DigitalSignatureDetails(certificate_holder, options)
        save_options = aw.saving.XpsSaveOptions()
        save_options.digital_signature_details = digital_signature_details
        
        # Αποθήκευση του υπογεγραμμένου εγγράφου ως XPS
        self.doc.save(file_name=output_path + 'SignedXpsDocument.xps', save_options=save_options)

# Παράδειγμα χρήσης:
signed_xps = SignedXpsDocument('YOUR_DOCUMENT_DIRECTORY/Document.docx')
signed_xps.sign_document('YOUR_DOCUMENT_DIRECTORY/morzal.pfx', 'aw', 'YOUR_OUTPUT_DIRECTORY/')
```

#### Εξήγηση

- **`sign_document(certificate_path, password, output_path)`**Αυτή η μέθοδος ρυθμίζει την ψηφιακή υπογραφή χρησιμοποιώντας ένα συγκεκριμένο πιστοποιητικό και αποθηκεύει το υπογεγραμμένο έγγραφο.
- **`CertificateHolder.create()`**: Αρχικοποιεί τον κάτοχο του πιστοποιητικού με το αρχείο ψηφιακού πιστοποιητικού σας.
- **`SignOptions()`**Ρυθμίζει τις λεπτομέρειες της υπογραφής, όπως την ώρα υπογραφής και τα σχόλια.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι το ψηφιακό πιστοποιητικό είναι έγκυρο και προσβάσιμο.
- Επαληθεύστε την ακρίβεια του κωδικού πρόσβασης για την πρόσβαση στο αρχείο πιστοποιητικού.
