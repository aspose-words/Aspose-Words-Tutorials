# Ανάκτηση Κατεστραμμένου DOCX – Πώς να Ενεργοποιήσετε τη Λειτουργία Ανάκτησης και να Λάβετε τον Αριθμό Σελίδων

Έχετε ποτέ χρειαστεί να **ανακτήσετε κατεστραμμένα docx** αρχεία και αναρωτηθήκατε αν υπάρχει προγραμματιστικός τρόπος για να το κάνετε; Δεν είστε μόνοι. Σε πολλά πραγματικά έργα ένα έγγραφο Word μπορεί να γίνει μη αναγνώσιμο λόγω κακής αποθήκευσης, δικτυακού σφάλματος ή απρόσμενης διακοπής λειτουργίας. Τα καλά νέα; Το Aspose.Words for Python via .NET σας παρέχει μια ενσωματωμένη μηχανή ανάκτησης που συχνά μπορεί να **διορθώσει κατεστραμμένο αρχείο Word** χωρίς χειροκίνητη παρέμβαση.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες για να **ενεργοποιήσετε τη λειτουργία ανάκτησης**, να φορτώσετε ένα κατεστραμμένο έγγραφο και να **λάβετε τον αριθμό σελίδων** ώστε να επαληθεύσετε ότι το αρχείο είναι χρησιμοποιήσιμο. Στο τέλος θα έχετε ένα έτοιμο‑για‑εκτέλεση script που αυτόματα προσπαθεί να **ανακτήσει κατεστραμμένα word** αρχεία και σας ενημερώνει αν η λειτουργία πέτυχε.

> **Προαπαιτούμενα** – Χρειάζεστε μια έγκυρη άδεια Aspose.Words (ή μπορείτε να εργαστείτε σε λειτουργία αξιολόγησης) και Python 3.8+ με το πακέτο `aspose-words` εγκατεστημένο (`pip install aspose-words`). Δεν απαιτούνται άλλες εξαρτήσεις.

## Βήμα 1: Εγκατάσταση και Εισαγωγή του Aspose.Words

Πριν μπορέσουμε να **ανακτήσουμε κατεστραμμένα docx**, χρειαζόμαστε τη βιβλιοθήκη. Αν δεν την έχετε εγκαταστήσει ακόμη, εκτελέστε:

Τώρα εισάγετε το πακέτο στο script σας:

```python
# Step 1: Import the Aspose.Words library
import aspose.words as aw
```

> **Συμβουλή:** Κρατήστε την έκδοση του Aspose.Words ενημερωμένη· η τελευταία έκδοση (από Μάρτιο 2026) προσθέτει νέες μεθόδους ανάκτησης που βελτιώνουν τις πιθανότητες διόρθωσης ενός κατεστραμμένου αρχείου.

## Βήμα 2: Προετοιμασία LoadOptions και Ενεργοποίηση Λειτουργίας Ανάκτησης

Η μαγεία συμβαίνει στο `LoadOptions`. Από προεπιλογή το Aspose.Words θα ρίξει εξαίρεση αν το αρχείο είναι κατεστραμμένο. Αλλάζουμε αυτή τη συμπεριφορά ενεργοποιώντας τη **λειτουργία ανάκτησης**.

```python
# Step 2: Create load options to control how the document is opened
load_options = aw.loading.LoadOptions()

# Step 3: Enable recovery mode so Aspose.Words attempts to fix a corrupted file
load_options.recovery_mode = aw.loading.RecoveryMode.RECOVER  # alternatives: THROW, AUTO
```

### Γιατί `RecoveryMode.RECOVER`;

- **RECOVER** – Το Aspose.Words σαρώει το αρχείο, απορρίπτει τα μη αναγνώσιμα τμήματα και προσπαθεί να ξαναχτίσει ένα χρησιμοποιήσιμο έγγραφο.  
- **THROW** – Η προεπιλογή· οποιαδήποτε κατεστραμμένη κατάσταση προκαλεί εξαίρεση.  
- **AUTO** – Αφήνει τη βιβλιοθήκη να αποφασίσει βάσει της σοβαρότητας· δεν είναι τόσο επιθετική όσο το `RECOVER`.

Αν διαχειρίζεστε δεδομένα κρίσιμης σημασίας, μπορείτε να ξεκινήσετε με `AUTO` και να επιστρέψετε στο `RECOVER` μόνο όταν είναι απαραίτητο.

---

## Βήμα 3: Φόρτωση του Πιθανώς Κατεστραμμένου Εγγράφου

Τώρα κατευθύνουμε το Aspose.Words στο αρχείο που υποπτευόμαστε ότι είναι κατεστραμμένο. Το `load_options` που διαμορφώσαμε θα εφαρμοστεί αυτόματα.

```python
# Step 4: Load the potentially corrupted document using the configured options
doc_path = "YOUR_DIRECTORY/Corrupted.docx"   # <-- replace with your actual path
document = aw.Document(doc_path, load_options)
```

Αν το αρχείο δεν μπορεί να ανοιχθεί ακόμη και σε λειτουργία ανάκτησης, το Aspose.Words θα εξακολουθήσει να ρίξει εξαίρεση. Τυλίξτε την κλήση σε μπλοκ `try/except` για να το διαχειριστείτε ομαλά:

```python
try:
    document = aw.Document(doc_path, load_options)
except Exception as e:
    print(f"Failed to recover the document: {e}")
    raise
```

## Βήμα 4: Επαλήθευση Επιτυχίας – Λήψη Αριθμού Σελίδων

Ένας γρήγορος τρόπος για να επιβεβαιώσετε ότι το έγγραφο φορτώθηκε σωστά είναι να διαβάσετε το `page_count`. Αυτό επίσης ικανοποιεί την απαίτηση **λήψης αριθμού σελίδων**.

```python
# Step 5: Verify that the document was loaded by printing its page count
print("Document loaded, page count:", document.page_count)
```

### Αναμενόμενη Έξοδος

```
Document loaded, page count: 12
```

Αν ο αριθμός σελίδων είναι `0`, η διαδικασία ανάκτησης πιθανώς αφαίρεσε όλο το περιεχόμενο, υποδεικνύοντας σοβαρά κατεστραμμένο αρχείο. Σε αυτήν την περίπτωση ίσως χρειαστεί να ζητήσετε από τον χρήστη μια νέα αντίγραφο.

## Πλήρες, Έτοιμο‑για‑Εκτέλεση Script

Παρακάτω βρίσκεται το πλήρες παράδειγμα, συμπεριλαμβανομένης της διαχείρισης σφαλμάτων και μιας μικρής βοηθητικής συνάρτησης που επιστρέφει boolean που υποδεικνύει επιτυχία.

```python
import aspose.words as aw

def recover_docx(file_path: str) -> bool:
    """
    Attempts to recover a corrupted DOCX file using Aspose.Words.
    Returns True if the document loads and has at least one page.
    """
    # Configure load options with recovery mode
    load_options = aw.loading.LoadOptions()
    load_options.recovery_mode = aw.loading.RecoveryMode.RECOVER

    try:
        # Load the document
        doc = aw.Document(file_path, load_options)
        # Output page count for verification
        print("Document loaded, page count:", doc.page_count)
        return doc.page_count > 0
    except Exception as exc:
        print(f"Failed to recover the document: {exc}")
        return False

# Example usage
if __name__ == "__main__":
    path = "YOUR_DIRECTORY/Corrupted.docx"   # Update this path
    if recover_docx(path):
        print("✅ Recovery succeeded!")
    else:
        print("❌ Recovery failed – consider obtaining a clean copy.")
```

Αποθηκεύστε το ως `recover_docx.py` και εκτελέστε:

```bash
python recover_docx.py
```

Θα πρέπει να δείτε τον αριθμό σελίδων να εκτυπώνεται, ακολουθούμενος από μήνυμα επιτυχίας ή αποτυχίας.

## Διαχείριση Ακραίων Περιπτώσεων & Συχνές Ερωτήσεις

### Τι γίνεται αν το αρχείο δεν είναι DOCX;

`LoadOptions` λειτουργεί για **.doc**, **.docx**, **.rtf**, **.pdf**, και πολλές άλλες μορφές. Αν περάσετε ένα αρχείο που δεν είναι Word, το Aspose.Words θα προσπαθήσει μετατροπή, αλλά οι μέθοδοι ανάκτησης είναι βελτιστοποιημένες για δομές ειδικές του Word. Για καλύτερα αποτελέσματα, ελέγξτε την επέκταση του αρχείου πριν καλέσετε το `recover_docx`.

### Μπορώ να ανακτήσω ένα αρχείο προστατευμένο με κωδικό;

Η λειτουργία ανάκτησης **δεν** παρακάμπτει την κρυπτογράφηση. Πρέπει να παρέχετε τον κωδικό μέσω `load_options.password`. Παράδειγμα:

```python
load_options.password = "mySecret"
```

### Πώς διαφέρει το **recover damaged word** από το απλό άνοιγμα του αρχείου στο Word;

Η ενσωματωμένη επισκευή του Microsoft Word συχνά σταματά στο πρώτο κρίσιμο σφάλμα, ενώ το Aspose.Words συνεχίζει τη σάρωση, απορρίπτοντας μόνο τα κατεστραμμένα τμήματα και διατηρώντας το υπόλοιπο. Αυτό μπορεί να αποδώσει ένα πιο χρησιμοποιήσιμο έγγραφο, ειδικά για μεγάλες συμβάσεις όπου μόνο μια παράγραφος είναι κατεστραμμένη.

### Πρέπει πάντα να χρησιμοποιώ το `RECOVER`;

Όχι απαραίτητα. Το `RECOVER` μπορεί να είναι επιθετικό και να αφαιρέσει περιεχόμενο που χρειάζεστε. Αν διαχειρίζεστε νομικά έγγραφα, ξεκινήστε με `AUTO` και ελέγξτε το αποτέλεσμα πριν προχωρήσετε σε πλήρη ανάκτηση.
