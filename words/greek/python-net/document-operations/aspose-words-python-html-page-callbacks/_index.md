### Οδηγός Εφαρμογής

#### Αποθήκευση κάθε σελίδας ως ξεχωριστό αρχείο HTML

Θα δείξουμε πώς να αποθηκεύσετε κάθε σελίδα εγγράφου του Word ως ξεχωριστό αρχείο HTML χρησιμοποιώντας το Aspose.Words. `IPageSavingCallback`.

##### Επισκόπηση

Προσαρμόστε τη διαδικασία αποθήκευσης εφαρμόζοντας μια επανάκληση που καθορίζει ονόματα αρχείων για τις σελίδες εξόδου.

##### Οδηγός βήμα προς βήμα

**1. Δημιουργία και Ρύθμιση Εγγράφου:**

Δημιουργήστε ή φορτώστε ένα έγγραφο χρησιμοποιώντας το Aspose.Words:

```python
import aspose.words as aw

doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Page 1.")
builder.insert_break(aw.BreakType.PAGE_BREAK)
builder.writeln("Page 2.")
builder.insert_image("path/to/image.jpg")
builder.insert_break(aw.BreakType.PAGE_BREAK)
builder.writeln("Page 3.")
```

**2. Ρύθμιση παραμέτρων σταθερών επιλογών αποθήκευσης HTML:**

Στήνω `HtmlFixedSaveOptions` και να αντιστοιχίσετε μια προσαρμοσμένη επιστροφή κλήσης με αποθήκευση σελίδας:

```python
html_fixed_save_options = aw.saving.HtmlFixedSaveOptions()
html_fixed_save_options.page_saving_callback = CustomFileNamePageSavingCallback(ARTIFACTS_DIR)
```

**3. Υλοποίηση προσαρμοσμένης κλάσης επανάκλησης:**

Ορίστε το `CustomFileNamePageSavingCallback` τάξη:

```python
class CustomFileNamePageSavingCallback(aw.saving.IPageSavingCallback):
    def __init__(self, output_dir):
        self.output_dir = output_dir

    def page_saving(self, args):
        # Καθορίστε το όνομα αρχείου για την τρέχουσα σελίδα
        args.page_file_name = f"{self.output_dir}/page_{args.page_index + 1}.html"
```

**4. Αποθηκεύστε το έγγραφο:**

Αποθηκεύστε το έγγραφό σας χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```python
doc.save(f"{ARTIFACTS_DIR}/output.html", html_fixed_save_options)
```
