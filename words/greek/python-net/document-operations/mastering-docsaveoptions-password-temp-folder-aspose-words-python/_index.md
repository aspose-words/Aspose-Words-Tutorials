## Οδηγός Εφαρμογής

### Προστασία με κωδικό πρόσβασης και διατήρηση αποσπάσματος δρομολόγησης (H2)

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να ορίσετε κωδικούς πρόσβασης για παλαιότερες μορφές εγγράφων του Microsoft Word, διασφαλίζοντας ότι τα έγγραφά σας είναι ασφαλή. Επιπλέον, διατηρεί τις πληροφορίες του δελτίου δρομολόγησης κατά τη διαδικασία αποθήκευσης.

##### Ρύθμιση του DocSaveOptions με προστασία με κωδικό πρόσβασης (H3)

Αρχικά, δημιουργήστε ένα νέο έγγραφο και ρυθμίστε τις παραμέτρους `DocSaveOptions`:

```python
import aspose.words as aw

def save_with_password_and_routing_slip():
    # Δημιουργήστε ένα νέο έγγραφο
    doc = aw.Document()
    builder = aw.DocumentBuilder(doc=doc)
    builder.write('Hello world!')

    # Ρύθμιση παραμέτρων DocSaveOptions για προστασία με κωδικό πρόσβασης
    options = aw.saving.DocSaveOptions(aw.SaveFormat.DOC)
    options.password = 'MyPassword'

    # Διατήρηση πληροφοριών δελτίου δρομολόγησης
    options.save_routing_slip = True

    # Αποθήκευση του εγγράφου
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithPasswordAndRoutingSlip.doc"
    doc.save(file_name=output_path, save_options=options)

    # Επαλήθευση με φόρτωση με κωδικό πρόσβασης
    load_options = aw.loading.LoadOptions(password='MyPassword')
    loaded_doc = aw.Document(file_name=output_path, load_options=load_options)
    assert 'Hello world!' == loaded_doc.get_text().strip()
```

**Επεξήγηση παραμέτρων:**
- `options.password`: Ορίζει τον κωδικό πρόσβασης για την προστασία εγγράφων.
- `options.save_routing_slip`Διατηρεί τις πληροφορίες του δελτίου δρομολόγησης.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου υπάρχει πριν από την αποθήκευση.
- Χρησιμοποιήστε έναν μοναδικό και ισχυρό κωδικό πρόσβασης για να βελτιώσετε την ασφάλεια.

### Προσωρινή χρήση φακέλου (H2)

#### Επισκόπηση

Όταν χειρίζεστε μεγάλα έγγραφα, η χρήση ενός προσωρινού φακέλου στο δίσκο μπορεί να βελτιώσει την απόδοση μειώνοντας τη χρήση μνήμης.

##### Ρύθμιση παραμέτρων DocSaveOptions για προσωρινούς φακέλους (H3)

Δείτε πώς μπορείτε να ρυθμίσετε έναν προσωρινό φάκελο:

```python
import os
import aspose.words as aw

def save_using_temp_folder():
    # Φόρτωση υπάρχοντος εγγράφου
    input_path = "YOUR_DOCUMENT_DIRECTORY/Rendering.docx"
    doc = aw.Document(file_name=input_path)

    # Ρύθμιση παραμέτρων του DocSaveOptions για χρήση προσωρινού φακέλου
    options = aw.saving.DocSaveOptions()
    temp_folder = "YOUR_OUTPUT_DIRECTORY/TempFiles"

    # Βεβαιωθείτε ότι υπάρχει ο προσωρινός φάκελος
    os.makedirs(temp_folder, exist_ok=True)
    options.temp_folder = temp_folder

    # Αποθήκευση χρησιμοποιώντας τον προσωρινό φάκελο
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithTempFolder.doc"
    doc.save(file_name=output_path, save_options=options)
```

**Βασικές επιλογές διαμόρφωσης:**
- `options.temp_folder`: Καθορίζει τη διαδρομή που θα χρησιμοποιηθεί για την ενδιάμεση αποθήκευση αρχείων.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Επαληθεύστε τα δικαιώματα εγγραφής για τον προσωρινό σας φάκελο.
- Βεβαιωθείτε ότι υπάρχει επαρκής χώρος στο δίσκο στον καθορισμένο κατάλογο.
