## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη μετατροπή αρχείων DOCX σε XAML σταθερής μορφής με διαχείριση πόρων. Θα εξετάσουμε κάθε λειτουργία βήμα προς βήμα.

### Μετατροπή εγγράφου σε XAML σταθερής μορφής

#### Επισκόπηση
Αυτό το μέρος εστιάζει στη χρήση του Aspose.Words. `save` μέθοδος για να μετατρέψετε το έγγραφό σας σε μορφή XAML σταθερής μορφής.

#### Βήμα 1: Φόρτωση του εγγράφου σας
Ξεκινήστε φορτώνοντας το αρχείο DOCX σε ένα Aspose.Words `Document` αντικείμενο:

```python
doc = aw.Document(MY_DIR + "Rendering.docx")
```

#### Βήμα 2: Δημιουργία επιλογών αποθήκευσης
Αρχικοποίηση `XamlFixedSaveOptions` για να προσαρμόσετε τη διαδικασία αποθήκευσης:

```python
options = aw.saving.XamlFixedSaveOptions()
```

#### Βήμα 3: Ρύθμιση παραμέτρων διαχείρισης πόρων
Ορίστε τον τρόπο διαχείρισης των συνδεδεμένων πόρων ορίζοντας το `resources_folder`, `resources_folder_alias`και μια συνάρτηση επανάκλησης.

```python
callback = ExXamlFixedSaveOptions.ResourceUriPrinter()
options.resource_saving_callback = callback
options.resources_folder = ARTIFACTS_DIR + "XamlFixedResourceFolder"
options.resources_folder_alias = ARTIFACTS_DIR + "XamlFixedFolderAlias"

# Βεβαιωθείτε ότι ο φάκελος ψευδωνύμων υπάρχει πριν από την αποθήκευση πόρων
os.makedirs(options.resources_folder_alias)
```

#### Βήμα 4: Αποθήκευση του εγγράφου
Τέλος, αποθηκεύστε το έγγραφό σας χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```python
doc.save(ARTIFACTS_DIR + "XamlFixedSaveOptions.resource_folder.xaml", options)
```

### URI πόρων παρακολούθησης
Για την παρακολούθηση και την εκτύπωση URI πόρων κατά τη μετατροπή, εφαρμόστε ένα `ResourceUriPrinter` κλάση που μετρά και καταγράφει κάθε URI.

#### Επισκόπηση
Ο μηχανισμός επανάκλησης βοηθά στην παρακολούθηση των πόρων που δημιουργήθηκαν κατά τη λειτουργία αποθήκευσης.

#### Υλοποίηση της κλάσης επανάκλησης
Δείτε πώς μπορείτε να ορίσετε μια προσαρμοσμένη επανάκληση για τη διαχείριση της εξοικονόμησης πόρων:

```python
class ResourceUriPrinter(aw.saving.IResourceSavingCallback):
    """Counts and prints URIs of resources created during conversion."""
    
    def __init__(self):
        self.resources = []  # τύπος: Λίστα[str]
    
    def resource_saving(self, args: aw.saving.ResourceSavingArgs):
        self.resources.append(f"Resource \"{args.resource_file_name}\"\n\t{args.resource_file_uri}")
        
        # Ανακατεύθυνση ροών στον φάκελο ψευδωνύμων
        args.resource_stream = open(args.resource_file_uri, 'wb')
        args.keep_resource_stream_open = False
```

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι όλοι οι κατάλογοι που καθορίζονται στο `resources_folder` και `resources_folder_alias` υπάρχουν πριν από την εκτέλεση του σεναρίου σας.
- Ελέγξτε ξανά τις διαδρομές αρχείων για τυχόν τυπογραφικά λάθη.
