## Οδηγός Εφαρμογής

### Χειρισμός αρχικών ονομάτων αρχείων σε αρχεία CHM

#### Επισκόπηση
Η διατήρηση εσωτερικών συνδέσμων κατά τη μετατροπή CHM σε HTML απαιτεί τον ορισμό του αρχικού ονόματος αρχείου χρησιμοποιώντας `ChmLoadOptions`Αυτό διασφαλίζει ότι όλες οι αναφορές συνδέσμων παραμένουν έγκυρες.

##### Βήμα 1: Δημιουργία στιγμιότυπου ChmLoadOptions
Δημιουργήστε μια παρουσία του `ChmLoadOptions` και ορίστε το αρχικό όνομα αρχείου:
```java
import com.aspose.words.ChmLoadOptions;
import java.nio.file.Files;
import java.nio.file.Paths;
import java.io.ByteArrayInputStream;

// Δημιουργήστε ένα αντικείμενο ChmLoadOptions
ChmLoadOptions loadOptions = new ChmLoadOptions();
loadOptions.setOriginalFileName("amhelp.chm"); // Ορίστε το αρχικό όνομα αρχείου CHM
```
**Εξήγηση**: Ρύθμιση `setOriginalFileName` βοηθά το Aspose.Words να κατανοήσει το περιεχόμενο του εγγράφου, διασφαλίζοντας ότι οι σύνδεσμοι μέσα στο αρχείο έχουν επιλυθεί σωστά.

##### Βήμα 2: Φόρτωση του αρχείου CHM
Φορτώστε το αρχείο CHM σε ένα Aspose.Words `Document` αντικείμενο χρησιμοποιώντας τις καθορισμένες επιλογές:
```java
import com.aspose.words.Document;

// Διαβάστε το αρχείο CHM ως πίνακα byte byte[] chmData = Files.readAllBytes(Paths.get("YOUR_DOCUMENT_DIRECTORY/Document with ms-its links.chm"));

// Φορτώστε το έγγραφο χρησιμοποιώντας το ChmLoadOptions
Document doc = new Document(new ByteArrayInputStream(chmData), loadOptions);
```
##### Βήμα 3: Αποθήκευση σε HTML
Αποθηκεύστε το φορτωμένο έγγραφο ως αρχείο HTML:
```java
// Αποθήκευση του εγγράφου ως HTML
doc.save("YOUR_OUTPUT_DIRECTORY/ExChmLoadOptions.OriginalFileName.html");
```
**Συμβουλές αντιμετώπισης προβλημάτων**: Εάν οι σύνδεσμοι δεν λειτουργούν, επαληθεύστε ότι `setOriginalFileName` ταιριάζει με το βασικό όνομα αρχείου που χρησιμοποιείται στην εσωτερική δομή του CHM και βεβαιωθείτε ότι η διαδρομή του αρχείου CHM είναι σωστή.
