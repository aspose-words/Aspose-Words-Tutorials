# Πώς να περιορίσετε τα επίπεδα επικεφαλίδων σε αρχεία XPS χρησιμοποιώντας το Aspose.Words για Java: Ένας πλήρης οδηγός

## Εισαγωγή

Η δημιουργία επαγγελματικών εγγράφων με ακριβή έλεγχο περιεχομένου είναι απαραίτητη, ειδικά κατά την εξαγωγή ως αρχείο XPS. Το Aspose.Words για Java απλοποιεί αυτήν την εργασία, επιτρέποντάς σας να διαχειρίζεστε αποτελεσματικά τα επίπεδα επικεφαλίδων κατά τη μετατροπή από Word σε μορφή XPS.

Σε αυτόν τον οδηγό, θα σας δείξουμε πώς να χρησιμοποιήσετε το `XpsSaveOptions` κλάση στο Aspose.Words για Java για να περιορίσετε ποιες επικεφαλίδες εμφανίζονται στο περίγραμμα ενός εξαγόμενου αρχείου XPS. Αυτό είναι ιδιαίτερα χρήσιμο για τη δημιουργία μιας καθαρής και εστιασμένης δομής πλοήγησης εγγράφων.

**Τι θα μάθετε:**

- Χρησιμοποιώντας `XpsSaveOptions` για τον έλεγχο των περιγραμμάτων των εγγράφων
- Εφαρμογή περιορισμών επιπέδου επικεφαλίδας κατά τη διάρκεια μετατροπών XPS

## Οδηγός Εφαρμογής

Τώρα, ας επικεντρωθούμε στην εφαρμογή της δυνατότητας περιορισμού των επιπέδων επικεφαλίδων σε ένα έγγραφο XPS χρησιμοποιώντας το Aspose.Words.

### Περιορισμός επιπέδων επικεφαλίδων σε έγγραφα XPS (H2)

#### Επισκόπηση

Κατά την εξαγωγή ενός εγγράφου Word ως αρχείου XPS, ο έλεγχος των επικεφαλίδων που εμφανίζονται στο περίγραμμα βοηθά στη διατήρηση της εστίασης και στην απλοποίηση της πλοήγησης. `XpsSaveOptions` Η κλάση επιτρέπει τον καθορισμό επιπέδων επικεφαλίδων που θα συμπεριληφθούν.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε το έγγραφό σας:**

Ξεκινήστε δημιουργώντας ένα νέο έγγραφο του Word χρησιμοποιώντας το Aspose.Words. `Document` και `DocumentBuilder` τάξεις:

```java
import com.aspose.words.*;

public class OutlineLevelsExample {
    public static void main(String[] args) throws Exception {
        // Αρχικοποίηση του εγγράφου
        Document doc = new Document();
        DocumentBuilder builder = new DocumentBuilder(doc);

        // Εισαγωγή επικεφαλίδων σε διάφορα επίπεδα
        builder.getParagraphFormat().setStyleIdentifier(StyleIdentifier.HEADING_1);
        builder.writeln("Heading 1");

        builder.getParagraphFormat().setStyleIdentifier(StyleIdentifier.HEADING_2);
        builder.writeln("Heading 1.1");
        builder.writeln("Heading 1.2");

        builder.getParagraphFormat().setStyleIdentifier(StyleIdentifier.HEADING_3);
        builder.writeln("Heading 1.2.1");
        builder.writeln("Heading 1.2.2");
    }
}
```

**2. Ρυθμίστε τις παραμέτρους του XpsSaveOptions:**

Στη συνέχεια, διαμορφώστε το `XpsSaveOptions` για να περιορίσετε τα επίπεδα επικεφαλίδων που εμφανίζονται στο περίγραμμα του εγγράφου:

```java
// Δημιουργήστε ένα αντικείμενο "XpsSaveOptions"
XpsSaveOptions saveOptions = new XpsSaveOptions();

// Ορισμός μορφής αποθήκευσης
saveOptions.setSaveFormat(SaveFormat.XPS);

// Περιορισμός επικεφαλίδων στο επίπεδο 2 στο περίγραμμα εξόδου
saveOptions.getOutlineOptions().setHeadingsOutlineLevels(2);
```

**3. Αποθηκεύστε το έγγραφο:**

Τέλος, αποθηκεύστε το έγγραφό σας με αυτές τις επιλογές:

```java
doc.save("output/DocumentWithLimitedOutlines.xps", saveOptions);
```

### Βασικές επιλογές διαμόρφωσης

- **`setSaveFormat(SaveFormat.XPS)`:** Καθορίζει την αποθήκευση ως αρχείο XPS.
- **`getOutlineOptions().setHeadingsOutlineLevels(int levels)`:** Τα στοιχεία ελέγχου περιελάμβαναν επίπεδα επικεφαλίδων στο περίγραμμα.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι όλες οι εξαρτήσεις έχουν προστεθεί σωστά για να αποφύγετε `ClassNotFoundException`.
- Βεβαιωθείτε ότι η άδειά σας έχει ρυθμιστεί σωστά για πλήρη λειτουργικότητα.
