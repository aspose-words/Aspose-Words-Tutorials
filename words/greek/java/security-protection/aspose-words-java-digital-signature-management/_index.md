## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην υλοποίηση βασικών λειτουργιών χρησιμοποιώντας το Aspose.Words για Java.

### Φόρτωση και επανάληψη ψηφιακών υπογραφών

#### Επισκόπηση
Η φόρτωση και η επανάληψη ψηφιακών υπογραφών σε ένα έγγραφο διασφαλίζει ότι μπορείτε να έχετε πρόσβαση στις λεπτομέρειες κάθε υπογραφής, κάτι που είναι κρίσιμο για τις διαδικασίες ελέγχου ή επαλήθευσης.

#### Βήματα για την εφαρμογή
##### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων

```java
import com.aspose.words.DigitalSignatureCollection;
import com.aspose.words.DigitalSignatureUtil;
```

##### Βήμα 2: Φόρτωση ψηφιακών υπογραφών
Φόρτωση των ψηφιακών υπογραφών από ένα έγγραφο χρησιμοποιώντας `DigitalSignatureUtil.loadSignatures`.

```java
String documentPath = "YOUR_DOCUMENT_DIRECTORY/\"Digitally signed.docx\"";
DigitalSignatureCollection digitalSignatures =
        DigitalSignatureUtil.loadSignatures(documentPath);
```

##### Βήμα 3: Επανάληψη υπογραφών
Επαναλάβετε τη συλλογή και εκτυπώστε τις λεπτομέρειες για κάθε υπογραφή.

```java
for (com.aspose.words.DigitalSignature ds : digitalSignatures) {
    if (ds != null)
        System.out.println(ds.toString()); // Εκτύπωση λεπτομερειών υπογραφής
}
```

#### Εξήγηση
- **ΨηφιακήΥπογραφήUtil.loadΥπογραφές**Αυτή η μέθοδος φορτώνει όλες τις ψηφιακές υπογραφές από ένα συγκεκριμένο έγγραφο.
- **Μέθοδος toString()**Παρέχει μια αναπαράσταση συμβολοσειράς των ιδιοτήτων της υπογραφής, βοηθώντας στην αποσφαλμάτωση και την επαλήθευση.

### Επικύρωση και έλεγχος ψηφιακών υπογραφών

#### Επισκόπηση
Η επικύρωση των ψηφιακών υπογραφών περιλαμβάνει τον έλεγχο της αυθεντικότητας και της ακεραιότητάς τους, επαληθεύοντας συγκεκριμένα χαρακτηριστικά, όπως η εγκυρότητα, ο τύπος, τα σχόλια, το όνομα του εκδότη και το όνομα του υποκειμένου.

#### Βήματα για την εφαρμογή
##### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων

```java
import com.aspose.words.DigitalSignature;
import com.aspose.words.DigitalSignatureCollection;
import com.aspose.words.DigitalSignatureType;
```

##### Βήμα 2: Φόρτωση ψηφιακών υπογραφών
Όπως και πριν, φορτώστε τις υπογραφές από το έγγραφό σας.

```java
digitalSignatures = DigitalSignatureUtil.loadSignatures("YOUR_DOCUMENT_DIRECTORY/\"Digitally signed.docx\"");
```

##### Βήμα 3: Επικύρωση ιδιοτήτων υπογραφής
Βεβαιωθείτε ότι υπάρχει ακριβώς μία υπογραφή και επικυρώστε τις ιδιότητές της.

```java
if (digitalSignatures.getCount() != 1) {
    throw new IllegalStateException("Expected one digital signature.");
}

DigitalSignature signature = digitalSignatures.get(0);

// Έλεγχος εγκυρότητας
if (!signature.isValid()) {
    throw new IllegalStateException("The digital signature is not valid.");
}

// Επαλήθευση τύπου υπογραφής
if (signature.getSignatureType() != DigitalSignatureType.XML_DSIG) {
    throw new IllegalStateException("Unexpected signature type.");
}

// Επιβεβαίωση σχολίων
if (!"Test Sign".equals(signature.getComments())) {
    throw new IllegalStateException("Unexpected comments in the signature.");
}

// Επικύρωση ονόματος εκδότη
String expectedIssuerName = "CN=VeriSign Class 3 Code Signing 2009-2 CA, OU=Terms of use at https://www.verisign.com/rpa (c)09, OU=Δίκτυο Εμπιστοσύνης VeriSign, O=\"VeriSign, Inc.\", C=ΗΠΑ";
if (!expectedIssuerName.equals(signature.getIssuerName())) {
    throw new IllegalStateException("Unexpected issuer name.");
}

// Ελέγξτε το όνομα του θέματος
String expectedSubjectName = "CN=Aspose Pty Ltd, OU=Digital ID Class 3 - Microsoft Software Validation v2, O=Aspose Pty Ltd, L=Lane Cove, S=New South Wales, C=AU";
if (!expectedSubjectName.equals(signature.getSubjectName())) {
    throw new IllegalStateException("Unexpected subject name.");
}
```

#### Εξήγηση
- **Μέθοδος isValid()**: Επιβεβαιώνει την αυθεντικότητα της υπογραφής.
- **getSignatureType()**: Διασφαλίζει ότι ο τύπος υπογραφής είναι ο αναμενόμενος (π.χ., XML_DSIG).
- **getComments(), getIssuerName() και getSubjectName()**Επαληθεύστε τα πρόσθετα μεταδεδομένα για λεπτομερή επικύρωση.
