# Ρύθμιση Αριθμών Σελίδων & Δημιουργία Πίνακα Περιεχομένων στο Aspose.Words για Java

Σε αυτό το tutorial θα ανακαλύψετε πώς να **ρυθμίσετε τους αριθμούς σελίδων** και να **εισάγετε έναν πίνακα περιεχομένων** (TOC) με το Aspose.Words για Java. Ένας καλά δομημένος Πίνακας Περιεχομένων κάνει τα μεγάλα έγγραφα εύκολα στην πλοήγηση, και η ακριβής ευθυγράμμιση των αριθμών σελίδων προσφέρει στους αναγνώστες μια επαγγελματική εμπειρία. Θα περάσουμε από τη δημιουργία ενός εγγράφου, την προσαρμογή των στυλ του TOC και τη ρύθμιση των διαστημάτων (tab stops) ώστε οι αριθμοί σελίδων να ευθυγραμμίζονται ακριβώς όπου θέλετε.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “ρύθμιση αριθμών σελίδων”;** Τροποποίηση των διαστημάτων (tab stops) που ευθυγραμμίζουν τους αριθμούς σελίδων σε έναν TOC.  
- **Μπορώ να εισάγω αυτόματα έναν πίνακα περιεχομένων;** Ναι – χρησιμοποιήστε την κλάση `FieldToc`.  
- **Χρειάζεται άδεια για την εκτέλεση του κώδικα;** Μια δωρεάν δοκιμαστική έκδοση λειτουργεί για ανάπτυξη· απαιτείται άδεια για παραγωγή.  
- **Ποια έκδοση του Aspose υποστηρίζεται;** Τα παραδείγματα λειτουργούν με την τελευταία έκδοση του Aspose.Words για Java.  
- **Μπορεί να προσαρμοστεί το στυλ του TOC;** Απόλυτα – μπορείτε να αλλάξετε γραμματοσειρές, έντονη γραφή και άλλα.

## Τι είναι ο Πίνακας Περιεχομένων στο Aspose.Words;
Ένας TOC είναι ένα πεδίο που σαρώσει το έγγραφο για στυλ επικεφαλίδων (π.χ., Heading 1, Heading 2) και δημιουργεί μια λίστα καταχωρήσεων με αριθμούς σελίδων. Το Aspose.Words σας επιτρέπει να εισάγετε αυτό το πεδίο προγραμματιστικά και να ελέγχετε πλήρως την εμφάνισή του.

## Γιατί να ρυθμίσετε τους αριθμούς σελίδων σε έναν TOC;
Η ρύθμιση των διαστημάτων (tab stops) σας δίνει ακριβή έλεγχο στο πού εμφανίζονται οι αριθμοί σελίδων, κάτι που είναι απαραίτητο για:

- Διατήρηση καθαρού, στήλης‑ευθυγραμμισμένου layout.  
- Συμμόρφωση με εταιρικά στυλ.  
- Βελτίωση της αναγνωσιμότητας σε έντυπα και ψηφιακά έγγραφα.

## Προαπαιτούμενα
- Το Aspose.Words για Java προστέθηκε στο έργο σας (Maven/Gradle).  
- Βασική εξοικείωση με τη σύνταξη της Java.  

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Δημιουργία νέου εγγράφου
Πρώτα, δημιουργήστε ένα κενό αντικείμενο `Document` που θα περιέχει το περιεχόμενό σας και τον TOC.

```java
Document doc = new Document();
```

### Βήμα 2: Προσαρμογή στυλ TOC
Μπορείτε να αλλάξετε την εμφάνιση κάθε επιπέδου του TOC. Σε αυτό το παράδειγμα κάνουμε τις καταχωρήσεις του πρώτου επιπέδου έντονες, κάτι που είναι συχνή απαίτηση μορφοποίησης.

```java
doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_1).getFont().setBold(true);
```

### Βήμα 3: Προσθήκη περιεχομένου στο έγγραφο
Εισάγετε επικεφαλίδες (π.χ., `Heading1`, `Heading2`) και κανονικές παραγράφους. Το πεδίο TOC θα εντοπίσει αυτές τις επικεφαλίδες αυτόματα. *(Ο κώδικας παραλείπεται για συντομία – η εστίαση είναι στη δημιουργία του TOC.)*

### Βήμα 4: Εισαγωγή του πεδίου TOC
Τοποθετήστε τον TOC εκεί που θέλετε – συνήθως στην αρχή του εγγράφου.

```java
// Insert a TOC field at the desired location in your document.
FieldToc fieldToc = new FieldToc();
doc.getFirstSection().getBody().getFirstParagraph().appendChild(fieldToc);
```

### Βήμα 5: Αποθήκευση του εγγράφου
Αποθηκεύστε το έγγραφο στο δίσκο. Μπορείτε να επιλέξετε οποιαδήποτε υποστηριζόμενη μορφή όπως DOCX, PDF ή HTML.

```java
doc.save("your_output_path_here");
```

## Προσαρμογή Διαστημάτων (Tab Stops) στον TOC (Ρύθμιση Αριθμών Σελίδων)
Αν το προεπιλεγμένο διάστημα δεν ευθυγραμμίζει τους αριθμούς σελίδων όπως χρειάζεστε, μπορείτε να περάσετε από όλες τις παραγράφους του TOC και να τροποποιήσετε τις θέσεις των διαστημάτων.

```java
Document doc = new Document("Table of contents.docx");

for (Paragraph para : (Iterable<Paragraph>) doc.getChildNodes(NodeType.PARAGRAPH, true))
{
    if (para.getParagraphFormat().getStyle().getStyleIdentifier() >= StyleIdentifier.TOC_1 &&
        para.getParagraphFormat().getStyle().getStyleIdentifier() <= StyleIdentifier.TOC_9)
    {
        // Get the first tab used in this paragraph, which aligns the page numbers.
        TabStop tab = para.getParagraphFormat().getTabStops().get(0);
        
        // Remove the old tab.
        para.getParagraphFormat().getTabStops().removeByPosition(tab.getPosition());
        
        // Insert a new tab at a modified position (e.g., 50 units to the left).
        para.getParagraphFormat().getTabStops().add(tab.getPosition() - 50.0, tab.getAlignment(), tab.getLeader());
    }
}

doc.save("output.docx");
```

Τώρα οι καταχωρήσεις του TOC εμφανίζουν τους αριθμούς σελίδων ακριβώς όπου θέλετε, δίνοντας στο έγγραφό σας ένα επαγγελματικό φινίρισμα.

## Συνηθισμένα Προβλήματα & Συμβουλές
- **Απουσία επικεφαλίδων στον TOC:** Βεβαιωθείτε ότι οι επικεφαλίδες σας χρησιμοποιούν ενσωματωμένα στυλ (`Heading1`, `Heading2`, κ.λπ.) ή αντιστοιχίστε προσαρμοσμένα στυλ σε επίπεδα TOC.  
- **Το διάστημα δεν εφαρμόζεται:** Ελέγξτε ότι η παράγραφος ανήκει σε στυλ TOC (`TOC_1`‑`TOC_9`).  
- **Απόδοση σε μεγάλα έγγραφα:** Καλέστε `doc.updateFields()` μετά την εισαγωγή του TOC για να ενημερώσετε τις καταχωρήσεις σε μία διεργασία.

## Συχνές Ερωτήσεις

**Ε: Πώς αλλάζω τη μορφοποίηση των καταχωρήσεων του TOC;**  
Α: Χρησιμοποιήστε `doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_X)` όπου *X* είναι το επίπεδο (1‑9) και τροποποιήστε τη γραμματοσειρά, το χρώμα ή τις ρυθμίσεις παραγράφου.

**Ε: Πώς μπορώ να προσθέσω περισσότερα επίπεδα στον TOC μου;**  
Α: Τροποποιήστε το switch του `FieldToc` `\o "1-3"` (για παράδειγμα) ώστε να συμπεριλάβει επιπλέον επίπεδα επικεφαλίδας, και έπειτα ενημερώστε τα αντίστοιχα στυλ `TOC_X`.

**Ε: Μπορώ να αλλάξω τις θέσεις των διαστημάτων για συγκεκριμένες καταχωρήσεις του TOC;**  
Α: Ναι – περάστε από τις παραγράφους όπως φαίνεται στην ενότητα “Προσαρμογή Διαστημάτων” και τροποποιήστε κάθε διάστημα ξεχωριστά.

**Ε: Είναι δυνατόν να δημιουργηθεί TOC σε έξοδο PDF;**  
Α: Απόλυτα. Αποθηκεύστε το έγγραφο ως PDF (`doc.save("output.pdf")`) μετά τη δημιουργία του TOC· το πεδίο αποδίδεται αυτόματα.

**Ε: Πρέπει να καλέσω το `updateFields()` χειροκίνητα;**  
Α: Όταν εισάγετε ένα `FieldToc`, το Aspose.Words το ενημερώνει κατά την αποθήκευση, αλλά η κλήση του `doc.updateFields()` παρέχει άμεσα αποτελέσματα για εντοπισμό σφαλμάτων.

Αυτές οι τεχνικές σας επιτρέπουν να δημιουργείτε καθαρά, πλοηγήσιμα και επαγγελματικά μορφοποιημένα έγγραφα που πληρούν οποιοδήποτε πρότυπο δημοσίευσης.
