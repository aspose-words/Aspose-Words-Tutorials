# Δημιουργία Πίνακα Περιεχομένων στο Aspose.Words για Java


## Εισαγωγή στη δημιουργία πίνακα περιεχομένων στο Aspose.Words για Java

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία δημιουργίας ενός Πίνακα Περιεχομένων (TOC) χρησιμοποιώντας το Aspose.Words για Java. Το TOC είναι μια κρίσιμη λειτουργία για τη δημιουργία οργανωμένων εγγράφων. Θα καλύψουμε τον τρόπο προσαρμογής της εμφάνισης και της διάταξης του TOC.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε εγκαταστήσει και ρυθμίσει το Aspose.Words για Java στο έργο Java σας.

## Βήμα 1: Δημιουργία νέου εγγράφου

Αρχικά, ας δημιουργήσουμε ένα νέο έγγραφο με το οποίο θα εργαστούμε.

```java
Document doc = new Document();
```

## Βήμα 2: Προσαρμογή στυλ περιεχομένων

Για να προσαρμόσετε την εμφάνιση του πίνακα περιεχομένων σας, μπορείτε να τροποποιήσετε τα στυλ που σχετίζονται με αυτόν. Σε αυτό το παράδειγμα, θα κάνουμε τις καταχωρήσεις πίνακα περιεχομένων πρώτου επιπέδου με έντονη γραφή.

```java
doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_1).getFont().setBold(true);
```

## Βήμα 3: Προσθήκη περιεχομένου στο έγγραφό σας

Μπορείτε να προσθέσετε το περιεχόμενό σας στο έγγραφο. Αυτό το περιεχόμενο θα χρησιμοποιηθεί για τη δημιουργία του Πίνακα Περιεχομένων.

## Βήμα 4: Δημιουργήστε τον Πίνακα Περιεχομένων

Για να δημιουργήσετε τον Πίνακα Περιεχομένων, εισαγάγετε ένα πεδίο Πίνακα Περιεχομένων στην επιθυμητή θέση στο έγγραφό σας. Αυτό το πεδίο θα συμπληρωθεί αυτόματα με βάση τις επικεφαλίδες και τα στυλ στο έγγραφό σας.

```java
// Εισαγάγετε ένα πεδίο TOC στην επιθυμητή θέση στο έγγραφό σας.
FieldToc fieldToc = new FieldToc();
doc.getFirstSection().getBody().getFirstParagraph().appendChild(fieldToc);
```

## Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφο με τον Πίνακα Περιεχομένων.

```java
doc.save("your_output_path_here");
```

## Προσαρμογή Tab Stop στο TOC

Μπορείτε επίσης να προσαρμόσετε τους στηλοθέτες στον Πίνακα Περιεχομένων σας για να ελέγχετε τη διάταξη των αριθμών σελίδων. Δείτε πώς μπορείτε να αλλάξετε τους στηλοθέτες:

```java
Document doc = new Document("Table of contents.docx");

for (Paragraph para : (Iterable<Paragraph>) doc.getChildNodes(NodeType.PARAGRAPH, true))
{
    if (para.getParagraphFormat().getStyle().getStyleIdentifier() >= StyleIdentifier.TOC_1 &&
        para.getParagraphFormat().getStyle().getStyleIdentifier() <= StyleIdentifier.TOC_9)
    {
        // Χρησιμοποιήστε την πρώτη καρτέλα που χρησιμοποιείται σε αυτήν την παράγραφο, η οποία ευθυγραμμίζει τους αριθμούς σελίδων.
        TabStop tab = para.getParagraphFormat().getTabStops().get(0);
        
        // Αφαιρέστε την παλιά καρτέλα.
        para.getParagraphFormat().getTabStops().removeByPosition(tab.getPosition());
        
        // Εισαγάγετε μια νέα καρτέλα σε μια τροποποιημένη θέση (π.χ., 50 μονάδες προς τα αριστερά).
        para.getParagraphFormat().getTabStops().add(tab.getPosition() - 50.0, tab.getAlignment(), tab.getLeader());
    }
}

doc.save("output.docx");
```

Τώρα έχετε έναν προσαρμοσμένο Πίνακα Περιεχομένων στο έγγραφό σας με προσαρμοσμένα στηλοθέτες για ευθυγράμμιση του αριθμού σελίδας.


## Σύναψη

Σε αυτό το σεμινάριο, εξερευνήσαμε πώς να δημιουργήσετε έναν Πίνακα Περιεχομένων (TOC) χρησιμοποιώντας το Aspose.Words για Java, μια ισχυρή βιβλιοθήκη για εργασία με έγγραφα του Word. Ένας καλά δομημένος Πίνακας Περιεχομένων είναι απαραίτητος για την οργάνωση και την πλοήγηση σε μεγάλα έγγραφα, και το Aspose.Words παρέχει τα εργαλεία για την εύκολη δημιουργία και προσαρμογή των Πίνακα Περιεχομένων.

## Συχνές ερωτήσεις

### Πώς μπορώ να αλλάξω τη μορφοποίηση των καταχωρίσεων TOC;

Μπορείτε να τροποποιήσετε τα στυλ που σχετίζονται με τα επίπεδα TOC χρησιμοποιώντας `doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_X)`, όπου X είναι το επίπεδο TOC.

### Πώς μπορώ να προσθέσω περισσότερα επίπεδα στον Πίνακα Περιεχομένων μου;

Για να συμπεριλάβετε περισσότερα επίπεδα στον Πίνακα Περιεχομένων σας, μπορείτε να τροποποιήσετε το πεδίο Πίνακα Περιεχομένων και να καθορίσετε τον επιθυμητό αριθμό επιπέδων.

### Μπορώ να αλλάξω τις θέσεις των στηλοθετών για συγκεκριμένες καταχωρήσεις TOC;

Ναι, όπως φαίνεται στο παραπάνω παράδειγμα κώδικα, μπορείτε να αλλάξετε τις θέσεις των στηλοθετών για συγκεκριμένες καταχωρήσεις πίνακα περιεχομένων (TOC) επαναλαμβάνοντας τις παραγράφους και τροποποιώντας τους στηλοθέτες ανάλογα.
