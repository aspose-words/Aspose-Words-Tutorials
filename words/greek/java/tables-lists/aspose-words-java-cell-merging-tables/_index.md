## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία συγχώνευσης κελιών πίνακα σε διαχειρίσιμα βήματα, εστιάζοντας τόσο στις κάθετες όσο και στις οριζόντιες συγχωνεύσεις.

### Κάθετη συγχώνευση κελιών

#### Επισκόπηση:
Η κάθετη συγχώνευση κελιών συνδυάζει πολλές γραμμές μέσα σε μία μόνο στήλη, ιδανική για τη δημιουργία κεφαλίδων ή την ομαδοποίηση σχετικών πληροφοριών.

#### Βήμα προς βήμα εφαρμογή:
**1. Δημιουργία εγγράφου και εργαλείου δημιουργίας:**
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

**2. Εισαγωγή κελιών με κάθετη συγχώνευση:**

- **Πρώτο κελί (Έναρξη συγχώνευσης):** Ορισμός ως αρχή μιας κάθετης συγχώνευσης.
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.FIRST); // Σημειώνει αυτό το κελί ως το σημείο εκκίνησης για τη συγχώνευση.
  builder.write("Text in merged cells.");
  ```

- **Δεύτερο κελί (μη συγχωνευμένο):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.NONE); // Δεν εφαρμόστηκε συγχώνευση εδώ.
  builder.write("Text in unmerged cell.");
  builder.endRow(); // Τερματίζει την τρέχουσα σειρά.
  ```

- **Τρίτο κελί (Συνέχεια συγχώνευσης):** Συγχωνεύεται με το πρώτο κελί κάθετα.
  ```java
  builder.insertCell();
  builder.getCellFormat().setVerticalMerge(CellMerge.PREVIOUS); // Συνεχίζει την κάθετη συγχώνευση από το προηγούμενο κελί.
  builder.endRow(); // Ολοκληρώστε τη δεύτερη σειρά.
  ```

**3. Αποθηκεύστε το έγγραφο:**
```java
doc.save("VerticalMergeOutput.docx");
```

### Οριζόντια συγχώνευση κελιών

#### Επισκόπηση:
Η οριζόντια συγχώνευση συνδυάζει κελιά σε μία μόνο γραμμή, ιδανική για τη δημιουργία ολοκληρωμένων κεφαλίδων ή πληροφοριών που εκτείνονται σε όλο το εύρος τους.

#### Βήμα προς βήμα εφαρμογή:
**1. Δημιουργία εγγράφου και εργαλείου δημιουργίας:**
Επαναχρησιμοποιήστε τον ίδιο κώδικα αρχικοποίησης όπως πριν.

**2. Εισαγωγή κελιών με οριζόντια συγχώνευση:**

- **Πρώτο κελί (Έναρξη συγχώνευσης):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.FIRST); // Ξεκινά η οριζόντια συγχώνευση.
  builder.write("Text in merged cells.");
  ```

- **Δεύτερο κελί (Συνέχεια συγχώνευσης):**
  ```java
  builder.insertCell();
  builder.getCellFormat().setHorizontalMerge(CellMerge.PREVIOUS); // Συνεχίζει από το πρώτο κελί οριζόντια.
  builder.endRow(); // Τερματίζει την τρέχουσα γραμμή, ολοκληρώνοντας την οριζόντια συγχώνευση.
  ```

**3. Αποθηκεύστε το έγγραφο:**
```java
doc.save("HorizontalMergeOutput.docx");
```

### Επένδυση κελιών

#### Επισκόπηση:
Η προσθήκη συμπλήρωσης στα κελιά βελτιώνει την αναγνωσιμότητα δημιουργώντας κενά μεταξύ κειμένου και περιγραμμάτων.

#### Βήμα προς βήμα εφαρμογή:
**1. Ορισμός συμπλήρωσης σε κελιά:**
```java
builder.getCellFormat().setPaddings(5.0, 10.0, 40.0, 50.0); // Επάνω, δεξιά, κάτω, αριστερά γεμίσματα σε σημεία.
```

**2. Εισαγωγή κελιού με συμπλήρωση:**
```java
builder.startTable();
builder.insertCell();
builder.write("Lorem ipsum dolor sit amet...");
builder.endRow();
builder.endTable();
doc.save("PaddingOutput.docx");
```

## Πρακτικές Εφαρμογές
Η κατανόηση του τρόπου συγχώνευσης κελιών και της προσθήκης συμπλήρωσης μπορεί να βελτιώσει τα έγγραφα με διάφορους τρόπους:
1. **Δημιουργία Τιμολογίου**Χρησιμοποιήστε κάθετες συγχωνεύσεις για περιγραφές στοιχείων που εκτείνονται σε πολλές γραμμές, βελτιώνοντας τη σαφήνεια.
2. **Δημιουργία Αναφοράς**Οι οριζόντιες συγχωνεύσεις είναι ιδανικές για ενοποιημένες κεφαλίδες ενοτήτων σε όλους τους πίνακες.
3. **Πρότυπα βιογραφικού σημειώματος**Προσθέστε συμπλήρωση για να διασφαλίσετε ότι το κείμενο μέσα στις ενότητες του βιογραφικού σημειώματος είναι ευχάριστο για τα μάτια.
