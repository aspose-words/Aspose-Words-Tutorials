### Επίλυση συγκρούσεων αρίθμησης λιστών

#### Επισκόπηση
Κατά τη συγχώνευση εγγράφων με πανομοιότυπες προσαρμοσμένες μορφές λίστας, ενδέχεται να προκύψουν συγκρούσεις αριθμών. Αυτή η λειτουργία σάς επιτρέπει να επιλέξετε εάν θα διατηρήσετε την αρχική αρίθμηση ή θα τα συγχωνεύσετε σε μια συνεχή ακολουθία.

#### Βήμα προς βήμα εφαρμογή

1. **Ρυθμίστε τα έγγραφά σας**
   Κλωνοποιήστε το έγγραφο προέλευσης για χειραγώγηση.
   ```java
   Document srcDoc = new Document("Custom list numbering.docx");
   Document dstDoc = srcDoc.deepClone();
   ```

2. **Ρύθμιση παραμέτρων επιλογών εισαγωγής**
   Χρήση `ImportFormatOptions` για να διαχειριστείτε τον τρόπο συνδυασμού των εγγράφων.
   ```java
   ImportFormatOptions importFormatOptions = new ImportFormatOptions();
   importFormatOptions.setKeepSourceNumbering(true); // ή false για συγχώνευση αρίθμησης
   ```

3. **Ρύθμιση Εισαγωγέα Κόμβου**
   Χρησιμοποιώ `NodeImporter` για τη διαχείριση λειτουργιών σε επίπεδο κόμβου κατά την εισαγωγή του εγγράφου.
   ```java
   NodeImporter importer = new NodeImporter(srcDoc, dstDoc, ImportFormatMode.KEEP_DIFFERENT_STYLES, importFormatOptions);
   ```

4. **Εισαγωγή και Προσθήκη Κόμβων**
   Επαναλάβετε τις παραγράφους στο έγγραφο προέλευσης και προσαρτήστε τις στον προορισμό.
   ```java
   for (Paragraph paragraph : srcDoc.getFirstSection().getBody().getParagraphs()) {
       Node importedNode = importer.importNode(paragraph, true);
       dstDoc.getFirstSection().getBody().appendChild(importedNode);
   }
   ```

5. **Ενημέρωση ετικετών λίστας**
   Βεβαιωθείτε ότι οι ετικέτες λίστας του εγγράφου είναι ενημερωμένες ώστε να αντικατοπτρίζουν την επιλεγμένη στρατηγική αρίθμησης.
   ```java
   dstDoc.updateListLabels();
   ```
