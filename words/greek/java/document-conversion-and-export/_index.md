- **Ποιες μορφές μπορώ να μετατρέψω;** DOCX, PDF, HTML, ODT και πολλές άλλες.

## Συχνές Ερωτήσεις

**Q: Πώς μπορώ να προσθέσω κείμενο ως υδατογράφημα σε PDF που δημιουργείται από αρχείο DOCX;**  
A: Χρησιμοποιήστε `DocumentBuilder.insertWatermarkText("Your Text")` πριν καλέσετε `document.save("output.pdf")`. Αυτό λειτουργεί κατά το βήμα **convert docx to pdf**.

**Q: Είναι δυνατόν να προσθέσω υδατογράφημα μόνο σε ορισμένες σελίδες;**  
A: Ναι. Εισάγετε το υδατογράφημα μέσα σε έναν βρόχο που ελέγχει τις ιδιότητες `PageSetup` για τις σελίδες που θέλετε να στοχεύσετε.

**Q: Μπορώ να συνδυάσω τη δημιουργία barcode με την υδατογράφημα;**  
A: Απόλυτα. Δημιουργήστε την εικόνα barcode, στη συνέχεια εισάγετε την ως υδατογράφημα εικόνας χρησιμοποιώντας `DocumentBuilder.insertWatermarkImage(barcodeImage)`.

**Q: Ποιες μορφές υποστηρίζουν υδατογραφήματα;**  
A: Όλες οι μορφές εξαγωγής που υποστηρίζονται από το Aspose.Words (PDF, DOCX, HTML, ODT κ.λπ.) διατηρούν τα υδατογραφήματα.

**Q: Χρειάζομαι ξεχωριστή άδεια για τις λειτουργίες barcode ή chart;**  
A: Όχι. Όλες οι λειτουργίες περιλαμβάνονται στην τυπική άδεια του Aspose.Words for Java.
