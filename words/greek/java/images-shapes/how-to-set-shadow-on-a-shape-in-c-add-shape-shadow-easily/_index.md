# Πώς να ορίσετε σκιά σε σχήμα σε C# – Προσθέστε εύκολα σκιά σε σχήμα

Έχετε αναρωτηθεί ποτέ **πώς να ορίσετε σκιά** σε ένα σχήμα χωρίς να σκάβετε μέσα σε ατελείωτες τεκμηριώσεις API; Δεν είστε μόνοι. Πολλοί προγραμματιστές συναντούν πρόβλημα όταν χρειάζονται μια διακριτική σκιά πτώσης για να κάνει ένα διάγραμμα να ξεχωρίσει, αλλά δεν μπορούν να βρουν ένα καθαρό παράδειγμα που δείχνει *και* το “τι” και το “γιατί”.  

Σε αυτό το tutorial θα περάσουμε βήμα-βήμα από την προσθήκη σκιάς σε σχήμα, την αλλαγή του χρώματος της σκιάς και τη λεπτομερή ρύθμιση του θολώματος, της μετατόπισης και της διαφάνειας — όλα χρησιμοποιώντας το Aspose.Words for .NET. Στο τέλος θα έχετε ένα έτοιμο κομμάτι κώδικα που μπορείτε να ενσωματώσετε σε οποιοδήποτε έργο C#, μαζί με μια σειρά συμβουλών για την προσαρμογή της σκιάς σχήματος σε πιο σύνθετα σενάρια.

> **Σημείωση:** Ο κώδικας λειτουργεί με Aspose.Words 22.9 ή νεότερη έκδοση και απαιτεί .NET 6+ (ή .NET Framework 4.7.2+).

## Βήμα‑βήμα: Πώς να ορίσετε σκιά σε σχήμα

### 1. Φορτώστε το έγγραφο Word

Ξεκινάμε ανοίγοντας το αρχείο `.docx`. Ο κατασκευαστής `Document` διαβάζει το αρχείο στη μνήμη, δίνοντάς μας πλήρη πρόσβαση στους κόμβους του.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;

// Load the source document
Document doc = new Document("YOUR_DIRECTORY/input.docx");
```

> **Γιατί;** Η φόρτωση του εγγράφου είναι το θεμέλιο — χωρίς αυτή δεν μπορείτε να διασχίσετε το δέντρο των σχημάτων.

### 2. Ανακτήστε το πρώτο σχήμα (ή οποιοδήποτε σχήμα χρειάζεστε)

Το Aspose.Words αποθηκεύει τα σχήματα ως κόμβους τύπου `NodeType.SHAPE`. Η μέθοδος `GetChild` μας επιτρέπει να πάρουμε το *n‑th* σχήμα· εδώ παίρνουμε το δείκτη 0, δηλαδή το πρώτο σχήμα.

```csharp
// Grab the first shape in the document (depth‑first search)
Shape firstShape = (Shape)doc.GetChild(NodeType.Shape, 0, true);
if (firstShape == null)
{
    throw new InvalidOperationException("No shape found in the document.");
}
```

> **Pro tip:** Αν χρειάζεται να **προσθέσετε σκιά σε σχήμα** σε ένα συγκεκριμένο σχήμα, αντικαταστήστε το δείκτη με την κατάλληλη τιμή ή επαναλάβετε μέσω `doc.GetChildNodes(NodeType.Shape, true)`.

### 3. Πρόσβαση στο αντικείμενο μορφοποίησης σκιάς

Κάθε `Shape` διαθέτει την ιδιότητα `ShadowFormat` που εκθέτει όλες τις ρυθμίσεις που αφορούν τη σκιά.

```csharp
ShadowFormat shadow = firstShape.ShadowFormat;
```

Τώρα μπορούμε να αρχίσουμε να ρυθμίζουμε τη σκιά.

### 4. Ορίστε την ακτίνα θολώματος – μαλακότερες άκρες

Μια μεγαλύτερη ακτίνα θολώματος κάνει τη σκιά να φαίνεται πιο διαχυμένη. Η τιμή είναι σε points (1 pt ≈ 1/72 inch).

```csharp
shadow.BlurRadius = 5.0; // 5 pt blur – looks nicely soft
```

> **Πότε να προσαρμόσετε;** Αν το σχήμα σας είναι μικρό, ένα θόλωμα 2–3 pt μπορεί να είναι αρκετό· για μεγάλα banner, αυξήστε το σε 8–10 pt.

### 5. Ορίστε τις οριζόντιες και κάθετες μετατοπίσεις

Οι μετατοπίσεις ελέγχουν πόσο μακριά θα μετακινηθεί η σκιά από το σχήμα. Θετικές τιμές μετακινούν τη σκιά δεξιά/κάτω· αρνητικές τιμές τη μετακινούν αριστερά/πάνω.

```csharp
shadow.DistanceX = 3.0; // 3 pt to the right
shadow.DistanceY = 3.0; // 3 pt downwards
```

### 6. Ρυθμίστε τη διαφάνεια (αδιαφάνεια)

Η `Transparency` κυμαίνεται από `0.0` (πλήρως αδιαφανής) έως `1.0` (εντελώς αόρατη). Μια τιμή γύρω στο `0.3` δίνει ένα διακριτικό, ημιδιαφανές αποτέλεσμα.

```csharp
shadow.Transparency = 0.3; // 30 % transparent
```

### 7. Επιλέξτε χρώμα σκιάς – **ορίστε χρώμα σκιάς** σε οποιοδήποτε `System.Drawing.Color`

Μπορείτε να επιλέξετε οποιοδήποτε προεπιλεγμένο χρώμα ή να δημιουργήσετε ένα προσαρμοσμένο με τιμές RGB.

```csharp
shadow.Color = Color.FromArgb(0, 120, 215); // A calm blue shade
```

Αν προτιμάτε την κλασική μαύρη σκιά, απλώς χρησιμοποιήστε `Color.Black`.

### 8. Αποθηκεύστε το τροποποιημένο έγγραφο

Τέλος, αποθηκεύστε τις αλλαγές. Μπορείτε να αντικαταστήσετε το αρχικό αρχείο ή να γράψετε σε νέα τοποθεσία.

```csharp
doc.Save("YOUR_DIRECTORY/output_with_shadow.docx");
```

## Πλήρες Παράδειγμα Εργασίας (Όλα τα Βήματα σε Ένα Block)

Αντιγράψτε‑και‑επικολλήστε το παρακάτω στη μέθοδο `Main` μιας εφαρμογής console. Συγκεντρώνεται ακριβώς, εφόσον το πακέτο NuGet είναι εγκατεστημένο.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;

class Program
{
    static void Main()
    {
        // 1. Load the Word document
        Document doc = new Document("YOUR_DIRECTORY/input.docx");

        // 2. Retrieve the first shape (add shape shadow)
        Shape shape = (Shape)doc.GetChild(NodeType.Shape, 0, true);
        if (shape == null)
        {
            System.Console.WriteLine("No shape found – aborting.");
            return;
        }

        // 3. Get the shadow formatting object
        ShadowFormat shadow = shape.ShadowFormat;

        // 4. Set blur radius
        shadow.BlurRadius = 5.0;

        // 5. Define offsets
        shadow.DistanceX = 3.0;
        shadow.DistanceY = 3.0;

        // 6. Adjust transparency (0 = opaque, 1 = fully transparent)
        shadow.Transparency = 0.3;

        // 7. Set shadow color (set shadow color)
        shadow.Color = Color.GetBlue(); // or any custom color

        // 8. Save the result
        doc.Save("YOUR_DIRECTORY/output_with_shadow.docx");

        System.Console.WriteLine("Shadow applied successfully!");
    }
}
```

**Αναμενόμενο αποτέλεσμα:** Ανοίξτε το `output_with_shadow.docx` στο Word· το πρώτο σχήμα εμφανίζει τώρα μια απαλή μπλε σκιά, μετατοπισμένη κατά 3 pt, με διακριτικό θόλωμα και 30 % διαφάνεια.

## Συνηθισμένες Παραλλαγές & Ακραίες Περιπτώσεις

### Προσθήκη σκιών σε *όλα* τα σχήματα

Αν το έγγραφό σας περιέχει πολλά διαγράμματα, ίσως θέλετε να κάνετε βρόχο σε κάθε σχήμα:

```csharp
NodeCollection shapes = doc.GetChildNodes(NodeType.Shape, true);
foreach (Shape s in shapes)
{
    ShadowFormat sf = s.ShadowFormat;
    sf.BlurRadius = 4.0;
    sf.DistanceX = 2.0;
    sf.DistanceY = 2.0;
    sf.Transparency = 0.25;
    sf.Color = Color.Gray;
}
```

### Επαναφορά μιας σκιάς

Μερικές φορές ένα σχήμα έχει ήδη σκιά που πρέπει να αφαιρεθεί. Ορίστε `ShadowFormat.Visible` σε `false`:

```csharp
shape.ShadowFormat.Visible = false;
```

### Χρήση προσαρμοσμένου χρώματος με άλφα (ημιδιαφανές)

```csharp
shadow.Color = Color.FromArgb(128, 255, 0, 0); // 50 % transparent red
```

### Σημείωση συμβατότητας

Το API `ShadowFormat` είναι σταθερό σε όλες τις εκδόσεις του Aspose.Words, αλλά παλαιότερες εκδόσεις (< 19.1) χρησιμοποιούσαν πεδία `ShadowFormat` με ελαφρώς διαφορετικές ονομασίες. Πάντα στοχεύετε το πιο πρόσφατο πακέτο NuGet για τα καλύτερα αποτελέσματα.

## Pro Tips για μια Πολυτελή Σκιά

- **Ισορροπήστε θόλωμα και μετατόπιση:** Ένα έντονο θόλωμα με μικρή μετατόπιση μπορεί να φαίνεται “λαμπερό” αντί για πραγματική σκιά πτώσης. Πειραματιστείτε με `BlurRadius` × `DistanceX/Y`.
- **Ταιριάξτε το θέμα του εγγράφου:** Αν το αρχείο Word χρησιμοποιεί σκούρο θέμα, μια ανοιχτή σκιά (`Color.White`) μπορεί να δημιουργήσει ένα διακριτικό εφέ ανύψωσης.
- **Απόδοση:** Η αλλαγή σκιών σε εκατοντάδες σχήματα μπορεί να προσθέσει μερικά χιλιοστά του δευτερολέπτου ανά σχήμα. Ομαδοποιήστε τη λειτουργία αν επεξεργάζεστε μεγάλες αναφορές.
- **Δοκιμή:** Ανοίξτε το παραγόμενο `.docx` τόσο στο Word Desktop όσο και στο Word Online για να διασφαλίσετε ότι η σκιά αποδίδεται σταθερά.
