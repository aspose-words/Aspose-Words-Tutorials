# Δημιουργία σχήματος ορθογωνίου στο Word με Java – Πλήρης Οδηγός

Κάποτε χρειάστηκε να **δημιουργήσετε σχήμα ορθογωνίου** σε ένα αρχείο Word αλλά δεν ήξερατε από πού να ξεκινήσετε; Δεν είστε οι μόνοι—πολλοί προγραμματιστές αντιμετωπίζουν αυτό το πρόβλημα όταν αυτοματοποιούν αναφορές ή τιμολόγια. Το καλό νέο; Με το Aspose.Words for Java μπορείτε να δημιουργήσετε ένα ορθογώνιο, να του προσθέσετε μια ωραία σκιά και να αποθηκεύσετε το έγγραφο Word με λίγες γραμμές κώδικα.

Σε αυτό το tutorial θα περάσουμε από όλα όσα χρειάζεστε: από την αρχικοποίηση ενός κεντρικού εγγράφου, στη διαμόρφωση μιας σκιάς, μέχρι την τελική αποθήκευση του αρχείου. Στο τέλος θα ξέρετε **πώς να προσθέσετε σκιά σε σχήμα**, πώς να **προσθέσετε σκιά σε σχήμα**, και πώς να **προσθέσετε σχήμα ορθογωνίου** σε οποιοδήποτε έγγραφο Word δημιουργείτε. Δεν απαιτούνται εξωτερικά έγγραφα—απλός, εκτελέσιμος κώδικας.

## Προαπαιτήσεις

- Java 8 ή νεότερη (το API λειτουργεί επίσης με Java 11+).  
- Βιβλιοθήκη Aspose.Words for Java (έκδοση 23.9 ή νεότερη).  
- Ένα IDE όπως IntelliJ IDEA ή Eclipse—οποιοδήποτε είναι αποδεκτό.  
- Βασική εξοικείωση με τη σύνταξη της Java.

> **Pro tip:** Αν χρησιμοποιείτε Maven, προσθέστε την εξάρτηση Aspose.Words στο `pom.xml` και αφήστε το IDE να διαχειριστεί τα υπόλοιπα.

## Βήμα 1: Αρχικοποίηση Νέου Εγγράφου – Πώς να **δημιουργήσετε σχήμα ορθογωνίου**  

Πρώτα απ' όλα: χρειάζεστε έναν καθαρό καμβά. Στο Aspose.Words αυτός ο καμβάς είναι ένα αντικείμενο `Document`.

```java
import com.aspose.words.*;

public class ShadowShapeExample {
    public static void main(String[] args) throws Exception {
        // Step 1: Create a new blank document
        Document document = new Document();
```

Η κλάση `Document` αντιπροσωπεύει ολόκληρο το αρχείο .docx. Σκεφτείτε το ως το σημειωματάριο όπου θα **προσθέσετε σχήμα ορθογωνίου** και τη σκιά του αργότερα.

## Βήμα 2: Δημιουργία του Ορθογωνίου – **Προσθήκη σχήματος ορθογωνίου**  

Τώρα κατασκευάζουμε το ορθογώνιο. Θα ορίσουμε το μέγεθός του, τη διάταξη και το χρώμα γεμίσματος.

```java
        // Step 2: Create a rectangle shape and set its size and layout
        Shape rectangleShape = new Shape(document, ShapeType.RECTANGLE);
        rectangleShape.setWidth(200);
        rectangleShape.setHeight(100);
        rectangleShape.setWrapType(WrapType.INLINE);
        rectangleShape.setFillColor(java.awt.Color.LIGHT_GRAY);
```

Γιατί `INLINE` wrap; Επειδή θέλουμε το σχήμα να συμπεριφέρεται όπως μια παράγραφος—ιδανικό για απλές αναφορές. Μπορείτε να το αλλάξετε σε `TOPBOTTOM` αν χρειαστεί το κείμενο να ρέει γύρω από το σχήμα αργότερα.

## Βήμα 3: Εφαρμογή Σκιάς – **Πώς να προσθέσετε σκιά σε σχήμα**  

Ένα επίπεδο ορθογώνιο φαίνεται λίγο βαρετό. Η προσθήκη σκιάς του δίνει βάθος και κάνει το έγγραφο πιο επαγγελματικό. Εδώ απαντάμε στην ερώτηση “**πώς να προσθέσετε σκιά σε σχήμα**” στην πράξη.

```java
        // Step 3: Configure the shape's shadow appearance
        rectangleShape.getShadowFormat().setVisible(true);
        rectangleShape.getShadowFormat().setColor(java.awt.Color.DARK_GRAY);
        rectangleShape.getShadowFormat().setBlurRadius(5.0);
        rectangleShape.getShadowFormat().setOffsetX(4.0);
        rectangleShape.getShadowFormat().setOffsetY(4.0);
        rectangleShape.getShadowFormat().setTransparency(0.3);
```

Κάθε ιδιότητα κάνει κάτι συγκεκριμένο:

- `setVisible(true)` ενεργοποιεί τη σκιά.  
- `setColor` επιλέγει ένα σκούρο γκρι για ήπιο αποτέλεσμα.  
- `setBlurRadius` ελέγχει πόσο μαλακές είναι οι άκρες.  
- `setOffsetX/Y` μετακινεί τη σκιά δεξιά και κάτω, μιμούμενο μια πηγή φωτός.  
- `setTransparency` την κάνει ελαφρώς διαφανή, ώστε το σχήμα να παραμένει στο επίκεντρο.

> **Σημείωση:** Αν χρειαστείτε χρωματιστή σκιά, απλώς περάστε ένα διαφορετικό `java.awt.Color` στο `setColor`.

## Βήμα 4: Εισαγωγή του Σχήματος στο Έγγραφο  

Με το ορθογώνιο και τη σκιά του έτοιμα, το τοποθετούμε στην πρώτη ενότητα του εγγράφου.

```java
        // Step 4: Add the shape to the first section of the document
        document.getFirstSection().getBody().appendChild(rectangleShape);
```

Η προσθήκη στο σώμα τοποθετεί το σχήμα εκεί που θα βρισκόταν μια νέα παράγραφος. Αν θέλετε το ορθογώνιο σε συγκεκριμένη θέση, μπορείτε να χρησιμοποιήσετε `insertBefore` ή να χειριστείτε τη συλλογή `Paragraph`.

## Βήμα 5: **Αποθήκευση εγγράφου Word** – Διατηρήστε τη δουλειά σας  

Το τελευταίο βήμα είναι η εγγραφή του αρχείου στο δίσκο. Αυτή είναι η στιγμή που πραγματικά **αποθηκεύετε το έγγραφο Word**.

```java
        // Step 5: Save the document with the shadowed shape
        document.save("YOUR_DIRECTORY/ShadowShape.docx");
    }
}
```

Αντικαταστήστε το `YOUR_DIRECTORY` με απόλυτη ή σχετική διαδρομή στον υπολογιστή σας. Μετά την εκτέλεση του προγράμματος, ανοίξτε το `ShadowShape.docx` στο Microsoft Word—θα πρέπει να δείτε ένα ανοιχτό‑γκρι ορθογώνιο με μια ήπια σκούρα σκιά.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις  

### Τι γίνεται αν χρειαστώ πολλά ορθογώνια;  

Απλώς επαναλάβετε τα **Βήμα 2** και **Βήμα 3** μέσα σε βρόχο, προσαρμόζοντας `setWidth`, `setHeight` ή `setFillColor` σε κάθε επανάληψη. Θυμηθείτε να δώσετε σε κάθε σχήμα μοναδικό όνομα μεταβλητής ή να τα αποθηκεύσετε σε λίστα.

### Μπορώ να εξάγω σε PDF αντί για DOCX;  

Απολύτως. Μετά την προσθήκη του σχήματος, καλέστε `document.save("output.pdf")`. Το Aspose.Words θα αναλάβει τη μετατροπή, διατηρώντας τη σκιά.

### Τι γίνεται με παλαιότερες εκδόσεις του Word;  

Χρησιμοποιήστε την υπερφόρτωση `document.save("file.doc", SaveFormat.DOC)`. Το API θα προσαρμόσει αυτόματα τις δυνατότητες, αλλά σημειώστε ότι ορισμένα στυλ σκιάς μπορεί να φαίνονται ελαφρώς διαφορετικά σε παλαιότερες μορφές.

### Πώς αλλάζω την κατεύθυνση της σκιάς;  

Τροποποιήστε τις `setOffsetX` και `setOffsetY`. Θετικό X μετακινεί τη σκιά δεξιά, αρνητικό αριστερά. Θετικό Y μετακινεί κάτω, αρνητικό πάνω. Παίξτε με αυτές τις τιμές για να προσομοιώσετε πηγή φωτός από οποιαδήποτε γωνία.

## Συμβουλές για Εργασία με Σχήματα  

- **Ομαδοποίηση σχημάτων**: Αν χρειάζεστε ετικέτα δίπλα στο ορθογώνιο, δημιουργήστε ένα `GroupShape` και προσθέστε τόσο το ορθογώνιο όσο και ένα `TextBox`.  
- **Η σειρά Z‑order έχει σημασία**: Χρησιμοποιήστε `shape.moveToFront()` ή `shape.moveToBack()` για να ελέγξετε ποιο σχήμα εμφανίζεται από πάνω.  
- **Απόδοση**: Η προσθήκη εκατοντάδων σχημάτων μπορεί να είναι αργή. Ομαδοποιήστε τα σε μία ενότητα και καλέστε `document.updatePageLayout()` μία φορά στο τέλος.
