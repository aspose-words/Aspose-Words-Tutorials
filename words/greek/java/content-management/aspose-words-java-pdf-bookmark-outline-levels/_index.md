# Διαχείριση Επιπέδων Περιγράμματος Σελιδοδεικτών σε PDF με Aspose.Words για Java

## Δημιουργία Ένθετων Σελιδοδεικτών
**Επισκόπηση**: Μάθετε πώς να δημιουργείτε ένθετους σελιδοδείκτες μέσα σε ένα έγγραφο Word χρησιμοποιώντας το Aspose.Words για Java.

### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί το περιβάλλον του εγγράφου Word όπου μπορείτε να αρχίσετε να εισάγετε περιεχόμενο.

### Βήμα 2: Εισαγωγή Ένθετων Σελιδοδεικτών
Ξεκινήστε δημιουργώντας έναν κύριο σελιδοδείκτη:
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

Τώρα, ενσωματώστε έναν ακόμη σελιδοδείκτη μέσα σε αυτόν:
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

Ολοκληρώστε τον εξωτερικό σελιδοδείκτη:
```java
builder.endBookmark("Bookmark 1");
```

### Βήμα 3: Προσθήκη Πρόσθετων Σελιδοδεικτών
Συνεχίστε να προσθέτετε σελιδοδείκτες όπως χρειάζεται. Για παράδειγμα, έναν ξεχωριστό τρίτο σελιδοδείκτη:
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

## Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
**Επισκόπηση**: Οργανώστε τους σελιδοδείκτες σας ορίζοντας τα επίπεδα περιγράμματος για καλύτερη πλοήγηση στο PDF.

### Βήμα 1: Ρύθμιση PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
Αυτό το απόσπασμα κώδικα αρχικοποιεί τις επιλογές που θα χρησιμοποιήσετε για την αποθήκευση του εγγράφου ως PDF με οργανωμένους σελιδοδείκτες.

### Βήμα 2: Προσθήκη Επιπέδων Περιγράμματος
Αναθέστε επίπεδα σε κάθε σελιδοδείκτη, τα οποία θα καθορίσουν την ιεραρχία τους:
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

### Βήμα 3: Αποθήκευση Εγγράφου
Τέλος, αποθηκεύστε το έγγραφό σας ως PDF με αυτές τις ρυθμίσεις:
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

## Συμβουλές Επίλυσης Προβλημάτων
- **Ελλιπείς Σελιδοδείκτες**: Βεβαιωθείτε ότι κάθε `startBookmark` έχει το αντίστοιχο `endBookmark`.  
- **Λανθασμένα Επίπεδα**: Επαληθεύστε την ιεραρχία ελέγχοντας τις σχέσεις γονέα‑παιδίου στον κώδικά σας.  
- **pdf bookmark troubleshooting**: Εάν οι σελιδοδείκτες δεν εμφανίζονται στο παραγόμενο PDF, ελέγξτε ξανά ότι το `PdfSaveOptions` περνιέται σωστά στη μέθοδο `doc.save()`.
