## Οδηγός Υλοποίησης
Θα περάσουμε από τον κώδικα βήμα‑βήμα. Κάθε απόσπασμα παραμένει αμετάβλητο από το αρχικό tutorial, εξασφαλίζοντας πλήρη συμβατότητα.

### Δημιουργία Φωλιασμένων Σελιδοδεικτών (create nested bookmarks java)
**Βήμα 1: Αρχικοποίηση Εγγράφου και Builder**  
Αυτό δημιουργεί ένα νέο έγγραφο Word που μπορείτε να γεμίσετε με περιεχόμενο και σελιδοδείκτες.

```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

**Βήμα 2: Εισαγωγή του πρώτου (γονικού) σελιδοδείκτη**  
Ξεκινήστε τον εξωτερικό σελιδοδείκτη και προσθέστε κάποιο κείμενο.

```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

**Βήμα 3: Φωλιάστε έναν δεύτερο σελιδοδείκτη μέσα στον πρώτο**  
Τώρα προσθέτουμε έναν παιδικό σελιδοδείκτη που ζει μέσα στον γονικό.

```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

**Βήμα 4: Κλείσιμο του εξωτερικού σελιδοδείκτη**  

```java
builder.endBookmark("Bookmark 1");
```

**Βήμα 5: Προσθήκη επιπλέον σελιδοδεικτών ανώτερου επιπέδου**  
Μπορείτε να συνεχίσετε να προσθέτετε περισσότερους σελιδοδείκτες όπως χρειάζεται.

```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδείκτη (save word pdf bookmarks)
**Βήμα 1: Ρύθμιση `PdfSaveOptions`**  
Αυτές οι επιλογές σας επιτρέπουν να ορίσετε πώς θα εμφανίζονται οι σελιδοδείκτες στο τελικό PDF.

```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

**Βήμα 2: Ανάθεση επιπέδων περιγράμματος σε κάθε σελιδοδείκτη**  
Το Επίπεδο 1 είναι μια εγγραφή ανώτερου επιπέδου, το Επίπεδο 2 είναι φωλιασμένο κάτω από το Επίπεδο 1, κ.ο.κ.

```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

**Βήμα 3: Αποθήκευση του εγγράφου ως PDF**  
Το PDF θα περιέχει πλέον ένα δομημένο πάνελ σελιδοδεικτών.

```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
