## Οδηγός Υλοποίησης
Θα χωρίσουμε την υλοποίηση σε δύο λογικά μέρη: δημιουργία ενσωματωμένων σελιδοδεικτών και διαμόρφωση των επιπέδων περιγράμματος.

### Πώς να δημιουργήσετε σελιδοδείκτες σε έγγραφο Word
**Επισκόπηση** – Αυτή η ενότητα δείχνει τον ακριβή κώδικα που χρειάζεστε για **how to create bookmarks** που μπορούν αργότερα να εξαχθούν ως ιεραρχία PDF.

#### Βήμα 1: Αρχικοποίηση εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Το αντικείμενο `Document` αντιπροσωπεύει το αρχείο Word, ενώ το `DocumentBuilder` σας επιτρέπει να εισάγετε κείμενο, εικόνες και σελιδοδείκτες.

#### Βήμα 2: Εισαγωγή ενσωματωμένων σελιδοδεικτών
Ξεκινήστε με έναν κύριο σελιδοδείκτη:
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```
Τώρα ενσωματώστε έναν άλλο σελιδοδείκτη μέσα στον πρώτο:
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```
Κλείστε τον εξωτερικό σελιδοδείκτη:
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 3: Προσθήκη πρόσθετων ανεξάρτητων σελιδοδεικτών
Μπορείτε να συνεχίσετε να προσθέτετε όσους χρειάζεστε. Για παράδειγμα, ένας ξεχωριστός τρίτος σελιδοδείκτης:
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Πώς να δημιουργήσετε PDF με σελιδοδείκτες και επίπεδα περιγράμματος
**Επισκόπηση** – Αφού οι σελιδοδείκτες υπάρχουν στο έγγραφο Word, διαμορφώνουμε την ιεραρχία τους πριν αποθηκεύσουμε ως PDF.

#### Βήμα 1: Ρύθμιση PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
Αυτές οι επιλογές λένε στο Aspose.Words πώς να μεταφράσει τους σελιδοδείκτες του Word σε καταχωρίσεις περιγράμματος PDF.

#### Βήμα 2: Ανάθεση επιπέδων περιγράμματος
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```
Ο ακέραιος ορίζει το βάθος – `1` είναι το ανώτερο επίπεδο, `2` είναι παιδί, κ.ο.κ.

#### Βήμα 3: Αποθήκευση του εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Το παραγόμενο PDF θα εμφανίζει ένα τακτοποιημένο παράθυρο σελιδοδεικτών που αντανακλά την ιεραρχία που ορίσατε.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Απουσία Σελιδοδεικτών** – Ελέγξτε ξανά ότι κάθε `startBookmark` έχει αντίστοιχο `endBookmark`.  
- **Λανθασμένα Επίπεδα** – Βεβαιωθείτε ότι οι αριθμοί επιπέδων αντιστοιχούν στη ζητούμενη σχέση γονέα‑παιδιού.
