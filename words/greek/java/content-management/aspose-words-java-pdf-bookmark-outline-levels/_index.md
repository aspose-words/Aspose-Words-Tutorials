# Δημιουργία Ενσωματωμένων Σελιδοδεικτών σε PDF με Aspose.Words Java

## Εισαγωγή
Αν έχετε ποτέ δυσκολευτεί να διατηρήσετε τους σελιδοδείκτες PDF οργανωμένους μετά τη μετατροπή ενός εγγράφου Word, δεν είστε μόνοι. Σε αυτό το tutorial θα **δημιουργήσετε ενσωματωμένους σελιδοδείκτες** και θα μάθετε πώς να **δημιουργήσετε PDF με σελιδοδείκτες** που είναι εύκολο να περιηγηθείτε. Θα περάσουμε από τη ρύθμιση του Aspose.Words, την κατασκευή μιας ιεραρχίας σελιδοδεικτών, την ανάθεση επιπέδων περιγράμματος και, τέλος, την εξαγωγή ενός καθαρού PDF.

**Τι θα μάθετε**
- Πώς να ρυθμίσετε το Aspose.Words για Java
- Πώς να **δημιουργήσετε ενσωματωμένους σελιδοδείκτες** μέσα σε ένα έγγραφο Word
- Πώς να διαμορφώσετε τα επίπεδα περιγράμματος των σελιδοδεικτών για σαφή πλοήγηση στο PDF
- Πώς να **δημιουργήσετε PDF με σελιδοδείκτες** που αντικατοπτρίζουν την ιεραρχία που ορίσατε

### Σύντομες Απαντήσεις
- **Ποια είναι η κύρια κλάση για τη δημιουργία εγγράφων;** `DocumentBuilder`
- **Ποια μέθοδος προσθέτει έναν σελιδοδείκτη;** `startBookmark(String name)`
- **Πώς ορίζετε ένα επίπεδο περιγράμματος για έναν σελιδοδείκτη;** `outlineLevels.add(name, level)`
- **Χρειάζομαι άδεια για παραγωγή;** Ναι, μια αγορασμένη άδεια ξεκλειδώνει όλες τις λειτουργίες.
- **Μπορώ να το χρησιμοποιήσω με Maven ή Gradle;** Απόλυτα – και τα δύο υποστηρίζονται.

### Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:
- **Aspose.Words for Java** (έκδοση 25.3 ή νεότερη).  
- Ένα εγκατεστημένο JDK και ένα IDE όπως το IntelliJ IDEA ή το Eclipse.  
- Βασικές γνώσεις Java και εξοικείωση με Maven ή Gradle.

## Τι είναι η «δημιουργία ενσωματωμένων σελιδοδεικτών»;
Η δημιουργία ενσωματωμένων σελιδοδεικτών σημαίνει την τοποθέτηση ενός σελιδοδείκτη μέσα σε έναν άλλο, σχηματίζοντας μια ιεραρχία γονέα‑παιδίου. Όταν το έγγραφο αποθηκευτεί ως PDF, αυτές οι σχέσεις εμφανίζονται ως αναδιπλώσιμες καταχωρήσεις στο πάνελ σελιδοδεικτών του PDF, καθιστώντας τα μεγάλα έγγραφα πολύ πιο εύκολα στην εξερεύνηση.

## Γιατί να χρησιμοποιήσετε επίπεδα περιγράμματος όταν δημιουργείτε PDF με σελιδοδείκτες;
Τα επίπεδα περιγράμματος ορίζουν την οπτική ιεραρχία των σελιδοδεικτών στον προβολέα PDF. Ένας σελιδοδείκτης επιπέδου 1 εμφανίζεται ως καταχώρηση κορυφαίου επιπέδου, ο επιπέδου 2 ως παιδί, κ.ο.κ. Τα σωστά επίπεδα περιγράμματος μετατρέπουν μια επίπεδη λίστα σελιδοδεικτών σε μια δομημένη λίστα περιεχομένων, κάτι που είναι ιδιαίτερα πολύτιμο για νομικές συμβάσεις, τεχνικές αναφορές και e‑books.

## Οδηγός Υλοποίησης
Παρακάτω ακολουθεί ένα βήμα‑βήμα walkthrough για τη δημιουργία ενός εγγράφου, την προσθήκη ενσωματωμένων σελιδοδεικτών, την ανάθεση επιπέδων περιγράμματος και την αποθήκευση του αποτελέσματος ως PDF.

### Βήμα 1: Αρχικοποίηση του Εγγράφου και του Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί ένα κενό έγγραφο Word και ένα αντικείμενο builder που θα χρησιμοποιήσετε για την εισαγωγή κειμένου και σελιδοδεικτών.

### Βήμα 2: Δημιουργία του Πρώτου (Γονικού) Σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```
Η κλήση `startBookmark` ανοίγει ένα νέο σελιδοδείκτη με όνομα **Bookmark 1**. Οτιδήποτε γράψετε μετά από αυτήν την κλήση θα ανήκει σε αυτόν τον σελιδοδείκτη μέχρι να τον κλείσετε.

### Βήμα 3: Ενσωμάτωση Δεύτερου Σελιδοδείκτη μέσα στον Πρώτο
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```
Επειδή αυτός ο σελιδοδείκτης ξεκινά **μετά** τον πρώτο και κλείνει **πριν** τον πρώτο, γίνεται παιδί του **Bookmark 1**.

### Βήμα 4: Κλείσιμο του Γονικού Σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```
Τώρα η ιεραρχία φαίνεται ως εξής:

- Σελιδοδείκτης 1 (επίπεδο 1)  
  - Σελιδοδείκτης 2 (επίπεδο 2)

### Βήμα 5: Προσθήκη Ανεξάρτητου Τρίτου Σελιδοδείκτη
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```
Αυτός ο σελιδοδείκτης βρίσκεται στο κορυφαίο επίπεδο, ξεχωριστός από τους πρώτους δύο.

### Βήμα 6: Διαμόρφωση Επιπέδων Περιγράμματος για Εξαγωγή PDF
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
Το αντικείμενο `PdfSaveOptions` σας επιτρέπει να ελέγξετε πώς εμφανίζονται οι σελιδοδείκτες στο τελικό PDF.

```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 1);
```
Εδώ αναθέτουμε επίπεδο 1 στους σελιδοδείκτες κορυφαίου επιπέδου και επίπεδο 2 στον ενσωματωμένο.

### Βήμα 7: Αποθήκευση του Εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Το παραγόμενο PDF θα εμφανίζει ένα καθαρό, αναδιπλώσιμο πάνελ σελιδοδεικτών που αντικατοπτρίζει την ιεραρχία που ορίσατε.

## Συχνά Προβλήματα και Λύσεις
- **Αγνοούμενοι Σελιδοδείκτες** – Κάθε `startBookmark` πρέπει να έχει ένα αντίστοιχο `endBookmark`. Η παράλειψη ενός θα κάνει τον σελιδοδείκτη να αγνοηθεί στο PDF.  
- **Λανθασμένα Επίπεδα Περιγράμματος** – Ελέγξτε ξανά τα ονόματα που περνάτε στο `outlineLevels.add`. Ένα τυπογραφικό λάθος σημαίνει ότι το επίπεδο δεν θα εφαρμοστεί.  
- **Μεγάλα Έγγραφα** – Για πολύ μεγάλα αρχεία, καλέστε `doc.removeMacros()` ή αφαιρέστε αχρησιμοποίητα στυλ πριν την αποθήκευση για να διατηρήσετε το μέγεθος του PDF λογικό.
