# Δημιουργία ενσωματωμένων σελιδοδεικτών σε PDF με Aspose.Words Java

## Εισαγωγή
Αν χρειάζεστε να **δημιουργήσετε ενσωματωμένους σελιδοδείκτες** σε ένα PDF που δημιουργείται από έγγραφο Word, βρίσκεστε στο σωστό μέρος. Σε αυτό το σεμινάριο θα περάσουμε από τη διαδικασία ολοκληρωμένα χρησιμοποιώντας το Aspose.Words for Java, από τη ρύθμιση της βιβλιοθήκης μέχρι τη διαμόρφωση των επιπέδων περιγράμματος των σελιδοδεικτών και τελικά **αποθήκευση των σελιδοδεικτών Word PDF** ώστε το τελικό PDF να είναι εύκολο στην πλοήγηση.

**Τι θα μάθετε**
- Πώς να ρυθμίσετε το Aspose.Words for Java
- Πώς να **δημιουργήσετε ενσωματωμένους σελιδοδείκτες** μέσα σε έγγραφο Word
- Πώς να αναθέσετε επίπεδα περιγράμματος για σαφή πλοήγηση στο PDF
- Πώς να **αποθηκεύσετε σελιδοδείκτες Word PDF** χρησιμοποιώντας το PdfSaveOptions

## Τι σημαίνει “δημιουργία ενσωματωμένων σελιδοδεικτών”;
Η δημιουργία ενσωματωμένων σελιδοδεικτών σημαίνει την τοποθέτηση ενός σελιδοδείκτη μέσα σε έναν άλλο, σχηματίζοντας μια ιεραρχική δομή που αντικατοπτρίζει τις λογικές ενότητες του εγγράφου σας. Αυτή η ιεραρχία εμφανίζεται στο πλαίσιο πλοήγησης του PDF, επιτρέποντας στους αναγνώστες να μεταβούν απευθείας σε συγκεκριμένα κεφάλαια ή υποενότητες.

## Γιατί να χρησιμοποιήσετε το Aspose.Words for Java για την αποθήκευση σελιδοδεικτών Word PDF;
Το Aspose.Words παρέχει ένα υψηλού επιπέδου API που αφαιρεί την ανάγκη για χαμηλού επιπέδου χειρισμό PDF, επιτρέποντάς σας να εστιάσετε στη δομή του περιεχομένου αντί για τις λεπτομέρειες του μορφότυπου αρχείου. Διατηρεί επίσης όλα τα χαρακτηριστικά του Word (στυλ, εικόνες, πίνακες) ενώ σας δίνει πλήρη έλεγχο πάνω στην ιεραρχία των σελιδοδεικτών.

## Οδηγός Υλοποίησης
Παρακάτω ακολουθεί μια βήμα‑βήμα περιγραφή. Κάθε μπλοκ κώδικα παραμένει αμετάβλητο για να διατηρηθεί η λειτουργικότητα.

### Πώς να δημιουργήσετε ενσωματωμένους σελιδοδείκτες σε έγγραφο Word
#### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Δημιουργεί ένα κενό έγγραφο Word και ένα αντικείμενο builder για την εισαγωγή περιεχομένου.

#### Βήμα 2: Εισαγωγή του πρώτου (γονικού) σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

#### Βήμα 3: Ενσωμάτωση δεύτερου σελιδοδείκτη μέσα στον πρώτο
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

#### Βήμα 4: Κλείσιμο του εξωτερικού σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 5: Προσθήκη ξεχωριστού τρίτου σελιδοδείκτη
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Πώς να αποθηκεύσετε σελιδοδείκτες Word PDF και να ορίσετε επίπεδα περιγράμματος
#### Βήμα 1: Διαμόρφωση του PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

#### Βήμα 2: Ανάθεση επιπέδων περιγράμματος σε κάθε σελιδοδείκτη
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Βήμα 3: Αποθήκευση του εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

## Συχνά Προβλήματα και Λύσεις
- **Αγνοούμενοι σελιδοδείκτες** – Επαληθεύστε ότι κάθε `startBookmark` έχει αντίστοιχο `endBookmark`.  
- **Λανθασμένη ιεραρχία** – Βεβαιωθείτε ότι οι αριθμοί επιπέδου περιγράμματος αντανακλούν τη ζητούμενη σχέση γονέα‑παιδίου (χαμηλότεροι αριθμοί = υψηλότερο επίπεδο).  
- **Μεγάλο μέγεθος αρχείου** – Αφαιρέστε αχρησιμοποίητα στυλ ή εικόνες πριν από την αποθήκευση, ή καλέστε `doc.optimizeResources()` εάν χρειάζεται.
