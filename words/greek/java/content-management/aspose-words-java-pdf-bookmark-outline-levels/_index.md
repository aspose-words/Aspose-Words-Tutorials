# Πώς να προσθέσετε σελιδοδείκτες σε PDF με το Aspose.Words Java

## Εισαγωγή
Αν ποτέ έχετε δυσκολευτεί να **προσθέσετε σελιδοδείκτες** που κάνουν ένα PDF εύκολο στην πλοήγηση, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα δούμε **πώς να προσθέσετε σελιδοδείκτες** και να ορίσετε επίπεδα περιγράμματος κατά τη μετατροπή εγγράφων Word σε PDF χρησιμοποιώντας το Aspose.Words για Java. Στο τέλος θα κατανοήσετε τη πλήρη ροή εργασίας — από τη δημιουργία ενσωματωμένων σελιδοδεικτών σε ένα αρχείο Word μέχρι την εξαγωγή ενός καθαρού, αναζητήσιμου PDF με λογική ιεραρχία σελιδοδεικτών.

**Τι θα μάθετε**
- Ρύθμιση του Aspose.Words για Java στο έργο σας  
- Δημιουργία ενσωματωμένων σελιδοδεικτών μέσα σε έγγραφο Word  
- Διαμόρφωση επιπέδων περιγράμματος σελιδοδεικτών για μια άψογη εμπειρία πλοήγησης PDF  
- Αποθήκευση του εγγράφου ως PDF διατηρώντας τη δομή των σελιδοδεικτών  

### Γρήγορες Απαντήσεις
- **Ποιο είναι το κύριο όφελος της προσθήκης σελιδοδεικτών;** Επιτρέπει στους αναγνώστες να μεταπηδούν απευθείας σε ενότητες, βελτιώνοντας τη χρηστικότητα.  
- **Ποια βιβλιοθήκη διαχειρίζεται τους σελιδοδείκτες PDF σε Java;** Aspose.Words for Java (με προαιρετικό Aspose.PDF για μετα‑επεξεργασία).  
- **Χρειάζομαι άδεια για αυτή τη λειτουργία;** Μια δοκιμαστική έκδοση λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να ελέγξω την ιεραρχία των σελιδοδεικτών;** Ναι, ορίζοντας επίπεδα περιγράμματος μέσω του `PdfSaveOptions`.  
- **Είναι αυτή η προσέγγιση κατάλληλη για μεγάλα έγγραφα;** Απολύτως—το Aspose.Words μεταδίδει το περιεχόμενο αποδοτικά.  

## Τι σημαίνει “πώς να προσθέσετε σελιδοδείκτες” στο πλαίσιο της μετατροπής PDF;
Η προσθήκη σελιδοδεικτών σημαίνει εισαγωγή ονομασμένων αγκυρών σε ένα έγγραφο Word που μεταφέρονται στο PDF. Όταν ανοίγει το PDF, αυτοί οι σελιδοδείκτες εμφανίζονται στο πλαίσιο πλοήγησης, επιτρέποντας στους χρήστες να εντοπίζουν κεφάλαια, ενότητες ή οποιαδήποτε προσαρμοσμένα σημεία άμεσα.

## Γιατί να χρησιμοποιήσετε το Aspose.Words για Java για τη μετατροπή σελιδοδεικτών Word → PDF;
Το Aspose.Words διατηρεί την ακριβή ιεραρχία σελιδοδεικτών που ορίζετε στο Word, σε αντίθεση με πολλούς δωρεάν μετατροπείς που την ισοπεδώνουν ή την αφαιρούν. Επίσης, σας επιτρέπει να αναθέσετε **επίπεδα περιγράμματος**, παρέχοντας λεπτομερή έλεγχο στην προβολή του πίνακα περιεχομένων του PDF.

## Οδηγός Βήμα‑Βήμα

### Δημιουργία Ενσωματωμένων Σελιδοδεικτών
**Επισκόπηση:** Θα δημιουργήσουμε ένα απλό έγγραφο Word με τρεις σελιδοδείκτες, όπου ένας σελιδοδείκτης είναι ενσωματωμένος μέσα σε έναν άλλο.

#### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Δημιουργεί ένα κενό έγγραφο Word και ένα αντικείμενο builder που μας επιτρέπει να εισάγουμε κείμενο και σελιδοδείκτες.

#### Βήμα 2: Εισαγωγή του Πρώτου (γονικού) Σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

#### Βήμα 3: Ενσωμάτωση Δεύτερου Σελιδοδείκτη Μέσα στον Πρώτο
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

#### Βήμα 4: Κλείσιμο του Γονικού Σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 5: Προσθήκη Ανεξάρτητου Τρίτου Σελιδοδείκτη
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```
Σε αυτό το σημείο το έγγραφο Word περιέχει μια σαφή ιεραρχία που μπορούμε αργότερα να μετατρέψουμε σε επίπεδα περιγράμματος PDF.

### Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
**Επισκόπηση:** Τα επίπεδα περιγράμματος λένε στον προβολέα PDF πόσο βαθιά κάθε σελιδοδείκτης βρίσκεται στο πλαίσιο πλοήγησης.

#### Βήμα 1: Προετοιμασία του `PdfSaveOptions`
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

#### Βήμα 2: Ανάθεση Επιπέδων σε Κάθε Σελιδοδείκτη
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```
Το επίπεδο 1 εμφανίζεται στο ανώτερο επίπεδο, το επίπεδο 2 ως παιδί, κ.λπ.

#### Βήμα 3: Αποθήκευση του Εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Το προκύπτον PDF θα εμφανίσει ένα δομημένο πλαίσιο σελιδοδεικτών που αντικατοπτρίζει την ιεραρχία που ορίσαμε.

## Συχνά Προβλήματα και Λύσεις

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|---------|--------------|-----|
| Οι σελιδοδείκτες εξαφανίζονται στο PDF | `PdfSaveOptions` δεν έχει ρυθμιστεί | Βεβαιωθείτε ότι τα `outlineLevels` έχουν προστεθεί πριν από την αποθήκευση. |
| Ο ενσωματωμένος σελιδοδείκτης εμφανίζεται στο ανώτερο επίπεδο | Λανθασμένος αριθμός επιπέδου | Επαληθεύστε ότι οι παιδικοί σελιδοδείκτες λαμβάνουν υψηλότερο αριθμητικό επίπεδο. |
| Λείπει η κλήση `endBookmark` | Μη ισορροπημένες κλήσεις start/end | Ελέγξτε ξανά ότι κάθε `startBookmark` έχει αντίστοιχη κλήση `endBookmark`. |
