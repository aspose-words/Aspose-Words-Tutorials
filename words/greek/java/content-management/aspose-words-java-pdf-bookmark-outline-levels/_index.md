## Οδηγός Υλοποίησης

### Πώς να δημιουργήσετε φωλιασμένους σελιδοδείκτες σε έγγραφο Word
Θα δημιουργήσουμε ένα απλό έγγραφο και θα προσθέσουμε τρεις σελιδοδείκτες, ένας εκ των οποίων περιέχει έναν άλλο σελιδοδείκτη.

#### Βήμα 1: Αρχικοποίηση του εγγράφου και του builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

#### Βήμα 2: Εισαγωγή του πρώτου (γονικού) σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

#### Βήμα 3: Φωλιάστε έναν δεύτερο σελιδοδείκτη μέσα στον πρώτο
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

#### Βήμα 4: Κλείσιμο του εξωτερικού σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 5: Προσθήκη ανεξάρτητου τρίτου σελιδοδείκτη
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Πώς να ορίσετε τα επίπεδα περιγράμματος των σελιδοδεικτών για εξαγωγή PDF
Τώρα θα διαμορφώσουμε την ιεραρχία περιγράμματος που θα εμφανιστεί στο τελικό PDF.

#### Βήμα 1: Προετοιμασία του `PdfSaveOptions`
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

#### Βήμα 2: Ανάθεση επιπέδων περιγράμματος σε κάθε σελιδοδείκτη
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Βήμα 3: Αποθήκευση του εγγράφου ως PDF με τις διαμορφωμένες σελιδοδείκτες
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

## Συχνά Προβλήματα και Λύσεις
- **Απουσία σελιδοδεικτών** – Επαληθεύστε ότι κάθε `startBookmark` έχει αντίστοιχο `endBookmark`.  
- **Λανθασμένη ιεραρχία** – Ελέγξτε ξανά τους αριθμούς επιπέδων που αναθέτετε· ένας μικρότερος αριθμός σημαίνει υψηλότερο (γονικό) επίπεδο.  
- **Άδεια δεν εφαρμόστηκε** – Εάν οι σελιδοδείκτες εξαφανιστούν, βεβαιωθείτε ότι το αρχείο άδειας έχει φορτωθεί πριν από οποιαδήποτε επεξεργασία εγγράφου.
