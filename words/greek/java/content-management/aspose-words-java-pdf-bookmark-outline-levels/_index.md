# Προσθήκη σελιδοδεικτών PDF και επιπέδων περιγράμματος με Aspose.Words Java

Οι σωστά οργανωμένοι σελιδοδείκτες κάνουν τα μεγάλα PDF εύκολα στην πλοήγηση, είτε πρόκειται για νομικά συμβόλαια, λεπτομερείς αναφορές ή υλικό e‑learning.

**Τι θα μάθετε**
- Ρύθμιση και χρήση **Aspose.Words for Java**
- **Δημιουργία ένθετων σελιδοδεικτών** σε ένα έγγραφο Word
- **Πώς να ορίσετε τα επίπεδα περιγράμματος των σελιδοδεικτών** για μια καθαρή ιεραρχία
- **Αποθήκευση εγγράφου ως PDF** με δομημένο δέντρο σελιδοδεικτών

Ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε πριν προχωρήσουμε.

## Οδηγός Υλοποίησης
Θα περάσουμε από την υλοποίηση βήμα‑βήμα, χωρίζοντας κάθε μέρος σε μικρά κομμάτια.

### Δημιουργία Ένθετων Σελιδοδεικτών
**Επισκόπηση**: Μάθετε πώς να **δημιουργήσετε ένθετους σελιδοδείκτες** σε ένα έγγραφο Word χρησιμοποιώντας Aspose.Words for Java.

#### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί ένα νέο έγγραφο Word και ένα αντικείμενο builder που σας επιτρέπει να εισάγετε περιεχόμενο και σελιδοδείκτες.

#### Βήμα 2: Εισαγωγή Ένθετων Σελιδοδεικτών
Ξεκινήστε δημιουργώντας έναν κύριο σελιδοδείκτη:
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```
Τώρα, ενσωματώστε έναν άλλο σελιδοδείκτη μέσα σε αυτόν:
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```
Ολοκληρώστε τον εξωτερικό σελιδοδείκτη:
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 3: Προσθήκη Πρόσθετων Σελιδοδεικτών
Μπορείτε να συνεχίσετε να προσθέτετε σελιδοδείκτες όπως χρειάζεται. Για παράδειγμα, ένας ξεχωριστός τρίτος σελιδοδείκτης:
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
**Επισκόπηση**: Οργανώστε τους σελιδοδείκτες σας ορίζοντας τα επίπεδα περιγράμματος, που καθορίζουν την ιεραρχία που θα βλέπετε σε προβολείς PDF.

#### Βήμα 1: Ρύθμιση PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
Αυτές οι επιλογές θα εφαρμοστούν όταν **αποθηκεύετε το έγγραφο ως PDF**.

#### Βήμα 2: Προσθήκη Επιπέδων Περιγράμματος
Αναθέστε επίπεδα σε κάθε σελιδοδείκτη· τα χαμηλότερα νούμερα εμφανίζονται πιο πάνω στο δέντρο περιγράμματος:
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Βήμα 3: Αποθήκευση Εγγράφου
Τέλος, δημιουργήστε το PDF με τη ρυθμισμένη ιεραρχία σελιδοδεικτών:
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

### Συμβουλές Επίλυσης Προβλημάτων
- **Αγνοούμενοι σελιδοδείκτες**: Επαληθεύστε ότι κάθε `startBookmark` έχει αντίστοιχο `endBookmark`.  
- **Λανθασμένα επίπεδα**: Ελέγξτε ξανά τη σειρά με την οποία προσθέτετε τα επίπεδα περιγράμματος· η ιεραρχία ακολουθεί το αριθμητικό επίπεδο που ορίζετε.  
- **Μεγάλα έγγραφα**: Χρησιμοποιήστε `doc.removeUnusedResources()` πριν την αποθήκευση για να μειώσετε το μέγεθος του PDF.
