# Πώς να ορίσετε σελιδοδείκτες σε PDF με Aspose.Words Java

## Εισαγωγή
Αν αντιμετωπίζετε δυσκολίες στη διαχείριση **πώς να ορίσετε σελιδοδείκτες** κατά τη μετατροπή εγγράφων Word σε PDF, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα περάσουμε από τη διαδικασία χρησιμοποιώντας το Aspose.Words για Java, από τη δημιουργία ένθετων σελιδοδεικτών μέχρι τη ρύθμιση των επιπέδων περιγράμματος ώστε το τελικό PDF να είναι καθαρό, επαγγελματικό και εύκολο στην πλοήγηση.

**Τι θα μάθετε**
- Ρυθμίστε το Aspose.Words για Java στο έργο σας  
- **Δημιουργήστε ένθετους σελιδοδείκτες** μέσα σε ένα έγγραφο Word  
- **Ρυθμίστε τα επίπεδα περιγράμματος των σελιδοδεικτών** για μια δομημένη διάρθρωση PDF  
- **Αποθηκεύστε PDF με σελιδοδείκτες** που αντικατοπτρίζουν την ιεραρχία που ορίσατε  

### Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για τη δημιουργία εγγράφων;** `DocumentBuilder`  
- **Ποια επιλογή ελέγχει την ιεραρχία των σελιδοδεικτών;** `PdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels()`  
- **Μπορώ να χρησιμοποιήσω Maven ή Gradle;** Ναι, και τα δύο υποστηρίζονται (δείτε τα αποσπάσματα κώδικα)

## Τι σημαίνει “πώς να ορίσετε σελιδοδείκτες” σε PDF;
Η δημιουργία σελιδοδεικτών σημαίνει ορισμό κλικ-μεταβλητών καταχωρήσεων στο πλαίσιο πλοήγησης του PDF που μεταβαίνουν σε συγκεκριμένα τμήματα του εγγράφου. Όταν οι σελιδοδείκτες είναι ένθετοι και έχουν εκχωρηθεί επίπεδα περιγράμματος, εμφανίζονται ως ένα αναδιπλούμενο δέντρο, καθιστώντας τα μεγάλα έγγραφα πολύ πιο εύκολα στην εξερεύνηση.

## Οδηγός Υλοποίησης
Παρακάτω υπάρχει ένας βήμα‑βήμα οδηγός που καλύπτει **δημιουργία ένθετων σελιδοδεικτών**, ορισμό των επιπέδων περιγράμματος και τελικά **αποθήκευση PDF με σελιδοδείκτες**.

### Δημιουργία Ένθετων Σελιδοδεικτών
**Επισκόπηση**: Δημιουργήστε ένα έγγραφο Word και ενσωματώστε σελιδοδείκτες που αντικατοπτρίζουν μια ιεραρχία.

#### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί ένα νέο έγγραφο έτοιμο για εισαγωγή περιεχομένου.

#### Βήμα 2: Εισαγωγή Ένθετων Σελιδοδεικτών
Ξεκινήστε με έναν κύριο σελιδοδείκτη, στη συνέχεια ενσωματώστε έναν δεύτερο μέσα σε αυτόν.

```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 3: Προσθήκη Πρόσθετων Σελιδοδεικτών
Μπορείτε να συνεχίσετε να προσθέτετε ανεξάρτητους σελιδοδείκτες όπως χρειάζεται.

```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Ρύθμιση Επιπέδων Περιγράμματος Σελιδοδεικτών
**Επισκόπηση**: Εκχωρήστε επίπεδα περιγράμματος ώστε το πλαίσιο σελιδοδεικτών του PDF να αντικατοπτρίζει την επιθυμητή ιεραρχία.

#### Βήμα 1: Ρύθμιση PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
Αυτές οι επιλογές θα χρησιμοποιηθούν κατά την αποθήκευση του εγγράφου ως PDF.

#### Βήμα 2: Προσθήκη Επιπέδων Περιγράμματος
Αντιστοιχίστε κάθε όνομα σελιδοδείκτη σε ένα επίπεδο περιγράμματος (1 = κορυφαίο επίπεδο, 2 = παιδί, κλπ.).

```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Βήμα 3: Αποθήκευση του Εγγράφου
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Το παραγόμενο PDF τώρα περιέχει ένα δομημένο δέντρο σελιδοδεικτών.

- **Μεγάλα έγγραφα** – Καλέστε `doc.removeUnusedResources()` πριν την αποθήκευση για να μειώσετε το μέγεθος του αρχείου.
