# Προσθήκη σελιδοδεικτών PDF και επιπέδων περιγράμματος με Aspose.Words Java

## Εισαγωγή
Αν αντιμετωπίζετε δυσκολίες με την **προσθήκη σελιδοδεικτών PDF** που παραμένουν οργανωμένοι κατά τη μετατροπή εγγράφων Word σε PDF, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα δούμε πώς να χρησιμοποιήσετε το Aspose.Words for Java για να δημιουργήσετε **ενσωματωμένους σελιδοδείκτες σε PDF**, να ορίσετε επίπεδα περιγράμματος και να παραγάγετε ένα καθαρό, πλοηγήσιμο αρχείο PDF.

**Τι θα μάθετε**

- Αποθήκευση του τελικού εγγράφου ως PDF με σωστά δομημένους σελιδοδείκτες  

### Σύντομες Απαντήσεις
- **Ποιο είναι το κύριο όφελος της προσθήκης σελιδοδεικτών PDF;** Βελτιώνει την πλοήγηση και την εμπειρία χρήστη σε μεγάλα έγγραφα.

- **Μπορώ να ορίσω διαφορετικά επίπεδα περιγράμματος για κάθε σελιδοδείκτη;** Ναι, χρησιμοποιώντας `BookmarksOutlineLevelCollection` στο `PdfSaveOptions`.  
- **Είναι αυτή η μέθοδος συμβατή με την τελευταία έκδοση του Aspose.Words;** Απόλυτα – λειτουργεί με την έκδοση 25.3 και νεότερη.

## Τι είναι η “προσθήκη σελιδοδεικτών PDF”;
Η προσθήκη σελιδοδεικτών PDF σημαίνει η εισαγωγή κλικ-μεγάλων καταχωρήσεων στο πλαίσιο πλοήγησης του PDF που οδηγούν σε συγκεκριμένα τμήματα του εγγράφου. Όταν συνδυάζονται με επίπεδα περιγράμματος, αυτοί οι σελιδοδείκτες σχηματίζουν μια δομή δέντρου που αντικατοπτρίζει την ιεραρχία του εγγράφου σας.

## Γιατί να χρησιμοποιήσετε ενσωματωμένους (nested) σελιδοδείκτες σε PDF;
Οι ενσωματωμένοι σελιδοδείκτες επιτρέπουν στους αναγνώστες να εμβαθύνουν από υψηλού επιπέδου ενότητες σε λεπτομερείς υποενότητες χωρίς να χρειάζεται κύλιση των σελίδων. Αυτό είναι ιδιαίτερα χρήσιμο για **νομικά συμβόλαια**, **τεχνικές εκθέσεις** και **εγχειρίδια e‑learning** όπου η γρήγορη αναφορά είναι απαραίτητη.

## Οδηγός Υλοποίησης
Ας χωρίσουμε την υλοποίηση σε σαφή, αριθμημένα βήματα.

### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί ένα νέο έγγραφο Word που θα γεμίσουμε με περιεχόμενο και σελιδοδείκτες.

### Βήμα 2: Εισαγωγή Ενσωματωμένων Σελιδοδεικτών
#### Δημιουργία του πρώτου (γονικού) σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

#### Ενσωμάτωση παιδικού σελιδοδείκτη μέσα στον γονέα
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

#### Κλείσιμο του γονικού σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```

#### Προσθήκη τρίτου, ανεξάρτητου σελιδοδείκτη
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Βήμα 3: Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδείκτη
#### Ρύθμιση του `PdfSaveOptions`
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

#### Ανάθεση επιπέδων ιεραρχίας
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Αποθήκευση του εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

### Συνηθισμένα Προβλήματα και Λύσεις
- **Απουσία σελιδοδεικτών** – Επαληθεύστε ότι κάθε `startBookmark` έχει αντίστοιχο `endBookmark`.  
- **Λανθασμένη ιεραρχία περιγράμματος** – Ελέγξτε ξανά τους αριθμούς επιπέδων· ένας μικρότερος αριθμός σημαίνει υψηλότερο επίπεδο στο πλαίσιο πλοήγησης.  
- **Μεγάλα έγγραφα** – Καλέστε `doc.optimizeResources()` πριν την αποθήκευση για μείωση της κατανάλωσης μνήμης.
