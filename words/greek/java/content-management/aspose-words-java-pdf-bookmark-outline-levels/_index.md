# Πώς να δημιουργήσετε σελιδοδείκτες και να ορίσετε επίπεδα περιγράμματος σε PDF χρησιμοποιώντας το Aspose.Words Java

## Εισαγωγή
Αν έχετε ποτέ δυσκολευτεί να **δημιουργήσετε σελιδοδείκτες** που παραμένουν οργανωμένοι κατά τη μετατροπή ενός εγγράφου Word σε PDF, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα περάσουμε από τη διαδικασία δημιουργίας ενός PDF με σελιδοδείκτες, την ένθεσή τους και την ανάθεση επιπέδων περιγράμματος ώστε το τελικό PDF να είναι εύκολο στην πλοήγηση. Στο τέλος θα μπορείτε να **μετατρέψετε Word PDF Java**‑style με μια καθαρή ιεραρχία σελιδοδεικτών που λειτουργεί σε οποιονδήποτε προβολέα PDF.

### Τι θα μάθετε
- Ρύθμιση του Aspose.Words για Java στο περιβάλλον ανάπτυξής σας.  
- **Πώς να δημιουργήσετε σελιδοδείκτες** προγραμματιστικά και να τους ενσωματώσετε.  
- Διαμόρφωση επιπέδων περιγράμματος σελιδοδεικτών για τη δημιουργία PDF με σελιδοδείκτες που αντικατοπτρίζουν τη δομή του εγγράφου.  
- Αποθήκευση του αρχείου Word ως PDF διατηρώντας την ιεραρχία των σελιδοδεικτών.

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για τη δημιουργία εγγράφων;** `DocumentBuilder`.  
- **Ποια επιλογή ελέγχει την ιεραρχία των σελιδοδεικτών;** `BookmarksOutlineLevelCollection` μέσα στο `PdfSaveOptions`.  
- **Μπορώ να χρησιμοποιήσω Maven ή Gradle;** Ναι – και τα δύο παρουσιάζονται παρακάτω.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται μόνιμη άδεια για παραγωγή.  
- **Είναι αυτή η προσέγγιση κατάλληλη για μεγάλα έγγραφα;** Ναι, αλλά σκεφτείτε τεχνικές βελτιστοποίησης μνήμης (π.χ., αφαίρεση αχρησιμοποίητων πόρων).

## Πώς να δημιουργήσετε σελιδοδείκτες σε PDF με Aspose.Words Java
Παρακάτω χωρίζουμε την υλοποίηση σε σαφή, αριθμημένα βήματα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από το αρχικό μπλοκ κώδικα (αμετάβλητο).

### Βήμα 1: Αρχικοποίηση ενός Document και ενός DocumentBuilder
Ξεκινάμε με μια νέα παρουσία `Document` και έναν `DocumentBuilder` που μας επιτρέπει να εισάγουμε περιεχόμενο και σελιδοδείκτες.

```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

### Βήμα 2: Εισαγωγή του Πρώτου (Γονικού) Σελιδοδείκτη
Δημιουργήστε έναν σελιδοδείκτη ανώτερου επιπέδου που θα περιέχει αργότερα έναν παιδικό σελιδοδείκτη.

```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

### Βήμα 3: Ένθεση Παιδικού Σελιδοδείκτη μέσα στον Γονικό
Τώρα προσθέτουμε έναν δεύτερο σελιδοδείκτη που ζει μέσα στον πρώτο, δείχνοντας την ένθεση.

```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

### Βήμα 4: Κλείσιμο του Γονικού Σελιδοδείκτη
Ολοκληρώστε τον εξωτερικό σελιδοδείκτη μετά το ένθετο περιεχόμενο.

```java
builder.endBookmark("Bookmark 1");
```

### Βήμα 5: Προσθήκη Ανεξάρτητου Τρίτου Σελιδοδείκτη
Μπορείτε πάντα να προσθέσετε περισσότερους σελιδοδείκτες που δεν είναι ενσωματωμένοι.

```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

## Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
Αφού οι σελιδοδείκτες είναι στη θέση τους, λέμε στο Aspose.Words πώς πρέπει να εμφανίζονται στο περίγραμμα του PDF (το αριστερό πλαίσιο πλοήγησης).

### Βήμα 6: Προετοιμασία του PdfSaveOptions
`PdfSaveOptions` μας δίνει πρόσβαση στις ρυθμίσεις του περιγράμματος.

```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

### Βήμα 7: Ανάθεση Επιπέδων Ιεραρχίας
Κάθε σελιδοδείκτης λαμβάνει έναν ακέραιο αριθμό επιπέδου· οι χαμηλότεροι αριθμοί είναι υψηλότερα στην ιεραρχία.

```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

### Βήμα 8: Αποθήκευση του Εγγράφου ως PDF
Τέλος, εξάγετε το έγγραφο Word σε PDF διατηρώντας το περίγραμμα των σελιδοδεικτών.

```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
