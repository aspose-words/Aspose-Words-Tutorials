# Δημιουργία Φωλιασμένων Σελιδοδεικτών Java με Aspose.Words PDF Επίπεδα

## Εισαγωγή
Αν χρειάζεστε να **create nested bookmarks Java**‑στυλ κατά τη μετατροπή εγγράφων Word σε PDF, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα περάσουμε από τη χρήση του Aspose.Words for Java για τη δημιουργία PDF με σελιδοδείκτες που είναι τακτοποιημένοι με επίπεδα περιγράμματος. Στο τέλος θα έχετε ένα επαγγελματικό PDF όπου οι αναγνώστες μπορούν να μεταβούν άμεσα σε οποιαδήποτε ενότητα.

**Τι Θα Μάθετε**

- Πώς να εξάγετε το έγγραφο ως PDF με δομημένους σελιδοδείκτες

- **Ποια είναι η κύρια κλάση για τη δημιουργία εγγράφων;** `DocumentBuilder`

## Τι είναι το “create nested bookmarks java”;
Η δημιουργία φωλιασμένων σελιδοδεικτών σημαίνει την τοποθέτηση ενός σελιδοδείκτη μέσα σε έναν άλλο, σχηματίζοντας μια ιεραρχία γονέα‑παιδιού. Όταν το έγγραφο αποθηκευτεί ως PDF, αυτές οι ιεραρχίες γίνονται συμπτυγμένες καταχωρίσεις στον πίνακα σελιδοδεικτών του PDF, καθιστώντας την πλοήγηση διαισθητική για τους αναγνώστες.

## Γιατί να δημιουργήσετε PDF με σελιδοδείκτες;
Η ενσωμάτωση σελιδοδεικτών σε PDF βελτιώνει την εμπειρία του χρήστη, ειδικά για νομικές συμβάσεις, εκτενείς αναφορές ή ηλεκτρονικά βιβλία. Οι αναγνώστες μπορούν άμεσα να μεταβούν σε κεφάλαια, ενότητες ή συγκεκριμένες ρήτρες χωρίς να κάνουν κύλιση των σελίδων.

## Οδηγός Υλοποίησης
Θα χωρίσουμε τη λύση σε σαφή, αριθμημένα βήματα.

### Βήμα 1: Δημιουργία Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Αυτό δημιουργεί ένα κενό έγγραφο Word και ένα αντικείμενο builder που θα χρησιμοποιήσετε για την εισαγωγή περιεχομένου και σελιδοδεικτών.

### Βήμα 2: Εισαγωγή Φωλιασμένων Σελιδοδεικτών
#### Κύριος σελιδοδείκτης
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```

#### Φωλιασμένος σελιδοδείκτης μέσα στον κύριο
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

#### Κλείσιμο του εξωτερικού σελιδοδείκτη
```java
builder.endBookmark("Bookmark 1");
```

#### Πρόσθετος ανεξάρτητος σελιδοδείκτης
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```

### Βήμα 3: Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
#### Ρύθμιση επιλογών αποθήκευσης PDF
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```

#### Ανάθεση επιπέδων ιεραρχίας
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```

#### Αποθήκευση ως PDF με το καθορισμένο περίγραμμα
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```

### Συνηθισμένα Προβλήματα και Λύσεις
- **Απουσία Σελιδοδεικτών** – Κάθε `startBookmark` πρέπει να έχει ένα αντίστοιχο `endBookmark`.  
- **Λανθασμένα Επίπεδα** – Ελέγξτε ξανά τους αριθμούς επιπέδων· ορίζουν τη σχέση γονέα‑παιδιού στο PDF.  
- **Μεγάλα Έγγραφα** – Χρησιμοποιήστε `Document.optimizeResources()` πριν την αποθήκευση για να μειώσετε την κατανάλωση μνήμης.
