## Οδηγός Υλοποίησης
Θα χωρίσουμε την υλοποίηση σε δύο λογικά μέρη: δημιουργία φωλιασμένων σελιδοδεικτών και διαμόρφωση των επιπέδων περιγράμματος.

### Δημιουργία Φωλιασμένων Σελιδοδεικτών
**Επισκόπηση** – Αυτή η ενότητα δείχνει πώς να ενσωματώσετε ιεραρχικούς σελιδοδείκτες απευθείας σε ένα έγγραφο Word.

#### Βήμα 1: Αρχικοποίηση Εγγράφου και Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
Το `DocumentBuilder` σας παρέχει έναν βολικό τρόπο για την εισαγωγή κειμένου, πινάκων και σελιδοδεικτών.

#### Βήμα 2: Εισαγωγή Πρωτεύοντος και Φωλιασμένου Σελιδοδείκτη
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```
Τώρα προσθέστε έναν υπο-σελιδοδείκτη μέσα στον πρώτο:
```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```
Κλείστε τον εξωτερικό σελιδοδείκτη:
```java
builder.endBookmark("Bookmark 1");
```

#### Βήμα 3: Προσθήκη Ξεχωριστού Σελιδοδείκτη Επάνω Επιπέδου
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```
Μπορείτε να επαναλάβετε αυτά τα βήματα για να δημιουργήσετε την απαιτούμενη βάθος ιεραρχίας.

### Διαμόρφωση Επιπέδων Περιγράμματος Σελιδοδεικτών
**Επισκόπηση** – Αφού υπάρχουν οι σελιδοδείκτες, ορίστε τα επίπεδα περιγράμματος ώστε οι προβολείς PDF να τα εμφανίζουν σωστά.

#### Βήμα 1: Ρύθμιση PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
`PdfSaveOptions` ελέγχει πώς το έγγραφο Word αποδίδεται ως PDF.

#### Βήμα 2: Ανάθεση Επιπέδων σε Κάθε Σελιδοδείκτη
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```
Το Επίπεδο 1 εμφανίζεται ως καταχώρηση κορυφαίου επιπέδου, το Επίπεδο 2 ως υπο‑επίπεδο, κ.ο.κ.

#### Βήμα 3: Αποθήκευση του Εγγράφου ως PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Το προκύπτον PDF θα εμφανίσει ένα τρι-επίπεδο πάνελ σελιδοδεικτών που αντικατοπτρίζει τη δομή που ορίσατε.
