## Ρύθμιση Aspose.Words
Παρακάτω είναι ο ελάχιστος κώδικας που χρειάζεστε για να αρχίσετε να εργάζεστε με μεταβλητές εγγράφου.

```java
import com.aspose.words.*;

class DocumentVariableExample {
    public static void main(String[] args) throws Exception {
        // Initialize a new Document instance.
        Document doc = new Document();
        
        // Access the variable collection from the document.
        VariableCollection variables = doc.getVariables();

        System.out.println("Aspose.Words setup complete.");
    }
}
```

## Πώς να Δημιουργήσετε Πρότυπο Τιμολογίου Χρησιμοποιώντας Μεταβλητές Εγγράφου
### Χαρακτηριστικό 1: Προσθήκη Μεταβλητών σε Συλλογές Εγγράφου
Η προσθήκη ζευγών κλειδί/τιμή είναι το πρώτο βήμα για τη δημιουργία ενός προτύπου τιμολογίου.

```java
Document doc = new Document();
VariableCollection variables = doc.getVariables();
```

```java
variables.add("InvoiceNumber", "INV-1001");
variables.add("CustomerName", "Acme Corp.");
variables.add("TotalAmount", "£1,250.00");
```

- **`add(String key, Object value)`** εισάγει μια νέα μεταβλητή ή ενημερώνει μια υπάρχουσα.  
- Χρησιμοποιήστε περιγραφικά κλειδιά που ταιριάζουν με τα placeholders στο πρότυπο Word.

### Χαρακτηριστικό 2: Ενημέρωση Μεταβλητών και Πεδία DOCVARIABLE
Εισάγετε ένα πεδίο `DOCVARIABLE` όπου θέλετε να εμφανίζεται η τιμή της μεταβλητής.

```java
DocumentBuilder builder = new DocumentBuilder(doc);
FieldDocVariable field = (FieldDocVariable) builder.insertField(FieldType.FIELD_DOC_VARIABLE, true);
field.setVariableName("InvoiceNumber");
field.update();
```

Όταν χρειάζεται να αλλάξετε μια τιμή (π.χ. μετά από επεξεργασία του τιμολογίου από τον χρήστη), απλώς ενημερώστε τη μεταβλητή και ανανεώστε το πεδίο.

```java
variables.add("InvoiceNumber", "INV-1002");
field.update(); // Reflects updated value.
```

### Χαρακτηριστικό 3: Έλεγχος και Αφαίρεση Μεταβλητών
Πριν γράψετε δεδομένα, είναι καλή πρακτική να **ελέγχετε την ύπαρξη της μεταβλητής** ώστε να αποφύγετε σφάλματα χρόνου εκτέλεσης.

```java
boolean containsCustomer = variables.contains("CustomerName");
boolean hasHighValue = IterableUtils.matchesAny(variables, s -> s.getValue().equals("£1,250.00"));
```

- **`contains(String key)`** επιστρέφει `true` αν η μεταβλητή υπάρχει.  
- **`IterableUtils.matchesAny(...)`** σας επιτρέπει να αναζητήσετε με βάση την τιμή.

Αν μια μεταβλητή δεν χρειάζεται πλέον, αφαιρέστε την καθαρά:

```java
variables.remove("CustomerName");
variables.removeAt(1);
variables.clear(); // Clears the entire collection.
```

### Χαρακτηριστικό 4: Διαχείριση Σειράς Μεταβλητών
Το Aspose.Words αποθηκεύει τα ονόματα μεταβλητών αλφαβητικά, κάτι που μπορεί να είναι χρήσιμο όταν χρειάζεστε προβλέψιμη σειρά.

```java
int indexInvoice = variables.indexOfKey("InvoiceNumber"); // Should be 0
int indexTotal = variables.indexOfKey("TotalAmount");    // Should be 1
int indexCustomer = variables.indexOfKey("CustomerName"); // Should be 2
```
