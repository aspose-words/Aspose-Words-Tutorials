### Διαχείριση Ενσωματωμένων Αναθεωρήσεων

#### Επισκόπηση

Κατά την παρακολούθηση αλλαγών σε ένα έγγραφο, η κατανόηση και η διαχείριση των ενσωματωμένων (inline) αναθεωρήσεων είναι κρίσιμη. Αυτές μπορεί να περιλαμβάνουν εισαγωγές, διαγραφές, αλλαγές μορφοποίησης ή μετακινήσεις κειμένου.

#### Υλοποίηση Κώδικα

Ακολουθεί ένας βήμα‑βήμα οδηγός για τον προσδιορισμό του τύπου αναθεώρησης ενός ενσωματωμένου κόμβου με το Aspose.Words Java:

```java
import com.aspose.words.Document;
import com.aspose.words.Paragraph;
import com.aspose.words.Run;
import com.aspose.words.Revision;
import org.testng.Assert;

public class RevisionHandler {
    public void handleRevisions() throws Exception {
        Document doc = new Document("Revision runs.docx");

        // Check the number of revisions
        Assert.assertEquals(6, doc.getRevisions().getCount());

        // Accessing a specific revision's parent node
        Run run = (Run) doc.getRevisions().get(0).getParentNode();

        Paragraph paragraph = run.getParentParagraph();
        com.aspose.words.RunCollection runs = paragraph.getRuns();

        Assert.assertEquals(runs.getCount(), 6);

        // Identifying different types of revisions
        Assert.assertTrue(runs.get(2).isInsertRevision());  // Insert revision
        Assert.assertTrue(runs.get(2).isFormatRevision());  // Format revision
        Assert.assertTrue(runs.get(4).isMoveFromRevision()); // Move from revision
        Assert.assertTrue(runs.get(1).isMoveToRevision());   // Move to revision
        Assert.assertTrue(runs.get(5).isDeleteRevision());   // Delete revision
    }
}
```

#### Επεξήγηση
- **Insert Revision:** Συμβαίνει όταν προστίθεται κείμενο ενώ η παρακολούθηση αλλαγών είναι ενεργή.  
- **Format Revision:** Ενεργοποιείται από τροποποιήσεις μορφοποίησης στο κείμενο.  
- **Move From/To Revisions:** Αντιπροσωπεύουν τη μετακίνηση κειμένου εντός του εγγράφου, εμφανιζόμενες σε ζεύγη.  
- **Delete Revision:** Σημαδεύει διαγραμμένο κείμενο που αναμένει αποδοχή ή απόρριψη.

### Πρακτικές Εφαρμογές

Ακολουθούν μερικά πραγματικά σενάρια όπου η διαχείριση αναθεωρήσεων είναι επωφελής:

2. **Ανασκόπηση Νομικών Εγγράφων:** Οι νομικοί μπορούν να παρακολουθούν τροποποιήσεις συμβάσεων, διασφαλίζοντας ότι όλα τα μέρη συμφωνούν στην τελική μορφή.
