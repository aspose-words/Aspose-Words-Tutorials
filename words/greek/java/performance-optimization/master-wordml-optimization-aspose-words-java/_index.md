### Οδηγός Εφαρμογής

#### Χαρακτηριστικό όμορφης μορφής
**Επισκόπηση:**
Η λειτουργία 'PrettyFormat' δημιουργεί WordML με όμορφα εσοχισμένη και ευανάγνωστη δομή XML, διευκολύνοντας τον εντοπισμό σφαλμάτων και την κατανόησή της.

##### Βήμα 1: Δημιουργία εγγράφου
Ξεκινήστε δημιουργώντας ένα νέο `Document` αντικείμενο και χρήση `DocumentBuilder` για να προσθέσετε περιεχόμενο:

```java
import com.aspose.words.Document;
import com.aspose.words.DocumentBuilder;

// Αρχικοποίηση εγγράφου.
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.writeln("Hello world!");
```

##### Βήμα 2: Ρύθμιση παραμέτρων του WordML2003SaveOptions
Στήνω `WordML2003SaveOptions` για να ενεργοποιήσετε την όμορφη μορφοποίηση:

```java
import com.aspose.words.WordML2003SaveOptions;

// Αρχικοποίηση επιλογών αποθήκευσης.
WordML2003SaveOptions options = new WordML2003SaveOptions();
options.setPrettyFormat(true); // Ενεργοποίηση όμορφης μορφής για έξοδο XML.

doc.save("YOUR_DOCUMENT_DIRECTORY/WordML2003SaveOptions.PrettyFormat.xml", options);
```

**Εξήγηση:**
- **`setPrettyFormat(true)`:** Ρυθμίζει τις παραμέτρους του εγγράφου ώστε να αποθηκεύεται με αναγνώσιμη μορφοποίηση, συμπεριλαμβανομένων εσοχών και αλλαγών γραμμής.

#### Λειτουργία βελτιστοποίησης μνήμης
**Επισκόπηση:**
Η αποτελεσματική διαχείριση της μνήμης είναι ζωτικής σημασίας κατά την επεξεργασία μεγάλων εγγράφων. Η λειτουργία «Βελτιστοποίηση μνήμης» βοηθά στη μείωση του αποτυπώματος μνήμης κατά τη διάρκεια των εργασιών αποθήκευσης.

##### Βήμα 1: Αρχικοποίηση εγγράφου
Δημιουργήστε ένα νέο `Document` αντικείμενο:

```java
import com.aspose.words.Document;
import com.aspose.words.DocumentBuilder;

// Δημιουργήστε ένα νέο έγγραφο.
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.writeln("Hello world!");
```

##### Βήμα 2: Ορισμός βελτιστοποίησης μνήμης
Διαμορφώστε τις επιλογές αποθήκευσης για να βελτιστοποιήσετε τη χρήση μνήμης:

```java
import com.aspose.words.WordML2003SaveOptions;

// Αρχικοποιήστε το WordML2003SaveOptions.
WordML2003SaveOptions options = new WordML2003SaveOptions();
options.setMemoryOptimization(true); // Ενεργοποίηση βελτιστοποίησης μνήμης.

doc.save("YOUR_DOCUMENT_DIRECTORY/WordML2003SaveOptions.MemoryOptimization.xml", options);
```

**Εξήγηση:**
- **`setMemoryOptimization(true)`:** Μειώνει το αποτύπωμα μνήμης κατά την αποθήκευση εγγράφων, κάτι που είναι απαραίτητο για την αποτελεσματική διαχείριση μεγάλων αρχείων.
