## Πώς να προσθέσετε σχόλιο Aspose Java;

Το Document αντιπροσωπεύει ένα αρχείο Word που έχει φορτωθεί στη μνήμη. Το DocumentBuilder είναι μια βοηθητική κλάση που χρησιμοποιείται για τη δημιουργία και την επεξεργασία ενός Document. Η μέθοδος insertComment προσθέτει έναν νέο κόμβο σχολίου στο έγγραφο. Φορτώστε το στόχο εγγράφου με `Document doc = new Document("input.docx")`, δημιουργήστε ένα `DocumentBuilder` και καλέστε `insertComment("Your comment text", "Author Name", new Date())`. Αυτή η μονή‑γραμμή ενέργεια εισάγει ένα πλήρως εξοπλισμένο σχόλιο που περιλαμβάνει συγγραφέα, κείμενο και χρονική σήμανση, και λειτουργεί σε όλα τα 35+ υποστηριζόμενα μορφότυπα χωρίς την ανάγκη εγκατάστασης του Microsoft Word.

## Πώς να αφαιρέσετε σημειώσεις Java;

Η Annotation είναι ένα στοιχείο σήμανσης όπως ένα σχόλιο, σημείωση ή επισήμανση. Η μέθοδος doc.getAnnotations() επιστρέφει τη συλλογή Annotation του εγγράφου. Ανακτήστε τη συλλογή `Annotation` μέσω `doc.getAnnotations()`, εντοπίστε τη σημείωση που θέλετε να διαγράψετε (με βάση ID, τύπο ή συγγραφέα) και καλέστε `annotation.remove()`. Η `annotation.remove()` διαγράφει αυτή τη σημείωση από το έγγραφο. Η αλλαγή αντικατοπτρίζεται αμέσως όταν αποθηκευτεί το αρχείο, επιτρέποντας καθαρό, αυτοματοποιημένο καθαρισμό των στοιχείων ανασκόπησης.

## Πώς να αυτοματοποιήσετε βρόχους ανατροφοδότησης με Aspose.Words;

Η μέθοδος removeAnnotation αφαιρεί μια συγκεκριμένη σημείωση από το έγγραφο. Δημιουργήστε μια παρτίδα εργασίας που φορτώνει κάθε έγγραφο, εφαρμόζει `insertComment` ή `removeAnnotation` ανάλογα με τις ανάγκες, και στη συνέχεια αποθηκεύει το αρχείο σε έναν προορισμένο φάκελο εξόδου. Συνδυάζοντας αυτές τις κλήσεις API μέσα σε βρόχο, μπορείτε αυτόματα να συλλέγετε την εισροή των ελεγκτών, να εφαρμόζετε μαζικές ενημερώσεις και να δημιουργείτε τα τελικά έγγραφα—όλα μέσα σε μια ενιαία, συντηρήσιμη διαδικασία Java.

## Συχνά Προβλήματα και Λύσεις

- **Τα σχόλια δεν εμφανίζονται στο UI** – Βεβαιωθείτε ότι το έγγραφο ανοίγει σε προβολέα που υποστηρίζει σχόλια (π.χ., Microsoft Word ή προεπισκόπηση Aspose.Words).  
- **Οι σημειώσεις εξαφανίζονται μετά την αποθήκευση** – Επαληθεύστε ότι αποθηκεύετε σε μορφότυπο που διατηρεί τις σημειώσεις (DOCX, PDF, κλπ.).  
- **Μείωση απόδοσης σε μεγάλα αρχεία** – Χρησιμοποιήστε `Document.optimizeResources()` πριν την επεξεργασία για μείωση χρήσης μνήμης. Το `Document.optimizeResources()` συμπιέζει ενσωματωμένους πόρους για χαμηλότερη χρήση μνήμης.

## Συχνές Ερωτήσεις

**Q: Μπορώ να προσθέσω σχόλια σε έγγραφα προστατευμένα με κωδικό;**  
A: Ναι. Ανοίξτε το έγγραφο με `new LoadOptions("password")`, στη συνέχεια προσθέστε σχόλια όπως συνήθως.

**Q: Η αφαίρεση μιας σημείωσης επηρεάζει άλλο περιεχόμενο;**  
A: Όχι. Η αφαίρεση μιας σημείωσης διαγράφει μόνο τον κόμβο σήμανσης· το γύρω κείμενο παραμένει αμετάβλητο.

**Q: Είναι δυνατόν η εξαγωγή σχολίων σε ξεχωριστή αναφορά;**  
A: Απόλυτα. Επανάληψη `doc.getComments()` και εγγραφή του συγγραφέα, του κειμένου και της ημερομηνίας κάθε σχολίου σε αρχείο CSV ή JSON.

**Q: Ποιες εκδόσεις Java υποστηρίζονται;**  
A: Το Aspose.Words for Java λειτουργεί με Java 8, 11 και νεότερες εκδόσεις LTS.

**Q: Πώς διαχειρίζομαι σχόλια στην έξοδο PDF;**  
A: Κατά την αποθήκευση σε PDF, ορίστε `PdfSaveOptions.setExportComments(true)` για να διατηρηθούν τα σχόλια στο τελικό PDF. Η `PdfSaveOptions.setExportComments(true)` ενημερώνει τον αποθηκευτή PDF να συμπεριλάβει τα σχόλια στην έξοδο.
