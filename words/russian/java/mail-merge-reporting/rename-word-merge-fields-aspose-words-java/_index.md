## Руководство по внедрению

В этом разделе мы проведем вас через процесс переименования полей слияния с помощью Aspose.Words.

### Функция: переименование полей слияния в документе Word

**Обзор**: Эта функция позволяет вам программно переименовывать поля слияния в шаблонах документов. Она упрощает управление шаблонами за счет автоматизации обновлений полей.

#### Шаг 1: Создайте и инициализируйте свой документ

Начните с создания нового `Document` объект и инициализируйте `DocumentBuilder`:

```java
import com.aspose.words.Document;
import com.aspose.words.DocumentBuilder;

Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

**Почему**: `DocumentBuilder` Класс предоставляет методы для вставки текста, полей и другого содержимого в ваш документ.

#### Шаг 2: Вставьте образцы полей слияния

Добавьте в документ несколько полей слияния:

```java
builder.write("Dear ");
builder.insertField("MERGEFIELD FirstName ");
builder.write(" ");
builder.insertField("MERGEFIELD LastName ");
builder.writeln(", ");
builder.insertField("MERGEFIELD CustomGreeting ");
```

**Почему**Этот шаг демонстрирует, что типичный документ Word может содержать поля слияния, которые необходимо переименовать.

#### Шаг 3: Определите и переименуйте поля слияния

Извлеките все начальные узлы полей, чтобы идентифицировать и переименовать поля слияния:

```java
import com.aspose.words.NodeCollection;
import com.aspose.words.NodeType;
import com.aspose.words.FieldStart;

NodeCollection fieldStarts = doc.getChildNodes(NodeType.FIELD_START, true);
for (FieldStart fieldStart : (Iterable<FieldStart>) fieldStarts) {
    if (fieldStart.getFieldType() == FieldType.FIELD_MERGE_FIELD) {
        MergeField mergeField = new MergeField(fieldStart);
        // Добавьте «_Renamed» к имени каждого поля слияния.
        mergeField.setName(mergeField.getName() + "_Renamed");
    }
}
```

**Почему**: Этот цикл ищет все поля слияния в документе и добавляет суффикс к их именам, обеспечивая их уникальную идентификацию.

#### Шаг 4: Сохраните документ

Наконец, сохраните обновленный документ с переименованными полями:

```java
doc.save("YOUR_DOCUMENT_DIRECTORY/RenameMergeFields.Rename.docx");
```

**Почему**: Сохранение документа гарантирует, что все изменения будут сохранены и могут быть использованы в последующих операциях.

### Класс Merge Field Facade для управления полями документа Word

В этом разделе представлен вспомогательный класс `MergeField` для упрощения процесса манипуляции полями. Класс предоставляет методы для получения или установки имен полей, обновления кодов полей и обеспечения согласованности между узлами документа.

#### Ключевые методы:

- **получитьИмя()**Возвращает текущее имя поля слияния.
  
  ```java
  String fieldName = mergeField.getName();
  ```

- **setName(строковое значение)**: Устанавливает новое имя для поля слияния.

  ```java
  mergeField.setName("NewFieldName");
  ```

- **updateFieldCode(строка fieldName)**: Обновляет код поля, чтобы отразить новое имя поля, гарантируя, что все ссылки в документе будут согласованы.
