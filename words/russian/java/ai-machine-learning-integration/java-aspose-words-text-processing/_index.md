# Мастер обработки текста на Java: использование Aspose.Words и моделей ИИ

Это руководство проведет вас через использование Aspose.Words для Java вместе с моделями ИИ, такими как OpenAI GPT-4 и Google Gemini 15 Flash для резюмирования и перевода текста.

**Что вы узнаете:**
- Настройка Aspose.Words с Maven или Gradle
- Реализация реферирования текста с использованием моделей ИИ

- Лучшие практики интеграции этих инструментов в приложения Java

Прежде чем приступить к внедрению, убедитесь, что у вас есть все необходимое.

## Предпосылки

Убедитесь, что вы соответствуете следующим требованиям:

### Требуемые библиотеки и версии
- **Aspose.Words для Java:** Версия 25.3 или более поздняя.
- **Комплект разработчика Java (JDK):** Установленный JDK (предпочтительно версии 8 или выше).
- **Инструменты сборки:** Maven или Gradle, в зависимости от ваших предпочтений.

### Требования к настройке среды
- Подходящая интегрированная среда разработки (IDE), например IntelliJ IDEA или Eclipse.
- Доступ к службам OpenAI и Google AI, для которых могут потребоваться ключи API.

### Необходимые знания
- Базовые знания программирования на Java.
- Знакомство с работой с внешними библиотеками в проекте Java.

## Настройка Aspose.Words

Чтобы начать использовать Aspose.Words для Java, добавьте необходимые зависимости в конфигурацию сборки.

### Зависимость Maven

Добавьте этот фрагмент в свой `pom.xml`:

```xml
<dependency>
  <groupId>com.aspose</groupId>
  <artifactId>aspose-words</artifactId>
  <version>25.3</version>
</dependency>
```

### Зависимость Gradle

Включите это в свой `build.gradle` файл:

```gradle
implementation 'com.aspose:aspose-words:25.3'
```

### Приобретение лицензии

Aspose.Words требует лицензию для полной функциональности. Вы можете приобрести:
- А **бесплатная пробная версия** для тестирования функций.
- А **временная лицензия** для расширенной оценки.
- А **лицензия на покупку** для использования в производстве.

Для настройки инициализируйте библиотеку и установите лицензию:

```java
License license = new License();
license.setLicense("path/to/your/license/file");
```

## Руководство по внедрению

### Резюмирование текста с помощью моделей ИИ

Резюмирование текста может быть бесценным при работе с объемными документами. Вот как это реализовать с помощью модели GPT-4 OpenAI.

#### Шаг 1: Инициализация документа и модели

Начните с загрузки документа и настройки модели ИИ:

```java
document = new Document(getMyDir() + "Big document.docx");
IAiModelText model = ((OpenAiModel) AiModel.create(AiModelType.GPT_4_O_MINI).withApiKey(apiKey))
        .withOrganization("YourOrg")
        .withProject("YourProject");
```

#### Шаг 2: Настройка параметров резюмирования

Укажите длину резюме и создайте `SummarizeOptions` объект:

```java
SummarizeOptions options = new SummarizeOptions();
options.setSummaryLength(SummaryLength.SHORT);
Document summarizedDoc = model.summarize(document, options);
```

#### Шаг 3: Сохраните сводку

Сохраните ваш обобщенный документ в желаемом месте:

```java
summarizedDoc.save(getArtifactsDir() + "AI.AiSummarize.One.docx");
```

### Перевод текста с использованием моделей ИИ

Легко переводите документы на разные языки с помощью модели Gemini от Google.

#### Шаг 1: Загрузите и подготовьте документ

Подготовьте документ к переводу:

```java
document = new Document(getMyDir() + "Document.docx");
IAiModelText translator = (IAiModelText) AiModel.create(AiModelType.GEMINI_15_FLASH).withApiKey(apiKey);
```

#### Шаг 2: Выполнение перевода

Перевести документ на арабский язык:

```java
Document translatedDoc = translator.translate(document, Language.ARABIC);
translatedDoc.save(getArtifactsDir() + "AI.AiTranslate.docx");
```
