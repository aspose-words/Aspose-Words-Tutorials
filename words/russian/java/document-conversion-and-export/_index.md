## Быстрые ответы
- **Каков основной сценарий использования?** Добавление водяных знаков при конвертации и экспорте документов.

- **Поддерживается ли Java 8+?** Абсолютно, библиотека работает с Java 8 и более новыми версиями.

## Что такое **how to add watermark**?
Добавление водяного знака означает наложение текста или изображения на каждую страницу документа. Это обычно используется для пометок «Confidential», «Draft» или в целях брендинга. С помощью Aspose.Words for Java вы можете программно вставлять водяные знаки во время процесса **convert docx to pdf** или любой другой операции экспорта.

- **Cross‑format support** – Один и тот же код работает с DOCX, PDF, HTML и другими форматами.  
- **Performance** – Высокоскоростная конвертация и наложение водяных знаков даже на больших файлах.  
- **Customization** – Регулируйте непрозрачность, вращение, цвет и позицию «на лету».

## Часто задаваемые вопросы

**Q: Как добавить текстовый водяной знак в PDF, сгенерированный из DOCX файла?**  
A: Используйте `DocumentBuilder.insertWatermarkText("Your Text")` перед вызовом `document.save("output.pdf")`. Это работает во время шага **convert docx to pdf**.

**Q: Можно ли добавить водяной знак только на определённые страницы?**  
A: Да. Вставляйте водяной знак внутри цикла, который проверяет свойства `PageSetup` для нужных страниц.

**Q: Можно ли комбинировать генерацию штрих‑кода с наложением водяного знака?**  
A: Конечно. Сгенерируйте изображение штрих‑кода, затем вставьте его как графический водяной знак с помощью `DocumentBuilder.insertWatermarkImage(barcodeImage)`.

**Q: Какие форматы поддерживают водяные знаки?**  
A: Все форматы экспорта, поддерживаемые Aspose.Words (PDF, DOCX, HTML, ODT и др.), сохраняют водяные знаки.

**Q: Нужна ли отдельная лицензия для функций штрих‑кода или диаграмм?**  
A: Нет. Все функции включены в стандартную лицензию Aspose.Words for Java.
