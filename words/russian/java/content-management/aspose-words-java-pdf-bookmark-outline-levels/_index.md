# Создание вложенных закладок PDF в Java с Aspose.Words

## Введение
В этом руководстве вы узнаете, как **создавать вложенные закладки PDF** с помощью Aspose.Words для Java, что позволит генерировать PDF с закладками и сохранять закладки Word в PDF с чистой иерархией оглавления. Мы пройдем настройку библиотеки, построение вложенных закладок, назначение уровней оглавления и экспорт окончательного PDF.

**Что вы узнаете**
- Установить и лицензировать Aspose.Words для Java
- Создать вложенные закладки внутри документа Word
- Настроить уровни оглавления закладок для структурированной навигации
- Сохранить документ в PDF, сохраняющий иерархию закладок

### Предварительные требования
- **Библиотеки и зависимости**: Aspose.Words для Java (25.3 или новее)  
- **Среда**: JDK 8+ и IDE, например IntelliJ IDEA или Eclipse  
- **Базовые навыки**: Знание Java, Maven или Gradle и понятие закладок PDF

  Это построение иерархии закладок, где дочерние закладки находятся внутри родительских, как главы и подп главы в книге.

## Что такое “создание вложенных закладок PDF”?
Вложенные закладки PDF — это древовидная структура, отображаемая в панелях навигации PDF‑просмотрщиков. Они позволяют читателям переходить непосредственно к разделам, подразделам или конкретным абзацам, улучшая удобство использования документа — особенно для юридических контрактов, технических отчетов или электронных книг.

## Почему использовать Aspose.Words для уровней оглавления закладок?
Aspose.Words предоставляет удобный API для определения закладок во время построения документа, затем автоматически сопоставляет эти закладки с записями оглавления PDF. Это устраняет необходимость ручной пост‑обработки и гарантирует, что навигация PDF отражает исходную иерархию Word.

## Руководство по реализации
Мы разделим реализацию на две логические части: создание вложенных закладок и настройку их уровней оглавления.

### Создание вложенных закладок
**Обзор** – В этом разделе показано, как внедрять иерархические закладки непосредственно в документ Word.

#### Шаг 1: Инициализация документа и Builder
```java
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```
`DocumentBuilder` предоставляет удобный способ вставлять текст, таблицы и закладки.

#### Шаг 2: Вставка основных и вложенных закладок
```java
builder.startBookmark("Bookmark 1");
builder.writeln("Text inside Bookmark 1.");
```
Теперь добавьте дочернюю закладку внутри первой:

```java
builder.startBookmark("Bookmark 2");
builder.writeln("Text inside Bookmark 1 and 2.");
builder.endBookmark("Bookmark 2"); // End the nested bookmark
```

Закройте внешнюю закладку:

```java
builder.endBookmark("Bookmark 1");
```

#### Шаг 3: Добавление отдельной закладки верхнего уровня
```java
builder.startBookmark("Bookmark 3");
builder.writeln("Text inside Bookmark 3.");
builder.endBookmark("Bookmark 3");
```
Вы можете повторять эти шаги, чтобы построить иерархию любой глубины.

### Настройка уровней оглавления закладок
**Обзор** – После создания закладок определите их уровни оглавления, чтобы PDF‑просмотрщики отображали их правильно.

#### Шаг 1: Настройка PdfSaveOptions
```java
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
BookmarksOutlineLevelCollection outlineLevels = pdfSaveOptions.getOutlineOptions().getBookmarksOutlineLevels();
```
`PdfSaveOptions` управляет тем, как документ Word преобразуется в PDF.

#### Шаг 2: Назначение уровней каждой закладке
```java
outlineLevels.add("Bookmark 1", 1);
outlineLevels.add("Bookmark 2", 2); // Nested under Bookmark 1
outlineLevels.add("Bookmark 3", 3);
```
Уровень 1 отображается как запись верхнего уровня, уровень 2 — как дочерняя, и т.д.

#### Шаг 3: Сохранение документа в PDF
```java
doc.save(getArtifactsDir() + "BookmarksOutlineLevelCollection.BookmarkLevels.pdf", pdfSaveOptions);
```
Полученный PDF покажет панель закладок из трёх уровней, отражающую заданную вами структуру.

### Советы по устранению неполадок
- **Отсутствующие закладки** – Убедитесь, что каждый `startBookmark` имеет соответствующий `endBookmark`.  
- **Неправильная иерархия** – Проверьте номера уровней оглавления; дочерняя закладка должна иметь более высокий уровень, чем её родитель.  
- **Ошибки лицензии** – Убедитесь, что файл лицензии загружен до вызова любых API Aspose; иначе вы увидите водяные знаки оценки.

## Часто задаваемые вопросы

**В: Как установить Aspose.Words для Java?**  
О: Добавьте зависимость Maven или Gradle, показанную выше, затем загрузите файл лицензии во время выполнения.

**В: Можно ли использовать закладки без установки уровней оглавления?**  
О: Да, но навигация PDF будет плоской, что усложнит читателям понимание иерархии документа.

**В: Есть ли ограничение на глубину вложения закладок?**  
О: Технически нет, но держите иерархию разумной (3‑5 уровней), чтобы обеспечить читаемость в большинстве PDF‑просмотрщиков.

**В: Как Aspose.Words обрабатывает очень большие документы?**  
О: Он потоково обрабатывает содержимое и предлагает `optimizeResources()` для снижения потребления памяти, однако всё равно следует тестировать с вашими конкретными размерами файлов.
