## Руководство по внедрению

### Удалить неиспользуемые стили и списки
#### Обзор
Эта функция помогает навести порядок в документах Word, удаляя все неиспользуемые стили и списки, уменьшая размер файла и улучшая управляемость.
##### Шаг 1: Создание и добавление пользовательских стилей
Начните с создания `Document` экземпляр и добавление пользовательских стилей:
```java
import com.aspose.words.Document;
import com.aspose.words.StyleType;

// Создайте новый экземпляр документа.
Document doc = new Document();

// Добавьте пользовательские стили в документ.
doc.getStyles().add(StyleType.LIST, "MyListStyle1");
doc.getStyles().add(StyleType.LIST, "MyListStyle2");
```
##### Шаг 2: Использование стилей в документе
Использовать `DocumentBuilder` чтобы применить эти стили и отметить их как используемые:
```java
import com.aspose.words.DocumentBuilder;

// Используйте DocumentBuilder для применения стилей.
DocumentBuilder builder = new DocumentBuilder(doc);
builder.getFont().setStyle(doc.getStyles().get("MyParagraphStyle1"));
builder.writeln("Hello world!");
```
##### Шаг 3: Настройка параметров очистки
Настраивать `CleanupOptions` чтобы указать, какие элементы следует очистить:
```java
import com.aspose.words.CleanupOptions;

// Настройте параметры очистки.
CleanupOptions cleanupOptions = new CleanupOptions();
cleanupOptions.setUnusedLists(true);
cleanupOptions.setUnusedStyles(true);
```
##### Шаг 4: Выполните очистку
Выполните операцию очистки, чтобы удалить неиспользуемые стили и списки:
```java
// Выполните операцию очистки.
doc.cleanup(cleanupOptions);
```
### Удалить дубликаты стилей
#### Обзор
Устраните дублирующиеся стили в документе, чтобы сохранить согласованность и сократить избыточность.
##### Шаг 1: Добавьте дублирующие стили
Создать новый `Document` и добавьте идентичные стили под разными именами:
```java
import com.aspose.words.Style;
import java.awt.Color;

// Создайте еще один экземпляр документа.
Document doc = new Document();

// Добавьте два одинаковых стиля с разными названиями.
Style myStyle = doc.getStyles().add(StyleType.PARAGRAPH, "MyStyle1");
myStyle.getFont().setSize(14.0);
```
##### Шаг 2: Применение стилей
Использовать `DocumentBuilder` для применения этих стилей:
```java
// Примените оба стиля к разным абзацам.
builder.getParagraphFormat().setStyleName(myStyle.getName());
builder.writeln("Hello world!");
```
##### Шаг 3: Настройте параметры очистки для дубликатов
Настраивать `CleanupOptions` для удаления дубликатов:
```java
// Настройте параметры очистки для удаления дубликатов стилей.
cleanupOptions.setDuplicateStyle(true);
```
##### Шаг 4: Выполните очистку
Выполните операцию очистки для устранения дубликатов:
```java
// Выполните операцию очистки.
doc.cleanup(cleanupOptions);
```
