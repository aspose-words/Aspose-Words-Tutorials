```java
public class FeatureDisplayAsposeWordsVersion {

        // Код будет добавлен здесь

```

**Шаг 3: Получите название и версию продукта**

Внутри `main` метод, использование `BuildVersionInfo` чтобы получить название и версию продукта:
