# Настройка номеров страниц и создание оглавления в Aspose.Words for Java

В этом руководстве вы узнаете, как **настраивать номера страниц** и **вставлять оглавление** (TOC) с помощью Aspose.Words for Java. Хорошо структурированное оглавление упрощает навигацию по длинным документам, а точная настройка выравнивания номеров страниц обеспечивает профессиональный вид для ваших читателей. Мы пройдём процесс создания документа, настройки стилей оглавления и изменения табуляций, чтобы номера страниц располагались точно там, где вам нужно.

## Быстрые ответы
- **Что означает «настройка номеров страниц»?** Изменение табуляций, которые выравнивают номера страниц в оглавлении.  
- **Можно ли автоматически вставить оглавление?** Да — используйте класс `FieldToc`.  
- **Нужна ли лицензия для запуска кода?** Бесплатная пробная версия подходит для разработки; для продакшна требуется лицензия.  
- **Какая версия Aspose поддерживается?** Примеры работают с последним выпуском Aspose.Words for Java.  
- **Можно ли настраивать стили оглавления?** Конечно — вы можете менять шрифты, жирность и многое другое.

## Что такое оглавление в Aspose.Words?
Оглавление — это поле, которое сканирует документ в поисках стилей заголовков (например, Heading 1, Heading 2) и генерирует список записей с номерами страниц. Aspose.Words позволяет вставлять это поле программно и полностью контролировать его внешний вид.

## Почему стоит настраивать номера страниц в оглавлении?
Настройка табуляций даёт точный контроль над тем, где отображаются номера страниц, что важно для:

- Поддержания чистого, выровненного по колонкам макета.  
- Соответствия корпоративным руководствам по стилю.  
- Повышения читаемости печатных и цифровых документов.

## Предварительные требования
- Aspose.Words for Java, добавленный в ваш проект (Maven/Gradle).  
- Базовое знакомство с синтаксисом Java.  

## Пошаговое руководство

### Шаг 1: Создание нового документа
Сначала создайте пустой объект `Document`, который будет содержать ваш контент и оглавление.

```java
Document doc = new Document();
```

### Шаг 2: Настройка стилей оглавления
Можно изменить внешний вид каждого уровня оглавления. В этом примере мы делаем записи первого уровня жирными — частый запрос к форматированию.

```java
doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_1).getFont().setBold(true);
```

### Шаг 3: Добавление контента в документ
Вставьте заголовки (например, `Heading1`, `Heading2`) и обычные абзацы. Поле оглавления позже автоматически обнаружит эти заголовки. *(Код опущен для краткости — основной фокус на генерации оглавления.)*

### Шаг 4: Вставка поля оглавления
Разместите оглавление там, где нужно — обычно в начале документа.

```java
// Insert a TOC field at the desired location in your document.
FieldToc fieldToc = new FieldToc();
doc.getFirstSection().getBody().getFirstParagraph().appendChild(fieldToc);
```

### Шаг 5: Сохранение документа
Сохраните документ на диск. Вы можете выбрать любой поддерживаемый формат, такой как DOCX, PDF или HTML.

```java
doc.save("your_output_path_here");
```

## Настройка табуляций в оглавлении (Настройка номеров страниц)
Если стандартная табуляция не выравнивает номера страниц так, как вам нужно, можно пройтись по всем абзацам оглавления и изменить их позиции табуляций.

```java
Document doc = new Document("Table of contents.docx");

for (Paragraph para : (Iterable<Paragraph>) doc.getChildNodes(NodeType.PARAGRAPH, true))
{
    if (para.getParagraphFormat().getStyle().getStyleIdentifier() >= StyleIdentifier.TOC_1 &&
        para.getParagraphFormat().getStyle().getStyleIdentifier() <= StyleIdentifier.TOC_9)
    {
        // Get the first tab used in this paragraph, which aligns the page numbers.
        TabStop tab = para.getParagraphFormat().getTabStops().get(0);
        
        // Remove the old tab.
        para.getParagraphFormat().getTabStops().removeByPosition(tab.getPosition());
        
        // Insert a new tab at a modified position (e.g., 50 units to the left).
        para.getParagraphFormat().getTabStops().add(tab.getPosition() - 50.0, tab.getAlignment(), tab.getLeader());
    }
}

doc.save("output.docx");
```

Теперь записи оглавления отображают номера страниц точно там, где вы хотите, придавая документу аккуратный вид.

## Распространённые проблемы и советы
- **Отсутствуют заголовки в оглавлении:** Убедитесь, что ваши заголовки используют встроенные стили (`Heading1`, `Heading2` и т.д.) или сопоставьте пользовательские стили с уровнями оглавления.  
- **Табуляция не применяется:** Проверьте, что абзац действительно принадлежит стилю оглавления (`TOC_1`‑`TOC_9`).  
- **Производительность на больших документах:** Вызовите `doc.updateFields()` после вставки оглавления, чтобы обновить записи за один проход.

## Часто задаваемые вопросы

**В: Как изменить форматирование записей оглавления?**  
О: Используйте `doc.getStyles().getByStyleIdentifier(StyleIdentifier.TOC_X)`, где *X* — уровень (1‑9), и изменяйте шрифт, цвет или параметры абзаца.

**В: Как добавить больше уровней в оглавление?**  
О: Отредактируйте переключатель `FieldToc` `\o "1-3"` (например), чтобы включить дополнительные уровни заголовков, затем обновите соответствующие стили `TOC_X`.

**В: Можно ли изменить позиции табуляций для отдельных записей оглавления?**  
О: Да — пройдитесь по абзацам, как показано в разделе «Настройка табуляций», и измените каждую табуляцию индивидуально.

**В: Можно ли генерировать оглавление в PDF?**  
О: Конечно. Сохраните документ как PDF (`doc.save("output.pdf")`) после генерации оглавления; поле будет автоматически отрисовано.

**В: Нужно ли вызывать `updateFields()` вручную?**  
О: При вставке `FieldToc` Aspose.Words обновляет его при сохранении, но вызов `doc.updateFields()` даёт мгновенный результат для отладки.
