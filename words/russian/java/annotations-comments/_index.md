## Как добавить комментарий Aspose Java?

Document представляет собой файл Word, загруженный в память. DocumentBuilder — вспомогательный класс, используемый для построения и редактирования Document. `insertComment` добавляет новый узел комментария в документ. Загрузите целевой документ с помощью `Document doc = new Document("input.docx")`, создайте `DocumentBuilder` и вызовите `insertComment("Your comment text", "Author Name", new Date())`. Эта однострочная операция вставляет полностью функциональный комментарий, включающий автора, текст и метку времени, и работает со всеми более чем 35 поддерживаемыми форматами без необходимости установки Microsoft Word.

## Как удалить аннотации Java?

Annotation — это элемент разметки, такой как комментарий, заметка или выделение. `doc.getAnnotations()` возвращает коллекцию Annotation документа. Получите коллекцию `Annotation` через `doc.getAnnotations()`, найдите аннотацию, которую нужно удалить (по ID, типу или автору), и вызовите `annotation.remove()`. `annotation.remove()` удаляет эту аннотацию из документа. Это мгновенно удаляет аннотацию, и изменение отражается при сохранении файла, позволяя выполнять чистую автоматическую очистку артефактов рецензирования.

## Как автоматизировать циклы обратной связи с Aspose.Words?

`removeAnnotation` удаляет указанную аннотацию из документа. Создайте пакетную задачу, которая загружает каждый документ, применяет `insertComment` или `removeAnnotation` по необходимости, а затем сохраняет файл в указанную выходную папку. Объединяя эти вызовы API внутри цикла, вы можете автоматически собирать ввод рецензентов, выполнять массовые обновления и генерировать финальные документы — всё в рамках единой поддерживаемой Java‑процедуры.

## Распространённые проблемы и решения

- **Комментарии не отображаются в UI** – Убедитесь, что документ открыт в просмотрщике, поддерживающем комментарии (например, Microsoft Word или предварительный просмотр Aspose.Words).  
- **Аннотации исчезают после сохранения** – Проверьте, сохраняете ли вы в формате, сохраняющем аннотации (DOCX, PDF и т.д.).  
- **Снижение производительности на больших файлах** – Используйте `Document.optimizeResources()` перед обработкой для снижения использования памяти. `Document.optimizeResources()` сжимает встроенные ресурсы, уменьшая потребление памяти.

## Часто задаваемые вопросы

**В: Можно ли добавлять комментарии в документы, защищённые паролем?**  
О: Да. Откройте документ с помощью `new LoadOptions("password")`, затем вставляйте комментарии как обычно.

**В: Влияет ли удаление аннотации на другое содержимое?**  
О: Нет. Удаление аннотации удаляет только узел разметки; окружающий текст остаётся без изменений.

**В: Можно ли экспортировать комментарии в отдельный отчёт?**  
О: Абсолютно. Пройдите по `doc.getComments()` и запишите автора, текст и дату каждого комментария в CSV или JSON файл.

**В: Какие версии Java поддерживаются?**  
О: Aspose.Words for Java работает с Java 8, 11 и более новыми LTS‑выпусками.

**В: Как обрабатывать комментарии при сохранении в PDF?**  
О: При сохранении в PDF установите `PdfSaveOptions.setExportComments(true)`, чтобы сохранить комментарии в конечном PDF. `PdfSaveOptions.setExportComments(true)` указывает PDF‑сохраняющему модулю включать комментарии в вывод.
