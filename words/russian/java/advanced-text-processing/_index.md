## Часто задаваемые вопросы

**Вопрос: Могу ли я программно изменить формат номера страницы (например, римские на арабские)?**  
Ответ: Да, вы можете установить свойство `PageNumberStyle` у поля `PageNumber` на любой поддерживаемый формат.

**Вопрос: Как пропустить нумерацию страниц на определённых страницах, например, на титульных?**  
Ответ: Вставьте поле `PageNumber` с переключателем `\* MERGEFORMAT` и установите атрибут `Suppress` в заголовке/нижнем колонтитуле для этого раздела.

**Вопрос: Можно ли автоматически сбрасывать нумерацию страниц для каждой главы?**  
Ответ: Конечно. Используйте класс `Section` для создания нового раздела для каждой главы и включите `PageSetup.RestartPageNumbering`.

**Вопрос: Обрабатывает ли API большие документы эффективно?**  
Ответ: Aspose.Words обрабатывает документы в потоковом режиме, минимизируя использование памяти даже для файлов с тысячами страниц.

**Вопрос: Какие варианты лицензирования доступны для продакшн‑использования?**  
Ответ: Вы можете начать с временной лицензии для оценки и перейти к бессрочной или подписной лицензии для коммерческого развертывания.
