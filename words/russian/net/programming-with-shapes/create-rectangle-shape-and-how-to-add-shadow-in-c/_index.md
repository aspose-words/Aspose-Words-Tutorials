# Создание прямоугольной фигуры и добавление тени в C#

Когда‑то вам нужно **создать прямоугольную фигуру** в документе Word, но вы не знаете, как добавить к ней лёгкую падающую тень? Вы не одиноки. Во многих сценариях отчётности или брендинга простой прямоугольник с мягкой полупрозрачной тенью может придать макету отполированный вид без больших усилий.

В этом руководстве мы пройдёмся по **созданию документа** с помощью Aspose.Words, затем покажем, **как добавить тень**, **применить размытие к тени** и даже **сделать тень прозрачной**. К концу вы получите готовый фрагмент C#, который за несколько минут создаст файл *.docx* с красиво затенённым прямоугольником.

## Что понадобится

- .NET 6 или новее (API также работает с .NET Framework 4.6+)
- Aspose.Words for .NET (бесплатная пробная версия подходит для этого примера)
- Редактор кода – Visual Studio, VS Code, Rider или любой другой
- Базовые знания C# – ничего сложного, просто возможность запустить консольное приложение

Если всё это у вас есть, можно сразу переходить к решению.

## Шаг 1 – Как создать документ и инициализировать холст

Первое, что нужно: пустой объект `Document`. Представьте его как чистый лист бумаги, который Aspose.Words позже превратит в файл Word.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;   // For Color

// Create a new blank document
Document doc = new Document();
```

Почему мы создаём `Document`, а не загружаем шаблон? Начинать с нуля гарантирует, что никаких скрытых стилей или секций не помешают нашему прямоугольнику. Это также сохраняет размер файла маленьким – полезная привычка, когда генерируете множество документов в цикле.

## Шаг 2 – Создание прямоугольной фигуры (ядро нашего основного ключевого слова)

Теперь мы действительно **создаём прямоугольную фигуру**. Класс `Shape` гибок: вы указываете тип (Rectangle), размеры и способ обтекания окружающим текстом.

```csharp
// Define a rectangular shape
Shape rect = new Shape(doc, ShapeType.Rectangle)
{
    Width = 200,               // Width in points (≈2.8 inches)
    Height = 100,              // Height in points (≈1.4 inches)
    WrapType = WrapType.Inline // Makes the shape behave like a character
};
```

Обратите внимание на синтаксис инициализатора объекта – он лаконичен и уменьшает риск забыть установить какое‑то свойство позже. Прямоугольник будет находиться внутри первого абзаца, который мы добавим на следующем шаге.

## Шаг 3 – Как добавить тень и настроить её внешний вид

Добавление тени – это не одна строка; есть несколько свойств, которые можно настроить. Здесь в игру вступают вторичные ключевые слова **apply blur to shadow** и **make shadow transparent**.

```csharp
// Configure the shadow
rect.Shadow.Format.Color = Color.DarkGray;   // Shadow colour
rect.Shadow.Format.BlurRadius = 5.0;         // Apply blur to shadow (points)
rect.Shadow.Format.OffsetX = 3;              // Horizontal offset
rect.Shadow.Format.OffsetY = 3;              // Vertical offset
rect.Shadow.Format.Transparency = 0.3;       // 30 % transparent (make shadow transparent)
```

Небольшое примечание о числах: `BlurRadius` = 5 даёт лёгкое размытие; увеличьте до 10 для более мягкого эффекта или уменьшите до 2 для чёткой границы. Значение `Transparency` варьируется от 0 (непрозрачная) до 1 (полностью прозрачная). Регулируйте в зависимости от контрастных требований вашего бренда.

### Pro tip

Если понадобится цветная тень (например, корпоративный синий), замените `Color.DarkGray` на `Color.FromArgb(80, 0, 120, 215)`. Первый аргумент – альфа‑канал; держите его низким для нежного эффекта.

## Шаг 4 – Вставка фигуры в документ

Когда прямоугольник и его тень готовы, мы помещаем его в первый абзац документа. Этот шаг гарантирует, что фигура окажется в самом верху файла.

```csharp
// Append the shape to the first paragraph of the first section
doc.FirstSection.Body.FirstParagraph.AppendChild(rect);
```

Почему именно первый абзац? Это безопасный вариант, который работает даже если документ полностью пуст. Если нужен конкретный участок (например, после заголовка), найдите соответствующий узел и вставьте фигуру туда.

## Шаг 5 – Сохранение файла и проверка результата

Наконец, сохраняем документ на диск. Вы можете указать любой путь, только убедитесь, что папка существует.

```csharp
// Save the document
doc.Save(@"C:\Temp\ShadowRectangle.docx");
```

Открыв *ShadowRectangle.docx* в Microsoft Word, вы увидите прямоугольник 200 × 100 пунктов с тёмно‑серой, слегка размытой, 30 % прозрачной тенью, смещённой на три пункта вправо и вниз. Эффект незаметный, но добавляет глубину плоским макетам.

## Общие варианты и граничные случаи

### Динамическое изменение цвета тени

Если ваше приложение поддерживает темы, цвет тени можно брать из конфигурационного файла:

```csharp
Color themeShadow = ColorTranslator.FromHtml(ConfigurationManager.AppSettings["ShadowColor"]);
rect.Shadow.Format.Color = themeShadow;
```

### Делание фигуры не‑встроенной

Иногда требуется, чтобы прямоугольник плавал над текстом. Переключите `WrapType` на `WrapType.Square` и задайте `RelativeHorizontalPosition` как `RelativeHorizontalPosition.Margin` для большего контроля.

```csharp
rect.WrapType = WrapType.Square;
rect.RelativeHorizontalPosition = RelativeHorizontalPosition.Margin;
rect.Left = 72; // 1 inch from the left margin
```

### Обработка нескольких страниц

Если нужен прямоугольник на каждой странице, пройдитесь по `doc.Sections` и добавьте клонированную фигуру в первый абзац каждой секции. Не забудьте вызвать `rect.Clone(true)`, чтобы также скопировать настройки тени.
