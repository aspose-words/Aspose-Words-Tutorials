# Создание прямоугольной фигуры в Word – Полное руководство Aspose.Words

Когда‑то вам нужно **создать прямоугольную фигуру** в документе Word, но вы не знали, с чего начать? Вы не одиноки — многие разработчики сталкиваются с этой проблемой при автоматизации отчётов или счетов. В этом руководстве мы пройдём через полностью готовый к запуску пример, показывающий, как **добавить фигуру в Word**, применить лёгкую тень и управлять прозрачностью фигуры, используя Aspose.Words для .NET.

К концу руководства у вас будет файл `.docx`, содержащий чистый прямоугольник с аккуратной тенью — идеально для брендинга, выделения или просто для более профессионального вида вашего документа. Никаких внешних инструментов, только несколько строк C#.

## Что вам понадобится

- **Aspose.Words for .NET** (последняя версия на начало 2026 года). Вы можете получить её из NuGet (`Install-Package Aspose.Words`).
- Среда разработки .NET (Visual Studio, Rider или VS Code с расширением C#).
- Базовое знакомство с синтаксисом C# — ничего сложного, только обычные `using`‑операторы и создание объектов.

Если всё это уже есть, отлично — приступаем.

## Создание прямоугольной фигуры — основные шаги

Ниже приведён полный исходный код. Скопируйте‑вставьте его в новый консольный проект, нажмите **F5**, и файл `ShadowDemo.docx` появится в указанной папке.

```csharp
using Aspose.Words;
using Aspose.Words.Drawing;
using System.Drawing;   // Needed for Color

// Step 1: Create a new blank document.
Document document = new Document();

// Step 2: Insert a rectangle shape and define its size.
Shape rectangleShape = new Shape(document, ShapeType.Rectangle)
{
    Width  = 200,   // Width in points (≈2.78 inches)
    Height = 100    // Height in points (≈1.39 inches)
};

// Step 3: Apply a shadow with fine‑grained control over its appearance.
rectangleShape.Shadow = new Shadow
{
    BlurRadius   = 5.0,                     // Softness of the shadow edge
    Distance     = 4.0,                     // How far the shadow is offset
    Direction    = 45,                      // Angle of the offset (degrees)
    Color        = Color.Gray,              // Shadow colour
    Transparency = 0.2,                     // Opacity (0 = opaque, 1 = fully transparent)
    Spread       = 0.3                      // Size of the shadow spread
};

// Step 4: Add the shape to the first paragraph of the document.
document.FirstSection.Body.FirstParagraph.AppendChild(rectangleShape);

// Step 5: Save the document with the shadowed shape.
document.Save("ShadowDemo.docx");
```

### Почему это работает

- **`Document`** — точка входа; представляет весь файл Word.
- **`Shape`** с `ShapeType.Rectangle` сообщает Aspose, что нам нужен прямоугольный объект рисования.
- Установка **`Width`** и **`Height`** задаёт фигуре определённый размер; иначе она будет иметь крошечный заполнитель.
- Объект **`Shadow`** позволяет точно настроить каждый визуальный аспект: размытие, расстояние, направление, цвет, прозрачность и распространение. Это и есть суть *apply shadow to shape*.
- Наконец, **`AppendChild`** вставляет фигуру в первый абзац документа, что является самым простым способом *add shape to Word* без работы с таблицами или колонтитулами.

Когда вы откроете `ShadowDemo.docx`, вы увидите серый прямоугольник, удобно расположенный в документе, его тень наклонена вниз‑вправо под углом 45°. Тень не представляет собой сплошной блок; радиус размытия смягчает края, а прозрачность делает её похожей на естественную падающую тень, а не на резкое наложение.

## Добавление фигуры в документ Word — варианты размещения

В примере используется **первый абзац**, потому что это самый быстрый способ увидеть результат. В реальных сценариях вы можете захотеть:

- Вставить фигуру в конкретный **section** или **header/footer**.
- Поместить её в **ячейку таблицы** для выравнивания с табличными данными.
- Обернуть её параметрами **text wrapping** (например, `WrapType.Square`), чтобы окружающий текст обтекал прямоугольник.

Ниже небольшая вариация, помещающая фигуру в новый абзац с пользовательским стилем:

```csharp
Paragraph para = new Paragraph(document);
para.ParagraphFormat.StyleIdentifier = StyleIdentifier.Heading2;
para.AppendChild(rectangleShape);
document.FirstSection.Body.AppendChild(para);
```

*Совет профессионала:* Всегда добавляйте фигуру **после** настройки её свойств; иначе может потребоваться вызвать `UpdateLayout` для обновления визуального отображения.

## Применение тени к фигуре — тонкая настройка внешнего вида

Тени могут значительно изменить эстетику документа. Класс `Shadow` раскрывает несколько свойств:

| Свойство      | Что контролирует                                   | Типичные значения |
|---------------|----------------------------------------------------|-------------------|
| `BlurRadius`  | Мягкость краёв тени                                 | 2.0 – 10.0        |
| `Distance`    | Насколько тень смещена от фигуры                    | 1.0 – 8.0         |
| `Direction`   | Угол в градусах (0 = влево, 90 = вверх)            | 0 – 360           |
| `Color`       | Цвет тени (любой `System.Drawing.Color`)           | Gray, Black, Custom |
| `Transparency`| Непрозрачность (0 = полностью непрозрачна, 1 = невидима) | 0.0 – 0.5         |
| `Spread`      | Расширение тени до применения размытия              | 0.0 – 1.0         |

Если вам нужен **тонкий, профессиональный вид**, держите `BlurRadius` около 4‑6 и `Transparency` около 0.2, как в коде выше. Для **драматического эффекта** увеличьте `Distance` до 6, задайте `Direction` 135°, и уменьшите `Transparency` до 0.05.

## Установка прозрачности фигуры и распространения тени

Прозрачность относится не только к тени; вы также можете сделать сам прямоугольник частично прозрачным:

```csharp
rectangleShape.FillColor = Color.LightBlue;
rectangleShape.Transparency = 0.3; // 30% transparent fill
```

Сочетание полупрозрачного заполнения с мягкой тенью часто создаёт современный UI‑вид — отлично подходит для панелей мониторинга или дизайн‑макетов, встроенных в отчёты.

### Крайние случаи, о которых стоит помнить

1. **Старые версии Word** (до 2007) не поддерживают некоторые свойства тени. Если вы целитесь в файлы `.doc`, рассмотрите упрощение тени (например, установить `BlurRadius` в 0).
2. **Экран с высоким DPI** может отобразить тень немного иначе. Тестируйте в целевой среде, если важна визуальная точность.
3. **Перекрывающиеся фигуры** — Aspose рендерит тени в порядке их добавления. Вставляйте фигуры от задних к передним, чтобы избежать нежелательного перекрытия.

## Сохранение и проверка результата

Метод `Document.Save` автоматически определяет формат вывода по расширению файла. Для **`.docx`** вы получаете формат Open XML, который понимают большинство современных процессоров Word. Если нужен **PDF** с тем же визуальным оформлением, просто измените расширение:

```csharp
document.Save("ShadowDemo.pdf");
```

Открытие сгенерированного `ShadowDemo.docx` (или `ShadowDemo.pdf`) должно показать чистый **прямоугольник с тенью**, подтверждая, что вы успешно *create rectangle shape* и *apply shadow to shape* с помощью Aspose.Words.

## Часто задаваемые вопросы

**В: Можно ли использовать другую фигуру, например, эллипс?**  
О: Конечно. Замените `ShapeType.Rectangle` на `ShapeType.Ellipse` (или любой другой элемент перечисления `ShapeType`). Свойства тени останутся теми же.

**В: Как сделать прямоугольник кликабельным?**  
О: Вы можете назначить гиперссылку фигуре:

```csharp
rectangleShape.Href = "https://example.com";
```

**В: Работает ли это на .NET 6+?**  
О: Да. Aspose.Words 23.11 и более новые версии полностью поддерживают .NET 6, .NET 7 и .NET 8. Просто подключите соответствующий пакет NuGet.

**В: Как изменить цвет тени, чтобы он соответствовал моему бренду?**  
О: Используйте любой `System.Drawing.Color`, который вам нужен:

```csharp
rectangleShape.Shadow.Color = Color.FromArgb(255, 30, 144, 255); // DodgerBlue
```
