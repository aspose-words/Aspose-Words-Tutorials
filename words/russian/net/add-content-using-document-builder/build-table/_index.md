# Создайте документ Word с таблицей с помощью Aspose.Words
  
В этом примере показано, как создать документ Word с таблицей с помощью библиотеки Aspose.Words для .NET. Код инициализирует новый документ, создает таблицу с двумя строками и форматирует ячейки с различными стилями и ориентациями. Вы научитесь управлять содержимым ячеек, выравниванием и программно создавать структурированный документ.

## Инструкция по установке:  
1. Установите Aspose.Words для .NET:
   - Использование менеджера пакетов NuGet:
     - Откройте свой проект в Visual Studio.
     - Щелкните правой кнопкой мыши проект в обозревателе решений и выберите «Управление пакетами NuGet».
     - Найдите «Aspose.Words» и нажмите «Установить».

   - Или воспользуйтесь консолью диспетчера пакетов:
     ```
     Install-Package Aspose.Words
     ```
