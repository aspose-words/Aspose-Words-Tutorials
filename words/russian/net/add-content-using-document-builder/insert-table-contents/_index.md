# Создать Word document with Table of Context
Этот пример показывает, как использовать Aspose.Words для .NET для создания документа Word, который включает интерактивную таблицу содержания (TOC), созданную с помощью класса документа Builder. Код иллюстрирует, как вставить заголовки различных уровней, форматировать их и сохранить полученный документ как файл .docx. Следуйте за указанными ниже шагами для легкой реализации.

## Инструкции по установке
1. Предпосылки: Убедитесь, что на вашей машине установлен .NET.
2. Установите азбуку. Заголовки: 
   * Использовать Konsole менеджер пакета NuGet:
     ```
     Install-Package Aspose.Words
     ```
   * Или поискать "Позиции. Заголовки" в GUI менеджера пакета NuGet.
