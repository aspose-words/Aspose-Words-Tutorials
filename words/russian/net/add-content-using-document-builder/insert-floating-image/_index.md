# Создать документ Word с плавающим изображением

В этом примере показано, как создать документ Word с помощью Aspose.Words для .NET и вставить изображение с плавающим макетом. Приведенный код инициализирует Document и DocumentBuilder, вставляет изображение с указанными размерами и позиционированием и, наконец, сохраняет документ. Это руководство идеально подходит для разработчиков, желающих улучшить свои возможности создания документов с помощью Aspose.Words.

## Инструкция по установке
1. Установите Aspose.Words: Добавьте пакет Aspose.Words NuGet в свой проект .NET. Это можно сделать, выполнив следующую команду в консоли диспетчера пакетов:
   ```
   Install-Package Aspose.Words
   ```

2. Добавьте требуемые пространства имен: убедитесь, что ваш файл кода включает следующие пространства имен:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
