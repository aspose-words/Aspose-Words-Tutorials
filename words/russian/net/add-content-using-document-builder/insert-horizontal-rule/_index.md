## Инструкция по установке
1. Установите Aspose.Words: используйте диспетчер пакетов NuGet в Visual Studio.
   - Откройте свой проект в Visual Studio.
   - Щелкните правой кнопкой мыши по вашему проекту в обозревателе решений.
   - Выберите «Управление пакетами NuGet».
   - Искать `Aspose.Words` и установите его.

2. Добавьте директиву Using: в свой файл кода включите следующую директиву для использования классов Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
