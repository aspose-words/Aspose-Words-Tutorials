# Вставить в word document inline изображение с помощью азпозиции.
Этот пример показывает, как вставить в документ Word, используя библиотеку Aspose.Words для .NET.`Бюллетень документации`Класс, вы можете легко разместить и вставить изображения в текущем месте курсора в вашем документе. Это руководство обеспечивает простую реализацию для того, чтобы вы могли начать работу с изображением в документах Word.

## Инструкции по установке
1. Установите азбуку. Заголовки для .NET:
   * Использовать менеджер пакета NuGet в Visual Studio.
   * Запустить следующую команду в консоле менеджера пакета:
     ```
     Install-Package Aspose.Words
     ```

2. Установите свой проект:
   * Создать новый проект .NET или открыть существующий.
   * Добавить необходимые указания в верхнюю часть файла кода:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
