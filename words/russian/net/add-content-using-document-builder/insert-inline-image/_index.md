# Вставка встроенного изображения в документ Word с помощью Aspose.Words
В этом примере показано, как вставить встроенное изображение в документ Word с помощью библиотеки Aspose.Words для .NET. Используя `DocumentBuilder` class, вы можете легко позиционировать и вставлять изображения в текущее местоположение курсора в вашем документе. Это руководство предоставляет простую реализацию, которая поможет вам начать работу с обработкой изображений в документах Word.

## Инструкция по установке
1. Установите Aspose.Words для .NET:
   - Используйте диспетчер пакетов NuGet в Visual Studio.
   - Выполните следующую команду в консоли диспетчера пакетов:
     ```
     Install-Package Aspose.Words
     ```

2. Настройте свой проект:
   - Создайте новый проект .NET или откройте существующий.
   - Добавьте необходимые директивы using в начало файла кода:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
