# Создайте многостраничный документ Word с помощью Aspose.Words

В этом примере показано, как создать документ Word с помощью Aspose.Words для .NET, что позволяет добавлять содержимое на несколько страниц. Он показывает, как инициализировать новый документ, использовать DocumentBuilder для вставки текста и разрывов страниц и, наконец, сохранить документ как файл .docx. Идеально подходит для разработчиков, желающих автоматизировать создание документов в своих приложениях .NET.

## Инструкция по установке
1. Установка Aspose.Words для .NET: Используйте NuGet Package Manager для установки Aspose.Words. Выполните следующую команду в консоли диспетчера пакетов:
```
Install-Package Aspose.Words
```
