# Создать многостраничный документ с помощью Aspose.Words

Этот пример показывает, как создать документ Word с использованием Aspose.Words для .NET, позволяющий добавлять контент на нескольких страницах. Он показывает, как инициировать новый документ, использовать DocumentBuilder для вставки текста и перерывов страниц, и наконец сохранить документ в виде файла .docx.

## Инструкции по установке
1. Устанавливать Aspose.Words для .NET: Используйте NuGet Пакетный менеджер для установки Aspose.Words. Используйте следующую команду в консоли Пакетный менеджер:
```
Install-Package Aspose.Words
```
