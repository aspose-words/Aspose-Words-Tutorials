# Вставьте пузырьковую диаграмму в Word с помощью Aspose.Words

Легко добавьте пузырьковую диаграмму в документ Word с помощью Aspose.Words для .NET! Это руководство проведет вас через создание нового документа, вставку пузырьковой диаграммы, добавление ряда данных со значениями X, Y и размера, а также сохранение документа. С Aspose.Words программная обработка документов Word никогда не была проще.

## Инструкция по установке  

1. Загрузить Aspose.Words для .NET  
   Получить библиотеку из [Страница загрузки Aspose.Words](https://releases.aspose.com/words/net/).  

2. Установить через менеджер пакетов NuGet  
   - Откройте проект Visual Studio.  
   - Перейдите в Инструменты > Диспетчер пакетов NuGet > Управление пакетами NuGet для решения.  
   - Искать `Aspose.Words` на вкладке «Обзор».  
   - Выберите пакет и нажмите «Установить».  

3. Установка через консоль диспетчера пакетов NuGet  
   Выполните следующую команду в консоли диспетчера пакетов:  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Добавить лицензию  
   Для полной функциональности примените лицензию. Вы можете получить временную лицензию от [Страница временной лицензии Aspose](https://purchase.aspose.com/temporary-license/).
