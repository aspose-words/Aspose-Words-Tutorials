# Вставка фигур в документы Word с помощью Aspose.Words
В этом руководстве показано, как использовать Aspose.Words для .NET для вставки и управления фигурами, такими как текстовые поля, в документах Word. Вы узнаете, как позиционировать фигуры абсолютно или относительно, устанавливать углы поворота и сохранять документ в формате ISO 29500:2008 Transitional DOCX с помощью `OoxmlSaveOptions`. Этот пример упрощает работу с фигурами в документах Word, позволяя разработчикам с легкостью создавать визуально привлекательный контент.

## Инструкция по установке  
1. Загрузите Aspose.Words для .NET:  
   Получите последнюю версию Aspose.Words для .NET с сайта [страница загрузки](https://releases.aspose.com/words/net/).  

2. Установить через NuGet:  
   Откройте свой проект в Visual Studio и установите библиотеку с помощью диспетчера пакетов NuGet:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Добавить ссылку на пространство имен:  
   Включите в свой код пространство имен для Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Активируйте лицензию (необязательно):  
   Если у вас есть действующая лицензия, примените ее в своем проекте, чтобы избежать ограничений. Вы можете [купить лицензию](https://purchase.aspose.com/buy) или получить [временная лицензия](https://purchase.aspose.com/temporary-license/).
