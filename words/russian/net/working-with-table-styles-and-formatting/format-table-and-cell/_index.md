# Создать Word document with Tribunal Tables and Shading ( &lt; &lt; Вертикальный документ &gt; &gt; ) и &lt; &lt; Обычный стол &gt; &gt; ( &lt; &lt; Границы &gt; &gt; ) и &lt; &lt; Шединг &gt; &gt;

Этот пример показывает, как использовать Aspose.Words для .NET для создания документа Word, который включает таблицу с адаптированной границей и стиля затенения для своих клеток. Вы научитесь инициализировать документ, устанавливать границы таблиц, применять фоновые цвета к отдельным ячейкам и сохранить документ в формате DOCX.

## Инструкции по установке
1. Загрузка Позиции. Установки для .NET:[Предложить веб- сайт](https://www.aspose.com)и загрузить последнюю версию Aspose.Words для .NET.
2. Добавить ссылку: в ваш проект .NET добавить ссылку на файл Aspose.Words DLL.
3. Установите через NuGet: В качестве альтернативы, вы можете установить пакет с помощью менеджера пакета NuGet со следующей командой:
   ```
   Install-Package Aspose.Words
   ```
4. Добавить в свой код следующую директиву:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   using System.Drawing;
   ```
