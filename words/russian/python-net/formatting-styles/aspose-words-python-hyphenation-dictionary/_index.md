## Руководство по внедрению

### Регистрация словаря переносов

#### Обзор
Регистрация словаря позволяет Aspose.Words применять правила переноса, зависящие от локали, поддерживая поток текста в многоязычных настройках.

#### Пошаговый процесс

**1. Укажите каталоги**

Определите пути для входного документа и выходного каталога:

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY'
arartifacts_directory = 'YOUR_OUTPUT_DIRECTORY'
```

**2. Зарегистрируйте словарь**

Используйте Aspose.Words для регистрации словаря переносов для локали «de-CH».

```python
aw.Hyphenation.register_dictionary('de-CH', document_directory + 'hyph_de_CH.dic')
```
*Параметры:*
- `'de-CH'`: Идентификатор локали.
- `document_directory + 'hyph_de_CH.dic'`: Путь к файлу словаря переносов.

**3. Подтвердите регистрацию**

Убедитесь, что словарь правильно зарегистрирован:

```python
assert aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be registered"
```

### Применение переносов

Откройте документ и сохраните его с применением переносов, используя недавно зарегистрированный словарь:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.registered.pdf')
```

### Отмена регистрации словаря переносов

#### Обзор
Отмена регистрации удаляет правила, специфичные для данного региона, и возвращается к поведению переносов по умолчанию.

**1. Отменить регистрацию словаря**

```python
aw.Hyphenation.unregister_dictionary('de-CH')
```
*Цель:* Удаляет регистрацию словаря «de-CH» для предотвращения его использования при будущей обработке документов.

**2. Подтвердите отмену регистрации**

Подтвердите, что словарь больше не активен:

```python
assert not aw.Hyphenation.is_dictionary_registered('de-CH'), "Dictionary should be unregistered"
```

### Сохранение без переносов

Откройте и сохраните документ еще раз, на этот раз без применения ранее зарегистрированных правил расстановки переносов:

```python
doc = aw.Document(document_directory + 'German text.docx')
doc.save(arartifacts_directory + 'Hyphenation.dictionary.unregistered.pdf')
```
