## Руководство по внедрению
### Создание пользовательских стилей
#### Обзор
Пользовательские стили обеспечивают единообразное форматирование всего документа без особых усилий. В этом разделе вы узнаете, как создать новый стиль с нуля.
#### Шаг 1: Определите стиль
Начните с определения свойств вашего пользовательского стиля, таких как имя, атрибуты шрифта, интервал между абзацами, границы и т. д.
```python
# Создать новый стиль в коллекции стилей документа
doc.styles.add(aw.StyleType.PARAGRAPH, "SEOStyle")
# Установить характеристики шрифта
font = doc.styles["SEOStyle"].font
font.name = "Arial"
font.size = 14
font.bold = True
# Настроить форматирование абзаца
paragraph_format = doc.styles["SEOStyle"].paragraph_format
paragraph_format.space_before = 10
paragraph_format.space_after = 10
```
#### Шаг 2: Применение стиля к тексту
Примените свой собственный стиль к определенной части документа.
```python
# Перейдите в конец документа и добавьте текст с новым стилем.
doc_builder = aw.DocumentBuilder(doc)
doc_builder.move_to_document_end()
doc_builder.write("This is a paragraph styled with SEOStyle.")
# Применить пользовательский стиль
doc_builder.current_paragraph.applied_style = doc.styles["SEOStyle"]
```
#### Шаг 3: Сохраните документ
После применения стилей сохраните документ, чтобы сохранить изменения.
```python
# Сохранить документ
doc.save("StyledDocument.docx")
```
