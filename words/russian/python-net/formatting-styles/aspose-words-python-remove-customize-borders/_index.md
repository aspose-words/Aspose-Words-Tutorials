## Руководство по внедрению

### Функция 1: Удалить все границы

#### Обзор

Эта функция позволяет вам очистить любое форматирование границ, примененное к абзацам в вашем документе. Это идеально подходит для документов, требующих единообразного стиля без отдельных границ абзацев.

#### Шаги по реализации

**Шаг 1:** Загрузить документ

```python
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Borders.docx')
```
- **Цель**: загрузить существующий документ, содержащий абзацы с границами.

**Шаг 2:** Повторяйте и очищайте границы

```python
for paragraph in doc.first_section.body.paragraphs:
    para_format = paragraph.as_paragraph().paragraph_format.borders
    para_format.clear_formatting()
```
- **Объяснение**: Этот цикл проходит по каждому абзацу, обращаясь к форматированию его границ, и очищает его. `clear_formatting()` метод удаляет все стили.

**Шаг 3:** Сохраните измененный документ

```python
doc.save('YOUR_OUTPUT_DIRECTORY/BorderCollection.RemoveAllBorders.docx')
```
- **Цель**: Сохраните изменения в новом файле в указанном каталоге.

#### Советы по устранению неполадок
- Убедитесь, что у вас есть права на запись в выходной каталог.
- Убедитесь, что путь к входному документу указан правильно и доступен.

### Функция 2: Настройка границ

#### Обзор

Эта функция демонстрирует, как итерировать границы абзацев, позволяя настраивать стиль, цвет и ширину. Это полезно, когда требуется различное оформление для разных частей документа.

#### Шаги по реализации

**Шаг 1:** Создать новый документ

```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
```
- **Цель**: Начните с пустого документа и инициализируйте DocumentBuilder для простоты использования.

**Шаг 2:** Настроить границы

```python
borders = builder.paragraph_format.borders
for border in borders:
    border.color = Color.green
    border.line_style = aw.LineStyle.WAVE
    border.line_width = 3
```
- **Объяснение**: Итерация по каждой границе формата абзаца, установка стиля зеленой волнистой линии шириной 3 пункта.

**Шаг 3:** Добавить текст и сохранить

```python
builder.writeln('Hello world!')
doc.save('YOUR_OUTPUT_DIRECTORY/BorderCollection.get_borders_enumerator.docx')
```
- **Цель**: Напишите текст, демонстрирующий изменения границ, затем сохраните документ.

#### Советы по устранению неполадок
- Если границы не отображаются должным образом, проверьте настройки стиля линии и цвета.
- Обязательно сохраняйте документ после внесения всех изменений.
