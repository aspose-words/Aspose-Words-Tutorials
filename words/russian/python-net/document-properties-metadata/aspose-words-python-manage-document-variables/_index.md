## Руководство по внедрению

### Добавление и обновление переменных

#### Обзор
Сохраняйте пары ключ-значение в документе для динамического управления содержимым. Вот как добавлять и обновлять эти переменные.

#### Шаги:
1. **Добавить переменные:**
   ```python
   variables = doc.variables
   variables.add('Home address', '123 Main St.')
   variables.add('City', 'London')
   ```
2. **Обновить существующие переменные:**
   Присвойте новое значение существующему ключу, чтобы обновить его:
   ```python
   variables.add('Home address', '456 Queen St.')
   ```

#### Отображение значений переменных

1. **Вставьте поля DOCVARIABLE:**
   Используйте поля для отображения значений переменных в тексте документа:
   ```python
   builder = aw.DocumentBuilder(doc)
   field = builder.insert_field(aw.fields.FieldType.FIELD_DOC_VARIABLE, True)
   field.variable_name = 'Home address'
   field.update()  # Обновите поле, чтобы отразить текущее значение
   ```

### Проверка и удаление переменных

#### Обзор
Эффективно управляйте переменными, проверяя их существование или удаляя, когда они больше не нужны.

#### Шаги:
1. **Проверка существования переменной:**
   ```python
   assert 'City' in variables
   ```
2. **Удалить переменные:**
   - По имени:
     ```python
     variables.remove('City')
     ```
   - По индексу:
     ```python
     variables.remove_at(0)  # Удалить первый элемент
     ```
3. **Очистить все переменные:**
   ```python
   variables.clear()
   ```
