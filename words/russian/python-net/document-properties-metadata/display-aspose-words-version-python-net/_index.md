Для работы с Aspose.Words вам необходимо сначала установить его с помощью `pip`.

### Базовая инициализация и настройка:
После установки инициализируйте Aspose.Words в вашем скрипте Python следующим образом:



- `BuildVersionInfo.version`: Предоставляет строку, содержащую номер версии.
