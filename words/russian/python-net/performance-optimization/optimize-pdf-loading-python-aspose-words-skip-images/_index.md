### Пропускать изображения PDF во время загрузки
Пропуск изображений может иметь решающее значение для приложений, где требуется только текстовое содержимое PDF-файла, сокращая время загрузки и использование памяти.

#### Шаг 1: Определите пути к документам
Сначала укажите пути для входных и выходных документов:
```python
YOUR_DOCUMENT_DIRECTORY = 'path/to/your/documents/'
YOUR_OUTPUT_DIRECTORY = 'path/to/output/directory/'

def skip_pdf_images_demo():
    file_name = YOUR_DOCUMENT_DIRECTORY + 'Images.pdf'
```
#### Шаг 2: Настройте параметры загрузки PDF
Создать `PdfLoadOptions` экземпляр и настройте его для пропуска или включения изображений:
```python
for is_skip_pdf_images in [True, False]:
    options = aw.loading.PdfLoadOptions()
    options.skip_pdf_images = is_skip_pdf_images
    options.page_index = 0
    options.page_count = 1
```
- **Параметры**:
  - `skip_pdf_images`: Логическое значение, определяющее, следует ли пропускать изображения.
  - `page_index` и `page_count`: Укажите страницы PDF-файла для загрузки.

#### Шаг 3: Загрузите документ
Загрузите документ с указанными параметрами:
```python
doc = aw.Document(file_name=file_name, load_options=options)
```

#### Шаг 4: Проверьте загрузку изображения
Проверьте наличие изображений на основе конфигурации:
```python
shape_collection = doc.get_child_nodes(aw.NodeType.SHAPE, True)

if is_skip_pdf_images:
    assert shape_collection.count == 0, 'Expected no images when skipping PDF images'
else:
    assert shape_collection.count != 0, 'Expected some images when not skipping PDF images'
# Выполнить демо
skip_pdf_images_demo()
```
