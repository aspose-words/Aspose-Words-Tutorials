### Растрирование сложных элементов в PCL
Растрирование сложных элементов гарантирует, что преобразования, такие как поворот или масштабирование, будут точно сохранены при печати. Вот как этого можно добиться:

#### Обзор
Возможность растеризации преобразованных элементов имеет важное значение для поддержания визуальной точности во время печати, особенно при работе со сложными конструкциями.

```python
import aspose.words as aw
# Загрузить документ
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Rendering.docx')
save_options = aw.saving.PclSaveOptions()
save_options.save_format = aw.SaveFormat.PCL
save_options.rasterize_transformed_elements = True  # Включить растеризацию преобразованных элементов
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.RasterizeElements.pcl', save_options=save_options)
```

**Объясняемые параметры:**
- `rasterize_transformed_elements`: Гарантирует, что любое преобразование, примененное к элементу, сохранится в печатном виде.

### Объявить резервный шрифт для PCL
Если указанный шрифт недоступен, наличие резервного варианта гарантирует, что ваш документ будет напечатан без пропущенных элементов. Вот как это можно настроить:

#### Обзор
Укажите заменяющий шрифт, который будет использоваться, если исходный шрифт не будет найден во время печати.

```python
import aspose.words as aw
doc = aw.Document()
builder = aw.DocumentBuilder(doc=doc)
builder.font.name = 'Non-existent font'  # Намеренно использовать недоступное имя шрифта
derived_text = builder.write('Hello world!')

save_options = aw.saving.PclSaveOptions()
save_options.fallback_font_name = 'Times New Roman'  # Установить резервный шрифт
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.SetPrinterFont.pcl', save_options=save_options)
```

**Объясняемые параметры:**
- `fallback_font_name`: Имя шрифта, который будет использоваться, если исходный шрифт недоступен.

### Добавить замену шрифта принтера в PCL
Замените определенные шрифты документа во время печати для лучшей совместимости:

#### Обзор
Замените указанный шрифт альтернативным при печати, обеспечив единообразный вид текста на разных устройствах.

```python
import aspose.words as aw
doc = aw.Document()
builder = aw.DocumentBuilder(doc=doc)
builder.font.name = 'Courier'
builder.write('Hello world!')

save_options = aw.saving.PclSaveOptions()
save_options.add_printer_font('Courier New', 'Courier')  # Замените «Курьер» на «Курьер Новый»
doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.AddPrinterFont.pcl', save_options=save_options)
```

**Объясняемые параметры:**
- `add_printer_font`: Заменяет исходный шрифт на аналогичный для печати.

### Сохранение информации о лотке для бумаги в PCL
Сохранение настроек лотка для бумаги имеет решающее значение при работе с многолотковыми принтерами:

#### Обзор
Поддерживайте определенные настройки лотка для различных разделов документа, гарантируя правильное использование бумаги во время заданий печати.

```python
import aspose.words as aw
doc = aw.Document('YOUR_DOCUMENT_DIRECTORY/Rendering.docx')

for section in doc.sections:
    section.page_setup.first_page_tray = 15  # Установите лоток для первой страницы на 15
    section.page_setup.other_pages_tray = 12  # Установите лоток для других страниц на 12

doc.save('YOUR_OUTPUT_DIRECTORY/PclSaveOptions.GetPreservedPaperTrayInformation.pcl')
```

**Объясняемые параметры:**
- `first_page_tray` и `other_pages_tray`: Определите лотки для бумаги для первой и последующих страниц.
