# Оптимизация файлов Excel с помощью Aspose.Words для Python: методы сжатия и настройки

Откройте для себя мощные методы эффективного сжатия, организации и повышения производительности ваших документов Excel с помощью Aspose.Words для Python. - Сжимайте документы XLSX для оптимального уменьшения размера файла

К концу этого руководства вы приобретете практические знания по повышению производительности и доступности файлов Excel.

## Руководство по внедрению
Мы разберем реализацию на отдельные функции, объяснив каждый шаг с помощью фрагментов кода и конфигураций.

### Функция 1: Сжатие документа XLSX
**Обзор**: эта функция помогает уменьшить размер файлов документов Excel, применяя максимальное сжатие при сохранении их в виде файлов XLSX.

#### Пошаговая реализация:
##### Загрузите ваш документ
Начните с загрузки документа, который вы хотите сжать:

```python
import aspose.words as aw

YOUR_DOCUMENT_DIRECTORY = 'path/to/your/document/directory'
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Shape with linked chart.docx')
```

##### Настроить параметры сжатия
Создать экземпляр `XlsxSaveOptions` и установите максимальный уровень сжатия:

```python
xlsx_save_options = aw.saving.XlsxSaveOptions()
xlsx_save_options.compression_level = aw.saving.CompressionLevel.MAXIMUM
xlsx_save_options.save_format = aw.SaveFormat.XLSX
```

##### Сохранить со сжатием
Наконец, сохраните документ, используя следующие параметры, чтобы получить сжатый файл XLSX:

```python
YOUR_OUTPUT_DIRECTORY = 'path/to/your/output/directory'
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'CompressedOutput.xlsx', save_options=xlsx_save_options)
```

### Функция 2: Сохранение документа в виде отдельных рабочих листов
**Обзор**: эта функция позволяет сохранять каждый раздел документа на отдельном листе, что способствует лучшей организации данных.

#### Пошаговая реализация:
##### Загрузите большой документ

```python
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Big document.docx')
```

##### Установить режим раздела
Настройте `XlsxSaveOptions` чтобы сохранить каждый раздел как отдельный рабочий лист:

```python
xlsx_save_options = aw.saving.XlsxSaveOptions()
xlsx_save_options.section_mode = aw.saving.XlsxSectionMode.MULTIPLE_WORKSHEETS
```

##### Сохранить с несколькими рабочими листами
Выполнить функцию сохранения:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'MultipleWorksheetsOutput.xlsx', save_options=xlsx_save_options)
```

### Функция 3: Укажите режим анализа даты и времени
**Обзор**: Включите автоматическое определение форматов даты и времени, чтобы обеспечить точность и единообразие ваших документов.

#### Пошаговая реализация:
##### Загрузите документ с данными о дате и времени

```python
doc = aw.Document(file_name=YOUR_DOCUMENT_DIRECTORY + 'Xlsx DateTime.docx')
```

##### Настройка анализа даты и времени
Настройте автоматическое определение форматов даты и времени с помощью `XlsxSaveOptions`:

```python
save_options = aw.saving.XlsxSaveOptions()
save_options.date_time_parsing_mode = aw.saving.XlsxDateTimeParsingMode.AUTO
```

##### Сохранение с использованием автоматически определенных форматов даты и времени
Сохраните документ, чтобы применить эти настройки:

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'DateTimeParsingModeOutput.xlsx', save_options=save_options)
```
