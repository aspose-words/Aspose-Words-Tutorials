## Руководство по внедрению

### Защита паролем и сохранение маршрутизации (H2)

#### Обзор

Эта функция позволяет вам устанавливать пароли для старых форматов документов Microsoft Word, обеспечивая безопасность ваших документов. Кроме того, она сохраняет информацию о маршрутном листе во время процесса сохранения.

##### Настройка DocSaveOptions с защитой паролем (H3)

Сначала создайте новый документ и настройте его. `DocSaveOptions`:

```python
import aspose.words as aw

def save_with_password_and_routing_slip():
    # Создать новый документ
    doc = aw.Document()
    builder = aw.DocumentBuilder(doc=doc)
    builder.write('Hello world!')

    # Настройте DocSaveOptions для защиты паролем
    options = aw.saving.DocSaveOptions(aw.SaveFormat.DOC)
    options.password = 'MyPassword'

    # Сохранение информации о маршрутном листе
    options.save_routing_slip = True

    # Сохранить документ
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithPasswordAndRoutingSlip.doc"
    doc.save(file_name=output_path, save_options=options)

    # Подтвердите, загрузив пароль
    load_options = aw.loading.LoadOptions(password='MyPassword')
    loaded_doc = aw.Document(file_name=output_path, load_options=load_options)
    assert 'Hello world!' == loaded_doc.get_text().strip()
```

**Объясняемые параметры:**
- `options.password`: Устанавливает пароль для защиты документа.
- `options.save_routing_slip`: Сохраняет информацию о маршрутном листе.

#### Советы по устранению неполадок

- Перед сохранением убедитесь, что путь к выходному каталогу существует.
- Для повышения безопасности используйте уникальный и надежный пароль.

### Использование временной папки (H2)

#### Обзор

При работе с большими документами использование временной папки на диске может повысить производительность за счет сокращения использования памяти.

##### Настройка DocSaveOptions для временных папок (H3)

Вот как настроить временную папку:

```python
import os
import aspose.words as aw

def save_using_temp_folder():
    # Загрузить существующий документ
    input_path = "YOUR_DOCUMENT_DIRECTORY/Rendering.docx"
    doc = aw.Document(file_name=input_path)

    # Настройте DocSaveOptions для использования временной папки
    options = aw.saving.DocSaveOptions()
    temp_folder = "YOUR_OUTPUT_DIRECTORY/TempFiles"

    # Убедитесь, что временная папка существует.
    os.makedirs(temp_folder, exist_ok=True)
    options.temp_folder = temp_folder

    # Сохраните, используя временную папку.
    output_path = "YOUR_OUTPUT_DIRECTORY/DocWithTempFolder.doc"
    doc.save(file_name=output_path, save_options=options)
```

**Основные параметры конфигурации:**
- `options.temp_folder`: Указывает путь, используемый для промежуточного хранения файлов.

#### Советы по устранению неполадок

- Проверьте права на запись для вашей временной папки.
- Убедитесь, что в указанном каталоге достаточно места на диске.
