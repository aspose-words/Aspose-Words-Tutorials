#### Загрузить документ из файла

**Обзор:**

Загрузка документа напрямую из локального системного пути к файлу проста. Вот как это можно сделать:

##### Шаг 1: Инициализация класса загрузчика

Создайте экземпляр нашего пользовательского класса, предназначенного для обработки загрузки документов.

```python
class LoadDocumentsWithComHelper:
    def __init__(self):
        self.com_helper = aw.ComHelper()
```

##### Шаг 2: Определите метод загрузки файла

Реализуйте метод, который берет путь к файлу и использует `com_helper.open` для загрузки документа.

```python
def open_document_from_file(self, file_path):
    """
    Opens a document using a local system filename.
    
    :param file_path: Path to the document file
    """
    doc = self.com_helper.open(file_name=file_path)
    return doc.get_text().strip()
```

**Объяснение:** The `open` метод считывает указанный файл и возвращает `Document` объект, из которого можно извлечь текст или другие данные.

#### Загрузить документ из потока

**Обзор:**

В сценариях, когда документы не хранятся локально, а вместо этого к ним осуществляется доступ через потоки (например, сетевые ответы), их эффективная загрузка имеет решающее значение.

##### Шаг 1: Определите метод потоковой загрузки

Реализуйте другой метод обработки загрузки документов из входного потока:

```python
from io import BytesIO

def open_document_from_stream(self, stream):
    """
    Opens a document using an input stream.
    
    :param stream: A BytesIO stream containing the document data
    """
    doc = self.com_helper.open(stream=stream)
    return doc.get_text().strip()
```

**Объяснение:** Этот метод использует `BytesIO` для имитации файловых объектов из потоков байтов, что обеспечивает бесперебойную загрузку документов без необходимости использования физического файла.
