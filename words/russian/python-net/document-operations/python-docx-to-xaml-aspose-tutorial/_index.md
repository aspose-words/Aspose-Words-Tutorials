## Руководство по внедрению

В этом разделе мы проведем вас через преобразование файлов DOCX в фиксированную форму XAML с обработкой ресурсов. Мы рассмотрим каждую функцию шаг за шагом.

### Преобразование документа в фиксированный формат XAML

#### Обзор
В этой части основное внимание уделяется использованию Aspose.Words. `save` метод преобразования вашего документа в фиксированный формат XAML.

#### Шаг 1: Загрузите документ
Начните с загрузки вашего файла DOCX в Aspose.Words `Document` объект:

```python
doc = aw.Document(MY_DIR + "Rendering.docx")
```

#### Шаг 2: Создайте параметры сохранения
Инициализировать `XamlFixedSaveOptions` чтобы настроить процесс сохранения:

```python
options = aw.saving.XamlFixedSaveOptions()
```

#### Шаг 3: Настройка обработки ресурсов
Определите, как управляются связанные ресурсы, установив `resources_folder`, `resources_folder_alias`и функция обратного вызова.

```python
callback = ExXamlFixedSaveOptions.ResourceUriPrinter()
options.resource_saving_callback = callback
options.resources_folder = ARTIFACTS_DIR + "XamlFixedResourceFolder"
options.resources_folder_alias = ARTIFACTS_DIR + "XamlFixedFolderAlias"

# Перед сохранением ресурсов убедитесь, что папка псевдонима существует.
os.makedirs(options.resources_folder_alias)
```

#### Шаг 4: Сохраните документ.
Наконец, сохраните документ, используя настроенные параметры:

```python
doc.save(ARTIFACTS_DIR + "XamlFixedSaveOptions.resource_folder.xaml", options)
```

### Отслеживание URI ресурсов
Для мониторинга и печати URI ресурсов во время преобразования реализуйте `ResourceUriPrinter` класс, который подсчитывает и регистрирует каждый URI.

#### Обзор
Механизм обратного вызова помогает отслеживать ресурсы, созданные во время операции сохранения.

#### Реализация класса обратного вызова
Вот как можно определить пользовательский обратный вызов для управления экономией ресурсов:

```python
class ResourceUriPrinter(aw.saving.IResourceSavingCallback):
    """Counts and prints URIs of resources created during conversion."""
    
    def __init__(self):
        self.resources = []  # тип: Список[стр]
    
    def resource_saving(self, args: aw.saving.ResourceSavingArgs):
        self.resources.append(f"Resource \"{args.resource_file_name}\"\n\t{args.resource_file_uri}")
        
        # Перенаправить потоки в папку псевдонима
        args.resource_stream = open(args.resource_file_uri, 'wb')
        args.keep_resource_stream_open = False
```

### Советы по устранению неполадок
- Убедитесь, что все каталоги, указанные в `resources_folder` и `resources_folder_alias` существовать до запуска вашего скрипта.
- Еще раз проверьте пути к файлам на предмет опечаток.
