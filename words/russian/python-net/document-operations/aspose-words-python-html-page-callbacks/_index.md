### Руководство по внедрению

#### Сохранение каждой страницы как отдельного HTML-файла

Мы покажем, как сохранить каждую страницу документа Word как отдельный HTML-файл с помощью Aspose.Words. `IPageSavingCallback`.

##### Обзор

Настройте процесс сохранения, реализовав обратный вызов, который указывает имена файлов для выходных страниц.

##### Пошаговое руководство

**1. Создание и настройка документа:**

Создайте или загрузите документ с помощью Aspose.Words:

```python
import aspose.words as aw

doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Page 1.")
builder.insert_break(aw.BreakType.PAGE_BREAK)
builder.writeln("Page 2.")
builder.insert_image("path/to/image.jpg")
builder.insert_break(aw.BreakType.PAGE_BREAK)
builder.writeln("Page 3.")
```

**2. Настройте параметры сохранения HTML Fixed:**

Настраивать `HtmlFixedSaveOptions` и назначить пользовательский обратный вызов для сохранения страницы:

```python
html_fixed_save_options = aw.saving.HtmlFixedSaveOptions()
html_fixed_save_options.page_saving_callback = CustomFileNamePageSavingCallback(ARTIFACTS_DIR)
```

**3. Реализуйте пользовательский класс обратного вызова:**

Определите `CustomFileNamePageSavingCallback` сорт:

```python
class CustomFileNamePageSavingCallback(aw.saving.IPageSavingCallback):
    def __init__(self, output_dir):
        self.output_dir = output_dir

    def page_saving(self, args):
        # Укажите имя файла для текущей страницы
        args.page_file_name = f"{self.output_dir}/page_{args.page_index + 1}.html"
```

**4. Сохраните документ:**

Сохраните документ, используя настроенные параметры:

```python
doc.save(f"{ARTIFACTS_DIR}/output.html", html_fixed_save_options)
```
