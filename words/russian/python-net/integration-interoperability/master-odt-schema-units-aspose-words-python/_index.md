## Руководство по внедрению

### Соответствие версиям схемы ODT

#### Обзор

Для обеспечения совместимости с определенными версиями спецификации OpenDocument (схема ODT) Aspose.Words позволяет вам определить, должен ли ваш документ строго соответствовать спецификациям версии 1.1.

**Шаг за шагом:**

##### Шаг 1: Настройка параметров сохранения
```python
import aspose.words as aw

doc = aw.Document('path/to/your/input.docx')
save_options = aw.saving.OdtSaveOptions()
```

##### Шаг 2: Настройте версию схемы ODT
```python
# Установите значение True для строгого соответствия ODT версии 1.1.
save_options.is_strict_schema11 = True
```

##### Шаг 3: Сохраните документ.
```python
doc.save('path/to/your/output.odt', save_options)
```

### Настройка единиц измерения

#### Обзор

Aspose.Words позволяет вам выбирать между метрическими (сантиметры) и имперскими (дюймы) единицами измерения при сохранении документов в формате ODT. Эта гибкость гарантирует, что параметры вашего стиля будут соответствовать требуемым стандартам.

**Шаг за шагом:**

##### Шаг 1: Выбор единицы измерения
```python
save_options = aw.saving.OdtSaveOptions()
# Выбирайте между САНТИМЕТРАМИ или ДЮЙМАМИ в зависимости от ваших потребностей.
save_options.measure_unit = aw.saving.OdtSaveMeasureUnit.CENTIMETERS
```

##### Шаг 2: Сохраните документ с единицами измерения
```python
doc.save('path/to/your/output.odt', save_options)
```

### Шифрование документов ODT/OTT

#### Обзор

Aspose.Words позволяет вам защитить ваши документы, зашифровав их. В этом разделе описывается, как применять защиту паролем при сохранении файла ODT или OTT.

**Шаг за шагом:**

##### Шаг 1: Инициализация документа и сохранение параметров
```python
doc = aw.Document()
builder = aw.DocumentBuilder(doc)
builder.writeln("Hello world!")
save_options = aw.saving.OdtSaveOptions(aw.SaveFormat.ODT)
```

##### Шаг 2: Установите защиту паролем
```python
# Установите пароль для шифрования
save_options.password = 'your_password_here'
doc.save('path/to/encrypted_output.odt', save_options)
```
