## Руководство по внедрению

### Резюме ИИ

Резюмирование текста необходимо для быстрого понимания больших документов. Вот как это можно сделать с помощью Aspose.Words и OpenAI:

#### Резюмирование отдельных документов
**Обзор:** Эта функция позволяет эффективно резюмировать отдельный документ.

- **Загрузить документ:**
  ```python
  first_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **Настроить модель ИИ:**
  - Для обобщения используйте модель GPT OpenAI.
  ```python
  api_key = 'YOUR_API_KEY'  
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model()
           .with_organization('Organization')
           .with_project('Project'))
  ```

- **Установите параметры резюмирования:**
  ```python
  options = aw.ai.SummarizeOptions()
  options.summary_length = aw.ai.SummaryLength.SHORT
  ```

- **Выполните обобщение:**
  ```python
  one_document_summary = model.summarize(source_document=first_doc, options=options)
  one_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.One.docx')
  ```

#### Резюмирование нескольких документов

Для обобщения нескольких документов одновременно:

- **Загрузить дополнительные документы:**
  ```python
  second_doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **Изменить длину резюме:**
  ```python
  options.summary_length = aw.ai.SummaryLength.LONG
  ```

- **Обобщение нескольких документов:**
  ```python
  multi_document_summary = model.summarize(source_documents=[first_doc, second_doc], options=options)
  multi_document_summary.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiSummarize.Multi.docx')
  ```

### Перевод с помощью искусственного интеллекта

Перевод документов на разные языки может открыть новые рынки и аудиторию.

#### Обзор:
Эта функция переводит текст с использованием моделей Google.

- **Загрузить документ:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Document.docx')
  ```

- **Настроить модель перевода:**
  - Используйте Google AI для переводов.
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GEMINI_15_FLASH)
           .with_api_key(api_key)
           .as_google_ai_model())
  ```

- **Перевести документ:**
  ```python
  translated_doc = model.translate(doc, aw.ai.Language.ARABIC)
  translated_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiTranslate.docx')
  ```

### Проверка грамматики с помощью искусственного интеллекта

Улучшение качества документа путем проверки грамматики.

#### Обзор:
Эта функция проверяет и исправляет грамматические ошибки в ваших документах.

- **Загрузить документ:**
  ```python
  doc = aw.Document(file_name='YOUR_DOCUMENT_DIRECTORY/Big document.docx')
  ```

- **Настроить модель грамматики:**
  - Используйте модель GPT OpenAI для проверки грамматики.
  ```python
  model = (aw.ai.AiModel.create(aw.ai.AiModelType.GPT_4O_MINI)
           .with_api_key(api_key)
           .as_open_ai_model())
  ```

- **Установить параметры грамматики:**
  ```python
  grammar_options = aw.ai.CheckGrammarOptions()
  grammar_options.improve_stylistics = True
  ```

- **Проверьте и сохраните документ:**
  ```python
  proofed_doc = model.check_grammar(doc, grammar_options)
  proofed_doc.save(file_name='YOUR_OUTPUT_DIRECTORY/AI.AiGrammar.docx')
  ```

## Раздел часто задаваемых вопросов

**В1: Нужна ли мне платная подписка на Aspose.Words?**
- **А:** Доступна бесплатная пробная версия, но для долгосрочного использования требуется покупка лицензии. Вы также можете получить временные лицензии.

**В2: Что произойдет, если мой ключ API будет скомпрометирован?**
- **А:** Немедленно отзовите старый ключ и сгенерируйте новый через панель управления вашего провайдера.

**В3: Могу ли я резюмировать более двух документов одновременно?**
- **А:** Да, `summarize` Метод поддерживает массив объектов документов для реферирования нескольких документов.

**В4: Как обрабатывать ошибки во время перевода?**
- **А:** Реализуйте блоки try-except в своем коде для эффективного перехвата и управления исключениями.

**В5: Можно ли дополнительно настроить длину резюме?**
- **А:** Да, отрегулируйте `summary_length` параметр в `SummarizeOptions` для более точного контроля длины выходного материала.
