### Анализ типа носителя с помощью API Aspose.Words

#### Обзор
Анализ типа носителя позволяет преобразовывать типы носителя IANA (типы MIME) в соответствующие форматы загрузки/сохранения Aspose. Эта функция обеспечивает совместимость между различными форматами документов во время операций с файлами.

#### Этапы внедрения
##### Шаг 1: Преобразование типов контента в форматы сохранения
В этом фрагменте показано, как найти подходящий формат сохранения для заданного типа MIME:

```python
from aspose.words import FileFormatUtil, SaveFormat

try:
    save_format = FileFormatUtil.content_type_to_save_format('image/jpeg')
except Exception as e:
    print("Exception:", e)

assert save_format == SaveFormat.JPEG
```
**Объяснение**: Этот код преобразует тип MIME «image/jpeg» в соответствующий ему формат сохранения Aspose, утверждая, что он соответствует `SaveFormat.JPEG`.

##### Шаг 2: Преобразование типов контента в форматы загрузки
Аналогично определяем формат загрузки:

```python
try:
    load_format = FileFormatUtil.content_type_to_load_format('application/msword')
except Exception as e:
    print("Exception:", e)

assert load_format == aw.LoadFormat.DOC
```
**Объяснение**: Фрагмент преобразует «application/msword» в формат загрузки Aspose, утверждая, что он соответствует `LoadFormat.DOC`.
