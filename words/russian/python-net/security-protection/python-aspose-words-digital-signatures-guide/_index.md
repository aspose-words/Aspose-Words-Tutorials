## Руководство по внедрению

Мы разберем реализацию на две основные функции: загрузку подписей и доступ к их свойствам.

### Функция 1: загрузка и итерация цифровых подписей

#### Обзор

Загрузка цифровых подписей из документа помогает проверить его подлинность. Давайте посмотрим, как это сделать с помощью Aspose.Words для Python.

#### Шаги по реализации

##### 1. Определите путь к документу

Сначала укажите путь к вашему документу с цифровой подписью:

```python
doc_path = 'path/to/your/Digitally_signed.docx'
```

Заменять `'path/to/your/Digitally_signed.docx'` с фактическим путем к файлу.

##### 2. Загрузить цифровые подписи

Использовать `DigitalSignatureUtil.load_signatures()` чтобы загрузить подписи из вашего документа:

```python
digital_signatures = dsignatures.DigitalSignatureUtil.load_signatures(doc_path)
```

Этот метод возвращает список объектов сигнатуры, которые можно перебирать.

##### 3. Повторите и распечатайте данные подписи

Просмотрите каждую подпись, чтобы вывести ее данные:

```python
for signature in digital_signatures:
    print(signature)
```

### Функция 2: Доступ к свойствам цифровой подписи

#### Обзор

Доступ к определенным свойствам позволяет проводить более детальную проверку и извлечение информации.

#### Шаги по реализации

##### 1. Доступ к специальной подписи

Если у вас несколько подписей, откройте первую из них:

```python
signature = digital_signatures[0]
```

##### 2. Извлечение свойств подписи

Вот как извлечь различные атрибуты подписи:
- **Действительность**:
  
  ```python
  is_valid = signature.is_valid
  ```

- **Тип подписи**:
  
  ```python
  signature_type = signature.signature_type
  ```

- **Время знака** (отформатировано):
  
  ```python
  sign_time = signature.sign_time.strftime('%m/%d/%Y %H:%M:%S %p')
  ```

- **Комментарии, эмитент и имена субъектов**:
  
  ```python
  comments = signature.comments
  issuer_name = signature.issuer_name
  subject_name = signature.subject_name
  ```

##### 3. Распечатать извлеченные свойства

Отобразить эти свойства для проверки:

```python
print(f"Signature Valid: {is_valid}")
print(f"Signature Type: {signature_type}")
print(f"Sign Time: {sign_time}")
print(f"Comments: {comments}")
print(f"Issuer Name: {issuer_name}")
print(f"Subject Name: {subject_name}")
```
