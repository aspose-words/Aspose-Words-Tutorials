## 常見問題

**Q: 我可以以程式方式變更頁碼格式（例如從羅馬數字改為阿拉伯數字）嗎？**  
A: 可以，您可以在 `PageNumber` 欄位上設定 `PageNumberStyle` 屬性為任何支援的格式。

**Q: 如何在特定頁面（例如標題頁）跳過頁碼？**  
A: 插入帶有 `\* MERGEFORMAT` 參數的 `PageNumber` 欄位，並在該節的頁首/頁尾設定 `Suppress` 屬性。

**Q: 能否自動為每章重新設定頁碼？**  
A: 完全可以。使用 `Section` 類別為每章建立新節，並啟用 `PageSetup.RestartPageNumbering`。

**Q: API 能有效處理大型文件嗎？**  
A: Aspose.Words 以串流方式處理文件，即使是上千頁的檔案也能將記憶體使用量降至最低。
