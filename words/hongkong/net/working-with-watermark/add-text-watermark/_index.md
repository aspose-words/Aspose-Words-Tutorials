## 安裝說明  
1. 下載 Aspose.Words for .NET：  
   從 [Aspose 下載頁面](https://releases。aspose.com/words/net/).  

2. 透過 NuGet 安裝：  
   - 開啟您的 Visual Studio 專案。  
   - 導覽至 NuGet 套件管理器（工具 > NuGet 套件管理器 > 管理解決方案的 NuGet 套件）。  
   - 搜尋“Aspose.Words”並點擊安裝。  

3. 新增命名空間引用：  
   在程式碼檔案頂部新增以下命名空間：  
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   using System.Drawing;
   ```  

4. 申請許可證（可選）：  
   要使用完整版本， [申請許可證](https://purchase.aspose.com/temporary-license/) 或使用 [免費試用](https://releases。aspose.com/).
