## 安裝說明  

1. 下載 Aspose.Words for .NET：  
   從官方取得最新版本 [Aspose.Words for .NET下載頁面](https://releases。aspose.com/words/net/).  

2. 透過 NuGet 安裝：  
   使用 NuGet 套件管理器將庫新增至您的專案。在程式包管理器控制台中執行以下命令：  
   ```bash
   Install-Package Aspose.Words
   ```  
