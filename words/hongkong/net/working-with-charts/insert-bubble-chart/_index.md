## 裝置指令  

1. 請下載Aspose.Words為 .NET  
   請從圖書館中拿下[關於Aspose.Words下載頁面](https://releases.aspose.com/words/net/)這就是我想要的.  

2. 透過 NuGet 包管理器安裝  
   * 打開你的視覺工作室項目.  
   * 移動到工具> NuGet 包管理器> 管理 NuGet 包解決方案.  
   * 搜尋`"說話. 字"`在 Browse 帳號上.  
   * 選擇包,然後點擊安裝.  

3. 透過 NuGet 包管理器控制台安裝  
   在包管理器控制台中執行下列命令:  
   ```bash
   Install-Package Aspose.Words
   ```  

4. 增加許可證  
   請申請許可證,[暫時許可證頁面](https://purchase.aspose.com/temporary-license/)這就是我想要的.
