# 使用 Aspose.Words 在 Word 文件中插入形狀

## 安裝說明  
1. 下載 Aspose.Words for .NET：  
   從取得最新版本的 Aspose.Words for .NET [下載頁面](https://releases。aspose.com/words/net/).  

2. 透過 NuGet 安裝：  
   在 Visual Studio 中開啟您的專案並使用 NuGet 套件管理器安裝庫：  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. 新增命名空間引用：  
   在您的程式碼中，包含 Aspose.Words 的命名空間：  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. 啟動許可證（可選）：  
   如果您擁有有效的許可證，請在您的專案中應用它以避免限制。你可以 [購買許可證](https://purchase.aspose.com/buy) 或得到 [臨時執照](https://purchase。aspose.com/temporary-license/).
