## 裝置指令 
1. 必須要確保您的電腦上安裝了 .NET.
2. 設置Aspose.Words: 
   * 在 Visual Studio 中使用 NuGet 包管理器:
     ```
     Install-Package Aspose.Words
     ```
   * 或直接從網路上下載DLL[目標網站](https://www.aspose.com)這就是我想要的.
