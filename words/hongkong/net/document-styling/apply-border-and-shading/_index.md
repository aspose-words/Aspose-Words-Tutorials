# 在Aspose.Words中,將邊界與影子應用到段落上

## 裝置指令  

1. 請下載Aspose.Words為 .NET  
   請從 網路上獲得最新版本.[提供 .NET 下載頁面的Aspose.Words](https://releases.aspose.com/words/net/)這就是我想要的.  

2. 透過 NuGet 設置  
   * 請在視覺工作室開啟您的項目.  
   * 請進入 NuGet 包管理器 (`工具`沒有任何相關信息`提供包裝管理器`沒有任何相關信息`管理 NuGet 包裝的解決方案..`該組織的組織  
   * 搜尋`"說話. 字"`點擊`設置`這就是我想要的.  

3. 包含名稱空間  
   加入必要的名稱空間:  
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
