# 建立一個包含內容表的字體文檔
這個例子展示了如何使用 .NET 的Aspose.Words來创建包含使用 DocumentBuilder 类生成的互動內容表 (TOC) 的 Word 文檔.該代碼顯示了如何插入各種層次的標題,格式化,並儲存結果文檔為 .docx 文件.

## 裝置指令
1. 必須要確保您的電腦上安裝了 .NET.
2. 設置Aspose.Words: 
   * 使用 NuGet 包管理器控制台:
     ```
     Install-Package Aspose.Words
     ```
   * 或在 NuGet 包管理器 GUI 中搜尋Aspose.Words.
