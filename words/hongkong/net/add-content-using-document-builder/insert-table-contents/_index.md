## 安裝說明
1. 先決條件：確保您的機器上安裝了 .NET。
2. 安裝 Aspose.Words： 
   - 使用 NuGet 套件管理器控制台：
     ```
     Install-Package Aspose.Words
     ```
   - 或者，在 NuGet 套件管理器 GUI 中搜尋「Aspose.Words」。
