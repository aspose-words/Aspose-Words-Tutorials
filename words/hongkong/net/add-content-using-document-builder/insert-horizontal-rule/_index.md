## 裝置指令
1. 設置Aspose.Words: 在 Visual Studio 中使用 NuGet 包管理器.
   * 請在視覺工作室開啟您的項目.
   * 在解決方案探險器中右鍵點擊您的項目.
   * 選擇 管理 NuGet 包.
   * 搜尋`"說話. 字"`請您繼續閱讀更多內容

2. 加入使用指令:在您的代碼檔案中, 加入以下指令,
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
