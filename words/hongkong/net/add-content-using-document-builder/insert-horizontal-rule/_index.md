## 安裝說明
1. 安裝 Aspose.Words：在 Visual Studio 中使用 NuGet 套件管理器。
   - 在 Visual Studio 中開啟您的專案。
   - 在解決方案資源管理器中以滑鼠右鍵按一下您的專案。
   - 選擇管理 NuGet 套件。
   - 搜尋 `Aspose.Words` 並安裝它。

2. 新增使用指令：在您的程式碼檔案中，包含以下指令以使用 Aspose.Words 類別：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
