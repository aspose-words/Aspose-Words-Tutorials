## 安裝說明：  
1. 安裝 Aspose.Words for .NET：
   - 使用 NuGet 套件管理器：
     - 在 Visual Studio 中開啟您的專案。
     - 在解決方案資源管理器中右鍵點選專案並選擇「管理 NuGet 套件」。
     - 搜尋“Aspose.Words”並點擊“安裝”。

   - 或者，使用套件管理器控制台：
     ```
     Install-Package Aspose.Words
     ```
