## 裝置指令:  
1. 設置 .NET 版本中的Aspose.Words:
   * 使用 NuGet 包管理器:
     * 請在視覺工作室開啟您的項目.
     * 在 Solution Explorer 中右鍵擊項目,並選擇"管理 NuGet 包".
     * 搜尋Aspose.Words並點擊"安裝".

   * 或使用包管理器控制台:
     ```
     Install-Package Aspose.Words
     ```
