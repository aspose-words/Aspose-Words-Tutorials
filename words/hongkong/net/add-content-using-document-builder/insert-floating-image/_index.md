## 安裝說明
1. 安裝 Aspose.Words：將 Aspose.Words NuGet 套件新增至您的 .NET 專案。您可以透過在程式包管理器控制台中執行以下命令來執行此操作：
   ```
   Install-Package Aspose.Words
   ```

2. 新增所需的命名空間：確保您的程式碼檔案包含以下命名空間：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
