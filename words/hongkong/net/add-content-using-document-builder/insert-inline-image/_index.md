# 使用 Aspose.Words 在 Word 文件中插入內嵌影像
此範例示範如何使用 .NET 的 Aspose.Words 程式庫將內嵌影像插入 Word 文件。透過利用 `DocumentBuilder` 類，您可以輕鬆地在文件中的當前遊標位置定位和插入影像。本指南提供了一個簡單的實作來幫助您開始在 Word 文件中處理圖像。

## 安裝說明
1. 安裝 Aspose.Words for .NET：
   - 在 Visual Studio 中使用 NuGet 套件管理器。
   - 在程式包管理器控制台中執行以下命令：
     ```
     Install-Package Aspose.Words
     ```

2. 設定您的項目：
   - 建立一個新的 .NET 專案或開啟一個現有專案。
   - 在程式碼檔案的頂部加入必要的使用指令：
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
