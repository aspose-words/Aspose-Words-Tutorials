## 安裝說明
1. 下載 Aspose.Words：訪問 [Aspose.Words for .NET](https://www.nuget.org/packages/Aspose.Words/) NuGet 頁面並下載最新版本。

2. 建立 .NET 專案：開啟 Visual Studio 並建立一個新的控制台或類別庫專案。

3. 新增 NuGet 套件：
   - 在解決方案資源管理器中以滑鼠右鍵按一下您的專案。
   - 選擇“管理 NuGet 套件”。
   - 搜尋“Aspose.Words”並點擊“安裝”。

4. 新增所需的使用語句：在程式碼檔案中包含必要的使用指令：
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Tables;
   ```
