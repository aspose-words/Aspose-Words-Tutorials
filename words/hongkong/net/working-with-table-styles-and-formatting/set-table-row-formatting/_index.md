## 安裝說明
1. 安裝 Aspose.Words for .NET：使用 NuGet 套件管理器安裝 Aspose.Words。您可以在程式包管理器控制台中執行以下命令：
   ```
   Install-Package Aspose.Words
   ```

2. 建立新專案：開啟 Visual Studio 並建立一個新的 C# 專案（控制台應用程式或任何您喜歡的類型）。

3. 新增引用：確保您的專案中引用了 Aspose.Words 庫。

4. 申請許可證（可選）：  
   要使用完整版本， [申請許可證](https://purchase.aspose.com/temporary-license/) 或使用 [免費試用](https://releases。aspose.com/words/net/).
