## 裝置指令  
執行這個例子,並使用 .NET 的Aspose.Words, 請按照以下步骤執行:  

1. 請下載Aspose.Words為 .NET  
   請從Aspose.Words最新版本獲得[標籤發布頁面](https://releases.aspose.com/words/net/)這就是我想要的.  

2. 透過 NuGet 包管理器安裝  
   在 Visual Studio 打開您的 .NET 項目,並在 NuGet 包管理器控制台中執行下列命令:  
   ```bash
   Install-Package Aspose.Words
   ```  

3. 建立暫時版本或授權版本  
   * 獲得一個[免費的暫時許可證](https://purchase.aspose.com/temporary-license/)或從該公司購買完整的許可證.[目標購買頁面](https://purchase.aspose.com/buy)這就是我想要的.  
   * 請將許可證用您的代碼:  
     ```csharp
     var license = new Aspose.Words.License();
     license.SetLicense("Aspose.Words.lic");
     ```
