## 安裝說明：

若要開始使用 Aspose.Words for .NET 並執行此範例，請依照以下簡單步驟操作：

1. 安裝 Aspose.Words for .NET：
   您可以透過 NuGet 輕鬆安裝 Aspose.Words for .NET。在程式包管理器控制台中執行以下命令：
   ```
   Install-Package Aspose.Words
   ```

2. 申請許可證（可選）：  
   要使用完整版本， [申請許可證](https://purchase.aspose.com/temporary-license/) 或使用 [免費試用](https://releases。aspose.com/words/net/).
