# Aspose.Words を使用して Word 文書に図形を挿入する
このガイドでは、Aspose.Words for .NETを使用して、テキストボックスなどの図形をWord文書に挿入および操作する方法を説明します。図形を絶対位置または相対位置で配置する方法、回転角度を設定する方法、そしてISO 29500:2008 Transitional DOCX形式で文書を保存する方法を学びます。 `OoxmlSaveOptions`この例では、Word 文書内の図形の操作が簡素化され、開発者は視覚的に魅力的なコンテンツを簡単に作成できるようになります。

## インストール手順  
1. Aspose.Words for .NET をダウンロード:  
   Aspose.Words for .NETの最新バージョンは、 [ダウンロードページ](https://releases。aspose.com/words/net/).  

2. NuGet 経由でインストール:  
   Visual Studio でプロジェクトを開き、NuGet パッケージ マネージャーを使用してライブラリをインストールします。  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. 名前空間参照を追加します:  
   コードに Aspose.Words の名前空間を含めます。  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. ライセンスをアクティブ化する（オプション）:  
   有効なライセンスをお持ちの場合は、制限を回避するためにプロジェクトに適用してください。 [ライセンスを購入する](https://purchase.aspose.com/buy) または [一時ライセンス](https://purchase。aspose.com/temporary-license/).
