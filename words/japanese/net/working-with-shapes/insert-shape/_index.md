# Aspose.Wordsを使って文字文書にフォームを挿入する
このガイドでは,Word文書にテキストボックスなどの形状を挿入し操作するために .NETのAspose.Wordsを使用する方法を示します. 形状を絶対的にまたは相対的に位置づけ,回転角度を設定し,文書をISO 29500:2008 移行DOCX形式で保存する方法について学びます.`OoxmlSaveOptions`この例では,Word文書の形状で作業を簡素化し,開発者が視覚的に魅力的なコンテンツを簡単に作成できるようにします.

## 設置指示  
1. .NET のためのAspose.Wordsをダウンロードする  
   ウェブの最新バージョンの Aspose.Words を入手してください[ダウンロードページ](https://releases.aspose.com/words/net/)やってくれ  

2. NuGet を介してインストールする  
   プロジェクトをVisual Studioで開き,NuGet パッケージ管理器を使用してライブラリをインストールします.  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. 名前空間参照を追加する  
   コードにはAspose.Wordsの名前空間を入力します.  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. ライセンス (オプション) を有効にする  
   ライセンスを取得する場合は 制限を避けるために プロジェクトに適用してください[ライセンスを購入する](https://purchase.aspose.com/buy) [派遣許可証](https://purchase.aspose.com/temporary-license/)やってくれ
