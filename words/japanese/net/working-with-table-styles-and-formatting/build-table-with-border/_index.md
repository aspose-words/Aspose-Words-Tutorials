# アクセシビリティを使って単語内の表の外枠を編集します。Do not translate the keyword between brackets (e.g. ServerName)
 
アクセシビリティ.NET は、言語ドキュメントの作成・編集・操作のための強力なライブラリです。この例では、既存の Word ドキュメントの取得の方法や、最初の表の境界の変更や、アクセシビリティを使ってカスタムフォーマットを適用する方法を学びます。このガイドブックは、既存の境界を消去し、表の周囲に緑色の境界を適用し、更新されたドキュメントを保存します。この機能により、単語ドキュメントのプレゼンテーションを効率的に改善することができます。
