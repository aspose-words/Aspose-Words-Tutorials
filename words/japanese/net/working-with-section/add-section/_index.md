# ワード文書に新しいセクションを追加

Association.NET は、複雑な文書構造を処理する強力な API を提供することにより、 Word ドキュメントの作業を行ないます。このチュートリアルは、Word 文書のプログラムに新しいセクションを追加することによって、ユーザーを巡回します。`文書`と`ドキュメントビルダー`クラスでは、ドキュメントを初期化し、テキストを追加し、新しいセクションを追加することができます。 自動的にワード文書を作成し、精密に操作しようとする開発者に適しています。
