# Aspose.Words を使って,文字に列表を挿入する

.NET の Aspose.Words は,Word 文書をプログラム的に作成し操作する非常に簡単です.この例では,コードの数行だけでWord 文書に列表を挿入する方法を示しています.強力な API を使用して,チャートを追加し,カテゴリーと値を定義し,ドキュメントを望ましいフォーマットで保存できます.このガイドに従って,あなたの .NET アプリケーションから直接プロのチャートを生成する方法はどれほど簡単であるかを見ることができます.

## 設置指示  

1. .NET のためのAspose.Wordsをダウンロードする  
   * 訪問する[Aspose.Wordsは.NETダウンロードページ](https://releases.aspose.com/words/net/)ダウンロードする  

2. 図書館をインストールする  
   * Visual Studio で NuGet パッケージマネージャーを使用して .NET のAspose.Wordsをインストールします.  
     ```
     Install-Package Aspose.Words
     ```

3. 名前空間を入力する  
   * プロジェクトに必要な名空を追加する:  
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```

4. 無料試用またはライセンス設定  
   * 始めましょう[無料試験](https://releases.aspose.com/)申請する[派遣許可証](https://purchase.aspose.com/temporary-license/)図書館の全機能を解除する
