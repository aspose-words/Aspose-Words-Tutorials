# Aspose.Wordsを使って表でWordドキュメントを作成する

## 設置指示:  
1. .NET に Aspose.Words をインストールする
   * NuGet パッケージマネージャーを使用する
     * プロジェクトをビジュアルスタジオで開け
     * Solution Explorer でプロジェクトを右クリックして"NuGet パッケージ管理"を選択します.
     * "Aspose.Words"を検索し",インストール"をクリックします.

   * または,パッケージ管理コンソールを使用します.
     ```
     Install-Package Aspose.Words
     ```
