# ワード文書を浮動画像で作成

この例は、Asdom.NET を使ってワード文書を作成する方法を示します。 NET 用の単語を作成し、浮動小数点配列で画像を挿入します。提供されたコードは、ドキュメントと文書ビルダーを初期化し、画像を指定した寸法と位置で挿入し、最後に文書を保存します。 このガイドは、ドキュメントの生成能力を高める開発者に適しています。

## インストールインジケータ
1. Asdomument.words: Assoft.News プロジェクトに NET プロジェクトに Access.ngetパッケージを追加します。これはパッケージマネージャのコンソールで以下のコマンドを実行することによって行うことができます。
   ```
   Install-Package Aspose.Words
   ```

2. 必要な名前空間を追加: コードファイルには次の名前空間を含む:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
