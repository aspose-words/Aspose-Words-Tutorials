# 浮動画像でWordドキュメントを作成する

この例では,NETのAspose.Wordsを使用してWord文書を作成し,浮動レイアウトを持つ画像を挿入する方法を示しています.提供されたコードは,ドキュメントとドキュメントビルダーを初期化し,指定した次元と位置付けのある画像を挿入し,最終的にドキュメントを保存します.このガイドは,開発者がAspose.Wordsを使用してドキュメント生成能力を向上させたいのに理想的です.

## 設置指示
1. Aspose.Wordsをインストールする. Aspose.Words NuGet パッケージをNETプロジェクトに追加する. パッケージ管理者コンソールで次のコマンドを実行することで,これを実行できます.
   ```
   Install-Package Aspose.Words
   ```

2. 必要な名空を追加する:コードファイルには以下の名空が含まれていることを確認してください.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
