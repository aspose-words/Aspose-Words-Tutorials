# フローティング画像付きのWord文書を作成する

この例では、Aspose.Words for .NET を使用してWord文書を作成し、フローティングレイアウトで画像を挿入する方法を示します。提供されているコードは、DocumentとDocumentBuilderを初期化し、指定されたサイズと位置で画像を挿入し、最後に文書を保存します。このガイドは、Aspose.Wordsを使用してドキュメント生成機能を強化したい開発者に最適です。

## インストール手順
1. Aspose.Words をインストールします。Aspose.Words NuGet パッケージを .NET プロジェクトに追加します。パッケージ マネージャー コンソールで次のコマンドを実行します。
   ```
   Install-Package Aspose.Words
   ```

2. 必要な名前空間を追加します。コード ファイルに次の名前空間が含まれていることを確認します。
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
