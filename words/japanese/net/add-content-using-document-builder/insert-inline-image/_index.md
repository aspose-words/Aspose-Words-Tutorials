# Aspose.Wordsを使用してWord ドキュメントにインライン画像を挿入する
この例では,NETのAspose.Wordsライブラリを使用して Word ドキュメントにインライン画像を挿入する方法を示しています.`ドキュメントビルダー`文書の現在のカーソルの位置に画像を簡単に配置し挿入できます.このガイドは,Word文書で画像処理を始めるのに役立つ簡単な実装を提供します.

## 設置指示
1. .NET に Aspose.Words をインストールする
   * Visual Studio で NuGet パッケージマネージャーを使用します.
   * パッケージ管理コンソールで次のコマンドを実行します.
     ```
     Install-Package Aspose.Words
     ```

2. 計画 を 設定 する
   * 新しい .NET プロジェクトを作成するか,既存のプロジェクトを開く.
   * コードファイルの上部には必要な指示を追加します
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
