# access.words を使って単語ドキュメントにインライン画像を挿入
この例は、Asfof.words ライブラリを使って、インラインイメージを単語ドキュメントに挿入する方法を示します。`ドキュメントビルダー`クラスでは、ドキュメント内の現在のカーソル位置に画像を簡単に位置付け、挿入できます。このガイドは、 Word ドキュメントで画像処理を開始するための簡単に実装を提供します。
