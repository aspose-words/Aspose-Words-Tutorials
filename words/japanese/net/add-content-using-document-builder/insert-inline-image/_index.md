# Aspose.Words を使用して Word 文書にインライン画像を挿入する
この例では、.NET用のAspose.Wordsライブラリを使用してWord文書にインライン画像を挿入する方法を示します。 `DocumentBuilder` クラスを使用すると、文書内の現在のカーソル位置に画像を簡単に配置して挿入できます。このガイドでは、Word文書で画像処理を始めるのに役立つ簡単な実装方法を紹介します。

## インストール手順
1. Aspose.Words for .NET をインストールします。
   - Visual Studio で NuGet パッケージ マネージャーを使用します。
   - パッケージ マネージャー コンソールで次のコマンドを実行します。
     ```
     Install-Package Aspose.Words
     ```

2. プロジェクトを設定する:
   - 新しい .NET プロジェクトを作成するか、既存のプロジェクトを開きます。
   - コード ファイルの先頭に必要な using ディレクティブを追加します。
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
