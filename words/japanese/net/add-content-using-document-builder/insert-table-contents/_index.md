## 設置指示
1. 必須条件: コンピュータに .NET がインストールされていることを確認します.
2. Aspose.Wordsをインストールする 
   * NuGet パッケージ管理コンソールを使用します:
     ```
     Install-Package Aspose.Words
     ```
   * または NuGet パッケージマネージャー GUI で "Aspose.Words" を検索してください.
