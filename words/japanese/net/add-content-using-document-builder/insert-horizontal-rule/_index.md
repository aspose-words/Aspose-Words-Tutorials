## 設置指示
1. Aspose.Wordsをインストールする. Visual Studio で NuGet パッケージマネージャーを使用する.
   * プロジェクトをビジュアルスタジオで開け
   * 解決法探査機でプロジェクトを右クリックします
   * NuGet パッケージを管理するを選択します.
   * 検索する`"言葉"`設置して

2. 使用指令を追加する:コードファイルにAspose.Wordsクラスを使用する次の指令を入力してください.
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
