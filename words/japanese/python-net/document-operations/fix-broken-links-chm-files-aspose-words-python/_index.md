## 実装ガイド

Aspose.Words API を使用して CHM リンクを最適化するために、実装を主要な手順に分解してみましょう。

### ChmLoadOptions で元のファイル名を指定する

**概要：**
この機能を使用すると、.chm ファイルの元のファイル名を指定して、すべての内部リンクが正しく解決されることを保証できます。

#### ステップ1: 必要なモジュールをインポートする
まずはインポートから `aspose.words` そして `io`：

```python
import aspose.words as aw
import io
```

#### ステップ2: ロードオプションを構成する
インスタンスを作成する `ChmLoadOptions` 元のファイル名を設定します。

```python
load_options = aw.loading.ChmLoadOptions()
load_options.original_file_name = 'amhelp.chm'
```
**説明：**
設定 `original_file_name` Aspose.Words が CHM ファイル内のリンクを正確に解決し、壊れた URL を防ぐのに役立ちます。

#### ステップ3: ドキュメントを読み込んで保存する
.chm ドキュメントを読み込むには、次のオプションを使用します。

```python
doc = aw.Document(
    stream=io.BytesIO(system_helper.io.File.read_all_bytes(YOUR_DOCUMENT_DIRECTORY + 'Document with ms-its links.chm')),
    load_options=load_options
)
```
修正したリンクを保持したまま、HTML ファイルとして保存します。

```python
doc.save(file_name=YOUR_OUTPUT_DIRECTORY + 'ExChmLoadOptions.OriginalFileName.html')
```
**トラブルシューティングのヒント:**
.chmファイルへのパスが正しく、アクセス可能であることを確認してください。パスが正しくない場合は、コード内で適切に修正してください。
