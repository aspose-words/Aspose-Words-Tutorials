# Nhập một hình ảnh vào Word Document Header

## Các hướng dẫn cài đặt  

Để bắt đầu với Aspose.Words cho .NET, hãy làm theo những bước đơn giản này:  

1. Tải xuống Aspose.Words  
   * [Tải xuống phiên bản mới nhất](https://releases.aspose.com/words/net/)của Aspose.Words cho .NET.  

2. Thiết lập Aspose.Words thông qua NuGet  
   * Mở dự án của bạn trong Visual Studio.  
   * Đi vào menu Công cụ và chọn NuGet Package Manager > Quản lý NuGet Packages for Solution.  
   * Tìm kiếm`"Thiếu nại.`trong quản lý gói NuGet và cài đặt nó.  

   Hoặc, sử dụng Console quản lý gói:  
   ```shell
   Install-Package Aspose.Words
   ```  

3. Đơn xin giấy phép  
   * Nhận một [Giấy phép thử nghiệm miễn phí](https://releases.aspose.com/)hoặc [mua giấy phép](https://purchase.aspose.com/buy)\- Tôi không biết.  
   * Sử dụng mã sau để áp dụng giấy phép trong dự án của bạn:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
