# Tạo một tài liệu Word có mục lục
Ví dụ này trình bày cách sử dụng Aspose.Words cho .NET để tạo một tài liệu Word bao gồm Mục lục (TOC) tương tác được tạo bằng lớp DocumentBuilder. Mã minh họa cách chèn tiêu đề ở nhiều cấp độ khác nhau, định dạng chúng và lưu tài liệu kết quả dưới dạng tệp .docx. Thực hiện theo các bước dưới đây để triển khai dễ dàng.

## Hướng dẫn cài đặt
1. Điều kiện tiên quyết: Đảm bảo bạn đã cài đặt .NET trên máy của mình.
2. Cài đặt Aspose.Words: 
   - Sử dụng NuGet Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
   - Hoặc tìm kiếm "Aspose.Words" trong GUI của Trình quản lý gói NuGet.
