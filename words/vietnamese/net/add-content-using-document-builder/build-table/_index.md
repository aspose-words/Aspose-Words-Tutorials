## Hướng dẫn cài đặt:  
1. Cài đặt Aspose.Words cho .NET:
   - Sử dụng Trình quản lý gói NuGet:
     - Mở dự án của bạn trong Visual Studio.
     - Nhấp chuột phải vào dự án trong Solution Explorer và chọn "Quản lý gói NuGet".
     - Tìm kiếm "Aspose.Words" và nhấp vào "Cài đặt".

   - Hoặc sử dụng Bảng điều khiển quản lý gói:
     ```
     Install-Package Aspose.Words
     ```
