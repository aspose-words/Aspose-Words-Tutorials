# Tạo tài liệu Word với hình ảnh nổi

Ví dụ này trình bày cách tạo tài liệu Word bằng Aspose.Words cho .NET và chèn hình ảnh có bố cục nổi. Mã được cung cấp sẽ khởi tạo Document và DocumentBuilder, chèn hình ảnh có kích thước và vị trí được chỉ định, và cuối cùng lưu tài liệu. Hướng dẫn này lý tưởng cho các nhà phát triển muốn nâng cao khả năng tạo tài liệu của họ bằng Aspose.Words.

## Hướng dẫn cài đặt
1. Cài đặt Aspose.Words: Thêm gói Aspose.Words NuGet vào dự án .NET của bạn. Bạn có thể thực hiện việc này bằng cách chạy lệnh sau trong Package Manager Console:
   ```
   Install-Package Aspose.Words
   ```

2. Thêm không gian tên bắt buộc: Đảm bảo tệp mã của bạn bao gồm các không gian tên sau:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
