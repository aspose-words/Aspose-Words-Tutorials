# Tạo tài liệu Word với thước ngang
Ví dụ này minh họa cách sử dụng Aspose.Words cho .NET để tạo tài liệu Word và thêm quy tắc ngang bằng DocumentBuilder. Thực hiện theo các bước dưới đây để tìm hiểu cách định dạng tài liệu của bạn hiệu quả.

## Hướng dẫn cài đặt
1. Cài đặt Aspose.Words: Sử dụng Trình quản lý gói NuGet trong Visual Studio.
   - Mở dự án của bạn trong Visual Studio.
   - Nhấp chuột phải vào dự án của bạn trong Solution Explorer.
   - Chọn Quản lý gói NuGet.
   - Tìm kiếm `Aspose.Words` và cài đặt nó.

2. Thêm Chỉ thị sử dụng: Trong tệp mã của bạn, hãy bao gồm chỉ thị sau để sử dụng các lớp Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
