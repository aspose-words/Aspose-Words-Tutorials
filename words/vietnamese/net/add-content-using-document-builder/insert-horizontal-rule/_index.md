# Tạo Word Document với quy tắc ngang
Ví dụ này cho thấy việc sử dụng Aspose.Words cho .NET để tạo tài liệu Word và thêm một quy tắc ngang bằng cách sử dụng DocumentBuilder. Hãy làm theo các bước dưới đây để tìm hiểu cách định dạng tài liệu của bạn hiệu quả.

## Các hướng dẫn cài đặt
1. Thiết lập Aspose.Words: Sử dụng NuGet Package Manager trong Visual Studio.
   * Mở dự án của bạn trong Visual Studio.
   * Nhấp chuột phải vào dự án của bạn trong Solution Explorer.
   * Chọn quản lý gói NuGet.
   * Tìm kiếm`"Thiếu nại.`và cài đặt nó.

2. Thêm Định hướng sử dụng: Trong tệp mã của bạn, hãy bao gồm hướng dẫn sau đây để sử dụng các lớp Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
