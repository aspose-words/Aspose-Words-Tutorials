# Tạo một tài liệu Word nhiều trang với Aspose.Words

Ví dụ này trình bày cách tạo tài liệu Word bằng Aspose.Words cho .NET, cho phép bạn thêm nội dung trên nhiều trang. Ví dụ này cho thấy cách khởi tạo tài liệu mới, sử dụng DocumentBuilder để chèn văn bản và ngắt trang, và cuối cùng lưu tài liệu dưới dạng tệp .docx. Hoàn hảo cho các nhà phát triển muốn tự động tạo tài liệu trong các ứng dụng .NET của họ.

## Hướng dẫn cài đặt
1. Cài đặt Aspose.Words cho .NET: Sử dụng NuGet Package Manager để cài đặt Aspose.Words. Chạy lệnh sau trong Package Manager Console:
```
Install-Package Aspose.Words
```
