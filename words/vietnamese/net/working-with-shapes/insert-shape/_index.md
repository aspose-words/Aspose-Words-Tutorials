# Chèn hình dạng vào tài liệu Word bằng Aspose.Words
Hướng dẫn này trình bày cách sử dụng Aspose.Words cho .NET để chèn và thao tác các hình dạng, chẳng hạn như hộp văn bản, trong tài liệu Word. Bạn sẽ học cách định vị các hình dạng tuyệt đối hoặc tương đối, đặt góc xoay và lưu tài liệu ở định dạng ISO 29500:2008 Transitional DOCX bằng cách sử dụng `OoxmlSaveOptions`. Ví dụ này giúp đơn giản hóa việc làm việc với các hình dạng trong tài liệu Word, cho phép các nhà phát triển tạo nội dung hấp dẫn về mặt thị giác một cách dễ dàng.

## Hướng dẫn cài đặt  
1. Tải xuống Aspose.Words cho .NET:  
   Nhận phiên bản mới nhất của Aspose.Words cho .NET từ [trang tải xuống](https://releases.aspose.com/words/net/).  

2. Cài đặt qua NuGet:  
   Mở dự án của bạn trong Visual Studio và cài đặt thư viện bằng NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Thêm tham chiếu không gian tên:  
   Trong mã của bạn, hãy bao gồm không gian tên cho Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```  

4. Kích hoạt Giấy phép (Tùy chọn):  
   Nếu bạn có giấy phép hợp lệ, hãy áp dụng nó vào dự án của bạn để tránh những hạn chế. Bạn có thể [mua giấy phép](https://purchase.aspose.com/buy) hoặc nhận được một [giấy phép tạm thời](https://purchase.aspose.com/temporary-license/).
