# Nhập hình dạng trong tài liệu Word bằng cách sử dụng Aspose.Words

Bạn sẽ học cách định vị hình dạng tuyệt đối hoặc tương đối, đặt góc quay và lưu tài liệu trong định dạng ISO 29500:2008 DOCX chuyển tiếp bằng cách sử dụng `OoxmlSaveOptions`Ví dụ này đơn giản hóa việc làm việc với hình dạng trong tài liệu Word, cho phép các nhà phát triển tạo ra nội dung hấp dẫn trực quan dễ dàng.

## Các hướng dẫn cài đặt  
1. Tải Aspose.Words cho .NET:  
   Nhận phiên bản mới nhất của Aspose.Words cho .NET từ [trang tải xuống](https://releases.aspose.com/words/net/)\- Tôi không biết.  

2. Thiết lập thông qua NuGet:  
   Mở dự án của bạn trong Visual Studio và cài đặt thư viện bằng cách sử dụng NuGet Package Manager:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Thêm tên miền:  
   Trong mã của bạn, bao gồm không gian tên cho Aspose.Words:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
