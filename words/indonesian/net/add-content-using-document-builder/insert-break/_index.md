# Buat Dokumen Word Multi-Halaman dengan Aspose.Words

Contoh ini menunjukkan cara membuat dokumen Word menggunakan Aspose.Words untuk .NET, yang memungkinkan Anda menambahkan konten di beberapa halaman. Contoh ini menunjukkan cara menginisialisasi dokumen baru, menggunakan DocumentBuilder untuk menyisipkan teks dan pemisah halaman, dan akhirnya menyimpan dokumen sebagai file .docx. Sempurna bagi pengembang yang ingin mengotomatiskan pembuatan dokumen dalam aplikasi .NET mereka.

## Petunjuk Instalasi
1. Instal Aspose.Words untuk .NET: Gunakan NuGet Package Manager untuk menginstal Aspose.Words. Jalankan perintah berikut di Package Manager Console:
```
Install-Package Aspose.Words
```
