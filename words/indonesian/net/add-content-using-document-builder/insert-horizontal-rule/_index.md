## Petunjuk Instalasi
1. Instal Aspose.Words: Gunakan NuGet Package Manager di Visual Studio.
   - Buka proyek Anda di Visual Studio.
   - Klik kanan pada proyek Anda di Solution Explorer.
   - Pilih Kelola Paket NuGet.
   - Pencarian untuk `Aspose.Words` dan menginstalnya.

2. Tambahkan Petunjuk Penggunaan: Dalam berkas kode Anda, sertakan petunjuk berikut untuk menggunakan kelas Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
