Contoh ini menunjukkan penggunaan Aspose.Words untuk .NET untuk membuat dokumen Word dan menambahkan aturan horizontal menggunakan DocumentBuilder. Ikuti langkah-langkah di bawah ini untuk belajar bagaimana memformat dokumen Anda secara efektif.

## Instruksi Instalasi
1. Instal Aspose.Words: Gunakan NuGet Package Manager di Visual Studio.
   * Buka proyekmu di Visual Studio.
   * Klik kanan pada proyek Anda di Solution Explorer.
   * Pilih Manage NuGet Packages.
   * Carilah`"Aspose. Kata-kata"`dan menginstalnya.

2. Tambahkan Direksi Penggunaan: Dalam file kode Anda, sertakan arahan berikut untuk menggunakan kelas Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
