## Instruksi Instalasi
1. Pasang aspose.Words: Gunakan Manajer Paket NuGet di Studio Visual.
   * Buka proyekmu di Studio Visual.
   * Klik kanan pada proyek Anda di Solusi Explorer.
   * Pilih Kelola Paket NuGet.
   * Cari`Sebagaipose.Words`dan memasangnya.

2. Tambahkan Penggunaan Direktif: Dalam berkas kode Anda, sertakan direktif berikut untuk menggunakan kelas Aspose.Words:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
