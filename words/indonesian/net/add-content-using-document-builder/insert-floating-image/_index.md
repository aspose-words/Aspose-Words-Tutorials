## Instruksi Instalasi
1. Pasang aspose.Words: Tambahkan paket Aspose.Words NuGet ke projek .NET Anda. Anda dapat melakukan ini dengan menjalankan perintah berikut dalam Konsol Manajer Paket:
   ```
   Install-Package Aspose.Words
   ```

2. Tambahkan Namespace yang Diperlukan: Pastikan berkas kode Anda termasuk namespace berikut:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
