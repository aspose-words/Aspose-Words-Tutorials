## Petunjuk Instalasi
1. Instal Aspose.Words: Tambahkan paket Aspose.Words NuGet ke proyek .NET Anda. Anda dapat melakukannya dengan menjalankan perintah berikut di Konsol Pengelola Paket:
   ```
   Install-Package Aspose.Words
   ```

2. Tambahkan Namespace yang Diperlukan: Pastikan file kode Anda menyertakan namespace berikut:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
