## Instruksi Instalasi
1. Instal Aspose.Words: Tambahkan paket NuGet Aspose.Words ke proyek .NET Anda. Anda dapat melakukan ini dengan menjalankan perintah berikut di Konsole Manajer Paket:
   ```
   Install-Package Aspose.Words
   ```

2. Tambahkan Nama Ruang yang Diperlukan: Pastikan file kode Anda mencakup nama ruang berikut:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
