# Masukkan Gambar Inline dalam Dokumen Word memakai Aspose.Words
Contoh ini menunjukkan bagaimana menyisipkan gambar inline ke dalam dokumen Word menggunakan Perpustakaan Aspose.Words untuk .NET. Dengan menggunakan`Pembangun Dokumen`kelas, anda dapat dengan mudah memposisikan dan menyisipkan gambar di lokasi kursor saat ini di dokumen anda. Panduan ini menyediakan implementasi langsung untuk membantu anda memulai dengan penanganan gambar dalam dokumen Word.

## Instruksi Instalasi
1. Pasang aspose.Words untuk .NET:
   * Gunakan Manajer Paket NuGet di Studio Visual.
   * Jalankan perintah berikut dalam Konsol Manajer Paket:
     ```
     Install-Package Aspose.Words
     ```

2. Atur Proyek Anda:
   * Buat projek .NET baru atau buka yang sudah ada.
   * Tambahkan penggunaan direktif yang diperlukan di puncak berkas kode Anda:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
