# Masukkan Gambar Inline di Word Document menggunakan Aspose.Words
Contoh ini menunjukkan bagaimana memasukkan gambar inline ke dalam dokumen Word menggunakan perpustakaan Aspose.Words untuk .NET. Dengan menggunakan `DocumentBuilder`kelas, Anda dapat dengan mudah memposisikan dan memasukkan gambar di lokasi kursor saat ini dalam dokumen Anda. Panduan ini memberikan implementasi yang mudah untuk membantu Anda memulai dengan penanganan gambar di dokumen Word.

## Instruksi Instalasi
1. Instal Aspose.Words untuk .NET:
   * Gunakan NuGet Package Manager di Visual Studio.
   * Jalankan perintah berikut di Konsole Paket Manager:
     ```
     Install-Package Aspose.Words
     ```

2. Buat Proyek Anda:
   * Buat proyek .NET baru atau buka yang sudah ada.
   * Tambahkan instruksi yang diperlukan di bagian atas file kode Anda:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
