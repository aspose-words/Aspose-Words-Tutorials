# Masukkan Gambar Sebaris dalam Dokumen Word menggunakan Aspose.Words
Contoh ini menunjukkan cara memasukkan gambar sebaris ke dalam dokumen Word menggunakan pustaka Aspose.Words untuk .NET. Dengan memanfaatkan `DocumentBuilder` kelas, Anda dapat dengan mudah memposisikan dan menyisipkan gambar di lokasi kursor saat ini dalam dokumen Anda. Panduan ini menyediakan implementasi langsung untuk membantu Anda memulai penanganan gambar dalam dokumen Word.

## Petunjuk Instalasi
1. Instal Aspose.Words untuk .NET:
   - Gunakan NuGet Package Manager di Visual Studio.
   - Jalankan perintah berikut di Konsol Manajer Paket:
     ```
     Install-Package Aspose.Words
     ```

2. Siapkan Proyek Anda:
   - Buat proyek .NET baru atau buka yang sudah ada.
   - Tambahkan perintah penggunaan yang diperlukan di bagian atas berkas kode Anda:
     ```csharp
     using Aspose.Words;
     using Aspose.Words.Drawing;
     ```
