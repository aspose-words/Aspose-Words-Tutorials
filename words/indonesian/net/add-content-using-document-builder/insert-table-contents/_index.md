# Buat Dokumen Kata dengan Daftar Isi
Contoh ini menunjukkan bagaimana menggunakan Apose.Words untuk .NET untuk membuat dokumen Word yang termasuk tabel interaktif dari Isi (TOC) yang dihasilkan dengan kelas Document Builder. Kode ini menggambarkan bagaimana memasukkan heading berbagai tingkat, format mereka, dan menyimpan dokumen yang dihasilkan sebagai berkas .dox. Ikuti langkah-langkah di bawah untuk implementasi yang mudah.

## Instruksi Instalasi
1. Prasyarat: Pastikan Anda telah .NET terpasang pada mesin Anda.
2. Pasang aspose.Words: 
   * Gunakan NuGet Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
   * Atau, cari "Aspose.Words" di NuGet Package Manager GUI.
