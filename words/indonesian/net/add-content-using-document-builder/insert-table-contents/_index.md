# Buat Dokumen Word dengan Tabel Konten
Contoh ini menunjukkan bagaimana menggunakan Aspose.Words untuk .NET untuk membuat dokumen Word yang mencakup Tabel Konten interaktif (TOC) yang dihasilkan menggunakan kelas DocumentBuilder. Kode ini menggambarkan bagaimana memasukkan judul dari berbagai tingkat, format mereka, dan menyimpan dokumen yang dihasilkan sebagai file .docx. Ikuti langkah-langkah di bawah untuk implementasi yang mudah.

## Instruksi Instalasi
1. Persyaratan: Pastikan Anda memiliki .NET diinstal pada mesin Anda.
2. Instal Aspose.Words: 
   * Gunakan NuGet Package Manager Console:
     ```
     Install-Package Aspose.Words
     ```
   * Atau, carilah untuk "Aspose.Words" di GUI NuGet Package Manager.
