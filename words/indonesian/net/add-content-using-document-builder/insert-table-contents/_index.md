Contoh ini menunjukkan cara menggunakan Aspose.Words untuk .NET guna membuat dokumen Word yang menyertakan Daftar Isi (TOC) interaktif yang dibuat menggunakan kelas DocumentBuilder. Kode tersebut mengilustrasikan cara menyisipkan judul dari berbagai tingkatan, memformatnya, dan menyimpan dokumen yang dihasilkan sebagai file .docx. ## Petunjuk Instalasi
1. Prasyarat: Pastikan Anda telah menginstal .NET di komputer Anda.
2. Instal Aspose.Words: 
   - Gunakan Konsol Manajer Paket NuGet:
     ```
     Install-Package Aspose.Words
     ```
   - Atau, cari "Aspose.Words" di GUI NuGet Package Manager.
