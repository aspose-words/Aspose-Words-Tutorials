# Membuat Dokumen Word dengan Tabel Menggunakan Aspose.Words
  
Contoh ini menunjukkan cara membuat dokumen Word dengan tabel menggunakan pustaka Aspose.Words untuk .NET. Kode tersebut menginisialisasi dokumen baru, membuat tabel dengan dua baris, dan memformat sel dengan berbagai gaya dan orientasi. Anda akan belajar memanipulasi konten sel, perataan, dan membangun dokumen terstruktur secara terprogram.

## Petunjuk Instalasi:  
1. Instal Aspose.Words untuk .NET:
   - Menggunakan NuGet Package Manager:
     - Buka proyek Anda di Visual Studio.
     - Klik kanan pada proyek di Solution Explorer dan pilih "Kelola Paket NuGet."
     - Cari "Aspose.Words" dan klik "Instal."

   - Atau, gunakan Konsol Manajer Paket:
     ```
     Install-Package Aspose.Words
     ```
