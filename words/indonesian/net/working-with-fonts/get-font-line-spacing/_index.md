# Dapatkan Spasi Font Line di Word Document
Aspose.Words for .NET simplifies Word document creation and manipulation. This example demonstrates how to initialize a Word document, set a custom font, and retrieve the line spacing of a specific font. Whether you're automating document generation or analyzing text styles, this guide provides clear instructions and practical code snippets to achieve your goals efficiently. Aspose.Words for .NET is a powerful library for developers working with Word documents in C# or VB.NET.

## Instruksi Instalasi  
Untuk menggunakan contoh ini, ikuti langkah-langkah berikut:  
1. Unduh dan Instal Aspose.Words untuk .NET  
   * Kunjungi [Aspose.Words untuk halaman download .NET](https://releases.aspose.com/words/net/)dan download versi terbaru.  
   * Keluarkan paket yang diunduh dan referensi `"Aspose. Words".`file dalam proyek Anda.  

2. Instal melalui NuGet  
   * Buka proyekmu di Visual Studio.  
   * Pergi ke Alat > NuGet Package Manager > Mengelola NuGet Packages untuk Solution.  
   * Carilah`"Aspose. Kata-kata"`di NuGet Package Manager dan menginstal paket.  

3. Minta Lisensi (Pilih tapi disarankan)  
   * Dapatkan lisensi dari [Aspose halaman pembelian](https://purchase.aspose.com/buy)Aku tidak tahu.  
   * Gunakan a [lisensi sementara](https://purchase.aspose.com/temporary-license/)untuk evaluasi jika diperlukan.  
   * Tambahkan kode berikut di awal aplikasi Anda untuk menerapkan lisensi:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
