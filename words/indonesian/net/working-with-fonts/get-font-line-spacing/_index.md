# Dapatkan Jarak Baris Fonta dalam Dokumen Word
Aspose.Words for .NET simplifies Word document creation and manipulation. This example demonstrates how to initialize a Word document, set a custom font, and retrieve the line spacing of a specific font. Whether you're automating document generation or analyzing text styles, this guide provides clear instructions and practical code snippets to achieve your goals efficiently. Aspose.Words for .NET is a powerful library for developers working with Word documents in C# or VB.NET.

## Instruksi Instalasi  
Untuk menggunakan contoh ini, ikutilah langkah - langkah ini:  
1. Unduh dan Instal Apose.Words untuk .NET  
   * Kunjungi[Sebagaipose.Words untuk halaman unduh .NET](https://releases.aspose.com/words/net/)dan download versi terbaru.  
   * Ekstrak paket yang diunduh dan referensi`Sebagaipose.Words.dll`file dalam proyek Anda.  

2. Pasang via NuGet  
   * Buka proyekmu di Studio Visual.  
   * Pergi ke Perkakas > Manajer Paket NuGet > Kelola Paket NuGet untuk Solusi.  
   * Cari`Sebagaipose.Words`di NuGet Package Manager dan instal paket.  

3. Terapkan suatu Lisensi (Opsional tetapi disarankan)  
   * Mendapatkan lisensi dari[aspose Pembelian page](https://purchase.aspose.com/buy).  
   * Gunakan[lisensi sementara](https://purchase.aspose.com/temporary-license/)untuk evaluasi jika diperlukan.  
   * Tambahkan kode berikut di awal aplikasi Anda untuk menerapkan lisensi:  
     ```csharp
     License license = new License();
     license.SetLicense("Aspose.Words.lic");
     ```
