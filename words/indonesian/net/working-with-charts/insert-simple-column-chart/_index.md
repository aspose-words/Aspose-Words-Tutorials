## Petunjuk Instalasi  
Untuk menggunakan contoh ini, pastikan Anda telah menginstal Aspose.Words for .NET. Ikuti langkah-langkah berikut:  

1. Unduh Aspose.Words untuk .NET:  
   Mengunjungi [Unduh Aspose.Words untuk .NET](https://releases.aspose.com/words/net/) dan unduh versi terbaru.  

2. Instal Perpustakaan:  
   - Tambahkan paket Aspose.Words NuGet ke proyek Anda. Jalankan perintah berikut di Konsol Pengelola Paket NuGet:  
     ```sh
     Install-Package Aspose.Words
     ```  
   - Atau, tambahkan `.dll` file ke referensi proyek Anda secara manual.  

3. Dapatkan Lisensi:  
   - Dapatkan lisensi dari [Aspose Pembelian](https://purchase.aspose.com/buy) atau minta uji coba gratis dari [Uji Coba Gratis Aspose](https://releases.aspose.com/).  

4. Terapkan Lisensi:  
   Tambahkan kode berikut ke proyek Anda untuk mengaktifkan lisensi:  
   ```csharp
   License license = new License();
   license.SetLicense("path-to-your-license-file.lic");
   ```
