## Petunjuk Instalasi  

1. Unduh Aspose.Words untuk .NET:  
   Dapatkan versi terbaru dari resmi [Halaman unduhan Aspose.Words untuk .NET](https://releases.aspose.com/words/net/).  

2. Instal melalui NuGet:  
   Tambahkan pustaka ke proyek Anda menggunakan NuGet Package Manager. Jalankan perintah berikut di Package Manager Console:  
   ```bash
   Install-Package Aspose.Words
   ```  

3. Dapatkan Lisensi:  
   - Untuk evaluasi, mintalah [lisensi sementara](https://purchase.aspose.com/temporary-license/).
