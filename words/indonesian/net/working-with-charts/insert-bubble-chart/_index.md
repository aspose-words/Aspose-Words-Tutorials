# Menyisipkan Bagan Gelembung di Word Menggunakan Aspose.Words

Tambahkan bagan Bubble ke dokumen Word Anda dengan mudah menggunakan Aspose.Words untuk .NET! Panduan ini memandu Anda membuat dokumen baru, menyisipkan bagan Bubble, menambahkan seri data dengan nilai X, Y, dan ukuran, serta menyimpan dokumen. Dengan Aspose.Words, menangani dokumen Word secara terprogram tidak pernah semudah ini.

## Petunjuk Instalasi  

1. Unduh Aspose.Words untuk .NET  
   Dapatkan perpustakaan dari [Halaman unduhan Aspose.Words](https://releases.aspose.com/words/net/).  

2. Instal melalui Pengelola Paket NuGet  
   - Buka proyek Visual Studio Anda.  
   - Navigasi ke Alat > Manajer Paket NuGet > Kelola Paket NuGet untuk Solusi.  
   - Pencarian untuk `Aspose.Words` di tab Telusuri.  
   - Pilih paket dan klik Instal.  

3. Instal melalui Konsol Pengelola Paket NuGet  
   Jalankan perintah berikut di Konsol Manajer Paket:  
   ```bash
   Install-Package Aspose.Words
   ```  

4. Tambahkan Lisensi  
   Untuk fungsionalitas penuh, terapkan lisensi. Anda bisa mendapatkan lisensi sementara dari [Halaman Lisensi Sementara Aspose](https://purchase.aspose.com/temporary-license/).
