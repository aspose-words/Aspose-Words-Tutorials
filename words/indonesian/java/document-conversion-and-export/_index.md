## Pertanyaan yang Sering Diajukan

**Q: Bisakah saya mengonversi DOCX ke PDF di server Linux?**  
A: Ya. Aspose.Words for Java bersifat platform‑independen dan bekerja pada sistem operasi apa pun dengan JRE yang didukung.

**Q: Apakah saya perlu menginstal Microsoft Office di server?**  
A: Tidak. Konversi dilakukan sepenuhnya oleh pustaka Aspose.Words.

**Q: Bagaimana saya dapat mempertahankan font kustom selama konversi?**  
A: Gunakan `PdfSaveOptions.setEmbedFullFonts(true)` untuk menyematkan semua font yang digunakan ke dalam PDF yang dihasilkan.

**Q: Apakah memungkinkan mengonversi file DOCX yang dilindungi kata sandi?**  
A: Tentu saja. Muat dokumen dengan `LoadOptions.setPassword("yourPassword")` sebelum menyimpan sebagai PDF.

**Q: Apa cara yang disarankan untuk batch‑convert banyak dokumen?**  
A: Buat metode yang dapat digunakan kembali yang memuat, mengonversi, dan membuang setiap dokumen, lalu jalankan dalam thread‑pool untuk pemrosesan paralel.
