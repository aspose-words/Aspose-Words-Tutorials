### Bagaimana cara memulai dengan Aspose.Words untuk .NET?
Kelas `Document` mewakili dokumen Word dan menyediakan metode untuk memuat, memodifikasi, dan menyimpannya. `DocumentBuilder` adalah kelas pembantu yang menyederhanakan penambahan dan pemformatan konten dalam sebuah `Document`. Mulailah dengan menginstal paket NuGet `Aspose.Words`, tambahkan referensi ke `Aspose.Words.dll`, lalu buat instance kelas `Document` dengan path ke file sumber Anda. Dari sana Anda dapat menggunakan `DocumentBuilder` untuk menambahkan konten, menerapkan gaya, atau menyimpan file dalam salah satu format yang didukung.
