# Aspose.Words kullanarak Word Belgelerine Şekiller Ekle
Bu kılavuz, Word belgelerinde metin kutuları gibi şekillerin yerleştirilmesi ve manipüle edilmesi için Aspose.NET'in nasıl kullanılacağını gösterir. Şekilleri kesinlikle veya nispeten konumlandırmayı, dönme açılarını ayarlamayı ve belgeyi ISO 29500:2008 Geçişli DOCX biçiminde kaydetmeyi öğreneceksiniz.`OoxmlSaveOptions`. Bu örnek, Word belgelerindeki şekillerle çalışmayı basitleştirir ve geliştiricilerin görsel olarak çekici içerik oluşturmasını kolaylaştırır.

## Kurulum Talimatları  
1. .NET için Aspose.Words İndir:  
   Aspose.Words'un en son sürümünü .NET'ten alın[indirme sayfası](https://releases.aspose.com/words/net/).  

2. NuGet üzerinden yükleyin:  
   Projenizi Visual Studio'da açın ve NuGet Paket Yöneticisi'ni kullanarak kütüphaneyi kurun:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. İsim Alanı Referansı Ekle:  
   Kodunuzda, Aspose.Words için ad alanı ekleyin:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
