# Aspose.Words Kullanarak Word Belgelerine Şekiller Ekleme
Bu kılavuz, Word belgelerine metin kutuları gibi şekiller eklemek ve bunları düzenlemek için Aspose.Words for .NET'in nasıl kullanılacağını gösterir. Şekilleri mutlak veya göreceli olarak nasıl konumlandıracağınızı, dönüş açılarını nasıl ayarlayacağınızı ve belgeyi ISO 29500:2008 Transitional DOCX biçiminde nasıl kaydedeceğinizi öğreneceksiniz. `OoxmlSaveOptions`Bu örnek, Word belgelerinde şekillerle çalışmayı basitleştirerek geliştiricilerin görsel olarak çekici içerikleri kolaylıkla oluşturmasını sağlar.

## Kurulum Talimatları  
1. .NET için Aspose.Words'ü indirin:  
   Aspose.Words for .NET'in en son sürümünü şu adresten edinin: [indirme sayfası](https://releases.aspose.com/words/net/).  

2. NuGet üzerinden kurulum:  
   Projenizi Visual Studio'da açın ve NuGet Paket Yöneticisi'ni kullanarak kütüphaneyi yükleyin:  
   ```bash  
   Install-Package Aspose.Words  
   ```  

3. Ad Alanı Referansı Ekle:  
   Kodunuza Aspose.Words için ad alanını ekleyin:  
   ```csharp  
   using Aspose.Words;  
   using Aspose.Words.Saving;  
   using Aspose.Words.Drawing;  
   ```
