# Yüzen Görüntülü Word Belgesi Oluşturun

Bu örnek, .NET için Aspose.Words kullanarak bir Word belgesinin nasıl oluşturulacağını ve yüzen düzene sahip bir görüntünün nasıl ekleneceğini gösterir. Sağlanan kod bir Belge ve Belge Oluşturucu başlatır, belirtilen boyutlara ve konumlandırmaya sahip bir görüntü ekler ve son olarak belgeyi kaydeder. Bu kılavuz, Aspose.Words kullanarak belge oluşturma yeteneklerini geliştirmek isteyen geliştiriciler için idealdir.

## Kurulum Talimatları
1. Aspose.Words'ü yükleyin: Aspose.Words NuGet paketini .NET projenize ekleyin. Bunu Paket Yöneticisi Konsolu'nda aşağıdaki komutu çalıştırarak yapabilirsiniz:
   ```
   Install-Package Aspose.Words
   ```

2. Gerekli Ad Alanlarını Ekleyin: Kod dosyanızın aşağıdaki ad alanlarını içerdiğinden emin olun:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```
