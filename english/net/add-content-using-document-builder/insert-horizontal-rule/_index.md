This example demonstrates the use of Aspose.Words for .NET to create a Word document and add a horizontal rule using DocumentBuilder. Follow the steps below to learn how to format your documents effectively.

## Installation Instructions
1. Install Aspose.Words: Use NuGet Package Manager in Visual Studio.
   - Open your project in Visual Studio.
   - Right-click on your project in the Solution Explorer.
   - Select Manage NuGet Packages.
   - Search for `Aspose.Words` and install it.

2. Add Using Directive: In your code file, include the following directive to use Aspose.Words classes:
   ```csharp
   using Aspose.Words;
   using Aspose.Words.Drawing;
   ```

3. Run Your Application: Execute your project to generate the Word document with the horizontal rule. Check the output path for your created file.
